Creazione di una risorsa Marketplace

Quando si seleziona l'opzione Crea pacchetto dal menu del mercato IDE, si accede alla seguente area di lavoro: Qui dovete innanzitutto selezionare l'identità dell'editore con cui desiderate pubblicare il pacchetto asset (che dovreste aver impostato dalla vostra Dashboard dell'editore), quindi indicare un Nome visualizzato e un Numero di versione. Verrà quindi creato un ID del pacchetto in base al nome visualizzato e all'account dell'editore utilizzato, ma è possibile modificarlo in un altro modo, purché mantenga il formato dell'URL inverso. È quindi necessario scegliere il tipo di pacchetto asset da creare:

  • Attività: si tratta di una raccolta di assets (sprites, scripts, objects, ecc...) che si desidera vendere sul Marketplace.
  • Demo: Si tratta di una demo per dimostrare un concetto o un'idea di gioco.
  • Tutorial: Il sito asset caricato è un tutorial, progettato per essere eseguito all'interno di GameMaker IDE (vedere questa pagina per maggiori dettagli).

Una volta fatto ciò, potete scegliere quali assets del progetto volete aggiungere al pacchetto Asset usando la sezione a destra. È possibile aggiungere tutti i assets o solo alcuni di essi, nonché rimuovere quelli aggiunti per errore. Si presume che il sito asset da caricare sia stato creato in precedenza in GameMaker e che il progetto attuale contenga tutto ciò che si desidera caricare.

RICORDA! Per poter creare o utilizzare i pacchetti di asset, è necessario aver precedentemente creato un certificato del profilo utente. È possibile farlo dalle Preferenze di Marketplace, come spiegato qui.

Quando si è soddisfatti della selezione di assets e si desidera caricare il proprio pacchetto su Marketplace, è sufficiente fare clic sul pulsante Ok in basso e il pacchetto asset verrà creato e caricato su Marketplace. Al termine del caricamento, si aprirà un nuovo workspace con una vista browser della pagina asset da modificare. Per ulteriori informazioni su come configurare questa operazione, consultare la pagina seguente:

  • Elencare un'attività

Se si deve aggiornare un pacchetto Marketplace Asset, si fa più o meno come quando si è creato il pacchetto originale, solo che ora si deve specificare il pacchetto da aggiornare da un elenco a discesa di pacchetti disponibili:

Durante l'aggiornamento, non sarà possibile modificare l'ID del pacchetto e si deve tenere presente che il numero di versione mostrato sarà il numero di versione corrente e si deve incrementare se necessario, altrimenti l'aggiornamento fallirà (poiché non si possono avere due pacchetti di risorse con lo stesso numero di versione). C'è anche un campo in cui è possibile aggiungere del testo per il registro delle modifiche di asset, che verrà caricato insieme a asset e visualizzato nella pagina Marketplace.

IMPORTANTE: Assets realizzato con la versione 2.3+ di GameMaker sarà disponibile solo per gli utenti che utilizzano la versione 2.3 o superiore. Se state aggiornando un asset da una versione precedente, quel asset sarà ancora disponibile per gli utenti che non hanno aggiornato e coesisterà con la versione aggiornata 2.3+.

Caricamento attraverso la pagina web

È possibile caricare assets attraverso la pagina web Marketplace piuttosto che attraverso IDE, cosa che può essere necessaria se il caricamento attraverso IDE non è riuscito per qualsiasi motivo. Per cominciare, si deve creare il pacchetto e tentare di caricarlo attraverso IDE come mostrato sopra; se il caricamento non riesce, si deve andare alla finestra di output e controllare il registro per trovare una riga che mostri dove è stato salvato il pacchetto creato prima di tentare il caricamento. Ad esempio:

Su Windows:

Created package file:
C:\Users\WINDOWSUSERNAME\AppData\Roaming\GameMaker\GM_USERNAME\Marketplace\Packages\com.publishername.assetname.yymp

Su macOS:

Created package file:
/Users/MACUSERNAME/.config/GameMaker/GM_USERNAME/Marketplace/Packages/com.publishername.assetname.yymp

È necessario navigare nella posizione del file e copiare il file *.YYMP in una posizione sicura, poiché questo sarà il file che verrà caricato sul sito Marketplace.

A questo punto è necessario aprire il sito Marketplace nel browser web e navigare nell'elenco asset per il sito asset, quindi andare alla sezione Versioni e fare clic sul pulsante Carica versione, come mostrato nell'immagine seguente:

In questo modo si accede alla pagina di caricamento della nuova versione, dove è possibile aggiungere informazioni sulla versione da caricare, se necessario, e fornire il file facendo clic sul pulsante Scegli file e selezionando il file *.YYMP creato in precedenza:

L'aggiunta del file caricherà le versioni e mostrerà una barra di avanzamento; quando il caricamento è terminato e si è soddisfatti dei dettagli, è possibile fare clic sul pulsante Salva per salvare il sito asset nell'elenco. Si noti che il sito asset rimarrà in fase di elaborazione per alcuni minuti prima di poter essere pubblicato.
