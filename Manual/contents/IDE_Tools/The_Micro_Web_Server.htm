Il micro server web

La funzione principale di questo piccolo strumento è quella di connettersi e servire file ad alcune delle piattaforme di destinazione disponibili per GameMaker (come la destinazione HTML5). Serve anche i file per il manuale di GameMaker quando viene usato offline.

Quando si esegue un gioco su un obiettivo che richiede il Micro Web Server, si vedrà una nuova finestra aperta mentre il gioco viene servito al dispositivo scelto:

NOTA La prima volta che eseguite un gioco su un obiettivo che richiede il Micro Web Server, potreste ottenere un avviso del firewall. Dovreste permettere al web server di comunicare attraverso il firewall se questo accade.

Quando Micro Web Server è in funzione, puoi accedere al tuo gioco da qualsiasi dispositivo locale (e anche in remoto, se il port forwarding è impostato correttamente). Basta accedere all'indirizzo IP locale del tuo computer attraverso la porta mostrata in questa finestra, e sarai in grado di giocare al tuo gioco da un altro dispositivo, come un telefono cellulare, o un altro computer.

Per esempio, diciamo che l'indirizzo IP locale del tuo computer è 192.168.0.110(leggi qui per sapere come trovarlo), e la porta mostrata nella finestra Micro Web Server è 51264. Dopo aver eseguito il gioco sul tuo computer, ti connetterai a 192.168.0.110:51264 da un browser sul tuo cellulare, e giocherai al tuo gioco.

Il Web Server mostrerà una certa quantità di informazioni di debug relative ai file che sono stati serviti, e ha un pulsante per cancellarle in basso. Accanto al pulsante "Clear" c'è il path al file JS principale del progetto, mentre sotto c'è l'opzione Network Speed. Questa è impostata di default su "No Limit", ma cliccandoci sopra si aprirà una lista di diverse opzioni di rete:

Cambiando questa impostazione cambierà anche l'opzione Dimensione pacchetto a destra. Puoi anche modificarla manualmente, nel qual caso la velocità di rete sarà impostata su "Custom". Impostandola su un valore diverso da quello massimo, la velocità di trasferimento della rete verrà effettivamente strozzata e ti permetterà di testare i giochi in un ambiente a bassa larghezza di banda.

Potete anche vedere l'elenco degli IP consentiti insieme al numero di porta utilizzato, ma questi non possono essere modificati in Micro Web Server. Dovrebbero invece essere impostati nelle impostazioni di Device Manager per il dato dispositivo di destinazione, o nelle Preferenze HTML5 quando si usa il dispositivo "Default".
