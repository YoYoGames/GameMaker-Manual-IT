Impostare il plugin Git

Prima di poter usare il plugin Git, dobbiamo impostare l'eseguibile Git da usare andando nella sezione Plugins - Source Control (Git) delle Preferenze:

L'eseguibile Git path dovrebbe essere riempito automaticamente se viene trovato da GameMaker, tuttavia se il campo è vuoto, puoi cliccare sul pulsante e selezionare manualmente il file git.exe incluso nella cartella cmd della tua installazione Git, che di solito è dentro C:\Program Files\Git\.

Per il resto delle opzioni di Git, vedi la pagina Preferenze del plugin.
