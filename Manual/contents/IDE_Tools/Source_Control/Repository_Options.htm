Opzioni di deposito

Avendo impostato l'eseguibile Git e abilitato Source Control per il tuo progetto, ora hai bisogno di collegare GameMaker IDE a un repository, il che è fatto in uno dei due modi:

  • Creare un repository di progetto
  • Clonare un repository esistente

Nel primo caso, creando un repository di progetto, questo creerà un repository Git direttamente dove si trova il tuo progetto. Nel gergo di Git questo è un "non-bare repository", quindi Push/Pull non ha senso in questo caso, ma puoi ancora fare il commit e tenere traccia delle modifiche al progetto, così come ripristinare le modifiche e le revisioni se necessario. Per impostarlo, devi solo cliccare su Source Control > Create Project Repository nel menu principale.

Come funzione di convenienza, puoi anche clonare un repository esterno attraverso IDE. Di nuovo, avrai bisogno del nome utente e della password per il repository a cui ti connetterai nelle tue preferenze di identità, e poi è solo un caso di cliccare su Source Control > Clone Repository.
