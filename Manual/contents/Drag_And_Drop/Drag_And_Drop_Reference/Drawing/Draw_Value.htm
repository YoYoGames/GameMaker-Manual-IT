Disegnare il valore

Con questa azione potete disegnare una didascalia string e/o una variabile o un valore sullo schermo. L'argomento della didascalia è un string che si desidera aggiungere prima del valore, e può essere lasciato vuoto se necessario, ma se si desidera includere del testo dovrebbe essere avvolto tra virgolette, e il valore può essere una variabile, un valore fisso, o un altro string - nel qual caso dovrebbe essere anch'esso racchiuso tra virgolette - ma non può essere lasciato vuoto. Poi si dà la posizione in cui il string dovrebbe essere disegnato, sia come posizione assoluta o come posizione relativa all'istanza che disegna il testo. Nota che il modo in cui il testo apparirà quando disegnato dipenderà dal font usato, dal colore usato e dall'allineamento. Tutte queste cose possono essere cambiate prima del disegno, come mostrato nell'esempio qui sotto.

NOTA: Questa azione è solo per l'uso nei vari eventi di disegno, e non disegnerà nulla se usata altrove.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Caption La didascalia da disegnare (un string), può essere lasciata vuota
Value Il valore da disegnare (reale, variabile o string)
x La posizione x a cui disegnare
y La posizione y a cui disegnare

Esempio:

Il codice del blocco d'azione qui sopra imposta un font, il colore font e poi allinea il testo per disegnare dal centro prima di disegnare effettivamente il valore "level_num" insieme alla didascalia "Livello: ".
