Impostare il disegno dell'alfa

Con questa azione si può impostare l'alfa da utilizzare per qualsiasi forma (rettangolo/ellisse/linea) o per qualsiasi testo che viene disegnato successivamente. Notate che questa azione non deve essere chiamata prima di ogni azione di disegno, e può essere impostata una sola volta all'inizio del gioco. Alpha è l'impostazione che controlla come le cose "trasparenti" saranno disegnate e si basa su una scala da 0 a 1, dove 0 è completamente trasparente e 1 è completamente opaco.

Argomenti:

Argomento Descrizione
Alpha Il valore alfa da usare (da 0 a 1, il default è 1)
