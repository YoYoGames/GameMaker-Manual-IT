Disegnare se stessi

Quando aggiungete qualsiasi tipo di azione o codice al Draw Event di un object, state dicendo a GameMaker che state prendendo il controllo e che gli direte cosa disegnare d'ora in poi (per default un'istanza disegnerà il sprite assegnatogli nell'editor di object quando non ci sono azioni nell'evento di disegno principale). Questo significa che qualsiasi sprite assegnato non sarà più disegnato a meno che non lo diciate esplicitamente, che è quando usereste questa azione. Questo semplicemente disegna il sprite assegnato per il object con qualsiasi trasformazione che può essere stata impostata in eventi precedenti (come il ridimensionamento, la fusione o i cambiamenti di alfa) nella sottoimmagine corrente.

NOTA: Questa azione è solo per l'uso nei vari eventi di disegno, e non disegnerà nulla se usata altrove.

Sintassi dell'azione:

Esempio:

Il codice del blocco d'azione di cui sopra disegna una "ombra" sprite nella stessa posizione relativa dell poi disegna l'istanza sprite sopra di essa.

Indietro: Azioni di disegno
Prossimo: Disegnare il valore
