Ottenere il carattere Draw

Con questa azione è possibile ottenere il valore ID del font attualmente assegnato per il disegno. Il valore restituito sarà o -1 per nessun font assegnato, o un valore >= 0 che rappresenta una costante font dal Browser Asset. Il valore restituito sarà memorizzato nella variabile di destinazione specificata, che può essere stata creata in precedenza o può essere una nuova variabile temporanea (se si seleziona la casella di controllo "Temp").
