Se il pulsante del gamepad è premuto

Con questa azione è possibile eseguire il polling del gamepad per vedere se un qualsiasi pulsante è attualmente premuto. L'azione restituirà true ogni volta che il pulsante del gamepad viene premuto (e tenuto premuto), o false altrimenti, anche se se si clicca il not è possibile controllare se il pulsante non è premuto, cioè: l'azione restituirà true mentre nessun pulsante è premuto e false se c'è. Se avete bisogno di controllare solo la pressione di un singolo pulsante allora utilizzare l'azione if gamepad button down.

Se il pulsante da controllare è un pulsante analogico, allora la pressione non sarà rilevata finché il valore del pulsante non avrà superato la soglia impostata usando l'azione Set Gamepad Button Threshold.

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite se il blocco "if" valuta true, mentre qualsiasi azione abbandonata altrove sarà eseguita dopo il blocco "if".

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Gamepad L'indice del gamepad.
Button Il nome del pulsante da controllare.
Not Negare il controllo (vero diventa falso e viceversa)

Esempio:

Il codice del blocco d'azione di cui sopra interroga lo stato dei pulsanti del gamepad 0 ad ogni passo e se un pulsante viene tenuto premuto, controlla la pressione iniziale del pulsante. Se il controllo è true sulla pressione iniziale, viene riprodotto un suono, poi, mentre il pulsante è tenuto premuto, la velocità dell'istanza viene impostata a 10. Se il pulsante non viene tenuto premuto, viene effettuato un controllo al rilascio del pulsante per resettare nuovamente la velocità a 0.
