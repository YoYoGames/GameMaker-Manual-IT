Scrivere nel file Ini

Con questa azione puoi scrivere un valore numerico o un string in un file Ini. Dovete aver precedentemente aperto il file Ini per la lettura/scrittura usando l'azione Open Ini File, e poi con questa azione date il tipo di dati da scrivere (un numero reale o un string racchiuso in ""), così come l'intestazione della sezione, la chiave all'interno di quella sezione e il valore della chiave. Per aiutare a visualizzare la struttura di un file Ini, vedi questa immagine:

Come potete vedere, i file Ini sono divisi in sezioni - dove ogni sezione ha il suo nome di intestazione - e poi ogni sezione è successivamente divisa in varie coppie chiave/valore (un po' come una struttura dati Map). Si noti che si possono creare i file Ini da soli usando qualsiasi software di elaborazione del testo (come Notepad su Windows) e poi includerli come file incluso per leggere e scrivere nel tuo gioco.

IMPORTANTE! Cercare di leggere i dati da un file Ini che non è stato aperto in precedenza utilizzando l'azione Open Ini File causerà un grave errore e il crash del gioco.

IMPORTANTE! Questo scrive solo i dati in memoria e per scrivere finalmente i dati nel file su disco, dovete chiamare Close Ini File, altrimenti i dati non saranno salvati.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Type Il tipo di dati da scrivere, o un string (racchiuso tra virgolette ""), o un real number
Section La sezione del file Ini su cui scrivere (come string)
Key La chiave all'interno della sezione su cui scrivere (come string)
Value Il valore da scrivere, o un string o un numero reale in base all'argomento "type" (sopra)

Esempio:

Il codice del blocco d'azione di cui sopra aprirà un ini poi scriverà i dati in tre chiavi diverse sotto la stessa intestazione di sezione (una string e due numeri reali) prima di chiudere nuovamente il file ini (scrivendo così i dati su disco).
