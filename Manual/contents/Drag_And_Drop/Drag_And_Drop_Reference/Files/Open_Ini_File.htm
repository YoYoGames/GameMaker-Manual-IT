Aprire il file Ini

Con questa azione potete aprire un file Ini per la lettura e/o la scrittura. Se il file Ini non esiste nella posizione che state controllando, GameMaker può crearne uno, ma solo se ci scrivete dei dati. Se avete solo letto informazioni da il file Ini, allora verranno restituiti i valori predefiniti per la funzione di lettura, ma il file Ini non verrà effettivamente creato.

Nota che puoi avere solo un file Ini aperto alla volta e ricordati di usare Close Ini File una volta che hai finito di leggere/scrivere dal file Ini perché le informazioni non sono effettivamente memorizzate sul disco fino ad allora (sono conservate in memoria fino a quando il file viene chiuso).

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Filename Il nome - come string e con il .ini extension - del file da aprire

Esempio:

Il codice del blocco d'azione di cui sopra aprirà un file ini per l'uso, quindi otterrà il valore associato a la chiave "name" sotto l'intestazione "player". Se il nome restituito corrisponde al valore predefinito del nome (cioè: il file, la sezione o la chiave non esistono) allora il file ha una variabile globale scritta prima di essere chiuso (e scrivere i nuovi dati sul disco).
