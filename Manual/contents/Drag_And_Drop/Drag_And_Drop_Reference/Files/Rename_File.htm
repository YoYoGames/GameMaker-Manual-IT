Rinominare il file

Con questa azione puoi rinominare qualsiasi file che è stato salvato dal tuo gioco. Date il nome del file da cambiare (come string) e poi il nuovo nome (sempre come string e includendo il file extension), e il file sarà rinominato. Nota che non puoi rinominare qualsiasi file che è stato fornito come file incluso con il tuo gioco (vedi la sezione sul File System per maggiori informazioni).

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Filename Il nome (come string) del file da rinominare
New Name Il nuovo nome (come string) per il file

Esempio:

Il codice del blocco d'azione di cui sopra controllerà per vedere se il file "checkpoint.sav" esiste, e se controlla se il file "checkpoint_OLD.sav" esiste. Se anche quel file esiste, allora viene cancellato, e poi il file originale "checkpoint.sav" viene rinominato in "checkpoint_OLD.sav". Infine un buffer viene salvato come "checkpoint.sav" (essenzialmente stiamo facendo il backup di un file salvato buffer ogni volta che salviamo, in modo che ci sia sempre un salvataggio "attuale" e un file di salvataggio "vecchio").
