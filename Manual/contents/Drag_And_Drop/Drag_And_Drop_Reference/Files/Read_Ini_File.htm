Leggere il file Ini

Con questa azione puoi leggere un pezzo di dati da un file Ini. Dovete aver precedentemente aperto il file Ini per la lettura/scrittura usando l'azione Open Ini File, e poi con questa azione date il tipo di dati da leggere (sia un numero reale o un string racchiuso in ""), così come l'intestazione della sezione e la chiave all'interno di quella sezione da cui prendere i dati. Si può anche fornire un valore predefinito che può essere sia un numero reale che un string, e il tipo dovrebbe coincidere con il "Tipo" impostato nell'azione. Questa azione predefinita sarà restituita dall'azione se si verifica una delle seguenti situazioni:

  • Il file Ini che hai aperto non esiste
  • L'intestazione della sezione a cui fai riferimento non esiste
  • La chiave da cui stai cercando di ottenere dati non esiste

In una qualsiasi delle circostanze di cui sopra, si otterrà il valore predefinito restituito piuttosto che un valore dal file Ini. Qualunque valore venga restituito sarà memorizzato nella variabile di destinazione che specificate, che può anche essere creata in quel momento se contrassegnata come variabile locale temporanea.

IMPORTANTE: cercare di leggere i dati da un file Ini che non è stato aperto in precedenza utilizzando l'azione Open Ini File causerà un grave errore e il crash del gioco.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Type Il tipo di dati da leggere, o un string (racchiuso tra virgolette ""), o un real number
Section La sezione del file Ini da cui leggere (come string)
Key La chiave all'interno della sezione da cui leggere (come string)
Default Il valore predefinito da restituire se il file, la sezione o la chiave non esistono
Target La variabile di destinazione per tenere il valore di ritorno dall'azione

Esempio:

Il codice del blocco d'azione di cui sopra aprirà un file Ini per la lettura, poi userà i dati letti da esso per impostare 3 diverse variabili globali prima di chiudere nuovamente il file.
