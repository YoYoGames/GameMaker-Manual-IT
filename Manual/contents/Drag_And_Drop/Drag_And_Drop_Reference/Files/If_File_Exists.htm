Se il file esiste

Con questa azione potete controllare se esiste un file con il nome dato. Si seleziona il file da controllare (come un pungiglione e includendo il extension) e la funzione restituirà true se esiste, e false se non esiste. Notate che questo restituirà true se il file esiste nel bundle del gioco come file incluso o se il file è stato salvato mentre il gioco era in esecuzione (vedere la sezione sul File System per maggiori informazioni).

Esempio:

Il codice del blocco d'azione qui sopra creerà una variabile per contenere i dati di buffer e poi controllerà per vedere se un file di salvataggio buffer è stato fatto in precedenza. Se il file esiste, allora viene caricato nella variabile buffer che abbiamo creato, ma se non esiste, viene creato, scritto e poi salvato pronto per essere caricato la prossima volta.
