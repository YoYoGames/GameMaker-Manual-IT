Buffer di carico

Con questa azione è possibile caricare un buffer che è stato precedentemente salvato con l'azione Save Buffer. L'azione creerà un nuovo buffer, caricherà il file di dati in esso e restituirà il valore ID unico per identificare il buffer alla variabile. Si fornisce il nome del file da caricare (come string) e una variabile in cui memorizzare l'ID restituito buffer. Notate che se contrassegnate la variabile come variabile locale temporanea, dovete assicurarvi di cancellare il buffer che questa funzione crea prima la fine dell'azione script o dell'evento, altrimenti si perderà il riferimento ad esso e si otterrà una perdita di memoria quando la variabile verrà scartata.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Filename Il nome (come string) del file da aprire
Target La variabile di destinazione per tenere il valore ID unico per il buffer creato al caricamento

Esempio:

Il codice del blocco d'azione di cui sopra creerà una variabile per contenere i dati buffer e poi controllerà per vedere se un buffer file di salvataggio è stato fatto in precedenza. Se il file esiste, allora viene caricato nella variabile buffer che abbiamo creato, ma se non esiste, viene creato, scritto e poi salvato pronto per essere caricato la prossima volta.
