Salvare il buffer

Con questa azione potete salvare un file contenente i dati di un buffer. Voi fornite l'ID unico di buffer (come restituito quando avete creato il buffer con Create Buffer) e date un nome al file (come string, con il extension di vostra scelta) e il buffer sarà scritto in questo file pronto per essere caricato di nuovo usando Load Buffer.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Buffer Il valore unico buffer ID del buffer da salvare
Filename Il nome (come string) del file da salvare

Esempio:

Il codice del blocco d'azione di cui sopra creerà una variabile per contenere i dati buffer e poi controllerà per vedere se un buffer file di salvataggio è stato fatto in precedenza. Se il file esiste, allora viene caricato nella variabile buffer che abbiamo creato, ma se non esiste, viene creato, scritto e poi salvato pronto per essere caricato la prossima volta.
