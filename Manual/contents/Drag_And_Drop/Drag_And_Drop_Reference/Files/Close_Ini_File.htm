Chiudere il file Ini

Questa azione chiuderà il file Ini attualmente aperto. Questa azione dovrebbe essere chiamata nel momento in cui hai finito di leggere o scrivere su qualsiasi file Ini aperto (i file Ini sono aperti usando l'azione Open Ini File). Se non usi questa azione dopo aver usato l'azione Write To Ini File, allora non verrà scritto nulla su disco perché le informazioni del file sono tenute in memoria fino a quando questa azione viene chiamata, a quel punto vengono prese dalla memoria e scritte nel file su disco. Notate anche che se provate ad aprire un file Ini senza averne precedentemente chiuso un altro (o lo stesso) otterrete un errore.

Sintassi dell'azione:

Esempio:

Il codice del blocco d'azione di cui sopra aprirà un file ini per l'uso, quindi otterrà il valore associato a la chiave "name" sotto l'intestazione "player". Se il nome restituito corrisponde al valore predefinito del nome (cioè: il file, la sezione o la chiave non esistono) allora il file ha una variabile globale scritta su di esso prima di essere chiuso (e scrivere i nuovi dati sul disco).
