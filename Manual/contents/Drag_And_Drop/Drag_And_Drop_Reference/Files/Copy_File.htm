Copiare il file

Con questa azione puoi copiare qualsiasi file che è stato salvato dal tuo gioco o che è stato messo in bundle come parte degli Included Files. Si dà il nome del file da copiare (come string) e poi il nome del file da creare come copia del file originale (anche come string e includendo il file extension). Il file dato verrà duplicato e gli verrà dato il nuovo nome.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Filename Il nome (come string) del file da copiare
Copy To Il nome (come string) del file da creare come copia

Esempio:

Il codice del blocco d'azione di cui sopra controllerà per vedere se un file ini non esiste e se non esiste allora viene creato copiando un altro file.
