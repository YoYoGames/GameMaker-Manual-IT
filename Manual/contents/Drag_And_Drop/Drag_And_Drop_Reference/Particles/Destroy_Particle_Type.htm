Distruggere il tipo di particella

Questa azione "distruggerà" il dato tipo di particella creato dall'azione Create Particle Type (cioè: libererà la memoria utilizzata dal tipo di particella). Questa azione dovrebbe essere chiamata ogni volta che non hai più bisogno un tipo di particella nel tuo gioco, o ogni volta che desideri ricreare il tipo di particella (per esempio, appena prima di chiamare un Game Restart).

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Type Il valore ID del tipo da distruggere

Esempio:

Il codice del blocco d'azione di cui sopra controllerà se il tasto "Spazio e se viene rilevato, rimuove il sistema di particelle e i diversi tipi di particelle definiti nelle variabili globali date e poi riavvia il gioco.

Indietro: Azioni delle particelle
Avanti: Imposta dimensione delle particelle
