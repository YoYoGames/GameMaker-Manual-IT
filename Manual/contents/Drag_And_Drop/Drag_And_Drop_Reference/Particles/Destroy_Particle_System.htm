Distruggere il sistema di particelle

Questa azione "distruggerà" il sistema di particelle dato (cioè: libererà la memoria utilizzata dal sistema di particelle). Tu fornisci il valore ID unico per il sistema da distruggere, come restituito quando hai creato il sistema con l'azione Create Particle System. Questa azione dovrebbe essere chiamata ogni volta che non hai più bisogno di un sistema di particelle nel tuo gioco, o ogni volta che desideri ricreare il sistema di particelle (per esempio, appena prima di chiamare un Restart Game). Nota che se il non è stato contrassegnato come persistente allora non hai bisogno di chiamare questa azione, poiché il sistema di particelle (e qualsiasi emettitore associato ad esso) sarà automaticamente ripulito su room fine.

IMPORTANTE! Questa funzione distruggerà e rimuoverà anche gli emettitori eventualmente creati e associati al sistema che viene distrutto (vedi Create Particle Emitter per maggiori informazioni).

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
System Il valore ID unico per il sistema da distruggere

Esempio:

Il codice del blocco d'azione di cui sopra controllerà se il tasto "Spazio e se viene rilevato, rimuove il sistema di particelle e i diversi tipi di particelle definiti nelle variabili globali date e poi riavvia il gioco.
