Distruggere l'emettitore di particelle

Questa azione "distruggerà" l'emettitore di particelle dato, cioè: libererà la memoria utilizzata dall'emettitore di particelle. Tu fornisci l'ID del sistema di particelle a cui l'emettitore appartiene (come restituito dall'azione Create Particle System) e poi l'effettivo ID dell'emettitore (come restituito dall'azione Create Particle Emitter). Questa azione dovrebbe essere chiamata ogni volta che non hai più bisogno di un emettitore di particelle nel tuo gioco, o ogni volta che desideri ricreare l'emettitore di particelle (per esempio, appena prima di chiamare un Game Restart). Nota che se il sistema di particelle a cui l'emettitore è stato assegnato non è stato contrassegnato come persistente allora non è necessario chiamare questa azione, poiché l'emettitore sarà automaticamente ripulito su room alla fine insieme al sistema di particelle.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
System Il valore ID del sistema che contiene l'emettitore
Emitter Il valore ID dell'emettitore da distruggere

Esempio:

Il codice del blocco d'azione di cui sopra controllerà una variabile di istanza e se è superiore a 100 distruggerà l'emettitore assegnato all'istanza e poi passerà al prossimo room.
