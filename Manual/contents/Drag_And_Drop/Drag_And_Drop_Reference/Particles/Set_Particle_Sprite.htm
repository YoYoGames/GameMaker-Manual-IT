Imposta sprite di particelle

Questa azione può essere utilizzata per impostare il sprite che una particella può utilizzare durante il disegno. Fornisci il valore ID unico per il tipo da impostare sprite (come restituito quando hai creato il tipo con l'azione Create Particle Type) e la risorsa sprite da utilizzare. Se sprite ha delle sotto-immagini, allora la particella si animerà attraverso i fotogrammi alla velocità impostata nell'Editor di immagini per sprite. Nota anche che se imposti un valore di colore per la particella (usando l'azione Set Particle Colour ) allora questo colore sarà miscelato con sprite usato, proprio come l'impostazione della miscela di immagini su un'istanza miscelerà un colore con l'istanza sprite (usa il bianco per non miscelare).

NOTA: Se usate questa azione allora non dovreste usare l'azione Set Particle Shape perché potete avere solo un sprite o una forma, ma non entrambi.
