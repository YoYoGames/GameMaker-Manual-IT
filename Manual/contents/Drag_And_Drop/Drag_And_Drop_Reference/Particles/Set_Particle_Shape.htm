Imposta forma della particella

Questa azione può essere utilizzata per impostare la forma incorporata di sprite che una particella dovrebbe utilizzare quando viene disegnata. Fornisci il valore ID unico per il tipo di cui impostare la forma (come restituito quando hai creato il tipo con l'azione Create Particle Type) e la forma da utilizzare. Le forme delle particelle sono semplicemente sprites bianche che sono state create per GameMaker per essere utilizzate specificamente come particelle e consistono nelle seguenti forme:

NOTA: Se usate questa azione allora non dovreste usare l'azione Set Particle Sprite perché potete avere solo un sprite o una forma, ma non entrambi.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Type Il valore ID per il tipo di cui impostare la forma
Shape La forma da usare per il tipo di particella

Esempio:

Il codice del blocco d'azione di cui sopra creerà un nuovo tipo di particella e assegnerà il suo unico ID a una variabile globale. Poi procede a impostare tutte le proprietà del tipo di particella.

Indietro: Azioni delle particelle
Avanti: Imposta il colore delle particelle
