Sistema a particelle chiare

Con questa azione è possibile cancellare il sistema di particelle dato al suo stato predefinito, rimuovendo tutti gli emettitori (non c'è anche bisogno di chiamare l'azione Destroy Particle Emitter in quanto questo è curato automaticamente) e cancellare tutte le particelle dallo schermo. Tu fornisci il valore ID unico per il sistema da cancellare, come restituito quando hai creato il sistema con l'azione Create Particle System. Bisogna fare attenzione quando si usa questa azione poiché distrugge tutti gli emettitori associati al sistema di particelle - questo significa che qualsiasi istanza in room che usa un emettitore può dare un errore poiché quell'emettitore non esiste più, quindi assicurati che l'azione sia chiamata solo quando non stai per creare nuove particelle da quell'emettitore (per maggiori informazioni sugli emettitori di particelle vedi Create Particle Emitter).

NOTA: Questa azione cancella solo le particelle visibili in room, ma non cancella le proprietà delle particelle né le rimuove dalla memoria. Per questo dovresti usare l'azione Destroy Particle Type.
