Scoppio di particelle

Questa azione ti permette di far scoppiare le particelle in qualsiasi momento senza fare affidamento su un emettitore. Fornisci il valore ID unico del sistema in cui creare le particelle (come restituito quando hai creato il sistema con l'azione Create Particle System) e l'ID unico del tipo di particella da creare (come restituito quando hai creato il tipo di particella con l'azione Create Particle Type), così come la posizione x/y in cui crearle e il numero da creare. La posizione può essere una posizione assoluta all'interno di room, oppure puoi segnalarla come una posizione relativa all'istanza che sta chiamando l'azione.

Potete chiamare questa azione ogni fotogramma di gioco per replicare l'effetto di flusso di un emettitore, o potete chiamarla una volta in un evento (come l'evento Destroy) per far semplicemente scoppiare le particelle.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
System Il valore ID del sistema di particelle da utilizzare
Type Il valore ID del tipo di particella da far scoppiare
X La posizione x per far scoppiare le particelle a
Y Si posizionano per far scoppiare le particelle a
Count Il numero di particelle da far scoppiare

Esempio:

Il codice del blocco d'azione di cui sopra controllerà per vedere se l'istanza ha cambiato posizione da quando l'ultimo fotogramma di gioco è stato aggiornato e, in caso affermativo, farà scoppiare alcune particelle nella nuova posizione.
