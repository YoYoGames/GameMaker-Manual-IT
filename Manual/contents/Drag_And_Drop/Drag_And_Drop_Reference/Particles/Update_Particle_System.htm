Aggiornare il sistema di particelle

Questa azione forzerà il sistema di particelle dato ad aggiornare la posizione, il colore, l'alfa, ecc. Tu fornisci il valore ID unico per il sistema da aggiornare (come restituito quando hai creato il sistema con l'azione Create Particle System), e ogni volta che chiami l'azione il sistema dato aggiornerà un fotogramma di gioco, il che significa che puoi - per esempio - chiamare questa azione all'interno di un Repeat loop e far aggiornare il sistema di particelle più volte in un singolo fotogramma di gioco (utile per quando vuoi che un livello abbia un qualsiasi effetto che richiede che le particelle siano presenti all'inizio, come la neve o la pioggia). Puoi anche usare questa azione quando il sistema di particelle è stato messo in pausa usando il Pause Particle System per aggiornare manualmente le particelle nel sistema.
