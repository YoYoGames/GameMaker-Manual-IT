Sistema di particelle Pause

Questa azione mette in pausa il sistema di particelle dato. Tu fornisci il valore ID unico per il sistema da mettere in pausa, come restituito quando hai creato il sistema con l'azione Create Particle System. Un sistema di particelle in pausa continuerà a disegnare le particelle che sono state generate, ma esse non aggiorneranno più il loro colore, la loro posizione o l'alfa, né completeranno la loro durata e scompariranno fino a quando non riprenderai il sistema. Puoi scegliere di mettere in pausa o riprendere il sistema usando l'opzione "Stato".

Nota che questo non mette in pausa gli emettitori che stanno trasmettendo particelle, né fermerà un emettitore che fa scoppiare particelle - gli emettitori continueranno a creare particelle quando chiamati, ma quando il sistema è in pausa le particelle non faranno nulla e saranno disegnate esattamente dove sono state create fino a quando non resetterai lo stato di pausa o chiamerai Update Particle System.
