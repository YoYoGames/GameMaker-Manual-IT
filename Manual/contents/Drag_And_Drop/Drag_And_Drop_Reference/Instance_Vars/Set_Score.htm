Impostare il punteggio

Con questa azione potete creare o impostare una variabile di istanza per il punteggio. Si fornisce il valore a cui impostare la variabile punteggio o da aggiungere alla variabile punteggio (se si spunta la casella di controllo relativa, allora il valore sarà aggiunto al valore corrente "score"). corrente). Si noti che, a differenza delle normali variabili di istanza, il nome della variabile "score" è memorizzato internamente, quindi per recuperare il valore del punteggio è necessario utilizzare l'azione Get Score.
