Set Vite

Con questa azione puoi creare o impostare una variabile di istanza per lives. Tu fornisci il valore a cui impostare la variabile lives o da aggiungere alla variabile lives (se spunti la casella di controllo relativa allora il valore sarà aggiunto all'attuale valore "lives"). corrente). Nota che a differenza delle normali variabili di istanza il nome della variabile "lives" è memorizzato internamente, quindi per recuperare il valore del punteggio devi usare l'azione Get Lives.
