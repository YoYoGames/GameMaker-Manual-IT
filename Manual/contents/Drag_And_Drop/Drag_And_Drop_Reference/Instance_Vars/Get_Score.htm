Ottieni un punteggio

Potete usare questa azione per recuperare il valore della variabile di istanza creata per il "punteggio". Si fornisce la variabile di destinazione per memorizzare il valore "score" restituito (che può essere contrassegnato come una variabile locale temporanea), ma si noti che se la variabile non è stata impostata in precedenza (usando Set Score) allora la variabile sarà creata per voi e l'azione restituirà 0 (a differenza delle normali variabili di istanza che daranno un errore se si cerca di accedervi senza averle prima impostate).

Questa variabile viene poi convertita in un string e memorizzata in un'altra variabile locale temporanea prima di essere disegnata sullo schermo.
