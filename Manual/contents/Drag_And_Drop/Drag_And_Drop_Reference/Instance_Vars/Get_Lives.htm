Ottenere vite

Potete usare questa azione per recuperare il valore della variabile di istanza creata per le "vite". Si fornisce la variabile di destinazione per memorizzare il valore "lives" restituito (che può essere contrassegnato come una variabile locale temporanea), ma si noti che se la variabile non è stata impostata in precedenza (usando Set Lives) allora la variabile sarà creata per voi e l'azione restituirà 0 (a differenza delle normali variabili di istanza che daranno un errore se si cerca di accedervi senza averle prima impostate).

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Target La variabile di destinazione per memorizzare il valore "vite" restituito (può essere contrassegnata come variabile locale temporanea)

Esempio:

Il codice del blocco d'azione di cui sopra recupererà il valore di "lives" e lo memorizzerà in una variabile locale temporanea variabile locale temporanea. Questa variabile viene poi utilizzata per decidere quanti sprites impilati devono essere disegnati.

Indietro: Azioni delle variabili di istanza
Prossimo: Imposta la salute
