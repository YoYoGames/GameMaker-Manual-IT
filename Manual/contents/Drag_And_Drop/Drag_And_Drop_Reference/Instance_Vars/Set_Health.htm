Imposta la salute

Con questa azione potete creare o impostare una variabile di istanza per la salute. Si fornisce il valore a cui impostare la variabile salute o da aggiungere alla variabile salute (se si spunta la casella di controllo relativa, allora il valore sarà aggiunto al valore corrente "salute" corrente). Si noti che se si desidera utilizzare l'azione Draw Instance Healthbar allora questo valore dovrebbe essere compreso tra 0 e 100, altrimenti può essere qualsiasi importo richiesto. Notate anche che a differenza delle le normali variabili di istanza il nome della variabile "salute" è memorizzato internamente, quindi per recuperare il valore del punteggio è necessario utilizzare l'azione Get Health.
