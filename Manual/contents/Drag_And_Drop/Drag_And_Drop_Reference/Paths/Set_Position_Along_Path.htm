Imposta la posizione lungo il percorso

Questa azione imposterà la posizione di un'istanza lungo un path. Il valore è normalizzato da 0 a 1, quindi se lo impostate, per esempio, a 0,5, l'istanza sarà spostata esattamente a metà del path. Notate che dovete aver chiamato l'azione Start Following Path per assegnare un path all'istanza perché questo abbia qualche effetto.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Position La posizione (da 0 a 1) lungo path per posizionare l'istanza

Esempio:

Il codice del blocco d'azione di cui sopra avvia l'istanza lungo un path e poi genera un valore decimale casuale tra 0 e 1. Questo valore viene memorizzato in una variabile locale temporanea e utilizzato per impostare la posizione dell'istanza lungo il path.
