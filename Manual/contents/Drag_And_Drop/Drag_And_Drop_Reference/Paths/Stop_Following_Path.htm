Smettere di seguire il percorso

Questa azione terminerà il movimento lungo l'attuale path che l'istanza sta seguendo.

Esempio:

Il codice del blocco d'azione di cui sopra controlla il valore di un allarme e se è inferiore a 0 viene controllato un Switch controllato. A seconda del string memorizzato nella variabile di istanza che viene passata al switch, l'istanza fermerà l'attuale path su cui si trova, o inizierà un nuovo path prima di impostare nuovamente l'allarme.
