Ottenere la posizione lungo il percorso

Questa azione otterrà la posizione di un'istanza lungo un path. Il valore restituito è normalizzato da 0 - 1, quindi se si ottiene un risultato di, per esempio, 0.5, l'istanza si trova esattamente a metà del path. Si noti che è necessario aver chiamato l'azione Start Following Path per assegnare un path all'istanza perché questa azione restituisca un valore diverso da 0. Il valore restituito sarà memorizzato nella variabile di destinazione fornita, oppure è possibile contrassegnare la variabile come una variabile locale temporanea per crearla e assegnarle il valore.

Il codice del blocco d'azione di cui sopra ottiene la posizione corrente sul percorso assegnato per l'istanza e se è inferiore a 0,5 viene impostata a 0,5.
