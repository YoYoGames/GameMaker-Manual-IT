Impostare Path Seguire la velocità

Questa azione imposterà la velocità, in pixel per fotogramma di gioco, che un'istanza dovrebbe seguire un path (si noti che è necessario aver chiamato l'azione Start Following Path per assegnare un path all'istanza perché questo abbia alcun effetto). Potete usare valori negativi per indicare che l'istanza dovrebbe seguire il path al contrario e vale la pena notare che la velocità che impostate qui può essere modificata dal modificatore di velocità impostato nell'editor path per ogni punto path, quindi se impostate una velocità path a 2 e in path uno dei punti ha un modificatore di velocità del 200% la velocità effettiva di pixel per frame di gioco in quel punto sarà 4.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Speed La velocità (in pixel per fotogramma di gioco) che l'istanza dovrebbe seguire il percorso

Esempio:

Il codice del blocco d'azione di cui sopra avvia un'istanza seguendo un path, poi genera un valore casuale tra 2 e 4 prima di usare questo valore per impostare la velocità con cui l'istanza si muove lungo path.
