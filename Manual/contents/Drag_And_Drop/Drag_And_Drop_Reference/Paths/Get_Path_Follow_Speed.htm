Ottieni Path Segui la velocità

Questa azione otterrà la velocità, in pixel per fotogramma di gioco, che un'istanza è impostata per seguire un path (si noti che è necessario aver chiamato l'azione Start Following Path per assegnare un path all'istanza perché questo restituire un valore diverso da 0). Il valore restituito sarà memorizzato nella variabile di destinazione che date, o potete contrassegnare la variabile come una variabile locale temporanea per crearla e assegnare il valore.

Argomenti:

Argomento Descrizione
Target La variabile di destinazione per tenere la velocità di inseguimento restituita path

Esempio:

Il codice del blocco d'azione di cui sopra ottiene la velocità attuale dell'istanza lungo il percorso assegnato path e se è inferiore a 4, incrementa la velocità path di 0,1.
