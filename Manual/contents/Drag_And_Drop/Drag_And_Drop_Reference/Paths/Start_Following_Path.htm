Iniziare a seguire il percorso

Questa azione avvia un'istanza che segue un path che è stato creato nel Browser di Asset. Dovreste quindi usare questa azione per dire alla vostra istanza quale path seguire, quale velocità seguire il path (misurata in pixel per fotogramma di gioco), come comportarsi quando raggiunge la fine di path, e se seguire la posizione assoluta o relativa di path. Quando si imposta la velocità si possono usare valori negativi per indicare che l'istanza dovrebbe seguire il path al contrario e vale la pena notare che la velocità che imposti qui può essere modificata dal modificatore di velocità impostato nell'editor di path per ogni punto di path, quindi se imposti la velocità di path a 2 e in path uno dei punti ha un modificatore di velocità del 200%, allora la velocità effettiva in quel punto sarà 4 pixel per fotogramma di gioco.

Una volta che hai impostato la velocità iniziale path puoi impostare come l'istanza dovrebbe comportarsi quando raggiunge la fine del path dalle seguenti opzioni (nota che quando un'istanza raggiunge la fine di un path questo innescherà anche un Other - Path Ended Event):

  • Stop: Termina il path, fermando l'istanza sull'ultimo punto path.
  • Riavviare: Continua il path dall'inizio, saltando di nuovo alla posizione iniziale se il path non è segnato come chiuso nell'Editor Path.
  • Loop: Continua dalla posizione attuale per eseguire di nuovo il path.
  • Invertire: Andare di nuovo all'indietro lungo il path (ottenuto invertendo la velocità di path follow), e quando raggiunge l'inizio del path, invertirà di nuovo, ecc...

Infine, è possibile contrassegnare path come Assoluto o no. Per impostazione predefinita l'istanza inizierà a seguire il path dalla sua posizione attuale (la posizione relativa), ma se si contrassegna la casella Absolute allora inizierà a seguire il path esattamente come l'hai disegnata e posizionata nell'editor di Path (la posizione assoluta). Argomenti:

Argomento Descrizione
Path Il path asset per iniziare a seguire
Speed La velocità (in pixel per fotogramma di gioco) per seguire il percorso
On End La risposta al raggiungimento della fine del sentiero
Absolute Se iniziare in posizione assoluta o relativa

Il blocco d'azione di cui sopra farà seguire all'istanza chiamante un path (partendo dalla sua posizione assoluta e continuando alla fine) quando viene premuto un tasto.
