Mostra il messaggio di debug

Con questa azione puoi inviare un messaggio di debug all'output del compilatore. Questo messaggio può essere sia un string che un numero reale, ma se vuoi usare entrambi allora il numero deve essere convertito in una stringa (vedi l'azione Number To String). Nota che puoi inviare più messaggi utilizzando questa azione cliccando sull'icona per espandere l'azione con ulteriori campi di output.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Message Il messaggio da mostrare nella finestra Output

Esempio:

Il codice del blocco d'azione di cui sopra controllerà una variabile e se restituisce true allora un paio di messaggi di debug saranno mostrati nella finestra Output.

Indietro: Azioni varie
Avanti: Impostare lo stato della finestra
