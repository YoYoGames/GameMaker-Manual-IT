Se viene visualizzata la tastiera virtuale

Questa azione può essere usata per controllare se la tastiera virtuale del sistema operativo viene mostrata o meno. L'azione valuterà true quando la tastiera è aperta o in apertura, o false se la tastiera virtuale è chiusa o in chiusura. Se si abilita il modificatore not, questa azione sarà invertita e si può memorizzare il valore di ritorno true / false in una variabile di destinazione, che può essere contrassegnata come temp (locale).

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite se il blocco "if" valuta true, mentre qualsiasi azione abbandonata altrove sarà eseguita dopo il blocco "if".

Esempio:

Il codice del blocco d'azione di cui sopra verifica la presenza di un'azione del mouse verso il basso, e se ne viene rilevata una, allora viene eseguito un controllo per vedere se la tastiera virtuale del sistema operativo è visibile. Se non lo è, allora viene impostata per essere mostrata - usando un tipo di tastierino numerico - e se è già mostrata, allora viene nascosta.
