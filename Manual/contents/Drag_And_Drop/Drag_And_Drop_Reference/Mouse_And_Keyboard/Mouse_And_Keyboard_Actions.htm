Libreria di azioni per mouse e tastiera

Il Mouse e Tastiera è dove puoi trovare le azioni richieste per eseguire il polling e processare l'input dalla tastiera o dai dispositivi del mouse. Le azioni disponibili sono tutte elencate qui sotto:

Se il mouse viene premuto
Se il mouse giù
Se il topo viene rilasciato
Se il tasto è premuto
Se tasto giù
Se la chiave viene rilasciata

Il resto delle azioni qui si riferisce alla tastiera virtuale. Se avete bisogno di testo o di input numerico nel vostro gioco su un dispositivo touchscreen, allora potete usare le azioni di tastiera virtuale fornite per usare la tastiera OS del dispositivo. Queste azioni richiameranno o nasconderanno la tastiera OS e accetteranno input da essa proprio come una tastiera fisica, quindi le normali azioni della tastiera di GameMaker dovrebbero funzionare tutte con le tastiere virtuali (per esempio, l'azione "If Key Released"), e qualsiasi input dalla tastiera virtuale sarà memorizzato nelle diverse variabili di tastiera, come keyboard_string o keyboard_last_char, ecc...).

L'apertura o la chiusura della tastiera virtuale genererà eventi asincroni di sistema, in cui la mappa async_load  DS sarà popolata con le seguenti coppie chiave/valore:

  • "event_type" - il tipo di evento di sistema ricevuto, che sarà "virtual keyboard status" per le tastiere virtuali.
  • "screen_height" - l'altezza della tastiera virtuale (in pixel). Questo sarà 0 se la tastiera è invisibile.
  • "keyboard_status" - lo stato attuale della tastiera, restituito come una delle seguenti stringhe:
    • "nascondere"
    • "nascosto"
    • "mostrando"
    • "visibile

Le azioni della tastiera virtuale disponibili sono:

Mostra la tastiera virtuale
Nascondere la tastiera virtuale
Ottieni l'altezza della tastiera virtuale
Se viene visualizzata la tastiera virtuale

Indietro: Indice
Prossimo: Azioni del gamepad
