Imposta alfa dell'istanza

Questo blocco d'azione imposta il valore image_alpha per l'istanza. L'alfa dell'immagine è il valore di "trasparenza" per disegnare qualsiasi sprite assegnato all'istanza e ha un valore tra 0 e 1, dove 0 sarebbe completamente trasparente e 1 sarebbe sarebbe completamente opaco. Notate che se sprite ha pixel trasparenti o semi-trasparenti, allora la loro trasparenza sarà scalata dalla quantità, quindi se avete alcuni pixel trasparenti al 50% in sprite e impostate l'alfa dell'istanza a 0.5, l'alfa finale per i pixel trasparenti sarebbe del 25%. Potete impostare questo valore per essere una quantità relativa

Si prega di notare che per i cambiamenti in questa variabile per essere visibili, l'istanza dovrebbe avere o nessun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata usando l'azione Draw Self .

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
alpha Il valore alfa da impostare (da 0 a 1, il default è 1).

Esempio:

Il codice del blocco d'azione di cui sopra imposta un nuovo sprite così come una serie di altre per come questo sprite deve essere visualizzato, compresa l'impostazione dell'alfa dell'immagine a 0,5.
