Imposta sprite

Questo blocco d'azione imposta il sprite_index dell'istanza su un altro sprite dal Browser Asset. Si seleziona il sprite a cui cambiare, e poi si dà l'immagine (frame di animazione) da mostrare. Notate che se non volete che sprite si animi dopo impostare il frame, dovrai usare l'azione Set Animation Speed e impostarla a 0, o se vuoi che sprite continui ad animarsi allora imposta il valore per usare la variabile incorporata image_index invece.

La modifica di sprite cambierà anche la maschera di collisione dell'istanza, a meno che tu non abbia anche specificato un mask_index separato nell'Editor degli oggetti. Si prega di notare che per le modifiche di questa azione siano visibili, l'istanza dovrebbe non avere alcun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata usando l'azione Draw Self.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Sprite Il nuovo sprite da usare (-1 se vuoi rimuovere il sprite).
Frame Il fotogramma dell'animazione da mostrare inizialmente (se non vengono usati fotogrammi extra, allora usa 0).
