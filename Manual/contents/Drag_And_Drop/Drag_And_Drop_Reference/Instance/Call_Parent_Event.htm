Chiama l'evento dei genitori

Questo blocco d'azione eseguirà l'evento del genitore object di cui l'istanza è figlia prima di continuare ad eseguire le azioni nell'istanza attuale. Quando si assegna un genitore object ad un'istanza, tutti gli eventi di quel object sono "ereditati" dal figlio. Tuttavia, se si aggiunge un evento al figlio object, questo sovrascrive l'evento del genitore. Usando questo blocco d'azione, si può forzare un figlio object ad eseguire sia l'evento genitore che l'evento nell'istanza stessa. Vedere la sezione Editor di oggetti per maggiori informazioni sui genitori.
