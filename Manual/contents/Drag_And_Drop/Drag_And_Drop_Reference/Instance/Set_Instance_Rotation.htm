Impostare la rotazione dell'istanza

Questo blocco d'azione imposta il image_angle (rotazione) dell'istanza. L'angolo è misurato in gradi, con la destra che è 0°, su che è 90°, a sinistra che è 180° e giù che è 270°. Imposta questa variabile a 0 per resettare l'istanza, il che significa che essa disegnare qualsiasi sprite assegnato come è stato definito nell'editor sprite. Si prega di notare che per i cambiamenti in questa azione per essere visibili, l'istanza dovrebbe avere o nessun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata usando l'azione Draw Self.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Angle Il nuovo angolo da utilizzare (da 0 a 360).

Esempio:

Il codice del blocco d'azione di cui sopra imposta un nuovo sprite così come una serie di altre per come questo sprite deve essere visualizzato, inclusa la rotazione di 180°.
