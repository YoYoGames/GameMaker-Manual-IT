Creare Object Istanza

Con questa azione si può creare un'istanza di qualsiasi object dal Browser di Asset. Si dà il nome del object e una posizione x/y all'interno del room dove deve essere creato, così come il nome del livello su cui creare l'istanza (deve essere un Instance Layer). Puoi spuntare le bandiere "relative" per posizionare l'istanza rispetto all'istanza che esegue i blocchi di codice dell'azione. Ogni volta che crei un'istanza nel gioco, le viene assegnato un valore ID unico con il quale puoi poi identificare l'istanza in ulteriori azioni, quindi questa azione vi permette di definire una variabile "target" per contenere questo valore ID (come mostrato nell'esempio qui sotto). Si noti che il target può essere contrassegnato come una variabile temporanea (locale), che sarà creata per l'azione e può essere usata in tutte le azioni successive.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Object Il nome del object di cui creare un'istanza.
x La posizione x in room per creare l'istanza (può essere relativa all'istanza chiamante).
y La posizione y in room per creare l'istanza (può essere relativa all'istanza chiamante).
Layer Il nome (un string) del livello di istanza su cui creare l'istanza.
Target La variabile a cui puntare per il valore di ritorno.

Esempio:

Il codice del blocco d'azione di cui sopra crea una nuova istanza e memorizza il suo valore ID unico in una variabile temporanea. Questa variabile viene poi utilizzata nell'azione Applies To... per impostare alcune proprietà dell'istanza creata.
