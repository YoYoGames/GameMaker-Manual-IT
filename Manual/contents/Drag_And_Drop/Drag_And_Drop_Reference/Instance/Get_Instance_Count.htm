Ottenere il conteggio delle istanze

Con questa azione potete trovare il numero di istanze di qualsiasi dato object attualmente attive all'interno di room. Potete selezionare qualsiasi object dalla lista delle risorse o potete usare la parola chiave speciale all per ottenere il numero di tutte le istanze attive. Dovete anche fornire una variabile di destinazione a cui restituire il valore, in modo da poterlo poi utilizzare in ulteriori azioni. Se contrassegnate la variabile di destinazione come "Temp", allora questo creerà una nuova variabile temporanea (locale) temporanea per contenere il valore di ritorno, come mostrato nell'esempio qui sotto.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Object Il nome del sito object da controllare.
Target La variabile a cui puntare per il valore di ritorno.

Esempio:

Il codice del blocco d'azione di cui sopra crea una variabile locale (temporanea) e poi assegna il numero di istanze di object "obj_Squirrel". Questo valore viene poi controllato e l'istanza viene distrutta se il numero di istanze è maggiore di 100.
