Imposta il colore dell'istanza

Questo blocco d'azione imposta il colore di image_blend per il "blending" con l'istanza sprite. Il valore predefinito è il codice esadecimale $FFFFFFFFFF (che rappresenta il colore bianco usando il formato ARGB ) e questo disegnerà il sprite usando nessuna miscelazione e piena alfa. Qualsiasi altro valore (incluse le costanti di colore interne come c_red, o c_aqua) fonderà il colore specificato con l'originale sprite così come imposterà il image_alpha per l'istanza, ma solo quando la casella Use Alpha è spuntata. Questo sovrascriverà qualsiasi valore alfa impostato in precedenza usando l'azione Set Instance Alpha.

Si prega di notare che per i cambiamenti in questa azione per essere visibili, l'istanza dovrebbe avere o nessun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata usando l'azione Draw Self. È importante notare anche che si dovrebbe cercare di limitare il blending sulla piattaforma HTML5 (a meno che non si usi WebGL), poiché ogni sprite mescolato deve essere memorizzato nella cache separatamente e quindi avere molti sprites mescolati può influire negativamente sulle prestazioni.
