Chiama l'evento utente

Con questa azione potete chiamare uno qualsiasi dei 12 Other - User Events disponibili. Un evento utente è un evento a cui si può aggiungere scripts o codice d'azione e poi chiamarlo in qualsiasi momento usando questa azione, rendendoli essenzialmente eventi object personalizzati.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Event Il numero dell'evento utente da chiamare (0 - 11).

Esempio:

Il codice del blocco d'azione di cui sopra controlla una variabile e poi chiama l'evento utente [0] o l'evento utente [1] a seconda che il controllo valuti true o false.

Indietro: Azioni dell'istanza
Avanti: Impostare la variabile di istanza
