Cercare il buffer

Questa azione può essere usata per spostarsi (cercare) in una posizione all'interno di un buffer. Si fornisce il valore ID unico buffer come restituito dall'azione Create Buffer e poi il valore di offset in byte verso cui spostarsi. Il valore di offset sarà sarà applicato dall'inizio del buffer, quindi chiamando questa funzione si sposterà la posizione di lettura/scrittura all'inizio del buffer poi si sposterà il numero di byte dato lungo e posizionerà la nuova posizione di lettura/scrittura lì. Tuttavia, se si contrassegna l'offset come relativo, sposterà la posizione di lettura/scrittura in avanti (o indietro se l'offset è un valore negativo) dalla posizione corrente.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Buffer L'indice buffer (memorizzato in una variabile)
offset L'offset (in byte) relativo alla posizione di ricerca corrente

Esempio:

Il codice del blocco d'azione di cui sopra ottiene la dimensione di buffer e poi usa quel valore per impostare la posizione di lettura/scrittura alla fine dei dati attualmente memorizzati.
