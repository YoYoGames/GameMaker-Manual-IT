Cancellare il buffer

Questa azione può essere usata per cancellare un buffer dalla memoria, rilasciando le risorse usate per crearlo e rimuovendo qualsiasi dato che può contenere attualmente. Si fornisce il valore ID unico di buffer come restituito dall'azione Create Buffer. Si noti che è possibile selezionare più buffers per la rimozione facendo clic sull'icona più accanto all'azione e aggiungendo un altro ID buffer.

NOTA: è importante rimuovere sempre qualsiasi risorsa creata dinamicamente dalla memoria quando non ne avete più bisogno per evitare perdite di memoria.

Argomenti:
