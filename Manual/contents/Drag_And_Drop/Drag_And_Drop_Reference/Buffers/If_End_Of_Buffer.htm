Se fine del buffer

Questa azione può essere usata per controllare se la posizione di lettura/scrittura ha raggiunto la fine di buffer. Si fornisce il valore ID unico di buffer come restituito dall'azione Create Buffer e poi l'azione restituirà true se la posizione di lettura/scrittura ha raggiunto la fine di tutti i dati disponibili in buffer, o false se non lo ha fatto. Si può flaggare "not" per far sì che l'azione controlli se la posizione di lettura/scrittura non è alla fine di buffer.
