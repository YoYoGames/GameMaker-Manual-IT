Ottenere Buffer Posizione

Questa azione può essere usata per recuperare la posizione corrente per leggere/scrivere su buffer. Quando si crea un buffer, la posizione di lettura/scrittura è impostata a 0 byte, e man mano che si aggiungono dati a buffer questa posizione si sposta in modo sequenziale, il che significa che potete usare questa azione per ottenere il numero di byte in buffer su cui è impostata in qualsiasi momento. Voi fornite il valore ID unico di buffer come restituito dall'azione Create Buffer e poi date una variabile di destinazione a cui restituire il valore di posizione. La variabile di destinazione può essere contrassegnata come una variabile locale temporanea, nel qual caso sarà creata per voi e rimossa dalla memoria alla fine dell'azione corrente script o dell'evento.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Buffer L'indice buffer (memorizzato in una variabile)
Position La posizione di lettura/scrittura (in byte) all'interno del buffer

Esempio:

Il codice del blocco d'azione di cui sopra ottiene la posizione di lettura/scrittura per il buffer dato e se non è 0 (cioè: all'inizio di buffer) viene impostata all'inizio di buffer.

Indietro: Azioni del buffer
Prossimo: Se Buffer esiste
