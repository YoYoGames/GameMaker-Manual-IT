Ottieni Buffer Dimensione

Questa azione può essere usata per recuperare la dimensione di un buffer, in byte. Si fornisce il valore ID unico di buffer come restituito dall'azione Create Buffer e poi si dà una variabile di destinazione a cui restituire il valore della dimensione. La variabile di destinazione può essere contrassegnata come una variabile locale temporanea, nel qual caso sarà creata per voi e rimossa dalla memoria alla fine dell'azione corrente script o dell'evento.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Buffer L'indice buffer (memorizzato in una variabile)
Target La variabile di destinazione che contiene il valore restituito

Esempio:

Il codice del blocco d'azione di cui sopra ottiene la dimensione di buffer e poi usa quel valore per impostare la posizione di lettura/scrittura alla fine dei dati attualmente memorizzati.
