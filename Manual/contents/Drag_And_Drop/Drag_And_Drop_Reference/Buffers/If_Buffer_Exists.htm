Se Buffer esiste

Questa azione può essere usata per controllare se un dato ID buffer è valido e se buffer esiste effettivamente in memoria. Se l'ID buffer è valido ed esiste, l'azione restituirà true altrimenti restituirà false. Si può mettere il flag "not" per far sì che l'azione controlli se un buffer non esiste.

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:
