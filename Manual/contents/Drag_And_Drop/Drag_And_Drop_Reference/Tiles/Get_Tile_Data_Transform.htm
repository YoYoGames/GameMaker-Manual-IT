Ottenere Tile Trasformazione dei dati

Con questa azione potete scoprire se i dati di tile per un tile sono stati trasformati in uno dei tre modi: capovolto, specchiato o ruotato. I dati di tile sono semplicemente un valore che riflette l'indice di tile insieme alle trasformazioni applicate, e si normalmente volete ottenere i dati di tile usando Get Tile Data In Cell o Get Tile Data At Pixel prima di usare questa azione. L'azione restituisce true se la trasformazione scelta è stata applicata al tile-data, o false altrimenti, e il valore restituito sarà quindi memorizzato nella variabile di destinazione che può essere stata creata in precedenza o può essere una nuova variabile temporanea (se si seleziona la casella "Temp ). Per maggiori informazioni sui dati tile, consultate la sezione GML sulle funzioni Tile Map.
