Set di piastrelle

Con questa azione si può impostare un livello per usare set di piastrelle diversi da quelli creati nel Browser di Asset. Si dà il nome del livello (un string, come definito nell'Editor Room ), e poi la risorsa tile set da usare, e tutti i tiles nel room su quel livello saranno disegnati con il nuovo tile set.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Layer Il livello su cui impostare il nuovo tile set.
Tileset Il tile set da usare per il livello.

Esempio:

Il codice del blocco d'azione di cui sopra controlla se tiles sul livello "Floor_Tiles" sta usando il tile set "tl_PalaceRuins", e se non lo fanno, allora sono impostati per usarlo.
