Tipi di dati Biblioteca d'azione

GameMaker ha una serie di diversi data types, come interi, strings, puntatori, ecc. e alcuni di questi possono essere convertiti in altri. Questa sezione del manuale mostra le diverse azioni disponibili per trattare e convertire diversi data types.

NOTA: potete scoprire di più sui diversi data types qui.

La libreria Types ha le seguenti azioni:

Da decimale a intero
Da stringa a numero
Da numero a stringa

Indietro: Indice
Prossimo: Azioni casuali
