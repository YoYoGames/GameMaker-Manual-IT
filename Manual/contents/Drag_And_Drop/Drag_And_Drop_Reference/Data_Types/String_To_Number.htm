Da stringa a numero

Questa azione può convertire un string in un numero. Tu fornisci il string da cambiare e poi dai una variabile di destinazione a cui restituire il numero reale (questa variabile può essere contrassegnata come una variabile locale temporanea). Notate che se string ha dei caratteri che non siano numeri e un punto decimale, non sarà convertito, anche se se string inizia con un numero allora questi saranno convertiti e il resto di string escluso. È importante notare che quando si passano i decimali in questa azione azione, il string restituito sarà arrotondato solo a due cifre decimali. Di seguito è riportata una tabella dei valori di input e di ciò che ci si aspetta che venga restituito

Stringa Valore restituito
"123" 123
"123 Hello World" 123
"Hello World" 0 (per impostazione predefinita, se string non può essere convertito, verrà restituito 0)
"Hello World 123" 0
"123.45678" 123.46

Argomenti:

Argomento Descrizione
Value Il string per convertire
Target La variabile di destinazione che contiene il valore restituito

Esempio:

Il codice del blocco d'azione qui sopra inizializza una variabile globale e poi carica un file con un buffer salvato in esso. Due strings vengono letti da buffer e memorizzati in una variabile locale temporanea prima che uno di essi venga convertito in un numero reale e memorizzato in un'altra variabile locale temporanea. Questi vengono poi assegnati a variabili globali, e il sito buffer viene cancellato.
