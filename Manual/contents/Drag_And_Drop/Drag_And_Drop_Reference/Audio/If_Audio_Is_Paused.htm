Se l'audio è in pausa

Questa azione può essere usata per controllare se un'istanza di qualsiasi risorsa sonora è attualmente in pausa nel vostro gioco. Date la risorsa sonora da controllare dall'esploratore asset e la funzione restituirà true if it is paused and false se non lo è, anche se se si controlla il flag "not", questo è invertito e la funzione restituirà true se il suono non è in pausa e false altrimenti.

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite se il "if" valuta true, mentre qualsiasi azione abbandonata altrove sarà eseguita dopo il blocco "if".
