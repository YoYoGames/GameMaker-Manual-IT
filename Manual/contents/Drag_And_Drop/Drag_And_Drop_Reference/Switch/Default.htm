Default

Questa azione creerà un nuovo caso Default all'interno di uno Switch. Il caso Default è quello che verrà eseguito se il valore di input non corrisponde a nessun'altra costante di caso. Le azioni che vengono aggiunte ad un caso Default dovrebbero essere lasciate a lato dell'azione, proprio come si aggiungerebbe un'azione ad un "se":

Esempio:

Il codice del blocco d'azione di cui sopra ottiene il valore memorizzato in una variabile globale e poi lo confronta con i diversi casi possibili. Se uno qualsiasi dei casi è uguale al valore, allora verrà eseguita l'azione sotto quel caso, altrimenti verrà valutato il caso successivo. Se nessun caso corrisponde al valore inserito, verrà invece eseguito il caso predefinito.
