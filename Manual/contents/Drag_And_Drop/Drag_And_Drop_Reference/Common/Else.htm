Altrimenti

L'azione "else" è usata solo in combinazione con If Variable (o qualsiasi altra azione "if"), ed è un modo di definire un insieme di azioni che dovrebbero essere eseguite se la valutazione "if" restituisce false. Per esempio, potreste avere un'azione "if" che controlla se la variabile salute dei giocatori è minore o uguale a 0, e se questa valutazione è vera distruggete l'istanza. Tuttavia, se si vuole eseguire un diverso Tuttavia, se si vuole eseguire un diverso insieme di azioni di codice se la salute del giocatore non è inferiore a 0, allora si può aggiungere un "else" e poi specificare le azioni da eseguire mentre l'if valuta a false.

Qualsiasi azione abbandonata altrove sarà eseguita dopo che il blocco "else" e il suo contenuto sono stati eseguiti.
