Se la variabile

Questa azione è usata per controllare lo stato di una data variabile contro un dato valore usando una specifica espressione. Un'istruzione "if" restituirà sempre o true o false e le espressioni di controllo possono essere una qualsiasi delle seguenti:

  • Equals to - La variabile e il valore sono entrambi esattamente uguali
  • Meno di - La variabile è inferiore al valore
  • Maggiore di - La variabile è maggiore del valore
  • Less than or Equal to - La variabile è inferiore o uguale al valore
  • Maggiore o uguale a - La variabile è maggiore o uguale al valore

Se contrassegnate l'argomento "Not", allora le precedenti saranno espressioni negate, per esempio "è uguale a" diventa "non è uguale a". Notate anche che né la variabile né gli argomenti di valore sono limitati a valori numerici e possono essere un string, un puntatore, un ID di risorsa o qualsiasi altra cosa che una funzione può restituire o utilizzare (vedere la sezione sui tipi di dati per maggiori informazioni), ma non è possibile confrontare due data types diversi senza ottenere un errore - per esempio, se la variabile contiene un string e si cerca di controllare se è uguale a un numero, si otterrà un errore di sintassi.

NOTA IMPORTANTE: A causa di problemi di precisione in virgola mobile, il controllo per vedere se due valori sono esattamente uguali può restituire false, poiché uno può essere esattamente 1, mentre l'altro potrebbe essere 1.00000000000001. Questo può essere evitato usando l'azione Decimal to Integer prima di controllare o usando le espressioni "maggiore di" o "minore di".

Argomento Descrizione

Expression Il tipo di espressione da usare per il controllo.
Value Il valore con cui controllare la variabile.
