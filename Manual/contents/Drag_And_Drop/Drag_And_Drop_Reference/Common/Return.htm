Ritorna

Questa azione è da usare quando si creano funzioni di blocco d'azione ed è usata per restituire un valore all'istanza che chiama script. Il valore di ritorno può essere qualsiasi cosa che una variabile può contenere - come un string, un valore numerico, un puntatore o un ID di risorsa - o può essere una variabile stessa. Notate che quando chiamiamo l'azione di ritorno in una funzione, la funzione uscirà - restituendo il valore dato - e non verranno chiamate altre azioni dopo il ritorno, anche se se avete azioni nell'evento dopo aver chiamato la funzione, queste saranno ancora eseguite.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Return Il valore di ritorno.

Esempio:

La funzione del blocco d'azione di cui sopra imposta una variabile temporanea alla parola chiave noone, e poi controlla una variabile di input per vedere quale valore ha attualmente. Se la variabile è maggiore di 100, allora viene creata un'istanza e il suo unico ID di istanza viene memorizzato nella variabile temporanea, che viene poi restituita dalla funzione.
