Applicare a...

Questa azione può essere usata per cambiare l'Action Scope per tutte le altre azioni concatenate a questa. Come con un'azione if azione, se si rilasciano ulteriori azioni azioni a destra dell'azione Apply to... saranno eseguite dall'ambito object o dall'istanza. L'ambito è scelto dal menu pop-up mostrato quando si clicca la piccola freccia in alto a destra dell'azione, e in questo modo si può dire un diverso insieme di objects - o anche una singola istanza se hai l'ID dell'istanza - di eseguire più azioni prima di continuare con il codice dell'attuale object.

Si noti che per aggiungere azioni nel blocco "Applies To", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite dal sito object o dall'istanza che è stata presa di mira dal blocco "Applies To", mentre qualsiasi azione lasciata altrove sarà eseguita dopo il blocco "Applies To".
