Ottieni variabile di visualizzazione

Questa azione permette di ottenere il valore di alcune variabili incorporate relative a una data porta di vista. Si seleziona la variabile della porta della vista da recuperare, poi si dà la porta della vista a cui puntare (da 0 a 7) e infine la variabile di destinazione per contenere il valore restituito (questo può essere contrassegnato come una variabile locale temporanea). La porta della vista è l'area dello schermo che verrà utilizzata per disegnare una data vista della telecamera (come impostata nell'editor room ), e le diverse variabili che si possono ottenere in questo modo sono delineate nella tabella sottostante:

Variabile Descrizione
Camera Il valore restituito sarà il valore ID unico della telecamera da assegnare alla porta (vedere qui per maggiori informazioni).
Visibility Questo restituirà un valore di true o false dove true è che la porta della vista è visibile (cioè: viene disegnata sullo schermo), e false significa che non viene disegnato nulla.
View Port X Coordinate Questa è la posizione X della porta della vista all'interno della finestra di gioco.
View Port Y Coordinate Questa è la posizione Y della porta di vista all'interno della finestra di gioco.
View Port Width Questa è la larghezza (in pixel) della porta di visualizzazione.
View Port Height Questa è l'altezza (in pixel) della porta di visualizzazione.
View Port Surface ID Questo restituirà il valore ID unico per la superficie assegnata alla porta di vista o -1 se non è stata assegnata alcuna superficie.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Variable La variabile di vista costruita per ottenere
View La porta di vista a cui puntare (da 0 a 7)
Target La variabile di destinazione per contenere il valore restituito (può essere contrassegnata come variabile locale temporanea)

Esempio:

Il codice del blocco d'azione di cui sopra ottiene l'ID corrente della telecamera assegnato alla vista porta [0] e poi controlla se è lo stesso di quello memorizzato in una variabile globale. Se non lo è, la telecamera viene impostata su quella nuova.
