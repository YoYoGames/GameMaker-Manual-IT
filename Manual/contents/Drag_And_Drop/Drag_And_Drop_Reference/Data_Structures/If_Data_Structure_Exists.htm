Se la struttura dei dati esiste

Con questa azione potete controllare una variabile per vedere se indicizza una struttura dati o no. Dovete prima scegliere il tipo di struttura dati da controllare (una List, una Map, una Grid o una Stack) e poi fornite la variabile che contiene l'indice della struttura dati (come restituito dalle azioni Create List, Create Map, Create Grid e Create Stack). L'azione restituirà true se la struttura dati esiste, e false se non esiste. Si noti che è possibile segnalare "not" nell'azione per controllare se una variabile non contiene un indice di struttura dati.
