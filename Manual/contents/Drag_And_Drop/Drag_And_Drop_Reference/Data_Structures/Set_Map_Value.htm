Imposta valore mappa

Con questa azione è possibile aggiungere una coppia chiave/valore nella struttura dati della mappa. Si fornisce la variabile che contiene l'indice della mappa (come restituito dall'azione Create Map) e poi si dà una "chiave" (che è l'identificatore all'interno della mappa per un valore), e poi il "valore" da associare a quella chiave. Entrambe le chiavi possono essere valori interi (come 0, 200, 36712, ecc...) o strings (come "Name", "Shield" ecc...), ma il valore può essere qualsiasi valido data type.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Map L'indice (memorizzato in una variabile) della mappa da impostare
Key La chiave da impostare (intero o string)
Value Il valore da memorizzare con la chiave data

Esempio:

Il codice del blocco d'azione di cui sopra crea una nuova struttura dati mappa e poi imposta tre voci chiave/valore in essa.

Indietro: Azioni della struttura dei dati
Prossimo: Ottenere il valore della mappa
