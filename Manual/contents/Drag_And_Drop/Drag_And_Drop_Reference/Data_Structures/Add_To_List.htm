Aggiungi alla lista

Questa azione può essere usata per aggiungere un nuovo valore (di qualsiasi data type) alla lista, e questo valore sarà aggiunto alla fine. Si fornisce la variabile che memorizza l'indice della lista (come restituito dall'azione Create List) e il valore da memorizzare.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
List L'indice (memorizzato in una variabile) dell'elenco da aggiungere a
Value Il valore da aggiungere nella lista

Esempio:

Il codice del blocco d'azione di cui sopra crea una variabile d'ambito globale e poi crea una nuova struttura dati della lista, assegnando il suo valore di indice alla variabile globale. L'ambito viene quindi modificato in modo che tutte le istanze di object "obj_Enemy_Parent" aggiungano il loro valore ID unico nell'elenco.

Indietro: Azioni della struttura dei dati
Prossimo: Rimuovi dalla lista
