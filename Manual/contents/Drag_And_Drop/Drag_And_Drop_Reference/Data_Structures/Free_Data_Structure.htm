Struttura dati libera

Questa azione libererà la struttura dati dalla memoria e dovrebbe essere chiamata ogni volta che una struttura dati non è più necessaria, per esempio nel Destroy anche di un'istanza o nell'evento Room End, ecc. Liberare una struttura di dati significa che non è possibile può essere utilizzata per fare riferimento ad essa, in quanto non esisterà più, anche se è possibile controllare questo utilizzando l'azione If Data Structure Exists.

Notate che, a meno che la struttura di dati non abbia un indice assegnato a una variabile di ambito globale, non sarà trasportata da un room all'altro. Per esempio, se si crea un'istanza di un giocatore e una struttura dati di lista è assegnata a una variabile in quell'istanza, l'elenco dovrà essere liberato se il giocatore viene distrutto o se room cambia con l'istanza del giocatore al suo interno (a meno che l'istanza del giocatore sia contrassegnata come persistente), quindi l'istanza del giocatore avrà due azioni Free Data Structure per entrambi questi eventi. È importante che tutte le strutture dati create siano distrutte in questo modo quando non sono più necessarie, per evitare perdite di memoria che possono causare problemi di prestazioni o anche crash del gioco.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Type Il tipo di struttura di dati che si sta liberando (Lista, Mappa, Griglia o Pila)
Variable La variabile che contiene l'indice della struttura di dati da liberare

Esempio:

Il codice del blocco d'azione di cui sopra ottiene la dimensione della lista referenziata nella variabile globale e poi usa un for loop per loop attraverso di essa e distruggere le istanze con gli ID memorizzati nella lista. Dopo che loop ha finito, la lista viene liberata dalla memoria.
