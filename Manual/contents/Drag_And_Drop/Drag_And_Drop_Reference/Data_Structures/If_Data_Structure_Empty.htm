Se la struttura dei dati è vuota

Con questa azione potete controllare se la struttura dati data è vuota o no. Dovete prima scegliere il tipo di struttura dati da controllare (o una List, o una Map, o una Stack - niente griglie perché non possono nemmeno essere vuote) e poi fornite la variabile che contiene l'indice della struttura dati (come restituito dalle azioni Create List, Create Map e Create Stack). L'azione restituirà true se la struttura dati è vuota, e false se non lo è. Notate che potete segnalare "not" nell'azione per controllare se una struttura dati contiene dati controllando se non è vuota.
