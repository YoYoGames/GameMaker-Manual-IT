Ottenere informazioni sul rollback

Questa funzione restituisce un struct contenente informazioni su un giocatore. Deve essere usata solo quando si gestiscono manualmente le istanze dei giocatori. Se si utilizza Define Player, ogni istanza di giocatore riceverà automaticamente queste informazioni come variabili.

Per informazioni sul contenuto del file struct restituito, vedere: rollback_get_info

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Player ID L'ID del giocatore per cui ottenere informazioni
Target La variabile in cui deve essere memorizzato struct restituito

Indietro: Azioni multigiocatore di rollback
Avanti: Ottenere l'input
