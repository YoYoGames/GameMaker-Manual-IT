Utilizzare l'avvio manuale (rollback)

Per impostazione predefinita, una partita multigiocatore inizia automaticamente quando tutti i giocatori si sono uniti.

Il richiamo di questa azione disabilita il comportamento di avvio automatico, per cui il gioco attende che l'utente lo avvii manualmente.

Se utilizzata, questa azione deve essere eseguita prima dell'azione di unione/creazione.

Esempio:

Il blocco precedente abilita l'avvio manuale, quindi si unisce o crea una nuova partita multigiocatore.
