Ottenere l'input (Rollback)

Questa azione restituisce un struct contenente i valori di input di un giocatore. Richiede un argomento opzionale che specifica l'ID del giocatore per il quale verranno restituiti gli input.

Si consiglia di usare Se tasto giù (Rollback), Se tasto premuto (Rollback ) e Se tasto su (Rollback) per controllare gli input in multiplayer.

Quando viene utilizzato in un'istanza di giocatore creata da Define Player (Rollback), non è necessario specificare un ID giocatore, quindi è sufficiente specificare player_id.

Quando viene utilizzato in altre istanze o in istanze di giocatori create manualmente, è necessario specificare un ID giocatore.

Per informazioni su quali ingressi vengono restituiti e su come possono essere modificati, vedere Definizione degli ingressi.
