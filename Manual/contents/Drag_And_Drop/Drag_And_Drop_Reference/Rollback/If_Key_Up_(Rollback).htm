Se tasto su (Rollback)

Questa azione controlla se l'input dato è stato appena rilasciato per il giocatore corrente.

Consente di collegare azioni a destra, che vengono eseguite solo quando viene trovato l'input dato.

Argomenti:

Argomento Descrizione
Input Il nome dell'ingresso da controllare
Not Attivare questa opzione per verificare se l'ingresso non è stato rilasciato.

Indietro: Azioni multigiocatore di rollback
Avanti: Se l'evento Rollback è stato attivato
