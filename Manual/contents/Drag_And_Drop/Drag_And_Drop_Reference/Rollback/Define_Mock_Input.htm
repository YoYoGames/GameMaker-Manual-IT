Definire il finto ingresso (rollback)

Questa azione può essere utilizzata durante il Sync Test per assegnare un input temporaneo "finto" al giocatore specificato. Questo sostituisce i controlli definiti per il giocatore e qualsiasi input casuale proveniente da Sync Test.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Player ID Il numero del giocatore che riceve il set di input mock
Input Il nome dell'ingresso da definire
Value La costante di ingresso da assegnare (vedere Controlli personalizzati)

Esempio:

L'azione di cui sopra definisce i finti input fuoco, sinistra, destra, su e giù per il giocatore 1.
