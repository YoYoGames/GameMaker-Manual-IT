Ottieni le preferenze del giocatore (Rollback)

Quando si gioca una partita utilizzando Sync Test, tutti i giocatori "remoti" ricevono valori casuali per i loro input definiti, come metodo di base per testare. È possibile disattivare o riattivare questo comportamento utilizzando questa azione.

Argomenti:

Argomento Descrizione
Enabled Abilitare o disabilitare l'input casuale per i giocatori remoti durante i test.

Indietro: Azioni multigiocatore di rollback
Avanti: Utilizzare l'avvio manuale
