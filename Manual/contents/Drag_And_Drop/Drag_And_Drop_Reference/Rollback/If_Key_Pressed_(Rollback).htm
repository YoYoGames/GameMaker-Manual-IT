Se il tasto viene premuto (Rollback)

Questa azione controlla se l'input dato è stato appena premuto per il giocatore corrente.

Consente di collegare azioni a destra, che vengono eseguite solo quando viene trovato l'input dato.

Argomenti:

Argomento Descrizione
Input Il nome dell'ingresso da controllare
Not Attivare questa opzione per verificare se l'input non è stato premuto

Indietro: Azioni multigiocatore di rollback
