Se il tasto è abbassato (Rollback)

Quest'azione controlla se l'input dato è presente per il giocatore corrente.

Consente di collegare azioni a destra, che vengono eseguite solo quando viene trovato l'input dato.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Input Il nome dell'ingresso da controllare
Not Abilitare questa opzione per verificare se l'ingresso non è stato mantenuto

Esempio:

Le azioni di cui sopra spostano il giocatore a sinistra o a destra, a seconda dell'input.

Indietro: Azioni multigiocatore di rollback
Avanti: Se il tasto viene premuto (Rollback)
