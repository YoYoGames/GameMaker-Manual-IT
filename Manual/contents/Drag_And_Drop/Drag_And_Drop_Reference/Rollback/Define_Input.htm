Definire l'ingresso (Rollback)

Questa funzione serve a definire gli input personalizzati per tutti i giocatori. Prende una coppia Input/Value, dove Input è il nome dell'input e Value è la costante di input ad esso assegnata.

Per sapere come assegnare gli ingressi personalizzati, vedere Controlli personalizzati.

Se utilizzata, questa funzione deve essere eseguita prima dell'azione di unione/creazione.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Input Il nome dell'ingresso da definire
Value La costante di ingresso da assegnare (vedere Controlli personalizzati)

Esempio:

L'azione di cui sopra definisce gli input fuoco, sinistra, destra, su e giù per tutti i giocatori.

Indietro: Azioni multigiocatore di rollback
