Se c'è Object sul posto

Questa azione è usata per controllare e vedere se c'è un'istanza di qualsiasi object posta in una posizione basata sulla maschera di collisione dell'istanza chiamante. Si dà la posizione, che può essere una posizione assoluta o una posizione relativa all'istanza, e l'azione restituirà true se c'è una collisione o false se non c'è. Si può anche controllare il flag "Not" per verificare se non c'è una collisione nella posizione data, e l'azione restituirà true se non vengono trovate collisioni e false altrimenti. Le collisioni sono calcolate in base alla maschera di collisione dell'istanza chiamante che si sovrappone alla maschera di collisione di qualsiasi istanza nella posizione (come se fosse "piazzata" nella posizione).

IMPORTANTE! Le collisioni saranno registrate solo per quelle istanze che hanno una maschera di collisione valida, cioè: hanno un sprite assegnato al sprite_index, o un sprite assegnato al mask_index. Se una delle istanze in la collisione non ha una maschera di collisione, allora la collisione non verrà rilevata, indipendentemente da ciò che l'istanza sta disegnando in quel momento.
