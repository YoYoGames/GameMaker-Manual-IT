Se Room è primo

Questa azione controllerà per vedere se l'attuale room è il primo room come elencato nel Room Manager. L'azione restituirà true se l'attuale room è il primo nel gestore Room e false se non lo è. È possibile contrassegnare la casella di controllo not per trasformarla in "If Room Is NOT first".

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite se il "if" valuta true, mentre qualsiasi azione abbandonate altrove saranno eseguite dopo il blocco "if".
