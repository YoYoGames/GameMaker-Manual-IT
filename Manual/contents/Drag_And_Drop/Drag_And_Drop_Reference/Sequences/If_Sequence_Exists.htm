Se Sequence esiste

Questa azione può essere usata per controllare se l'elemento dato sequence esiste nel gioco room. Si dà l'ID dell'elemento, come restituito dall'azione Create Sequence, e se l'elemento sequence esiste in room allora l'azione restituirà true altrimenti restituirà false.

Se si contrassegna l'argomento "Not", allora l'azione controllerà per vedere se non esiste un tale elemento sequence e se non ne viene trovato nessuno restituirà true.

Notate che per aggiungere azioni nel blocco "if", esse devono essere lasciate cadere a lato dell'azione, come mostrato nell'immagine qui sotto:

Queste azioni saranno ora eseguite se il blocco "if" valuta a true, mentre qualsiasi azione abbandonata altrove sarà eseguita dopo il blocco "if".
