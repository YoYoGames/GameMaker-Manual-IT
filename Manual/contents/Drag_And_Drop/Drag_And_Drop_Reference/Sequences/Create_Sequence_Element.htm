Creare una sequenza

Con questa azione si può creare un elemento sequence di qualsiasi sequence dal Browser Asset. Si dà il nome dell'elemento sequence e una posizione x/y all'interno di room dove deve essere creato, così come il livello su cui deve essere creato (questo livello deve essere un livello asset). Puoi spuntare i flag "relativi" per posizionare sequence relativamente all'istanza che esegue i blocchi di codice dell'azione. Ogni volta che si crea un sequence nel gioco in questo modo, gli viene dato un valore ID unico dell'elemento con il quale si può poi identificare il sequence in ulteriori azioni, quindi questa azione permette di definire una variabile "target" per contenere questo valore ID (come mostrato nell'esempio qui sotto). Si noti che il target può essere contrassegnato come una variabile temporanea (locale), che sarà creata per l'azione e può essere utilizzata in tutte le azioni successive per l'evento in cui si trova l'azione.

Sintassi dell'azione:

Argomenti:

Argomento Descrizione
Sequence Il nome del sequence di cui creare un elemento.
x La posizione x in room per creare il sequence (può essere relativa all'istanza chiamante).
y La posizione y in room per creare il sequence (può essere relativa all'istanza chiamante).
layer Il nome (un string) del livello asset su cui creare il sequence.
Target La variabile a cui puntare per il valore di ritorno.

Esempio:

Il codice del blocco d'azione di cui sopra crea un nuovo sequence e memorizza il suo unico valore ID dell'elemento in una variabile. Questa variabile viene poi usata per mettere in pausa sequence usando l'azione Pause Sequence.
