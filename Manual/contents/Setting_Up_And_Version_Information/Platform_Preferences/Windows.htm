Preferenze di Windows

Le preferenze di Windows hanno le seguenti opzioni:

  • Posizione di Visual Studio: Questa opzione serve per puntare GameMaker all'installazione di Microsoft Visual Studio sul PC di costruzione.
  • Tipo di impacchettamento predefinito: Quando si costruisce un pacchetto eseguibile finale Windows, si può scegliere se creare un installatore (usando NSIS) o un file compresso ZIP che conterrà il gioco e qualsiasi file aggiuntivo al suo interno. Di default ti verrà mostrata una finestra di dialogo per scegliere tra queste due opzioni durante la compilazione, ma puoi impostare questa preferenza per usare sempre l'una o l'altra. Nota che quando questa opzione è impostata su Show Dialog, nella finestra di dialogo che appare in fase di compilazione, l'abilitazione di "Remember Packaging Option" imposterà questa preferenza sull'opzione selezionata.
