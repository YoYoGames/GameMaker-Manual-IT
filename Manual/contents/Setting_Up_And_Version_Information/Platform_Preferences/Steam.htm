Preferenze del vapore

L'SDK Steam è usato per le build di Windows, macOS e Ubuntu (Linux) che devono essere distribuite attraverso la piattaforma Steam. Questa sezione contiene le opzioni di configurazione necessarie per farlo funzionare nei vostri progetti:

  • Percorso a Steam SDK: Il sistema path dove avete installato l'SDK Steam. Una volta impostato questo path all'SDK, sarà necessario attivare Steam nelle Opzioni di gioco e poi per ogni piattaforma aggiungere l'App ID.
