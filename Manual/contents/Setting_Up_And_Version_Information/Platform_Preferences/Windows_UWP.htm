Preferenze UWP di Windows

Le preferenze di Windows UWP hanno le seguenti opzioni:

  • Posizione di Visual Studio: Questa opzione serve per puntare GameMaker all'installazione di Microsoft Visual Studio sul PC di costruzione.
  • Win UWP Tools location: Questa opzione serve per puntare GameMaker all'installazione degli strumenti UWP path sul PC di costruzione.
  • Architettura predefinita del pacchetto: Per impostazione predefinita, viene mostrata una finestra di dialogo per scegliere tra le architetture x86, x64 e ARM quando si compila un eseguibile UWP. Puoi impostare questa opzione su una di queste tre, che sarà poi selezionata di default ogni volta che si sceglie di compilare un eseguibile.
  • Risposta automatica alla creazione del pacchetto del negozio: nella finestra di dialogo mostrata durante la compilazione, puoi scegliere di creare un pacchetto del negozio. Puoi impostare questa opzione su Yes o No , che sarà usata per default ogni volta che scegli di compilare un eseguibile.
