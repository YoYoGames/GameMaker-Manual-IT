Preferenze tvOS

Le preferenze di tvOS hanno le seguenti opzioni:

  • Risposta automatica a tvOS avviso di limitazione della memoria persistente: i giochi tvOS hanno una memoria locale limitata per i file, e quando si compila per questo obiettivo un avviso verrà mostrato di default per ricordarvi questo fatto. È possibile cambiare l'impostazione qui per sopprimere l'avviso selezionando "OK". L'impostazione predefinita è "Show Message".
  • Sopprimere la compilazione e l'esecuzione: Per impostazione predefinita, quando si crea un progetto tvOS e poi si costruisce un pacchetto eseguibile finale, GameMaker costruirà il pacchetto e poi tenterà di eseguirlo su qualsiasi dispositivo collegato. Selezionando questa opzione sopprimerà la costruzione dell'app e la sua esecuzione sul dispositivo e creerà semplicemente un progetto XCode sul Mac host.
  • Percorso di installazione di macOS: L'installazione path per i file del progetto sul Mac di costruzione.
  • Identificatore di squadra predefinito: Qui puoi aggiungere il tuo ID di squadra predefinito, come ti è stato assegnato da Apple. Questo ID di squadra sarà usato quando i file di gioco saranno inviati a Xcode per costruire l'app e permetterà a Xcode di generare i certificati di firma richiesti. Si noti che questa impostazione sarà applicata di default a tutti i giochi costruiti per tvOS, ma può essere annullata per progetto dalle General tvOS Opzioni di gioco.
