Preferenze HTML5

Le preferenze di HTML5 hanno le seguenti opzioni:

  • Tipo di packaging predefinito: Quando si costruisce un pacchetto eseguibile HTML5 finale, è possibile scegliere tra Package as loose files - che genererà semplicemente tutti i file necessari in una cartella - o Package as Zip - che creerà un file compresso ZIP che avrà il gioco ed eventuali file aggiuntivi all'interno. Per impostazione predefinita ti verrà data la possibilità di scegliere quale di queste due opzioni vuoi scegliere in fase di compilazione, ma puoi impostare questa preferenza per usare sempre l'una o l'altra. Nota che quando questa opzione è impostata su Show Dialogue, nella finestra di dialogo che appare in fase di compilazione, scegliendo "Remember Packaging Option" imposterai questa preferenza sull'opzione che hai selezionato.
  • Default Web Server Port: Questo permette di cambiare la porta predefinita per il web HTML5 server. Il valore predefinito è 51264.
  • Indirizzo web predefinito Server: Qui potete cambiare l'indirizzo web predefinito server per il web HTML5 server. Il valore predefinito è 127.0.0.1.
  • Default Web Server Allowed IPs: Qui è possibile aggiungere qualsiasi IP consentito per il web server e saranno impostati come predefiniti per tutti i dispositivi aggiunti attraverso il Target Manager.

IMPORTANTE Quando si usa un particolare dispositivo di destinazione (cioè il browser), i valori IP e di porta impostati per quel particolare dispositivo sono usati al posto di queste preferenze. Controlla la Gestione dispositivi per vedere un elenco dei tuoi dispositivi con i loro dettagli.
