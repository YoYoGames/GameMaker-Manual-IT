Preferenze di Opera GX

Le preferenze di Opera GX presentano le seguenti opzioni:

  • Posizione dell'SDK Emscripten: Questo è il percorso dell'SDK Emscripten, necessario per le build YYC realizzate per Opera GX. Per informazioni sulla configurazione dell'SDK, leggere Impostazione di Opera GX.
  • Porta predefinita del sito Server : consente di modificare la porta predefinita del sito Opera GX server. Il valore predefinito è 51264.
  • Indirizzo web Server predefinito: Qui è possibile modificare l'indirizzo web server predefinito per il sito web di Opera GX server. Il valore predefinito è 127.0.0.1.
  • IP consentiti predefiniti per il Web Server: Qui si possono aggiungere tutti gli IP consentiti per il web server e saranno impostati come predefiniti per tutti i dispositivi aggiunti tramite Target Manager.
