Preferenze dell'editor di curve di animazione

Le Preferenze delle curve di animazione sono utilizzate per definire alcune proprietà per quando si lavora con l'Editor delle curve di animazione. Le opzioni disponibili sono:

  • Risposta automatica all'eliminazione della curva: Questa opzione può essere usata per controllare il messaggio pop-up che viene mostrato quando si elimina una curva dall'editor di curve di animazione. L'azione predefinita è quella di mostrare un messaggio di avviso, ma potete impostare su "Ok" e la finestra di dialogo non verrà mostrata.
