Preferenze dell'editor di caratteri

Le preferenze dell'Editor di caratteri sono utilizzate per definire il comportamento dell'Editor di caratteri. Le diverse opzioni disponibili sono elencate di seguito insieme ad una descrizione di ciò che ciascuna fa.

  • Aggiorna Fonts importato da progetti GameMaker 1: Quando questa opzione è selezionata, GameMaker tenterà di aggiornare fonts e rigenerare il loro texture. Questo accadrà solo se:
    • Questa preferenza è abilitata
    • Il sito font è disponibile sulla macchina
    • GameMaker può individuare una faccia corrispondente nella famiglia font (questa è la causa più probabile di incoerenza nell'importazione)
    Notare che accadrà anche se il sito font di GameMaker 1 non ha una cache font texture. Se un font legacy non può essere aggiornato, allora GameMaker utilizzerà di default la cache font texture e l'Editor di Font non vi permetterà più di modificare font, mostrando un'opzione per il backup e l'aggiornamento su un altro:

    Facendo clic su questo vi permetterà di modificare il font, aggiungendo il font mancante o impostandone un altro, e il backup di font è memorizzato nella cartella font asset, come <font name>.yy.gms1 e <font name>.png.gms1. Per ripristinare questi file, cancellate manualmente i file <font name>.yy e sostituiteli con questi rimuovendo il .gms1 extension dai nomi dei file. Importante! Tutti i file assets memorizzano i loro nomi nel file .yy, quindi è necessario modificare questi e mettere il nome giusto della risorsa anche nel campo <name> (se diverso).
  • Scelta automatica per il dialogo di rigenerazione del contenuto fallito: Quando si va a rigenerare un font asset texture , viene mostrato un avvertimento. Impostando questa opzione su "OK", il sito font verrà rigenerato senza che venga mostrato.
  • Scelta automatica per dialogo di rigenerazione fallita di font: Quando la rigenerazione di un font asset texture fallisce, viene mostrato un avviso. Impostando questa opzione su "OK" si eviterà di mostrare il messaggio.
  • Scelta automatica per l'aggiornamento dei font legacy importati: Quando importate un progetto legacy, potreste ricevere un avviso su fonts e l'offerta di aggiornamento. Impostando questa opzione su "OK" l'aggiornamento andrà bene automaticamente senza mostrare questo messaggio.
