Preferenze generali

Le Preferenze generali vengono utilizzate per definire il comportamento generale dell'IDE GameMaker. Le più generali sono elencate nella pagina principale, mentre molte altre sono suddivise in più sottocategorie, spiegate più avanti in questa pagina.

  • Lingua IDE: Questa opzione consente di selezionare la lingua per il sito IDE dall'elenco delle opzioni fornite. L'impostazione predefinita è l'inglese.
  • Font IDE: Questa opzione consente di cambiare il font utilizzato in tutto il sito IDE. L'impostazione predefinita è Open Sans.
  • Framerate massimo dell'editor di GameMaker: Questa opzione consente di impostare il framerate massimo a cui GameMaker IDE si aggiornerà, permettendo di impostare il framerate in base alla velocità di aggiornamento del monitor. Il valore predefinito è 60.
  • Pelle IDE: Qui si può scegliere tra le diverse skin che IDE può utilizzare. Per impostazione predefinita, GameMaker viene fornito con due skin, una chiara e una scura, e la configurazione iniziale utilizza la skin scura.
  • Dispositivo audio predefinito: Qui è possibile scegliere il dispositivo audio predefinito da utilizzare per GameMaker IDE. È possibile aggiornare l'elenco facendo clic sul pulsante Aggiorna .
  • Risposta automatica alla conferma di uscita: Questa opzione consente di impostare se visualizzare un messaggio di conferma quando si esce da GameMaker. Per impostazione predefinita è impostata su "mostra messaggio", ma può essere impostata in modo da saltare il messaggio come se si fosse fatto clic su OK.
  • Risposta automatica alla finestra di dialogo Sistema Font modificato: Questa opzione consente di impostare se visualizzare un messaggio di conferma quando un sistema font è stato aggiunto, rimosso o modificato. Per impostazione predefinita è impostata su "mostra messaggio", ma può essere impostata in modo da saltare il messaggio come se si fosse fatto clic su OK.
  • Abilita le notifiche: Selezionando questa opzione, il sito IDE sarà in grado di mostrare le notifiche relative ai nuovi siti runtimes, ecc. Questa opzione è attiva per impostazione predefinita.
  • Timeout di notifica (ms): Imposta la durata (in millisecondi) per la quale viene visualizzata una notifica. L'impostazione predefinita è 6000.
  • Logout all'uscita: se questa opzione è selezionata, IDE disconnetterà automaticamente la sessione dell'utente alla chiusura di GameMaker. Questa opzione è rivolta principalmente agli utenti del settore educativo, in quanto, se selezionata, significa che ogni volta che il programma si chiude l'utente viene disconnesso, liberando così il posto della sessione per altri utenti. Questa opzione è disattivata per impostazione predefinita.
  • Mostra schermata di benvenuto all'avvio: Questa opzione è attiva per impostazione predefinita e indica semplicemente a GameMaker di mostrare una breve schermata iniziale all'avvio del programma. Se si deseleziona questa opzione, la schermata iniziale non verrà visualizzata.
  • Consentire la chiusura delle schede vuote tramite una scorciatoia: è possibile chiudere tutte le schede aperte utilizzando la scorciatoia da tastiera / + W, ma se si disabilita questa opzione la scorciatoia non funzionerà più. Questa opzione è attivata per impostazione predefinita.
  • Aprire nuove risorse per la modifica trascinando i file nell'IDE: Alcune risorse (come sprites o i suoni) possono essere trascinate sul sito principale Workspace di GameMaker IDE e verranno rilevate e aggiunte come nuove risorse nel Browser Asset. Per impostazione predefinita, l'editor di risorse corrispondente verrà aperto per la modifica quando ciò accade, ma se si deseleziona questa opzione, gli editor non verranno aperti e sarà necessario aprirli manualmente.
  • Aprire le nuove risorse per la modifica al momento della creazione: Se questa opzione è selezionata, ogni volta che si crea una nuova risorsa nel Browser Asset, l'editor di risorse corrispondente verrà aperto automaticamente. Questa opzione è selezionata per impostazione predefinita e, se viene deselezionata, sarà necessario aprire manualmente tutte le risorse appena create.
  • Ricarica automaticamente i file modificati: Se questa opzione è selezionata, IDE scansiona e controlla automaticamente i file per vedere se sono stati modificati al di fuori di GameMaker e, in caso affermativo, li ricarica.
  • Disabilitare le animazioni di transizione di IDE: GameMaker IDE dispone di una serie di transizioni animate per l'apertura di finestre e così via... Tuttavia, non sono di gradimento a tutti e, se disabilitate, potrebbero aumentare le prestazioni su macchine vecchie o poco potenti. È possibile disattivarle qui e sono attive per impostazione predefinita.
  • Opt-in per i sondaggi di feedback: Quando questa opzione è selezionata, si comunica a GameMaker che si desidera partecipare ai sondaggi di feedback; ciò significa che di tanto in tanto, durante il lavoro, verrà visualizzato un messaggio relativo all'area su cui si sta lavorando, con la richiesta di una valutazione e di un feedback. Questa opzione è attiva per impostazione predefinita.
  • Abilita la sostituzione dei DPI: Questa opzione consente di sovrascrivere l'impostazione DPI predefinita con la propria. È necessario prima abilitare l'opzione e poi impostare il valore Raw DPi Override (il valore predefinito è 96) o impostare la percentuale del DPi nativo (valore predefinito 100%); una volta modificata, verrà richiesto di riavviare IDE, il che potrebbe richiedere di salvare il lavoro svolto sul progetto attualmente aperto. Si noti che il DPi ha una percentuale minima del 50% e massima del 500%; se si passa il mouse sull'icona delle informazioni, si può vedere l'impostazione massima consigliata per il display in uso:

    Se si supera l'impostazione DPi consigliata per il monitor, viene visualizzata una notifica che consente di ripristinare la DPi.
  • Use US Date Format: This setting changes the format used to display dates in the IDE. When enabled, dates are displayed as e.g. "Thursday, April 27, 2023", when disabled, dates are displayed as e.g. "27 April 2023" instead. By default this preference is unchecked.
