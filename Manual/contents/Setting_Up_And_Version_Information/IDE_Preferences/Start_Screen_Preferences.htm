Preferenze della schermata iniziale

Le Preferenze della schermata iniziale ti permettono di cambiare il comportamento della pagina iniziale. Le opzioni principali sono:

  • Mostra Tour: Questo permette di selezionare se il tour introduttivo viene mostrato quando si avvia IDE. Questo viene automaticamente disabilitato la prima volta che si salta il tour.
  • Show Setup Wizard: Questo permette di selezionare se il Setup Wizard viene mostrato quando si avvia IDE. Il Setup Wizard essenzialmente ti porta attraverso i passi necessari per creare un nuovo progetto.
  • Stile elenco progetti recenti: Questo ti permette di cambiare la visualizzazione dell'elenco dei progetti recenti nella pagina iniziale tra "Elenco" e "Griglia".
