Preferenze di sfondo

Le preferenze di Sfondo si occupano dello sfondo di Workspace e di cosa visualizzerà così come di come verrà visualizzato. Le opzioni sono:

  • Immagine di sfondo: Questo è il file immagine che sarà usato come sfondo per i diversi Workspaces. C'è un'immagine predefinita fornita ed è abilitata di default, ma puoi selezionare la tua immagine qui. I tipi di immagine che potete selezionare sono *.png, *.gif, *.jpg, *.jpeg, *.tif, *.tiff, *.ico, *.bmp, *.2bp o *.mdi, ma nota che indipendentemente dal formato iniziale dell'immagine, tutti i file saranno salvati come file *.png, quindi i file animati *.gif per esempio saranno resi solo come un singolo frame iniziale.
  • Scalatura immagine: Questo controlla come l'immagine di sfondo sarà scalata all'interno dei diversi workspaces. Le scelte sono:
    • Stretch - L'immagine verrà allungata per adattarsi al sito workspace, cambiando scala a seconda delle banchine visibili
    • Tile - L'immagine verrà piastrellata per coprire lo spazio di lavoro
    • Best Fit - L'immagine sarà scalata per adattarsi a workspace proporzionalmente, con l'aggiunta di barre intorno al bordo se non è un adattamento 1:1
    • Desktop Stretch - Lo sfondo coprirà l'intero workspace, e si muoverà piuttosto che cambiare scala quando i dock vengono aperti/chiusi
    • Screen Stretch - Lo sfondo coprirà un'area della dimensione della finestra e sarà ritagliato/esteso per mostrare più o meno dell'immagine quando i dock vengono aperti/chiusi.
  • Mostra immagini di sfondo sulle aree di lavoro: Questa opzione controlla se l'immagine di sfondo selezionata deve essere mostrata o meno. Per impostazione predefinita è selezionata come on.
