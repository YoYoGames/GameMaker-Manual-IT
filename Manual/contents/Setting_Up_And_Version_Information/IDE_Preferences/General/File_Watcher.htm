Preferenze di File Watcher

Il File Watcher è il nome dato alla parte di IDE che guarda i file relativi a un progetto per qualsiasi modifica dovuta a software esterno (ad esempio: la modifica di un'immagine in un editor esterno) o dovuta a modifiche interne come la modifica di uno script asset. In generale, queste impostazioni non dovrebbero aver bisogno di essere modificate, ma su alcuni sistemi - come quelli che usano un SSD come unità di salvataggio per i file del progetto - il file watcher potrebbe informarti dei cambiamenti quando fai cose come salvare semplicemente il o ogni volta che modifichi qualsiasi progetto asset, nel qual caso potresti aver bisogno di cambiare queste impostazioni.

Le impostazioni disponibili sono:

  • Abilita File Watcher: Questo può essere usato per abilitare o disabilitare il sistema File Watcher. Se questo è disabilitato, IDE non raccoglierà più i file modificati da fonti esterne, quindi bisogna fare attenzione quando si disabilita questa caratteristica. Il File Watcher è abilitato di default.
  • Imposta la tolleranza del File Watcher in MS dopo il salvataggio nell'IDE: Questo regola il tempo tra il salvataggio di un file e il controllo del File Watcher per le modifiche. Potresti regolare questo valore se stai salvando il tuo progetto e il File Watcher stava erroneamente appariva dicendo che qualcosa era cambiato, nel qual caso aumentereste il ritardo fino a quando il messaggio non appare più. Il valore predefinito è 2000ms, ma può variare da 1ms fino a 10.000ms.
