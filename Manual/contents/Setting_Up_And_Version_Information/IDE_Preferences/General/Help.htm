Preferenze di aiuto

Il manuale di aiuto è progettato per funzionare come una pagina web e anche se i file possono essere memorizzati localmente (a seconda delle impostazioni), richiede l'uso di un browser esterno per visualizzarli. Le seguenti impostazioni relative al manuale d'aiuto possono essere cambiate qui:

  • Lingua del manuale: Questo imposta la lingua del manuale. L'impostazione predefinita è "Inglese".
  • Usa il manuale online: Abilita questo per usare il sito web del manuale online ogni volta che il manuale viene aperto, o disabilitalo per eseguire il manuale localmente. Questo è disabilitato per impostazione predefinita.
  • Help manual port: Questo imposta il numero di porta per il micro web server manuale da usare, se "Use online manual" è disabilitato. Il valore predefinito è 51290.
