Preferenze di risparmio energetico

Le preferenze di risparmio energetico qui influenzeranno quanta potenza della CPU e della GPU sarà usata per generare il GameMaker IDE (essenzialmente questo abbasserà il framerate di base a cui il programma gira e quindi abbasserà il consumo energetico).

Le opzioni qui sono:

  • Framerate dell'alimentazione AC (%): Il modificatore di framerate di base IDE quando è collegato a un ingresso di alimentazione. Il valore predefinito è 100%.
  • Framerate della batteria (%): Il modificatore di framerate di base IDE quando si usa la batteria per alimentare la macchina. Il valore predefinito è 35%.
  • Framerate di sfondo (%): Il modificatore di framerate di base IDE quando GameMaker non è a fuoco (indipendentemente dall'alimentazione). Il valore predefinito è 15%.
