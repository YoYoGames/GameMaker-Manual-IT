Preferenze delle risorse

Le preferenze qui contengono opzioni per come certe risorse saranno disegnate nei vostri progetti:

  • Colore di disegno iniziale di default per i nuovi progetti: Quando usate un font o disegnate una forma usando le funzioni di GameMaker o le azioni di GML Visual senza specificare un colore, il colore impostato qui è quello che verrà usato. Il colore predefinito è il bianco ( ARGB: $FFFFFFFFFF).
  • Colore iniziale predefinito della finestra per i nuovi progetti: Questa opzione è usata per impostare il colore di sfondo della pagina HTML per il target HTML5, così come per impostare il colore mostrato fuori da rooms su tutti i target (incluso HTML5). Il colore predefinito è nero (ARGB: $000000FF).
