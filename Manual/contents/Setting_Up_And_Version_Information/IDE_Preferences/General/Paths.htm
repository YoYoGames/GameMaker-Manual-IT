Preferenze di percorso

Le preferenze Path controllano i diversi file paths richiesti dagli elementi in GameMaker IDE così come per qualsiasi editor esterno richiesto. Esistono le seguenti opzioni per l'IDE:

  • Directory temporanea: La posizione per salvare tutte le cartelle temporanee. Per impostazione predefinita su Windows questa è

    \Users\<Username>\AppData\Local\Temp\GameMaker

    E su macOS è:

    /var/folders/<Path_Hash>/GameMaker

  • IDE cache directory: La posizione per il salvataggio di IDE cache. Per default su Windows questa è

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    E su macOS è:

    /Users/<Username>/.config/GameMaker/Cache

  • Directory delle risorse cache: La posizione per salvare il asset cache per ogni progetto. Per impostazione predefinita su Windows questo è

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    E su macOS è:

    /Users/<Username>/.config/GameMaker/Cache

  • Posizione "I miei progetti": La posizione in cui GameMaker creerà inizialmente i nuovi progetti. Per impostazione predefinita su Windows questo è

    \Users\<Username>\Documents\GameMaker

    E su macOS è

    /Users/<Username>/GameMaker

  • Elimina automaticamente la cartella temporanea alla chiusura: Selezionando questa opzione si forza GameMaker a cancellare automaticamente la cartella Temp che crea per ogni progetto per la compilazione, ecc... Questa impostazione è attiva per impostazione predefinita, e deselezionandola switch la disattiverà (il che significa che dovrete rimuovere manualmente qualsiasi file temporaneo in seguito).
  • Elimina automaticamente asset cache alla chiusura: Se questa opzione è selezionata, la cartella asset del compilatore cache sarà rimossa quando si chiude GameMaker. Questo è disattivato per impostazione predefinita, e abilitarlo significherà che ogni volta che caricate ed eseguite qualsiasi progetto il cache dovrà essere ricostruito (il che può richiedere tempo a seconda della dimensione del gioco).
  • Elimina automaticamente IDE cache alla chiusura: Se questa opzione è selezionata, la cartella IDE del compilatore cache sarà rimossa quando si chiude GameMaker. Questo è disattivato per impostazione predefinita, e abilitarlo significherà che ogni volta che si avvia GameMaker il cache dovrà essere ricostruito.
  • Elimina la cartella temporanea: Facendo clic su questo pulsante si cancella la cartella temporanea del progetto.
  • Cancella Asset Cache: Facendo clic su questo pulsante si cancella il compilatore asset cache per il progetto.
  • Cancella IDE Cache: Facendo clic su questo pulsante si cancella la cache di IDE

Esistono le seguenti opzioni per impostare paths agli editor esterni:

  • Percorso all'editor/visualizzatore esterno per i file SWF: Se stai lavorando con il formato SWF sprites, puoi impostare questo a path del tuo visualizzatore/editor preferito e quando fai clic sul pulsante Modifica immagine nell'editor Sprite allora si aprirà il programma dato. Il valore predefinito qui è di non avere path.
  • Percorso all'editor/visualizzatore esterno per i file Spine: Se stai lavorando con il formato Spine sprites , puoi impostare questo valore su path del tuo visualizzatore/editor preferito e quando fai clic sul pulsante Modifica immagine nell'editor Sprite allora si aprirà il programma dato. Il valore predefinito qui è di non avere path.
