Preferenze dell'editor di set di piastrelle

Le preferenze dell'Editor di set di piastrelle hanno la seguente opzione per l'Auto Tiling:

Auto tile background colour: Questa opzione imposta il colore di sfondo all'interno dell'editor di tile set nella sezione per la creazione di Auto Tiles. Il colore predefinito è (rgba) #808080FF.

Le preferenze di Tile Set Editor hanno anche una sottosezione Confirm Dialogs :

L'opzione disponibile qui controlla il messaggio che appare quando si cambia la dimensione della cella di tile set e questo influenzerà il modo in cui i livelli di room tilemap sono resi (risultando in livelli tagliati). Per impostazione predefinita ti verrà richiesto di continuare o annullare l'operazione, e questa opzione ti permette di sopprimere questo messaggio selezionando "Okay" che andrà avanti con la modifica delle celle di tile set senza considerare l'impatto sui livelli di room tilemap. Il valore predefinito è "Show Message".
