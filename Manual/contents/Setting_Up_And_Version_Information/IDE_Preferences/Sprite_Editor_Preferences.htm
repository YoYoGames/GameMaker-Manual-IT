Preferenze dell'editor di sprite

Le preferenze dell'editor di sprite sono utilizzate per definire l'aspetto e il funzionamento dell'editor di sprite. Esistono le seguenti opzioni:

  • Default sprite origin: Questa opzione vi permette di impostare la posizione di origine predefinita per ogni nuova risorsa sprite che create. Il valore predefinito è in alto a sinistra.
  • Tipo di maschera di collisione predefinita sprite: Questa opzione permette di impostare il tipo predefinito di maschera di collisione da utilizzare per i nuovi sprites. Le scelte sono rettangolo, rettangolo ruotato, ellisse, diamante, o preciso / preciso per frame. L'impostazione predefinita è rettangolo.
  • Modalità di animazione predefinita di sprite: Questa opzione permette di impostare la modalità di animazione predefinita per sprites nell'editor di sprite. Potete scegliere tra " Frames per second" o " Frames per game frame", l'impostazione predefinita è " Frames per second".
  • Velocità di animazione di default sprite : questa opzione permette di impostare il valore della velocità di animazione di default per quando si crea un nuovo sprite asset. Notate che questo valore è legato alla modalità di animazione e - per esempio - 3 fotogrammi al secondo non è la stessa cosa di 30 fotogrammi per fotogramma di gioco, quindi assicuratevi di impostare questo valore correttamente in riferimento alla modalità preferita. Il valore predefinito è 30.
  • Default sprite width: Questa opzione permette di impostare la larghezza di default per tutti i nuovi sprites creati. Il valore predefinito è 64.
  • Altezza predefinita sprite: Questa opzione permette di impostare l'altezza predefinita per tutti i nuovi sprites. Il valore predefinito è 64.
  • Apply resize and scale to sprite origin: Quando è selezionata, questa opzione cambierà l'origine di sprite quando sprite è ridimensionato o scalato, con la nuova origine che viene spostata/scalata per corrispondere ai nuovi valori per sprite. Si noti che questo è applicabile solo per le origini sprite personalizzate che non sono bloccate nell'editor sprite. Se l'origine è bloccata o il sprite usa una delle posizioni predefinite - ad esempio: "Center" o "Top Left", ecc... - allora questa opzione sarà sarà grigia e non disponibile per la modifica. L'impostazione predefinita è abilitata.

Esiste anche la seguente sottocategoria per l'editor Sprite:

Confermare le finestre di dialogo

Le opzioni disponibili qui controllano i messaggi che appaiono quando viene richiesto di confermare un'azione. Puoi cambiare queste impostazioni per mostrare sempre il messaggio o per respingerlo con un'azione specifica (Sì o No):

  • Risposta automatica all'importazione di file: Quando si importa una nuova immagine e ne esiste già una, per default viene richiesto di continuare l'importazione e sovrascrivere l'immagine esistente o di annullare l'operazione. Questa opzione permette di sopprimere questo messaggio e annullare sempre l'importazione selezionando "No", o sovrascrivere l'attuale sprite selezionando "Sì". Il valore predefinito è "Show Message".
  • Risposta automatica al fallimento del caricamento delle immagini: Quando importi una nuova immagine e l'importazione fallisce, per default ti verrà mostrato un messaggio di avviso che ti informa che c'è stato un fallimento. Questa opzione ti permette di sopprimere questo messaggio selezionando "Ok". Il valore predefinito è "Show Message".
