Note di rilascio

Ogni versione di GameMaker viene fornita con il proprio set di note di rilascio. Queste contengono informazioni dettagliate sui cambiamenti che sono stati fatti così come una lista completa di tutti i bug che sono stati risolti. Questo è uno strumento inestimabile per decidere se aggiornare la vostra versione corrente, o quando trovate qualcosa che potrebbe essere un bug dopo una release.

Le note di rilascio sono divise in due serie diverse a causa del fatto che sia il IDE che l'attuale Runtimes possono essere aggiornati indipendentemente. Puoi trovare le attuali note di rilascio per IDE dal link qui sotto:

  • Note di rilascio dell'IDE

Potete trovare le note di rilascio attuali per Runtimes dal link qui sotto:

  • Note di rilascio del runtime
