Preferenze IDE

La finestra delle preferenze è una parte essenziale di GameMaker, in quanto è il luogo in cui è possibile impostare e personalizzare la propria esperienza d'uso di IDE, nonché impostare qualsiasi SDK specifico per la piattaforma ecc... In questa sezione vengono trattate le aree specifiche di IDE della finestra delle preferenze; per le singole piattaforme di destinazione, consultare la sezione Preferenze della piattaforma di destinazione. Si noti che, dopo aver modificato le preferenze in questa finestra, è necessario fare clic sul pulsante Applica per renderle effettive.

Se, dopo aver modificato le opzioni qui per personalizzare GameMaker secondo i propri gusti, si desidera eseguire il backup delle preferenze impostate, è necessario andare in questa cartella di Windows:

%AppData%/GameMaker/<your_account_name>/

O questa cartella su Mac:

~/.config/GameMaker

Quindi copiare i due file seguenti in un luogo sicuro:

  • local_settings.json
  • cloud_settings.json

Tuttavia, se si desidera ripristinare le preferenze modificate, è necessario fare clic sul pulsante Ripristina impostazioni predefinite nella parte superiore della finestra. Si noti che questa operazione ripristina le impostazioni predefinite solo per la sezione in corso di modifica.
