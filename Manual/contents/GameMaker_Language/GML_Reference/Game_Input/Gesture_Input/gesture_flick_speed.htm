gesto_flick_velocità

Questa funzione è usata per impostare la velocità necessaria affinché un evento Flick Gesture sia attivato quando viene rilasciato un tocco o un clic. La velocità è misurata in pollici al secondo e ha un valore predefinito di 2.0.

Sintassi:

gesture_flick_speed(speed);

ArgomentoTipo Descrizione
speed The speed (in inches per second) to set for flick gesture event detection.

Ritorna:

Esempio:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

Il codice qui sopra controlla se la velocità di flick per i gesti è impostata a 2 pollici al secondo e se non lo è la imposta a quel valore.
