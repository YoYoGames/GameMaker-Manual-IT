gesto_get_double_tap_distance

Questa funzione è usata per ottenere la distanza entro la quale si deve toccare/cliccare di nuovo lo schermo dopo un singolo tocco per attivare un Double Tap Gesture. La distanza è misurata in pollici e ha un valore predefinito di 0,1.

Sintassi:

gesture_get_double_tap_distance();

Ritorna:

(inches)

Esempio:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

Il codice qui sopra controlla se la distanza del doppio tocco per i gesti è impostata a 0,1 pollici e se non lo è la imposta su quel valore.
