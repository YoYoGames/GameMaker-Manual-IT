velocità del gesto_get_flick

Questa funzione è usata per ottenere la velocità necessaria affinché un evento Flick Gesture sia attivato quando viene rilasciato un tocco o un clic. La velocità è misurata in pollici al secondo e ha un valore predefinito di 2.0.

Sintassi:

gesture_get_flick_speed();

Ritorna:

(inches per second)

Esempio:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

Il codice qui sopra controlla se la velocità di flick per i gesti è impostata a 2 pollici al secondo e se non lo è la imposta a quel valore.
