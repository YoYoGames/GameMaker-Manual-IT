gesto_doppio_tap_tempo

Questa funzione è usata per impostare il tempo necessario tra due tocchi/click per innescare un evento Double Tap Gesture. Il tempo è misurato in secondi e ha un valore predefinito di 0,16.

Sintassi:

gesture_double_tap_time(time);

Argomento Tipo Descrizione
time Real The time (in seconds) to set for double tap gesture event detection.

Ritorna:

N/A

Esempio:

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

Il codice qui sopra controlla se il tempo del doppio tocco per i gesti è impostato a 0,16 secondi e se non lo è lo imposta su quel valore.
