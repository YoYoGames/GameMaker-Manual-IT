gesto_doppio_tap_distanza

Questa funzione è usata per impostare la distanza entro la quale si deve toccare/cliccare di nuovo lo schermo dopo un singolo tocco per attivare un Double Tap Gesture. La distanza è misurata in pollici e ha un valore predefinito di 0,1.

Sintassi:

gesture_double_tap_distance(distance);

ArgomentoTipo Descrizione
distance The distance (in inches) to set for double tap gesture event detection.

Ritorna:

Esempio:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_get_double_tap_distance(0.1);
}

Il codice qui sopra controlla se la distanza del doppio tocco per i gesti è impostata a 0,1 pollici e se non lo è la imposta su quel valore.

Indietro: Ingresso gestuale
