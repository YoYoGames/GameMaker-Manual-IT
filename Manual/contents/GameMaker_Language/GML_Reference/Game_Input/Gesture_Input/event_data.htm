dati_evento

Questa variabile ha uno scopo globale e viene usata per contenere una mappa DS quando viene usata nell'evento appropriato (ad es. eventi gestuali e messaggi broadcast), e -1 in tutte le altre occasioni. Il contenuto effettivo della mappa DS dipenderà dal tipo di evento che l'ha attivata, quindi fate riferimento alle singole sezioni per quegli eventi.

Sintassi:

event_data;

Ritorna:

DS Map ID

Esempio:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

Il codice qui sopra è preso dall'evento del gesto Drag End e controlla se lo sfogo è un evento "flick", e se lo è estrae i dati richiesti dalla mappa event_data DS e li usa per impostare alcune variabili. Se non viene rilevato un evento "flick", allora le stesse variabili vengono impostate a 0.

Indietro: Variabili globali
