tempo di trascinamento del gesto

Questa funzione è usata per impostare il tempo necessario affinché un evento Gesto di trascinamento sia attivato da un tocco o da un clic. Questo tempo influenzerà anche il modo in cui l'evento Tap viene innescato, poiché un tocco/click e un rilascio prima che questo tempo sia scaduto sarà considerato un Tap. Il tempo è misurato in secondi e ha un valore predefinito di 0,16.

Sintassi:

gesture_drag_time(time);

ArgomentoTipo Descrizione
time The time (in seconds) to set for drag gesture event detection.

Ritorna:

Esempio:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

Il codice qui sopra controlla se il tempo di trascinamento per i gesti è impostato a 0,16 secondi e se non lo è lo imposta a quel valore.
