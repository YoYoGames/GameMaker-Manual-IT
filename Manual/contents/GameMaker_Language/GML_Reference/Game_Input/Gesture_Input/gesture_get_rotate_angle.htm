tempo di rotazione del gesto

Questa funzione è usata per ottenere l'angolo che una coppia di tocchi deve superare per innescare un gesto di inizio rotazione. L'angolo è misurato in gradi e ha un valore predefinito di 5°.

Sintassi:

gesture_get_rotate_angle();

Ritorna:

(inches)

Esempio:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

Il codice qui sopra controlla se l'angolo di rotazione per i gesti è impostato a 5° e se non lo è lo imposta a quel valore.

Indietro: Ingresso gestuale
Il prossimo: gesture_drag_time
