tempo di rotazione del gesto

Questa funzione è usata per ottenere il tempo entro il quale una coppia di tocchi deve ruotare in una direzione coerente affinché un gesto Rotate Start sia attivato. Il tempo è misurato in secondi e ha un valore predefinito di 0.16s.

IMPORTANTE! Attualmente per un gioco a 60fps è possibile impostare solo un massimo di un secondo, altrimenti non verrà rilevata alcuna rotazione. Questo aumenterà per un framrate inferiore, per esempio un gioco a 30fps può avere un tempo massimo di 2 secondi.

Sintassi:

gesture_get_rotate_time();

Ritorna:

(inches)

Esempio:

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

Il codice qui sopra controlla se il tempo di rotazione per i gesti è impostato a 0,1 secondi e se non lo è lo imposta a quel valore.
