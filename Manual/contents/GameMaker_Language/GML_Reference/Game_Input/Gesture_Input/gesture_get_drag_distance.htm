distanza del gesto_get_drag

Questa funzione è usata per ottenere la distanza necessaria affinché un evento Gesto di trascinamento sia attivato dal movimento di un tocco o di un clic. La distanza è misurata in pollici e ha un valore predefinito di 0,1.

Sintassi:

gesture_get_drag_distance();

Ritorna:

(inches)

Esempio:

if gesture_get_drag_distance() != 0.1
{
    gesture_drag_distance(0.1);
}

Il codice qui sopra controlla se la distanza di trascinamento per i gesti è impostata a 0,1 pollici e se non lo è la imposta a quel valore.
