gesture_get_double_tap_time

Questa funzione viene utilizzata per ottenere il tempo necessario tra due tocchi/clic per attivare un evento Gesto di doppio tocco. Il tempo è misurato in secondi e ha un valore predefinito di 0,10.

Sintassi:

gesture_get_double_tap_time();

Restituzione:

Real (seconds)

Esempio:

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

Il codice precedente controlla se il tempo di doppio tocco per i gesti è impostato su 0,16 secondi e, in caso contrario, lo imposta su tale valore.
