gesto_pinch_angle_away

Questa funzione è usata per impostare l'angolo entro il quale un tocco deve allontanarsi da un altro tocco prima che un gesto di pizzicamento possa essere potenzialmente avviato. L'angolo è misurato in gradi e ha un valore predefinito di 45°.

Sintassi:

gesture_pinch_angle_away();

Ritorna:

(inches)

Esempio:

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

Il codice qui sopra controlla se l'angolo di pinch out per i gesti è impostato a 45° e se non lo è lo imposta a quel valore
