tastiera_get_numlock

Potete usare questa funzione per trovare lo stato del blocco del numero della tastiera con true restituito per on, e false restituito per off.

NOTA Questa funzionalità è disponibile solo nelle build di Windows exe e non funzionerà su nessun altro dispositivo.

Sintassi:

keyboard_get_numlock();

Ritorna:

Esempio:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

Il codice dell'esempio precedente otterrà lo stato del tasto numberlock e se è acceso (true) lo imposterà su off (false) e viceversa.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_key_press
