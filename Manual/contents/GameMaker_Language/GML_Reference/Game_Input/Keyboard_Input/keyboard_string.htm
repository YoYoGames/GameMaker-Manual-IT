stringa_tastiera

Questa variabile contiene un string contenente gli ultimi (al massimo) 1024 caratteri digitati sulla tastiera. Questo string conterrà solo caratteri stampabili digitati, ma risponderà correttamente alla pressione del tasto backspace cancellando l'ultimo carattere. Questa variabile non è di sola lettura e potete cambiarla, per esempio impostarla a "" (un string vuoto) se l'avete già gestita, e potete usare le Funzioni Stringa per manipolarla. Notate che quando usate la Tastiera Virtuale su schermo, solo questa variabile sarà aggiornata con l'input della tastiera.

Sintassi:

keyboard_string;

Ritorna:

Esempio:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

Il codice qui sopra limiterà la lunghezza della tastiera string a 15 caratteri, rimuovendo quelli che sono oltre quel limite copiando i primi quindici caratteri nella variabile.

Indietro: Ingresso da tastiera
Il prossimo: io_clear
