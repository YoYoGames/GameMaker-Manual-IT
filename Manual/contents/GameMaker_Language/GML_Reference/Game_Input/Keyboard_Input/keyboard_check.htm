controllo tastiera

Con questa funzione potete controllare se un tasto è tenuto premuto o no. A differenza della funzione keyboard_check_pressed() o keyboard_check_released() che vengono attivate solo una volta quando il tasto viene premuto o rilasciato, questa funzione viene attivata ogni volta che il tasto viene tenuto premuto. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale Keyboard Input.

Sintassi:

keyboard_check(key);

Argomento Tipo Descrizione
key Costante chiave virtuale (vk_*) The key to check the down state of.

Ritorna:

Boolean

Esempio:

if keyboard_check(vk_left)
{
    x -= 5;
}

Il codice di cui sopra controllerà per vedere se il tasto freccia è stato premuto e sposterà l'istanza di 5 pixel a sinistra ogni passo che ritorna vero.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_check_pressed
