tastiera_get_map

A volte si può desiderare di ottenere il codice ascii per una chiave mappata (per vedere se è già mappata, per esempio) che è ciò che questa funzione restituisce. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale Keyboard Input.

Sintassi:

keyboard_get_map(key);

ArgomentoTipo Descrizione
key This is the key that you wish to get the mapped value from

Ritorna:

Esempio:

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(ord("A"), keyboard_lastkey);
}

Il codice dell'esempio precedente controllerà prima se "A" è stato mappato su un altro tasto, e se non lo è lo mapperà sull'ultimo tasto che l'utente ha premuto.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_unset_map
