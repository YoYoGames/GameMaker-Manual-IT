tastiera_unset_map

Con questa funzione potete cancellare tutti i tasti rimappati in modo che ritornino al loro stato predefinito, cioè: tutti i tasti da mappare a se stessi.

Sintassi:

keyboard_get_map();

Ritorna:

N/A

Esempio:

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

Il codice dell'esempio precedente resetterà tutti i tasti mappati alle loro impostazioni predefinite se l'utente preme il tasto "escape".
