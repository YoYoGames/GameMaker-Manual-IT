tastiera_check_direct

Questa funzione restituirà true se il tasto con il particolare keycode è premuto, o false se non lo è, controllando direttamente l'hardware. Permette alcuni controlli in più, in particolare potete usare i keycode vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol e vk_ralt per controllare se il tasto shift sinistro o destro, control o alt è premuto. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale Keyboard Input.

NOTA Questa funzione è disponibile solo per il target standard Windows e il risultato è indipendente da quale applicazione ha il focus.

Sintassi:

keyboard_check_direct(key);

ArgomentoTipo Descrizione
key The key to check the down state of.

Ritorna:

Esempio:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

Il codice qui sopra controllerà per vedere se i tasti alt sinistro o destro sono stati premuti, e se lo sono, imposta la variabile "crouch" a true.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_clear
