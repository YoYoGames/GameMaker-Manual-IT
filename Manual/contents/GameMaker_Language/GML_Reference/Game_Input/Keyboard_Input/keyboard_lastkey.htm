tastiera_ultimo tasto

Questa variabile si riferisce al valore che keyboard_key era nel frame precedente, restituendo il keycode di quel tasto (vengono restituite tutte le costanti keycode standard). Questa variabile non è di sola lettura e potete cambiarla, per esempio impostarla a -1 se l'avete già gestita.

Sintassi:

keyboard_lastkey;

Ritorna:

Virtual Key Constant

Esempio:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Il codice qui sopra controlla per vedere se la variabile lastkey non è uguale a -1, e se lo è aggiunge qualunque sia l'ultimo tasto come string alla variabile "str", poi resetta la variabile keyboard_lastkey per accettare ulteriori input.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_lastchar
