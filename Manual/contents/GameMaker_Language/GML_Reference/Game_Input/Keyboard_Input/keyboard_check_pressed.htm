controllo tastiera premuto

Con questa funzione potete controllare se un tasto è stato premuto o no. A differenza della funzione keyboard_check() questa funzione verrà eseguita solo una volta per ogni volta che il tasto viene premuto, quindi per farla scattare di nuovo, il tasto deve essere prima rilasciato e poi premuto di nuovo. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale Keyboard Input.

Sintassi:

keyboard_check_pressed(key);

Argomento Tipo Descrizione
key Costante chiave virtuale (vk_*) The key to check the pressed state of.

Ritorna:

Boolean

Esempio:

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

Il codice di cui sopra avanzerà al prossimo room se il giocatore preme uno qualsiasi dei tasti della tastiera (lavorando come un prompt "Press Any Key to Continue").

Indietro: Ingresso da tastiera
Il prossimo: keyboard_check_released
