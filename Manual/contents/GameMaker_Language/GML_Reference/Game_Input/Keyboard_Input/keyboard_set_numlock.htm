impostare il numero di serrature

Potete usare questa funzione per switch attivare o disattivare il blocco dei numeri della tastiera (impostato su true per on, e false per off).

NOTA Questa funzionalità è disponibile solo nelle build del target Windows e non funzionerà su nessun altro dispositivo.

Sintassi:

keyboard_set_numlock(value);

Argomento Tipo Descrizione
value Booleano Set this to true for "on" and false for "off"

Ritorna:

N/A

Esempio:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

Il codice dell'esempio precedente otterrà lo stato del tasto numberlock e se è acceso (true) lo imposterà su off (false) e viceversa.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_get_numlock
