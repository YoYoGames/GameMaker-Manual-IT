Rilascio dei tasti della tastiera

Con questa funzione potete simulare il rilascio di qualsiasi tasto della tastiera. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti di vk_* elencate nella pagina principale Keyboard Input.

Sintassi:

keyboard_key_release(key);

ArgomentoTipo Descrizione
key The key to simulate a release of.

Ritorna:

Esempio:

keyboard_key_release(vk_space);

Questo simulerà un rilascio della barra spaziatrice.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_key
