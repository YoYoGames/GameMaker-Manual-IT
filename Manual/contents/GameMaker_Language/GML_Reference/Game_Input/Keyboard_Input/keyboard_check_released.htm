tastiera_check_released

Con questa funzione potete controllare se una chiave è stata rilasciata o meno. A differenza della funzione keyboard_check() questa funzione verrà eseguita solo una volta per ogni volta che il tasto viene sollevato, quindi per farla scattare di nuovo, il tasto deve essere prima premuto e poi rilasciato di nuovo. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale Keyboard Input.

Sintassi:

keyboard_check_released(key);

ArgomentoTipo Descrizione
key The key to check the released state of.

Ritorna:

Esempio:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Il codice di cui sopra controllerà se il tasto "P" è stato rilasciato e, in caso affermativo, creerà un'istanza di "obj_Pause".
