cancellare la tastiera

Con questa funzione potete cancellare lo stato corrente della tastiera, il che significa essenzialmente che se il tasto è tenuto premuto, non sarà più riconosciuto fino a quando non sarà rilasciato di nuovo (che non attiverà nemmeno l'evento Keyboard Key Released in questa occasione) e premuto di nuovo. La funzione prenderà un valore di keycode come restituito dalla funzione ord() (solo lettere maiuscole da A-Z o numeri da 0-9), o una qualsiasi delle costanti vk_* elencate nella pagina principale di Keyboard Input.

Sintassi:

keyboard_clear(key);

ArgomentoTipo Descrizione
key The key to clear.

Ritorna:

Esempio:

keyboard_clear(vk_space);

Il codice qui sopra cancella lo stato della barra spaziatrice.

Indietro: Ingresso da tastiera
