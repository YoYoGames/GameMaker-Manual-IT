tastiera_ultimo carattere

Questa variabile memorizza un string dell'ultimo tasto premuto. Questa variabile non è di sola lettura e potete cambiarla, per esempio impostarla su "" (un string vuoto) se l'avete già gestita.

Sintassi:

keyboard_lastchar;

Ritorna:

Esempio:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = vk_nokey;
}

Il codice qui sopra controlla per vedere se la variabile lastkey è maggiore di -1, e se lo è aggiunge qualunque sia l'ultima chiave come string alla variabile "str".

Indietro: Ingresso da tastiera
Il prossimo: keyboard_string
