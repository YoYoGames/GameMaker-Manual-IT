tasto_tastiera

Con questa variabile potete ottenere il keycode del tasto attualmente premuto e restituirà 0 se nessun tasto è stato premuto al momento del controllo.

Sintassi:

keyboard_key;

Ritorna:

Virtual Key Constant

Esempio:

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

Il codice qui sopra usa il valore della variabile keyboard_key per impostare una variabile allo stesso valore di un array.

Indietro: Ingresso da tastiera
Il prossimo: keyboard_lastkey
