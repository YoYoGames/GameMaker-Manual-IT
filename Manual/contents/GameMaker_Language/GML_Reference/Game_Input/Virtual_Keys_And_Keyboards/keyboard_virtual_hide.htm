Nascondi tastiera virtuale

Questa funzione può essere utilizzata per nascondere la tastiera virtuale sul dispositivo che esegue il gioco. Chiamando questa funzione verrà generato un evento asincrono di sistema, in cui la mappa async_load DS verrà popolata con le seguenti coppie chiave/valore:

  • "event_type" - questo sarà "virtual keyboard status" quando viene attivato dalle funzioni della tastiera virtuale.
  • "screen_height" - l'altezza della tastiera virtuale (in pixel). Questo sarà 0 se la tastiera è invisibile.
  • "keyboard_status" - lo stato attuale della tastiera, restituito come una delle seguenti stringhe:
    • "nascondere"
    • "nascosto"
    • "mostrando"
    • "visibile

Sintassi

keyboard_virtual_hide();

Ritorna:

Esempio:

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

Il codice qui sopra nasconderà la tastiera virtuale del sistema operativo se la variabile data non è impostata su false.
