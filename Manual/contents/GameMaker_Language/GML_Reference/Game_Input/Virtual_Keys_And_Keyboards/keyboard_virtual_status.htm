stato_tastiera_virtuale

Questa funzione può essere usata per ottenere lo stato della tastiera virtuale sul dispositivo che esegue il gioco. La funzione restituirà true se la tastiera virtuale del sistema operativo è visibile/da mostrare o false se è nascosta/nascosta.

Sintassi

keyboard_virtual_status();

Ritorna:

Esempio:

var _status = keyboard_virtual_status();
if _status == false
{
    keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

Il codice di cui sopra mostrerà la tastiera virtuale del sistema operativo se lo stato attuale è false.
