altezza tastiera virtuale

Questa funzione restituisce l'altezza attuale in pixel della tastiera virtuale, in base alle dimensioni del display. Se la tastiera non è visibile, verrà restituito 0.

Sintassi

keyboard_virtual_height();

Ritorna:

Esempio:

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

Il codice qui sopra controllerà lo stato della tastiera virtuale del sistema operativo, e se è visibile imposterà una variabile all'altezza della tastiera.
