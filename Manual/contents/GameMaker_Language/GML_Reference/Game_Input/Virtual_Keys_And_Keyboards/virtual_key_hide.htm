Nascondi chiave virtuale

Una volta che avete creato la vostra chiave virtuale per i vostri dispositivi, potreste aver bisogno di fare il debug e testare che sia posizionata correttamente e che funzioni correttamente e per questo dovreste chiamare la funzione virtual_key_show(). Tuttavia, potreste aver bisogno di attivare e disattivare questa vista, quindi potete anche nascondere la chiave virtuale con questa funzione, utilizzando l'indice della chiave virtuale che volete nascondere (precedentemente creato e memorizzato utilizzando virtual_key_add()). Una volta che si è soddisfatti, si consiglia di creare e posizionare il proprio grafico nella sua area.

Sintassi

virtual_key_hide(index);

Argomento Tipo Descrizione
index ID della chiave virtuale The index of the virtual key to hide.

Ritorna:

N/A

Esempio:

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

Il codice di cui sopra controlla la variabile globale "global.Debug" e se risulta vero allora il tasto virtuale indicizzato nella variabile "global.Left" sarà disegnato sullo schermo, e se è falso allora il tasto sarà nascosto.
