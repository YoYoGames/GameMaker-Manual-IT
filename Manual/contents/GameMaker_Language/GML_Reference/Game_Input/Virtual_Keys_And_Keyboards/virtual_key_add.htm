chiave virtuale aggiunta

Questa funzione vi permette di mappare i "tocchi" di un'area dello schermo agli eventi della tastiera. Questo significa che una volta che avete assegnato un'area a un tasto virtuale, tutti i tocchi su quell'area attiveranno l'evento della tastiera corrispondente al tasto che avete mappato all'area. Puoi anche assegnare ogni tasto virtuale che definisci a una variabile, che può poi essere usata nelle ulteriori funzioni dei tasti virtuali per mostrarli, nasconderli e cancellarli. Questi tasti sono assegnati per ogni stanza e saranno automaticamente rimossi da GameMaker quando si cambia rooms.

La posizione effettiva della chiave virtuale è basata sulla posizione dello schermo piuttosto che su room e quindi i valori x/y sono assoluti sullo schermo. Questo significa che non dovete preoccuparvi dell'uso delle viste o delle coordinate relative room, e potete semplicemente definire le chiavi nell'evento Create di un object (dovete definire una chiave virtuale solo una volta per stanza), poi disegnare la vostra chiave sprites nell'evento Draw GUI, dimensionando il livello GUI per essere uguale allo schermo.

Sintassi

virtual_key_add(x, y, w, h, keycode);

Argomento Tipo Descrizione
x Real The x coordinate (left side) of the virtual key on the screen
y Real The y coordinate (top side) of the virtual key on the screen
w Real The width of the virtual key
h Real The height of the virtual key
keycode Costante chiave virtuale (vk_*) Which keyboard key event should be triggered by touching this area

Ritorna:

Virtual Key ID

Esempio:

global.Left = virtual_key_add(32, 32, 64, 64, vk_left);

Il codice qui sopra crea un tasto virtuale quadrato di 64x64 pixel, posizionato sullo schermo a (32, 32) che attiverà l'evento vk_left quando viene toccato e assegna l'indice di questo tasto virtuale a una variabile globale.
