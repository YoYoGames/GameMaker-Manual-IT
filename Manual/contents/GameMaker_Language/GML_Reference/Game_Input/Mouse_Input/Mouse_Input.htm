Ingresso del mouse

L'input del mouse è accettato su tutte le piattaforme (sui dispositivi mobili è accettato come un singolo tocco dello schermo - se hai bisogno di usare il multi-touch, dovresti usare le funzioni specifiche del dispositivo) e ha alcune costanti che sono usate per specificare i pulsanti che vengono premuti. Queste costanti sono mostrate nella seguente tabella:

Le seguenti funzioni esistono per i controlli standard dei pulsanti del mouse:

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

Si noti che c'è anche un insieme di funzioni di finestra relative all'uso del mouse sui bersagli del desktop:

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
