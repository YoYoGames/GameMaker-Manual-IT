mouse_clear

Questa funzione cancella lo stato corrente del dato pulsante del mouse. Ciò significa che i controlli sul fatto che sia tenuto premuto non restituiranno true finché il giocatore non rilascerà il pulsante e lo premerà di nuovo (ma lo stato di rilascio sarà ancora rilevato se la cancellazione viene fatta mentre il pulsante del mouse è tenuto premuto).

Potete fornire una delle costanti del pulsante del mouse elencate in questa pagina.

Sintassi:

mouse_clear(button);

Argomento Tipo Descrizione
button Costante del pulsante del mouse Which mouse button constant to check for.

Ritorna:

Boolean

Esempio:

mouse_clear(mb_any);

Il codice di cui sopra cancellerà lo stato down di tutti i pulsanti del mouse.

Indietro: Ingresso del mouse
Il prossimo: mouse_lastbutton
