ruota del mouse su

Questa funzione restituisce true se la rotella del mouse viene ruotata verso l'alto e false altrimenti.

Sintassi:

mouse_wheel_up();

Ritorna:

Esempio:

if mouse_wheel_up()
{
    y -= 10;
}

Questo sposta l'istanza corrente verso l'alto dello schermo se la rotella del mouse viene ruotata verso l'alto.

Indietro: Ingresso del mouse
Il prossimo: mouse_wheel_down
