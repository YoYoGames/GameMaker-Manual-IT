mouse_x

Questa variabile di sola lettura restituisce la posizione corrente dell'asse x del mouse all'interno di room.

Sintassi:

mouse_x;

Ritorna:

Esempio:

x = median(64, mouse_x, room_width - 64);

Il codice di cui sopra manterrà l'istanza nella posizione x del mouse finché è entro i limiti di 64 pixel da entrambi i lati di room.

Indietro: Ingresso del mouse
Il prossimo: mouse_y
