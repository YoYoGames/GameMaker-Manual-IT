tasto_mouse

Questa variabile di sola lettura restituisce il pulsante del mouse che è attualmente premuto (attualmente, come in questo passo) e può restituire qualsiasi costante speciale del mouse eccetto mb_any.

Sintassi:

mouse_button;

Ritorna:

Mouse Button Constant (except mb_any)

Esempio:

if mouse_button = mb_left
{
    x -= 1;
}

Questo sposta l'istanza corrente a sinistra se il tasto sinistro del mouse è abbassato.

Indietro: Ingresso del mouse
Il prossimo: mouse_check_button
