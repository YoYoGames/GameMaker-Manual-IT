mouse_check_button_pressed

Questa funzione restituisce true se il pulsante del mouse controllato è stato premuto o false se non lo è stato. Questa funzione sarà attivata solo una volta per ogni pulsante del mouse quando viene premuto per la prima volta e per attivarla di nuovo il pulsante dovrà essere stato rilasciato e premuto di nuovo. Si noti che sarà considerata attivata per la durata del passo, e per tutte le istanze che hanno eventi del mouse o che usano questa stessa funzione.

Si fornisce il pulsante del mouse per controllare da una delle seguenti costanti:

Sintassi:

mouse_check_button_pressed(numb);

Argomento Tipo Descrizione
numb Costante del pulsante del mouse Which mouse button constant to check for.

Ritorna:

Boolean

Esempio:

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

Il codice di cui sopra controllerà per vedere se il pulsante sinistro del mouse è stato premuto e, in caso affermativo, aggiungerà 50 al punteggio.

Indietro: Ingresso del mouse
Il prossimo: mouse_check_button_released
