ruota del mouse giù

Questa funzione restituisce true se la rotella del mouse viene ruotata verso il basso e false altrimenti.

Sintassi:

mouse_wheel_down();

Ritorna:

Esempio:

if mouse_wheel_down()
{
    y += 10;
}

Questo sposta l'istanza corrente verso il basso dello schermo se la rotella del mouse viene ruotata verso l'alto.

Indietro: Ingresso del mouse
Il prossimo: mouse_x
