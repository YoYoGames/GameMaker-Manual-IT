mouse_check_button_released

Questa funzione restituirà true se il pulsante del mouse controllato è stato rilasciato o false se non lo è stato. Questa funzione sarà attivata solo una volta per ogni pulsante del mouse quando viene rilasciato e per attivarla di nuovo il pulsante dovrà essere stato premuto e rilasciato di nuovo. Si fornisce il pulsante del mouse da controllare da una delle seguenti costanti:

Sintassi:

mouse_check_button_released(numb);

Argomento Tipo Descrizione
numb Costante del pulsante del mouse Which mouse button constant to check for.

Ritorna:

Boolean

Esempio:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

Il codice qui sopra controllerà per vedere se il pulsante destro del mouse è stato rilasciato e se lo è, imposterà la velocità dell'istanza a un decimo della distanza tra la posizione x/y corrente e la posizione x/y del mouse.

Indietro: Ingresso del mouse
Il prossimo: mouse_clear
