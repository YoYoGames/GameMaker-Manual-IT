pulsante di controllo del mouse

Questa funzione restituisce true se il pulsante del mouse controllato è tenuto premuto o false se non lo è. Si fornisce il pulsante del mouse da controllare da una delle seguenti costanti:

Sintassi:

mouse_check_button(numb);

Argomento Tipo Descrizione
numb Costante del pulsante del mouse Which mouse button constant to check for.

Ritorna:

Boolean

Esempio:

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

Il codice di cui sopra controllerà il pulsante sinistro del mouse e ogni passo che viene tenuto premuto creerà un'istanza di object indicizzata in "obj_Star".

Indietro: Ingresso del mouse
Il prossimo: mouse_check_button_pressed
