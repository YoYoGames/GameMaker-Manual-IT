pulsante del mouse

Questa variabile restituisce l'ultimo pulsante del mouse che è stato premuto e può restituire una qualsiasi delle costanti speciali del mouse tranne mb_any (potete anche impostare questa variabile su una delle costanti).

Sintassi:

mouse_lastbutton;

Ritorna:

Mouse Button Constant (except mb_any)

Esempio:

if mouse_lastbutton = mb_left
{
    x -= 1;
}

Questo sposta l'istanza corrente a sinistra se l'ultimo pulsante premuto era il pulsante sinistro del mouse.

Indietro: Ingresso del mouse
Il prossimo: mouse_wheel_up
