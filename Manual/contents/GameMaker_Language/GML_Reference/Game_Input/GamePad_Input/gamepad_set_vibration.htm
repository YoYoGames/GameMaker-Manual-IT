gamepad_set_vibrazione

Con questa funzione potete impostare la vibrazione dei motori del gamepad, con entrambi i motori che utilizzano un valore da 0 (nessuna vibrazione) a 1 (vibrazione completa). Notate che non c'è un limite di tempo su questa funzione, quindi dovrete usare una variabile o un allarme per switch spegnere la vibrazione (impostare i motori a 0) dopo che è passato un certo tempo, altrimenti il gamepad continuerà a vibrare indefinitamente.

NOTA: Questa funzione è attualmente disponibile solo per i moduli target standard di Windows, PS4 e Xbox One.

Sintassi:

gamepad_set_vibration(device, left_motor, right_motor);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Ritorna:

Esempio:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

Il codice di cui sopra verrebbe usato (per esempio) in un evento di collisione per far vibrare il gamepad inserito nello "slot" 0 per mezzo secondo, con l'allarme impostato che viene usato per switch farlo ripartire una volta trascorso questo tempo.
