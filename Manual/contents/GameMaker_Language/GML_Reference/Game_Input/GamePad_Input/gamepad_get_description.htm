gamepad_get_description

Questa funzione restituirà un string con la descrizione del gamepad dato, per esempio, un controller PS3 potrebbe restituire un string simile a "PLAYSTATION(R)3 Controller", o un controller XBox360 potrebbe restituire "Xbox 360 Controller (XInput STANDARD GAMEPAD)". Questo string dipende dall'hardware e il valore restituito dipenderà dal gamepad inserito nello "slot" del dispositivo che viene controllato.

Sintassi:

gamepad_get_description(device);

ArgomentoTipo Descrizione
device Which gamepad "slot" to get the name of.

Ritorna:

Esempio:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    var _gamepad = gamepads[i];
    if (gamepad_is_connected(_gamepad))
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(_gamepad));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "Gamepad not found");
    }
}

Il codice di cui sopra si trova su loop attraverso tutti gli slot per gamepad per controllare i dispositivi collegati e poi disegnare del testo sullo schermo in base al fatto che un gamepad sia collegato allo slot o meno.
