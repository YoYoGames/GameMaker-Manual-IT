gamepad_è_supportato

Con questa funzione potete scoprire se la piattaforma di destinazione supporta i game pad (restituisce true) o no (restituisce false).

Sintassi:

gamepad_is_supported();

Ritorna:

Esempio:

global.GP = gamepad_is_supported();

Il codice qui sopra controlla se un gamepad è supportato e memorizza il valore di ritorno in una variabile globale per controlli futuri.

Indietro: Ingresso Gamepad
Successivo: gamepad_is_connected
