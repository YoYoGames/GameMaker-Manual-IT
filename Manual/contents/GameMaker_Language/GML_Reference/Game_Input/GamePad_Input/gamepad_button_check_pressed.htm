gamepad_button_check_pressed

Questa funzione restituirà true o false a seconda che il dato pulsante del gamepad sia stato rilevato come premuto o meno. Si noti che questa funzione si attiverà solo una volta per il pulsante la prima volta che viene premuto. Per farla scattare di nuovo, il pulsante deve essere prima rilasciato e poi premuto di nuovo. Se avete bisogno di controllare una pressione continua del pulsante dovreste usare la funzione gamepad_button_check().

Se state controllando un pulsante analogico, allora il controllo prenderà in considerazione l'impostazione della soglia e restituirà true solo quando il valore grezzo dell'ingresso supera la soglia data (potete ottenere questo valore grezzo usando la funzione gamepad_button_value()).

Sintassi:

gamepad_button_check_pressed(device, button);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to check.
button Costante del pulsante del gamepad Which gamepad button constant to check for.

Ritorna:

Boolean

Esempio:

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

Il codice di cui sopra rileverà se il pulsante "start" del gamepad collegato al dispositivo "slot" 0 è stato premuto o meno e cambierà room se lo è stato.

Indietro: Ingresso Gamepad
Successivo: gamepad_button_check_released
