valore_pulsante_del_gamepad

Con questa funzione è possibile ottenere il valore corrente di un pulsante analogico, da 0 a 1, dove 0 è nessuna pressione e 1 è la pressione massima. Si fornisce l'indice dello slot del gamepad da controllare, insieme a una costante del pulsante (come quella elencata qui), oppure un valore intero compreso tra 0 e gamepad_button_count() - 1.

Si noti che questo restituisce il valore grezzo del pulsante e non tiene conto dell'impostazione della soglia (per ulteriori informazioni sulle soglie, vedere qui ).

Sintassi:

gamepad_button_value(device, button);

Argomento Tipo Descrizione
device Reale Which gamepad device "slot" to check.
button Pulsante del gamepad costante Which gamepad button constant to check for.

Restituzione:

Real

Esempio:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Il codice precedente utilizza il valore del trigger analogico del gamepad collegato al dispositivo "slot" 0 per impostare la velocità dell'istanza.
