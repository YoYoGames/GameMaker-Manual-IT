gamepad_hat_count

Questa funzione restituisce il numero totale di cappelli disponibili per il gamepad collegato al dato "slot" del dispositivo. I cappelli generalmente si riferiscono ai pulsanti su/giù/sinistra/destra, e si noti che sull'obiettivo Windows, i cappelli sono disponibili solo sui controller DirectInput (quindi, dallo slot 4 in su).

Sintassi:

gamepad_hat_count(device);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.

Ritorna:

Esempio:

h_num = gamepad_hat_count(4);

Il codice di cui sopra restituirà il numero di cappelli disponibili sul gamepad inserito nel dispositivo "slot" 4 e memorizzerà il valore nella variabile "h_num".
