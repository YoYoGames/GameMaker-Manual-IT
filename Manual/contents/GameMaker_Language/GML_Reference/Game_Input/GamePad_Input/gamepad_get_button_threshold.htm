gamepad_get_button_threshold

Questa funzione può essere utilizzata per rilevare l'impostazione corrente della soglia dei pulsanti analogici per un dato dispositivo. La soglia predefinita per tutti i pulsanti analogici è 0,5, con una gamma che va da 0 a 1. La soglia definisce a che punto il pulsante viene considerato come "premuto" per i giochi che richiedono di agire come un pulsante digitale.

Sintassi:

gamepad_get_button_threshold(device);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.

Ritorna:

Esempio:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Il codice qui sopra controlla la soglia del pulsante analogico del gamepad collegato al dispositivo "slot" 0 e se non è il valore predefinito di 0,5, viene impostato su quel valore.
