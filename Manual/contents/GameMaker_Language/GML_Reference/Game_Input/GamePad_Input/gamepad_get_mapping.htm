gamepad_get_mapping

Questa funzione può essere usata per recuperare la mappatura string per il gamepad. Questo string sarà o:

  • La mappa attuale string per il gamepad, per esempio:

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • Un string con "nessuna mappatura" se non c'è nessuna mappatura impostata (questo non è lo stesso di quando non è disponibile nessuna mappatura)
  • Un string con "device index out of range" se l'indice dello slot non è un indice di gamepad valido
  • Un string vuoto "" se non è disponibile alcuna mappatura per il gamepad collegato

Per maggiori informazioni sulla formattazione della mappa restituita string, si veda la funzione gamepad_test_mapping().

Sintassi:

gamepad_get_mapping(index);

ArgomentoTipo Descrizione
index Which gamepad index "slot" to get the mapping from.

Ritorna:

Esempio:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

Il codice qui sopra ottiene la mappa string per il dato slot del gamepad, e poi la invia alla console per debugging.
