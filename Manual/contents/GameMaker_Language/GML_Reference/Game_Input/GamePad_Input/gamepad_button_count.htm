numero di pulsanti del gamepad

Questa funzione restituisce il numero totale di pulsanti disponibili per il gamepad collegato al dato dispositivo "slot".

Sintassi:

gamepad_button_count(device);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.

Ritorna:

Esempio:

b_num = gamepad_button_count(0);

Il codice di cui sopra restituirà il numero di pulsanti disponibili sul gamepad inserito nel dispositivo "slot" 0 e memorizzerà il valore nella variabile "b_num".
