valore dell'asse del gamepad

Potete usare questa funzione per ottenere il valore dei diversi assi da un dato slot del gamepad. Si fornisce l'indice dello slot del gamepad da controllare, insieme a una costante dell'indice dell'asse (come elencato qui), o un valore intero tra 0 e gamepad_axis_count()-1. Il valore restituito sarà compreso tra -1 e 1 per ciascuno degli assi orizzontali e verticali disponibili.

Si noti che il valore restituito è normalizzato per prendere in considerazione il valore impostato per la zona morta dell'asse (utilizzando la funzione gamepad_set_axis_deadzone()), e quindi sarà sempre tra -1 e 0, e 0 e 1. Per esempio, impostando la zona morta a 0.2 significherà che spingendo lo stick verso destra inizierà a restituire un valore da 0 a 1 solo quando il valore grezzo dell'asse è superiore a 0.2 - così per esempio, quando il valore grezzo è 0.2, il valore restituito sarà 0, quando il valore grezzo è a 0.5, il valore restituito sarà 0.375, o se il valore grezzo è 0.9, allora il valore restituito sarà 0.875.

Sintassi:

gamepad_axis_value(device, axisIndex);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to check.
axisIndex Costante dell'asse del gamepad The axis index to check (see the constants list).

Ritorna:

Real

Esempio:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Il codice precedente imposta la velocità e la direzione dell'istanza in base ai valori restituiti dal controller analogico sinistro del gamepad inserito nel dispositivo "slot" 0.
