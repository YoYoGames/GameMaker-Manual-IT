impostazione del gamepad

Questa funzione può essere usata per impostare qualsiasi opzione del gamepad disponibile. Si fornisce lo "slot" del gamepad su cui impostare l'opzione, così come l'opzione string da impostare e il valore da utilizzare. L'opzione disponibile string dipenderà dalla piattaforma su cui viene eseguito il progetto, come elencato di seguito:

Opzione Piattaforma Descrizione Valore
"permetti_rotazione" tvOS Imposta se il Siri Remote sarà bloccato alla rotazione (cioè: può essere usato solo verticalmente) o no. Quando è sbloccato (un valore di true) la rotazione del controllo cambierà l'input, in modo tale che una rotazione di 90º a sinistra farà tornare il lato "destro" del touch pad "su", ecc... in modo tale che tutti gli input saranno relativi all'orientamento del dispositivo. Quando è bloccato (un valore di false) allora l'input non sarà cambiato rispetto alla posizione del controller e sarà sempre trattato come se il controller fosse in posizione verticale. Booleano
"dpad_absolute" tvOS Imposta se il touchpad di Siri Remote accetta un input assoluto o relativo. In modalità assoluta (un valore di true) il centro del touch pad è considerato la posizione (0,0) - cioè: dove lo stick analogico è a riposo o dove sarebbe il centro del dpad - e il movimento intorno a questa posizione genererà l'input. Quando è impostato su relativo (un valore di false allora la posizione (0,0) sarà considerata come quella in cui viene effettuato il tocco iniziale sul pad e tutti gli input saranno generati da quella posizione, non dal centro del pad. Booleano

Sintassi:

gamepad_set_option(device, option_name, value);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to set.
option_name Stringa The name of the option to set (a string, see the table above)
value Reale, booleano o stringa The value to set the option to (can be boolean, real or string - see the table above)

Ritorna:

N/A

Esempio:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Il codice di cui sopra loops attraverso tutti gli slot del gamepad e controlla il "Siri Remote" sulla piattaforma tvOS. Se ne viene rilevato uno, imposta il telecomando per consentire la rotazione degli input.
