gamepad_è_connesso

Questa funzione restituisce se un gamepad è collegato allo "slot" dato (restituisce true) o no (restituisce false). Normalmente si usa questa funzione insieme alla funzione gamepad_get_device_count() per ottenere il numero corretto di game pad e/o "slot" di gamepad disponibili. Si noti che potrebbe esserci un leggero ritardo tra il momento in cui l'utente connette il gamepad e quello in cui GameMaker lo rileva come connesso (questo è il caso soprattutto quando si ha a che fare con controller connessi via Bluetooth).

NOTA: Su alcune piattaforme - specialmente le console - questa funzione può restituire false immediatamente dopo che un gamepad è stato collegato/selezionato e può aver bisogno di essere controllata in un allarme alcuni fotogrammi (passi) dopo perché rilevi correttamente il gamepad.

Sintassi:

gamepad_is_connected(numb);

ArgomentoTipo Descrizione
device Which gamepad "slot" to check.

Ritorna:

Esempio:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

Il codice di cui sopra loops attraverso i game pad disponibili (o slot gamepad) e poi controlla ognuno di essi per un gamepad collegato. il valore restituito viene quindi utilizzato per impostare un array globale su true o false per l'utilizzo nei controlli futuri.

Indietro: Ingresso Gamepad
Successivo: gamepad_get_guid
