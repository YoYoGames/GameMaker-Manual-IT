gamepad_set_button_threshold

Questa funzione consente di impostare la soglia corrente dei pulsanti analogici per un determinato dispositivo. La soglia predefinita per tutti i pulsanti analogici è 0,5, mentre l'intervallo è compreso tra 0 e 1.

La soglia definisce il momento in cui il pulsante viene considerato "premuto" per i giochi che richiedono di agire come un pulsante digitale.

Questa funzione influisce sugli stati di controllo, pressione e rilascio dei pulsanti analogici, ma non sul valore restituito dalla funzione gamepad_button_value()che restituirà sempre il valore grezzo del pulsante.

Sintassi:

gamepad_set_button_threshold(device, threshold);

Argomento Tipo Descrizione
device Reale Which gamepad device "slot" to check.
threshold Reale The new threshold value (from 0 - 1, default 0.5).

Restituzione:

N/A

Esempio:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Il codice precedente controlla la soglia del pulsante analogico del gamepad collegato al dispositivo "slot" 0 e, se non è il valore predefinito di 0,5, viene impostato su tale valore.
