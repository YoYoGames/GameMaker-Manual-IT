gamepad_get_device_count

Questa funzione indica il numero di game pad collegati o il numero di "slot" disponibili per il collegamento dei game pad.

Il valore di ritorno effettivo dipenderà dalla piattaforma e dalla sua configurazione interna; pertanto, questa funzione deve essere utilizzata insieme alla funzione gamepad_is_connected() per essere certi del numero esatto di pad collegati in qualsiasi momento.

Sintassi:

gamepad_get_device_count();

Restituzione:

Real

Esempio:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

Il codice sopra riportato loops passa in rassegna i game pad (o slot gamepad) disponibili e controlla che ognuno di essi sia collegato. il valore restituito viene quindi utilizzato per impostare un array globale su true o false da utilizzare per i controlli futuri.
