gamepad_remove_mapping

Questa funzione può essere usata per rimuovere la mappatura corrente del dispositivo dall'indice dello slot del gamepad dato. Una volta chiamato, l'indice dello slot dovrà essere rimappato usando la funzione gamepad_test_mapping() se si vuole essere in grado di utilizzare le costanti del gamepad per rilevare correttamente l'input (l'input diretto può sempre essere recuperato utilizzando le funzioni gamepad_axis/button/hat_count() e gamepad_axis/button/hat_value() insieme).

Sintassi:

gamepad_remove_mapping(index);

ArgomentoTipo Descrizione
index Which gamepad index "slot" to remove the mapping from.

Ritorna:

Esempio:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

Il codice qui sopra rimuoverà la mappatura dal dato slot di indice del gamepad in base al valore di una variabile.

Indietro: Ingresso Gamepad
Il prossimo: gamepad_is_supported
