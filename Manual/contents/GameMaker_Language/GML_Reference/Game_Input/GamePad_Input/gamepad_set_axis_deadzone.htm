gamepad_set_axis_deadzone

Questa funzione può essere usata per impostare la "zona morta" dell'asse del joystick. Si specifica lo slot del dispositivo da impostare, e poi si imposta un valore da 0 a 1 e se la quantità di input è inferiore al valore dato, l'asse del joystick è considerato a 0. Si noti che questa è un'impostazione globale che influenzerà tutti gli assi di tutti i joystick collegati allo slot del dispositivo specificato.

L'uso di questa funzione non significa che il valore dell'asse partirà dal valore della zona morta, poiché la funzione gamepad_axis_value() restituirà sempre un valore normalizzato tra -1 e 0 o 0 e 1. Per esempio, impostando la zona morta a 0,2, significa che spingendo lo stick verso destra si inizierà a restituire un valore da 0 - 1 solo quando il valore grezzo dell'asse è superiore a 0,2 - quindi quando il valore grezzo è 0,2, il valore di ritorno sarà 0, quando il valore grezzo è a 0,5, il valore di ritorno sarà 0,375, o se il valore grezzo è 0,9, allora il valore di ritorno sarà 0,875.

Sintassi:

gamepad_set_axis_deadzone(device, deadzone);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Ritorna:

Esempio:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

Il codice qui sopra imposterà la zona morta dell'asse del joystick su uno qualsiasi dei 4 dispositivi collegati a 0,05.
