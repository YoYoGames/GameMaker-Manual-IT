numero di assi del gamepad

Questa funzione restituisce il numero di controlli "asse" sul dispositivo controllato. Questi controlli sono le "levette" analogiche di direzione sulla maggior parte dei controller.

Sintassi:

gamepad_axis_count(device);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.

Ritorna:

Esempio:

axis = gamepad_axis_count(0))

Il codice precedente memorizza il numero di assi disponibili per il gamepad collegato al dispositivo "slot" 0 nella variabile "axis".

Indietro: Ingresso Gamepad
Successivo: gamepad_axis_value
