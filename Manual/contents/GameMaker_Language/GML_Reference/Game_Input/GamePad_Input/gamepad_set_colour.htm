gamepad_set_colour

Questa funzione può essere usata per impostare il colore dei LED all'interno di un controller PlayStation. Si specifica lo slot del dispositivo da impostare, e poi si dà un colore, che può essere una qualsiasi delle costanti di colore o un valore di colore creato utilizzando le funzioni di colore specifiche o un valore HEX (come $FFFFFFFFF).

Sintassi:

gamepad_set_colour(device, colour);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to set.
colour The colour to use.

Ritorna:

Esempio:

if health < 10
{
    gamepad_set_colour(0, c_red);
}

Il codice di cui sopra imposterà i LED del controller PlayStation sul rosso se la variabile health scende sotto 10.
