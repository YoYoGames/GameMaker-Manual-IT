controllo dei pulsanti del gamepad

Questa funzione restituirà true o false a seconda che il dato pulsante del gamepad sia rilevato come tenuto premuto o meno. Se state controllando un pulsante analogico, allora il controllo prenderà in considerazione l'impostazione della soglia e restituirà true solo quando il valore grezzo dell'input è sopra la soglia data (potete ottenere questo valore grezzo usando la funzione gamepad_button_value()).

Sintassi:

gamepad_button_check(device, button);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to check.
button Costante del pulsante del gamepad Which gamepad button constant to check for.

Ritorna:

Boolean

Esempio:

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

Il codice di cui sopra rileverà se il pulsante inferiore della faccia superiore (la "X" su un controller ps3) viene tenuto premuto e, in tal caso, sparerà un'istanza "proiettile" e imposterà un allarme.

Indietro: Ingresso Gamepad
Successivo: gamepad_button_check_pressed
