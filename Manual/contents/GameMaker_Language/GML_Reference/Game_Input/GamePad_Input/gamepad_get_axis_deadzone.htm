gamepad_get_axis_deadzone

Questa funzione può essere usata per ottenere il valore della "zona morta" dell'asse del joystick. Si specifica lo slot del dispositivo da ottenere, e la funzione restituirà un valore tra 0 e 1, dove il valore riflette la soglia sotto la quale l'asse del joystick è considerato a 0.

Sintassi:

gamepad_get_axis_deadzone(device);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.

Ritorna:

Esempio:

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

Il codice qui sopra controlla la soglia dell'asse analogico del gamepad collegato al dispositivo "slot" memorizzato nella variabile globale e se non è il valore predefinito di 0,5, viene impostato su quel valore.
