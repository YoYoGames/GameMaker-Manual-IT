gamepad_button_check_released

Questa funzione restituirà true o false a seconda che il dato pulsante del gamepad sia stato rilevato come rilasciato o meno. Si noti che questa funzione si attiverà solo una volta per il pulsante nel momento in cui è stato rilasciato. Per farla scattare di nuovo, il pulsante deve essere prima premuto e poi rilasciato ancora una volta. Se state controllando un pulsante analogico, allora il controllo prenderà in considerazione l'impostazione della soglia e restituirà true solo quando il valore grezzo dell'input va sotto la soglia data (potete ottenere questo valore grezzo usando la funzione gamepad_button_value()).

Sintassi:

gamepad_button_check_released(device, button);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to check.
button Costante del pulsante del gamepad Which gamepad button constant to check for.

Ritorna:

Boolean

Esempio:

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

Il codice di cui sopra rileverà se il pulsante "select" del gamepad collegato al dispositivo "slot" 0 è stato premuto o meno e attiverà la variabile globale "Pause".
