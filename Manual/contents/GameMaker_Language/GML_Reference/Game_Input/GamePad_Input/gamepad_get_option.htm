gamepad_get_option

Questa funzione può essere usata per recuperare il valore dell'opzione data. Si fornisce lo "slot" del gamepad da controllare, così come l'opzione string e la funzione restituirà il valore associato. L'opzione disponibile string dipenderà dalla piattaforma su cui viene eseguito il progetto, come elencato di seguito:

Opzione Piattaforma Descrizione Restituisce
"permetti_rotazione" tvOS Restituisce se il telecomando Siri sarà bloccato alla rotazione (cioè: può essere usato solo verticalmente) o no. Quando è sbloccato, la rotazione del controllo cambierà l'input, in modo tale che una rotazione di 90º a sinistra farà tornare il lato "destro" del touch pad "su", ecc... in modo tale che tutti gli input saranno relativi all'orientamento del dispositivo Booleano
"dpad_absolute" tvOS Restituisce se il touchpad di Siri Remote accetta un input assoluto o relativo. In modalità assoluta, il centro del touch pad è considerato la posizione (0,0) - cioè: dove lo stick analogico è a riposo o dove sarebbe il centro del dpad - e il movimento intorno a questa posizione genererà input. Quando è impostato su relativo, allora la posizione (0,0) sarà considerata come il punto in cui viene effettuato il tocco iniziale sul pad e tutti gli input saranno generati da quella posizione, non dal centro del pad. Booleano

Sintassi:

gamepad_get_option(device, option_name);

ArgomentoTipo Descrizione
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Ritorna:

Boolean, Real, String, or N/A (depends on platform and option)

Esempio:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Il codice di cui sopra loops attraverso tutti gli slot del gamepad e controlla il "Siri Remote" sulla piattaforma tvOS. Se ne viene rilevato uno, imposta il telecomando per consentire la rotazione degli input.
