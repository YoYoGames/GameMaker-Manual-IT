valore del gamepad

Con questa funzione è possibile ottenere il valore corrente di un "cappello" del gamepad. Ogni valore del cappello è una maschera di bit per le diverse direzioni in cui:

  • Su = 1
  • Destra = 2
  • Giù = 4
  • Sinistra = 8

Notate che questi possono essere combinati (per esempio, fornendo un indice del cappello di 3 si controllerebbe su e a destra, oppure si può usare l'operatore | per combinare i vari valori) ma solo al massimo 2 bit possono essere premuti contemporaneamente. La funzione restituirà un valore reale tra 0 e 1, dove 0 è non premuto e 1 è completamente premuto (e ci possono essere valori intermedi a seconda che il gamepad supporti l'input analogico per i cappelli).

Sintassi:

gamepad_hat_value(device, hatindex);

Argomento Tipo Descrizione
device Real Which gamepad device "slot" to check.
hatindex Real Which gamepad hat (or hats) to check for.

Ritorna:

Real

Esempio:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

Il codice qui sopra memorizza lo stato del cappello "0" in una variabile locale, poi controlla quale sia il valore di ritorno e agisce di conseguenza.
