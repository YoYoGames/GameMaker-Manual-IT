gamepad_get_guid

Con questa funzione potete recuperare il GUID per il gamepad collegato all'indice dello slot dato. Voi fornite lo slot del gamepad da controllare (da 0 a 11), e la funzione restituirà una delle seguenti stringhe:

  • "none" se non è disponibile alcun GUID o se nessun gamepad è collegato
  • "indice del dispositivo fuori portata" se l'indice dello slot del gamepad è fuori portata
  • Il GUID string, qualcosa come "050000005e040000fd020000ffff3f00", se lo slot ha un gamepad assegnato e può ottenere il GUID per esso

Questa funzione verrebbe di solito usata insieme a gamepad_get_description() per generare parte dell'SDL string necessario per rimappare i controllori su Windows, Android e macOS.

Note specifiche per la piattaforma
Piattaforma Nota
Opera GX Sulla piattaforma di destinazione Opera GX, questa funzione restituirà sempre "nessuno" per i dispositivi non DirectInput

Sintassi:

gamepad_get_guid(index);

ArgomentoTipo Descrizione
index Which gamepad "slot" index to check

Ritorna:

Esempio:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Il codice di cui sopra ottiene il GUID e la descrizione del nome strings, poi li concatena e memorizza il finale string in una variabile globale per riferimento futuro.
