pulsante di controllo del mouse del dispositivo

Questa funzione restituisce true o false a seconda che il dato pulsante del mouse sia tenuto premuto sul dato dispositivo. Per il pulsante del mouse potete fornire una delle costanti elencate in questa pagina.

Questa funzione può essere usata per i touch screen, e l'argomento del dispositivo può essere da 0 a n dove n è il numero massimo di "dita" che possono toccare lo schermo in una volta (il numero massimo di tocchi che possono essere essere rilevato dipenderà molto dal dispositivo utilizzato e dal sistema operativo che esegue, ma la maggior parte dei dispositivi dovrebbe rilevare almeno fino a 4). Si noti che mb_right sarà rilevato solo se viene rilevato un doppio tocco (questo comportamento può essere disabilitato utilizzando la funzione device_mouse_dbclick_enable()).

Sintassi:

device_mouse_check_button(device, button);

ArgomentoTipo Descrizione
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

Ritorna:

Esempio:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

Il codice di cui sopra controlla se i due tocchi sono tenuti premuti allo stesso tempo e se lo sono allora passa a un altro room.
