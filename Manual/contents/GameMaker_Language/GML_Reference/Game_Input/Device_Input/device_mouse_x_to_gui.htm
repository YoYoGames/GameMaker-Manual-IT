dispositivo_mouse_x_a_gui

Questa funzione restituisce la posizione x del tocco sul dispositivo in relazione al livello della GUI. Poiché il livello della GUI può essere impostato per essere di dimensioni diverse indipendentemente dalla risoluzione dello schermo e dalla dimensione di room, a volte può essere complicato ottenere il punto esatto sul livello della GUI in cui un utente ha toccato lo schermo. Con questa funzione è possibile ottenere quella posizione in base alla dimensione effettiva della GUI, rendendo i pulsanti GUI e gli elementi HUD più facili da creare e con cui interagire. Se state eseguendo questa funzione su un modulo HTML5 o PC e Mac, allora questo valore viene aggiornato costantemente, finché il dispositivo (di solito un mouse) è collegato, tuttavia per i dispositivi mobili, questo sarà aggiornato solo mentre lo schermo viene toccato, e notate che il numero massimo di tocchi che può essere rilevato dipenderà molto dal dispositivo utilizzato e dal sistema operativo che esegue.

Sintassi:

device_mouse_x_to_gui(device);

ArgomentoTipo Descrizione
device The device (from 0 - n) that is being checked.

Ritorna:

Esempio:

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_x_to_gui(0) > 0 && device_mouse_x_to_gui(0) < 32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

Il codice qui sopra controlla per vedere se il mouse del dispositivo viene premuto e, in tal caso, esamina la posizione x del dispositivo sul livello GUI per vedere se rientra nei parametri. Se lo è, imposta la variabile "pressed" su true, altrimenti la imposta su false.
