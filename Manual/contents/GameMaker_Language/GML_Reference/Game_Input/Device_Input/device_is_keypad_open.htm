dispositivo_è_tastiera_aperta

Questo fa un controllo del dispositivo per una tastiera e se una è disponibile restituisce true altrimenti restituisce false. Si prega di notare che questa funzione è principalmente per l'uso con dispositivi Android

Per gli utenti con un Sony Xperia Play, c'è un'impostazione della mappa dei tasti all'interno di GameMaker, quindi puoi usare le costanti della tastiera vk_up, vk_down, vk_left, vk_right per i tasti del joypad e Triangle è ord("T"), Square è ord("S"), Circle è vk_alt + vk_backspace mentre il pulsante back è semplicemente vk_backspace, Cross è vk_space, Select è vk_return e Start è vk_rshift, il L Trigger è ord("L") e R Trigger è ord("R").

Sintassi:

device_is_keypad_open

Ritorna:

Esempio:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

Il codice qui sopra controlla la presenza di una tastiera e poi cambia una variabile globale a seconda del valore restituito.

Indietro: Ingresso dispositivo
Il prossimo: device_get_tilt_x
