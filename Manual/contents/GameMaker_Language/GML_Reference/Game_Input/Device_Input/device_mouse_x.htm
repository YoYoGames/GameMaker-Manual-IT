dispositivo_mouse_x

Questa funzione restituisce la posizione x di un tocco sul dispositivo. Se state eseguendo questo su un modulo HTML5 o PC e Mac, allora questo valore viene aggiornato costantemente, finché il dispositivo (di solito un mouse) è collegato, tuttavia per i dispositivi mobili, questo sarà aggiornato solo mentre lo schermo viene toccato e notate che il numero massimo di tocchi che possono essere rilevati dipenderà molto dal dispositivo utilizzato e dal sistema operativo che esegue.

Sintassi:

device_mouse_x(device);

ArgomentoTipo Descrizione
device The device (from 0 - n) that is being checked.

Ritorna:

Esempio:

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

Il codice di cui sopra controlla se il dispositivo viene premuto e, in tal caso, esegue il polling della posizione x del dispositivo per vedere se rientra nei parametri. Se lo è, imposta la variabile "pressed" su true, altrimenti la imposta su false.
