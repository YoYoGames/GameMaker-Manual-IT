dispositivo_mouse_check_button_pressed

Questa funzione restituisce true o false a seconda che il dispositivo specificato sia stato "toccato" (cliccato) o meno. L'argomento del dispositivo si riferisce al numero del tocco, che può essere da 0 a n e il numero massimo di tocchi che possono essere rilevati dipenderà molto dal dispositivo utilizzato e dal sistema operativo che esegue (la maggior parte dei dispositivi supporterà almeno 4 tocchi simultanei). Questa funzione viene attivata solo una volta dall'effettiva azione di pressione, e le costanti elencate in questa pagina possono essere utilizzate per controllare i pulsanti del mouse. Si noti che mb_right sarà rilevato solo se viene rilevato un doppio tocco e tenuto sul secondo tocco (questo comportamento può essere disabilitato utilizzando la funzione device_mouse_dbclick_enable()).

Sintassi:

device_mouse_check_button_pressed(device, button);

ArgomentoTipo Descrizione
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Ritorna:

Esempio:

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

Il codice qui sopra controlla se il dispositivo è stato premuto e imposta una variabile a true se è stato premuto.

Indietro: Ingresso dispositivo
Il prossimo: device_mouse_check_button_released
