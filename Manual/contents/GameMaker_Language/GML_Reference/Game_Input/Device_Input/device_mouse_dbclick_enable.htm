dispositivo_mouse_dbclick_enable

Questa funzione può essere usata per impostare il dispositivo per rilevare un doppio tocco di mb_left (pulsante sinistro del mouse) come un input mb_right (pulsante destro del mouse).

Per impostazione predefinita questo è impostato su true, il che significa che ogni volta che l'utente tocca lo schermo del dispositivo due volte velocemente e consecutivamente, il valore di ritorno è lo stesso come se il tasto destro del mouse fosse stato cliccato. Quando questo è attivo, il primo tocco sarà rilevato come mb_left, e il secondo come mb_right, quindi assicuratevi che ogni codice che usate ne tenga conto.

Sintassi:

device_mouse_dbclick_enable(bool);

Argomento Tipo Descrizione
bool Booleano Set double-click detection on (true) o off (false).

Ritorna:

Boolean

Esempio:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

Il codice di cui sopra controlla se il dispositivo che esegue il gioco è un PC Windows o un Mac e se è uno dei due, disabilita la funzione del doppio tocco.
