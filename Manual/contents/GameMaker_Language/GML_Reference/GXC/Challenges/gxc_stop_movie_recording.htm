gxc_stop_movie_recording

Questa funzione è usata per fermare una registrazione GXC che è stata precedentemente avviata con gxc_start_movie_recording(). Invia anche la registrazione all'URL specificato negli argomenti, tuttavia il comportamento è diverso a seconda del tipo di string specificato:

  • Se si specifica un nome di file, la registrazione sarà salvata con quel nome nella cartella Downloads
  • Se si specifica un URL, la registrazione sarà caricata su quell'URL

Il metodo URL dovrebbe essere usato quando la registrazione deve essere caricata su GXC, e il file dovrebbe essere salvato localmente solo per controllare se la registrazione ha avuto successo.

Sintassi:

gxc_stop_movie_recording(filename);

Argomento Tipo Descrizione
filename Stringa The file name or URL where the recording is to be saved or uploaded

Ritorna:

N/A

Esempio:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

Il codice di cui sopra utilizza la GXC API Library per caricare un punteggio alla sfida attualmente attiva, e quando viene ricevuta una risposta nel suo metodo di callback, recupera l'URL di caricamento del video e vi carica la registrazione.

Indietro: Opera GX e GXC
Dopo: gxc_pause_movie_recording
