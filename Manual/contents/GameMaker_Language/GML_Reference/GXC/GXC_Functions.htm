GX.games Functions

This section outlines the functions provided for the GX.games target platform. The functions listed on this page allow you to record your screen and upload it to GXC along with your challenge score, however do note that you require use of the GXC API Library to achieve this. Please refer to the GX.games section on the YoYo Games Helpdesk for instructions on implementing the GXC API Library and working with challenges.

The following functions are provided for uploading video recordings with challenges:

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
