Opera GX e GXC

Questa sezione delinea le funzioni fornite per la piattaforma target Opera GX. Le funzioni elencate in questa pagina ti permettono di registrare il tuo schermo e di caricarlo su GXC insieme al tuo punteggio della sfida, tuttavia tieni presente che è necessario utilizzare la libreria API GXC per ottenere questo risultato. Si prega di fare riferimento alla sezione Opera GX sul YoYo Games Helpdesk per le istruzioni sull'implementazione della GXC API Library e sul lavoro con le sfide.

Le seguenti funzioni sono previste per il caricamento di registrazioni video con sfide:

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
