file_bin_write_byte

Questa funzione scriverà un byte di dati nel file identificato dall'ID del file nella posizione di scrittura corrente. Voi fornite il valore dell'ID del file, come restituito dalla funzione file_bin_open() e il byte di dati da scrivere.

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Sintassi:

file_bin_write_byte(binfile, byte);

ArgomentoTipo Descrizione
binfile The ID of the file to write to.
byte The data to write.

Ritorna:

Esempio:

file_bin_write_byte(file, data);

Questo scriverebbe un byte nel file selezionato.

Indietro: File binari
