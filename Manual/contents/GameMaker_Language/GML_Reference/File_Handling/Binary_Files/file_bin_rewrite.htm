file_bin_rewrite

Questa funzione prende l'handle del nome del file come restituito dalla funzione file_bin_open() e poi riscrive il file, cancellandolo da tutti i dati precedenti per iniziare a scrivere dall'inizio del file.

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Sintassi:

file_bin_rewrite(binfile);

ArgomentoTipo Descrizione
binfile The ID of the file to rewrite.

Ritorna:

Esempio:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

Questo aprirebbe un file dalla stessa directory del gioco e assegnerebbe il suo indice alla variabile "file". Poi riscriverebbe il file (ripulendolo dai dati).
