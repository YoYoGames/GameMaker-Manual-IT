File binari

In rare situazioni potreste aver bisogno di leggere dati da file binari. Queste funzioni possono essere usate per questo scopo, anche se in generale, vi raccomandiamo di usare le funzioni buffer dedicate per creare, salvare e caricare dati binari piuttosto che queste funzioni.

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Le seguenti routine di basso livello esistono per questo:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
