file_bin_close

Chiude un file binario precedentemente aperto. Si fornisce il valore ID del file, come restituito dalla funzione file_bin_open().

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Sintassi:

file_bin_close(binfile);

ArgomentoTipo Descrizione
binfile The ID of the file to close.

Ritorna:

Esempio:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Questo aprirebbe un file dalla stessa directory del gioco, e assegnerebbe il suo indice alla variabile "file". Poi riscriverebbe il file (ripulendolo dai dati) e lo chiuderebbe di nuovo.

Indietro: File binari
