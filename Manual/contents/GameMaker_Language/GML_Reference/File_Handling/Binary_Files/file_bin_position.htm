posizione file_bin

Questa funzione restituisce la posizione corrente in byte, dove 0 è la prima posizione, del file con l'id del file dato. Si fornisce il valore dell'ID del file, come restituito dalla funzione file_bin_open().

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Sintassi:

file_bin_position(binfile);

ArgomentoTipo Descrizione
binfile The ID of the file to get the position in.

Ritorna:

Esempio:

pos = file_bin_position(file);

Questo memorizzerebbe la posizione attuale nella variabile "pos".

Indietro: File binari
Il prossimo: file_bin_seek
