cerca file_bin

Questa funzione sposta la posizione di lettura corrente all'interno del file nella posizione indicata. Si fornisce il valore ID del file, come restituito dalla funzione file_bin_open()e per aggiungere un file, spostate la posizione alla dimensione del file prima di scrivere.

NOTA: Queste funzioni non funzionano quando il modulo di destinazione è HTML5.

Sintassi:

file_bin_seek(binfile, pos);

ArgomentoTipo Descrizione
binfile The ID of the file to read from.
pos The position in the file to move to.

Ritorna:

Esempio:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Questo aprirebbe un file dalla stessa directory del gioco, e assegnerebbe il suo indice alla variabile "file", poi otterrebbe la dimensione del file e imposterebbe la prossima posizione di scrittura a quella dimensione.

Indietro: File binari
Il prossimo: file_bin_write_byte
