elimina file

Questa funzione eliminerà il file specificato dal sistema. Va notato che questa funzione cancellerà solo quei file che GameMaker è in grado di creare e analizzare: file ini, file di testo e file binari, o quei file fatti per memorizzare risorse create dal gioco come sprites o superfici. Tuttavia, non cancellerà nessun altro file. La funzione restituirà anche true se il file è stato rimosso con successo, o false in qualsiasi altra circostanza.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Si prega di consultare la sezione sui limiti del file system per maggiori informazioni.

Sintassi:

file_delete(fname);

ArgomentoTipo Descrizione
fname The name of the file to delete.

Ritorna:

Esempio:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Questo controllerebbe la presenza di un file e se esiste viene cancellato.
