cambiare nome al file

Questa funzione restituisce il nome del file indicato con il extension (incluso il punto) cambiato nel nuovo extension. Usando un string vuoto come nuovo extension si può rimuovere la parte extension tutta insieme.

Sintassi:

filename_change_ext(fname, newext);

ArgomentoTipo Descrizione
fname The file to use.
newext The new extension to use.

Ritorna:

Esempio:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

Il codice di cui sopra ottiene il nome del file (come string) con la parte extension rimossa.

Indietro: File System
Il prossimo: get_open_filename
