File System

Esiste in GameMaker una serie di funzioni che permettono di manipolare file di tutti i tipi in modi diversi:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Le seguenti funzioni possono essere usate per cambiare i nomi dei file:

NOTA Queste funzioni non lavorano sui file veri e propri! Si occupano solo del strings relativo ai diversi elementi di path e dovrebbero essere usati insieme alle altre funzioni di file.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Infine ci sono alcune funzioni speciali per ottenere nomi di file che possono essere usati per caricare e salvare file con filtri di tipo file. Queste funzioni apriranno una finestra di explorer, e quindi faranno perdere il focus al gioco. Questo significa che qualsiasi input da cui dipende il gioco potrebbe non essere registrato correttamente, e come tale raccomandiamo che (se usate queste funzioni in un evento della tastiera) usiate l'evento tasto su e non gli eventi premuto o giù.

NOTA Queste funzioni sono solo per Windows e macOS.

ATTENZIONE Queste funzioni potrebbero non funzionare come vi aspettate a causa della sandbox di GameMaker! Si prega di consultare la sezione sul File System per maggiori informazioni.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
