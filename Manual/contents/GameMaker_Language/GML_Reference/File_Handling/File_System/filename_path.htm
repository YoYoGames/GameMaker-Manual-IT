percorso_file

L'uso di questa funzione restituisce la parte path del file indicato path, compreso il backslash finale.

Sintassi:

filename_path(fname);

ArgomentoTipo Descrizione
fname The file name and path to use.

Ritorna:

Esempio:

path = filename_path(working_directory + "Test.ini");

Il codice qui sopra ottiene il path (come string) del file specificato.

Indietro: File System
Il prossimo: filename_dir
