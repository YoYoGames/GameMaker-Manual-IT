rinominare il file

Questa funzione rinominerà il file specificato con il nome specificato. La funzione restituisce true se il file è stato rinominato con successo, o false in qualsiasi altra circostanza.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

Sintassi:

file_rename(oldname, newname);

ArgomentoTipo Descrizione
oldname The name of the file to change.
newname The new name to give the file.

Ritorna:

Esempio:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

Questo controllerebbe la presenza di un file e se esiste viene rinominato.
