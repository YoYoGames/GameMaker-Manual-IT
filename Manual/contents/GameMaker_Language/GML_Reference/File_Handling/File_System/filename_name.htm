nome_file

Usando questa funzione restituisce la parte di nome del file indicato, con il extension ma senza il path.

Sintassi:

filename_name(fname);

ArgomentoTipo Descrizione
fname The file to use.

Ritorna:

String

Esempio:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

Il codice precedente ottiene il nome (come string) del primo file di tipo "doc" trovato nella directory specificata.

Indietro: File System
Il prossimo: filename_path
