salva il nome del file

Questa funzione apre un dialogo e chiede al giocatore un nome di file in cui salvare con il filtro dato. Il filtro ha la forma "name1|mask1|name2|mask2|... ", dove la maschera può contenere le diverse opzioni con un punto e virgola tra loro e si può anche usare un "*" per indicare qualsiasi string. Per esempio: "bitmaps|*.bmp;*.wmf", chiederebbe all'utente di selezionare il nome di un file bitmap dal sistema, ma solo quelli che hanno i due extensions specificati (bmp e png). questa funzione non salva il file stesso! Restituisce solo un string con il nome completo e path del file. Se l'utente preme "Cancel" viene restituito un string vuoto "".

Una cosa importante da notare quando si usa questa funzione è che vi concede certi permessi per quel file, per la durata del vostro gioco. Quindi, una volta ottenuto il file path dall'utente, potrete accedervi ancora e ancora senza doverlo chiedere, e il permesso si applica sia alla lettura che alla scrittura.

NOTA: Questa funzione è valida solo sugli obiettivi Windows e macOS.

Sintassi:

get_save_filename(filter, fname);

ArgomentoTipo Descrizione
filter The file type filter and/or name.
fname The suggested filename to use.

Ritorna:

Esempio:

var file;
file = get_save_filename("screenshot|*.png", "");
if (file != "")
{
    screen_save(file);
}

Il codice di cui sopra chiederà all'utente di selezionare un file di nome immagine per il salvataggio, poi controllerà il nome del file restituito e path, e se non è un vuoto string prenderà uno screenshot e lo salverà usando il nome dato.
