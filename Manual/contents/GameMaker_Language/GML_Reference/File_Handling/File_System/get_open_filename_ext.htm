get_open_filename_ext

Questa funzione apre un dialogo e chiede al giocatore un nome di file da aprire con il filtro dato. Il filtro ha la forma "name1|mask1|name2|mask2|... ", dove la maschera può contenere le diverse opzioni con un punto e virgola tra loro e si può anche usare un "*" per indicare qualsiasi string. Per esempio: "bitmaps|*.bmp;*.wmf", chiederebbe all'utente di selezionare il nome di un file bitmap dal sistema, ma solo quelli che hanno i due extensions specificati (bmp e png). questa funzione non apre il file stesso! Restituisce solo un string con il nome completo e path del file. Se l'utente preme "Cancel" viene restituito un string vuoto "". Potete anche aggiungere una directory da cui iniziare la ricerca, e una didascalia da mostrare in cima alla finestra al posto di quella predefinita.

Una cosa importante da notare quando si usa questa funzione è che vi concede certi permessi per quel file, per la durata del vostro gioco. Quindi, una volta ottenuto il file path dall'utente, potrete accedervi ancora e ancora senza doverlo chiedere. Tuttavia, poiché questa funzione è per la lettura di un file, i permessi concessi sono solo per la lettura e il salvataggio non sarà permesso dal sistema operativo. Se avete bisogno di leggere e scrivere sul file, potete ottenere i permessi usando la funzione get_save_filename().

NOTA: Questa funzione è valida solo sugli obiettivi Windows e macOS.

Sintassi:

get_open_filename_ext(filter, fname, directory, caption);

ArgomentoTipo Descrizione
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

Ritorna:

Esempio:

var file;
file = get_open_filename_ext("text file|*.txt", "", working_directory, "Open a saved level");
if file != ""
{
    file_text_open_read(file);
}

Il codice di cui sopra chiederà all'utente di selezionare un file di testo da aprire da working_directory e poi controllerà il nome del file restituito e path, e se esiste aprirà il file per la lettura.
