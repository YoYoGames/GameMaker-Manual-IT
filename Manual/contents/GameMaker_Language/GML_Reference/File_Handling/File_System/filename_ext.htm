nome_file_ext

Questa funzione restituisce la parte extension del nome del file indicato, compreso il punto iniziale.

Sintassi:

filename_ext(fname);

ArgomentoTipo Descrizione
fname The file to use.

Ritorna:

Esempio:

ext = filename_ext(file_find_first("*.*", 0));

Il codice qui sopra ottiene il extension (come string) del file specificato.

Indietro: File System
