file_copy

Questa funzione copierà il file specificato, dandogli il nuovo nome che avete scelto.

Se state copiando un file su Android dall'interno della vostra area Included Files (che sarà all'interno del vostro APK) esso verrà copiato prima in memoria, quindi, siate consapevoli che copiare file molto grandi dall'APK può causare l'esaurimento della memoria dei dispositivi. Sul target HTML5, usiamo l'API HTML5 Local Storage per il salvataggio lato client, e le funzioni non supportano le operazioni sui file binari, quindi si presume che tu stia copiando un file di testo.

La funzione fallisce se il file newname esiste già.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Si prega di consultare la sezione sui limiti del file system per maggiori informazioni.

Sintassi:

file_copy(fname, newname);

ArgomentoTipo Descrizione
fname The name of the file to copy.
newname The name of the new, copied file.

Ritorna:

N/A

Esempio:

if file_exists("level1.txt")
{
    file_copy("level1.txt", "level2.txt");
}

Questo controllerebbe un file e se esiste viene copiato.
