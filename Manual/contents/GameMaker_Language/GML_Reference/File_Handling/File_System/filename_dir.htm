nome_file_dir

Questa funzione restituisce la parte di directory del nome del file indicato, che normalmente è lo stesso di path tranne che per la barra rovesciata finale.

Sintassi:

filename_dir(fname);

ArgomentoTipo Descrizione
fname The file to use.

Ritorna:

Esempio:

dir = filename_dir("Test.ini");

Il codice precedente ottiene la directory (come string) del file specificato.

Indietro: File System
Il prossimo: filename_drive
