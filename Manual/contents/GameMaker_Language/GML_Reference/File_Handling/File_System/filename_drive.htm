nome file_drive

Questa funzione restituisce le informazioni sull'unità del nome del file.

Sintassi:

filename_drive(fname);

ArgomentoTipo Descrizione
fname The file to use.

Ritorna:

Esempio:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

Il codice qui sopra ottiene le informazioni sull'unità (come string) del file specificato.

Indietro: File System
Il prossimo: filename_ext
