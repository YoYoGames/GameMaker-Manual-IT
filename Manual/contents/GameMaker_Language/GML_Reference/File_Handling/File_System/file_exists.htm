file_exists

Questa funzione restituisce true se il file specificato esiste e false se non esiste. Si noti che la funzione può essere utilizzata solo per controllare i file locali, ma non i file memorizzati su un sito remoto server.

Sintassi:

file_exists(fname);

ArgomentoTipo Descrizione
fname The name of the file to check for.

Ritorna:

Esempio:

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

Questo controllerebbe la presenza di un file e se esiste viene aperto per la lettura.
