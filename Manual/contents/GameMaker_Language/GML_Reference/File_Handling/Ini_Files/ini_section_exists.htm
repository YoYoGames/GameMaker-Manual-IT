ini_section_exists

Questa funzione controlla se una sezione esiste nell'ini attualmente aperto e restituisce true se esiste o false altrimenti. Questo non è un controllo necessario per prevenire errori poiché, quando una sezione non esiste, la lettura da una sezione inesistente restituirà semplicemente un valore predefinito, tuttavia può essere utile per vedere se un file ini ha salvato dati specifici.

Sintassi:

ini_section_exists(section);

ArgomentoTipo Descrizione
section The section in the open .ini file to check for.

Ritorna:

Esempio:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

Questo imposterà la variabile global.savegame a "1" se "save1" NON viene trovato. Con un for loop, questo potrebbe essere usato come un modo di base per capire quanti giochi di salvataggio ci sono per un gioco (cioè: creare un sistema per più giochi di salvataggio simultanei per un gioco).
