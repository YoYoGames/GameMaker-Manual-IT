ini_close

Questa funzione deve essere richiamata al termine della lettura o della scrittura di qualsiasi file ini aperto. Se non si usa questa funzione dopo aver usato una qualsiasi delle funzioni di scrittura dei file ini, non verrà scritto nulla sul disco, poiché le informazioni del file vengono tenute in memoria fino a quando non viene richiamata questa funzione, che forza la scrittura. Se si tenta di aprire un file ini senza averne precedentemente chiuso un altro (o lo stesso), si otterrà un errore.

La funzione restituisce anche un string con il file ini codificato al suo interno. Questo string può essere salvato in un server e/o usato di nuovo con la funzione ini_open_from_string() per ricreare l'ini.

Sintassi:

ini_close();

Restituzione:

String

Esempio:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Questo aprirà "savedata.ini" e leggerà un valore dalla sezione "save1" con la chiave "score", quindi chiuderà il file ini. Se non c'è alcun valore in "save1" chiamato "score" o se non è presente alcun file "savedata.ini", verrà restituito 0 e verrà creato un nuovo file ini.
