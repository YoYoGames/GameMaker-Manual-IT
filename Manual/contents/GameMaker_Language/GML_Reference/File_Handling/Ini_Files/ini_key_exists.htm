ini_key_exists

Questa funzione controlla se una chiave esiste nell'ini attualmente aperto e restituisce true se esiste o false altrimenti. Questo non è un controllo necessario per prevenire errori poiché, quando una chiave non esiste, la lettura da una chiave inesistente restituirà semplicemente un valore predefinito. Può essere utile per vedere se un file ini ha salvato dati specifici e alcune altre cose, comunque.

Sintassi:

ini_key_exists(section, key);

ArgomentoTipo Descrizione
section The section in the open .ini file to check a key in.
key The key to check for.

Ritorna:

Esempio:

ini_open("savedata.ini");
if (!ini_key_exists("save1", "name"))
{
    global.name = "Player1";
}
ini_close();

Questo imposterà la variabile global.name su "Player1" se non esiste una chiave come "name".
