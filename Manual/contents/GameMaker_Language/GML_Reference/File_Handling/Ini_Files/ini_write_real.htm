ini_write_real

Potete usare questa funzione per scrivere un valore (numerico) in un file dati ini. I file ini sono divisi in sezioni e poi ogni sezione è successivamente divisa in coppie chiave-valore. Quindi un tipico file ini sarebbe qualcosa di simile a questo:

Sintassi:

ini_write_real(section, key, value);

ArgomentoTipo Descrizione
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

Ritorna:

Esempio:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Questo imposterà score a 1000, poi aprirà "savedata.ini" e scriverà questo valore in "save1" > "Score". Poi, imposterà score2 al valore reale sotto "save1" > "score" in esso, 1000, e chiuderà il .ini di nuovo. Se non avesse trovato questo valore, score2 avrebbe finito per essere impostato come 0.
