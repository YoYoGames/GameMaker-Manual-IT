cancellazione ini_key

Con questa funzione potete rimuovere la chiave selezionata (e il suo valore corrispondente) da un file ini.

Sintassi:

ini_key_delete(section, key);

ArgomentoTipo Descrizione
section The section to delete a key from.
key The key to delete.

Ritorna:

Esempio:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

Questo esempio aprirà "savedata.ini" e scriverà un valore in "save1" > "Score". Poi cancellerà la chiave "Score" e chiuderà il file .ini.

Indietro: Ini Files
Il prossimo: ini_section_delete
