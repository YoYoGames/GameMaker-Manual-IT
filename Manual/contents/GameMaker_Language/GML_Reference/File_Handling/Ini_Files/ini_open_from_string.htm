ini_open_from_string

Questa funzione creerà un file ini temporaneo da un string e lo aprirà per la lettura/scrittura. Il string deve essere formattato correttamente come un file ini (cioè: con sezioni, interruzioni di riga chiavi e valori) altrimenti il file ini non sarà creato correttamente. Notate che questo file ini è temporaneo e verrà rimosso dalla memoria nel momento in cui verrà chiuso, perdendo qualsiasi informazione che vi era memorizzata, tuttavia la ini_close() restituisce un string del file ini completo che può essere salvato su un server o su disco.

Sintassi:

ini_open_from_string(string);

ArgomentoTipo Descrizione
string The string containing all the ini information.

Ritorna:

N/A

Esempio:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Il codice qui sopra imposta un ini temporaneo per contenere le informazioni da string "str", poi legge dall'ini prima di chiuderlo di nuovo.
