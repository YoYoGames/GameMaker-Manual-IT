ini_read_real

Potete usare questa funzione per leggere un numero da un file di dati ini. I file ini sono divisi in sezioni e poi ogni sezione è successivamente divisa in coppie chiave-valore. Quindi un tipico file ini sarebbe qualcosa di simile a questo:

Sintassi:

ini_read_real(section, key, default);

ArgomentoTipo Descrizione
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Ritorna:

Esempio:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Questo aprirà "savedata.ini" e imposterà il punteggio al valore sotto "save1" > "score" in esso, poi chiuderà nuovamente il .ini. Se non c'è nessun valore sotto "save1" > "score", o non c'è nessun file "savedata.ini" presente, il punteggio sarà impostato a 0.
