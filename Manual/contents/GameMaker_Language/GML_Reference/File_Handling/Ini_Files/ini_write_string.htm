ini_write_string

Potete usare questa funzione per scrivere un string (testo) in un file dati ini. I file ini sono divisi in sezioni e poi ogni sezione è successivamente divisa in coppie chiave-valore. Quindi un tipico file ini sarebbe qualcosa di simile a questo:

Vale la pena notare che l'uso di virgolette e caratteri di escape quando si scrive ini strings può causare problemi quando in seguito si rileggono i dati perché potrebbero essere troncati prematuramente. Per esempio, scrivere questo sarà un problema:

ini_write_string("test2", "section", "hello \"Fritz\"");

Sintassi:

ini_write_string(section, key, value);

Argomento Tipo Descrizione
section Stringa The section of the .ini to write to.
key Stringa The key within the relevant section of the .ini to write to.
value Stringa The string to write to the relevant destination.

Ritorna:

N/A

Esempio:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Il codice qui sopra apre un file ini per leggere e scrivere, poi scrive il string memorizzato nella variabile globale "Name" nella sezione "Save" con la chiave "Player" prima di chiudere di nuovo il file.
