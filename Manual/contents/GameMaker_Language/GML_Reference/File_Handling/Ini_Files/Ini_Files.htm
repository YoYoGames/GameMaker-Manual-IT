File Ini

I file INI sono file piccoli e leggeri, compatibili con la maggior parte delle piattaforme. Sono ideali per memorizzare piccole informazioni, come le preferenze dell'interfaccia, i punteggi più alti in locale, i dati dei livelli, ecc... e sono quindi un modo semplice e veloce per iniziare a salvare le informazioni dai giochi. Se siete alla ricerca di modi più avanzati per salvare le informazioni nei file, potete dare un'occhiata alle altre funzioni dei file, come i file di testo, i file binari e i buffer.

IMPORTANTE! Tutti i progetti GameMaker hanno un file chiamato "options.ini", quindi non è possibile assegnare a un file un nome simile (si otterrà un errore uid quando si cercherà di compilare il progetto di gioco).

Si noti che per i giochi con localizzazione è molto importante che le lettere accentate possano essere lette da file esterni. Ciò significa che è necessario creare prima il file ini in formato UTF8 e poi aggiungerlo a GameMaker come file incluso, in modo che venga esportato durante l'esecuzione del gioco e utilizzato al posto del file ini in formato ANSI predefinito che viene creato dalle funzioni ini quando non ne è stato fornito alcuno. In questo modo, è possibile leggerlo e scriverlo correttamente, mantenendo tutti gli accenti e le lettere non romane.

Per i file ini esistono le seguenti funzioni:

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
