directory di lavoro

Working_directory può effettivamente restituire due valori diversi a seconda dell'uso che se ne fa.

Se state scrivendo un file su disco, working_directory punta all'area di memoria locale che è riservata al vostro gioco sul dispositivo di destinazione (in Windows questo è "%LOCALAPPDATA%/gamedir/" dove "gamedir" è la directory con il nome del vostro gioco).

Tuttavia, quando si legge da disco, working_directory può essere sia la memoria locale (game_save_id) o l'area dove sono memorizzati i file inclusi.

NOTA La variabile working_directory restituirà l'indirizzo path compreso il backslash finale.

Per esempio, se avete un file di default *.txt incluso nel vostro gioco e lo leggete in memoria usando working_directory, allora quel file sarà preso dall'area dove sono memorizzati i file inclusi. Se poi scegliete di scrivere quelle informazioni in un file di testo usando working_directory, questo scriverà il file nella memoria locale della piattaforma di destinazione, da dove sarà letto la volta successiva, a meno che non ordiniate a GameMaker di cancellare il file, nel qual caso working_directory punterà ancora una volta all'area dove sono memorizzati i file inclusi.

Vale la pena notare che in generale non c'è bisogno di usare questo e semplicemente specificare il nome del file stesso (più qualsiasi informazione aggiuntiva path ), che è sufficiente. Per esempio, questo:

file_text_open_read(working_directory + "my_file.txt");

sarebbe meglio fare così:

file_text_open_read("my_file.txt");

ATTENZIONE Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Si prega di vedere la sezione sul File System per maggiori informazioni.

Sintassi:

working_directory

Ritorna:

String

Esempio:

ini_open(working_directory + "temp_ini.ini");

Questo aprirà un file ini dalla directory di lavoro del gioco (creandolo se non esiste già). Questo potrebbe essere l'archivio locale o l'area in cui si trovano i file inclusi, a seconda che il file da cercare esista in uno dei due posti.
