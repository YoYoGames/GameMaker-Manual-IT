esiste una directory

Questa funzione restituirà true se la directory indicata esiste o false se non esiste. Il nome specificato deve includere l'intero path, non un relativo path e per impostazione predefinita non è possibile accedere a nessuna directory dall'esterno del bundle del gioco, poiché tutti i giochi sono sandboxed (vedere la sezione sul File System per maggiori informazioni).

Sintassi:

directory_exists(dname)

ArgomentoTipo Descrizione
dname The name of the directory to look for.

Ritorna:

Esempio:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Questo controllerà per vedere se la directory specificata esiste, poi, se esiste, ci andrà e restituirà il primo file "read only" doc trovato.

Indietro: Cartelle di file
Il prossimo: directory_create
