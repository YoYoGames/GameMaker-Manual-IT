creare directory

Questa funzione crea una directory con il nome dato nell'area di salvataggio.

NOTA: Questa funzione non funzionerà su HTML5 poiché non è possibile creare o rimuovere le directory nella memoria locale del browser.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

Sintassi:

directory_create(dname)

ArgomentoTipo Descrizione
dname The name of the directory to create.

Ritorna:

Esempio:

if !directory_exists("Games")
{
    directory_create("Games");
}

Questo controllerà se la directory specificata esiste nella cartella locale dei dati e, se non esiste, la crea per voi.

Indietro: Cartelle di file
Il prossimo: directory_destroy
