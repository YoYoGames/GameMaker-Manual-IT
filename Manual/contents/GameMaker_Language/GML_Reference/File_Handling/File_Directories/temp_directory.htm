directory_tempore

Questo può essere usato per restituire la directory temporanea creata per il vostro gioco ogni volta che viene eseguito (la root non contiene il "\" finale). Questa directory conterrà dei file e vi si potrà accedere mentre il gioco è in esecuzione, ma verrà rimossa (insieme con tutti i file che contiene) quando il gioco viene chiuso.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

Sintassi:

temp_directory

Ritorna:

Esempio:

ini_open(temp_directory + "\temp_ini.ini");

Questo aprirà un file ini nella directory temporanea del gioco (creandolo se non esiste già).
