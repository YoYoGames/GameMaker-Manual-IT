directory del programma

Questo restituirà la directory in cui è memorizzato l'eseguibile del gioco. Tuttavia questo potrebbe non essere sempre utile, in particolare perché alcuni dispositivi eseguono l'exe da un file *.zip, quindi questo restituirebbe lo stesso indipendentemente da dove il gioco è effettivamente in esecuzione.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

NOTA: se disabilitate la sandbox, non sarete comunque in grado di scrivere in questa directory per prevenire qualsiasi possibile problema di cancellazione dei file di gioco necessari.

Sintassi:

program_directory

Ritorna:

Esempio:

dir = program_directory;

Questo memorizzerà la directory in cui l'eseguibile è memorizzato in una variabile.

Indietro: Cartelle di file
Il prossimo: directory_exists
