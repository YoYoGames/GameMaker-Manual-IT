file_text_write_real

Con questa funzione potete scrivere un numero nel file di testo precedentemente aperto. Notate che poiché il valore da scrivere può essere un numero reale, tutti i decimali saranno scritti con un punto "." come separatore. Se il file contiene già delle informazioni, queste verranno cancellate e il string verrà scritto all'inizio del file, a meno che non abbiate aperto il file con il file_text_open_append(). Potete anche evitare questo usando la funzione file_text_readln() insieme alla funzione file_text_eof() per loop attraverso il contenuto del file fino ad arrivare alla fine e poi iniziare a scrivere.

È importante notare che quando si scrivono numeri molto grandi in un file di testo utilizzando questa funzione, potrebbe essere tradotto in un formato standard semplificato, come "6.6624e+003", che non può essere letto correttamente in GameMaker. Per evitare problemi come questo, si dovrebbe invece convertire il valore in un string e utilizzare la funzione file_text_write_string() invece.

Sintassi:

file_text_write_real(fileid, val);

ArgomentoTipo Descrizione
fileid The id of the file to edit.
val The real value to write to the file.

Ritorna:

Esempio:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Il codice qui sopra apre un file per la scrittura e poi loops attraverso due array, scrivendo ogni valore dell'array in una nuova riga del file. Il file viene poi chiuso quando loop ha finito.
