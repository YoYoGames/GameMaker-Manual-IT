file_text_readln

Con questa funzione potete saltare il resto della riga corrente di un dato file di testo aperto e passare all'inizio della successiva. La funzione restituisce anche l'intera riga come string, rendendola un modo semplice per leggere "pezzi" completi di dati da analizzare in seguito.

Sintassi:

file_text_readln(fileid);

ArgomentoTipo Descrizione
fileid The id of the file to read from.

Ritorna:

Esempio:

var file = file_text_open_read(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    scr[i] = file_text_read_real(file);
    file_text_readln(file);
    scr_name[i] = file_text_read_string(file);
    file_text_readln(file);
}
file_text_close(file);

Il codice qui sopra apre un file per la lettura e poi loops attraverso le linee del file leggendo alternativamente un valore di numero reale e un string in due diversi array per uso futuro. Il file viene poi chiuso quando il loop ha finito.
