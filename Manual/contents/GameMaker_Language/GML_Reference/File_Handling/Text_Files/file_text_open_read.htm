file_text_open_read

Questa funzione apre il file di testo con il nome del file indicato per la sola lettura, restituendo l'id univoco del file che dovrebbe essere memorizzato in una variabile in quanto verrà utilizzato per tutte le ulteriori azioni che riguardano quel file. Se il file non esiste, la funzione restituisce il valore -1.

NOTA: potete avere solo un massimo di 32 file aperti alla volta. Dovresti anche chiudere sempre i file quando hai finito, perché questo scrive le informazioni e libera la memoria associata al file.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

Sintassi:

file_text_open_read(fname);

ArgomentoTipo Descrizione
fname The name of the file to read from.

Ritorna:

Text File ID or -1

Esempio:

file = file_text_open_read(working_directory + "level.txt");

Questo aprirebbe "level.txt" dalla stessa directory del gioco e memorizzerebbe l'id del file nella variabile "file".
