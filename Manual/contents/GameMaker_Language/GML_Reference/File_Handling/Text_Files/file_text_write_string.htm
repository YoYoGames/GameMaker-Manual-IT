file_text_write_string

Con questa funzione potete scrivere un string in un file di testo precedentemente aperto. Se il file contiene già informazioni, queste saranno cancellate e il string sarà scritto all'inizio del file, a meno che non abbiate aperto il file con il file_text_open_append(). Potete anche evitare questo usando la funzione file_text_readln() insieme alla funzione file_text_eof() per loop attraverso il contenuto del file fino ad arrivare alla fine e poi iniziare a scrivere.

Sintassi:

file_text_write_string(fileid, str);

ArgomentoTipo Descrizione
fileid The id of the file to edit.
str The string to write to the file.

Ritorna:

Esempio:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Il codice qui sopra apre un file per la scrittura e poi loops attraverso due array, scrivendo ogni valore dell'array in una nuova riga del file. Il file viene poi chiuso quando loop ha finito.
