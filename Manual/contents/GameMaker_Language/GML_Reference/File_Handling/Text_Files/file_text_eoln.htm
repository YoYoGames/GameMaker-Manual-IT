file_text_eoln

Con questa funzione potete far sì che GameMaker controlli il file attualmente aperto per vedere se la riga che viene letta è finita. La funzione restituisce true se la fine della linea è stata raggiunta e false altrimenti.

Sintassi:

file_text_eoln(fileid);

ArgomentoTipo Descrizione
fileid The id of the file to check.

Ritorna:

Esempio:

var file = file_text_open_read(working_directory + "Game_Data.txt");
var num = 0; while (!file_text_eoln(file))
{
    score_array[num] = file_text_read_real(file);
    num++;
}
file_text_close(file);

Il codice qui sopra apre un file per la lettura, poi legge i valori da una singola linea in un array.
