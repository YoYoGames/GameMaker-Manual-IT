file_text_eof

Questa funzione restituisce true quando è stata raggiunta la fine di un dato file di testo aperto o false in caso contrario.

Sintassi:

file_text_eof(fileid);

ArgomentoTipo Descrizione
fileid The id of the file to check.

Ritorna:

Esempio:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

Il codice qui sopra apre un file per la scrittura poi loops attraverso le linee di testo già scritte nel file fino a raggiungere la fine, memorizzando ogni linea nell'array "str".

Indietro: File di testo
Il prossimo: file_text_close
