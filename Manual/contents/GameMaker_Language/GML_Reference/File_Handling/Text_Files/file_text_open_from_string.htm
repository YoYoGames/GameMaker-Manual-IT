file_text_open_from_string

Questa funzione creerà un file di testo da un string e lo aprirà per la lettura, restituendo il file "handle" che dovrebbe essere usato in tutte le ulteriori chiamate alla funzione file per leggere da questo file. Notate che questo file è temporaneo e di sola lettura, e come tale verrà rimosso dalla memoria nel momento in cui verrà chiuso.

NOTA: potete avere solo un massimo di 32 file aperti alla volta. Dovresti anche chiudere sempre i file quando hai finito, perché questo scrive le informazioni e libera la memoria associata al file.

ATTENZIONE! Questa funzione potrebbe non funzionare come vi aspettate a causa della sandbox di GameMaker! Vedere la sezione sul File System per maggiori informazioni.

Sintassi:

file_text_open_from_string(string);

ArgomentoTipo Descrizione
string The string to create the file from.

Ritorna:

File ID or -1

Esempio:

file = file_text_open_from_string(reset_str);

Il codice qui sopra prende il string memorizzato nella variabile "reset_str" e lo usa per creare un file di testo di sola lettura. L'"handle" per questo file è poi memorizzato nella variabile "file" per tutte le altre funzioni di file da usare.
