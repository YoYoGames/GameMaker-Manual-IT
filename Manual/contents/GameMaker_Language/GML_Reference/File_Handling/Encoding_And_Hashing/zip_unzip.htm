zip_unzip

Questa funzione aprirà un file zip memorizzato ed estrarrà il suo contenuto nella directory data. Si noti che se non si fornisce un path completo alla directory ZIP, allora verrà utilizzata la radice dell 'unità corrente, e se si desidera posizionarla in un path relativo alla directory di lavoro del bundle del gioco, allora si dovrebbe utilizzare la variabile working_directory come parte di path ( paths relativo usando "." o ".." non funzionerà come previsto quindi dovrebbe essere evitato). Si noti inoltre che lo zip deve essere o parte del bundle del gioco (cioè: un file incluso) o essere stato scaricato nell'area di memorizzazione usando http_get_file().

La funzione restituisce un valore che indica il numero di file estratti, oppure restituisce 0 o meno se l'estrazione è fallita.

Sintassi:

zip_unzip(zip_file, target_directory)

ArgomentoTipo Descrizione
zip_file The zip file to open
target_directory The target directory to extract the files to

Ritorna:

Esempio:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

Il codice qui sopra aprirà il file zip memorizzato nella directory "downloads" ed estrarrà il suo contenuto nella directory "extracted" (creando tale directory se non esiste già) e poi controllerà che l'estrazione sia stata corretta, mostrando un messaggio di debug se dovesse fallire.
