md5_file

In crittografia, MD5 (Message-Digest algorithm 5) è una funzione di hash crittografica ampiamente utilizzata con un valore di hash a 128 bit ed è stata impiegata in un'ampia varietà di applicazioni di sicurezza. È anche comunemente usato per controllare l'integrità dei file e strings. Questa funzione prenderà il file dato e genererà un MD5 unico per quel file che può essere memorizzato per un uso successivo.

NOTA: MD5 non è completamente sicuro e può essere rotto. Vedi questa pagina per maggiori informazioni.

Sintassi:

md5_file(filename)

ArgomentoTipo Descrizione
filename The file to generate the MD5 hash for.

Ritorna:

Esempio:

hash = md5_file(working_directory + "game_data.ini")

Il codice sopra genererà un hash MD5 per il file specificato e memorizzerà il valore restituito nella variabile "hash".

Indietro: Codifica dei file
Il prossimo: sha1_string_utf8
