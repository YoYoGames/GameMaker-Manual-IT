sha1_file

In crittografia, SHA-1 è una funzione di hash crittografica progettata dalla United States National Security Agency ed è impiegata in diverse applicazioni molto diffuse come Git, dove è usata per verificare le modifiche ai file, e i protocolli TLS e SSL, PGP, SSH, S/MIME e IPsec. Questa funzione prenderà un file di input e restituirà un message digest a 160 bit in formato ASCII unico per quel file da utilizzare per la verifica dell'integrità in qualsiasi momento successivo.

NOTA: SHA-1 non è completamente sicuro e può essere rotto. Vedi questa pagina per maggiori informazioni.

Sintassi:

sha1_file(filename)

Argomento Tipo Descrizione
filename Stringa The file to generate the sha1 hash for.

Ritorna:

String

Esempio:

hash = sha1_file(working_directory + "game_data.ini")

Il codice sopra genererà un hash sha1 per il file specificato e memorizzerà il valore restituito nella variabile "hash".
