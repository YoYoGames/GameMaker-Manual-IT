base64_encode

Questa funzione convertirà un string in un formato base64 codificato string. Questo è uno schema di codifica comunemente usato che è spesso utilizzato per qualsiasi media che deve essere memorizzato o trasferito su Internet come testo, e rende l'output illeggibile all l'occhio umano.

Sintassi:

base64_encode(string)

ArgomentoTipo Descrizione
string The string to encode.

Ritorna:

Esempio:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

Il codice di cui sopra convertirà il string memorizzato in "game_data" in un string codificato in base64 che viene poi memorizzato in un file di testo esterno.
