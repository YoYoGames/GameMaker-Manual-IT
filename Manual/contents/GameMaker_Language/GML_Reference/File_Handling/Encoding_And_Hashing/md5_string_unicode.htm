md5_stringa_unicode

In crittografia, MD5 (Message-Digest algorithm 5) è una funzione di hash crittografica ampiamente utilizzata con un valore di hash a 128 bit ed è stata impiegata in un'ampia varietà di applicazioni di sicurezza. È anche comunemente usato per controllare l'integrità dei file e strings. Questa funzione prenderà un input unicode string (che è 16bits per ogni char) e restituirà l'hash MD5 esadecimale di 32 caratteri che è unico per quel string. In questo modo è possibile generare una chiave sicura che può essere memorizzata e utilizzata per controllare l'integrità delle informazioni inviate a (o ricevute da) un server esterno (per esempio).

NOTA Ci sono due formati per la codifica MD5, UTF-8 e unicode. Entrambi sono forniti per facilitare la comunicazione con diverse impostazioni server, ma il più comune da usare è l'unicode.

NOTA: MD5 non è completamente sicuro e può essere rotto. Vedi questa pagina per maggiori informazioni.

Sintassi:

md5_string_unicode(string)

ArgomentoTipo Descrizione
string The string to hash.

Ritorna:

Esempio:

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

Il codice di cui sopra codifica in base64 un string e poi genera un hash MD5. Infine, sia l'hash che il string codificato sono inviati a un server.
