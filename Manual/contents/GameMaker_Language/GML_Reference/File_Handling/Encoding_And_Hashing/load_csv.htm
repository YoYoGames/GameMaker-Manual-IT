caricare_csv

Questa funzione carica un file in formato CSV e lo converte in una griglia DS, restituendo il valore ID univoco della griglia creata.

Il file CSV deve seguire le specifiche CSV. Se la sintassi del file CSV non è corretta, si otterranno risultati imprevisti al momento del caricamento del file.

Sintassi:

load_csv(filename)

Argomento Tipo Descrizione
filename Stringa The name of the file to open (as a string)

Restituzione:

DS Grid ID

Esempio:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

Il codice precedente aprirà il file CSV dato e memorizzerà la griglia DS restituita nella variabile "file_grid". Questa griglia viene poi analizzata in un paio di pagine di for e il contenuto viene disegnato sullo schermo.

Indietro: Codifica dei file
Il prossimo: json_encode
