creare un riferimento debole

Con questa funzione potete creare un riferimento debole a un struct che può essere usato per controllare se il struct è ancora "vivo" (referenziato) o meno nel gioco. Voi fornite il riferimento a struct che volete tracciare, e la funzione restituirà un altro struct che è un riferimento debole a quel struct. Notate che potete controllare se un riferimento è "forte" o "debole" usando la funzione instanceof(), poiché un riferimento forte restituirà o "struct" o il nome della funzione costruttrice che ha creato il struct, o "weakref" se è un riferimento debole. Notate anche che il riferimento debole struct avrà una variabile "ref" a cui si può accedere per ottenere il riferimento forte al struct in questione, a meno che non sia stato garbage collected, nel qual caso restituirà undefined.

Sintassi:

weak_ref_create(struct_to_track);

Argomento Tipo Descrizione
struct_to_track Struttura The struct that you want to create a weak reference for

Ritorna:

Struct Weak Reference

Esempio:

inventory_ref = weak_ref_create(inventory);

Il codice qui sopra crea un riferimento debole a un struct e lo memorizza in una variabile di istanza per un uso successivo.
