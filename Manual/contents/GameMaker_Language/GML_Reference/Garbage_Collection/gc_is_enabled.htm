è abilitato

Con questa funzione potete controllare se il garbage collector è abilitato o meno. La funzione restituisce true se è abilitato o false altrimenti.

Sintassi:

gc_is_enabled();

Ritorna:

Esempio:

if (!gc_is_enabled())
{
    gc_enable(true);
}

Il codice qui sopra controlla se il garbage collector è abilitato e se non lo è lo abilita.
