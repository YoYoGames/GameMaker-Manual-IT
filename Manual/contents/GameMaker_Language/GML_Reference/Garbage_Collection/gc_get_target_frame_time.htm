gc_get_target_frame_time

Con questa funzione potete recuperare il valore attuale del target frame per il garbage collector. Il valore restituito è in microsecondi (dove 1.000.000 di microsecondi equivale a un secondo) e il tempo predefinito del frame di destinazione è 100 microsecondi. Se volete cambiare questo valore allora dovreste usare la funzione gc_target_frame_time().

Sintassi:

gc_get_target_frame_time(time);

Ritorna:

Esempio:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

Il codice di cui sopra controlla l'obiettivo di tempo del frame corrente per il garbage collector e se non è 50 microsecondi allora è impostato su questo valore.
