gc_collect

Con questa funzione potete attivare il garbage collector, forzandolo ad eseguire alla fine del frame corrente (passo). Vale la pena notare che il garbage collector non ha bisogno di essere attivo per funzionare. Chiamando questa funzione dopo aver disabilitato il garbage collector (usando la funzione gc_enable()) abiliterà il garbage collector per un frame in cui tutti i objects che sono stati segnalati per la raccolta saranno rimossi dalla memoria prima che il garbage collector sia nuovamente disabilitato.

Sintassi:

gc_collect();

Ritorna:

Esempio:

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

Il codice di cui sopra controlla una variabile globale e un tasto premuto e se questi sono true allora viene attivata la garbage collection per la fine del frame (passo).

Indietro: Raccolta della spazzatura
Dopo: gc_target_frame_time
