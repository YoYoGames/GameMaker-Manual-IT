weak_ref_alive

Con questa funzione potete controllare il riferimento debole a un struct per vedere se è ancora "vivo" o no. Voi fornite il riferimento debole da controllare (come restituito dalla funzione weak_ref_create()), e la funzione restituirà true se il struct è ancora referenziato da qualche parte o false se non lo è ed è stato smaltito. Notate che se fornite un valore che non è un riferimento debole, la funzione restituirà undefined.

Sintassi:

weak_ref_alive(weak_ref);

Argomento Tipo Descrizione
weak_ref Riferimento debole della struttura The weak reference to the struct you want to check.

Ritorna:

Boolean (or undefined)

Esempio:

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

Il codice qui sopra controlla un riferimento debole a un struct e se è ancora vivo la variabile che lo referenzia viene impostata a -1, de-referenziando il struct e permettendogli di essere smaltito.
