weak_ref_any_alive

Con questa funzione potete controllare i riferimenti deboli a varie strutture per vedere se sono ancora "vivi" o no. Fornite un array di riferimenti deboli da controllare (ogni riferimento debole dovrebbe essere stato creato utilizzando la funzione weak_ref_create()), e la funzione restituirà true se qualsiasi delle strutture sono ancora referenziate da qualche parte o false se non lo sono e sono state smaltite. Notate che se fornite un array in cui uno qualsiasi dei valori non è un riferimento debole, la funzione restituirà undefined.

Notate che la funzione prende anche due argomenti opzionali, dove il primo permette di fornire un indice iniziale nella matrice da cui controllare, così come un argomento per impostare il numero di posizioni (lunghezza) da quell'indice da controllare. Se specificato, verranno controllati solo gli indici della matrice all'interno di questi parametri.

Sintassi:

weak_ref_any_alive(array, [index], [length]);

Argomento Tipo Descrizione
array Matrice di riferimenti deboli della struttura Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL L'indice nell'array da cui iniziare il controllo
[length] Real OPTIONAL Il numero di posizioni, a partire dall'indice dato, da controllare per

Ritorna:

Boolean (or undefined)

Esempio:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

Il codice precedente controlla un array di riferimenti deboli e se qualcuno è ancora vivo, allora un'istanza viene distrutta.
