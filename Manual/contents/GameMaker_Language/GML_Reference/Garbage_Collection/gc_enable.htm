gc_enable

Con questa funzione potete abilitare o disabilitare il garbage collector. Chiamando la funzione con true come argomento lo si abilita e usando false lo si disabilita (non consigliato). È abilitato di default.

Sintassi:

gc_enable(enable);

ArgomentoTipo Descrizione
enable enable (true) o disabilitare (false) il garbage collector.

Ritorna:

Esempio:

if (global.debug == true)
{
    gc_enable(false);
}

Il codice qui sopra disabilita la garbage collection se la variabile globale data è true.
