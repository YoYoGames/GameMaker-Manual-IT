disegnare_percorso

Con questa funzione potete far disegnare a GameMaker un path sullo schermo. Il path sarà disegnato come una semplice linea, e può essere relativo all'istanza chiamante o nella posizione assoluta in cui è stato creato nell'editor path o attraverso il codice. Questa funzione è estremamente utile quando debugging dinamico paths (per esempio, quelli creati per le istanze con la mp_grid_path() funzione).

NOTA: Se volete disegnare una forma usando un shader, dovete sapere che la maggior parte di shaders si aspetta i seguenti input: vertex, texture, Colour. Tuttavia, quando si usa questa funzione, vengono passati solo i dati del vertice e del colore, e quindi shader potrebbe non disegnare nulla (o disegnare qualcosa ma non correttamente). Se avete bisogno di disegnare forme in questo modo, allora shader dovrebbe essere personalizzato con questo in mente.

Sintassi:

draw_path(path, x, y, absolute);

ArgomentoTipo Descrizione
path The path to draw
x The x coordinate of where the path is drawn
y The y coordinate of where the path is drawn
absolute Whether the path is drawn at the absolute position (true) o la posizione relativa (false)

Ritorna:

Esempio:

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
{
    draw_path(path, x, y, false);
}

Il codice precedente userà la funzione mp_grid_path per generare un path e memorizzarlo nella variabile "path". Se il path viene creato con successo, viene poi disegnato sullo schermo in una posizione relativa all'istanza che esegue il codice.
