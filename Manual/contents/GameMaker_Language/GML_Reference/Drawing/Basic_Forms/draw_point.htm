disegnare un punto

Con questa funzione potete disegnare un singolo pixel in qualsiasi punto dello schermo, utilizzando il colore di disegno e l'alfa attualmente impostati.

Sintassi:

draw_point(x, y);

ArgomentoTipo Descrizione
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

Ritorna:

Esempio:

draw_set_colour(c_yellow);
draw_point(100,100);

Questo disegnerà un pixel giallo nella posizione (100,100).
