pausa_video

Questa funzione mette in pausa il file video attualmente caricato. Potete riprenderlo chiamando video_resume() in qualsiasi momento dopo questa funzione.

Sintassi:

video_pause();

Ritorna:

N/A

Esempio:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Il codice qui sopra ottiene lo stato del video e poi controlla se il giocatore ha premuto Space. In questo caso mette in pausa il video se è in riproduzione, e lo riprende se è in pausa.

Indietro: Riproduzione video
Dopo: video_resume
