video_chiusura

Questa funzione chiude il file video attualmente caricato. Assicuratevi che sia chiamata solo dopo una chiamata a video_open(), altrimenti non farà nulla.

Sintassi:

video_close();

Ritorna:

N/A

Esempio:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

Il codice di cui sopra chiude il video quando viene premuto il tasto Escape.
