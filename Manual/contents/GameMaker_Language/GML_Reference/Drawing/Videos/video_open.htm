video_open

Questa funzione apre un file video utilizzando l'indirizzo path specificato. L'indirizzo path può riferirsi a un file nei file inclusi del vostro progetto, un file presente localmente sul dispositivo dell'utente, o un URL per caricare un file da internet (che potrebbe non funzionare su tutte le piattaforme).

Poiché si può caricare solo un video alla volta, questa funzione carica il video specificato nell'unico "slot video" disponibile, che viene poi manipolato tramite le altre funzioni Video.

Ci sono callback asincroni associati alla riproduzione video. Vedere Callbacks asincroni per maggiori informazioni.

Sintassi:

video_open(path);

Argomento Tipo Descrizione
path Stringa Path to the video file

Ritorna:

N/A

Esempio:

my_video = video_open("splash.mp4");

Il codice qui sopra carica splash.mp4 dai file inclusi nel gioco.

Indietro: Riproduzione video
Dopo: video_close
