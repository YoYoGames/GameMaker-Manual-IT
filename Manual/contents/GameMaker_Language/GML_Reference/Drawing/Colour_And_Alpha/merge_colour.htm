fondere_colore

Con questa funzione potete prendere due colori e poi fonderli insieme per fare un nuovo colore. La quantità di ciascuno dei colori componenti può essere definita cambiando l'argomento "amount", dove un valore di 0 restituirà il primo colore (col1), un valore di 1 restituirà il secondo colore (col2) e un valore intermedio restituirà il mix corrispondente. Per esempio, un valore di 0,5 mescolerà equamente i due colori. L'immagine seguente illustra come funziona, fondendo i colori rosso e blu insieme:

Sintassi:

merge_colour(col1, col2, amount);

ArgomentoTipo Descrizione
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

Ritorna:

Colour Constant

Esempio:

col = merge_colour(c_lime, c_orange, 0.5);

Il codice qui sopra usa la funzione per creare un colore fondendo lime e arancione 50/50 e poi memorizza il suo valore nella variabile "col" per un uso successivo.
