disegnare_chiaro

Questa funzione può essere usata per cancellare l'intero schermo (senza alpha blend) al colore dato, ed è solo per l'uso nell'evento draw di un'istanza (non si vedrà se usato in qualsiasi altro evento). Può anche essere utile per cancellare le superfici quando sono appena create.

Sintassi:

draw_clear(col);

ArgomentoTipo Descrizione
col The colour with which the screen will be cleared

Ritorna:

Esempio:

draw_clear(c_blue);

Questo cancellerà lo schermo con il colore blu.

Indietro: Colore e alfa
Il prossimo: draw_clear_alpha
