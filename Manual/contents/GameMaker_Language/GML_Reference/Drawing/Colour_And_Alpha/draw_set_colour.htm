draw_set_colour

Con questa funzione è possibile impostare il colore di base del disegno per il gioco.

Questo influisce sul disegno di fonts, forme, primitive e 3D, ma non su sprites (disegnato manualmente o da un'istanza). Se gli elementi grafici interessati sono disegnati con i propri valori di colore, questo valore verrà ignorato.

Sintassi:

draw_set_colour(col);

Argomento Tipo Descrizione
col Colore The colour to set for drawing.

Restituzione:

N/A

Esempio:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Il codice precedente disegna un testo nella posizione specificata con un effetto ombra creato dalla modifica dell'alfa e del colore del disegno.
