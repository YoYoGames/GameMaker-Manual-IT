draw_enable_swf_aa

Con questa funzione è possibile abilitare o disabilitare l'anti-aliasing (AA) per il formato SWF vettoriale sprites. L'AA semplicemente smussa i bordi delle immagini vettoriali per dare loro un aspetto più piacevole. La quantità di AA utilizzata dipenderà dal valore impostato utilizzando la funzione draw_set_swf_aa_level(). Per impostazione predefinita questo è disabilitato.

Sintassi:

draw_enable_swf_aa(enable);

ArgomentoTipo Descrizione
enable Enable (true) o disabilitare (false) AA per tutti gli SWF sprites.

Ritorna:

Esempio:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Il codice qui sopra controllerà il valore AA per il formato SWF sprites, e se è 0 abilita AA e imposta il valore a 0.5.

Indietro: Sprites e piastrelle
