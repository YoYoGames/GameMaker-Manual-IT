disegnare_tilemap

Questa funzione può essere usata per disegnare una data tilemap ovunque in room. Si dà l'ID dell'elemento tilemap (che si ottiene quando si crea un elemento tilemap usando layer_tilemap_create() o quando usi la funzione layer_tilemap_get_id()), poi dai la posizione x/y per il disegno (in room). Nota che questo disegnerà semplicemente la tilemap nel punto specificato, usando la profondità di livello dell'istanza che sta chiamando la funzione. Non sposta la tilemap - né la cambia in alcun modo - e non importa se la tilemap è contrassegnata come visibile o meno.

Sintassi:

draw_tilemap(tilemap_element_id, x, y);

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tilemap element to draw
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

Il codice di cui sopra ottiene l'ID del livello e poi lo usa per ottenere un ID specifico di tilemap che a sua volta è usato per disegnare la tilemap nella posizione del mouse.
