disegnare piastrelle

Con questa funzione potete disegnare qualsiasi dato tile da un tile set - competere con rotazioni, capovolgimenti e specchi - impostando la cornice (se animata) e la posizione all'interno del room. Il valore ID di tile set è l'indice tile set come impostato nel IDE e i dati tile sono il "blob" di dati che contiene tutte le informazioni sull'indice tile e le trasformazioni che sono state applicate. È possibile utilizzare la funzione tilemap_get() per ottenere i dati di tile che possono poi essere manipolati usando le funzioni Tile, oppure potete costruirli voi stessi usando le seguenti costanti di tile insieme al valore dell'indice di tile da creare:

Costante Descrizione
tile_rotate Usato per impostare/ottenere il bit di rotazione di un blob di dati tile.
tile_mirror Usato per impostare/ottenere il bit speculare di un blob di dati tile.
tile_flip Usato per impostare/ottenere il flip bit di un blob di dati tile.
tile_index_mask Una costante speciale che serve per "e" con il blob di dati tile per estrarre l'indice tile.


Dovreste fare un "or" bitwise di queste costanti con il valore dell'indice tile per ottenere il tiledata finale:

var tiledata = 8 | tile_rotate | tile_flip;

Il codice di cui sopra crea semplicemente una variabile locale per memorizzare i tiledata per il tile con l'indice di 8 sul tile set, e viene ruotato e capovolto. Questo può poi essere usato per disegnare il tile a quell'indice dal tile set che specifichi. questo non è collegato a nessun livello di tilemap e il tile sarà disegnato alla profondità del livello dell'istanza che fa il disegno, nella posizione x/y che specifichi nel room. Nota che puoi anche impostare il frame da disegnare. Se il tile indicizzato fa parte di un set di animazione, allora potete impostare qui il numero di frame (da 0 al numero di frame - 1) e il tile indicizzato sarà semplicemente l'indicatore di quale tile set animato state usando mentre sarà il dato frame ad essere disegnato. Se il tile non fa parte di un set animato allora il frame dovrebbe essere impostato a 0.

Sintassi:

draw_tile(tileset, tiledata, frame, x, y)

Argomento Tipo Descrizione
tileset Tile Set Asset The ID of the tile set to use
tiledata Dati sulle piastrelle The tile data to use
frame Real The frame number for the animated tile to use (default 0 for non-animated tiles)
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Ritorna:

Real

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

Il codice qui sopra otterrà i dati su un tile dalla tilemap data, poi imposterà il flag "flip" e userà questi dati per disegnare il tile capovolto nella posizione dell'istanza.
