draw_get_swf_aa_level

Questa funzione può essere usata per ottenere il livello di anti-aliasing (AA) per il formato SWF vettoriale sprites. Il valore di ritorno sarà compreso tra 0 e 1 e mostra quanto "lisci" saranno disegnati i bordi di questi sprites. È possibile impostare il livello AA utilizzando la funzione draw_set_swf_aa_level().

Sintassi:

draw_get_swf_aa_level();

Ritorna:

Esempio:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Il codice qui sopra controllerà il valore AA per il formato SWF sprites, e se è 0 abilita AA e imposta il valore a 0.5.
