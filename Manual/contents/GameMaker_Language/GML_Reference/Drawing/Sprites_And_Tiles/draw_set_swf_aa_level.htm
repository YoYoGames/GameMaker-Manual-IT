draw_set_swf_aa_level

Questa funzione può essere usata per impostare il livello di anti-aliasing (AA) per il formato SWF vettoriale sprites. Questo può essere un valore reale da 0 a 1 e "smusserà" i bordi di questi sprites. Si noti che per vedere questo effetto, è necessario aver prima abilitato AA utilizzando la funzione draw_enable_swf_aa().

Sintassi:

draw_set_swf_aa_level(AA);

ArgomentoTipo Descrizione
AA The anti-aliasing value to use from 0 to 1.

Ritorna:

Esempio:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Il codice qui sopra controllerà il valore AA per il formato SWF sprites, e se è 0 abilita AA e imposta il valore a 0.5.

Indietro: Sprites e piastrelle
