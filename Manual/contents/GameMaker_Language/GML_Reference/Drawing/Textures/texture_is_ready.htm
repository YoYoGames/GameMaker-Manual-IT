è pronto

Questa funzione può essere usata per controllare se una specifica pagina texture è stata spacchettata ed è pronta all'uso, o se un gruppo di pagine texture è stato spacchettato ed è pronto all'uso. Si fornisce l'ID univoco della pagina della texture (come restituito dalle funzioni texturegroup_* ) o l'ID del gruppo texture string (come definito nel Texture Group Editor), e la funzione restituirà true se sono state spacchettate, o false altrimenti.

Sintassi:

texture_is_ready(tex_id);

Argomento Tipo Descrizione
tex_id Puntatore alla texture o stringa The texture page pointer or a texture group name (a string)

Ritorna:

Boolean

Esempio:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Il codice di cui sopra recupererà gli ID di pagina texture per il gruppo texture "MainMenu", poi controllerà per vedere se sono spacchettati, e se non lo sono vengono preimpostati nella VRAM.
