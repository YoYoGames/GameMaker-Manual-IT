lavaggio_tessuto

Questa funzione può essere usata per "lavare" una pagina texture o un gruppo di pagine texture, cioè: rimuoverle dalla VRAM quando non sono più necessarie. Si fornisce l'ID unico della pagina della texture (come restituito dalle funzioni texturegroup_* ) per cancellare una singola pagina, oppure si può fornire il nome di un gruppo di texture (come definito nel Texture Group Editor) per cancellare tutte le pagine texture nel gruppo.

Sintassi:

texture_flush(tex_id);

Argomento Tipo Descrizione
tex_id Puntatore alla texture o stringa The texture page pointer or a texture group name (a string)

Ritorna:

N/A

Esempio:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

Il codice di cui sopra farà scorrere tutte le pagine di texture sotto il gruppo texture "MainMenu".
