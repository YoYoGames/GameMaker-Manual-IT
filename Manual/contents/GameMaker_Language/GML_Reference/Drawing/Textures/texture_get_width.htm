larghezza della texture

Restituisce la larghezza di texture con l'id dato, che è sempre un valore compreso nell'intervallo 0 - 1. Questo può essere usato quando si mappano textures a modelli o primitive.

Sintassi:

texture_get_width(tex);

Argomento Tipo Descrizione
tex Puntatore di texture The texture page asset pointer to use

Ritorna:

Real

Esempio:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Il codice qui sopra otterrà la larghezza del texture preso da un sprite asset.
