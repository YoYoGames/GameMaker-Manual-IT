gruppo texture_get_fonts

Con questa funzione è possibile recuperare l'indice font di ciascuna delle fonts assegnate alle pagine texture all'interno del dato gruppo texture. Si fornisce l'ID del gruppo texture string (come definito nel texture Group Editor) e la funzione restituisce un array 1D dove ogni voce contiene l'indice font per una risorsa font. Se la funzione fallisce - cioè: viene fornito un gruppo non valido, o il gruppo non ha texture assegnato ad esso - allora l'array sarà vuoto (lunghezza 0).

Sintassi:

texturegroup_get_fonts(tex_id);

Argomento Tipo Descrizione
tex Stringa The name of the texture group to check (a string)

Ritorna:

Array

Esempio:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
}

Il codice qui sopra recupererà gli indici di font per il gruppo texture "MainMenu", poi mostrerà questi ID nella finestra di output della console.
