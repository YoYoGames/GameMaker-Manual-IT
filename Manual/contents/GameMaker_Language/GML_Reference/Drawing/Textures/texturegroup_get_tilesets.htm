gruppo texture_get_tilesets

Con questa funzione è possibile recuperare l'indice del tileset di ciascuno dei tilesets assegnati alle pagine texture all'interno del dato gruppo texture. Si fornisce l'ID del gruppo texture string (come definito nel texture Group Editor) e la funzione restituirà un array 1D dove ogni voce contiene l'indice del tileset per una risorsa del tileset. Se la funzione fallisce - cioè: viene dato un gruppo non valido, o il gruppo non ha texture assegnato ad esso - allora l'array sarà vuoto (lunghezza 0).

Sintassi:

texturegroup_get_tilesets(tex_id);

Argomento Tipo Descrizione
tex Stringa The name of the texture group to check (a string)

Ritorna:

Array

Esempio:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

Il codice qui sopra recupererà gli indici del tileset per il gruppo texture "MainMenu", poi mostrerà questi ID nella finestra di output della console.
