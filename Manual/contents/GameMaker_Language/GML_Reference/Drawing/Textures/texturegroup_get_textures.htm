texturegroup_get_textures

Questa funzione può essere utilizzata per recuperare gli ID delle pagine texture delle singole pagine che compongono un gruppo texture. Si fornisce l'ID del gruppo texture string (come definito nell'editor di gruppi Texture ) e la funzione restituisce un array 1D, in cui ogni voce dell'array è un singolo ID di pagina texture. Se la funzione fallisce, cioè se viene fornito un gruppo non valido o se al gruppo non è assegnato texture, l'array sarà vuoto (lunghezza 0).

Sintassi:

texturegroup_get_textures(tex_id);

Argomento Tipo Descrizione
tex_id Stringa The name of the texture group to check (a string)

Restituzione:

Array of Texture Pointers

Esempio:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Il codice sopra riportato recupera gli ID delle pagine di texture per il gruppo "MainMenu" di texture, quindi controlla se sono scompattate e, in caso affermativo, le inserisce nella VRAM.
