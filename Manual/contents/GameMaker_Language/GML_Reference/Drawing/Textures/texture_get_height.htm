altezza texture_get

Restituisce l'altezza di texture con l'id dato, che è sempre un valore compreso nell'intervallo 0 - 1. Questo può essere usato quando si mappano textures a modelli o primitive.

Sintassi:

texture_get_height(tex);

Argomento Tipo Descrizione
tex Puntatore di texture The texture page asset pointer to use

Ritorna:

Real

Esempio:

tex_h = texture_get_height(surface_get_texture(global.Surf));

Il codice di cui sopra otterrà l'altezza di texture presa da una superficie creata in precedenza.
