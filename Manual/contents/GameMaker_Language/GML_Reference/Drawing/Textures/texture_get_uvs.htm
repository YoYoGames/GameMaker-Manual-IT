texture_get_uvs

Questa funzione restituisce un array 1D con 4 elementi che rappresentano le coordinate UV per il puntatore texture dato, riempiendo l'array con i seguenti valori:

  • [0] = sinistra
  • [1] = top
  • [2] = destra
  • [3] = fondo
  • [4] = quantità di pixel che il compilatore asset ha tagliato dal lato sinistro di sprites (solosprite assets )
  • [5] = quantità di pixel che il compilatore asset ha tagliato dal lato superiore di sprites ( solosprite assets )
  • [6] = percentuale normalizzata di dati in pixel della larghezza originale di sprites che è stata salvata nella pagina texture ( solosprite assets )
  • [7] = percentuale normalizzata di dati pixel dell'altezza originale di sprites che è stata salvata nella pagina texture (solosprite assets )

Questo valore può essere utilizzato in altre funzioni di disegno, in particolare nelle funzioni 3D generali e in alcune primitive 2D, nonché nelle funzioni di Shader. Se si ha bisogno dell'UVS per un sprite, si può usare il sprite_get_uvs() e per un font, font_get_uvs().

Sintassi:

texture_get_uvs(texid)

Argomento Tipo Descrizione
texid Struttura The texture pointer to get the UVS for

Restituzione:

Array (4 - 8 elements)

Esempio:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

Il codice precedente recupera innanzitutto il sito texture per la superficie memorizzata in surf_back, quindi ottiene le coordinate UV per quel sito texture. Quindi recupera le coordinate UV sinistra, superiore, destra e inferiore dall'array restituito e le memorizza in variabili locali.
