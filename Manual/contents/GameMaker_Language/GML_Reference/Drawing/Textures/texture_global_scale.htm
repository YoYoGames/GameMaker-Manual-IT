scala_globale

Questa funzione permette di controllare il ridimensionamento delle pagine di texture al caricamento dal file WAD che fa parte dell'eseguibile finale del gioco. Il valore di input deve essere un valore di potenza di due e funzionerà in modo tale che:

  • Per impostare le pagine di texture come 1:1 usare texture_global_scale(1)
  • Per metà textures usare sempre texture_global_scale(2)
  • Per acquartierarli usa texture_global_scale(4), ecc...

In questo modo è possibile controllare meglio l'uso della memoria della pagina texture su piattaforme con problemi di bassa memoria. Notate che le pagine textures sono create su richiesta dal WAD, e quindi potete chiamarlo all'inizio del gioco per caricare tutte le pagine textures in scala, oppure potete può essere chiamata in momenti specifici per caricare specifiche textures, nel qual caso probabilmente vorrete chiamare draw_texture_flush() prima di chiamare questa funzione, quindi utilizzare le funzioni di pre-fetch per ottenere le texture pagine in memoria.

Sintassi:

texture_global_scale(pow2integer);

ArgomentoTipo Descrizione
pow2integer The scale factor to use (1, no scale, 2, half scale, 4, quarter scale, etc...)

Ritorna:

Esempio:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

Il codice di cui sopra cancellerà tutto textures dalla memoria, poi imposterà il ridimensionamento di texture a 2 (quindi le pagine di texture sono grandi la metà) e infine pre-fetcherà la pagina texture in cui è contenuto lo "spr_Trees" di sprite.
