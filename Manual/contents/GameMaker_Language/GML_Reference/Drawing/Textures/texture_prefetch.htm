prelievo texture

Questa funzione può essere usata per "prefetchare" una pagina texture o un gruppo di pagine texture, cioè: caricarle nella VRAM quando richiesto. Si fornisce l'ID unico della pagina della texture (come restituito dalle funzioni texturegroup_* ) per prefetchare una singola pagina, oppure si può fornire il nome di un gruppo di texture (come definito nel Texture Group Editor) per prefetchare tutte le pagine texture nel gruppo.

Sintassi:

texture_prefetch(tex_id);

Argomento Tipo Descrizione
tex_id Puntatore alla texture o stringa The texture page pointer or a texture group name (a string)

Ritorna:

N/A

Esempio:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

Il codice di cui sopra preleverà tutte le pagine texture sotto il gruppo texture "MainMenu".
