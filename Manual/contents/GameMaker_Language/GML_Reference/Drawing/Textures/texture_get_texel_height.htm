altezza deltexel della texture

Un texel, o elemento texture, è l'unità fondamentale dello spazio texture usato nella computer grafica. Textures è rappresentato da array di texel, proprio come le immagini sono rappresentate da array di pixel, e questa funzione restituisce l'altezza di un singolo texel dalla pagina texture dell'immagine asset usata.

Sintassi:

texture_get_texel_height(tex);

Argomento Tipo Descrizione
tex Puntatore di texture The texture page asset pointer to use

Ritorna:

Real

Esempio:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Il codice qui sopra otterrà la larghezza e l'altezza del texel di texture preso da un sprite asset.
