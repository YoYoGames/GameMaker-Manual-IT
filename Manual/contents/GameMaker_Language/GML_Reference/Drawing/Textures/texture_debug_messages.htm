messaggi di debug delle texture

Questa funzione può essere usata per abilitare o disabilitare i messaggi di debug di texture. Quando è abilitata (impostata a true), informazioni aggiuntive sull'uso della pagina texture saranno inviate alla finestra di output. Impostate su false per disabilitare nuovamente questo output.

Sintassi:

texture_debug_messages(enable);

ArgomentoTipo Descrizione
enable Enable or disable the texture debug messages

Ritorna:

Esempio:

if global.debug
{
    texture_debug_messages(true);
}

Il codice qui sopra controllerà il valore di una variabile globale e se è impostato su true allora texture messaggi di debug sono abilitati.
