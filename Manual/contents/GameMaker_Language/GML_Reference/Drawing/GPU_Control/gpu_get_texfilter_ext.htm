gpu_get_texfilter_ext

Con questa funzione si può verificare se l'interpolazione texture (interpolazione lineare) è abilitata (restituisce true) o meno (restituisce false) per un dato campionatore shader texture .

NOTA: sul target HTML5, questa funzione funziona solo con WebGL abilitato.

Sintassi:

gpu_get_texfilter_ext(sampler_id);

Argomento Tipo Descrizione
sampler_id Maniglia Shader Sampler The sampler id from the shader.

Restituzione:

Boolean

Esempio:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Il codice precedente controlla se il filtraggio di texture è attivo o meno per uno specifico ID di campionatore (memorizzato in una variabile locale) e lo attiva di conseguenza.
