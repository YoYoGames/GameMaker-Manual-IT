gpu_set_texfilter_ext

Questa funzione può essere usata per impostare l'interpolazione lineare per un singolo "slot" di campionamento quando si usano gli shader in GameMaker. Quando questo è abilitato (true) il campionatore texture sarà smussato e se questo è disabilitato (false) allora le immagini saranno disegnate in base al pixel più vicino. Il valore predefinito è quello impostato dalle Global Game Options per il vostro gioco, o quello impostato utilizzando la funzione gpu_set_texfilter().

NOTA: Questa impostazione sarà sovrascritta dal valore impostato quando si chiama la funzione gpu_set_texfilter().

NOTA: sul target HTML5, questa funzione funziona solo con WebGL abilitato.

Sintassi:

gpu_set_texfilter_ext(sampler_id, enable);

ArgomentoTipo Descrizione
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Ritorna:

Esempio:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Il codice di cui sopra controlla se il filtraggio texture è attivato o disattivato per uno specifico ID del campionatore (memorizzato in una variabile locale) e lo commuta di conseguenza.
