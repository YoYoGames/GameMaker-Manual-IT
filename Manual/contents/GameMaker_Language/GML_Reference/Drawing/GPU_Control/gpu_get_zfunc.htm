gpu_get_zfunc

Questa funzione può essere usata per recuperare la modalità di confronto z. Il valore restituito sarà una delle seguenti costanti (il valore predefinito è cmpfunc_lessequal):

Funzione di confronto Costante
Costante Descrizione
cmpfunc_never Mai
cmpfunc_less Meno
cmpfunc_equal Uguale
cmpfunc_lessequal Meno o uguale
cmpfunc_greater Maggiore
cmpfunc_notequal Non uguale
cmpfunc_greaterequal Maggiore o uguale
cmpfunc_always Sempre

Sintassi:

gpu_get_zfunc();

Ritorna:

Comparison Function Constant (see table above)

Esempio:

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

Il codice qui sopra controlla per vedere se il metodo z-testing è impostato su cmpfunc_greater e se non lo è allora è impostato su quella costante.
