gpu_get_state

Questa funzione otterrà lo stato corrente della GPU, restituendolo come DS Map. Questo può quindi essere manipolato o anche salvato, ed è possibile restituire questa mappa alla GPU utilizzando la funzione gpu_set_state().

Sintassi:

gpu_get_state();

Ritorna:

DS Map ID

Esempio:

gpu_map = gpu_get_state();

Il codice precedente memorizza lo stato attuale della GPU in una variabile.

Indietro: Controllo GPU
Il prossimo: gpu_set_state
