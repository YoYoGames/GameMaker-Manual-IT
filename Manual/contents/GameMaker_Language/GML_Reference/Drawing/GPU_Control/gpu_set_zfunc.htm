gpu_set_zfunc

Questa funzione può essere usata per impostare la modalità di confronto del test dello z-buffer (vedere gpu_set_ztestenable() per maggiori informazioni). I valori disponibili per l'uso sono una qualsiasi delle seguenti costanti (il valore predefinito è cmpfunc_lessequal):

Costante Descrizione
cmpfunc_never Mai
cmpfunc_less Meno
cmpfunc_equal Uguale
cmpfunc_lessequal Meno o uguale
cmpfunc_greater Maggiore
cmpfunc_notequal Non uguale
cmpfunc_greaterequal Maggiore o uguale
cmpfunc_always Sempre

Sintassi:

gpu_set_zfunc(cmp_func);

Argomento Tipo Descrizione
cmp_func Funzione di confronto Costante The comparison mode to use (see list above)

Ritorna:

N/A

Esempio:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Il codice qui sopra attiva il test dello z-buffer e imposta la sua modalità di confronto prima di disegnare uno sfondo sprite e poi lo disattiva di nuovo per continuare a disegnare.
