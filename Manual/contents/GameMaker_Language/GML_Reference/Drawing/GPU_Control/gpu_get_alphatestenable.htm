gpu_get_alphatestenable

Con questa funzione potete controllare se l'alpha testing è abilitato (restituisce true) o no (restituisce false). Per maggiori informazioni sull'alpha testing, si veda la funzione gpu_set_alphatestref().

Sintassi:

gpu_get_alphatestenable();

Ritorna:

Esempio:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Il codice qui sopra controllerà per vedere se l'alpha testing è abilitato e se non lo è switch su alpha testing e imposterà la soglia di test a 128 (solo i pixel con un alpha superiore a 0.5 saranno disegnati).
