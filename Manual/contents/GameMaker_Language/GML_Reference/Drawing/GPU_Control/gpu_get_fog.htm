gpu_get_fog

Questa funzione può essere usata per recuperare le impostazioni della nebbia. La funzione restituisce un array 1D di 4 elementi con le seguenti informazioni:

  • [0] = abilitato alla commutazione (un booleano, o true o false), predefinito false
  • [1] = Costante di colore, predefinito c_black
  • [2] = distanza iniziale(reale), default 0
  • [3] = distanza finale(reale), default 1

Notate che potete cambiare questi valori e passare l'intero array alla funzione gpu_set_fog() come singolo argomento.

Sintassi:

gpu_get_fog();

Ritorna:

Array (4 elements only; see above for details)

Esempio:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Il codice qui sopra ottiene le impostazioni correnti della nebbia e poi imposta l'elemento colore dell'array su c_red prima di impostare nuovamente la nebbia usando l'array modificato.
