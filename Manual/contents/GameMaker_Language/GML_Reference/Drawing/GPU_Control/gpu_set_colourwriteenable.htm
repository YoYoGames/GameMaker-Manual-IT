gpu_set_colourwriteenable

Con questa funzione potete switch attivare o disattivare i canali di colore e il canale alfa per tutti i disegni successivi. Per esempio, potete usare questa funzione per creare superfici con solo il canale alfa (switch off la scrittura RGB prima di usare la superficie), o per creare effetti speciali mentre disegnate sullo schermo. Il valore predefinito per ogni componente è true, e può essere fornito come quattro argomenti unici o come un array 1D di 4 elementi con i seguenti elementi che saranno o true (abilitato) o false (disabilitato):

  • [0] = canale rosso abilitato/disabilitato
  • [1] = canale verde abilitato/disabilitato
  • [2] = canale blu abilitato/disabilitato
  • [3] = canale alfa abilitato/disabilitato

Sintassi:

gpu_set_colorwriteenable(red, green, blue, alpha);

Argomento Tipo Descrizione
red Booleano Enable/disable the red channel
green Booleano Enable/disable the green channel
blue Booleano Enable/disable the blue channel
alpha Booleano Enable/disable the alpha channel

Ritorna:

N/A

Esempio:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Il codice qui sopra ottiene i valori di scrittura del colore corrente e poi imposta il componente alfa su false.
