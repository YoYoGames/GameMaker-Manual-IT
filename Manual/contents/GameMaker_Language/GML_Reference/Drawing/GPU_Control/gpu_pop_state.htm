stato gpu_pop

Questa funzione estrae lo stato precedente della GPU dallo stack e lo applica. Vedere gpu_push_state() per maggiori informazioni.

Sintassi:

gpu_pop_state();

Ritorna:

Esempio:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

Il codice di cui sopra "salverà" lo stato attuale della GPU sullo stack, poi cambierà alcune impostazioni della GPU e disegnerà un gruppo di istanze prima di resettare lo stato della GPU a quello che era in precedenza.
