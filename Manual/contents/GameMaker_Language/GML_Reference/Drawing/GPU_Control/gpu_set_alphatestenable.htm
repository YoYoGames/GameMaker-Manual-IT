gpu_set_alphatestenable

Questa funzione abiliterà o disabiliterà l'alpha testing per il vostro gioco (di default è disabilitato). Attivando l'alpha testing potete usare la funzione companion gpu_set_alphatestref() per impostare il valore "cut-off" al quale tutti i valori alfa saranno impostati a 0. L'immagine qui sotto mostra la differenza che può essere vista quando l'alpha testing è attivato o disattivato:

NOTA: Questa funzione può influenzare negativamente le prestazioni su iOS e dispositivi Android.

Sintassi:

gpu_set_alphatestenable(enable);

ArgomentoTipo Descrizione
enable Enable or disable alpha testing (true / false)

Ritorna:

Esempio:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Il codice qui sopra controllerà per vedere se l'alpha testing è abilitato e se non lo è switch su alpha testing e imposterà la soglia di test a 128 (solo i pixel con un alpha superiore a 0.5 saranno disegnati).
