gpu_get_blendmode_destalpha

Questa funzione può essere usata per recuperare l'attuale fattore alfa della modalità di fusione estesa della destinazione. Il valore restituito sarà una delle seguenti costanti ("s" denota un valore preso dall'origine mentre una "d" denota un valore dalla destinazione) con solo la componente "A" utilizzata durante il disegno:

Costante Fattore di fusione (rosso, verde, blu, alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (annuncio, annuncio, annuncio, annuncio)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) dove f = min(As, 1-Ad)

Sintassi:

gpu_get_blendmode_destalpha();

Ritorna:

Blend Mode Factor Constant (see above table)

Esempio:

var bm;
bm[0] = gpu_get_blendmode_srcalpha();
bm[1] = gpu_get_blendmode_destalpha();
gpu_set_blendmode_ext_sepalpha(bm_inv_src_alpha, bm_inv_dest_colour, bm[0], bm[1]);

Il codice di cui sopra crea una matrice locale e ottiene i fattori di miscelazione di origine e destinazione correnti per la componente alfa. Questo array viene poi utilizzato per manipolare la componente RGB dei fattori di miscelazione.
