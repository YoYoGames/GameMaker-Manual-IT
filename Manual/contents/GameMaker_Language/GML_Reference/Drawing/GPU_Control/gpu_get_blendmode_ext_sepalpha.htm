gpu_get_blendmode_ext_sepalpha

Questa funzione può essere usata per recuperare la modalità di fusione estesa correntemente usata per il disegno, separando le modalità di fusione in due parti - la componente RGB e la componente Alpha. La funzione restituisce un array 1D di 4 elementi con i seguenti elementi:

  • [0] = Modalità di fusione della sorgente meno la componente alfa (l'impostazione predefinita è bm_src_alpha (As, As, As))
  • [1] = Modalità di fusione della destinazione meno la componente alfa (l'impostazione predefinita è bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = componente alfa della sorgente (l'impostazione predefinita è bm_src_alpha (As))
  • [3] = componente alfa di destinazione (l'impostazione predefinita è bm_inv_src_alpha (1-As))

I valori dell'array saranno una delle seguenti costanti ("s" denota un valore preso dalla sorgente mentre una "d" denota un valore dalla destinazione):

Costante Fattore di fusione (rosso, verde, blu, alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (annuncio, annuncio, annuncio, annuncio)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) dove f = min(As, 1-Ad)

Quando si creano le modalità di fusione estese in questo modo, le componenti RGB saranno prese da una modalità di fusione e la componente alfa sarà presa da un'altra, così che - per esempio - se si usa bm_zero (0, 0, 0, 0) come modalità di fusione RGB e bm_one come modalità di fusione alfa, si otterrà una miscela composta di (0,0,0,1).

Notate che potete cambiare questi valori direttamente all'interno dell'array e passare l'intero array alla funzione come singolo argomento (come mostrato nell'esempio qui sotto).

Sintassi:

gpu_get_blendmode_ext_sepalpha();

Ritorna:

(4 elements only; see above for constants)

Esempio:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

Il codice qui sopra crea una matrice locale e ottiene la modalità di fusione estesa corrente con valori alfa separati. Questo array viene poi manipolato e restituito per impostare la modalità di fusione.
