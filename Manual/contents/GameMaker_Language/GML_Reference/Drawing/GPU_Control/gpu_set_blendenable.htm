gpu_set_blendenable

Questa funzione può essere usata per attivare e disattivare l'alpha-blending. Fondamentalmente, se avete questa funzione impostata su false, tutte le immagini disegnate saranno disegnate al 100% opache, il che significa che qualsiasi area trasparente o semi-trasparente di un sprite o dello sfondo sarà sarà visibile come un colore solido. È una buona idea avere l'alpha blending off ogni volta che è possibile (specialmente quando si sviluppa per i dispositivi mobili) perché questo aumenta notevolmente la velocità di disegno.

Sintassi:

gpu_set_blendenable(enable);

ArgomentoTipo Descrizione
enable Enable or disable alpha blending value (true o false).

Ritorna:

Esempio:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

Il codice qui sopra spegne l'alpha blending per disegnare uno sfondo sprite e poi lo riaccende per continuare a disegnare.
