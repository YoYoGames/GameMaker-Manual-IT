gpu_set_fog

Questa funzione può essere utilizzata per attivare o disattivare il disegno della nebbia. La nebbia può essere utilizzata nei giochi 3D per far apparire sbiadite o addirittura invisibili le istanze in lontananza, contribuendo a creare l'atmosfera e a mascherare il fatto che non si stanno disegnando istanze lontane.

Si imposta se è abilitata (true) o disabilitata (false), il colore che la nebbia deve usare per la sfumatura e le distanze di inizio e fine disegno. I valori della distanza sono essenzialmente valori di profondità in pixel e sono relativi alla posizione della telecamera.

Ad esempio, una posizione pari a 0 è la più vicina alla telecamera e ad ogni aumento si allontana. Per impostazione predefinita, la telecamera di una vista è posizionata a -16000 di profondità, quindi se si vuole che la nebbia inizi a profondità 0 e finisca a profondità 1000, si deve impostare la distanza iniziale a 16000 e quella finale a 17000.

La funzione può accettare quattro singoli argomenti (indicati sopra) o un singolo array con la seguente struttura (il codice di esempio qui sotto mostra questo metodo):

  • [0] = attivazione del toggle (un booleano, true o false), predefinito false
  • [1] = colore (reale), predefinito c_black
  • [2] = distanza iniziale (reale), valore predefinito 0
  • [3] = distanza finale (reale), valore predefinito 1

Sintassi:

gpu_set_fog(enable, colour, start, end);

Argomento Tipo Descrizione
enable Booleano Enable or disable fog
colour Colore The fog colour
start Reale The distance to start applying fog from (relative to the camera)
end Reale The distance to end the fog (relative to the camera)

Restituzione:

N/A

Esempio:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Il codice precedente ottiene le impostazioni correnti della nebbia e quindi imposta l'elemento colore dell'array su c_red prima di impostare nuovamente la nebbia utilizzando l'array modificato.
