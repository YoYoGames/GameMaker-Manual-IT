gpu_set_cullmode

Questa funzione può essere usata per impostare la modalità di abbattimento della faccia posteriore. Un poligono (primitiva) ha un lato anteriore e uno posteriore. Il lato anteriore è detto essere il lato in cui i vertici sono definiti in ordine antiorario. Normalmente vengono disegnati entrambi i lati, ma se si fa una forma chiusa (come una piramide) allora questo è uno spreco di potenza di elaborazione perché il lato posteriore del triangolo non può mai essere visto. È in questi casi che si può switch sul backface culling, dato che questo risparmia circa la metà del tempo di disegno, anche se significa che si ha il compito di definire i poligoni nel modo corretto per evitare problemi.

Il valore di cull mode utilizzato deve essere una delle seguenti costanti (il valore predefinito è cull_noculling):

Costante Descrizione
cull_noculling Nessun abbattimento sarà fatto
cull_clockwise Tutti i triangoli in senso orario saranno eliminati
cull_counterclockwise Tutti i triangoli in senso antiorario saranno eliminati

Sintassi:

gpu_set_cullmode(cullmode);

Argomento Tipo Descrizione
cullmode Modalità di abbattimento costante The cull mode to use (see list above)

Ritorna:

N/A

Esempio:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Il codice di cui sopra ottiene la modalità di abbattimento corrente e se non è cull_clockwise è impostato su quella costante.
