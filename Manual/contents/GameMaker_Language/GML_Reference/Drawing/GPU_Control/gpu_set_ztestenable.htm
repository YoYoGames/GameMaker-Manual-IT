gpu_set_ztestenable

Questa funzione può essere usata per attivare o disattivare il test dello z-buffer, influenzando il modo in cui le cose saranno disegnate (in generale è utile solo quando si lavora con progetti 3D). Essenzialmente, per impostazione predefinita, quando lo z-testing è disattivato e si hanno due objects che disegnano nello stesso spazio, entrambi objects saranno renderizzati senza considerare se uno disegnerà più dell'altro, con il risultato di chiamate di disegno non necessarie. Se switch questo è on allora lo z-buffer è testato per vedere se un object sarà "visibile" e non disegnato se non lo è no. Si noti che questo è il comportamento predefinito, ma è possibile cambiarlo cambiando il tipo di confronto utilizzato per il test dello z-buffer (vedere la funzione gpu_set_zfunc().

Per default il test dello z-buffer è disattivato (false).

Sintassi:

gpu_set_ztestenable(enable);

ArgomentoTipo Descrizione
enable Enable or disable z-buffer testing (true o false).

Ritorna:

Esempio:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Il codice qui sopra attiva il test dello z-buffer per disegnare uno sfondo sprite e poi lo disattiva di nuovo per continuare a disegnare.
