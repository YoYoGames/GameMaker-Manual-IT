gpu_get_texrepeat_ext

Con questa funzione si può controllare se la ripetizione texture è abilitata (ritorna true) o no (ritorna false) per un dato campionatore shader texture .

Sintassi:

gpu_get_texrepeat_ext(sampler_id);

ArgomentoTipo Descrizione
sampler_id The sampler id from the shader.

Ritorna:

Esempio:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if !gpu_get_texrepeat_ext(s_tex)
{
    gpu_set_texrepeat_ext(s_tex, true);
}

Il codice qui sopra controlla se texture filtraggio off per uno specifico ID del campionatore (memorizzato in una variabile locale) e lo attiva se non lo è.
