gpu_set_texfilter

Questa funzione può essere usata per impostare il filtraggio texture (interpolazione lineare) di tutte le immagini disegnate sullo schermo di gioco. Quando questo è abilitato (true) tutti i textures saranno smussati quando disegnati (questo include sprites poiché anch'essi sono considerati textures), il che significa che quando vengono scalati o spostati se non c'è un rapporto pixel 1:1 allora ci sarà una "sbavatura" attraverso vari pixel che può far apparire le immagini sfocate a seconda dello stile artistico usato. Se questo è disabilitato (false) allora le immagini saranno disegnate in base al pixel più vicino quando vengono scalate o spostate, il che può portare a immagini "a blocchi". Il valore predefinito è false, e questo può anche essere cambiato nelle Global Game Options per le singole piattaforme di destinazione.

NOTA: Questa impostazione sovrascrive qualsiasi interpolazione dello stadio texture impostata per shaders usando la funzione gpu_set_texfilter_ext().

NOTA: sul target HTML5, questa funzione funzionerà solo con WebGL abilitato.

Sintassi:

gpu_set_texfilter(enable);

ArgomentoTipo Descrizione
enable Enable or disable texture filtering (true / false)

Ritorna:

Esempio:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

Il codice di cui sopra controlla se il filtraggio texture è on o off e lo commuta di conseguenza.
