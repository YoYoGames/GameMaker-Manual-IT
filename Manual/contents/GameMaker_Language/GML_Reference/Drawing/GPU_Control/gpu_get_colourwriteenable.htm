gpu_get_colourwriteenable

Questa funzione può essere usata per recuperare i valori correnti di abilitazione alla scrittura dei colori. La funzione restituisce un array 1D di 4 elementi con i seguenti elementi che saranno o true (abilitato) o false (disabilitato). Per impostazione predefinita, tutta la scrittura dei colori è impostata su true:

  • [0] = canale rosso abilitato/disabilitato
  • [1] = canale verde abilitato/disabilitato
  • [2] = canale blu abilitato/disabilitato
  • [3] = canale alfa abilitato/disabilitato

Sintassi:

gpu_get_colorwriteenable();

Ritorna:

(4 elements only; see above for details)

Esempio:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Il codice qui sopra ottiene i valori di scrittura del colore corrente e poi imposta il componente alfa su false.
