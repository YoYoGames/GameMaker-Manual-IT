gpu_get_alphatestref

Potete usare questa funzione per trovare il valore corrente per il riferimento del test alfa (il valore predefinito è 0, ma potete usare gpu_set_alphatestref() per impostare questo valore su qualcosa di diverso da questo).

Sintassi:

gpu_get_alphatestref();

Ritorna:

(from 0 - 255)

Esempio:

if gpu_get_alphatestenable()
{
    if gpu_get_alphatestref() < 254
    {
        gpu_set_alphatestref(254);
    }
}

Il codice qui sopra controlla se il test alfa è abilitato, e se lo è, controlla il valore corrente di riferimento del test alfa e lo imposta se è inferiore a 254.
