gpu_set_texrepeat

Questa funzione può essere usata per indicare a GameMaker se textures deve ripetersi (true) o meno (false) quando viene usato per modelli e primitive. Normalmente le larghezze e le altezze di texture sono comprese nell'intervallo 0 - 1, ma se si specifica un valore maggiore di 1 di default il resto non viene disegnato (l'intero texture sarà disegnato con una sezione "vuota" corrispondente alla percentuale oltre 1 che viene utilizzata. Così un valore di 2 per un texture larghezza o altezza disegnerebbe il texture in metà dello spazio e lascerebbe l'altra metà vuota). Tuttavia, utilizzando questa funzione per impostare la ripetizione su true, texture viene ripetuto automaticamente su tutti i modelli e le primitive. Il caso d'uso probabile per queste funzioni è la ripetizione di un texture in 3D, ma per far sì che funzioni e non tiri le immagini dal resto della pagina texture, il sprite usato dovrà essere segnato come se fosse su una "Pagina separata Texture " nell'editor Sprite.

NOTA: Sprites e gli sfondi sono sempre disegnati senza ripetizione, quindi una volta che si disegna un sprite o uno sfondo questo valore viene resettato a false. Questo comportamento di default non influenzerà le impostazioni dello stadio texture (cioè: chiamando la funzione lo farà, ma l'impostazione interna a false non lo farà).

NOTA: Questa impostazione sovrascrive qualsiasi texture ripetizione di fase impostata per shaders quando si usa la funzione gpu_set_texrepeat_ext().

Sintassi:

gpu_set_texrepeat(enable);

ArgomentoTipo Descrizione
enable Enable or disable texture filtering (true / false)

Ritorna:

Esempio:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

Il codice qui sopra controlla se texture repeating è on o off e lo commuta di conseguenza.
