gpu_get_blendenable

Questa funzione può essere usata per recuperare lo stato di alpha-blending. Se restituisce true allora l'alpha-blending è abilitato, e se restituisce false è disabilitato. Per impostazione predefinita questo è on e quindi la funzione restituirà true

Sintassi:

gpu_get_blendenable();

Ritorna:

Esempio:

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

Il codice di cui sopra controlla lo stato dell'alpha blending e se è disabilitato viene poi abilitato di nuovo.
