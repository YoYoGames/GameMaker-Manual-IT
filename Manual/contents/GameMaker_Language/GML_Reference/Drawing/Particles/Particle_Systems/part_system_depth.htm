profondità del sistema parziale

Con questa funzione è possibile impostare la profondità di disegno per il sistema di particelle, più o meno come è possibile impostare la profondità di rendering dei diversi livelli all'interno di room, dove una bassa profondità di disegno significa che apparirà in cima a tutte le cose disegnate con una profondità maggiore, e un'alta profondità di disegno che lo pone sotto tutto ciò che ha una profondità di disegno inferiore.

Sintassi:

part_system_depth( ind, depth );

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to change.
depth Real The depth at which to set the particle system.

Ritorna:

N/A

Esempio:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

Il codice di cui sopra creerà un sistema di particelle e memorizzerà il suo indice nella variabile globale "Sname". A questo sistema viene poi data una profondità bassa di -1000, il che significa che apparirà sopra tutto ciò che ha una profondità di disegno superiore.
