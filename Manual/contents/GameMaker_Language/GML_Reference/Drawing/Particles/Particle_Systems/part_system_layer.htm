strato_sistema_parte

Questa funzione può essere utilizzata per switch un sistema di particelle dal suo livello corrente a uno nuovo. Si fornisce il valore dell'ID del sistema di particelle (come restituito dalla funzione part_system_create_layer()) e poi l'ID unico del livello (come restituito dalla funzione layer_create() o il nome del livello da usare come string - per esempio "instance_layer" - come definito nell'editor room ), e il sistema sarà spostato nel nuovo livello.

Sintassi:

part_system_layer(ps, layer);

Argomento Tipo Descrizione
ps ID del sistema di particelle The particle system ID
layer Stringa o ID del livello The new layer ID

Ritorna:

N/A

Esempio:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Il codice qui sopra controllerà un sistema di particelle per vedere su quale livello si trova e se non è sul livello dato verrà cambiato.
