part_system_get_layer

Questa funzione può essere utilizzata per recuperare il valore unico dell'ID del livello per il sistema di particelle dato. Si fornisce il valore ID del sistema di particelle (come restituito dalla funzione part_system_create_layer()) e la funzione restituirà il valore ID del livello.

Sintassi:

part_system_get_layer(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The particle system ID value to get the layer ID of

Ritorna:

Layer ID or 0 (if the layer is an internally managed one)

Esempio:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, _effects_layer);
}

Il codice qui sopra controllerà un sistema di particelle per vedere su quale livello si trova e se non è sul livello dato verrà cambiato.
