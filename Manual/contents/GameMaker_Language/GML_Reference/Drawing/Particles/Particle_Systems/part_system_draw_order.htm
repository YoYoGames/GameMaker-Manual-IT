ordine di prelievo del sistema

Con questa funzione puoi impostare il modo in cui le particelle sono disegnate quando vengono create sullo schermo. Il sistema predefinito utilizza un look vecchio > nuovo (la funzione è impostata su true), dove le particelle vecchie sono disegnate a una profondità maggiore di quelle nuove e quindi appaiono "sotto" le particelle nuove, ma impostando questa funzione su false è possibile invertire questo ordine e avere le particelle nuove disegnate più in alto e quindi appaiono "sotto" quelle vecchie. Le immagini qui sotto illustrano questo, con l'immagine a sinistra che è il valore di default di true e l'immagine a destra che è false:

NOTA: Quando le particelle vengono disegnate con una modalità di fusione additiva, l'effetto di questa funzione potrebbe non essere sempre evidente.

Sintassi:

part_system_draw_order(ind, oldtonew);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to change.
oldtonew Booleano Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Ritorna:

N/A

Esempio:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Questo creerà un nuovo sistema di particelle con l'indice "mysystem", e poi imposta le particelle per disegnare le particelle più recenti in cima a quelle più vecchie.
