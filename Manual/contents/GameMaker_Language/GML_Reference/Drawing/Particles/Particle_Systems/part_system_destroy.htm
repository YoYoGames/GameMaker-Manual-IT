distruggere il sistema

Con questa funzione puoi distruggere un dato sistema di particelle e rimuoverlo dalla memoria. Questo dovrebbe sempre essere chiamato quando il sistema non è più necessario, come alla fine di un room, o nell'evento di distruzione di un'istanza, altrimenti si potrebbe finire con le particelle che appaiono in rooms e nessun modo per rimuoverle, così come una perdita di memoria che alla fine farà crashare il gioco.

NOTA: Questa funzione distruggerà e rimuoverà anche tutti gli emettitori che possono essere stati creati e associati al sistema che viene distrutto.

Sintassi:

part_system_destroy(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to destroy.

Ritorna:

N/A

Esempio:

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

Il codice di cui sopra controlla se il valore di una variabile globale è inferiore o uguale a zero, e se lo è allora distrugge il sistema di particelle a cui fa riferimento la variabile data e poi va al prossimo room.

Indietro: Sistemi di particelle
