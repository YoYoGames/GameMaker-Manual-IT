part_system_create_layer

Questa funzione creerà un nuovo sistema di particelle su un dato livello. Si dà l'ID univoco del livello come restituito dalla funzione layer_create() o il nome del livello da usare come string - per esempio "instance_layer" - e quindi contrassegnare il sistema come persistente o no. Se il sistema non è contrassegnato come persistente allora sarà automaticamente distrutto alla fine del room in cui è stato creato (questo è lo stesso che se aveste chiamato la funzione part_system_destroy() e distruggerà anche qualsiasi emettitore associato al sistema). Tuttavia, quando è contrassegnato come persistente, il sistema sarà portato al prossimo room quando il room è cambiato, e se il seguente room non ha un livello con lo stesso nome o profondità di quello assegnato, allora un nuovo livello sarà creato per il sistema che sta persistendo attraverso il rooms, e sarà chiamato come il livello originale. Quando si cambia rooms, se c'è un altro livello nel seguente rooms con lo stesso nome, allora l'istanza persistente sarà assegnata al livello con quel nome indipendentemente dalla profondità del livello. Infine, se un sistema persistente si sposta su un room con un livello alla stessa profondità del livello su cui il sistema è stato creato, non verrà assegnato a questo livello, ma verrà invece creato un nuovo livello alla stessa profondità (con lo stesso nome del livello originale).

La funzione restituirà un valore ID unico per il sistema di particelle che dovrebbe essere usato in tutte le ulteriori chiamate di funzioni in cui è necessario dare un ID di sistema.

IMPORTANTE! Se contrassegnate il sistema di particelle come persistente, allora esso (e qualsiasi emettitore ad esso assegnato) dovrà essere ripulito manualmente usando le funzioni di distruzione appropriate quando non è in uso, altrimenti rischiate una perdita di memoria che avrà un impatto negativo sul vostro gioco finale.

Sintassi:

part_system_create_layer(layer_id, persistent);

Argomento Tipo Descrizione
layer Stringa o ID del livello The layer ID (or name) to assign the particle system to (can be any layer type)
persistent Booleano Flag the particle system as persistent (set to true) o no (impostato su false)

Ritorna:

Particle System ID

Esempio:

global.p_sys = part_system_create_layer("effects_layer", true);

Il codice di cui sopra creerà un nuovo sistema di particelle sul livello dato e lo segnalerà come persistente sui successivi rooms. L'ID del sistema di particelle viene memorizzato in una variabile di portata globale per riferimenti futuri.
