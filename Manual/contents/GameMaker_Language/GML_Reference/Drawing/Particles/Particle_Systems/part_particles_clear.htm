parte_particelle_clear

Con questa funzione è possibile cancellare tutte le particelle attualmente create dal sistema da room. Non azzera o rimuove i tipi di particelle stesse, solo la loro rappresentazione visiva, e se avete qualsiasi object particelle in streaming da un emettitore, queste particelle scompaiono ma inizieranno ad apparire di nuovo il passo successivo dopo aver chiamato questo codice.

Sintassi:

part_particles_clear(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system.

Ritorna:

N/A

Esempio:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

Il codice qui sopra controllerà il valore della variabile "lives" e se è uguale a 0, cancella tutte le particelle dal sistema e poi cambia room.
