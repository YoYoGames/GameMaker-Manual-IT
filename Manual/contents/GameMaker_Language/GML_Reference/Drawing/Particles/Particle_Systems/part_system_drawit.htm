aggiornamento del sistema parziale

Con questa funzione potete forzare un sistema di particelle a disegnare tutte le particelle correnti sullo schermo. Se part_system_automatic_draw() è disattivato allora questa funzione mostrerà le particelle quando viene utilizzata nell'evento draw di un'istanza, oppure può essere utilizzata quando il target di disegno è impostato su una superficie (utilizzando surface_set_target()) per disegnare le particelle del sistema su quella superficie.

Sintassi:

part_system_drawit(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to draw the particles of.

Ritorna:

N/A

Esempio:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

Il codice qui sopra controlla se la superficie indicizzata nella variabile "surf" esiste, e se esiste imposta il target di disegno sulla superficie, disegna il sistema di particelle con l'ID memorizzato nella variabile globale, e poi resetta il target di disegno in modo che tutto il disegno normale appaia di nuovo sullo schermo.
