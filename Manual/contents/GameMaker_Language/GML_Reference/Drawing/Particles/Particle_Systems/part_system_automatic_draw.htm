sistema_parte_automatico_disegno

Questa funzione può essere utilizzata per switch disattivare il disegno di un sistema di particelle in modo che qualsiasi aggiornamento fatto al sistema (automatico o altro) non sarà visto. Questa è un'opzione puramente visiva e quando è impostata su false non sarai in grado di vedere le particelle perché non sono disegnate, ma esistono ancora e stanno cambiando posizione, colore ecc... come farebbero normalmente. Quando il disegno automatico è disattivato, è possibile ordinare esplicitamente a GameMaker di disegnare lo stato attuale del sistema di particelle utilizzando la funzione part_system_drawit()e se si imposta nuovamente questa funzione su true è possibile switch riattivare il disegno automatico.

Una cosa da notare è che se state usando i semplici effetti creati dalle funzioni effect_create_above() o effect_create_below() allora puoi usare i valori 0 (per gli effetti sotto) o 1 (per gli effetti sopra) come indice del sistema di particelle e quindi attivare il disegno automatico anche per questi (questo funzionerà anche per attivare il disegno per gli effetti particellari di GML Visual).

Sintassi:

part_system_automatic_draw(ind, automatic);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to change.
automatic Booleano Whether automatic drawing is on (true) or not (false).

Ritorna:

N/A

Esempio:

part_system_automatic_draw(global.Sname, false);

Il codice qui sopra disattiva il disegno automatico per il sistema di particelle indicizzato nella variabile globale "Sname".

Indietro: Sistemi di particelle
Il prossimo: part_system_update
