aggiornamento del sistema parziale

Questa funzione farà avanzare il dato sistema di particelle di un passo nel tempo di gioco e può essere usata per simulare il comportamento del sistema di particelle quando l'aggiornamento automatico è disattivato, oppure può essere usata per far avanzare un sistema di particelle fino a un punto specifico in un solo passo. Per esempio, se avete un sistema di particelle per la neve, e volete che sembri che la neve stia cadendo dal momento in cui il giocatore entra in room allora potreste voler far avanzare le vostre particelle di neve per alcuni secondi nell'evento di creazione di un'istanza in modo che la prima volta che vengono disegnate, ci siano particelle su tutto lo schermo (l'esempio di codice qui sotto lo mostra). Potete anche usare questa funzione quando l'aggiornamento automatico del sistema è disattivato dalla funzione part_system_automatic_update() per spostare il sistema.

Sintassi:

part_system_update(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to update.

Ritorna:

N/A

Esempio:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

Il codice di cui sopra farà avanzare il sistema di particelle indicizzato nella variabile globale "RainSys" di 3 secondi.
