creare_particelle

Questa funzione è ideale per quegli effetti che non richiedono nessuna delle funzionalità offerte dagli emettitori di particelle (per esempio, per creare del fumo da un missile, o un semplice effetto di esplosione) poiché permette di creare rapidamente e facilmente delle particelle in qualsiasi posizione del gioco room. Si noti che è necessario aver creato il sistema di particelle e il tipo di particelle in precedenza perché questa funzione possa essere utilizzata.

Sintassi:

part_particles_create(ind, x, y, parttype, number);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Tipo di particella ID The index (type) of the particles to be created.
number Real The number of particles to create.

Ritorna:

N/A

Esempio:

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

Il codice di cui sopra controlla che il pulsante del mouse sia premuto e se ritorna true genera 5 particelle nella posizione del mouse.
