posizione del sistema di parti

Con questa funzione è possibile impostare la posizione di base per il sistema di particelle rispetto alla posizione (0,0) di room, il che significa che tutte le ulteriori funzioni di particelle relative a questo sistema saranno ora disegnate rispetto alla nuova posizione. Per impostazione predefinita questa posizione è sempre (0,0), ma in alcune circostanze molto speciali potresti desiderare di cambiarla in qualcos'altro.

NOTA: Questa funzione cambierà tutto all'interno del sistema di particelle, quindi se avete un emettitore in posizione (100,100) e poi impostate la posizione del sistema di particelle a (0,100), l'emettitore sarà ora disegnato a (100,200). Lo stesso vale se spostate il sistema e poi create l'emettitore, perché anche se lo create a (100,100) sarà disegnato a (100,200).

Sintassi:

part_system_position(ind, x, y);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to change.
x Real The new x coordinate of the particle system.
y Real The new y coordinate of the particle system.

Ritorna:

N/A

Esempio:

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

Il codice di cui sopra controllerà la pressione del pulsante del mouse e se ne rileva uno, il sistema di particelle indicizzato nella variabile globale "Sname" viene spostato nella posizione x/y del mouse
