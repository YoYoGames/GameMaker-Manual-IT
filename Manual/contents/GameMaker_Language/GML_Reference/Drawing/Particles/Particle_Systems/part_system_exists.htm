esiste un sistema parziale

Con questa funzione potete controllare se il dato sistema di particelle indicizzato esiste o meno nel gioco. Notate che se la variabile da controllare è una variabile non inizializzata (alla quale un sistema di particelle avrebbe altrimenti assegnato il suo indice) questo lancerà un errore.

Sintassi:

part_system_exists(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to check for.

Ritorna:

Boolean

Esempio:

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

Il codice di cui sopra controlla se il sistema di particelle a cui si fa riferimento nella variabile globale esiste e se non esiste viene creato.

Indietro: Sistemi di particelle
Il prossimo: part_system_create
