aggiornamento automatico del sistema parziale

Questa funzione controlla se GameMaker deve aggiornare il sistema di particelle automaticamente o meno, e normalmente non è necessario utilizzare questa funzione poiché il valore predefinito di true (l'aggiornamento automatico è attivo) è ciò che si desidera che accada. Tuttavia, per effetti speciali o per mettere in pausa il gioco, è possibile impostare questo valore su false e ciò impedirà l'esecuzione di qualsiasi aggiornamento sul sistema di particelle dato, a meno che non sia esplicitamente comandato dall'uso della funzione part_system_update() o si utilizzi nuovamente questa funzione per attivare gli aggiornamenti automatici. Nessun aggiornamento significa che le particelle create in questo sistema non cambieranno più posizione, colore o qualsiasi altro parametro e che anche gli emettitori cesseranno di funzionare, "congelando" il sistema nel punto esatto in cui l'aggiornamento automatico è stato impostato su false.

Una cosa da notare è che se state usando i semplici effetti creati dalle funzioni effect_create_above() o effect_create_below() allora puoi usare i valori 0 (per gli effetti sotto) o 1 (per gli effetti sopra) come indice del sistema di particelle e quindi attivare l'aggiornamento automatico anche per questi (questo funzionerà anche per attivare l'aggiornamento per gli effetti delle particelle di GML Visual).

Sintassi:

part_system_automatic_update(ind, automatic);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system to change.
automatic Booleano Whether automatic updating is on (true) o no (falso).

Ritorna:

N/A

Esempio:

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

Il codice qui sopra switch spegnerà gli aggiornamenti delle particelle se la variabile globale "Paused" è vera.
