parte_particelle_creare_colore

Questa funzione è ideale per quegli effetti che non richiedono nessuna delle funzionalità offerte dagli emettitori di particelle (ad esempio, per creare fumo da un missile, o un semplice effetto di esplosione) in quanto permette di creare rapidamente e facilmente particelle in qualsiasi posizione nel gioco room. Potete anche colorare le particelle "al volo" mentre vengono create con questa funzione, e questo colore sovrascrive il colore predefinito della particella, ma non fonde questo colore durante la vita della particella. Nota che devi aver creato il sistema di particelle e il tipo di particelle in precedenza perché questa funzione possa essere utilizzata.

Sintassi:

part_particles_create_colour(ind, x, y, parttype, colour, number);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Tipo di particella ID The index (type) of the particles to be created.
colour Colore The colour blending for the particles.
number Real The number of particles to create.

Ritorna:

N/A

Esempio:

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

Il codice di cui sopra genererà una singola particella ogni passo che l'istanza con il codice ha una velocità maggiore di uno. Queste particelle saranno di un colore casuale.
