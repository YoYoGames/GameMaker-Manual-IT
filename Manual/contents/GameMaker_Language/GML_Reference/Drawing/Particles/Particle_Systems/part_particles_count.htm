numero di particelle

Con questa funzione è possibile controllare se un sistema di particelle ha attualmente qualche particella creata in room, e restituirà anche il numero di esse.

Sintassi:

part_particles_count(ind);

Argomento Tipo Descrizione
ind ID del sistema di particelle The index of the particle system.

Ritorna:

Real

Esempio:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

Il codice di cui sopra controllerà il numero di particelle nel sistema di particelle locale indicizzato nella variabile "Sname" e se non ce ne sono, distruggerà il sistema e quindi l'istanza.

Indietro: Sistemi di particelle
