effetto_creare_sopra

Con questa funzione potete creare un semplice effetto sopra tutte le istanze del vostro room (in realtà viene creato ad una profondità di -100000). Se l'effetto è qualcosa di diverso da ef_rain o ef_snow allora potete definire una posizione x/y per creare l'effetto, e la dimensione può essere un valore di 0, 1, o 2, dove 0 è piccolo, 1 è medio e 2 è grande.

Vale la pena notare che questi effetti possono avere il loro disegno attivato e disattivato, così come il loro disegno messo in pausa, utilizzando le funzioni part_system_automatic_draw() e part_system_automatic_update() con il valore appropriato per l'indice del sistema di particelle (dove 0 è per gli effetti sotto e 1 è per gli effetti sopra).

Le costanti disponibili per i diversi tipi di particelle sono:

Tipo di effetto Costante
Costante Esempio Descrizione
ef_cloud An effect that creates random cloud particles of varying sizes
ef_ellipse An effect that creates expanding ellipses
ef_explosion An effect that creates expanding fading explosions
ef_firework An effect that creates multiple small particles to generate a firework explosion
ef_flare An effect that generates a brilliant point that flares up and fades out
ef_rain An effect that generates rain particles coming down from the top of the screen
ef_ring An effect that generates expanding and fading circles
ef_smoke An effect that generates little puffs of smoke
ef_smokeup An effect that creates a smoke plume that rises up the screen
ef_snow An effect that generates multiple snow particles falling down the screen
ef_spark An effect that generates a small spark
ef_star An effect that generates star particles

Sintassi:

effect_create_above(kind, x, y, size, colour);

Argomento Tipo Descrizione
kind Tipo di effetto Costante The kind of effect (use one of the constants listed above).
x Real The x positioning of the effect if relevant.
y Real The y positioning of the effect if relevant.
size Real The size of the effect.
colour Colore The colour of the effect.

Ritorna:

N/A

Esempio:

if health <= 0
{
    effect_create_above(ef_explosion, x, y, 1, c_yellow);
    instance_destroy();
}

Il codice qui sopra creerà un'esplosione media, gialla, sopra l'istanza e la distruggerà se la variabile "salute" sarà inferiore o uguale a 0.
