effetto_chiaro

Questo cancellerà tutte le particelle che sono attualmente visibili in room che sono state fatte usando effect_create_above() o effect_create_below().

Sintassi:

effect_clear();

Ritorna:

Esempio:

effect_clear();

Chiamando questo cancellerà il sito room da qualsiasi particella fatta con le funzioni effect_create_*.

Indietro: Particelle
Il prossimo: effect_create_below
