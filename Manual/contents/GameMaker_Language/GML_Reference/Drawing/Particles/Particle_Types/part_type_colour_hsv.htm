tipo_parte_colore_hsv

Con questa funzione è possibile impostare una gamma di tonalità, saturazione e valore per tutte le particelle del tipo dato. Fornite un valore minimo e un valore massimo per ciascuno dei tre componenti e le particelle create avranno un colore casuale basato sulla gamma di parametri data. In questo modo puoi creare particelle della stessa tonalità ma con saturazioni diverse, o di tonalità diverse ma con lo stesso valore (luminosità) ecc. Tutti i valori devono essere compresi tra 0 e 255.

Sintassi:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
hmin Real The minimum the final colour's hue component can be.
hmax Real The maximum the final colour's hue component can be.
smin Real The minimum the final colour's saturation component can be.
smax Real The maximum the final colour's saturation component can be.
vmin Real The minimum the final colour's value component can be.
vmax Real The maximum the final colour's value component can be.

Ritorna:

N/A

Esempio:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Il codice precedente imposta ogni particella emessa del tipo di particella indicizzata nella variabile globale "Stars" per avere colori e saturazioni diverse, ma lo stesso valore (luminosità).
