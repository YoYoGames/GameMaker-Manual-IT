tipo_parte_alfa1

Questa funzione è usata per impostare un tipo di particella per avere un singolo valore alfa (trasparenza) per la durata totale della vita di ogni singola particella, e questo può essere da 0 (trasparente) a 1 (opaco).

Sintassi:

part_type_alpha1(ind, alpha1);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
alpha1 Real The alpha of the particle.

Ritorna:

N/A

Esempio:

part_type_alpha1(global.Snow_Part, 0.5);

Il codice qui sopra imposterà tutte le particelle create del tipo di particella indicizzata nella variabile globale "Snow_Part" per avere un valore alfa di 0,5 (semi-trasparente).
