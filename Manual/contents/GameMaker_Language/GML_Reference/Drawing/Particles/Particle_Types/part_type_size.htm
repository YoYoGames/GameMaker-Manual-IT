tipo_parte_dimensione

Questa funzione è usata per determinare la dimensione della particella quando viene creata e può anche essere usata per far aumentare o diminuire la dimensione della particella nel corso della sua vita. I valori di dimensione minima e massima sono predefiniti a 1 (il che significa che la dimensione è lo stesso rapporto della particella di base dopo che la part_type_scale() scala è stato impostato) e se sono impostati allo stesso valore, la dimensione iniziale per tutte le particelle sarà la stessa, ma se impostati a valori diversi, ogni particella inizierà con una dimensione casuale all'interno dell'intervallo, per esempio se hai un minimo di 0,5 e un massimo di 2, tutte le particelle saranno create con una dimensione tra la metà e il doppio della scala di base.

Puoi anche impostare un valore di incremento che farà crescere (se è un numero positivo) o restringere (se è un numero negativo) la particella sprite nel corso della vita della particella, con la quantità data che viene aggiunta alla dimensione della particella ad ogni passo. Questo valore può essere un minimo di (+/-)0,01.

Infine, è possibile impostare il fattore "wiggle". Questo è un valore che sarà aggiunto o sottratto casualmente alla dimensione ad ogni passo della vita delle particelle. Ovviamente i valori più grandi sono più pronunciati di quelli più piccoli, e questo valore può anche essere negativo con un range massimo tra -20 e 20.

Sintassi:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
size_min Real The minimum size the particle can start at.
size_max Real The maximum size the particle can start at.
size_incr Real How much the particle should increase or decrease per step.
size_wiggle Real How much should randomly be added or subtracted from the particles size per step.

Ritorna:

N/A

Esempio:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

Il codice di cui sopra imposterà vari valori per le particelle, compresa la dimensione che è impostata sul doppio della dimensione quando ogni particella viene creata, e poi incrementerà la dimensione di 0,01 ogni passo con un wiggle di 0,1 anche.
