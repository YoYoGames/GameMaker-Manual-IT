tipo_parte_direzione

Questa funzione è usata per determinare la direzione della particella quando viene creata e può anche essere usata per far aumentare o diminuire la direzione della particella durante la sua vita. I valori di default della direzione minima e massima sono 0, il che significa che la particella si muoverà verso destra senza alcuna variazione, ma questi valori possono essere modificati per limitare la direzione del movimento seguendo le direzioni standard di GameMaker: 0 è destra, 90 è su, 180 è sinistra e 270 è giù. Se li impostate allo stesso valore, le particelle si muoveranno solo in una direzione.

Potete anche impostare un valore di incremento che aggiungerà (se è un numero positivo) o sottrarrà (se è un numero negativo) una quantità di gradi alla direzione durante la sua durata. Questo valore può essere un minimo di (+/-) 0,01.

Infine, è possibile impostare il fattore "wiggle". Questo è un valore che sarà aggiunto o sottratto casualmente alla direzione ad ogni passo della vita delle particelle. Ovviamente i valori più grandi sono più pronunciati di quelli più piccoli, e questo valore può anche essere negativo con un range massimo tra -20 e 20.

Sintassi:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
dir_min Real The minimum direction the particle can start at.
dir_max Real The maximum direction the particle can start at.
dir_incr Real How much the particle direction should increase or decrease per step.
dir_wiggle Real How much should randomly be added or subtracted from the particle's direction per step.

Ritorna:

N/A

Esempio:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Il codice qui sopra imposterà vari valori per le particelle, compresa la direzione che sarà un valore casuale tra 0 e 359 gradi. Salirà anche di 1 grado ad ogni passo e ha una quantità casuale aggiunta ad essa di qualsiasi valore compreso tra 0 e 20 ad ogni passo.
