tipo_parte_distruzione

Con questa funzione puoi rimuovere il tipo di particella specificato dal gioco. Quando usi questa funzione, tutte le particelle del tipo dato scompariranno da room e la particella stessa viene rimossa dalla memoria, quindi questa funzione dovrebbe essere usata solo quando non hai più bisogno della particella.

Sintassi:

part_type_destroy(ind);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to destroy.

Ritorna:

N/A

Esempio:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

Il codice qui sopra controlla un sistema di particelle per vedere se qualche particella è attualmente visibile in room e se non lo è, le particelle, il sistema e l'istanza vengono distrutti.
