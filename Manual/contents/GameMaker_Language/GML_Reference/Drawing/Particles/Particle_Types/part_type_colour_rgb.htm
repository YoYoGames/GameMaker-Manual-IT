tipo_parte_colore_rgb

Con questa funzione potete impostare il mix di colori rosso, verde e blu per tutte le particelle create del tipo dato. Fornisci un valore minimo e un valore massimo per ciascuno dei tre componenti e le particelle create avranno un colore casuale basato sull'intervallo di parametri dato. Tutti i valori devono essere compresi tra 0 e 255.

Sintassi:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Ritorna:

N/A

Esempio:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Il codice precedente imposta ogni particella emessa del tipo di particella indicizzata nella variabile globale "Blood_Part" per essere solo diverse tonalità di rosso.
