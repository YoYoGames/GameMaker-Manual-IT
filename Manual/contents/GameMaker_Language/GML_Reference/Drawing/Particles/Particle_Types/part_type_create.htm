creare un tipo di parte

Con questa funzione è possibile creare un nuovo tipo di particella e il valore di ritorno dovrebbe essere memorizzato in una variabile per l'uso in tutte le successive funzioni di particelle.

Sintassi:

part_type_create();

Ritorna:

Particle Type ID

Esempio:

mypart = part_type_create();

Questo creerà un nuovo tipo di particella, memorizzando il suo indice nella variabile "mypart".

Indietro: Tipi di particelle
Il prossimo: part_type_destroy
