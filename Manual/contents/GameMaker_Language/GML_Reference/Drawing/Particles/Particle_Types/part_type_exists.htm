tipo_parte_esiste

Con questa funzione potete controllare se il dato tipo di particella indicizzato esiste o meno nel gioco. Notate che se la variabile da controllare è una variabile non inizializzata (alla quale un tipo di particella avrebbe altrimenti assegnato il suo indice) questo lancerà un errore.

Sintassi:

part_type_exists(ind);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to check for.

Ritorna:

Boolean

Esempio:

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

Il codice qui sopra controlla se la variabile globale "p1" memorizza l'indice di un tipo di particella e se non lo fa ne crea uno.
