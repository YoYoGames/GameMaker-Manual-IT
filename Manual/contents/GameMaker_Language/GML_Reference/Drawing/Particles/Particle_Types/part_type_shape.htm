tipo_parte_forma

Con questa funzione è possibile impostare la forma sprite da utilizzare per il tipo di particella, con le seguenti costanti accettabili:

Costante di forma delle particelle
Costante Descrizione
pt_shape_pixel Un pixel 1x1. (Questa è l'impostazione predefinita).
pt_shape_disk Un cerchio pieno.
pt_shape_square Un quadrato riempito.
pt_shape_line Una linea orizzontale larga 8px.
pt_shape_star Una stella a cinque punte riempita.
pt_shape_circle Un cerchio delineato di 3px.
pt_shape_ring Un cerchio con un bagliore verso l'interno (sembra una bolla).
pt_shape_sphere Un cerchio con un bagliore verso l'esterno ' solido al centro, incandescente verso l'esterno.
pt_shape_flare Un punto fortemente incandescente (sembra una vera e propria stella nella notte).
pt_shape_spark Un effetto scintilla ' come una stella con più punti in dissolvenza.
pt_shape_explosion Una nuvola squadrata di fumo ' richiede più colori per assomigliare a un'esplosione.
pt_shape_cloud Una nuvola sottile, richiede un up scaling e particelle multiple per assomigliare a una nuvola.
pt_shape_smoke Una versione liscia dell'effetto esplosione. Usane più di uno per creare una nuvola di fumo.
pt_shape_snow Una forma generica di fiocco di neve.

Sintassi:

part_type_shape(ind, shape);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to set.
shape Costante di forma delle particelle The shape to choose.

Ritorna:

N/A

Esempio:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Il codice qui sopra imposta il tipo di particella indicizzato nella variabile globale "p1" alla forma di un quadrato, poi imposta varie altre proprietà della particella.
