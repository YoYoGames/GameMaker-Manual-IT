tipo_parte_sprite

Questa funzione può essere usata per impostare un tipo di particella per usare un sprite personalizzato dal gioco assets. È possibile selezionare il sprite e poi dire a GameMaker di utilizzare il sprite e animarlo o meno, e se si sceglie animato, allora si può anche scegliere di avere le sotto-immagini cambiate ad un intervallo tale che l'animazione inizierà e finirà all'inizio e alla fine della durata di ogni particella creata (in modo che se si ha una durata impostata su un minimo di 15 passi e un massimo di 30 passi, non importa quale valore abbia la durata l'animazione sarà fatta per adattarsi, con un funzionamento più veloce per le durate più brevi e più lento per quelle più lunghe). Puoi anche scegliere di scegliere una sotto-immagine casuale in modo che se il sprite non è animato, un'immagine diversa sarà scelta a caso per creare il sprite, o se la particella è animata inizierà l'animazione in un punto casuale.

Sintassi:

part_type_sprite(ind, sprite, animate, stretch, random);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
sprite Sprite Asset The index of the sprite to set the particle type to.
animate Booleano Whether to follow the sprite's animation normally (true) or not (false).
stretch Booleano Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Booleano Whether to choose a random sub-image (true) or not (false).

Ritorna:

N/A

Esempio:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Il codice qui sopra imposta il tipo di particella indicizzata nella variabile globale "p1" alla forma del sprite indicizzata in "spr_Coins", e poi imposta varie altre proprietà della particella.
