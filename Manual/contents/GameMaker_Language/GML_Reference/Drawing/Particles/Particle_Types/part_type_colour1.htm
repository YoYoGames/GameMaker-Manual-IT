tipo_parte_colore1

Questa funzione viene utilizzata per impostare un tipo di particella su un unico colore per la durata totale della vita di ogni singola particella.

Sintassi:

part_type_colour1(ind, colour1);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
colour1 Colore The single colour to make the particle type.

Ritorna:

N/A

Esempio:

part_type_colour1(global.Snow_Part, c_white);

Il codice qui sopra imposterà tutte le particelle create del tipo di particella indicizzata nella variabile globale "Snow_Part" per essere solo bianca.
