tipo_parte_chiaro

Con questa funzione puoi "resettare" una particella, riportando tutti i valori di ciascuna delle funzioni relative alla particella (vita, colore, alfa, orientamento ecc...) ai loro valori di default. Nota che questa funzione non rimuove dallo schermo le particelle attualmente visibili in room, per questo dovresti usare part_particles_clear().

Sintassi:

part_type_clear(ind);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to clear.

Ritorna:

N/A

Esempio:

part_type_clear(global.explode_part);

Il codice di cui sopra cancellerà il tipo di particella indicizzato nella variabile globale "explode_part" ai suoi valori predefiniti.
