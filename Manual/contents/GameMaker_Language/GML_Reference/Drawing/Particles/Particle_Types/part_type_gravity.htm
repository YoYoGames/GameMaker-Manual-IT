tipo_parte_gravità

Questa funzione imposterà la gravità che influenzerà ogni particella del tipo dato che viene creata. Il valore della forza di gravità viene aggiunto alla velocità della particella ad ogni passo e di solito è un valore piccolo come 0,5, mentre la direzione è la direzione della "forza di gravità" e segue le direzioni standard di GameMaker di 0° a destra, 90° in alto, 180° a sinistra e 270° in basso.

Sintassi:

part_type_gravity(ind, grav_amount, grav_direction);

Argomento Tipo Descrizione
ind Tipo di particella ID The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Ritorna:

N/A

Esempio:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Il codice di cui sopra imposterà vari valori di particelle tra cui la gravità che aggiungerà 0,2 alla velocità ad ogni passo con una direzione di 270º, quindi tirerà la particella "giù" verso il fondo dello schermo.
