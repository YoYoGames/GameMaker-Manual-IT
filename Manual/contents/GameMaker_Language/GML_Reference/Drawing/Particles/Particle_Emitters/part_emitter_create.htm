emettitore_parte_creato

Questa funzione viene utilizzata per creare un nuovo emettitore di particelle nel sistema di particelle indicato. La funzione restituisce il valore dell'indice del nuovo emettitore, che deve essere memorizzato in una variabile e utilizzato in tutte le altre funzioni che fanno riferimento all'emettitore; l'emettitore stesso deve essere distrutto quando non viene più utilizzato, per evitare perdite di memoria (ciò può essere ottenuto utilizzando le funzioni specifiche di distruzione dell'emettitore o distruggendo l'intero sistema di particelle a cui l'emettitore appartiene).

NOTA Gli ID degli emettitori restituiti da questa funzione vengono riutilizzati, ad esempio se si creano per la prima volta tre emettitori di particelle, questi avranno gli ID 0, 1 e 2. Se poi si distrugge il secondo emettitore (che ha ricevuto un ID 1) usando la funzione part_emitter_destroy e poi si crea un altro emettitore, quest'ultimo riutilizzerà l'ID 1. Poiché l'ID dell'emettitore 2 è in uso, l'emettitore successivo avrà un ID 3 e così via.

Sintassi:

part_emitter_create(ps);

Argomento Tipo Descrizione
ps ID del sistema di particelle The particle system to create the emitter in.

Restituzione:

Particle Emitter ID

Esempio:

p_emit1 = part_emitter_create(Sname);

Questo crea un nuovo emettitore di particelle e ne memorizza l'indice nella variabile "p_emit".

Indietro: Emettitori di particelle
Il prossimo: part_emitter_clear
