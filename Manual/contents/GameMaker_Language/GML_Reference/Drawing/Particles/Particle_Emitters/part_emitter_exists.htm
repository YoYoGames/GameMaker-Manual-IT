parte_emittente_esiste

Con questa funzione potete controllare se l'emettitore di particelle dato indicizzato esiste o meno nel sistema dato. Notate che se la variabile da controllare è una variabile non inizializzata (alla quale un emettitore di particelle avrebbe altrimenti assegnato il suo indice), questa funzione darà un errore.

Sintassi:

part_emitter_exists(ps, ind);

Argomento Tipo Descrizione
ps ID del sistema di particelle The particle system to check for an emitter.
ind ID emettitore di particelle The index of the emitter to search for.

Ritorna:

Boolean

Esempio:

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

Il codice di cui sopra controllerà l'emettitore indicizzato nella variabile "permit" e se esiste, farà scoppiare 30 particelle da esso.

Indietro: Emettitori di particelle
