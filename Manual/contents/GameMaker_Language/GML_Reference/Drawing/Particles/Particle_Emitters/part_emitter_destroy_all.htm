distruggi_parte_distruggi_tutto

Questa funzione rimuoverà tutti gli emettitori definiti dal sistema dato e li cancellerà dalla memoria (questo fermerà anche qualsiasi particella dall'essere prodotta dall'emettitore dato, ma NON li rimuove dal room). Questa funzione dovrebbe sempre essere chiamata quando gli emettitori non sono più necessari per il sistema per prevenire perdite di memoria ed errori.

Sintassi:

part_emitter_destroy_all( ps );

ArgomentoTipo Descrizione
ps The particle system to destroy all emitters from.

Ritorna:

Esempio:

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

Il codice qui sopra controlla la variabile globale incorporata "lives" e se è 0, distrugge tutti gli emettitori di particelle e poi cambia room.

Indietro: Emettitori di particelle
Il prossimo: part_emitter_exists
