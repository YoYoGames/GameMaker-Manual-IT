distruggi_emettitore_parte

Questa funzione rimuoverà l'emettitore specificato dal sistema dato e lo cancellerà dalla memoria (questo fermerà anche qualsiasi particella dall'essere prodotta dall'emettitore dato, ma NON lo rimuoverà dal room). Questa funzione dovrebbe sempre essere chiamata quando l'emettitore dato non è più necessario per il sistema per prevenire perdite di memoria ed errori.

Sintassi:

part_emitter_destroy( ps, ind );

Argomento Tipo Descrizione
ps ID del sistema di particelle The particle system to destroy the emitter from.
ind ID emettitore di particelle The index of the emitter to destroy.

Ritorna:

N/A

Esempio:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

Il codice di cui sopra controllerà se l'emettitore di particelle indicizzato nella variabile "p_emit" esiste nel sistema di particelle dato e se lo fa viene distrutto.

Indietro: Emettitori di particelle
Il prossimo: part_emitter_destroy_all
