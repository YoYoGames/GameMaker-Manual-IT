esplosione dell'emettitore

Questa funzione permette di impostare un emettitore per far scoppiare un tipo specifico di particelle ed è tipicamente usata negli allarmi e negli eventi di distruzione poiché è un codice una tantum che crea il numero di particelle specificato tutto in una volta seguendo la distribuzione, la forma e la posizione impostata dalla funzione part_emitter_region(). Se hai bisogno che le particelle appaiano ad ogni passo, dovresti usare la funzione part_emitter_stream() piuttosto che chiamare questa funzione ad ogni passo.

Sintassi:

part_emitter_burst(ps, ind, parttype, number);

Argomento Tipo Descrizione
ps ID del sistema di particelle The particle system that the emitter is in.
ind ID emettitore di particelle The index of the emitter to burst from.
parttype Tipo di particella ID The index (type) of the particles to be created.
number Real The number of particles to create.

Ritorna:

N/A

Esempio:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

Il codice di cui sopra farà scoppiare un numero casuale di particelle tra 30 e 60 dall'emettitore scelto.

Indietro: Emettitori di particelle
Il prossimo: part_emitter_stream
