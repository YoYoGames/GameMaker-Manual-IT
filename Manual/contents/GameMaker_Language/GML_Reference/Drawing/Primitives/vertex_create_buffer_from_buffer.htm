vertex_create_buffer_from_buffer

Come per la funzione vertex_create_buffer()questa funzione creerà un nuovo vertice buffer, solo che ora i dati dei vertici che memorizza sono copiati dal regolare buffer che è specificato come fonte. Il buffer creato è uno speciale buffer che è pre-formattato con il formato dei vertici per costruire primitive da usare con (per esempio) shaders. Questa funzione richiede che si fornisca il puntatore a una buffer regolare creata in precedenza, e un formato di vertice che deve essere applicato ai dati copiati.

Sintassi:

vertex_create_buffer_from_buffer(buffer, format);

Argomento Tipo Descrizione
buffer ID del buffer The buffer to create the vertex buffer from.
format Costante di tipo primitivo The primitive vertex format to use.

Ritorna:

Vertex Buffer ID

Esempio:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

Il codice di cui sopra creerà un nuovo formato di vertice e poi creerà un nuovo vertice buffer da un regolare buffer creato in precedenza, applicando il formato di vertice personalizzato ad esso.
