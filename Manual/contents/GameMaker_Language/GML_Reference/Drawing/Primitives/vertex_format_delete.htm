cancellazione del formato dei vertici

Questa funzione deve essere chiamata ogni volta che avete finito di usare qualsiasi formato vettoriale creato. Voi fornite il valore dell'ID del formato (come restituito dalla funzione vector_format_end()), e questa funzione libererà la memoria associata ad esso. Notate che se cercate di utilizzare nuovamente questo formato dopo aver chiamato questa funzione, otterrete un errore.

Sintassi:

vertex_format_delete(formatID);

Argomento Tipo Descrizione
formatID ID del formato dei vertici The vertex format to delete

Ritorna:

N/A

Esempio:

vertex_format_delete(myFormat);

Il codice di cui sopra rimuoverà dalla memoria il formato dei vertici creato nella variabile "myFormat".
