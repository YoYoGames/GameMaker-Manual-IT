vertex_argb

Questa funzione imposterà i valori ARGB per il vertice attualmente definito per la primitiva personalizzata. Si fornisce l'indirizzo buffer in cui scrivere i dati e i valori rosso, verde, blu e alfa che si desidera utilizzare come un singolo intero senza segno a 32 bit - campione alfa negli 8 bit più alti, seguito dal campione rosso, verde e infine dal campione blu negli 8 bit più bassi. Puoi usare la notazione esadecimale ($AARRGGBB) un numero reale o usare una qualsiasi delle funzioni di make_colour_*() per definire il valore del colore.

Sintassi:

vertex_argb(buffer, argb);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to write the information to.
argb Colore The ARGB value to set.

Ritorna:

N/A

Esempio:

vertex_argb(buff, $FFFFFFFF);

Il codice qui sopra imposterà i valori ARGB del vertice che viene definito su bianco.
