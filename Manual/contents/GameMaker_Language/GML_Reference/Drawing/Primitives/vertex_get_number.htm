vertex_get_number

Con questa funzione potete trovare il numero di singoli vertici definiti in qualsiasi vertice dato buffer.

Sintassi:

vertex_get_number(buffer);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The vertex buffer to check.

Ritorna:

Real

Esempio:

v_num = vertex_get_number(v_buffer);

Il codice di cui sopra memorizzerà il numero di vertici memorizzati nel vertice dato buffer in una variabile.

Indietro: Primitivi e formati di vertici
