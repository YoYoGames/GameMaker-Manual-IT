draw_primitive_begin_texture

Questa funzione deve essere chiamata prima di definire i vertici di una primitiva testurizzata. È necessario fornire il tipo di primitiva da utilizzare (vedere draw_primitive_begin() per maggiori informazioni) e l'id di un texture da utilizzare, che può essere un sprite o un'immagine di sfondo asset. Questo id asset può essere ottenuto dalle funzioni sprite_get_texture(), per esempio (usate -1 per nessun texture).

NOTA Perché un texture si ripeta deve essere una potenza di due in dimensioni, cioè: 32x32, 128x128, ecc...

Sintassi:

draw_primitive_begin_texture(kind, tex)

Argomento Tipo Descrizione
kind Costante di tipo primitivo The kind of primitive you are going to draw.
tex Puntatore di texture The texture to use with the primitive.

Ritorna:

N/A

Esempio:

draw_set_colour(c_white);
var _tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, _tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Il codice di cui sopra disegnerà una striscia di triangolo a 4 vertici (facendo un rettangolo) texturizzato con il texture tenuto nella variabile "tex", e l'intero texture sarà usato per coprire la primitiva completata.
