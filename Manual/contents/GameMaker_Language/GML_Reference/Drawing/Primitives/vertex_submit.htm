vertex_submit

Potete usare questa funzione per inviare il contenuto di un vertice buffer alla pipeline grafica per l'uso con un shader. Voi fornite l'indice buffer da usare, il tipo primitivo di base da usare (vedete le costanti sotto) e il texture che deve essere usato. Il tipo primitivo di base è usato solo per assegnare l'ordine in cui i vertici che avete memorizzato in buffer sono disegnati e collegati, ma i dati effettivi usati per ciascuno dei vertici saranno quelli che avete definito quando avete creato il vertice buffer.

NOTA: Questa funzione può essere usata solo nell'evento Draw.

Per un esempio visivo delle diverse primitive di base disponibili, vedi l'immagine qui sotto:

Costante Descrizione
pr_pointlist Una primitiva che consiste in una lista di punti.
pr_linelist Una primitiva formata da singole righe di una lista.
pr_linestrip Una primitiva costituita da una striscia consecutiva di linee.
pr_trianglelist Una primitiva composta da singoli triangoli in una lista.
pr_trianglestrip Una primitiva composta da una striscia consecutiva di triangoli.

Sintassi:

vertex_submit(buffer, primitive, texture);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to use.
primitive Costante di tipo primitivo The primitive base type.
texture Puntatore di texture The texture to use (or -1 for no texture).

Ritorna:

N/A

Esempio:

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

Il codice sopra presenterà il vertice buffer indicizzato nella variabile "buff" per il disegno con il shader "shader_prim", usando una lista di triangoli come primitiva di base e il texture del sprite per l'istanza che esegue il codice.
