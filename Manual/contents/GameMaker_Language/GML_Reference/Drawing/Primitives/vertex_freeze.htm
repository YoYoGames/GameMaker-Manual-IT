vertex_freeze

Questa funzione può essere usata per "congelare" un vertice buffer. Questo buffer diventa di sola lettura, il che significa che se avete bisogno di cambiarlo, dovrete cancellare l'intero buffer e ricrearlo. Un buffer congelato può essere sottoposto al shader più velocemente di un normale, dinamico buffer ed è raccomandato per quegli effetti che richiedono una primitiva personalizzata immutabile per la durata di un livello o del gioco.

La funzione restituisce 0 se congela con successo il vertice specificato buffer, tuttavia se fallisce per qualsiasi motivo, restituisce -1.

Sintassi:

vertex_freeze(buffer);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The vertex buffer to freeze.

Ritorna:

Real (0 for success, -1 for failure)

Esempio:

// Create event
vertex_freeze(vbuff);

Il codice di cui sopra congelerà il vertice dato buffer nell'evento Create, in modo che possa essere usato come "statico" buffer in altri eventi per un accesso più veloce.
