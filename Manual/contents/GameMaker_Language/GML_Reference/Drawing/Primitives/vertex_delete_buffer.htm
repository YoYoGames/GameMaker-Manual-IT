vertex_delete_buffer

Questa funzione può essere usata per rimuovere un vertice precedentemente creato buffer (vedere vertex_create_buffer()) dalla memoria di sistema. Questo rimuoverà anche tutti i dati dei vertici per qualsiasi primitiva personalizzata che conteneva.

Sintassi:

vertex_delete_buffer(buffer);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The vertex buffer to be removed.

Ritorna:

N/A

Esempio:

vertex_delete_buffer(v_buff);

Il codice qui sopra cancellerà dalla memoria il buffer indicizzato nella variabile "v_buff".

Indietro: Primitivi e formati di vertici
Il prossimo: vertex_begin
