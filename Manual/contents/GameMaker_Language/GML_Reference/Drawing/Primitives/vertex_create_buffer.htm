vertex_create_buffer

Con questa funzione potete creare un nuovo vertice buffer. Questa è una speciale crescita buffer creata da GameMaker che è pre-formattata per l'uso quando si costruiscono le primitive (per l'uso con shaders, per esempio). La funzione restituirà un indice per il buffer che dovrebbe poi essere usato in tutte le ulteriori chiamate ad esso.

Quando si usa un vertice buffer creato con questa funzione si chiama semplicemente vertex_begin() per iniziare ad assegnare i dati dei vertici ad esso per iniziare a definire la vostra primitiva personalizzata, che sarà poi tenuta nel buffer pronta per essere presentata al shader. Il buffer può essere riutilizzato quando necessario (a meno che non abbiate usato la vertex_freeze() ), con ogni chiamata di vertex_begin() cancellando i dati precedenti buffer pronti ad accettare i nuovi dati.

Sintassi:

vertex_create_buffer();

Ritorna:

Vertex Buffer ID

Esempio:

v_buff = vertex_create_buffer();

Il codice di cui sopra creerà un nuovo vertice buffer e memorizzerà il suo handle nella variabile "v_buff".
