vertex_get_buffer_size

Con questa funzione potete ottenere la dimensione del vertice dato buffer in byte.

Sintassi:

vertex_get_buffer_size(buffer);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to get the size of.

Ritorna:

Real

Esempio:

bufferBytes = vertex_get_buffer_size(buff);

Il codice precedente otterrà il numero di byte utilizzati dal vertice buffer dato e memorizzerà il valore nella variabile "bufferBytes".
