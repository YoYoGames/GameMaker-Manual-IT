vertex_format_end

Questa funzione deve essere chiamata dopo aver definito qualsiasi nuovo formato di vertice. Restituisce il nuovo formato "handle" (indice) che deve essere usato in tutte le altre funzioni di vertice che fanno riferimento a questo nuovo formato.

Sintassi:

vertex_format_end();

Ritorna:

Vertex Format ID

Esempio:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Il codice qui sopra creerà un nuovo formato di vertice con solo i valori di colore e posizione e poi memorizzerà l'id del formato nella variabile "my_format".
