vertex_create_buffer_ext

Come per la funzione vertex_create_buffer()questa funzione creerà un nuovo vertice buffer. Questo è uno speciale buffer creato da GameMaker che è pre-formattato per l'uso quando si costruiscono le primitive da usare con shaders. È possibile specificare una dimensione iniziale di partenza per il buffer (in byte) e restituirà un indice per il buffer che dovrebbe poi essere usato in tutte le ulteriori chiamate al buffer.

Sintassi:

vertex_create_buffer_ext(size);

ArgomentoTipo Descrizione
size The initial size of the buffer (in bytes).

Ritorna:

Vertex Buffer ID

Esempio:

v_buff = vertex_create_buffer_ext(1024 * 1024);

Il codice di cui sopra creerà un nuovo vertice buffer, inizialmente di 1MB di dimensione, e memorizzerà il suo handle nella variabile "v_buff".
