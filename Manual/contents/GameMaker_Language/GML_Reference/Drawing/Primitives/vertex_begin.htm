vertice_inizio

Con questa funzione si inizia la definizione di una primitiva personalizzata. Si assegna un buffer su cui scrivere la primitiva e il formato dei vertici da utilizzare (precedentemente definito con le funzioni di formato dei vertici). Definirete quindi i punti necessari per ogni vertice della primitiva prima di chiamare vertex_end() per finalizzare la creazione della primitiva.

Sintassi:

vertex_begin(buffer, format);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to be written to.
format ID del formato dei vertici The vertex format to use.

Ritorna:

N/A

Esempio:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

Il codice di cui sopra definirà un nuovo formato di vertice, creerà un nuovo buffer e inizierà il processo di definizione di una nuova primitiva.
