vertice_texcoord

Questa funzione imposterà le coordinate texture da usare per il vertice attualmente definito per la primitiva personalizzata. Si fornisce il buffer in cui scrivere i dati e la posizione UV all'interno del texture da utilizzare.

Sintassi:

vertex_texcoord(buffer, u, v);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Ritorna:

N/A

Esempio:

vertex_texcoord(buff, 0, 0);

Il codice di cui sopra imposterà i valori UV del vertice che viene definito.

Indietro: Primitivi e formati di vertici
