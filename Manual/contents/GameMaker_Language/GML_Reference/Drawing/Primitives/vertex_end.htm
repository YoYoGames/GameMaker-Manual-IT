fine del vertice

Con questa funzione si termina la costruzione della primitiva personalizzata. Una volta chiamato questo comando, la primitiva può essere usata nella funzione vertex_submit() per l'uso in un shader o si può congelare il buffer (rendendo il vertice buffer usato in sola lettura e molto più veloce) usando vertex_freeze().

Sintassi:

vertex_end(buffer);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to end writing to.

Ritorna:

N/A

Esempio:

vertex_end(buff);

Il codice qui sopra terminerà la definizione di una primitiva personalizzata.
