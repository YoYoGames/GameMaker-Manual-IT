disegnare_prima_fine

Questa funzione deve essere chiamata quando avete finito di definire i punti della vostra primitiva. Se non chiamate questa funzione, non verrà disegnato nulla, poiché questa funzione dice effettivamente a GameMaker che avete finito e che ora può disegnare la primitiva definita.

Sintassi:

draw_primitive_end()

Ritorna:

Esempio:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Il codice qui sopra disegnerà una striscia di triangolo a 4 vertici texturizzata con il texture tenuto nella variabile "tex".

Indietro: Primitivi e formati di vertici
Il prossimo: draw_vertex
