colore del vertice

Se il vostro formato di vertice definito prende un valore di colore, potete usare questa funzione per aggiungere quei dati al vertice definito per la primitiva corrente. La funzione ha bisogno di un buffer in cui memorizzare i dati e prenderà una costante di colore, o un valore esadecimale (usando il formato standard GameMaker di BGR, ad esempio: $FF0000 per il blu) così come un valore alfa da 0 (trasparente) a 1 (completamente opaco).

Sintassi:

vertex_colour(buffer, colour, alpha);

Argomento Tipo Descrizione
buffer Vertex Buffer ID The buffer to write the information to.
colour Colore The colour for this vertex (can be a constant or a hex value).
alpha Real The alpha value for the vertex (from 0 to 1).

Ritorna:

N/A

Esempio:

vertex_colour(b, c_white, 1);

Il codice qui sopra imposterà il vertice definito come bianco con un valore alfa di 1.
