disegnare_diventa_illuminazione

Questa funzione restituisce se l'illuminazione è abilitata (true) o no (false) per tutta la scena.

Sintassi:

draw_get_lighting()

Ritorna:

Esempio:

if !draw_get_lighting()
{
    draw_set_lighting(true);
}

Il codice qui sopra controllerà per vedere se l'illuminazione è abilitata per la scena, e se non lo è la abilita.
