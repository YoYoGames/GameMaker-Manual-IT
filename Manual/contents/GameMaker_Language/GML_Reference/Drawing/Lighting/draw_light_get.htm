disegnare_luce_get

Questa funzione otterrà i parametri di luce specificati come un array con i seguenti 6 elementi -

  • [0] = abilitato / disabilitato (true / false)
  • [1] = Costante del tipo di luce (vedi tabella sotto)
  • [2] = posizione x
  • [3] = posizione y
  • [4] = posizione z
  • [5] = raggio di luce (solo per le luci puntiformi, sarà 0 per le direzionali)
  • [6] = colore chiaro (un reale)

Il tipo di luce (elemento [1]) può essere una delle due costanti seguenti:

Tipo di luce costante
Costante Descrizione
lighttype_dir La luce è una luce direzionale
lighttype_point La luce è una luce puntiforme

Sintassi:

draw_light_get(ind);

Argomento Tipo Descrizione
ind Real The index number of the light (from 0 to 7)

Ritorna:

Array

Esempio:

light_a = draw_light_get(1);
if (light_a[5] < 200)
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Il codice di cui sopra otterrà i valori usati per definire la luce indicizzata come "1", poi controllerà il raggio e se è inferiore a 200 sarà aumentato e il raggio della luce impostato al nuovo valore.
