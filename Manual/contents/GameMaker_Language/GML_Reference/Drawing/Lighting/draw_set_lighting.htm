disegnare_impostare_illuminazione

Questa funzione è usata per abilitare tutti gli effetti di luce. L'impostazione predefinita è disabilitata (false).

Sintassi:

draw_set_lighting(enable);

Argomento Tipo Descrizione
enable Booleano Enable or disable all lighting (true o falso)

Ritorna:

N/A

Esempio:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Il codice di cui sopra abiliterà l'illuminazione per l'intera scena, poi definirà una luce bianca direzionale nello spazio room, e infine accenderà quella luce.
