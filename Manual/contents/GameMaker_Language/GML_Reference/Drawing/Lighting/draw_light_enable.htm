disegna_luce_abilita

Questa funzione è usata per attivare una luce definita. Quando si definisce una luce posizionale o direzionale è necessario assegnarle un numero di indice che viene poi utilizzato da questa funzione per switch accendere o spegnere la luce. L'impostazione predefinita è disabilitata (false).

NOTA: Ci sono solo 8 luci hardware disponibili, quindi solo 8 luci definite possono essere abilitate in qualsiasi momento (anche se se ne possono definire di più).

Sintassi:

draw_light_enable(ind, enable);

Argomento Tipo Descrizione
ind Real The index number of the light (from 0 to 7)
enable Booleano Enable or disable all lighting (true o false)

Ritorna:

N/A

Esempio:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Il codice di cui sopra abiliterà l'illuminazione per l'intera scena, poi definirà una luce bianca direzionale nello spazio room, e infine accenderà quella luce.

Indietro: Illuminazione
Il prossimo: draw_set_lighting
