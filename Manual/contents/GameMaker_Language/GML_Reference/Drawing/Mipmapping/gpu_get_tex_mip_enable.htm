gpu_get_tex_mip_enable

Con questa funzione si può ottenere se il mipmapping è disattivato, attivato per tutto o attivato solo per i gruppi texture selezionati nel Texture Group Manager. La funzione restituirà una delle costanti elencate di seguito, con l'impostazione predefinita che è mip_markedonly.

Sintassi:

gpu_get_tex_mip_enable();

Ritorna:

Constant

Costante Mipmapping
Costante Descrizione
mip_off Il mipmapping è disabilitato.
mip_on Mipmapping per tutti i textures è abilitato.
mip_markedonly Mipmapping è abilitato per textures che lo ha abilitato nelle opzioni di Texture Group (default).

Esempio:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

Il codice qui sopra controllerà per vedere se il mipmapping è abilitato e se non lo è, lo abiliterà.
