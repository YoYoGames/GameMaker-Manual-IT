gpu_get_tex_min_mip

Con questa funzione potete ottenere il livello minimo di mipmap che è attualmente usato, dove un valore di 0 è la risoluzione più alta, 1 è usare la prima mipmap, 2 è la seconda ecc.

Sintassi:

gpu_get_tex_min_mip();

Ritorna:

(default: 0)

Esempio:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

Il codice di cui sopra controllerà il livello minimo corrente di mipmap e se non è 0 viene impostato a 0.
