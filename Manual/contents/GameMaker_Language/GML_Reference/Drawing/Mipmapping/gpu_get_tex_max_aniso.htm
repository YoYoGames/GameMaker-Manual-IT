gpu_get_tex_max_aniso

Con questa funzione si può ottenere il livello massimo attuale di anisotropia quando si usa la modalità filtro tf_anisotropic (vedi gpu_get_tex_mip_filter() per maggiori informazioni). Il valore restituito sarà compreso tra 1 e 16.

Sintassi:

gpu_get_tex_max_aniso();

Ritorna:

(default: 16)

Esempio:

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

Il codice di cui sopra controllerà il livello massimo corrente di filtraggio anisotropico e se non è 8 viene impostato a 8.
