Mipmapping

Questa sezione contiene tutte le funzioni relative all'uso delle funzioni di mipmapping. Prima di usare queste funzioni dovreste aver abilitato il mipmapping per le pagine richieste texture nel Texture Group Editor e/o abilitato il mipmapping per quelle sprites che sono state impostate per usare solo una singola pagina texture dalle General Game Options. L'immagine qui sotto mostra la differenza che il mipmapping può fare durante il rendering del tuo progetto:

Una volta che avete abilitato il mipmapping per il progetto o una pagina texture, potete usare le seguenti funzioni per ottenere e impostare valori che cambieranno l'aspetto del vostro progetto quando viene eseguito.

Le funzioni elencate di seguito possono essere utilizzate per ottenere i diversi valori associati al mipmapping in GameMaker:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

Le funzioni elencate di seguito possono essere utilizzate per impostare i diversi valori associati al mipmapping in GameMaker:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
