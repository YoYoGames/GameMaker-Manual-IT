gpu_get_tex_mip_enable_ext

Con questa funzione potete ottenere se il mipmapping è disattivato, attivato per tutto o attivato solo per i gruppi texture selezionati nel Texture Group Manager su un campionatore shader. Si fornisce il valore dell'indice per il campionatore shader (come restituito dalla funzione shader_get_sampler_index()e la funzione restituirà una delle costanti elencate di seguito.

Sintassi:

gpu_get_tex_mip_enable_ext(sampler_index);

Argomento Tipo Descrizione
sampler_index Manico Shader Sampler The index of the shader sampler to get

Ritorna:

Constant

Costante Descrizione
mip_off Il mipmapping è disabilitato.
mip_on Mipmapping per tutti i textures è abilitato.
mip_markedonly Mipmapping è abilitato per textures che lo ha abilitato nelle opzioni di Texture Group.

Esempio:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Il codice qui sopra abilita il mipmapping per il dato campionatore shader texture se non è già stato abilitato.
