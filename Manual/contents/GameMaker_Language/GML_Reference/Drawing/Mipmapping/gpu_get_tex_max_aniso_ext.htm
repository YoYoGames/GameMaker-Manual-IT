gpu_get_tex_max_aniso_ext

Con questa funzione si può ottenere il livello massimo attuale di anisotropia quando si usa la modalità filtro tf_anisotropic (vedi gpu_get_tex_mip_filter() per maggiori informazioni) su un campionatore shader. Voi fornite il valore dell'indice per il campionatore shader (come restituito dalla funzione shader_get_sampler_index()e la funzione restituirà un valore compreso tra 1 e 16.

Sintassi:

gpu_get_tex_max_aniso_ext(sampler_index);

ArgomentoTipo Descrizione
sampler_index The index of the shader sampler to get

Ritorna:

(default: 16)

Esempio:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8
{
    gpu_set_tex_max_aniso_ext(_sampleIndex, 8);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Il codice qui sopra imposta il livello massimo di anisotropia a 8 per il dato campionatore shader texture se non è già stato impostato a 8.
