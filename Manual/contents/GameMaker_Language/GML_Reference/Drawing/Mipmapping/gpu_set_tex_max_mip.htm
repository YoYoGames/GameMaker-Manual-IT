gpu_set_tex_max_mip

Con questa funzione puoi impostare il livello massimo di mipmap attualmente impostato che deve essere usato, dove un valore di 0 è la risoluzione più alta, 1 è usare la prima mipmap, 2 la seconda ecc. Nota che questo può essere molto utile per evitare artefatti di bleeding quando si renderizza textures, per esempio, impostando il bordo della pagina texture a 8px e poi impostando il livello massimo di mipmap a 3 ti assicurerà di non avere problemi di bleeding a distanze di rendering maggiori.

Sintassi:

gpu_set_tex_max_mip(maxmip);

ArgomentoTipo Descrizione
maxmip The maximum mipmap level (default: 16)

Ritorna:

Esempio:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

Il codice di cui sopra controllerà il livello massimo di mipmap corrente e se non è 4 viene impostato a 4.
