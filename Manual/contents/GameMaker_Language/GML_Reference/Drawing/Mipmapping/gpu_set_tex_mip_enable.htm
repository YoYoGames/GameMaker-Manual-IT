gpu_set_tex_mip_enable

Con questa funzione potete cambiare se il mipmapping è disattivato, attivato per tutto o attivato solo per i gruppi texture selezionati nel Texture Group Manager. La funzione richiede una delle costanti elencate di seguito.

Costante Descrizione
mip_off Il mipmapping è disabilitato.
mip_on Mipmapping per tutti i textures è abilitato.
mip_markedonly Mipmapping è abilitato per textures che lo ha abilitato nelle opzioni di Texture Group (default).

Sintassi:

gpu_set_tex_mip_enable(setting);

Argomento Tipo Descrizione
setting Costante Mipmapping The mipmap setting (a constant, default: mip_markedonly)

Ritorna:

N/A

Esempio:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

Il codice qui sopra controllerà se il mipmapping è abilitato e se non lo è, lo abiliterà.
