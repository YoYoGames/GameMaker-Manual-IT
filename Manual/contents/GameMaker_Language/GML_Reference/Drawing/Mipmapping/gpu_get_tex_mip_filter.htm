gpu_get_tex_mip_filter

Con questa funzione potete ottenere la modalità corrente del filtro mip. Il valore di modalità restituito dalla funzione sarà una delle costanti elencate di seguito.

Sintassi:

gpu_get_tex_mip_filter();

Ritorna:

Mipmapping Filter Constant (listed below):

Costante del filtro Mipmapping
Costante Descrizione
tf_point Questo significa che il blending tra i livelli di mipmap è disabilitato, il che può causare transizioni visibili su texture, ma dà le migliori prestazioni.
tf_linear Questo significa che il blending tra i livelli di mipmap è abilitato (questo è anche conosciuto come filtraggio trilineare), che appiana le transizioni di texture, ma darà un piccolo colpo alle prestazioni.
tf_anisotropic Questo significa che il filtraggio anisotropico è abilitato, che migliora notevolmente la qualità della transizione texture e può ridurre la sfocatura visibile con altre modalità di filtraggio, ma ha il maggior impatto sulle prestazioni.

Esempio:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

Il codice di cui sopra controlla la tastiera e se il tasto specificato è premuto, otterrà il filtro mip corrente e passerà il valore a quello successivo, scorrendo le diverse modalità.
