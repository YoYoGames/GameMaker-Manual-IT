gpu_set_tex_mip_filter

Con questa funzione potete impostare la modalità corrente del filtro mip su uno dei tre tipi supportati. Si dà la costante che si riferisce alla modalità di filtraggio mip che si richiede, dove si può scegliere tra filtraggio puntuale (impostazione predefinita, cioè nessun filtraggio), filtraggio lineare (questo è anche conosciuto come filtraggio trilineare) o filtraggio aniostropico (vedere la tabella delle costanti sotto).

Costante Descrizione
tf_point Questo disabiliterà il filtraggio tra i livelli di mipmap, che può causare transizioni visibili su texture, ma dà le migliori prestazioni.
tf_linear Questo abiliterà il filtraggio lineare tra i livelli di mipmap (questo è anche conosciuto come filtraggio trilineare), che smussa le transizioni di texture, ma darà un piccolo colpo alle prestazioni.
tf_anisotropic Questo abiliterà il filtraggio anisotropico tra i livelli mipmap, che migliora notevolmente la qualità della transizione texture e può ridurre la sfocatura visibile con altre modalità di filtraggio, ma ha il più alto impatto sulle prestazioni.

Sintassi:

gpu_set_tex_mip_filter(filter);

Argomento Tipo Descrizione
filter Costante del filtro Mipmapping The mip filter mode to use (a constant, default: tf_point).

Ritorna:

N/A

Esempio:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

Il codice di cui sopra controlla la tastiera e se il tasto specificato è premuto, otterrà il filtro mip corrente e passerà il valore a quello successivo, scorrendo le diverse modalità.
