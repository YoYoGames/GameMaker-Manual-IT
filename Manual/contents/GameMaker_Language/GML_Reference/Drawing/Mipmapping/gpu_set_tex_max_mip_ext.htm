gpu_set_tex_max_mip_ext

Con questa funzione potete ottenere il livello massimo di mipmap attualmente impostato che deve essere usato per un dato campionatore shader. Si fornisce il valore dell'indice per il campionatore shader (come restituito dalla funzione shader_get_sampler_index()e la funzione restituirà l'attuale livello massimo di mipmap, dove un valore di 0 è la risoluzione più alta, 1 è usare la prima mipmap, 2 la seconda ecc. Notate che questo può essere molto utile per evitare artefatti di bleeding quando si renderizza textures, per esempio, impostando il bordo della pagina texture a 8px e poi impostando il livello massimo di mipmap a 3 vi assicurerà di non avere problemi di bleeding a distanze di rendering maggiori.

Sintassi:

gpu_set_tex_max_mip_ext(sampler_index, maxmip);

ArgomentoTipo Descrizione
sampler_index The index of the shader sampler to get
maxmip The maximum mipmap level value to use (default: 16)

Ritorna:

Esempio:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Il codice qui sopra imposta il livello massimo di mipmap a 4 per il dato campionatore shader texture se non è già stato impostato a 4.
