gpu_set_tex_mip_bias

Con questa funzione è possibile impostare il valore di bias della mipmap, dove un valore di 0 equivale a nessun bias, 1 equivale alla prima mipmap, 2 equivale alla seconda mipmap ecc. Questo controlla la velocità con cui la mipmap viene scambiata e generalmente renderà textures più sfocato quanto più alto è il valore e quanto maggiore è la "distanza" che viene vista. Si noti che questa funzione può assumere anche valori negativi, nel qual caso il rendering di textures sarà più nitido su una distanza maggiore quanto più basso è il valore.

Sintassi:

gpu_set_tex_mip_bias(bias);

ArgomentoTipo Descrizione
bias The mipmap bias value to use (default: 0)

Ritorna:

Esempio:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

Il codice di cui sopra controllerà il bias di mipmap corrente e se non è 0 viene impostato a 0.
