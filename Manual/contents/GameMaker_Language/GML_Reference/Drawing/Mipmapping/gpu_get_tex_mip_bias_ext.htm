gpu_get_tex_mip_bias_ext

Con questa funzione potete recuperare il valore di bias di mipmap per un dato campionatore shader. Si fornisce il valore dell'indice per il campionatore shader (come restituito dalla funzione shader_get_sampler_index()e la funzione restituirà un valore di 0,0 per nessun bias, o un valore maggiore dove 1 è uguale alla prima mipmap, 2 è uguale alla seconda mipmap ecc. Questo controlla la velocità con cui la mappa mip viene scambiata e generalmente renderà il shader textures più sfocato quanto più alto è il valore e quanto più grande è la "distanza" che viene vista. Si noti che può restituire anche valori negativi, nel qual caso shader textures sarà più nitido su una distanza maggiore quanto più basso è il valore.

Sintassi:

gpu_get_tex_mip_bias_ext(sampler_index);

ArgomentoTipo Descrizione
sampler_index The index of the shader sampler to get

Ritorna:

(default: 0)

Esempio:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Il codice qui sopra imposta il bias del filtro mip a 0 per il dato campionatore shader texture se non è già stato impostato a 0.
