disegna_testo_ext_colore

Questa funzione è una combinazione della funzione base draw_text() con la funzione draw_text_ext() e draw_text_colour() che permette di definire i colori del gradiente per il testo, così come l'interlinea e la larghezza massima per linea, tutto insieme.

NOTA La miscelazione dei gradienti non è disponibile per l'obiettivo HTML5 a meno che WebGL sia abilitato, anche se è ancora possibile impostare i colori di miscelazione e si fonderà il font con il primo colore dato. Tuttavia tutte le miscelazioni in questo modo creano un duplicato di font che viene poi memorizzato in cache e utilizzato quando richiesto, il che è tutt'altro che ottimale e se si utilizzano più cambi di colore rallenterà le prestazioni del gioco. È possibile impostare la dimensione di font cache per cercare di limitare questo fenomeno, se necessario, utilizzando la funzione font_set_cache_size().

Sintassi:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Argomento Tipo Descrizione
x Real The x coordinate of the drawn string.
y Real The y coordinate of the drawn string.
string Stringa The string to draw.
sep Real The distance in pixels between lines of text.
w Real The maximum width in pixels of the string before a line break.
c1 Colore The colour for the top left of the drawn text.
c2 Colore The colour for the top right of the drawn text.
c3 Colore The colour for the bottom right of the drawn text.
c4 Colore The colour for the bottom left of the drawn text.
alpha Real The alpha for the text.

Ritorna:

N/A

Esempio:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

Il codice di cui sopra disegnerà qualsiasi testo digitato dall'utente sulla tastiera, dividendolo su nuove linee ogni volta che la lunghezza di string per quella linea supera i 300 pixel. il codice manterrà anche una separazione di 5 pixel tra le linee se ciò dovesse accadere. Ogni linea del testo sarà colorata usando un gradiente blu, con il blu chiaro in alto e il blu scuro in basso.
