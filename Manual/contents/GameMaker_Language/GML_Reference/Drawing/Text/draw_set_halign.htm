disegnare_set_align

Questa funzione è usata per allineare il testo lungo l'asse orizzontale e cambiare l'allineamento orizzontale cambierà la posizione e la direzione in cui tutto il testo successivo è disegnato con il valore predefinito che è fa_left. Sono accettate le seguenti costanti:

Costante Allineamento
fa_left
fa_center
fa_right

Sintassi:

draw_set_halign(halign);

Argomento Tipo Descrizione
halign Costante di allineamento orizzontale Horizontal alignment constant (from the table above).

Ritorna:

N/A

Esempio:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

Il codice qui sopra disegnerà due strings sulla stessa linea, con il punteggio allineato a sinistra e la salute allineata a destra.
