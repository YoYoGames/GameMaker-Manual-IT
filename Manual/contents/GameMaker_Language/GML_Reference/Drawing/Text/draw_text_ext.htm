disegnare il testo

Questa funzione disegnerà il testo in modo simile a draw_text() solo che ora è possibile impostare lo spazio tra ogni riga di testo - se il testo occupa più di una riga - e limitare la larghezza (in pixel) del string per linea in modo che se qualsiasi linea dovesse superare questo valore, GameMaker dividerà automaticamente il testo alla linea successiva al più vicino spazio bianco disponibile (se il testo non ha spazi bianchi allora supererà questo massimo larghezza massima). Notate che qualsiasi spazio bianco posto all'inizio di string sarà rimosso prima di essere analizzato per il disegno a causa di questo. Notate anche che un valore di -1 per l'argomento di separazione delle linee avrà come impostazione predefinita una separazione basata sulla altezza del carattere "M" nel font scelto.

Sintassi:

draw_text_ext(x, y, string, sep, w);

ArgomentoTipo Descrizione
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Ritorna:

Esempio:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Il codice di cui sopra disegnerà qualsiasi testo digitato dall'utente sulla tastiera, dividendolo su nuove linee ogni volta che la lunghezza di string per quella linea supera i 300 pixel. il codice manterrà anche una separazione di 3 pixel tra le linee se ciò dovesse accadere.
