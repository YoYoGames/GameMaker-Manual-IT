disegnare_set_font

Questa funzione imposterà il font da usare per tutti i successivi disegni di testo. Questo font deve essere stato aggiunto nelle risorse di font del gioco o essere stato creato usando il metodo font_add() o font_add_sprite().

Sintassi:

draw_set_font(font);

ArgomentoTipo Descrizione
font The name of the font to use.

Ritorna:

Esempio:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

Il codice qui sopra disegnerà il testo dato usando il font indicizzato nella variabile "fnt_Game" e colorato di blu.
