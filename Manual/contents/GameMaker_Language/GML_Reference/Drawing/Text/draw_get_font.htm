disegnare_get_font

Questa funzione otterrà il font attualmente assegnato per disegnare il testo. La funzione restituirà -1 se nessun font è impostato, o il valore ID (un intero positivo) per il font asset assegnato.

Sintassi:

draw_get_font();

Ritorna:

Font Asset or -1

Esempio:

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

Il codice qui sopra prende il font attualmente applicato ed esegue una dichiarazione switch su di esso, applicando un diverso valore di offset Y a seconda del font. Poi usa quel valore di offset mentre disegna del testo.
