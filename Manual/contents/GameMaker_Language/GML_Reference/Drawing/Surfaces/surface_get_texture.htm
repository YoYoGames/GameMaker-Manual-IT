superficie_getta_tessuto

Questa funzione restituisce un puntatore speciale per la superficie texture. Questo valore può essere utilizzato in altre funzioni di disegno, in particolare nelle funzioni 3D generali e in alcune primitive 2D.

NOTA Quando si lavora con le superfici, c'è la possibilità che queste cessino di esistere in qualsiasi momento, perché sono memorizzate nella memoria di texture. È consigliabile verificare SEMPRE l'esistenza di una superficie utilizzando il comando surface_exists() prima di farvi riferimento direttamente.

Sintassi:

surface_get_texture(surface_id);

Argomento Tipo Descrizione
surface_id ID superficie The ID of the surface to get the width of.

Restituzione:

Texture

Esempio:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

Il codice precedente disegna una striscia di triangolo a 4 vertici con la texture texture contenuta nella variabile "tex", a sua volta presa da una superficie creata in precedenza.
