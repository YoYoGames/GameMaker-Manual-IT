superficie_set_target

Con questa funzione impostate tutti gli ulteriori disegni sulla superficie di destinazione piuttosto che sullo schermo e in questo modo potete dire a GameMaker di disegnare solo cose specifiche sulla superficie specificata. Notate che se non chiamate surface_reset_target() dopo che avete finito, niente sarà disegnato sullo schermo poiché tutti i disegni successivi (anche in altre istanze) saranno fatti sulla superficie. Dovreste anche rendervi conto che nulla sarà visto se la superficie stessa non è disegnata sullo schermo nell'evento draw di un'istanza. Potete anche controllare il valore di ritorno di questa funzione come strumento di debug per verificare se il target della superficie è stato impostato o meno, con un valore di ritorno di 0 che rappresenta un fallimento nell'impostare il target e qualsiasi altro valore positivo che rappresenta un successo.

Una cosa che va notata è che le superfici sono impilate, quindi non si può saltare da un obiettivo all'altro e poi tornare al normale obiettivo di disegno alla fine, ma piuttosto bisogna aprire e chiudere gli obiettivi di rendering. Per esempio, questo non funzionerà correttamente:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

Invece, dovete reimpostare l'obiettivo per ciascuna delle superfici che impostate, proprio come dovete usare le parentesi di apertura e di chiusura {} per i blocchi di codice. Quindi quanto sopra dovrebbe essere scritto come questo:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_reset_target();
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

o come questo:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();
surface_reset_target();

NOTA: Quando si lavora con le superfici c'è la possibilità che esse possano cessare di esistere in qualsiasi momento perché sono memorizzate in texture memory. Dovreste SEMPRE controllare che una superficie esista usando surface_exists() prima di farvi riferimento direttamente.

Sintassi:

surface_set_target(surface_id);

ArgomentoTipo Descrizione
surface_id The ID of the surface to set as the drawing target.

Ritorna:

Esempio:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Il codice qui sopra controllerà per vedere quale vista è attualmente disegnata, e se è view[0] imposta il draw target su una superficie e disegna tutte le istanze di object "obj_Effect" prima di resettare nuovamente il draw target. Se la vista non è view[0], la superficie viene disegnata sullo schermo.
