superficie_reset_target

Con questa funzione resettate tutti gli ulteriori disegni dalla superficie di destinazione allo schermo. Notate che per iniziare a disegnare su una superficie dovete prima aver chiamato la funzione surface_set_target() e poi questa dopo aver finito, per ogni destinazione di superficie che avete impostato, altrimenti non verrà disegnato nulla sullo schermo, poiché tutti gli ulteriori disegni (anche in altre istanze) verranno fatti sulla superficie. Dovreste anche rendervi conto che non si vedrà nulla se la superficie stessa non è disegnata sullo schermo nell'evento draw di un'istanza.

NOTA: se non avete precedentemente impostato un obiettivo di rendering con la funzione surface_set_target()l'uso di questa funzione terminerà silenziosamente (cioè senza alcun messaggio di errore) tutte le ulteriori esecuzioni di codice per l'evento.

Sintassi:

surface_reset_target();

Ritorna:

Esempio:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Il codice qui sopra controllerà per vedere quale vista è attualmente disegnata, e se è view[0] imposta il draw target su una superficie e disegna tutte le istanze di object "obj_Effect" prima di resettare nuovamente il draw target. Se la vista non è view[0], la superficie viene disegnata sullo schermo.
