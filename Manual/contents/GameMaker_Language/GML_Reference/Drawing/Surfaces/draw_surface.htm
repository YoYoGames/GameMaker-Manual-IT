disegnare_superficie

Con questa funzione potete disegnare una superficie in una data posizione all'interno di room, con l'angolo superiore sinistro della superficie che viene disegnato nella posizione x/y specificata.

Sintassi:

draw_surface(id, x, y);

Argomento Tipo Descrizione
id ID superficie The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.

Ritorna:

N/A

Esempio:

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

Il codice di cui sopra disegna la superficie indicizzata in "surf" nella stessa posizione della vista della telecamera[0].
