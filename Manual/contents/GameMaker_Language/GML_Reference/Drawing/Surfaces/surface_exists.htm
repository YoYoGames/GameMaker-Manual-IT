superficie_esiste

Questa funzione è essenziale quando si lavora con le superfici a causa della loro natura volatile. Le superfici sono sempre tenute in texture memoria, il che significa che possono essere distrutte da un momento all'altro (per esempio, quando si avvia uno screensaver su windows, o quando si minimizza su un dispositivo Android ), quindi si dovrebbe sempre controllare che una superficie esista prima di fare qualcosa con essa (questo include disegnarla sullo schermo). Il codice di esempio qui sotto mostra un uso tipico di questo comando nell'evento draw di un'istanza per controllare una superficie e ricrearla se è stata rimossa (si noti che la superficie sarà stata originariamente creata nell'evento create di object).

Sintassi:

surface_exists(surface_id);

ArgomentoTipo Descrizione
surface_id The ID of the surface to check.

Ritorna:

Esempio:

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
}
else
{
    if view_current = 1
    {
        draw_surface(surf,0,0);
    }
}

Il codice qui sopra controllerà per vedere se una superficie indicizzata nella variabile "surf" esiste, e se non esiste, la ricreerà. Se esiste, controlla per vedere quale vista è attualmente disegnata e se è la vista[1], disegna la superficie.
