disegna_superficie_stretta_ext

Questa funzione fa esattamente la stessa cosa della funzione draw_surface_stretched() con l'aggiunta della capacità di impostare la miscelazione dei colori e il valore alfa per la superficie quando viene disegnata (simile alla funzione draw_surface_ext()).

Sintassi:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Argomento Tipo Descrizione
id ID superficie The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
w Real The width at which to draw the surface.
h Real The height at which to draw the surface.
colour Colore The colour with which to colour the surface.
alpha Real The alpha with which to blend the surface.

Ritorna:

N/A

Esempio:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Questo disegnerà la superficie data con il suo angolo sinistro nella posizione x/y dell'istanza e sarà allungata per occupare un'area di 200x200 pixel senza fusione, ma con una trasparenza parziale.
