disegnare_superficie_ext

Questa funzione disegnerà la superficie data come nella funzione draw_surface() ma con opzioni aggiuntive per cambiare la scala, la sfumatura, la rotazione e l'alfa della superficie disegnata. Cambiare questi valori non modifica in alcun modo la risorsa (solo il modo in cui viene disegnata).

Sintassi:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Argomento Tipo Descrizione
id ID superficie The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
xscale Real The horizontal scale.
yscale Real The vertical scale.
rot Real The rotation or angle to draw the surface.
col Colore The colour with which to blend the surface.
alpha Real The alpha transparency for drawing the surface.

Ritorna:

N/A

Esempio:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

Il codice qui sopra disegnerà una superficie indicizzata nella variabile "surf" alla posizione (0,0) in room, con il doppio della scala originale, rosso sfumato e semi trasparente.
