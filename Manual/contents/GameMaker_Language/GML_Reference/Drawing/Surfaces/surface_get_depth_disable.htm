superficie_get_depth_disable

Questa funzione controlla se la generazione automatica della profondità buffer per le superfici è abilitata. Normalmente tutte le superfici hanno profondità buffers così se disegnate 3D objects su di esse allora le ordinerà correttamente in base alla profondità, tuttavia allocare la profondità buffers essenzialmente raddoppia la dimensione delle superfici, il che potrebbe essere un overhead eccessivo e non necessario specialmente se il vostro gioco è molto intensivo di memoria o prevalentemente 2D. In questi casi potete controllare la cosa usando questa funzione e disabilitare la profondità buffer per le superfici se necessario usando la funzione surface_depth_disable().

Sintassi:

surface_get_depth_disable();

Ritorna:

Esempio:

if (!surface_exists(surf))
{
    if surface_get_depth_disable() == false
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

Il codice qui sopra controllerà per vedere se la superficie data esiste, e se non esiste, allora controlla lo stato attuale della profondità della superficie buffer e se è abilitata, la disabiliterà invece, prima di creare finalmente la superficie.
