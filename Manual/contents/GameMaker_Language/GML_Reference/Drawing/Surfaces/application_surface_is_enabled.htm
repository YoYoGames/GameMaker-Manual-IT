la superficie dell'applicazione è abilitata

Questa funzione restituisce true se la superficie dell'applicazione viene utilizzata per il disegno, o false se viene utilizzato lo schermo buffer.

Sintassi:

application_surface_is_enabled();

Ritorna:

Esempio:

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

Il codice di cui sopra controlla la pressione di un tasto e attiva o disattiva la superficie dell'applicazione a seconda del suo stato (come in un menu di opzioni).
