superficie_getta_obiettivo

Con questa funzione è possibile ottenere l'ID della superficie corrente a cui si vuole disegnare. Se la superficie è stata creata con surface_create()allora questo sarà un numero intero positivo che rappresenta l'ID univoco di quella superficie.

Se non è stata impostata alcuna superficie personalizzata, ma la application_surface predefinita è ancora abilitata, la funzione restituirà tipicamente 0. Tuttavia, se si è manipolata la superficie in precedenza, si potrebbe verificare che l'ID ottenuto non corrisponda al valore attuale della variabile application_surface, nel caso in cui non sia più 0. Tuttavia, se si è manipolata la superficie_applicazione in precedenza, si potrebbe verificare che l'ID ottenuto non corrisponda al valore attuale della variabile superficie_applicazione, nel caso in cui questo non sia più 0.

Se non è stata impostata alcuna superficie personalizzata e la superficie_applicazione predefinita è stata disattivata, la funzione restituirà -1.

Sintassi:

surface_get_target();

Restituzione:

Surface ID or -1 (if no target surface is set)

Esempio:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

Il codice precedente controlla se il target di rendering corrente è una superficie o meno e, in caso affermativo, reimposta il target e disegna una superficie.
