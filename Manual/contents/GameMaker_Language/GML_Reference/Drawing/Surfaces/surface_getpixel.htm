superficie_getpixel

Questa funzione può essere usata per ottenere il colore di un pixel specifico da una superficie, usando le coordinate locali della superficie dove (0,0) è l'angolo superiore sinistro. Questa funzione non dovrebbe essere usata molto spesso perché è estremamente lenta e può causare una pausa nel vostro gioco.

NOTA: Quando si lavora con le superfici c'è la possibilità che esse possano cessare di esistere in qualsiasi momento perché sono memorizzate in texture memory. Dovreste SEMPRE controllare che una superficie esista usando surface_exists() prima di farvi riferimento direttamente.

Sintassi:

surface_getpixel(surface_id, x, y);

ArgomentoTipo Descrizione
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

Ritorna:

Colour Constant

Esempio:

col = surface_getpixel(surf, 56, 78 );

Questo restituirà il colore del pixel alle coordinate (56,78) della superficie indicizzata nella variabile "surf".
