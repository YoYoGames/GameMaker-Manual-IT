superficie_salvataggio

Questa funzione salverà una superficie su disco usando il nome di file dato. La superficie deve essere salvata come file in formato *.png.

Sintassi:

surface_save(surface_id, fname);

ArgomentoTipo Descrizione
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.

Ritorna:

Esempio:

if keyboard_check_pressed(ord("S")
{
    surface_save(surf, "test.png");
}

Il codice qui sopra controllerà per vedere se l'utente preme il tasto "S" sulla tastiera e se lo fa salverà la superficie indicizzata nella variabile "surf" sul disco.
