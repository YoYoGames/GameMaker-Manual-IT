copia_superficie

Questa funzione prende semplicemente l'immagine da una superficie e la copia su un'altra nella posizione locale specificata all'interno di quella superficie (dove la posizione (0,0) è l'angolo superiore sinistro della superficie di destinazione). Se la superficie di destinazione ha già delle informazioni, queste saranno sovrascritte dalla copia, e la funzione non cambia la superficie di origine in alcun modo.

NOTA: Quando si lavora con le superfici c'è la possibilità che esse possano cessare di esistere in qualsiasi momento perché sono memorizzate in texture memory. Dovreste SEMPRE controllare che una superficie esista usando surface_exists() prima di farvi riferimento direttamente.

Sintassi:

surface_copy(destination, x, y, source);

Argomento Tipo Descrizione
destination ID superficie The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source ID superficie The ID of surface to be copied.

Ritorna:

N/A

Esempio:

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

Il codice di cui sopra controlla la vista corrente che viene disegnata e se è view[0] copia la superficie indicizzata nella variabile "temp_surf" sulla superficie indicizzata nella variabile "surf". Se la vista corrente è diversa da view[0], la superficie "surf" viene disegnata sullo schermo.
