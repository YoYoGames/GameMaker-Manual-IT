superficie_get_target_ext

Questa funzione recupera l'ID della superficie assegnata a uno dei 4 obiettivi di rendering disponibili per le superfici. Si fornisce l'indice del target di rendering da controllare, e la funzione restituirà -1 se nessuna superficie è assegnata, o un valore intero >= 0, che rappresenta l'ID della superficie assegnata (come restituito dalla funzione surface_create()).

Sintassi:

surface_get_target_ext(index);

ArgomentoTipo Descrizione
index The render target index to check (from 0 to 3).

Ritorna:

Surface ID or -1 (if no target surface is set)

Esempio:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

Il codice di cui sopra prima controllerà e vedrà se il shader render target 0 è stato impostato su una superficie, e se non lo è, allora ne viene assegnato uno.
