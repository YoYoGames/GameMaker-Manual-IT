istanza di

Questa funzione può essere usata per ottenere il nome della funzione costruttore che è stata usata per creare un struct. Il struct stesso dovrebbe essere stato creato usando il metodo new insieme al costruttore stesso. Si fornisce la variabile con il riferimento a struct da controllare e la funzione restituirà un string con il nome del costruttore o undefined.

Si noti che se si passa alla funzione un letterale struct (cioè un struct che è stato creato senza usare una funzione costruttore), la funzione restituirà semplicemente la "struct" string .

Questa funzione può essere utilizzata anche per verificare se un riferimento struct è un riferimento debole o meno. riferimento debole o meno, nel qual caso la funzione restituirà il string "weakref" invece del nome della funzione che ha creato il struct. Per ulteriori informazioni, vedere la funzione weak_ref_create().

Il nome del costruttore restituito includerà del testo aggiuntivo se la funzione costruttore non è stata creata in un script. Se è stata creata all'interno di un object, il suo nome includerà il object e l'evento in cui è stata creata. Per esempio, un costruttore chiamato test_constructor creato nell'evento Create di Object1 sarà restituito come "test_constructor_gml_Object_Object1_Create_0". Per evitare questo problema, creare i costruttori in un file script.

Sintassi:

instanceof(struct_id);

Argomento Tipo Descrizione
struct Struttura The struct reference to use.

Restituzione:

String or undefined

Esempio esteso:

In questo esempio, dobbiamo prima definire la funzione che sarà usata come costruttore per il nostro struct. La seguente funzione è definita in una risorsa script:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Questa funzione può essere usata insieme all'operatore new per creare un struct e popolarlo con le variabili impostate sui valori degli argomenti usati nella funzione:

mystruct = new init_struct(10, 100, "Hello World");

Possiamo quindi ottenere il nome della funzione del costruttore che è stata utilizzata in questo modo:

var _name = instanceof(mystruct);
if is_string(_name)
{
    show_debug_message(_name);
}

In questo modo si stamperà nel log di output string "init_struct", che è il nome della funzione costruttore definita nel suo script.
