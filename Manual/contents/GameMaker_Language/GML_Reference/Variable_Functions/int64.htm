int64

Questa funzione tenta di convertire un valore dato in un intero firmato a 64 bit, dove il valore deve essere un reale, un string, un int64, un int32, o un ptr. Qualsiasi altro valore causerà l'arresto del gioco con un messaggio di errore. È possibile verificare se una variabile contiene un int64 utilizzando la funzione is_int64().

Sintassi:

int64(val);

Argomento Tipo Descrizione
val Reale, Stringa, o Puntatore The value to convert.

Restituzione:

int64 (signed 64-bit integer)

Esempio:

steam_handle = int64(global.fileReadString);

Il codice precedente converte il valore contenuto nella variabile globale in un numero intero a 64 bit.
