variabile_instanza_esiste

Con questa funzione è possibile verificare se una variabile di ambito dell'istanza esiste o meno. Si fornisce il valore dell'ID univoco dell'istanza (che può essere trovato nelle Proprietà dell'istanza nell'editor di room, oppure viene restituito quando si chiama la funzione instance_create_layer()) e il nome della variabile da controllare come stringa (si veda il codice di esempio qui sotto). La funzione restituirà true se per l'istanza esiste una variabile con il nome indicato e false altrimenti.

Sintassi:

variable_instance_exists(instance_id, name);

Argomento Tipo Descrizione
instance_id ID istanza The unique ID value of the instance to check
name Stringa The name of the variable to check for

Restituzione:

Boolean

Esempio:

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

Il codice qui sopra controlla se la variabile "scudi" esiste nell'istanza che esegue il codice e, in caso contrario, viene creata e inizializzata a 0.
