variabile_globale_esiste

Con questa funzione è possibile verificare se una variabile di ambito globale esiste o meno. Si fornisce il nome della variabile globale da controllare come stringa (si veda il codice di esempio qui sotto) e la funzione restituirà true se esiste una variabile globale con il nome dato o false altrimenti.

Sintassi:

variable_global_exists(name);

Argomento Tipo Descrizione
name Stringa The name of the global variable to check for (as a string)

Restituzione:

Boolean

Esempio:

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

Il codice precedente controlla se la variabile globale chiamata "enemy_num" esiste e, in caso contrario, viene creata e impostata con un valore.
