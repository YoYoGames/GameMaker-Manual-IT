è_reale

Questa funzione restituisce se una data variabile è un numero reale (singolo, doppio o intero) o no. In alcuni casi volete controllare e vedere se una variabile contiene un numero reale, ed è allora che usereste questa funzione. Non restituisce il numero reale ma piuttosto true o false, quindi un valore di, per esempio, "pesce" restituirebbe false, tuttavia un valore di 200 restituirebbe true.

NOTA: Questa funzione restituirà false per qualsiasi enum valori in quanto sono memorizzati come int64s. La funzione corretta per controllare quel data type è is_int64().

Sintassi:

is_real(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

if is_real(val)
{
    score += val;
}

Il codice qui sopra controlla la variabile "val" per vedere se è un numero reale e se lo è lo aggiunge al punteggio.
