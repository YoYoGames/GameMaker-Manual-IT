ridimensionamento dell'array

Con questa funzione potete ridimensionare una dimensione di un array esistente ad una nuova dimensione. Voi fornite l'array da ridimensionare così come il nuovo numero di indici per l'array, e la funzione ridimensionerà quell'array. Notate che questa funzione è progettata per ridimensionare una matrice ad una lunghezza inferiore, poiché potete ridimensionare in alto semplicemente impostando un nuovo indice nella matrice. Detto questo, se la usate per ridimensionare un array, ogni nuovo indice sarà impostato al valore predefinito di 0.

Questa funzione può essere usata anche per gli array multidimensionali, purché specifichiate quale dimensione volete ridimensionare quando fornite l'indice dell'array, seguendo questo schema:

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

Il codice afferma che solo il primo slot della seconda dimensione viene ridimensionato, poiché gli slot in qualsiasi dimensione dell'array non sono interconnessi e possono avere dimensioni diverse; per esempio:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

Nel codice precedente, la lunghezza della seconda dimensione dell'array è 10 nel primo slot ma 22 nel secondo slot.

Sintassi:

array_resize(array_index, new_size);

ArgomentoTipo Descrizione
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Ritorna:

Esempio:

if array_length(a) > 10
{
    array_resize(a, 10);
}

Il codice qui sopra controlla la lunghezza di un array e se ha più di 10 indici, viene ridimensionato.
