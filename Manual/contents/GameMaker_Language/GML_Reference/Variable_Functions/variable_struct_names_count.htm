numero di nomi di strutture variabili

Con questa funzione potete trovare il numero totale di variabili definite per un struct. Voi fornite l'ID di struct da controllare, e la funzione restituirà un valore intero per il numero di variabili incontrate, oppure (se non esiste nessun struct dell'ID dato) -1.

Sintassi:

variable_struct_names_count(struct_id);

ArgomentoTipo Descrizione
struct_id The unique ID value of the struct to check.

Ritorna:

Esempio:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Il codice qui sopra recupererà il numero di variabili nel dato struct e mostrerà un messaggio di debug nell'output della console con questo valore.
