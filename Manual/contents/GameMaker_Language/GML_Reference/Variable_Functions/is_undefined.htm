is_undefined

Questa funzione restituisce se una data variabile è definita o no. In alcuni casi si vuole controllare e vedere cosa data type contiene una variabile in GameMaker ed è allora che si usa questa funzione. Restituisce true o false a seconda che il valore sia definito o meno.

NOTA: Questa funzione non può essere usata per verificare l'esistenza di una variabile. Usate variable_instance_exists() o variable_global_exists() invece.

Sintassi:

is_undefined(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

Il codice qui sopra controlla la variabile "val" per vedere se è indefinita o no e mostra un messaggio di debug se lo è.
