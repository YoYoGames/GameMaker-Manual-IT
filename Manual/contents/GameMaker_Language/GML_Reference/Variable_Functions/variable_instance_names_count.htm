numero di nomi di istanze variabili

Con questa funzione potete trovare il numero totale di variabili definite per un'istanza. Si fornisce l'ID dell'istanza da controllare, e la funzione restituisce un valore intero per il numero di variabili incontrate, o (se non esiste alcuna istanza dell'ID dato) -1.

Sintassi:

variable_instance_names_count(instance_id);

Argomento Tipo Descrizione
instance_id ID istanza The unique ID value of the instance to check.

Ritorna:

Real

Esempio:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

Il codice qui sopra recupererà il numero di variabili nell'istanza data e mostrerà un messaggio di debug nell'output della console con quel valore.
