è_int32

Questa funzione restituisce se una data variabile è un intero a 32 bit o no. In alcuni casi si vuole controllare e vedere quale data type contiene una variabile in GameMaker ed è allora che si usa questa funzione. Restituisce true o false a seconda che il valore sia un int 32 o meno.

Sintassi:

is_int32(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

if !is_int32(val)
{
    show_debug_message("Not a 32 bit integer!");
}

Il codice qui sopra controlla la variabile "val" per vedere se contiene un int32 e se non lo è allora mostra un messaggio nella console di debug.
