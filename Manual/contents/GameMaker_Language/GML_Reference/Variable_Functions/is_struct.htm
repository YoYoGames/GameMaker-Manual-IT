is_struct

Questa funzione controlla se il valore fornito è un struct. Restituisce true se lo è, altrimenti restituisce false.

Si noti che le variabili dei metodi restituiranno anche true, e le istanze degli oggetti restituiranno false.

Sintassi:

is_struct(val);

Argomento Tipo Descrizione
val Variabile The value to check.

Ritorna:

Boolean

Esempio:

if is_struct(a)
{
    delete(a);
}

Il codice qui sopra controlla una variabile per vedere se è un struct, e se la funzione restituisce true, il struct viene cancellato.
