variabile_globale_get

Con questa funzione potete ottenere il valore da una data variabile globale nominata. Voi fornite il nome della variabile globale di cui ottenere il valore come una stringa (vedete il codice di esempio qui sotto) e la funzione restituirà il valore tenuto dalla variabile globale o undefined se la variabile non esiste.

IMPORTANTE! Se la variabile globale che state ottenendo non esiste, allora la funzione restituirà la parola chiave undefined e potreste ottenere degli errori che impediranno al gioco di funzionare, quindi se avete dei dubbi usate la funzione variable_global_exists prima.

Sintassi:

variable_global_get(name);

ArgomentoTipo Descrizione
name The name of the global variable to get (as a string)

Ritorna:

Variable (any data type) or undefined (if the named variable does not exist)

Esempio:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

Il codice qui sopra controllerà per vedere se esiste una variabile globale e, se esiste, viene emessa nella console.
