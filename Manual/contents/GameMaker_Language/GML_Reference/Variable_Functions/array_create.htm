creare array

Con questa funzione potete creare un array 1D di una data dimensione. Dite alla funzione la lunghezza dell'array da creare, ed essa restituirà l'"handle" per l'array che potrete poi assegnare ad una variabile. Gli array creati in questo modo avranno ogni voce inizializzata a 0, a meno che non specifichiate un valore di inizializzazione (opzionale). Se fornite il valore extra per l'inizializzazione dell'array, allora tutti gli indici all'interno del nuovo array saranno impostati a quello invece che a 0, ma notate che la funzione avrà un maggiore sovraccarico di prestazioni in questo caso.

Sintassi:

array_create(size, [value]);

Argomento Tipo Descrizione
size Real The size of the array to create.
value Variabile OPTIONAL Il valore da usare per inizializzare tutti gli indici dell'array.

Ritorna:

Array

Esempio:

instance_array = array_create(100, noone);

Il codice qui sopra creerà una nuova matrice di 100 voci, ciascuna impostata sulla parola chiave noone, e poi la assegnerà alla variabile "instance_array".
