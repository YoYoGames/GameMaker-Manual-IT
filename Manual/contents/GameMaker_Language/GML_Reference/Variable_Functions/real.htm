Questa funzione può essere utilizzata per trasformare un dato string in un numero reale. Quando si usa questa funzione, i numeri, i segni meno, i punti decimali e le parti esponenziali in string sono presi in considerazione, mentre altri caratteri (come le lettere) causeranno un errore un errore. Se sapete, o sospettate, che un string possa avere altri caratteri, allora potete usare string_digits() per rimuovere tutti i caratteri non numerici, prima di usare questa funzione per trasformare la stringa risultante in un numero reale.

Sintassi:

real(string);

ArgomentoTipo Descrizione
string The string to be converted to a real value.

Ritorna:

Esempio:

var t_str = string_digits(input_str);
age = real(t_str);

Il codice di cui sopra prenderà l'input string, lo spoglierà di tutti i caratteri diversi dai numeri e poi imposterà la variabile "age" per contenere il valore numerico reale del ritorno string.

Indietro: Corde
Il prossimo: string
