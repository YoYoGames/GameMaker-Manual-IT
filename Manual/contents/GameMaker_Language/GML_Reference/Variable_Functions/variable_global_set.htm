variabile_globale_set

Con questa funzione potete impostare il valore di una data variabile globale. Si fornisce il nome della variabile globale di cui impostare il valore come una stringa (vedi esempio di codice qui sotto), e poi il valore da impostare (può essere qualsiasi tipo di dato valido). Se la variabile globale non esiste già nel gioco, verrà creata e poi le verrà assegnato il valore.

Sintassi:

variable_global_set(name, val);

Argomento Tipo Descrizione
name Stringa The name of the global variable to set (as a string)
val Variabile The value to set the global variable to

Ritorna:

N/A

Esempio:

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

Il codice qui sopra controllerà per vedere se esiste una variabile globale e se non esiste allora viene creata e impostata a 0.
