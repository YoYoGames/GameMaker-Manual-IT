è_stringa

Questa funzione restituisce se una data variabile è un numero string oppure no. In alcuni casi si vuole controllare e vedere se una variabile in GameMaker contiene un string e non un reale ed è allora che si usa questa funzione. Non restituisce il string ma piuttosto true o false, quindi un valore di, per esempio, "pesce" per n restituirà true, ma un valore di 200 per n restituirà false.

Sintassi:

is_string(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

if is_string(val)
{
    name = "Player: " + val;
}

Il codice qui sopra controlla la variabile "val" per vedere se contiene un string e se lo fa lo aggiunge in un altro string che viene poi assegnato alla variabile "name".
