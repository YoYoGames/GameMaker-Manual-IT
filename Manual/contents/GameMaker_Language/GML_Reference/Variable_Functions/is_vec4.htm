è_vec4

In alcuni casi si vuole controllare e vedere cosa data type contiene una variabile in GameMaker ed è allora che si usa questa funzione. Restituisce true o false a seconda che il valore sia un vettore di 4 valori o meno.

Sintassi:

is_vec4(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

if !is_vec4(val)
{
    show_debug_message("Not a 4 value vector!");
}

Il codice qui sopra controlla la variabile "val" per vedere se contiene un vec4 e se non lo è allora mostra un messaggio nella console di debug.
