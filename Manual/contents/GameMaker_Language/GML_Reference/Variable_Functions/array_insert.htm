inserire array

Con questa funzione potete inserire un valore (o più valori) in una matrice in qualsiasi posizione. La funzione richiede di fornire una variabile che contenga la matrice, l'indice (posizione) nella matrice da inserire, così come almeno un valore da inserire, sebbene possiate opzionalmente fornire ulteriori argomenti e saranno tutti inseriti nella matrice in ordine consecutivo dall'indice dato.

Si noti che se l'indice dato è al di fuori della lunghezza dell'array, i valori saranno aggiunti all'indice dato, e qualsiasi slot vuoto nell'array tra il suo ultimo valore e i valori inseriti sarà impostato a un valore predefinito di 0.

Sintassi:

array_insert(variable, index, value, [value], [value], [etc...]);

Argomento Tipo Descrizione
variable Array The variable that holds the array.
index Real The index (position) in the array to insert the value(s) 
value Variabile The value to insert
[value], [value], [etc...] Variabile OPTIONAL Altri valori da inserire nell'array

Ritorna:

N/A

Esempio:

array_insert(score_array, score_rank, current_score);

Il codice qui sopra inserirà il punteggio corrente nella matrice del punteggio, all'indice definito dalla variabile score_rank.
