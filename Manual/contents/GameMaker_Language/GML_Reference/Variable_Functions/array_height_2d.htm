array_height_2d DEPRECATO

ATTENZIONE! Questa funzione è deprecata (e sostituita da array_length()) poiché gli array non sono più limitati a solo 1 o 2 dimensioni, e come tale questa funzione è fornita solo per il supporto dei progetti legacy. Tutti i nuovi progetti dovrebbero usare il modo attuale di creare e accedere agli array multidimensionali (vedere qui per maggiori informazioni).

Con questa funzione potete ottenere l'altezza (numero di voci) della prima dimensione di una matrice 2D. Voi fornite la matrice da controllare e l'output della funzione vi dice quante voci iniziali contiene. Potete ottenere il numero di voci per la seconda dimensione dell'array usando la funzione array_length_2d.

Sintassi:

array_height_2d(array_index);

Argomento Tipo Descrizione
array_index Array The index of the array to check.

Ritorna:

Real

Esempio:

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

Il codice qui sopra loop attraverso un array 2D e imposta ogni voce a -1.
