lunghezza dell'array

Con questa funzione potete ottenere la lunghezza (numero di voci) di una dimensione di matrice. Fornite il valore dell'indice dell'array e la funzione restituirà un valore intero che rappresenta il numero di voci che l'array contiene. Questa funzione può essere usata anche per array multidimensionali, purché specifichiate di quale dimensione volete ottenere la lunghezza quando fornite l'indice dell'array, seguendo questo schema:

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc...

Sintassi:

array_length(array_index);

ArgomentoTipo Descrizione
array_index The index of the array to check.

Ritorna:

Esempio:

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

Il codice qui sopra loop attraverserà un array e imposterà ogni voce a -1.
