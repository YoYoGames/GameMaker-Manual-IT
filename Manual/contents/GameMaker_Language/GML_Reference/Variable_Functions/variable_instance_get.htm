variabile_instanza_get

Con questa funzione potete ottenere il valore di una data variabile nominata. Si fornisce il valore unico dell'ID dell'istanza (che può essere trovato dalle proprietà dell'istanza nell'editor di room, o viene restituito quando si chiama la funzione instance_create_layer()) così come il nome della variabile di cui ottenere il valore come una stringa (vedere il codice di esempio qui sotto). La funzione restituirà il valore della variabile, oppure undefined se la variabile non esiste.

IMPORTANTE! Se la variabile che state ottenendo non esiste, allora la funzione restituirà la parola chiave undefined e potreste ottenere degli errori che impediranno al gioco di funzionare, quindi se avete dei dubbi usate la funzione variable_instance_exists prima.

Sintassi:

variable_instance_get(instance_id, name);

Argomento Tipo Descrizione
instance_id ID istanza The unique ID value of the instance to use
name Stringa The name of the variable to get (as a string)

Ritorna:

Variable (any data type) or undefined (if the named variable does not exist)

Esempio:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

Il codice qui sopra controllerà per vedere se una variabile esiste e se esiste, allora il valore che contiene viene recuperato e memorizzato in una variabile locale. Se non esiste, la variabile locale viene impostata a -1.
