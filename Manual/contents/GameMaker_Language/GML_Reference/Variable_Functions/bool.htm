bool

Questa funzione tenterà di convertire un valore dato in un booleano data type, dove il valore sarà restituito come true se è maggiore o uguale a 0,5, e false altrimenti.

Sintassi:

bool(n);

Argomento Tipo Descrizione
n Real The value to convert.

Ritorna:

Boolean

Esempio:

if !is_bool(val)
{
    val = bool(val);
}

Il codice qui sopra controlla la variabile "val" per vedere se è un booleano e se non lo è allora viene convertito in uno.
