array_get

Con questa funzione potete recuperare il valore da un indice in una matrice. La funzione richiede di fornire una variabile che contiene l'array e l'indice per ottenere il valore da quell'array. Notate che se l'indice dell'array fornito è fuori dai limiti, il gioco si bloccherà con un errore. Questa funzione può essere usata anche per array multidimensionali, a patto che specifichiate quale dimensione volete ottenere quando fornite l'indice dell'array, seguendo questo schema:

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

Sintassi:

array_get(variable, index);

Argomento Tipo Descrizione
variable Array The variable that holds the array.
index Real The index of the array element to get the value from.

Ritorna:

Variable (Any valid data type that an array can hold)

Esempio:

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

Il codice di cui sopra emetterà i primi 10 elementi del dato nella console.
