Funzioni variabili

Le funzioni in questa pagina sono progettate per trattare tutte le diverse variabili e tipi di variabili quando si usa GameMaker Language nei vostri giochi. Per maggiori informazioni sulle variabili, consultate la pagina sulle variabili e lo scopo delle variabili.

Sono disponibili le seguenti funzioni variabili:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variabile_struttura_rimuovi
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Ci sono anche alcune funzioni associate all'uso degli array. Queste sono progettate per darvi flessibilità nel vostro codice, e vi permetteranno di creare array più funzionali e dinamici quando fate i vostri giochi. Queste funzioni sono:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(deprecato)
  • array_length_2d(deprecato)
  • array_height_2d(deprecato)

Queste prossime funzioni sono tutte legate all'uso delle variabili di metodo:

  • method
  • method_get_self
  • method_get_index

  • is_bool
  • is_array

  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
