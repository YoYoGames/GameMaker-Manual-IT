tipo di

Questa funzione restituisce il valore data type di una data variabile come string. I possibili valori di ritorno sono elencati nella tabella seguente:

Stringa Descrizione
"number" La variabile contiene un numero reale (in virgola mobile) - può includere NaN e infinity
"string" La variabile contiene una stringa
"array" La variabile fa riferimento a un array
"bool" La variabile contiene un booleano (true / false)
"int32" La variabile contiene un numero intero a 32 bit
"int64" La variabile contiene un numero intero a 64 bit
"ptr" La variabile contiene un puntatore
"undefined" La variabile è indefinita
"null" La variabile contiene un valore nullo (questo non dovrebbe essere visto normalmente)
"vec3" La variabile contiene un vettore di 3 valori
"vec4" La variabile contiene un vettore di 4 valori
"method" La variabile contiene un riferimento a una funzione
"struct" La variabile contiene un riferimento a struct
"ref" La variabile contiene un riferimento all'istanza
"unknown" Il valore è sconosciuto. Questo valore non dovrebbe mai essere visto e indica che qualcosa è andato storto al livello più elementare, come una sovrascrittura della memoria.

Si noti che in alcuni casi questa funzione potrebbe non restituire il valore corretto per un metodo. Si considerino le seguenti due definizioni di funzione:

a = function()
{
    // something
}

function b()
{
    // Something
}

Tecnicamente, sono entrambi considerati metodi, in quanto stanno legando una funzione a una variabile dell'ambito di istanza, tuttavia la chiamata a typeof() sulla funzione b restituirà "numero" e non "metodo", mentre la chiamata a a restituirà "metodo". Ciò è dovuto al fatto che ai metodi creati come quello di b vengono assegnati gli indici di script (che sono valori interi), poiché questo è il modo in cui il compilatore riconosce le funzioni di script, anche se la funzione non è stata definita in un script.

Sintassi:

typeof(variable);

Argomento Tipo Descrizione
variable Qualsiasi The variable to get the data type of.

Restituzione:

String (see table above)

Esempio:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Il codice precedente ottiene data type dalla variabile globale indicata e restituisce string a una variabile locale, che viene poi utilizzata per inviare un messaggio alla console.
