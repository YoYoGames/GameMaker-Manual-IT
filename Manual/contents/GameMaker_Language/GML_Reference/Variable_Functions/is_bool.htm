è_bool

Questa funzione restituisce se una data variabile è un booleano (true ior false) o no. In alcuni casi si vuole controllare e vedere se una variabile in GameMaker contiene un valore booleano, ed è allora che si usa questa funzione.

Sintassi:

is_bool(n);

Argomento Tipo Descrizione
n Variabile The argument to check.

Ritorna:

Boolean

Esempio:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

Il codice qui sopra controlla la variabile "val" per vedere se è un numero reale e se lo è lo usa per impostare una variabile globale.
