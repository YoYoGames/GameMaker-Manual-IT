nomi_di_struttura_variabile

Con questa funzione potete recuperare un array popolato con i nomi delle variabili da un struct. Passate il riferimento a struct da controllare, e ogni voce nell'array sarà una stringa dei nomi delle variabili che struct contiene.

Sintassi:

variable_struct_get_names(struct);

ArgomentoTipo Descrizione
struct The struct reference to check.

Ritorna:

(each entry is a string)

Esempio:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

Il codice qui sopra recupererà un array di nomi di variabili per il dato struct e poi mostrerà questi insieme ai loro valori nell'output di debug.
