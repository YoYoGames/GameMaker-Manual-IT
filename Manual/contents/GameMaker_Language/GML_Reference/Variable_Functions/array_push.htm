array_push

Con questa funzione potete spingere un valore (o più valori) alla fine di una matrice senza dover conoscere la lunghezza della matrice. La funzione richiede che forniate una variabile che contenga l'array e almeno un valore da spingere, anche se potete opzionalmente fornire ulteriori argomenti e saranno tutti spinti nell'array in ordine consecutivo.

Sintassi:

array_push(variable, value, [value], [value], [etc...]);

Argomento Tipo Descrizione
variable Array The variable that holds the array.
value Variabile The value to push onto the end of the array
[value], [value], [etc...] Variabile OPTIONAL Altri valori da spingere nell'array

Ritorna:

N/A

Esempio:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

Il codice qui sopra spingerà quattro valori alla fine dell'array dato.
