array_equals

Con questa funzione potete controllare se due array sono uguali (equivalenti o uguali). Voi date i due array da controllare, e la funzione restituirà true se sono uguali (o equivalenti o uguali) o false se non lo sono. Notate che questo non è lo stesso che controllare se due array sono uguali usando ==, che non controllerà per vedere se i due array contengono valori equivalenti, ma solo per vedere se gli array fanno riferimento allo stesso array iniziale. Per esempio:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Notate che ci sono alcune costanti che possono non essere uguali a se stesse, il che può far fallire questa funzione. Ecco un esempio:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Vedere la Tabella dell'uguaglianza per maggiori informazioni.

Sintassi:

array_equals(var1, var2);

Argomento Tipo Descrizione
var1 Array The first array.
var2 Array The second array.

Ritorna:

Boolean

Esempio:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Il codice qui sopra controllerà due array per vedere se contengono valori equivalenti, e se non lo fanno allora il codice copierà l'intero contenuto dell'array "inventory_array" nell'array "item_array".
