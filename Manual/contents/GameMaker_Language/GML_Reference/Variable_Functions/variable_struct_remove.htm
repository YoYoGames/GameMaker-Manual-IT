struttura_variabile_rimuovi

Con questa funzione è possibile rimuovere una variabile da struct. Si fornisce l'ID di struct da cui rimuovere la variabile e il nome string della variabile da rimuovere.

Sintassi:

variable_struct_remove(struct, name);

Argomento Tipo Descrizione
struct Struttura The struct reference to remove the variable from
name Stringa The name of the variable to remove (as a string)

Restituzione:

N/A

Esempio:

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

Il codice precedente controlla se la variabile data esiste nel sito struct e, in caso affermativo, la rimuove.
