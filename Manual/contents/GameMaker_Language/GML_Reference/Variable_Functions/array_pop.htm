array_pop

Questa funzione rimuove l'ultimo elemento dell'array dato e restituisce il suo valore. Se l'array è vuoto, viene restituito undefined.

Sintassi:

array_pop(array);

ArgomentoTipo Descrizione
array The array to remove the last value from.

Ritorna:

Variable or undefined

Esempio:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Il codice qui sopra estrarrà l'ultimo valore dall'array dato, e poi lo disegnerà insieme a del testo sullo schermo. La voce recuperata viene anche rimossa dall'array del punteggio.
