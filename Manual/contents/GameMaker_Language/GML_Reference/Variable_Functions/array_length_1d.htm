array_length_1d DEPRECATO

ATTENZIONE! Questa funzione è deprecata (e sostituita da array_length()) poiché gli array non sono più limitati a solo 1 o 2 dimensioni, e come tale questa funzione è fornita solo per il supporto dei progetti legacy. Tutti i nuovi progetti dovrebbero usare il modo attuale di creare e accedere agli array multidimensionali (vedere qui per maggiori informazioni).

Con questa funzione potete ottenere la lunghezza (numero di voci) di un array 1D. Per gli array 2D dovreste usare le funzioni array_height_2d e array_length_2d le funzioni.

IMPORTANTE! Se la matrice ha più di 32.000 voci, questa funzione restituirà un valore errato e non dovrebbe essere usata.

Sintassi:

array_length_1d(array_index);

Argomento Tipo Descrizione
array_index Array The index of the array to check.

Ritorna:

Real

Esempio:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
    a[i] = -1;
}

Il codice qui sopra loop attraverserà un array e imposterà ogni voce a -1.
