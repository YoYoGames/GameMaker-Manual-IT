è_array

Questa funzione può essere usata per controllare e vedere se una variabile contiene un array (restituirà true) o no (nel qual caso restituirà false).

Sintassi:

is_array(n);

Argomento Tipo Descrizione
n Variabile The variable to check.

Ritorna:

Boolean

Esempio:

if is_array(a)
{
    a = -1;
}

Il codice qui sopra controlla una variabile per vedere se è un array, e se la funzione restituisce true, l'array viene cancellato impostando la variabile a -1.
