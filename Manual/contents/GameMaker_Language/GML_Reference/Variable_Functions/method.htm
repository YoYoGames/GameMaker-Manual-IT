metodo

Con questa funzione potete legare una funzione esistente (o un metodo) all'istanza data o a struct, creando una nuova variabile di metodo che può essere usata in seguito.

Voi fornite l'ID dell'istanza da usare (un'istanza che è attiva in room, e non un indice object ) o un riferimento a struct, così come l'ID della funzione (o il riferimento al metodo) che volete legare. La funzione restituirà un nuovo metodo che può essere chiamato dalla variabile a cui è assegnato (vedi l'esempio di codice qui sotto).

Il metodo restituito sarà "legato" all'istanza data o a struct, il che significa che verrà sempre eseguito nell'ambito di quell'istanza/struttura.

Potete legare funzioni built-in così come funzioni/metodi definiti dall'utente, e potete anche fornire undefined come argomento di istanza/struttura, il che significa che l'ambito corrente sarà usato per il self corrente sarà usato per il collegamento.

Sintassi:

method(struct_ref_or_instance_id, func);

Argomento Tipo Descrizione
struct_ref_or_instance_id IDstruttura o istanza The unique reference or ID value of the struct or instance to use (can be self o undefined)
func Funzione o metododelloscript The ID of the function (or method reference) to use

Ritorna:

Method

Esempio:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
    enemy_func = method(_inst, enemy_ai);
}

Il codice di cui sopra controllerà se esiste un'istanza del nemico nella posizione del mouse. Se esiste, allora il metodo enemy_ai è legato all'istanza del nemico e restituito come nuova variabile del metodo "enemy_func".
