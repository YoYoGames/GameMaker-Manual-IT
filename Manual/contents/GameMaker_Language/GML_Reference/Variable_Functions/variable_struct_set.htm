set di strutture variabili

Con questa funzione si può impostare il valore di una data variabile in un struct. Si fornisce il riferimento a struct così come il nome della variabile di cui impostare il valore come una stringa (vedi codice di esempio qui sotto), e poi finalmente il valore da impostare (può essere qualsiasi tipo di dato valido). Se la variabile non esiste già in struct verrà creata e poi le verrà assegnato il valore.

Sintassi:

variable_struct_set(struct, name, val);

Argomento Tipo Descrizione
struct Struttura The struct reference to set
name Stringa The name of the variable to set (as a string)
val Variabile The value to set the variable to

Ritorna:

N/A

Esempio:

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

Il codice qui sopra controllerà per vedere se la variabile data esiste nel dato struct e se non esiste allora viene creata e impostata a 0.
