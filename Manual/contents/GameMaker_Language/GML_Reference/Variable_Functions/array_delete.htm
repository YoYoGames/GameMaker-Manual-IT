eliminare array

Con questa funzione potete cancellare un valore (o più valori) da una matrice in qualsiasi posizione. La funzione richiede di fornire una variabile che contiene la matrice, l'indice (posizione) nella matrice da cui cancellare, nonché il numero di valori da cancellare.

Sintassi:

array_delete(variable, index, number);

ArgomentoTipo Descrizione
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

Ritorna:

Esempio:

array_delete(score_array, 0, 10);

Il codice qui sopra cancellerà i primi 10 valori nell'array dato a partire dall'indice 0 (quindi gli indici da 0 a 9 saranno rimossi dall'array).
