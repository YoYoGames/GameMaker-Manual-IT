variabile_struttura_get

Con questa funzione potete ottenere il valore di una data variabile nominata all'interno di un struct. Voi fornite il riferimento struct così come il nome della variabile di cui ottenere il valore come una stringa (vedete il codice di esempio qui sotto). La funzione restituirà il valore della variabile o undefined se la variabile nominata non esiste.

IMPORTANTE! Se la variabile che state ottenendo non esiste, allora la funzione restituirà la parola chiave undefined e potreste ottenere degli errori che impediranno al gioco di funzionare, quindi se avete dei dubbi usate la funzione variable_struct_exists() prima.

Sintassi:

variable_struct_get(struct, name);

ArgomentoTipo Descrizione
struct The struct reference to use
name The name of the variable to get (as a string)

Ritorna:

Variable (any data type) or undefined (if the named variable does not exist)

Esempio:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

Il codice qui sopra controllerà per vedere se una variabile esiste nel dato struct e se esiste allora il valore che contiene viene recuperato e memorizzato in una variabile locale. Se non esiste, la variabile locale viene impostata a -1.
