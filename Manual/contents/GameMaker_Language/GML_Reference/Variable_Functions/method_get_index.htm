metodo_get_index

Con questa funzione potete recuperare l'indice Script per il script dove il metodo è stato definito. Se il metodo non è stato definito in un script allora la funzione restituirà -1, altrimenti restituirà il valore dell'indice per il script.

Sintassi:

method_get_index(method);

Argomento Tipo Descrizione
method Metodo The method variable to check

Ritorna:

Script Asset or Real

Esempio:

var _index = method_get_index(light_setup);
if (_index != undefined)
{
    show_debug_message(script_get_name(_index));
}

Il codice qui sopra controllerà l'indice script per il metodo "light_setup" e se non è -1 mostrerà il nome script alla console.
