nomi delle istanze variabili

Con questa funzione potete recuperare un array popolato con i nomi delle variabili di istanza per un'istanza, o le variabili globali per un gioco. Quando passate un valore di ID di istanza, ogni voce nell'array sarà un string del nome della variabile che corrisponde a una variabile dell'ambito di istanza che è stata creata nell'istanza. Tuttavia, se passate la parola chiave global, ogni voce nell'array sarà un string del nome della variabile che corrisponde a una variabile di ambito globale.

Sintassi:

variable_instance_get_names(instance_id/global);

Argomento Tipo Descrizione
instance_id/global ID dell'istanza o global The unique ID value of the instance to check or the keyword global

Ritorna:

Array (each entry is a string)

Esempio:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

Il codice qui sopra recupererà un array di tutte le variabili dell'ambito dell'istanza per l'istanza che esegue il blocco di codice e poi le mostrerà insieme ai loro valori nell'output di debug.
