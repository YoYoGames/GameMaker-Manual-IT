array_length_2d DEPRECATO

ATTENZIONE! Questa funzione è deprecata (e sostituita da array_length()) poiché gli array non sono più limitati a solo 1 o 2 dimensioni, e come tale questa funzione è fornita solo per il supporto dei progetti legacy. Tutti i nuovi progetti dovrebbero usare il modo attuale di creare e accedere agli array multidimensionali (vedere qui per maggiori informazioni).

Con questa funzione potete ottenere la lunghezza (numero di voci) della seconda dimensione di un array. Fornite il numero di entrata per la prima dimensione e la funzione restituirà il numero di entrate della seconda dimensione che la matrice ha (per trovare la lunghezza della prima dimensione usate la funzione array_height_2D()). La funzione restituirà 0 se la variabile fornita non è una matrice o 1 se la variabile è una matrice 1D (poiché c'è ancora 1 riga).

Sintassi:

array_length_2d(array_index, n);

Argomento Tipo Descrizione
array_index Array The index of the array to check.
n Real The entry of the array to get the length of.

Ritorna:

Real

Esempio:

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

Il codice qui sopra loop attraverso un array 2D e imposta ogni voce a -1.
