ptr

Questa funzione tenterà di convertire un dato valore in un puntatore data type, dove il valore deve essere o un real, un string, un int64, un int32, o un ptr. Qualsiasi altra cosa causerà il crash del gioco con un messaggio di errore.

Sintassi:

ptr(n);

Argomento Tipo Descrizione
n Reale, stringa o puntatore The value to convert.

Ritorna:

Pointer

Esempio:

if !is_ptr(val)
{
    val = ptr(application_surface);
}

Il codice qui sopra controlla la variabile "val" per vedere se contiene un puntatore e se non lo contiene, gliene viene assegnato uno.

Indietro: Funzioni variabili
Il prossimo: int64
