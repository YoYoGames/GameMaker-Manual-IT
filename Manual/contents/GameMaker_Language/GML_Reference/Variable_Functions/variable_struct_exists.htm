struttura_variabile_esiste

Con questa funzione è possibile verificare se una variabile esiste o meno all'interno del sito struct. Si fornisce il riferimento a struct da utilizzare e il nome della variabile da controllare come stringa (si veda il codice di esempio qui sotto). La funzione restituirà true se esiste una variabile con il nome dato per struct e false altrimenti.

Sintassi:

variable_struct_exists(struct, name);

Argomento Tipo Descrizione
struct Struttura The struct reference to check
name Stringa The name of the struct variable to check for (as a string)

Restituzione:

Boolean

Esempio:

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

Il codice qui sopra controlla se la variabile "scudi" esiste nel sito struct e se non esiste viene creata e inizializzata a 0.
