imposta_gioco_velocità

Questa funzione può essere usata per impostare la velocità di gioco. Potete impostarla in uno dei due modi - come fotogrammi di gioco al secondo (FPS) o come microsecondi per fotogramma di gioco (MPF) - usando una delle seguenti due costanti:

Costante Descrizione
gamespeed_fps Imposta la velocità di gioco usando i fotogrammi al secondo.
gamespeed_microseconds Imposta la velocità di gioco usando microsecondi per fotogramma.

Così, per esempio, una velocità di gioco di 30 fotogrammi al secondo sarebbe 33333 microsecondi per fotogramma di gioco, che sarebbe poi espresso da questa funzione come:

game_set_speed(30, gamespeed_fps);

oppure:

game_set_speed(33333, gamespeed_microseconds);

Sintassi:

game_set_speed(speed, type);

Argomento Tipo Descrizione
speed Real The new game speed (as either FPS or MPF).
type Velocità di gioco costante The type of method used to set the game speed (see the constants above).

Ritorna:

N/A

Esempio:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Il codice qui sopra controlla se il gioco è in esecuzione in un browser e imposta la velocità di gioco di conseguenza come valore FPS.
