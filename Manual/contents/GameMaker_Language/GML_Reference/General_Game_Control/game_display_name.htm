nome_display del gioco

Questa variabile di sola lettura restituisce il nome di visualizzazione del vostro gioco per la piattaforma di destinazione, come impostato nelle Opzioni di gioco.

Sintassi:

game_display_name

Ritorna:

Esempio:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

Il codice di cui sopra ottiene il nome del display e il numero di versione del gioco e li disegna.

Indietro: Controllo generale del gioco
Il prossimo: game_project_name
