game_id DEPRECATED

Questa variabile di sola lettura restituisce l'identificatore unico per il gioco che avete creato. Puoi usarla se hai bisogno di un nome di file unico, o qualsiasi altra cosa che abbia bisogno di qualcosa che identifichi solo il tuo gioco. Questo può essere impostato nelle Opzioni di gioco.

Sintassi:

game_id

Ritorna:

Real

Esempio:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

Il codice qui sopra esegue una crittografia molto semplice sul punteggio aggiungendovi il game_id prima di salvare. Quando lo si rilegge nel gioco, si deduce il game_id per ottenere di nuovo il valore corretto.
