valore del punteggio più alto

Con questa funzione potete recuperare il valore del punteggio che è stato memorizzato nella lista dei punteggi più alti nella posizione data. Se non è stato inserito alcun punteggio, la funzione restituisce 0.

Sintassi:

highscore_value(place);

ArgomentoTipo Descrizione
place The place on the table (1-10).

Ritorna:

Esempio:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

Il codice di cui sopra si collega a loop attraverso la lista dei punteggi più alti e memorizza tutti i punteggi in un array.
