nome del progetto del gioco

Questa variabile di sola lettura restituisce il nome visualizzato del vostro gioco per la piattaforma di destinazione in un formato "save-friendly" per la piattaforma di destinazione. Se il nome visualizzato contiene dei caratteri che non sono permessi per un nome di file, essi saranno saranno sostituiti automaticamente con "_". Il nome visualizzato può essere impostato nelle Opzioni di gioco. Nota che, poiché non ci sono restrizioni sui nomi dei file per HTML5, questo string sarà probabilmente lo stesso di quello restituito da game_display_name.

Sintassi:

game_project_name

Ritorna:

Esempio:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

Il codice di cui sopra ottiene il nome del progetto e lo usa per aprire (o creare) un file ini che viene poi scritto.

Indietro: Controllo generale del gioco
Il prossimo: game_end
