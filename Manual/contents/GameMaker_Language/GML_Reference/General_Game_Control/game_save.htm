salvataggio del gioco

Questa è una funzione legacy che può essere usata per salvare lo stato corrente del gioco, e non è più raccomandata per l'uso. Usa invece le funzioni File per creare un sistema di salvataggio personalizzato in cui salvare e caricare solo dati di gioco specifici.

Questa funzione salverà lo stato del gioco così com'è, ma non salverà le risorse dinamiche utilizzate in quel momento, come le strutture dati, le superfici, assets aggiunte su runtime, ecc. Se un tale file di salvataggio viene caricato, qualsiasi aggiornamento del gioco che è stato fatto dopo il salvataggio non sarà visibile, in quanto verrà ripristinata la versione del gioco che è stata utilizzata per creare il salvataggio.

Si noti che i file di salvataggio creati con questa funzione potrebbero non essere supportati dalle versioni aggiornate di Runtime, quindi sostituire questa funzione con un nuovo sistema di salvataggio è essenziale per garantire la compatibilità con i futuri aggiornamenti di GameMaker e, per le ragioni elencate sopra, gli aggiornamenti fatti al proprio gioco.

Sintassi:

game_save(filename);

ArgomentoTipo Descrizione
filename The name of the file to save the game to.

Ritorna:

Esempio:

if keyboard_check_pressed(ord("S"))
{
    global.Saved = true;
    game_save("Save.dat");
}

Il codice qui sopra imposterà una variabile globale a true e poi salverà il gioco nel file "Save.dat" quando il tasto "S" viene premuto.
