carico_di_gioco

Con questa funzione potete caricare un gioco che è stato precedentemente salvato usando game_save(). Si noti che ripristinerà la versione del gioco che è stata utilizzata per creare il salvataggio, quindi qualsiasi aggiornamento fatto dopo non sarà visibile. Per maggiori informazioni, leggete la pagina su game_save().

Sintassi:

game_load(filename);

Argomento Tipo Descrizione
filename Stringa The name of the file to load from.

Ritorna:

N/A

Esempio:

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

Il codice qui sopra caricherà un gioco precedentemente salvato se una variabile globale è true quando il giocatore preme il tasto "L".

Indietro: Controllo generale del gioco
