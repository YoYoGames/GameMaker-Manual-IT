carico_gioco_buffer

Con questa funzione è possibile caricare uno stato di gioco che è stato salvato in precedenza. Il gioco viene caricato da un "grow" buffer creato in precedenza (vedi Buffers) e il buffer deve avere uno stato di gioco salvato in esso usando game_save_buffer() funzione.

Sintassi:

game_load_buffer(buffer);

ArgomentoTipo Descrizione
buffer The buffer id to load from.

Ritorna:

Esempio:

if keyboard_check_pressed(ord("L"))
{
    if (global.Checkpoint) game_load_buffer(save_buff);
}

Il codice qui sopra caricherà uno stato di gioco precedentemente salvato da buffer indicizzato nella variabile "save_buff", solo se la variabile globale è true, quando il giocatore preme il tasto "L".

Indietro: Controllo generale del gioco
Il prossimo: game_save
