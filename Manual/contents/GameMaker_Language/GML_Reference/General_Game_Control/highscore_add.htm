aggiungi punteggio alto

Con questa funzione è possibile aggiungere un nome e un punteggio alla lista globale interna dei punteggi più alti. Non c'è bisogno di controllare il valore per vedere se è abbastanza alto da essere inserito nella lista dei punteggi, poiché GameMaker memorizza solo i valori che sono maggiori della decima posizione memorizzata.

Sintassi:

highscore_add(str, numb);

Argomento Tipo Descrizione
str Stringa The string (name) to attribute the score to.
numb Real The number (score) to add.

Ritorna:

N/A

Esempio:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

Il codice di cui sopra controllerà le vite e se sono impostate a 0, aggiungerà il punteggio corrente e il string tenuto nella variabile globale "Name" nella lista dei punteggi più alti, prima di impostare finalmente il punteggio a 0 e inviare il giocatore al room indicizzato in "rm_MainMenu".

Indietro: Controllo generale del gioco
