salvataggio del gioco

Questa variabile di sola lettura restituisce l'ID completo path della directory utilizzata dal vostro gioco per salvare i file. Questa directory può essere visibile o meno alle altre applicazioni, a seconda della piattaforma.

Sul target HTML5 restituirà un string vuoto.

NOTA Questa variabile esiste solo per informazione e non dovrebbe essere usata per ottenere un file path o qualcosa di simile, poiché non funzionerà correttamente su tutte le piattaforme.

Sintassi:

game_save_id

Ritorna:

String

Esempio:

save_dir = game_save_id;

Questo memorizzerà la directory per il salvataggio dei file in una variabile per uso futuro.
