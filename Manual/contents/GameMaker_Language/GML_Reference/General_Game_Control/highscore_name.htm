nome del punteggio più alto

Con questa funzione è possibile recuperare il nome string che è stato memorizzato nella lista dei punteggi più alti nella posizione data. Se non è stato inserito alcun nome, verrà restituito l'indirizzo string "Unknown".

Sintassi:

highscore_name(place);

ArgomentoTipo Descrizione
place The place on the table (1-10).

Ritorna:

Esempio:

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

Il codice di cui sopra si collega a loop attraverso la lista dei punteggi più alti e memorizza tutti i nomi in un array.
