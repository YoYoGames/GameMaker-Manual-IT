fisica_applica_impulso_angolare

Questa funzione darà un impulso angolare a un'istanza fisica abilitata. Questo impulso imposterà la rotazione angolare della quantità data, ignorando la coppia corrente, essenzialmente impostando la quantità di "spin" che un'istanza ha. Se volete applicare una forza angolare ad un'istanza usando la coppia, allora dovreste usare la funzione physics_apply_torque().

Sintassi:

physics_apply_angular_impulse(impulse)

ArgomentoTipo Descrizione
impulse The impulse (in Newton metres) to be applied

Ritorna:

Esempio:

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

Il codice qui sopra darà all'apparecchio abilitato alla fisica un impulso angolare se viene premuto un tasto.

Indietro: Forze
Il prossimo: physics_apply_force
