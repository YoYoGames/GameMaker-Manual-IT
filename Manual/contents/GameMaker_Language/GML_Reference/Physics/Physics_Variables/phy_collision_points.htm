punti di collisione fittizi

Questa variabile di sola lettura restituisce il numero di punti di collisione rilevati tra i due objects nella collisione

NOTA: Questa variabile è disponibile solo nell'evento di collisione di un'istanza abilitata alla fisica.

Sintassi:

phy_collision_points;

Ritorna:

(integer value, or undefined if the instance is not physics enabled)

Esempio:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

Il codice di cui sopra crea particelle in tutti i punti definiti di una collisione tra due istanze abilitate alla fisica.

Indietro: Variabili fisiche
Il prossimo: phy_collision_x
