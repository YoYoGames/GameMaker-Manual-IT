rotazione_fissa

Questa variabile può essere usata per impostare se l'istanza può essere influenzata o meno da forze rotazionali (il default è false). Se questo è impostato a true, nessuna forza esterna (sia da impulsi o forze codificate, sia da collisioni) influenzerà il valore di rotazione dell'istanza e questo dovrebbe essere impostato manualmente usando la phy_rotation variabile

Sintassi:

phy_fixed_rotation;

Ritorna:

(or undefined if the instance is not physics enabled)

Esempio:

phy_rotation = 0;
phy_fixed_rotation = true;

Il codice di cui sopra switch l'istanza avrà una rotazione fissa, poi imposterà l'angolo di rotazione.
