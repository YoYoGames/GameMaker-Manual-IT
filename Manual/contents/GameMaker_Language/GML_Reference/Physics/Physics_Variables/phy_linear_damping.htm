smorzamento_lineare

Questa variabile può essere usata per impostare lo smorzamento lineare dell'istanza, o può essere usata per ottenere lo smorzamento lineare corrente. Lo smorzamento è la quantità di "resistenza" al movimento in avanti che l'istanza abilitata alla fisica ha, con un valore più basso che permette all'istanza di muoversi e accelerare più velocemente e un valore più alto che richiede una spinta più forte

Sintassi:

phy_linear_damping;

Ritorna:

(single precision floating point value, or undefined if the instance is not physics enabled)

Esempio:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

Il codice di cui sopra controllerà una collisione tra l'istanza chiamante e le istanze di "obj_Water" e cambierà lo smorzamento lineare di conseguenza.
