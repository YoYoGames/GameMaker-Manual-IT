phy_kinematic

Questa variabile di sola lettura restituirà true se l'istanza è classificata come cinematica object, o false se non lo è. Un'istanza cinematica è un'istanza che ha una massa infinita (una densità di 0) ma può muoversi. Quindi, per rendere un'istanza cinematica, dovreste prima creare un'istanza statica e poi impostare una o più variabili di istanza relative al movimento (es: phy_speed_x, phy_speed_y, o phy_angular_velocity)

Sintassi:

phy_kinematic;

Ritorna:

(or undefined if the instance is not physics enabled)

Esempio:

if !phy_kinematic
{
    phy_speed_x = 5;
}

Il codice qui sopra controlla se l'istanza è cinematica e se non lo è, imposta la velocità orizzontale a 5.
