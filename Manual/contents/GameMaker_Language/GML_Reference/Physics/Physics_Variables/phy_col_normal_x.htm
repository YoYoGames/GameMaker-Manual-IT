phy_col_normal_x

Questa variabile di sola lettura restituisce la componente x della normale di collisione corrispondente al phy_collision_x valore della matrice. Per ogni punto di contatto c'è una normale di contatto associata (che di solito è la stessa normale per tutti i punti di contatto nella collisione). Questa normale di contatto è un vettore unitario che punta da un'istanza della collisione ad un'altra, e può essere usata, per esempio, per calcolare la corretta direzione di "spinta" per risolvere le collisioni

NOTA: Questa variabile è disponibile solo nell'evento di collisione di un'istanza abilitata alla fisica.

Sintassi:

phy_col_normal_x;

Ritorna:

(single precision floating point value, or undefined if the instance is not physics enabled)

Esempio:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

Il codice qui sopra usa la normale di contatto per impostare la direzione del movimento per un'istanza creata nell'evento di collisione tra due istanze abilitate alla fisica.
