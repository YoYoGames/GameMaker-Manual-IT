Velocità_angolare

Questa variabile può essere usata per impostare la velocità angolare dell'istanza, o può essere usata per ottenere la velocità angolare corrente, in gradi al secondo e il valore usato può essere positivo (per la rotazione in senso orario) o negativo (per la rotazione in senso antiorario). Se lo imposti su un'istanza che prima era statica (cioè: ha una densità di 0) diventerà un cinematico object e inizierà a ruotare

Sintassi:

phy_angular_velocity;

Ritorna:

(single precision floating point value, or undefined if the instance is not physics enabled)

Esempio:

if abs(phy_angular_velocity) > 0
{
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
    phy_angular_velocity = 0;
}

Il codice di cui sopra controllerà la velocità angolare dell'istanza e se non è 0 aggiungerà (o sottrarrà) una piccola quantità ad ogni passo finché il valore non sarà 0.
