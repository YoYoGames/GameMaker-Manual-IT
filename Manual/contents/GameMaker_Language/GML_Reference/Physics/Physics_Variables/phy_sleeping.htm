dormiente

Questa variabile di sola lettura restituisce se l'istanza è attualmente "dormiente" (true) o no (false), Un'istanza "dormiente" è una che non è attivamente impegnata in alcuna simulazione fisica. GameMaker metterà objects a dormire per risparmiare cicli di simulazione quando un'istanza è a riposo e non in collisione con un'altra istanza

Sintassi:

phy_sleeping;

Ritorna:

(or undefined if the instance is not physics enabled)

Esempio:

if phy_sleeping
{
    instance_destroy();
}

Il codice di cui sopra controlla se il sito object viene simulato attivamente o no e se non lo è viene distrutto.
