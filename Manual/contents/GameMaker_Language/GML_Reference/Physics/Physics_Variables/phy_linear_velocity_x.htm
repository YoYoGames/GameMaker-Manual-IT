velocità_lineare_x

Questa variabile può essere usata per ottenere o cambiare la componente x del vettore di velocità lineare dell'istanza ed è definita in pixel al secondo (per i pixel per passo, vedere phy_speed_x). Modificando questa variabile per un'istanza statica (cioè: un'istanza con 0 densità) la si trasformerà in un'istanza cinematica

Sintassi:

phy_linear_velocity_x;

Ritorna:

(single precision floating point value, or undefined if the instance is not physics enabled)

Esempio:

if (phy_linear_velocity_x != 0)
{
    phy_linear_velocity_x = 0;
}

Il codice di cui sopra controllerà la componente x del vettore velocità lineare e se è diversa da 0 la imposterà su 0.
