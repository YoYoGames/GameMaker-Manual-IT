velocità_fisica

Questa variabile di sola lettura restituisce la velocità corrente dell'istanza fisica abilitata, definita in pixel per passo. Se avete bisogno di cambiare questo valore, dovete farlo cambiando i vettori x e y utilizzando le variabili phy_speed_x e phy_speed_y.

Sintassi:

phy_speed;

Ritorna:

(single precision floating point value, or undefined if the instance is not physics enabled)

Esempio:

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

Il codice qui sopra controlla la velocità dell'istanza fisica abilitata e poi cambia lo smorzamento lineare in base al valore restituito.
