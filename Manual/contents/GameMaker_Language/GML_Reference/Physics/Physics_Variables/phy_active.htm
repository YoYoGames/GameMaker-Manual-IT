phy_active

Questa variabile controlla se l'istanza è attualmente "attiva" o meno. Impostandola su false si impedirà all'istanza di partecipare al mondo della fisica, e impostandola su true la si farà partecipare di nuovo. Si prega di notare che questo non è lo stesso che disattivare l'istanza, in quanto l'istanza è ancora visibile sullo schermo e può ancora essere modificata attraverso il codice, piuttosto questa funzione le impedisce solo di partecipare alla simulazione fisica

Sintassi:

phy_active;

Ritorna:

(or undefined if the instance is not physics enabled)

Esempio:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

Il codice di cui sopra rileverà la pressione di un tasto della lettera "P" e poi commuterà la variabile globale "Pause" da true a false e viceversa. Questa variabile viene poi utilizzata per impostare se la fisica è attiva o meno nelle istanze dei figli di object indicizzate nella variabile "obj_Parent".
