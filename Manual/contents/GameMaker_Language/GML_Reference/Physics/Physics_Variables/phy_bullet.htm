pallottola_fisica

Questa variabile definisce se l'istanza si muove estremamente velocemente (per esempio un proiettile). Il valore predefinito è false ma se impostato su true questo dice a GameMaker che l'istanza si muoverà a velocità così elevate da richiedere un rilevamento delle collisioni più costoso per assicurarsi che non passi attraverso altre istanze senza essere rilevata

Sintassi:

phy_bullet;

Ritorna:

(or undefined if the instance is not physics enabled)

Esempio:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

Il codice qui sopra crea una nuova istanza e poi la definisce come un "proiettile" nel mondo della fisica prima di darle un impulso massiccio lungo l'asse y.

Indietro: Variabili fisiche
Il prossimo: phy_speed
