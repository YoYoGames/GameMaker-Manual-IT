sovrapposizione di test fisici

Questa funzione può essere usata per controllare e vedere se un corpo fisico (cioè: l'attrezzatura di un'istanza) si sovrappone, o si sovrapporrà, quando viene ruotata e posta in una data posizione in room. l'argomento "angle" è l'angolo di rotazione che l'istanza chiamante ha (o avrà) nella posizione da controllare, e l'argomento "obj" può essere sia un singolo id di istanza, e object indice o il parole chiave tutti o altri.

Sintassi:

physics_test_overlap(xpos, ypos, angle, obj);

Argomento Tipo Descrizione
xpos Real The x position in the room to check
ypos Real The y position in the room to check
angle Real The angle to check (of the calling instance)
obj ID dell'asset dell'oggetto o dell'istanza The object to check for

Ritorna:

Boolean

Esempio:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

Il codice qui sopra controllerà una posizione per una sovrapposizione di apparecchi fisici, e se c'è, imposta una variabile e un allarme.
