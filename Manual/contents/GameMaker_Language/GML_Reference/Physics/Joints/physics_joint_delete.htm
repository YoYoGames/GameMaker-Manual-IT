cancellazione_giunto_fisico

Una volta che due istanze con rappresentazioni fisiche sono state legate da un giunto, questo può essere cancellato di nuovo in qualsiasi momento. Normalmente questo avviene automaticamente quando una delle due istanze viene distrutta, o quando il sito room finisce, ma ci sono momenti in cui si può desiderare di farlo manualmente. In quei casi userete questa funzione.

Va anche notato che ogni volta che un'istanza che fa parte di un giunto dell'ingranaggio viene distrutta, il giunto dell'ingranaggio dovrebbe essere cancellato usando questa funzione prima di qualsiasi istanza coinvolta nella formazione dell'ingranaggio (ma i restanti giunti saranno cancellati automaticamente), per esempio nell'evento destroy dell'istanza.

Sintassi:

physics_joint_delete(joint)

Argomento Tipo Descrizione
joint ID congiunto di fisica The index of the joint that you wish to delete

Ritorna:

N/A

Esempio:

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

Il codice di cui sopra controlla se la variabile "shipJoint" contiene un indice del giunto e se lo contiene, calcola la forza applicata a quel giunto usando le due costanti. Infine, se la forza totale è maggiore di 2, il giunto viene eliminato.
