valore di physics_joint_set

Alcune proprietà del giunto possono essere cambiate e impostate anche dopo la creazione del giunto. Ci sono un certo numero di costanti che possono essere utilizzate in questa funzione e possono essere trovate qui: Costanti dei giunti fisici.

Sintassi:

physics_joint_set_value(joint, field, value)

Argomento Tipo Descrizione
joint ID congiunto di fisica The index of the joint that you wish to change
field Fisica Costante comune The constant for the joint property that you wish to change
value Real The new value for the joint property

Ritorna:

N/A

Esempio:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

Il codice di cui sopra controlla se la coppia massima del motore dei giunti è impostata a meno di 2 e se lo è la imposta a 2.
