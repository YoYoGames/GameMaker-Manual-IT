valore di physics_joint_get_value

Usando una serie di costanti predefinite, potete chiedere a GameMaker di dirvi una serie di cose sullo stato di qualsiasi giunto. Questo è molto utile perché vi dà la possibilità di eliminare i giunti o cambiare il comportamento di un'istanza a seconda delle vostre esigenze del momento. Ci sono un certo numero di costanti che possono essere utilizzate in questa funzione e possono essere trovate qui: Physics Joint Constants, ma siate consapevoli che vengono fatti calcoli complessi quando li chiamate, quindi dovrebbero essere usati con attenzione e solo quando necessario e notate che molti sono unici per un tipo specifico di giunto.

Se la proprietà non esiste (se, per esempio, si controlla un giunto di puleggia per la traslazione prismatica) allora 0 sarà il valore di ritorno.

Sintassi:

physics_joint_get_value(joint, value)

Argomento Tipo Descrizione
joint ID congiunto di fisica The index of the joint that you wish to test
value Costante congiunta di fisica The constant for the joint property that you wish to test

Ritorna:

Real

Esempio:

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

Il codice di cui sopra controlla se la variabile "shipJoint" contiene un indice del giunto e se lo contiene, calcola la forza applicata a quel giunto usando le due costanti. Infine, se la forza totale è maggiore di 2, il giunto viene eliminato.
