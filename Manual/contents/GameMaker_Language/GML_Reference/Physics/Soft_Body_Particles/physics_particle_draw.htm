disegno_fisica_particella

Questa funzione disegnerà un sprite nella posizione di tutte le particelle che condividono lo stesso valore di categoria definito dall'utente e che hanno delle bandiere che coincidono con quelle impostate per la maschera. Il valore della maschera è definito nello stesso modo in cui definiresti le bandiere durante la creazione della particella (vedi la funzione physics_particle_create() per un esempio), cioè: usando il bitwise o "|" per mascherare i bit appropriati per ogni flag.

NOTA: Questa funzione disegnerà tutte le particelle della simulazione fisica con le bandiere che coincidono con quelle della maschera, che siano particelle singole o gruppi, se la categoria dell'utente è la stessa.

La particella disegnata userà il colore e i valori alfa impostati quando l'hai creata, e ruoterà con il momento angolare della particella. Tuttavia, se desideri scalare sprite o cambiare qualsiasi altra proprietà disegnata, dovresti utilizzare la funzione physics_particle_draw_ext().

Sintassi:

physics_particle_draw(typemask, category, sprite, subimg)

Argomento Tipo Descrizione
typemask Fisica Particella Bandiera Costante(e) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite Asset The sprite to use for each particle.
subimg Real The image_index dello sprite da disegnare.

Ritorna:

N/A

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

Il codice qui sopra disegnerà un sprite per tutte le particelle nella categoria "2" definita dall'utente che hanno le stesse bandiere della maschera data.
