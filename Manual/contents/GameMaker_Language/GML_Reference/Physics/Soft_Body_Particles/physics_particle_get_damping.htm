fisica_particella_get_damping

Con questa funzione potete scoprire qual è l'attuale smorzamento lineare per le particelle nella simulazione fisica (potete impostare questo valore usando physics_particle_set_damping()).

Sintassi:

physics_particle_get_damping()

Ritorna:

Esempio:

if (physics_particle_get_damping() < 1)
{
    physics_particle_set_damping(physics_particle_get_damping() + 0.01);
}

Il codice di cui sopra controllerà il valore di smorzamento corrente per tutte le particelle nel sistema e se è inferiore a 1 allora vi aggiungerà 0,01.

Indietro: Particelle del corpo morbido
