physics_particle_get_group_flags

Con questa funzione puoi recuperare le bandiere di gruppo per un gruppo di particelle. Il valore group è quello che è stato restituito quando hai creato il gruppo di particelle utilizzando la funzione physics_particle_group_end()e la funzione restituisce un valore che è il valore combinato delle bandiere attualmente impostate.

Sintassi:

physics_particle_get_group_flags(group)

Argomento Tipo Descrizione
group ID del gruppo di particelle della fisica The particle group to get.

Ritorna:

Real

Esempio:

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

Il codice di cui sopra creerà una variabile per memorizzare il valore delle bandiere e poi la userà per controllare le bandiere del gruppo indicizzato nella variabile "group". Se non sono uguali, il gruppo viene impostato con queste bandiere.
