physics_particle_get_data_particle

Questa funzione restituisce varie informazioni su una singola particella nella simulazione fisica utilizzando i flags controllati. L'indice della particella (il suo ID) è quello restituito dalla funzione physics_particle_create()e il buffer utilizzato deve essere stato creato in precedenza con la funzione buffer_create(). Dovrebbe essere di tipo "grow", con una dimensione approssimativamente pari a quella dei dati di ritorno previsti. Le bandiere stesse sono impostate utilizzando le costanti indicate di seguito, e si dovrebbe utilizzare il bitwise o "|" per creare un singolo valore di bandiera per ottenere l'informazione desiderata.

Costante Descrizione Tipo di dati
phy_particle_data_flag_typeflags Il valore delle bandiere per la particella. buffer_u32
phy_particle_data_flag_position La posizione x e y della particella. 2 x buffer_f32
phy_particle_data_flag_velocity La velocità orizzontale e verticale. 2 x buffer_f32
phy_particle_data_flag_colour Il colore e il valore alfa (esadecimale). buffer_f32
phy_particle_data_flag_category La categoria della particella (come definita quando hai creato la particella). buffer_u32

Sintassi:

physics_particle_get_data_particle(ind, buffer, flags)

Argomento Tipo Descrizione
ind Particella fisica ID The index (ID) of the particle to get the data from.
buffer ID del buffer The (previously created) buffer to use to store the data.
flags Fisica Particella Dati Flag Costante(i) The flags to use to extract data about specific particle types.

Ritorna:

N/A

Esempio:

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

Il codice precedente ottiene la posizione e la velocità della particella indicizzata dalla variabile "part" e memorizza i dati in un certo numero di variabili.
