gruppo_fisico_di_particelle_unito

Con questa funzione è possibile unire due gruppi di particelle, e i gruppi uniti si comporteranno come se fossero entrambi parte di una singola entità. I gruppi dovrebbero essere stati creati con bordi sovrapposti, poiché, se non sono già in contatto, non verranno uniti. La funzione prende gli ID unici dei gruppi da unire, come restituiti dalla funzione physics_particle_group_end()ed è possibile utilizzare la funzione qualsiasi numero di volte per un singolo gruppo per unire vari corpi molli insieme.

Sintassi:

physics_particle_group_join(to, from)

Argomento Tipo Descrizione
to ID del gruppo di particelle della fisica The first particle group to join.
from ID del gruppo di particelle della fisica The second particle group to join.

Ritorna:

N/A

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x- 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g1 = physics_particle_group_end();
physics_particle_group_begin(flags, groupflags, mouse_x + 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g2 = physics_particle_group_end();
physics_particle_group_join(g1, g2);

Il codice qui sopra crea due gruppi di particelle circolari e li unisce.

Indietro: Particelle del corpo morbido
