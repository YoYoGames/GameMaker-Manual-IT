physics_particle_draw_ext

Questa funzione disegnerà un sprite nella posizione di tutte le particelle che condividono lo stesso valore di categoria definito dall'utente e che hanno delle bandiere che coincidono con quelle impostate per la maschera. Il valore della maschera è definito nello stesso modo in cui definiresti le bandiere durante la creazione delle particelle (vedi le funzioni physics_particle_create() per un esempio), cioè: usando il bitwise o "|" per mascherare i bit appropriati per ogni flag.

NOTA: Questa funzione disegnerà tutte le particelle della simulazione fisica con le bandiere che coincidono con quelle della maschera, che siano particelle singole o gruppi, se la categoria dell'utente è la stessa.

La particella disegnata ignorerà i valori di colore e alfa impostati quando l'hai creata, e sarà invece disegnata utilizzando i parametri di scala, angolo, colore e alfa dati in questa funzione. Nota che questo è puramente per disegnare un sprite nella posizione della particella e non influenza in alcun modo il comportamento della particella.

Sintassi:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Argomento Tipo Descrizione
typemask Fisica Particella Bandiera Costante(e) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite Asset The sprite to use for each particle.
subimg Real The image_index dello sprite da disegnare.
xscale Real The scale factor to apply to the sprite along the x axis.
yscale Real The scale factor to apply to the sprite along the y axis.
ang Real The angle (in degrees) to draw the sprite at.
col Colore The colour to blend with the sprite (default is c_white).
alpha Real The alpha value to use when drawing (0 - 1).

Ritorna:

N/A

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

Il codice qui sopra disegnerà un sprite per tutte le particelle nella categoria definita dall'utente "2" che hanno le stesse bandiere della maschera data, scalate di 2 lungo entrambi gli assi e con un'alfa di 0,5.
