gruppo_fisica_particella_fine

Questa funzione viene utilizzata per terminare la definizione di una forma di gruppo di particelle. Chiamando questa funzione creerà le particelle all'interno dei parametri di forma dati, e restituirà anche un valore ID che può essere memorizzato e utilizzato in ulteriori funzioni per le interazioni con il gruppo di particelle.

Sintassi:

physics_particle_group_end()

Ritorna:

Physics Particle Group ID

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Il codice di cui sopra memorizza le bandiere per il tipo di particella e le proprietà del gruppo di particelle in variabili e le utilizza per creare un gruppo di particelle circolare con un raggio di 100px nella posizione del mouse. L'ID del gruppo che è stato creato è memorizzato nella variabile "mLastGroup".
