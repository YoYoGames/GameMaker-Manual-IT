fisica_particella_get_densità

Con questa funzione potete scoprire qual è la densità di corrente per le particelle nella simulazione fisica (potete impostare questo valore usando physics_particle_set_density()).

Sintassi:

physics_particle_get_damping()

Ritorna:

Esempio:

if (physics_particle_get_density() < 1)
{
    physics_particle_set_density(physics_particle_get_density() + 0.01);
}

Il codice di cui sopra controllerà il valore di densità attuale per tutte le particelle nel sistema e se è inferiore a 1 allora vi aggiungerà 0,01.

Indietro: Particelle del corpo morbido
Il prossimo: physics_particle_get_damping
