creare_fisica_particella

Con questa funzione potete creare una singola particella in qualsiasi punto del vostro gioco room, impostando alcune bandiere e proprietà. Le "bandiere" sono il valore di ritorno di una combinazione delle seguenti costanti:

Fisica Particella Costante di bandiera
Costante Descrizione
phy_particle_flag_water Le proprietà predefinite per una particella soft body.
phy_particle_flag_zombie Una particella zombie è una particella che verrà distrutta dopo un solo passo, con tutte le altre contrassegnate in questo modo.
phy_particle_flag_wall Questo definisce la particella come statica, essenzialmente creandola come un object immobile nella simulazione fisica, in quanto rimarrà in una posizione fissa non importa cosa si scontra con loro. Si dovrebbe usare questo flag piuttosto che impostare la densità a 0.
phy_particle_flag_spring Le particelle a molla producono l'effetto di essere attaccate l'una all'altra, come da una molla. Le particelle create con questa bandiera sono "collegate" a coppie, con ogni particella collegata a quella che le era più vicina al momento della creazione. Una volta accoppiate, le particelle non cambiano "partner", e più una forza esterna le allontana l'una dall'altra, maggiore sarà la potenza con cui si scontreranno quando quella forza esterna verrà rimossa. Si noti che non importa quanto le particelle accoppiate si allontanino l'una dall'altra, la connessione tra loro non si spezzerà.
phy_particle_flag_elastic Le particelle elastiche si deformano e possono anche rimbalzare quando si scontrano con altri corpi rigidi nella simulazione fisica.
phy_particle_flag_viscous Una particella viscosa è una particella che mostra "aderenza" o "viscosità", come l'olio. Le particelle viscose si raggruppano e si attaccano di più.
phy_particle_flag_powder Le particelle di polvere producono un effetto di dispersione come si potrebbe vedere con la sabbia o la polvere.
phy_particle_flag_tensile Le particelle di tensione sono usate per produrre l'effetto della tensione superficiale, o la curvatura tesa sulla superficie di un corpo di liquido. Possono essere usate, per esempio, per creare la tensione superficiale che si vede su una goccia d'acqua. Una volta che la tensione è rotta, le particelle rimbalzano come se fossero elastiche, ma continuano anche ad attrarsi l'un l'altra. Di conseguenza, le particelle tendono a formare degli ammassi mentre rimbalzano.
phy_particle_flag_colourmixing Le particelle che mischiano i colori assumono parte del colore delle altre particelle con cui si scontrano. Nota che se solo una delle due particelle che si scontrano è una particella color-mixing, l'altra particella mantiene il suo colore pre-collisione.

Questi flag utilizzano il bit-masking per creare un valore di uscita finale che viene poi controllato per impostare le diverse proprietà di base della particella (con la proprietà di base che è sempre quella di phy_particle_flag_water). Per esempio, se vuoi simulare un liquido viscoso con tensione superficiale, useresti il bitwise o "|" per contrassegnare i bit appropriati come mostrato nell'esempio qui sotto. In questo modo puoi impostare diverse proprietà (oltre a quelle globali) per ogni singola particella creata.

Oltre alle bandiere, puoi anche impostare la posizione in room in cui creare la particella, la velocità orizzontale e verticale iniziale, il colore iniziale e l'alfa, così come la categoria utente. I valori di colore e alfa sono utilizzati solo quando si disegna la particella utilizzando la funzione physics_particle_draw()ma il valore della categoria è usato in molte altre funzioni. Questo valore è un valore intero arbitrario che dai ad ogni particella che vuoi che abbia proprietà simili, e può essere usato per impostare le bandiere in seguito per quella specifica categoria così come una serie di altre cose. Per esempio, se avete definito una particella "acqua" e una particella "olio", dovreste dare a tutte le particelle contrassegnate come "acqua" la categoria utente 1 e a tutte le particelle create come "olio" la categoria utente 2, permettendovi di puntare successivamente a una singola categoria per cambiarne i flag o ottenere dati specifici.

NOTA: Potete usare qualsiasi valore intero per la categoria, eccetto 0 che è riservato dalla simulazione ed è usato per selezionare tutte le categorie in altre funzioni (come per il disegno).

La funzione restituisce un valore unico di indice (o ID) per la particella che può essere memorizzato in una variabile per essere utilizzato in altre funzioni che fanno riferimento direttamente alla particella.

Sintassi:

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)

Argomento Tipo Descrizione
flags Fisica Particella Bandiera Costante(s) The flags to set on the particle.
x Real The x position to create the particle at.
y Real The y position to create the particle at.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
col Colore The base colour to use for the particle.
alpha Real The base alpha to use for the particle.
category Real The user defined category to which the particle belongs.

Ritorna:

Physics Particle ID

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1);

Il codice di cui sopra creerà delle variabili per impostare il tipo di particella e la sua velocità iniziale, poi le userà per creare una particella nella posizione x/y dell'istanza.
