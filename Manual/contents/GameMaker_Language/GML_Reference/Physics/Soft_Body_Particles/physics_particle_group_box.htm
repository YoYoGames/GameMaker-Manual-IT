casella_fisica_gruppo_di_particelle

Questa funzione imposta la forma del gruppo di particelle che si sta creando. Dovete prima aver iniziato la definizione del gruppo utilizzando la funzione physics_particle_group_begin()e poi userete questa funzione per impostare approssimativamente metà larghezza e metà altezza del gruppo in pixel - approssimativamente perché la larghezza e l'altezza esatte dipenderanno dalle dimensioni delle particelle di base, definite dalla funzione physics_particle_set_radius() poiché la simulazione fisica cerca di "inserire" il maggior numero possibile di particelle nella forma definita. Infine è necessario chiamare physics_particle_group_end() per creare il gruppo di particelle in room.

Sintassi:

physics_particle_group_box(halfWidth, halfHeight)

ArgomentoTipo Descrizione
halfWidth The half width of the box.
halfHeight The half height of the box.

Ritorna:

N/A

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_box(100, 100);
mLastGroup = physics_particle_group_end();

Il codice di cui sopra memorizza le bandiere per il tipo di particella e le proprietà del gruppo di particelle in variabili poi le usa per creare un gruppo di particelle rettangolare con lati di 200px nella posizione del mouse.
