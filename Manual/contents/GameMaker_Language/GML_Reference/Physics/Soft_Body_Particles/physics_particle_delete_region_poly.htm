physics_particle_delete_region_poly

Con questa funzione è possibile eliminare (rimuovere) tutte le particelle che rientrano nei limiti dell'area poligonale definita dalla simulazione fisica nel corrente room. La funzione prende un (precedentemente creato) ds_list() contenente la posizione x/y di ogni punto del poligono, con le posizioni pari nella lista che sono le coordinate x, e le posizioni dispari le coordinate y, cioè: per un triangolo, la vostra lista avrebbe sei voci, con le voci 0, 2, e 4 che sono le coordinate x e 1, 3, e 5 le coordinate y. Il poligono sarà poi utilizzato per eliminare tutte le particelle che rientrano nell'area definita.

Si noti che il poligono definito deve avere almeno tre punti e al massimo 8.

Sintassi:

physics_particle_delete_region_poly(pointList)

ArgomentoTipo Descrizione
pointList A ds_list of points to use to create the polygon.

Ritorna:

Esempio:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

Il codice di cui sopra cancellerà tutte le particelle che si trovano nell'area poligonale definita dai punti aggiunti a DS List.
