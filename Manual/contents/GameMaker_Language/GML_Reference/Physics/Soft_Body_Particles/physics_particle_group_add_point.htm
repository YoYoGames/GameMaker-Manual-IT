punto di aggiunta di gruppo fisico_particella

Questa funzione imposta un punto in room per definire la forma di un poligono che sarà utilizzato per creare un gruppo di particelle soft body. Dovete aver precedentemente segnalato a GameMaker che state per definire la forma di un poligono utilizzando la funzione physics_particle_group_polygon() e poi usare questa funzione per definire i singoli punti del poligono. Dovete dare almeno tre punti quando definite la forma del poligono, ma potete darne fino a otto, e la funzione permetterà la definizione di poligoni concavi. Tuttavia, se si genera un poligono con delle cavità, i punti al suo interno verranno ignorati e verrà creata una forma convessa per il gruppo di particelle.

Sintassi:

physics_particle_group_add_point(x, y)

ArgomentoTipo Descrizione
x The x position in the room for the point.
y The y position in the room for the point.

Ritorna:

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Il codice di cui sopra memorizza le bandiere per il tipo di particella e le proprietà del gruppo di particelle in variabili, poi le usa per creare un gruppo di particelle poligonali di tre lati nella posizione del mouse.
