physics_particle_set_radius

Con questa funzione potete impostare il raggio (in pixel) per le particelle in una simulazione fisica. Questa funzione è di portata globale, nel senso che cambierà il raggio non solo per le nuove particelle create dopo la modifica, ma anche per quelle già presenti nella simulazione.

Sintassi:

physics_particle_set_radius(radius)

ArgomentoTipo Descrizione
radius The radius (in pixels) of the particle fixture.

Ritorna:

Esempio:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Il codice qui sopra imposterà le proprietà di base per tutte le particelle nella simulazione.

Indietro: Particelle del corpo morbido
