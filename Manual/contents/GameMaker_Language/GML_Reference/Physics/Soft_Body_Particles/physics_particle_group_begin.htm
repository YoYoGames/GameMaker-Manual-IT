gruppo_fisica_particella_begin

Con questa funzione è possibile creare un gruppo di particelle in un room. Un gruppo di particelle è quello in cui le particelle necessarie per creare una forma specifica sono create tutte insieme come un gruppo, permettendo di creare corpi molli simulati con varie proprietà impostate dai flags utilizzati. Le "bandiere" sono il valore di ritorno di una combinazione di costanti di particelle (vedi la funzione physics_particle_create() per i dettagli di queste), mentre le "groupFlags" sono una combinazione delle seguenti costanti di gruppo (o 0 per nessuna bandiera):

Fisica Gruppo di particelle Costante di bandiera
Costante Descrizione
phy_particle_group_flag_solid Un gruppo di particelle solide impedisce ad altri elementi di alloggiare al suo interno. Se qualcosa dovesse penetrarvi, il gruppo di particelle solide spinge l'apparecchio incriminato verso la sua superficie, facendo sì che un gruppo di particelle solide possieda una forza repulsiva particolarmente forte.
phy_particle_group_flag_rigid I gruppi di particelle rigide sono quelli la cui forma non cambia, anche quando si scontrano con altri apparecchi.

Questi flag di gruppo usano la mascheratura dei bit per creare un valore di uscita finale che viene poi controllato per impostare le diverse proprietà di base del gruppo, oltre a quelle impostate dai flag delle particelle. Puoi usare nessuna, una o entrambe queste costanti usando il bitwise o "|" per mascherare i bit appropriati (come mostrato nell'esempio qui sotto).

Oltre alle bandiere, potete anche impostare la posizione in room per creare il gruppo di particelle (la posizione data sarà il centro di massa del gruppo finito), la velocità orizzontale e verticale iniziale, così come la velocità angolare del gruppo, permettendovi di creare gruppi mobili e rotanti.

Potete anche impostare il colore iniziale e l'alfa di tutte le particelle nel gruppo, con i valori che vengono utilizzati quando si disegnano le particelle utilizzando la funzione physics_particle_draw(). Dopodiché imposti la forza di coesione delle particelle nel gruppo, che è un altro modo per dire quanto fortemente si "attaccano" le une alle altre. La forza può essere un valore tra 0 e 1, dove 0 è nessuna coesione e 1 è la massima coesione.

Infine devi dare la categoria per il tuo gruppo di particelle. Questo valore è un valore intero arbitrario che dai ad ogni gruppo di particelle che vuoi avere proprietà simili, e può essere usato per impostare le bandiere in seguito per quella specifica categoria così come una serie di altre cose.

NOTA: Potete usare qualsiasi valore intero per la categoria, eccetto 0 che è riservato dalla simulazione ed è usato per selezionare tutte le categorie in altre funzioni (come per il disegno).

È importante notare che chiamare questa funzione non crea il gruppo nella fisica room. Questa funzione inizia semplicemente il processo e imposta le proprietà di base delle particelle per il gruppo. Per creare effettivamente il gruppo di particelle è necessario dargli una forma (vedi l'esempio qui sotto) e poi chiamare la funzione physics_particle_group_end()a questo punto verrà creato.

NOTA: I flag del tipo di particella che impostate influenzano anche la coesione del gruppo di particelle, in modo tale che se contrassegnate le particelle come di tipo acqua (per esempio), quando il gruppo si scontra con un'altra fixture si romperà, mentre se contrassegnate la particella come tipo molla, allora manterrà la sua coesione alla collisione.

Sintassi:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argomento Tipo Descrizione
flags Fisica Particella Bandiera Costante(s) The flags to set on the particle.
groupflags Fisica Particella Gruppo Bandiera Costante(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Colore The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Ritorna:

N/A

Esempio:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();
