recupero_fisico

Quando si lega una fixture ad un'istanza usando physics_fixture_bind() questo restituisce un "id" per la fixture legata. Potete usare questo id per ottenere la restituzione (la proprietà "bounciness") della fixture legata, non della fixture "base", in qualsiasi momento usando questa funzione.

Sintassi:

physics_get_restitution(fixture)

ArgomentoTipo Descrizione
fixture the id of the bound fixture

Ritorna:

Esempio:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Il codice qui sopra ottiene il valore di restituzione corrente per le proprietà fisiche vincolate dell'istanza e poi le imposta su un valore diverso.
