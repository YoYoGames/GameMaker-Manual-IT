imposta_fisica_restituzione

Quando si lega una fixture ad un'istanza usando physics_fixture_bind() questo restituisce un "id" per l'attrezzatura legata. Potete usare questo id per impostare il valore di restituzione della fixture legata, non la fixture "base", in qualsiasi momento usando questa funzione. La restituzione è solitamente impostata come un valore tra 0 e 1, ma potete usare valori più alti se necessario, anche se i risultati possono essere imprevedibili.

Sintassi:

physics_set_restitution(fixture, restitution)

ArgomentoTipo Descrizione
fixture the id of the bound fixture
restitution the new restitution value to apply

Ritorna:

Esempio:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Il codice qui sopra ottiene il valore di restituzione corrente per le proprietà fisiche vincolate dell'istanza e poi le imposta su un valore diverso.
