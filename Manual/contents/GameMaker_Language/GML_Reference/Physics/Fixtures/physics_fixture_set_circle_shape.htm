physics_fixture_set_circle_shape

Questa funzione definisce una forma di cerchio per il vostro apparecchio con un raggio definito dall'argomento "rad".

Sintassi:

physics_fixture_set_circle_shape(fixture, rad)

ArgomentoTipo Descrizione
fixture the index of the fixture
rad radius of the circle

Ritorna:

Esempio:

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

Il codice qui sopra applicherà una forma circolare all'apparecchio indicizzato nella variabile "fix_Ball" con un raggio uguale a quello della larghezza della sprite "spr_Ball" diviso per 2.
