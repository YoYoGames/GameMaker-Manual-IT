attrito fisico

Quando si lega una fixture ad un'istanza usando physics_fixture_bind() questo restituisce un "id" per l'attrezzatura legata. Potete usare questo id per impostare il valore di attrito dell'attrezzatura legata, non dell'attrezzatura "base", in qualsiasi momento usando questa funzione. Notate che l'attrito è solitamente impostato su un valore tra 0 e 1, ma potete usare qualsiasi valore non negativo se necessario.

Sintassi:

physics_set_friction(fixture, friction)

ArgomentoTipo Descrizione
fixture the id of the bound fixture
friction the new friction value to apply

Ritorna:

Esempio:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Il codice qui sopra ottiene il valore di attrito corrente per le proprietà fisiche vincolate dell'istanza e poi le imposta su un valore diverso.
