physics_fixture_set_chain_shape

Questa funzione definisce una forma di fixture "a catena". Una fixture a catena è composta da un certo numero di punti, che sono poi collegati insieme usando forme di bordo per unirli. La funzione stessa prende l'indice (ID) della fixture e può dire alla fixture di loop oppure no. Impostando questo su true si collegherà l'ultimo punto al primo punto con un bordo, mentre impostandolo su false non lo farà. Notate che questa funzione prepara la fixture ad accettare i punti richiesti per fare la catena, e questi dovrebbero essere aggiunti dopo aver chiamato questa funzione usando physics_fixture_add_point(), proprio come fareste quando costruite una fixture poligonale.

Quando si crea un dispositivo a catena, bisogna dargli almeno due punti, ma non si è limitati nel numero di punti successivi che si possono aggiungere per comporre il dispositivo finito.

Sintassi:

physics_fixture_set_chain_shape(fixture, loop)

ArgomentoTipo Descrizione
fixture The index of the fixture
loop Whether to close the chain (true) o no (false).

Ritorna:

Esempio:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

Il codice di cui sopra creerà una linea di apparecchi "bordo" con una varietà di altezze sulla lunghezza del room.
