physics_remove_fixture

Questa funzione rimuove (o "slega") una fixture da una o più istanze. Richiede l'"id" unico dell'attrezzatura vincolata (come restituito dalla funzione physics_fixture_bind() e rimuoverà tutte le proprietà fisiche attualmente definite per l'istanza, permettendovi di ridefinire una nuova fixture e legarla all'istanza. In questo modo potete cambiare le proprietà fisiche dell'istanza senza doverla distruggere e ricreare.

Sintassi:

physics_remove_fixture(id, fixture)

Argomento Tipo Descrizione
id ID istanza The ID of the instance to remove the fixture from
fixture ID apparecchio fisico The ID of the fixture that is to be removed from the instance

Ritorna:

N/A

Esempio:

physics_remove_fixture(id, my_fix);

Il codice qui sopra rimuoverà il dispositivo con l'"id" memorizzato nella variabile "my_fix" dall'istanza.
