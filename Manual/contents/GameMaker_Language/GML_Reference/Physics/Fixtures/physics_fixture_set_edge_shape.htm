physics_fixture_set_edge_shape

Questa funzione definisce una forma di fixture "bordo". Una forma di bordo è semplicemente una linea che genererà una collisione quando altre fixture la superano, e può essere molto utile per generare (per esempio) un terreno, o per creare bordi intorno a un room. La posizione del bordo è definita usando lo spazio locale, cioè: la posizione x/y dell'istanza è considerata (0,0), quindi questo dovrebbe essere preso in considerazione quando si creano (nell'esempio di codice qui sotto, l'istanza sarebbe stata posta a (0,0) in room per evitare complicazioni).

Sintassi:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

ArgomentoTipo Descrizione
fixture the index of the fixture
local_x1 start x position for the edge
local_y1 start y position for the edge
local_x2 end x position for the edge
local_y2 end y position for the edge

Ritorna:

Esempio:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
    var fix = physics_fixture_create();
    physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
    physics_fixture_bind(fix, id);
    physics_fixture_delete(fix);
    xx += 50;
    y1 = y2;
    y2 = room_height - 50 - irandom(100);
}

Il codice di cui sopra creerà una linea di apparecchi "bordo" con una varietà di altezze sulla lunghezza del room.
