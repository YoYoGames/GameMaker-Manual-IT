attrito fisico

Quando si lega una fixture ad un'istanza usando physics_fixture_bind() questo restituisce un "id" per l'attrezzatura legata. Potete usare questo id per ottenere il valore di attrito della fixture legata(non la fixture "base") in qualsiasi momento usando questa funzione.

Sintassi:

physics_get_friction(fixture)

Argomento Tipo Descrizione
fixture ID apparecchio fisico the id of the bound fixture

Ritorna:

Real

Esempio:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Il codice qui sopra ottiene il valore di attrito corrente per le proprietà fisiche vincolate dell'istanza e poi le imposta su un valore diverso.
