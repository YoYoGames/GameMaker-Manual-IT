physics_fixture_set_box_shape

Questa funzione definisce una forma di scatola per il vostro apparecchio. Prende la metà della larghezza e dell'altezza perché il mondo della fisica usa questo valore molto più dei valori interi di larghezza/altezza per determinare cose come le collisioni.

Sintassi:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

ArgomentoTipo Descrizione
fixture the index of the fixture
halfWidth the half width of the box
halfHeight the half height of the box

Ritorna:

Esempio:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

Il codice qui sopra applicherà una forma di scatola all'apparecchio indicizzato nella variabile "fix_Border" con una larghezza di room e un'altezza di 20 pixel.
