mp_linear_path_object

Questa funzione calcola una linea retta path dalla posizione attuale dell'istanza al punto specificato dai valori "xgoal" e "ygoal". Utilizza la dimensione del passo indicata, la stessa della funzione mp_linear_step(). Il path indicato deve già esistere e sarà sovrascritto dal nuovo path e la funzione restituirà se è stato trovato un path completo (true) o no (false). Un path completo viene trovato solo se non c'è stata collisione con il object o l'istanza specificata e se viene restituito false allora viene ancora generato un path, ma verrà eseguito solo fino alla posizione in cui il path è stato bloccato.

NOTA: Questa funzione non sposta l'istanza. Imposta solo un path ed è necessario utilizzare le funzioni Path per il movimento.

Sintassi:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

Argomento Tipo Descrizione
path Percorso patrimoniale The index of the path to be used
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj ID dell'asset dell'oggetto o dell'istanza The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

Ritorna:

Boolean

Esempio:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

Il codice di cui sopra controlla se c'è una collisione con "obj_Wall" lungo il path tra il object che esegue il codice e la posizione x/y del mouse. Se non c'è collisione e viene generato l'intero path allora inizierà il object lungo il path generato.
