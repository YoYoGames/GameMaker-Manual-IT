mp_grid_clear_all

Con questa funzione si può cancellare una griglia MP da tutte le celle "proibite".

Sintassi:

mp_grid_clear_all(id);

Argomento Tipo Descrizione
id ID griglia MP Index of the mp_grid that is to be used

Ritorna:

Boolean

Esempio:

if !instance_exists(obj_Player)
{
    mp_grid_clear_all(grid);
}

Il codice qui sopra cancellerà la mp_grid indicizzata nella variabile "grid", segnando tutte le celle come libere, se un'istanza di object "obj_Player" non esiste più in room.

Indietro: Pianificazione del movimento
Il prossimo: mp_grid_clear_cell
