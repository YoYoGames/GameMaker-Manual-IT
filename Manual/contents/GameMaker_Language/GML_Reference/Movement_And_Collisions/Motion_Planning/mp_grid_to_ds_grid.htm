mp_grid_to_ds_grid

Con questa funzione potete copiare la griglia MP data in una griglia DS. La griglia DS dovrebbe avere la stessa dimensione della griglia MP, anche se non deve necessariamente esserlo (i dati saranno persi se è più piccola, e se è più grande tutte le celle extra della griglia saranno 0). Una cella nella griglia DS conterrà il valore -1 se era segnalata come occupata nella griglia MP, o 0 se non lo era.

Sintassi:

mp_grid_to_ds_grid(source, destination);

Argomento Tipo Descrizione
source ID griglia MP Index of the mp_grid that is to be used
destination ID griglia DS Index of the ds_grid that is to be used to copy the grid data to.

Ritorna:

Boolean

Esempio:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

Il codice qui sopra creerà una nuova griglia DS e poi copierà i dati della griglia MP contenuti nella variabile "mp_grid" nella nuova griglia DS.
