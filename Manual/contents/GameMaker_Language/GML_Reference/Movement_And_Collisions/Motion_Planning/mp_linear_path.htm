mp_linear_path

Questa funzione calcola una linea retta path dalla posizione attuale dell'istanza al punto specificato dai valori "xgoal" e "ygoal". Utilizza la dimensione del passo indicata, la stessa della funzione mp_linear_step(). Il path indicato deve già esistere e sarà sovrascritto dal nuovo path e la funzione restituirà se è stato trovato un path completo (true) o no (false). Se viene restituito false, allora viene ancora generato un path, ma verrà eseguito solo fino alla posizione in cui il path è stato bloccato.

NOTA: Questa funzione non sposta l'istanza. Imposta solo un path ed è necessario utilizzare le funzioni Path per il movimento.

Sintassi:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);

ArgomentoTipo Descrizione
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

Ritorna:

Esempio:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

Questo fa sì che object controlli e veda se c'è un path lineare dalla sua posizione attuale alla posizione di "obj_Player". Se c'è, allora inizia il path.
