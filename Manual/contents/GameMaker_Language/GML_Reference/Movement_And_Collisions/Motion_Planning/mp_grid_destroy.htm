mp_grid_destroy

Potete usare questa funzione per distruggere la mp_grid indicata (come definita dalla funzione mp_grid_create()) e liberare la memoria allocata. È essenziale che chiamiate questa funzione ogni volta che la griglia MP è finita o potreste potenzialmente ottenere una perdita di memoria, il che significa che il vostro gioco rallenterà nel tempo e alla fine andrà in crash.

NOTA: l'uso delle funzioni mp_grid_* su una griglia dopo che è stata distrutta provocherà un errore.

Sintassi:

mp_grid_destroy(id);

Argomento Tipo Descrizione
id ID griglia MP Index of the mp_grid that is to be destroyed

Ritorna:

N/A

Esempio:

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

Il codice di cui sopra distruggerà la mp_grid indicizzata nella variabile "grid" se la variabile "timer" è uguale a 0, e poi andrà su un altro room.
