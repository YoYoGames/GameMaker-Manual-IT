mp_linear_step

Con questa funzione dite a un'istanza di fare un "passo" verso un punto specifico, specificato dai valori xgoal e ygoal. La dimensione del passo (che è quanti pixel l'istanza dovrebbe muovere ad ogni passo) è indicata da stepsize, e se l'istanza è già nella posizione si fermerà e non si muoverà ulteriormente, contrariamente ad altre funzioni più semplici come move_towards_point(). Lo stepsize è anche la distanza "avanti" che object controllerà ad ogni passo per una collisione, ed è possibile scegliere se l'istanza si ferma ad una collisione con qualsiasi istanza o solo con quelle che sono contrassegnate come solide. La funzione restituirà se ha raggiunto la posizione dell'obiettivo (true) o se ha fallito (false).

NOTA Questa funzione non cerca di fare deviazioni se incontra un ostacolo, semplicemente fallisce e smette di muoversi.

Sintassi:

mp_linear_step(xgoal, ygoal, stepsize, checkall);

ArgomentoTipo Descrizione
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Ritorna:

Esempio:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Il codice di cui sopra farà muovere object verso il mouse ad una velocità di 5 pixel per passo. Una volta raggiunta la posizione del mouse, creerà un object "obj_Explosion" e si distruggerà.
