mp_potential_step_object

Questa funzione permette all'istanza di fare un passo verso una particolare posizione definita da xgoal/ygoal, mentre cerca di evitare gli ostacoli. Quando l'istanza si imbatte in un'istanza di object specificata dall'argomento "obj", cambierà la direzione del movimento per cercare di evitare quell'istanza e muoversi intorno ad essa. Questo approccio non è garantito per funzionare, ma nella maggior parte dei casi facili muoverà effettivamente l'istanza verso l'obiettivo. La funzione restituisce se l'obiettivo è stato raggiunto o meno.

Sintassi:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

Argomento Tipo Descrizione
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj ID dell'asset dell'oggetto o dell'istanza The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

Ritorna:

Boolean

Esempio:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

Il codice di cui sopra prima controlla se ci sono istanze di "obj_Enemy" in room. Se ci sono, trova quella più vicina e memorizza il suo id in una variabile. Questa variabile viene poi usata per dire a mp_potential_step_object di spostare l'istanza con il codice verso la posizione x/y del object che è stata trovata ad una velocità di 5 pixel per passo evitando solo le istanze di "obj_Wall" di object.
