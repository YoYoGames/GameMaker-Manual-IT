mp_grid_get_cell

Con questa funzione è possibile controllare qualsiasi cella della mp_grid per vedere se è stata segnalata come occupata o meno, fornendo l'indice della mp_grid (precedentemente creata) e le coordinate x e y della cella da controllare.

Se è stata occupata o se la posizione controllata è al di fuori dei limiti della griglia, la funzione restituirà -1, altrimenti restituirà 0.

Sintassi:

mp_grid_get_cell(id, x , y);

Argomento Tipo Descrizione
id ID griglia MP Index of the mp_grid that is to be used
x1 Reale The x coordinate of the grid to check.
y1 Reale The y coordinate of the grid to check.

Restituzione:

Real

Esempio:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

Il codice precedente controlla la cella mp_grid che corrisponde alla posizione del mouse e, se è occupata, imposta la variabile image_blend sul rosso, mentre se non è occupata la imposta sul verde.

Indietro: Pianificazione del movimento
