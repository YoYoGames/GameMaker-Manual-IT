cambiamento di posizione

Questa funzione controllerà una posizione per una collisione con qualsiasi istanza nel punto dato, e se c'è, cambierà tutte le istanze in collisione per essere istanze del object scelto. Potete impostare l'argomento "perf" su true che forzerà GameMaker ad eseguire gli eventi Destroy e Clean Up dell'istanza trovata così come l'evento Create della nuova istanza, oppure false a non eseguire nessuno di questi eventi. Si prega di notare che se si sceglie di non eseguire gli eventi Destroy, Clean Up e Create, qualsiasi istanza creata che utilizza una variabile normalmente definita nell'evento Create manderà in crash il gioco perché quella variabile non esisterà più.

Sintassi:

position_change(x, y, obj, perf);

Argomento Tipo Descrizione
x Real The x coordinate of where to change colliding instances.
y Real The y coordinate of where to change colliding instances.
obj Oggetto Attività The new object the calling object will change into.
perf Booleano Whether to perform that new object's Create event (true) or not (false).

Ritorna:

N/A

Esempio:

position_change(200, 200, obj_Bird, true);

Questo cambierà tutte le istanze che si scontrano nel punto (200,200) in un'istanza di "obj_Bird", eseguendo l'evento Create di "obj_Bird" per ognuno di loro nel processo.

Indietro: Collisioni
Il prossimo: position_destroy
