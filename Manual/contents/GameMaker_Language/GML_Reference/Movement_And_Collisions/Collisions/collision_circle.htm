cerchio di collisione

Collision_circle crea una zona circolare del raggio dato dall'utente intorno ai punti x1,y1 e poi controlla questa zona per una collisione con qualsiasi istanza di object specificata dall'argomento "obj". Questo controllo può essere preciso o meno, ma perché le collisioni precise siano abilitate, l'object o l'istanza che si sta controllando deve anche avere collisioni precise abilitate per il suo sprite. In caso contrario, il controllo di default è basato sui bounding box. L'immagine seguente illustra come funziona:

Qui, l'istanza al centro sta usando un cerchio di collisione per controllare la palla objects. Ora, quelle blu non hanno un bounding box preciso e come potete vedere, anche se sprite non sta effettivamente toccando il cerchio, la collisione può ancora avvenire (anche se impostate l'opzione precisa nella funzione a true) poiché il bounding box di quella sprite sovrappone l'area circolare definita da collision_circle. D'altra parte, le palline verdi saranno considerate in collisione solo se l'attuale sprite si sovrappone al cerchio definito. Ricordate, perché le collisioni precise siano considerate, sia il object sprite che la funzione collision devono avere precise marcato come on. Va anche notato che il valore di ritorno della funzione può essere l'id di una qualsiasi delle istanze considerate in collisione.

Notate che invece di un indice object potete fornire un'istanza id per controllare un'istanza specifica, o le parole chiave di istanza all, o other (a seconda dell'evento e dello scopo del codice corrente).

Sintassi:

collision_circle(x1, y1, rad, obj, prec, notme);

Argomento Tipo Descrizione
x1 Real The x coordinate of the center of the circle to check.
y1 Real The y coordinate of the center of the circle to check.
rad Real The radius (distance in pixels from its center to its edge).
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instance collisions.
prec Booleano Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Ritorna:

Instance ID or noone

Esempio:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
    image_index = 1;
}
else image_index = 0;

Il codice qui sopra controllerà un'area circolare con un raggio di 20pixel per una collisione con "obj_Cursor" e se ce n'è una imposterà l'image_index del object che esegue il codice a 1, ma se non c'è imposterà l'image_index del object a 0.
