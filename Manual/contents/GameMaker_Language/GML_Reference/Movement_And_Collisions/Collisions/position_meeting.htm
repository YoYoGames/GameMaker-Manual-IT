riunione_di_posizione

Con questa funzione potete controllare una posizione per una collisione con un'altra istanza o con tutte le istanze di un object. Quando usate questa funzione state controllando un singolo punto in room per un'istanza o un object. Questo controllo sarà fatto contro il bounding box dell'istanza o contro la maschera dell'istanza se tale istanza ha collisioni precise controllate. Se è necessario ottenere l'istanza unica id se il sito object con cui si sta entrando in collisione si dovrebbe usare instance_position(). Questa funzione accetta anche le parole chiave speciali all e other.

Sintassi:

position_meeting(x, y, obj);

Argomento Tipo Descrizione
x Real The x position to check.
y Real The y position to check.
obj ID dell'asset dell'oggetto o dell'istanza The object (or instance id, or the keywords "all" or "other") to check for a collision with.

Ritorna:

Boolean

Esempio:

if mouse_check_button(mb_left)
{
    if (!position_meeting(mouse_x, mouse_y, all))
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

Il codice di cui sopra controlla il pulsante sinistro del mouse, e se è premuto controlla la posizione x/y del mouse per una collisione con qualsiasi istanza. Se non ce n'è nessuna, allora viene creata un'istanza di "obj_Wall".
