collisione_rettangolo

Collision_rectangle usa i primi quattro argomenti (x1,y1,x2,y2) per definire un'area all'interno del corrente room e poi controlla per vedere se qualsiasi object che è definito dall'argomento "obj" è in collisione con quell'area. Questa collisione può essere controllata come precisa o meno, e si può anche scegliere di controllare l'istanza che esegue il codice stesso o meno. Considerate questa immagine:

Qui, l'istanza al centro sta usando un rettangolo di collisione per controllare la palla objects. Ora, quelle blu non hanno un bounding box preciso e come potete vedere, anche se sprite non sta effettivamente toccando il rettangolo, la collisione avverrà comunque (anche se impostate l'opzione precisa nella funzione a true) poiché il bounding box di quel sprite supera il rettangolo di collisione. D'altra parte, le palline verdi saranno considerate in collisione solo se l'attuale sprite supera il rettangolo. Ricordate, perché le collisioni precise siano considerate, sia il object sprite che la funzione di collisione devono avere preciso segnato come on. Va anche notato che il valore di ritorno della funzione può essere l'id di una qualsiasi delle istanze considerate in collisione.

Notate che invece di un indice object potete fornire un'istanza id per controllare un'istanza specifica, o le parole chiave di istanza all, o other (a seconda dell'evento e dello scopo del codice corrente).

Sintassi:

collision_rectangle(x1, y1, x2, y2, obj, prec, notme);

Argomento Tipo Descrizione
x1 Real The x coordinate of the left side of the rectangle to check.
y1 Real The y coordinate of the top side of the rectangle to check.
x2 Real The x coordinate of the right side of the rectangle to check.
y2 Real The y coordinate of the bottom side of the rectangle to check.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueche è più lento) o il suo bounding box in generale (false, più veloce).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Ritorna:

Instance ID or noone

Esempio:

var inst;
inst = collision_rectangle(50, 50, 200, 100, obj_Ball, false, true);
if (inst != noone)
{
    instance_destroy(inst);
}

Questo breve codice usa collision_rectangle per controllare un'area in room da 50x, 50y (in alto a sinistra del rettangolo) a 200x, 200y (in basso a destra del rettangolo) per un'istanza di un object chiamato "obj_ball". Memorizza il valore di ritorno in una variabile temporanea che viene poi controllata per vedere se quel valore è un id di istanza, o la parola chiave nessuno. Se non è nessuno allora usa l'id dell'istanza memorizzata per distruggere il object.
