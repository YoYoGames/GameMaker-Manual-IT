luogo_riunione

Con questa funzione potete controllare una posizione per una collisione con un'altra istanza o con tutte le istanze di un object usando la maschera di collisione dell'istanza che esegue il codice per il controllo. Quando usate questa funzione state effettivamente chiedendo a GameMaker di spostare l'istanza nella nuova posizione, controllare la collisione, tornare indietro e dirvi se una collisione è stata trovata o meno. Questo funzionerà per collisioni precise, ma solo se sia l'istanza che il object che viene controllato hanno maschere di collisione precise selezionate, altrimenti vengono applicate solo le collisioni del bounding box. Questa funzione accetta anche le parole chiave speciali all e other. Si prega di notare che se si ha bisogno di ottenere l'id unico dell'istanza di object con cui si sta collidendo, si dovrebbe usare instance_place().

Si noti che le coordinate x/y date saranno flottate al numero intero più vicino prima che il controllo venga eseguito.

Sintassi:

place_meeting(x, y, obj);

Argomento Tipo Descrizione
x Real The x position to check.
y Real The y position to check.
obj ID dell'asset dell'oggetto o dell'istanza The instance or object to check for.

Ritorna:

Boolean

Esempio:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

Il codice di cui sopra controlla se non c'è una collisione a sinistra dell'istanza e sposta l'istanza se non c'è.
