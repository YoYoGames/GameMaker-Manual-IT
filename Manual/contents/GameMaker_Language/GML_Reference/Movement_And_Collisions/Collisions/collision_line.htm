linea di collisione

Collision_line controlla lungo una linea dal punto x1,y1 al punto x2,y2 per una collisione con qualsiasi istanza di object specificata dall'argomento "obj". questo controllo può essere preciso o no, ma per abilitare le collisioni precise, l'object o l'istanza che si sta controllando deve anche avere collisioni precise abilitate per il suo sprite. Se no, il controllo di default è basato sui bounding box. L'immagine seguente illustra come funziona:

Ricordate, perché le collisioni precise siano considerate, sia il sito object sprite che la funzione di collisione devono avere precise marcate come on. Va anche notato che il valore di ritorno della funzione può essere l'id di una qualsiasi delle istanze considerate in collisione lungo la linea, quindi se tre istanze si sovrappongono alla linea definita, uno qualsiasi dei loro id potrebbe essere il valore di ritorno della funzione.

Notate che invece di un indice object potete fornire un'istanza id per controllare un'istanza specifica, o le parole chiave di istanza all, o other (a seconda dell'evento e dello scopo del codice corrente).

Sintassi:

collision_line(x1, y1, x2, y2, obj, prec, notme);

Argomento Tipo Descrizione
x1 Real The x coordinate of the start of the line.
y1 Real The y coordinate of the start of the line.
x2 Real The x coordinate of the end of the line.
y2 Real The y coordinate of the end of the line.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueche è più lento) o il suo bounding box in generale (false, più veloce).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Ritorna:

Instance ID or noone

Esempio:

with (obj_Enemy)
{
    if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

Questo codice fa sì che tutte le istanze di "obj_Enemy" controllino lungo una linea da 100x, 400y a 100x, 600y per una collisione con se stesse, e se ce n'è una allora vengono distrutte.
