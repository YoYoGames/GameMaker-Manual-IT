collisione_ellisse

Collision_ellipse usa i primi quattro argomenti (x1,y1,x2,y2) per definire la larghezza e l'altezza di un'ellisse all'interno del corrente room e poi controlla per vedere se qualsiasi object che è definito dall'argomento "obj" è in collisione con quell'area. Questa collisione può essere controllata come precisa o meno, e si può anche scegliere di controllare l'istanza che esegue il codice stesso o meno. Considerate questa immagine:

Qui, l'istanza al centro sta usando un'ellisse di collisione per controllare la palla objects. Ora, quelle blu non hanno un bounding box preciso e come potete vedere, anche se sprite non sta effettivamente toccando l'ellisse, la collisione può ancora avvenire (anche se impostate l'opzione precisa nella funzione a true) poiché il bounding box di quel sprite supera l'area ellittica definita da collision_circle. D'altra parte, le palline verdi saranno considerate in collisione solo se l'attuale sprite supera l'ellisse definita. Ricordate, perché le collisioni precise siano considerate sia il object sprite che la funzione collision devono avere precise marcato come on. Va anche notato che il valore di ritorno della funzione può essere l'id di una qualsiasi delle istanze considerate in collisione.

Notate che invece di un indice object potete fornire un'istanza id per controllare un'istanza specifica, o le parole chiave di istanza all, o other (a seconda dell'evento e dello scopo del codice corrente).

Sintassi:

collision_ellipse(x1, y1, x2, y2, obj, prec, notme);

Argomento Tipo Descrizione
x1 Real The x coordinate of the left side of the ellipse to check.
y1 Real The y coordinate of the top side of the ellipse to check.
x2 Real The x coordinate of the right side of the ellipse to check.
y2 Real The y coordinate of the bottom side of the ellipse to check.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueche è più lento) o il suo bounding box in generale (false, più veloce).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).

Ritorna:

Instance ID or noone

Esempio:

if collision_ellipse(50, 50, 200, 100, obj_Player, false, true)
{
    instance_create_layer(obj_Player.x, obj_Player.y, "Effects", obj_Splash);
}

Questo controllerà una zona ellittica entro i limiti di 50x, 50y e 200x, 100y per il object "obj_Player". Se c'è una collisione con quel object, allora creerà un'istanza di "obj_Splash" alle coordinate x/y di obj_Player.
