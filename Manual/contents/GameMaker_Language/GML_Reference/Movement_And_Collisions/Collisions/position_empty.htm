posizione_vuota

Questa funzione controlla se una data posizione entra in collisione con qualsiasi istanza con una maschera di collisione valida nella posizione data.

Sintassi:

position_empty(x, y);

ArgomentoTipo Descrizione
x The x position to check.
y The y position to check.

Ritorna:

Esempio:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if (position_empty(xx, yy))
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Questo controllerà una posizione casuale da qualche parte in room per una collisione e se non ce n'è nessuna creerà un'istanza di obj_Bomb.
