elenco punti di collisione

Questa funzione è la stessa della funzione collision_point() solo che invece di rilevare solo un'istanza in collisione alla volta, rileverà istanze multiple. Si fornisce la posizione x/y del punto da controllare e si può impostare il controllo per essere preciso (nel qual caso tutte le istanze controllate dovranno avere maschere di collisione precise ) e se il controllo deve includere o meno l'istanza chiamante.

Si fornisce anche una lista DS, quindi i id di tutte le istanze che collidono con l'istanza chiamante saranno aggiunte alla fine della lista data. Potete cancellare la lista prima di chiamare questa funzione in modo che contenga solo i risultati di questa chiamata di funzione. Avete anche la possibilità di ordinare le istanze in base alle loro distanze dal punto dato alle loro origini. La funzione restituisce il numero di istanze trovate, o 0 se non ne viene trovata nessuna.

Notate che invece di un indice object potete fornire la parola chiave istanza all, per controllare tutte le istanze nella corrente room, che può includere l'istanza che esegue il codice (a seconda dell'argomento notme ).

Sintassi:

collision_point_list(x, y, obj, prec, notme, list, ordered);

Argomento Tipo Descrizione
x Real The x coordinate of the point to check.
y Real The y coordinate of the point to check.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueche è più lento) o il suo bounding box in generale (false, più veloce).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) o no (false).
list ID elenco DS The DS list to use to store the IDs of colliding instances.
ordered Booleano Whether the list should be ordered by distance (true) o no (false).

Ritorna:

Real (The number of instances found to be in collision)

Esempio:

var _list = ds_list_create();
var _num = collision_point_list(x, y, obj_Enemy, false, true, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Il codice qui sopra controllerà la posizione dell'istanza chiamante per collisioni con istanze di "obj_Enemy". Se ci sono delle collisioni, allora la lista pre-creata viene analizzata e ogni istanza che era nella collisione viene distrutta.
