sposta_contatti_tutti

Questa funzione sposterà l'istanza che esegue il codice di un certo numero di pixel nella direzione specificata finché non incontrerà qualsiasi altra istanza con una maschera valida. Potete usare -1 o 0 per il maxdist essendo un default di 1000px, cioè: GameMaker muoverà l'istanza continuamente su 1000 pixel fino a quando non sarà fuori dalla collisione.

Sintassi:

move_contact_all(dir, maxdist);

ArgomentoTipo Descrizione
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Ritorna:

Esempio:

if !place_meeting(x, y + 1, all)
{
    move_contact_all(270, -1);
}

Il codice di cui sopra controllerà sotto l'istanza per una collisione, e se non ce n'è nessuna, allora la sposterà verso il basso fino a quando non ce n'è o fino a quando 1000 pixel sono stati coperti.
