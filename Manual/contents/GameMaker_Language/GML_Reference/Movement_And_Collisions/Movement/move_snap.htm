spostare_snap

Questa funzione è usata per "agganciare" l'istanza a una griglia di una data dimensione. Verrà agganciata alla posizione corrispondente più vicina sulla griglia "invisibile" che i valori hsnap e vsnap definiscono.

Sintassi:

move_snap(hsnap, vsnap);

ArgomentoTipo Descrizione
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Ritorna:

Esempio:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Il codice di cui sopra controlla tutte le istanze di "obj_Pieces" per vedere se sono snappate in una griglia di 32x32 pixel, e se non lo sono le snappa alla posizione più vicina in quella griglia.
