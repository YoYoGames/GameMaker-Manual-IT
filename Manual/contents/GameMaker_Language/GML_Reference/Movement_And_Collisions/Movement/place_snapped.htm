posto_snapped

Con questa funzione potete controllare e vedere se l'origine di un'istanza (la sua posizione x e y) è allineata a una griglia con i valori hsnap e vsnap specificati da voi.

Sintassi:

place_snapped(hsnap, vsnap);

ArgomentoTipo Descrizione
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Ritorna:

Esempio:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Il codice di cui sopra controlla tutte le istanze di "obj_Pieces" per vedere se sono snappate in una griglia di 32x32 pixel, e se non lo sono le snappa alla posizione più vicina in quella griglia.
