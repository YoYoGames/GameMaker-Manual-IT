sposta_contatto_solido

Questa funzione sposterà l'istanza che esegue il codice di un certo numero di pixel nella direzione specificata fino a quando non incontra un'istanza con solido segnalata come vera. Potete usare -1 o 0 per il maxdist essendo un default di 1000px, cioè: GameMaker muoverà l'istanza continuamente su 1000 pixel fino a quando non sarà fuori dalla collisione.

Sintassi:

move_contact_solid( dir, maxdist );

ArgomentoTipo Descrizione
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Ritorna:

Esempio:

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

Il codice di cui sopra controllerà sotto l'istanza per una collisione con "obj_Floor", e se non ce n'è nessuna, allora la sposterà in basso fino a quando non ci sarà una collisione, o saranno stati coperti 1000 pixel.
