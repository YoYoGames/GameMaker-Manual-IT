spostare_fuori_tutto

Con questa funzione si può dire a un'istanza di muoversi fuori da una collisione in qualsiasi direzione e in qualsiasi numero di pixel ogni passo, con un valore di -1 o 0 per il maxdist essendo un default di 1000px, cioè: GameMaker muoverà l'istanza continuamente su 1000 pixel fino a quando non è fuori dalla collisione.

Sintassi:

move_outside_all(dir, maxdist);

ArgomentoTipo Descrizione
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Ritorna:

Esempio:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

Il codice di cui sopra sposterà l'istanza verso l'alto di 1 pixel per ogni passo in cui è in collisione con qualsiasi altra istanza.
