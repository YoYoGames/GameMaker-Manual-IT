spostare_fuori_solido

Con questa funzione potete dire ad un'istanza di muoversi fuori da una collisione con qualsiasi istanza segnalata come solido in qualsiasi direzione e in qualsiasi numero di pixel per ogni passo, con un valore di -1 o 0 per il maxdist essendo un default di 1000px, cioè: GameMaker muoverà l'istanza continuamente su 1000 pixel fino a quando non sarà fuori dalla collisione.

Sintassi:

move_outside_solid(dir, maxdist);

ArgomentoTipo Descrizione
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Ritorna:

Esempio:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

Il codice di cui sopra andrebbe in un evento di collisione e controlla per vedere se l'istanza "altra" nella collisione è contrassegnata come "solida". Se lo è, sposterà l'istanza fuori dalla collisione.
