salva schermo

Con questa funzione è possibile salvare una schermata del gioco così come appare attualmente nella finestra di gioco.

Il file verrà salvato nella directory di lavoro del gioco, che in Windows si trova a <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ e in Mac a ~/Library/Application Support/[Game Name]/. L'immagine catturata sarà il rendering finale della superficie dell'applicazione del gioco, oppure (se la superficie dell'applicazione è stata disattivata) sarà un'immagine dell'intero display o finestra.

È importante notare che la funzione è stata progettata per essere chiamata nell'evento Draw GUI End, che dovrebbe dare risultati coerenti su tutte le piattaforme. L'uso di questa funzione in qualsiasi altro evento potrebbe non funzionare come previsto e potrebbe dare risultati diversi su diversi target e persino dispositivi.

Se si desidera un maggiore controllo su ciò che viene salvato, si deve utilizzare la funzione screen_save_part().

NOTA Questa funzione non funziona con il target HTML5.

Sintassi:

screen_save(fname);

Argomento Tipo Descrizione
fname Stringa The filename for the screenshot to save

Restituzione:

N/A

Esempio:

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

Il codice qui sopra scatta una schermata del gioco quando l'utente preme il tasto "S". Questa schermata viene salvata in una directory specifica e numerata con una variabile che viene poi incrementata.
