imposta la dimensione della finestra

Con questa funzione potete cambiare le dimensioni della finestra di gioco. Sulle piattaforme desktop questo di solito si riferisce all'area client della finestra, escludendo la barra del titolo e i bordi:

Se si usa questa funzione per cambiare la dimensione della finestra di gioco, il contenuto della finestra sarà scalato per adattarsi con un rapporto 1:1 alla tela di sfondo che manterrà l'immagine nitida e chiara (anche se scalata, se la finestra è diversa dalla dimensione di room o della vista). Per esempio, se avete una finestra di 200 x 150 pixel, e il giocatore allunga la finestra trascinando i bordi, la dimensione finale della finestra può essere di 400 x 300 pixel, ma la tela di disegno è ancora 200 x 150, il che dà un'immagine sfocata perché il rapporto dei pixel è ora 1:2. Tuttavia, se si usa questa funzione per cambiare la dimensione della finestra, il rapporto 1:1 viene mantenuto, anche se l'immagine viene scalata per adattarsi alla nuova dimensione della finestra.

Sintassi:

window_set_size( w, h );

ArgomentoTipo Descrizione
w The new width of the window.
h The new height of the window.

Ritorna:

N/A

Esempio:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

Il codice qui sopra cambierà la dimensione della finestra in base al valore della variabile "global.scale" quando viene premuto il pulsante del mouse.
