window_mouse_get_x

Con questa funzione è possibile ottenere la posizione x del cursore del mouse (in pixel) all'interno del browser se si tratta di un gioco HTML5 o all'interno del display se si tratta di un gioco Windows, Ubuntu (Linux) o macOS.

NOTA: Per le funzioni regolari del mouse vedere la sezione sull'ingresso del mouse

Sintassi:

window_mouse_get_x();

Ritorna:

Esempio:

wx = window_mouse_get_x();

Il codice precedente memorizza l'attuale posizione della finestra sull'asse x del mouse nella variabile "wx".

Indietro: La finestra di gioco
