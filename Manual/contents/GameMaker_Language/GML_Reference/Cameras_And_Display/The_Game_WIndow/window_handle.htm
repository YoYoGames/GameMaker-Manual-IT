handle_finestra

Con questa funzione potete ottenere il valore id interno di Windows (l'HWND, un puntatore). Questa funzione è davvero utile solo per gli scrittori di extension che hanno bisogno di window handle per chiamare le API di Windows nel codice DLL (il puntatore restituito dovrebbe essere castato in un string e poi nel C++ semplicemente castato in un HWND). La tabella qui sotto mostra le piattaforme supportate insieme a ciò che restituiscono:

Piattaforma Restituisce
Windows Finestra HWND
macOS Classe NSWindow
Ubuntu (Linux) Impugnatura XWindow
HTML5 Tela ID

Sintassi:

window_handle();

Ritorna:

Pointer

Esempio:

win_id = windows_handle();
