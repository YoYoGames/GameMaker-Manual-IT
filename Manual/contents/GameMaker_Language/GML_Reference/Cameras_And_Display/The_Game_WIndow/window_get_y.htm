ottenere la finestra_y

Con questa funzione è possibile ottenere la posizione y (in pixel) all'interno del browser se si tratta di un gioco HTML5 o all'interno del display se si tratta di un gioco Windows, Ubuntu (Linux) o macOS.

Sintassi:

window_get_y();

Ritorna:

Esempio:

wy = window_get_y();

Il codice precedente memorizza la posizione corrente della finestra dell'asse y nella variabile "wy".

Indietro: La finestra di gioco
Il prossimo: window_get_cursor
