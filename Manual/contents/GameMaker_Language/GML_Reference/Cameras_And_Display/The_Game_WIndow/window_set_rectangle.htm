impostare il rettangolo della finestra

Con questa funzione puoi impostare la posizione della finestra di gioco all'interno del browser (HTML5) o del display (Windows, Ubuntu (Linux) o macOS) e impostare anche la scala della finestra. Per maggiori informazioni sulla posizione e la dimensione della finestra, vedi window_set_position() e window_set_size().

Sintassi:

window_set_rectangle(x, y, w, h);

ArgomentoTipo Descrizione
x The new x coordinate of the window.
y The new y coordinate of the window.
w The new width of the window.
h The new height of the window.

Ritorna:

Esempio:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

Il codice di cui sopra imposterà la finestra di gioco per occupare l'intera area di visualizzazione (o il browser o lo schermo, a seconda della piattaforma di destinazione).

Indietro: La finestra di gioco
Il prossimo: window_set_cursor
