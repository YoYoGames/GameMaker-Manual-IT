finestra_ha_focus

Con questa funzione potete interrogare lo stato della finestra (o della scheda) e se perde il focus la funzione restituirà false altrimenti restituirà true. Nella maggior parte dei casi si può semplicemente usare la funzione os_is_paused() per testare questo, ma in alcuni casi molto specifici (per esempio i giochi su Chrome Apps) quella funzione non si attiverà, nel qual caso si dovrebbe usare invece questa funzione.

NOTA: Questa funzione è valida solo per HTML5, Windows, e MacOS.

Sintassi:

window_has_focus();

Ritorna:

Esempio:

if !window_has_focus()
{
    PauseGame();
}

Il codice di cui sopra controllerà per vedere se la finestra di gioco è a fuoco o no, e se la funzione ritorna false verrà chiamata una funzione.

Indietro: La finestra di gioco
