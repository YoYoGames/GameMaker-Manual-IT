visualizza_set_ui_visibilità

Questa funzione può essere usata per mostrare o nascondere l'interfaccia utente di sistema solo su Android. La funzione richiede di fornire uno o più flag di sistema come valore intero. Quando si usa più di un singolo flag, questi devono essere uniti usando il bitwise "or", come mostrato nell'esempio qui sotto. Potete trovare un elenco di flag di sistema Android qui.

Sintassi:

display_set_ui_visibility(flags);

Argomento Tipo bandiere
flags Real The Android SYSTEM flags to use (as an integer value).

Ritorna:

N/A

Esempio:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Il codice precedente userà i flag di sistema Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" e "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" per impostare la visibilità del display.
