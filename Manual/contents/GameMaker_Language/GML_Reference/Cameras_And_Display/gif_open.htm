aprire una gif

Con questa funzione potete creare un'immagine vuota in formato GIF, pronta per essere aggiunta ai dati. Quando chiamate la funzione, dovete specificare la larghezza e l'altezza della GIF (in pixel, e non c'è un limite superiore alla dimensione se non quello della memoria disponibile), e la funzione restituirà il valore ID unico usato per identificare il regalo nelle funzioni successive, oppure restituirà -1 se la gif non può essere inizializzata (per esempio, se la larghezza/altezza è troppo grande per la memoria disponibile). Potete anche specificare un argomento opzionale per impostare il "colore chiaro" per la GIF. Questo è un valore RGB (nessuna componente alfa), e cancellerà la gif a questo colore prima che vi venga aggiunto qualcosa. Se non si fornisce un argomento per il colore chiaro, verrà usato il colore predefinito nero.

Si noti che quando si usa questa funzione è necessario chiamare gif_save() per terminare la creazione della GIF prima di aprire un altro file per la registrazione (quindi per ogni gif_open() ci deve essere un accompagnamento gif_save()).

Sintassi:

gif_open(width, height, [clear_colour]);

Argomento Tipo Descrizione
width Real The width of the gif to create
height Real The height of the gif to create
[clear_colour] Colore OPTIONAL Il colore (RGB) per cancellare la gif a

Ritorna:

GIF ID

Esempio:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Il codice qui sopra creerà un file immagine GIF con 30 fotogrammi presi dalla superficie dell'applicazione e poi lo salverà.
