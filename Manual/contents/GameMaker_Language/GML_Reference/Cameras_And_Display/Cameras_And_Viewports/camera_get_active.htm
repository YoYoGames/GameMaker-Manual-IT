aggiungi attivo

Questa funzione può essere usata per recuperare il valore ID unico della telecamera attualmente attiva.

Sintassi:

camera_get_active()

Ritorna:

Camera ID

Esempio:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = _active;
}

Il codice di cui sopra ottiene l'ID della telecamera attiva e imposta la telecamera di visualizzazione per la porta[0] per utilizzarla.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: view_camera
