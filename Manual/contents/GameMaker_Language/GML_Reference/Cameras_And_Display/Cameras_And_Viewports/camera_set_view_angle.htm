angolo di visuale della telecamera

Si può usare questa funzione per aggiornare l'angolo della telecamera all'interno di room. Si dà il valore unico dell'ID della telecamera (come restituito dalle diverse camera_create() funzioni) e poi date l'angolo che desiderate desiderate impostare la vista della telecamera. Il valore di default è 0° con valori positivi che ruotano la telecamera in senso antiorario, cioè: impostando il valore a 90 la telecamera ruoterà di 90° a sinistra.

Sintassi:

camera_set_view_angle(camera_id, angle)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera
angle The new angle to set the camera view to

Ritorna:

Esempio:

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

Il codice di cui sopra imposterà l'angolo della telecamera per la telecamera assegnata alla porta di visualizzazione[0].
