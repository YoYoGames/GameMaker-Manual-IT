vista_set_camera

Questa funzione imposta una telecamera su una specifica porta di visualizzazione. Si indica la porta di visualizzazione da impostare (da 0 a 7) e si fornisce il valore ID unico della telecamera (come restituito dalle funzioni camera_create() o quando usate view_get_camera()). Se dai un valore di -1 come ID della telecamera allora stai rimuovendo una telecamera dalla porta di visualizzazione e nota che se quella porta di visualizzazione è abilitata e visibile potresti ottenere dei risultati imprevedibili.

Sintassi:

view_set_camera(view_port, camera_id)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Esempio:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

Il codice di cui sopra creerà una nuova telecamera e poi le assegnerà la view port[0].

Indietro: Telecamere e porte di visualizzazione
