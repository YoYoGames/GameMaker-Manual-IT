vista_get_camera

Questa funzione può essere usata per recuperare il valore unico dell'ID della telecamera assegnato alla data porta di visualizzazione (da 0 a 7). Se nessuna telecamera è assegnata, la funzione restituisce -1.

Sintassi:

view_get_camera(view_port)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)

Ritorna:

Camera ID

Esempio:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Il codice di cui sopra recupererà l'ID della telecamera assegnata alla view port[0] e poi cambierà la sua posizione.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: view_get_visible
