vista_telecamera_y

Questa funzione può essere usata per recuperare la posizione y della vista per una data telecamera. Si noti che questa funzione è valida solo per le telecamere create utilizzando camera_create_view() o per quelle aggiunte nell'editor room editor.

Sintassi:

camera_get_view_y(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Esempio:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

Il codice qui sopra recupera la posizione della telecamera assegnata alla view port[0] e poi disegna il testo relativo a quella posizione.

Indietro: Telecamere e porte di visualizzazione
