camera_get_begin_script

Questa funzione può essere usata per recuperare l'ID della funzione di script assegnata come begin script per la telecamera data. Se nessuna funzione script è assegnata, la funzione restituirà -1.

Sintassi:

camera_get_begin_script(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Script Function or -1 if no function is assigned

Esempio:

var scr = camera_get_begin_script(camera_view[0]);
if scr != cutScene
{
    camera_set_update_script(camera_view[0], cutScene);
}

Il codice di cui sopra controlla la funzione script assegnata come begin script per la telecamera assegnata alla view port[0] e se non è "cutScene" è impostata su quella funzione.
