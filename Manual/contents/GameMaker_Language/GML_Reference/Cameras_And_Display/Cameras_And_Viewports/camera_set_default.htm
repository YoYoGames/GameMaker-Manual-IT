camera_set_default

Questa funzione può essere utilizzata per impostare la telecamera di default per utilizzare una telecamera personalizzata che avete precedentemente creato utilizzando una delle camera_create() funzioni. Quando si crea un room senza porte di vista attive o telecamere di visualizzazione, GameMaker utilizza comunque una telecamera per mostrare l'azione nel gioco. Questa telecamera è chiamata telecamera di default e può essere impostata e manipolata (e anche distrutta) proprio come qualsiasi altra telecamera.

Sintassi:

camera_set_default(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Esempio:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

Il codice qui sopra crea una nuova telecamera e poi la assegna come telecamera predefinita da usare.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: camera_get_view_mat
