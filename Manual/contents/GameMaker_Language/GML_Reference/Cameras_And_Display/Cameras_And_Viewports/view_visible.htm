visibile

Questa variabile può essere usata per scoprire se una particolare porta di visualizzazione è attualmente visibile o meno. Potete anche impostare questa variabile per "accendere" o "spegnere" efficacemente una vista impostando il valore a true (visibile) o false (invisibile). Si noti che anche se si ha una porta di vista impostata su visible, se le porte di vista non sono abilitate (usando la variabile incorporata view_enabled o abilitandole nell'editor di Room ) allora non saranno disegnate sullo schermo.

Sintassi:

view_visible[0 ... 7];

Ritorna:

(true: enabled, false: disabled)

Esempio:

if !view_visible[0]
{
    view_visible[0] = true;
}

Il codice qui sopra controlla se view[0] è visibile o no e se non lo è, lo imposta come visibile.
