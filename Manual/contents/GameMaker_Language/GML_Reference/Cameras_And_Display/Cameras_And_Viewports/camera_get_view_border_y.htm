camera_get_view_border_y

Questa funzione può essere utilizzata per recuperare il valore del bordo per l'oggetto/istanza seguente della telecamera data lungo l'asse y (bordo verticale). Il valore di ritorno sarà in pixel.

Sintassi:

camera_get_view_border_y(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Esempio:

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

Il codice qui sopra recupera i valori di xborder e yborder della telecamera assegnata alla view port[0] e poi controlla se corrisponde al valore dato. Se non lo fa, allora la telecamera viene impostata sul valore dato.
