imposta la porta d'ingresso

Questa funzione può essere usata per impostare la larghezza della porta della vista data. Si dà l'indice della porta di visualizzazione (da 0 a 7) e la nuova larghezza da utilizzare.

Sintassi:

view_set_wport(view_port, w)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)
w The new width (in pixels)

Ritorna:

Esempio:

if view_get_wport(0) != (display_get_width () / 2)
{
    view_set_wport(0, display_get_width() / 2);
}

Il codice di cui sopra controllerà la larghezza della porta di visualizzazione[0] e se non è la metà della larghezza del display è impostata su quel valore.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: view_set_hport
