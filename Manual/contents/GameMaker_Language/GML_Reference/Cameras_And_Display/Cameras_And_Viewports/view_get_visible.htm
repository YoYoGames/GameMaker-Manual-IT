visualizza_get_visibile

Questa funzione può essere usata per controllare la visibilità di una data porta di vista. La funzione restituisce true se è visibile e false se non lo è.

Sintassi:

view_get_visible(view_port)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)

Ritorna:

Boolean

Esempio:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Il codice di cui sopra controllerà per vedere se la porta della vista[0] è visibile e se non lo è viene impostata su visible.

Indietro: Telecamere e porte di visualizzazione
