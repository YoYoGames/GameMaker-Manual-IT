vista_set_visibile

Questa funzione può essere usata per impostare la visibilità di una data porta di vista. La funzione prende l'indice della porta della vista (da 0 a 7) e un booleano true se è visibile e false se non lo è.

Sintassi:

view_set_visible(view_port, visible)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)
visible Visibility toggle (true è visibile e false è invisibile)

Ritorna:

N/A

Esempio:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Il codice di cui sopra controllerà per vedere se la porta della vista[0] è visibile e se non lo è viene impostata su visible.
