obiettivo della telecamera

Potete usare questa funzione per impostare il follow target della telecamera all'interno di room. Date il valore unico dell'ID della telecamera (come restituito dalle diverse camera_create() funzioni) e poi dai l'istanza di destinazione o l'ID di object su cui vuoi impostare la vista della telecamera. Si noti che se si sceglie un ID object e c'è più di un'istanza di quel object in room, non c'è modo per GameMaker di sapere quale istanza si desidera seguire e quindi potrebbe essere una qualsiasi di esse.

Sintassi:

camera_set_view_target(camera_id, instance_id/object_id)

Argomento Tipo Descrizione
camera_id ID della macchina fotografica The unique camera ID value returned when you created the camera
instance_id/object_id ID dell'asset dell'oggetto o dell'istanza Instance or object to have the camera target for following

Ritorna:

N/A

Esempio:

camera_set_view_target(view_camera[0], id);

Il codice di cui sopra imposterà il target della telecamera assegnata alla view port[0] su quello dell'istanza che esegue il codice.

Indietro: Telecamere e porte di visualizzazione
