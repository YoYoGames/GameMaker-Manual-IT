obiettivo della telecamera_get_view_target

Questa funzione può essere usata per recuperare il follow target della telecamera data. Il valore di ritorno può essere un indice object, un ID di istanza o -1 se non è stato impostato alcun follow target.

Sintassi:

camera_get_view_target(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Object Asset, Instance ID or -1

Esempio:

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_Player);
}

Il codice qui sopra recupera il follow target della telecamera assegnata alla view port[0] e poi lo controlla per vedere se corrisponde all'indice object "obj_Player". Se non corrisponde, allora la telecamera è impostata per seguire un'istanza di quel object.
