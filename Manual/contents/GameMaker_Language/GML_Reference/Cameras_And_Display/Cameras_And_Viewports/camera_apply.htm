applicare la telecamera

Questa funzione applicherà immediatamente le impostazioni della telecamera date alla vista della telecamera corrente in fase di rendering. Mentre è possibile impostare qualsiasi vista della telecamera utilizzando le funzioni specifiche di camera_set_*, le nuove impostazioni non saranno utilizzate per il rendering fino al prossimo fotogramma di gioco, ma con questa funzione è possibile apportare tali modifiche immediatamente. Se state usando più viste allora la funzione dovrebbe essere usata nel Draw Event e potete usare la variabile view_current per applicare selettivamente la telecamera data solo alla vista della telecamera attualmente in fase di rendering. Notate che questo NON cambia il valore dell'ID della telecamera, ma solo le impostazioni per quella telecamera, e che se usate questa funzione sulla telecamera di default (cioè: la telecamera interna che GameMaker usa quando nessuna telecamera della vista è attiva) le modifiche saranno sovrascritte nel prossimo fotogramma di gioco dalle impostazioni interne per il disegno di room.

Sintassi:

camera_apply(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera.

Ritorna:

Esempio:

if view_current == 0
{
    camera_apply(cutscene_cam);
}

Il codice di cui sopra (in un Draw Event) controlla per vedere quale vista della telecamera è attualmente in fase di rendering e se è la vista della telecamera[0] allora le impostazioni per la telecamera a cui fa riferimento la variabile "cutscene_cam" sono applicate ad essa.
