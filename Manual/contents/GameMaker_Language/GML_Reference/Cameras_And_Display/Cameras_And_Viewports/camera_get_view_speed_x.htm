camera_get_view_speed_x

Questa funzione può essere utilizzata per recuperare la velocità di movimento della telecamera data lungo l'asse x (movimento orizzontale). Il valore di ritorno sarà in pixel per fotogramma di gioco.

Sintassi:

camera_get_view_speed_x(camera_id)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera

Ritorna:

Esempio:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

Il codice qui sopra recupera la xspeed e la yspeed della telecamera assegnata alla view port[0] e poi controlla se corrisponde al valore dato. Se non lo fa, allora la telecamera viene impostata su quella velocità.

Indietro: Telecamere e porte di visualizzazione
