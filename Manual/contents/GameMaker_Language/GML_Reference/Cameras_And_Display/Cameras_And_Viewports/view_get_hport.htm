vista_get_hport

Questa funzione può essere usata per recuperare l'altezza della porta della vista data.

Sintassi:

view_get_hport(view_port)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)

Ritorna:

Real

Esempio:

if view_get_hport(0) != (display_get_height () / 2)
{
    view_set_hport(0, display_get_height() / 2);
}

Il codice di cui sopra controllerà l'altezza della porta della vista[0] e se non è la metà dell'altezza del display viene impostata a quel valore.

Indietro: Telecamere e porte di visualizzazione
