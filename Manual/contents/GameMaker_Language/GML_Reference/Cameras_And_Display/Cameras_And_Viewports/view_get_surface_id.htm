visualizza_get_surface_id

Questa funzione può essere usata per recuperare il valore ID unico per la superficie assegnata alla data porta di vista (restituirà -1 se nessuna superficie è stata assegnata).

Sintassi:

view_get_surface_id(view_port)

ArgomentoTipo Descrizione
view_port The view port to target (0 - 7)

Ritorna:

Surface ID

Esempio:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

Il codice di cui sopra controllerà per vedere se una superficie è stata assegnata alla porta della vista[0] e se non lo è, allora ne viene assegnata una.

Indietro: Telecamere e porte di visualizzazione
