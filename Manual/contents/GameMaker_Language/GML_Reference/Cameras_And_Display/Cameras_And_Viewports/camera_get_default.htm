camera_get_default

Questa funzione può essere usata per recuperare il valore unico dell'ID della telecamera di default (la telecamera che GameMaker usa quando nessuna vista o porta della telecamera è attiva in un gioco room).

Sintassi:

camera_get_default()

Ritorna:

Camera ID

Esempio:

var def = camera_get_default();
view_camera[0] = def;

Il codice precedente ottiene l'ID della telecamera per la telecamera di default e imposta la telecamera di visualizzazione per la porta[0] per usarla.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: camera_get_active
