camera_set_view_mat

Questa funzione imposterà la matrice di vista per una data telecamera. Si fornisce il valore unico dell'ID della telecamera (come restituito dalle diverse camera_create() funzioni) e una matrice di vista da applicare. Puoi trovare maggiori informazioni sulla creazione di matrici di vista nella sezione Funzioni di matrice, in particolare matrix_build_lookat().

Nota che se la tua telecamera fa il tracking automatico object - cioè: è stata creata usando camera_create_view() con un indice object / instance ID che non è -1, o stai impostando una telecamera definita nell'editor Room, o stai impostando la telecamera di default - allora questa matrice verrà sovrascritta al prossimo fotogramma di gioco.

Sintassi:

camera_set_view_mat(camera_id, matrix)

ArgomentoTipo Descrizione
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Ritorna:

Esempio:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Il codice di cui sopra crea una nuova telecamera e la assegna alla view port[0]. Questa telecamera ha poi le sue matrici di vista e di proiezione impostate.

Indietro: Telecamere e porte di visualizzazione
Il prossimo: camera_set_proj_mat
