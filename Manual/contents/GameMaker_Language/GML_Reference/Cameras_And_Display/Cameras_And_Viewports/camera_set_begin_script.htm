camera_set_begin_script

Questa funzione può essere usata per impostare una funzione di script che sarà chiamata all'inizio di ogni fotogramma di gioco in cui la telecamera è assegnata a una porta di vista visibile e attiva. Si fornisce il valore unico dell'ID della telecamera (come restituito dalle diverse funzioni di camera_create() ) e l'ID della funzione da chiamare. L'ordine in cui scripts è collegato alle telecamere e l'effettivo rendering della vista della telecamera è il seguente:

  • Le telecamere per tutte le porte di vista visibili e attive hanno la loro funzione di aggiornamento chiamata
  • Poi per ogni vista
    • Viene chiamata la funzione di inizio della telecamera per quella vista
    • Vengono eseguiti gli eventi di disegno per quella vista
    • Viene chiamata la funzione end per la telecamera
  • Passare alla prossima vista visibile attiva e ripetere...

Sintassi:

camera_set_begin_script(camera_id, script)

Argomento Tipo Descrizione
camera_id ID della macchina fotografica The unique camera ID value returned when you created the camera.
script Funzione Script The script function to run each game frame

Ritorna:

N/A

Esempio:

camera_set_begin_script(view_camera[0], beginCamera);

Il codice qui sopra imposta la funzione begin script per la telecamera assegnata alla porta di visualizzazione [0].

Indietro: Telecamere e porte di visualizzazione
Il prossimo: camera_set_end_script
