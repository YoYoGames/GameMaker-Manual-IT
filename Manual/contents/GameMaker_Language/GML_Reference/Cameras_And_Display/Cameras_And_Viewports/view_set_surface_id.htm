imposta_superficie_id

Con questa variabile potete impostare il contenuto di una porta di vista da disegnare su una superficie. Quando si lavora con le superfici, è spesso richiesto di catturare l'intera regione visibile dello schermo alla superficie, e quindi la si assegnerebbe a una view port usando questa funzione. Questo significa che tutto ciò che viene mostrato nella porta di vista scelta sarà ora disegnato sulla superficie assegnata e il contenuto di quella porta di vista non sarà più visualizzato, il che significa che dovrete o:

  • Abilitare una nuova vista e disegnare la superficie solo in quella vista (usando view_current per controllare quale vista viene disegnata)
  • Disegna la superficie nell'evento Draw GUI o Post Draw di un'istanza, poiché questi eventi sono indipendenti dalle viste.

Quando si usa questa funzione si dà l'indice della porta della vista (da 0 a 7) e un indice di superficie (o il application_surface o il valore di indice unico restituito dalla funzione surface_create()) o, se una superficie è stata precedentemente assegnata e si desidera rimuoverla, un valore di -1. Per ulteriori esempi sull'impostazione della porta della vista su una superficie, vedere la variabile view_surface_id.

Sintassi:

view_set_surface_id(view_port, surf)

Argomento Tipo Descrizione
view_port Real The view port to target (0 - 7)
surf ID superficie The surface to apply to the view

Ritorna:

Real

Esempio:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

Il codice di cui sopra controllerà per vedere se una superficie è stata assegnata alla porta della vista[0] e se non lo è, allora ne viene assegnata una.
