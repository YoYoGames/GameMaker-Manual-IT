gif_add_surface

Con questa funzione potete salvare un fotogramma in un file GIF da una superficie. Fornite l'ID del file GIF (come restituito dalla funzione gif_open()) e un ID di superficie (come restituito dalla funzione surface_create()), così come un tempo di ritardo tra i fotogrammi. Il tempo di ritardo è calcolato in 1/100 di secondo, ma notate che una volta che la GIF è stata creata, se la usate per scopi promozionali per esempio, i diversi browser possono interpretare il ritardo del fotogramma in modo leggermente diverso - vedete qui per maggiori dettagli. Potete (opzionalmente) fornire valori di offset X e Y per la superficie da aggiungere alla GIF, nel qual caso la parte disegnata nella GIF inizierà dall'offset e non dalla posizione predefinita (0,0), permettendovi di selezionare solo una sezione della superficie da aggiungere. Potete anche fornire un valore di quantizzazione, che è un intero compreso tra 0 e 3, dove 0 è la quantizzazione completa e 3 è la quantizzazione zero (valori più bassi avranno un successivo degrado della qualità, ma creano una GIF più piccola). Si noti che non c'è molta differenza tra la qualità predefinita di 2 e la più alta di 3, e l'uso di 3 rallenterà significativamente la creazione della GIF, quindi bisogna fare attenzione quando si usa il valore massimo.

Notate che non ci sono limiti incorporati sul numero di fotogrammi o sulla dimensione di quei fotogrammi nella vostra GIF, ma se usate troppa memoria allora la funzione fallirà e restituirà -1, altrimenti restituirà 0.

Sintassi:

gif_add_surface(gif_index, surface, delay_time, [xoffset], [yoffset]);

Argomento Tipo Descrizione
gif_index ID GIF The ID of gif to add the surface data to
surface ID superficie The ID of the surface to use as the added frame
delay_time Real The delay time for the frame (in 100ths of a second)
[xoffset] Real OPTIONAL L'offset lungo l'asse X della superficie da aggiungere da
[yoffset] Real OPTIONAL L'offset lungo l'asse Y della superficie da aggiungere da
[quantization] Real OPTIONAL La quantità di quantizzazione da 0 a 3 (0 qualità più bassa, 3 qualità più alta, il default è 2)

Ritorna:

Real

Esempio:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Il codice qui sopra creerà un file immagine GIF con 30 fotogrammi presi dalla superficie dell'applicazione e poi lo salverà.
