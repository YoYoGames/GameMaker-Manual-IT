gif_save

Con questa funzione potete salvare un'animazione GIF. Si fornisce l'indice GIF (come restituito dalla funzione gif_open()) e un nome di file con cui salvarla. Notate che GameMaker non aggiunge automaticamente il file .gif extension , quindi dovreste includerlo come parte del nome del file string se volete che il file salvato sia identificato come una GIF. La GIF creata sarà pallettizzata usando la Universal 884 Palette (vedere qui per maggiori informazioni).

Notate che se la funzione ha successo e la GIF viene salvata correttamente, allora restituirà 0, altrimenti restituirà -1.

Sintassi:

gif_save(gif_index, fname);

ArgomentoTipo Descrizione
gif_index The ID of gif to save
fname The filename to use for the gif

Ritorna:

Esempio:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Il codice qui sopra creerà un file immagine GIF con 30 fotogrammi presi dalla superficie dell'applicazione e poi lo salverà.
