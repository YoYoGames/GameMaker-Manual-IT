imposta colore finestra

Questa funzione può impostare il colore di sfondo della finestra di gioco. Questo colore rappresenta quello che verrà utilizzato per quelle aree della finestra di gioco che non sono occupate da alcuna vista. L'immagine seguente lo illustra:

L'immagine qui sopra ha due viste con due porte di visualizzazione, ciascuna disegnata in posizioni diverse. Questo allunga la finestra di gioco per ospitare entrambe le porte e usa il colore della finestra per colorare lo sfondo dove non viene mostrata alcuna vista.

Sintassi:

window_set_colour(colour);

ArgomentoTipo Descrizione
colour The colour to set the region.

Ritorna:

Esempio:

if window_get_colour() != c_black
{
    window_set_colour(c_black);
}

Il codice qui sopra controllerà il colore della finestra per vedere se è impostato come nero o no, e se non lo è lo imposta come nero.
