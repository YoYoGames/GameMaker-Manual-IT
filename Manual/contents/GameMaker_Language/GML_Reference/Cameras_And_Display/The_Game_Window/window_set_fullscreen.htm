imposta finestra_fullscreen

Con questa funzione è possibile impostare la finestra di gioco per essere a schermo intero (true) o no (false). Si prega di notare che per il target macOS, è necessario aver deselezionato l'opzione "Start In Fullscreen" e selezionato l'opzione "Allow the player to resize the game window" nelle Game Options, altrimenti questa funzione fallirà. Nota anche che questa funzione non funzionerà su HTML5 a meno che non sia aggiunta come callback "cliccabile" (vedi qui per maggiori dettagli).

Sintassi:

window_set_fullscreen(full);

ArgomentoTipo Descrizione
full Whether to set the screen to fullscreen (true) or not (false).

Ritorna:

Esempio:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

Il codice di cui sopra controlla la pressione di un pulsante del mouse e poi imposta la finestra a schermo intero se non lo è già, o la imposta a finestra se lo è già.
