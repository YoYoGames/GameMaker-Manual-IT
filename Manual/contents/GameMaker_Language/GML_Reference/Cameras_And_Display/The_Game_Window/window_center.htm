centro_finestra

Con questa funzione puoi centrare la finestra di gioco nel display quando il modulo di destinazione è Windows, Ubuntu (Linux) o macOS, oppure puoi centrarla nel browser se il modulo di destinazione è HTML5. Questa funzione non ha effetto su nessun altro dispositivo.

NOTA Se avete ridimensionato la finestra di gioco nel passo corrente (per esempio, passando da schermo intero a finestra o usando window_set_size()), e volete centrare la nuova finestra, questa funzione dovrebbe essere chiamata almeno un passo dopo (per esempio, usando un alarm) altrimenti non funzionerà correttamente.

Sintassi:

window_center();

Ritorna:

Esempio:

if !window_get_fullscreen()
{
    window_center();
}

Il codice qui sopra centrerà la finestra di gioco all'interno del display se non è in modalità schermo intero.
