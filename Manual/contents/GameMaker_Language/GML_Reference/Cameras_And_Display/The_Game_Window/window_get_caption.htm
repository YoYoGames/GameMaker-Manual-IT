aggiungi didascalia alla finestra

Questa funzione restituisce la didascalia della finestra (questo è il testo che appare nella parte superiore della finestra, accanto alla sua icona) e per impostazione predefinita questa mostra la didascalia del room in cui vi trovate attualmente.

Sintassi:

window_get_caption();

Ritorna:

Esempio:

if window_get_caption() != ""
{
    window_set_caption("");
}

Il codice qui sopra controllerà la didascalia della finestra per vedere se ha del testo o no, e se ce l'ha la imposta su un string vuoto in modo che non venga visualizzata alcuna didascalia.
