imposta la didascalia della finestra

Con questa funzione puoi cambiare o impostare la didascalia della finestra di room in cui ti trovi attualmente. Questa didascalia appare nella parte superiore della finestra, accanto all'icona del gioco, quando il gioco non è in modalità schermo intero.

Sintassi:

window_set_caption(caption);

ArgomentoTipo Descrizione
caption The new caption.

Ritorna:

Esempio:

if window_get_caption() != ""
{
    window_set_caption("");
}

Il codice qui sopra controllerà la didascalia della finestra per vedere se ha del testo o no, e se ce l'ha la imposta su un string vuoto in modo che non venga visualizzata alcuna didascalia.

Indietro: La finestra di gioco
