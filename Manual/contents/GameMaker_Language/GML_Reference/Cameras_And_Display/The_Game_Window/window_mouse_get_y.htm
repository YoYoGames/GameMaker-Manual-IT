window_mouse_get_y

Con questa funzione è possibile ottenere la posizione y del cursore del mouse (in pixel) all'interno del browser se si tratta di un gioco HTML5 o all'interno del display se si tratta di un gioco Windows, Ubuntu (Linux) o macOS.

NOTA: Per le funzioni regolari del mouse vedere la sezione sull'ingresso del mouse

Sintassi:

window_mouse_get_y();

Ritorna:

Esempio:

wy = window_mouse_get_y();

Il codice precedente memorizza l'attuale posizione della finestra dell'asse y del mouse nella variabile "wy".

Indietro: La finestra di gioco
Il prossimo: window_mouse_set
