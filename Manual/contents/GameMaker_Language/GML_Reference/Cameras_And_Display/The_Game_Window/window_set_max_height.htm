imposta altezza massima della finestra

Questa funzione può essere usata per impostare un'altezza massima della finestra per il vostro gioco. Se si abilita l'opzione di ridimensionamento della finestra nelle Opzioni di gioco per la piattaforma di destinazione, allora il giocatore può ridimensionare la finestra di gioco a qualsiasi dimensione desiderata, tuttavia utilizzando questa funzione puoi limitare l'altezza massima alla dimensione da te specificata. Se volete tornare al comportamento di default (cioè: nessun minimo), allora usate un valore di -1.

NOTA: Questa funzione è disponibile solo sulle piattaforme Windows, macOS e Linux.

Sintassi:

window_set_max_height(height);

ArgomentoTipo Descrizione
height The maximum height in pixels for the game window.

Ritorna:

Esempio:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

Il codice qui sopra imposterà la larghezza e l'altezza minima e massima della finestra di gioco.

Indietro: La finestra di gioco
Il prossimo: window_has_focus
