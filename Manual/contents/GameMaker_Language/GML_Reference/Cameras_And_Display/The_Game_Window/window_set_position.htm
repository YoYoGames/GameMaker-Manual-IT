posizione della finestra

Con questa funzione è possibile impostare la finestra di gioco in una posizione specifica all'interno del display (su macOS, Linux(Ubuntu) e Windows) o all'interno del browser (HTML5).

NOTA: se il vostro gioco HTML5 usa un index.html personalizzato e che imposta il canvas in una posizione fissa, allora questa funzione non avrà alcun effetto sulla posizione della finestra.

Sintassi:

window_set_position(x, y);

ArgomentoTipo Descrizione
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Ritorna:

Esempio:

window_set_position(0, 0);

Il codice di cui sopra posizionerà la finestra di gioco nell'angolo superiore sinistro del browser o del display (a seconda del modulo di destinazione utilizzato).

Indietro: La finestra di gioco
