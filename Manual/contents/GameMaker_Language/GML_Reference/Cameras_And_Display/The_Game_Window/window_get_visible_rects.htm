ottieni i diritti visibili dalla finestra

Con questa funzione potete trovare la regione di sovrapposizione del rettangolo definito da (x1,y1) a (x2,y2) su ciascuno dei display collegati. La funzione restituirà un array con 8 valori per display (cioè: se avete due display, l'array avrà una lunghezza di 16 indici), dove i valori [0 ... 3] corrispondono ai valori overlapx1, overlapy1, overlapx2, overlapy2 - che definiscono la regione di sovrapposizione su questo display e saranno impostati a 0,0,0,0 se nessuna sovrapposizione - e i valori [4 ... 7] corrispondono ai valori monitorx1, monitory1, monitorx2, monitory2 - le coordinate del display nello spazio di visualizzazione virtuale. Questo può essere usato per testare se una posizione della finestra salvata sarà visibile o meno (l'utente potrebbe aver scollegato un monitor esterno o spostato la finestra fuori dallo schermo, lasciando la posizione della finestra salvata non valida), per esempio.

Sintassi:

window_get_visible_rects(x1, y1, x2, y2);

ArgomentoTipo Descrizione
x1 The left edge of the rectangle to check
y1 The top edge of the rectangle to check.
x2 The right edge of the rectangle to check
y2 The bottom edge of the rectangle to check.

Ritorna:

Esempio:

var wx = window_get_x();
var wy = window_get_y();
var ww = window_get_width();
var wh = window_get_height();
display_data = window_get_visible_rects(wx, wy, wx + ww, wy + wh);
display_num = array_length(display_data) / 8;

Il codice di cui sopra genererà un array 1D contenuto nella variabile display_data contenente le informazioni sui display, così come creerà la variabile display_num per contenere il numero di display attivi trovati.
