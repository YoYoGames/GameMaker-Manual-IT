visualizza_get_orientamento

Questa funzione restituirà una delle quattro costanti che GameMaker ha per dirvi se il dispositivo che esegue il gioco è tenuto in modalità orizzontale o verticale (vedi la tabella qui sotto).

Si noti che questa funzione potrebbe non rilevare correttamente l'orientamento del dispositivo quando viene utilizzata nel modulo di destinazione HTML5. Tuttavia questo è facilmente imitabile con l'uso del seguente script:

return (browser_width < browser_height);

Una tale funzione restituirebbe true per il ritratto e false per il paesaggio.

Sintassi:

display_get_orientation()

Ritorna:

Display Orientation Constant

Costante di orientamento del display
Costante Descrizione
display_landscape Il dispositivo è tenuto orizzontalmente, cioè: il bordo più lungo è da sinistra a destra, e il pulsante del menu è sulla destra.
display_landscape_flipped Come sopra, solo che ora il pulsante del menu è a sinistra.
display_portrait Il dispositivo è tenuto verticalmente, cioè: il bordo più lungo è dall'alto verso il basso, e il pulsante del menu è in basso.
display_portrait_flipped Come sopra, solo che ora il pulsante del menu è in alto.

Esempio:

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

Il codice qui sopra controlla l'orientamento del dispositivo e imposta una variabile globale a seconda del valore restituito.
