visualizza_get_sleep_margin

Questa funzione può essere utilizzata per ottenere il valore corrente del margine di sospensione utilizzato per il tempo di rendering del vostro gioco, e restituirà un valore in millisecondi. Per maggiori informazioni sui tempi di visualizzazione, vedere display_set_timing_method().

Sintassi:

display_get_sleep_margin();

Ritorna:

Esempio:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

Il codice di cui sopra controllerà il metodo di temporizzazione e poi se non è impostato su tm_sleep sarà impostato su quello e il margine di sonno impostato su 20.
