metodo di visualizzazione dei tempi

Questa funzione può essere utilizzata per recuperare il metodo di temporizzazione da utilizzare per il rendering del gioco. Il metodo può essere una delle costanti elencate di seguito.

Per ulteriori informazioni sui diversi metodi di temporizzazione, vedere display_set_timing_method().

Sintassi:

display_get_timing_method();

Ritorna:

Display Timing Method Constant

Metodo di temporizzazione del display Costante
Costante Descrizione
tm_sleep Il valore del margine di sonno è il principale metodo di temporizzazione
tm_countvsyncs Il timing Vsync è il metodo di timing principale (predefinito per tutte le piattaforme supportate)

Esempio:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

Il codice di cui sopra controllerà il metodo di temporizzazione e poi se non è impostato su tm_sleep sarà impostato su quello e il margine di sonno impostato su 20.
