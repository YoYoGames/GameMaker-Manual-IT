visualizza_get_frequenza

Questa funzione restituisce la frequenza (o refresh rate) del display su cui si sta giocando. Restituirà un valore reale come fotogrammi al secondo, quindi per esempio se il vostro monitor è a 60hz avrete 60, se sta girando a 144hz allora avrete 144, e così via.

NOTA Questa funzione non è supportata su HTML5.

Sintassi:

display_get_frequency();

Ritorna:

Esempio:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Il codice di cui sopra ottiene la frequenza del display ed esegue alcune condizioni per impostare il gioco a 240, 120, 60 o 30 fps. Questo significa che se il vostro display è a 90hz il gioco girerà a 60 fps, se è a 144hz allora girerà a 120 fps, ecc. Naturalmente, è possibile passare la frequenza del display direttamente in game_set_speed() in modo che sia usata come frame rate del gioco.
