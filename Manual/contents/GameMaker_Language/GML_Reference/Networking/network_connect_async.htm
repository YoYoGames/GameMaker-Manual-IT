rete_connetti_async

Con questa funzione potete inviare una richiesta di connessione a un server. La funzione prende il socket id attraverso cui connettersi (vedi network_create_socket()) e richiede di fornire l'indirizzo IP a cui connettersi (un string) così come la porta a cui connettersi, e se la connessione fallisce verrà restituito un valore inferiore a 0. La connessione utilizza un protocollo speciale che assicura che solo i giochi GameMaker si connettano tra loro, tuttavia se avete bisogno di di connettersi a un server che non è un gioco GameMaker, è possibile utilizzare network_connect_async_raw(). Si noti che questa funzione è asincrona, generando un evento Asynchronous Networking del tipo network_type_non_blocking_connect.

Sintassi:

network_connect_async(socket, url, port);

ArgomentoTipo Descrizione
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Ritorna:

Esempio:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

Il codice di cui sopra creerà un nuovo socket TCP poi tenterà di effettuare una connessione asincrona attraverso di esso all'indirizzo IP dato sulla porta 6510.
