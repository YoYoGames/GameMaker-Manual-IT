rete_set_timeout

Con questa funzione potete impostare il timeout per leggere e scrivere dati da/verso un server attraverso il socket dato. Notate che il timeout non genera alcun tipo di evento, quindi dovrete occuparvi voi stessi dei timeout usando gli allarmi (per esempio). Si noti che questo valore influisce solo sull'invio e la ricezione di dati, e se si desidera modificare il valore di timeout della connessione allora si dovrebbe utilizzare la funzione network_set_config().

Sintassi:

network_set_timeout(socket, read_timeout, write_timeout);

ArgomentoTipo Descrizione
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Ritorna:

Esempio:

network_set_timeout(sock, 3000, 3000);

Il codice qui sopra imposterà il timeout per leggere e scrivere dati attraverso il socket indicizzato nella variabile "sock" a 3 secondi.
