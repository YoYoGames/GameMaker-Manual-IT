rete_creare_presa_ext

Questa funzione serve a creare un nuovo socket client per il gioco per comunicare in rete. È necessario definire il tipo di socket (vedere l'elenco di costanti riportato di seguito) e fornire una porta da utilizzare; la funzione restituirà un id univoco che dovrà essere utilizzato in tutte le ulteriori chiamate di funzione per quel socket, oppure un valore inferiore a 0 se la connessione fallisce.

Sintassi:

network_create_socket_ext(protocol, port);

Argomento Tipo Descrizione
protocol Tipo di presa Costante The network protocol to use
port Reale The port to use

Restituzione:

Network Socket ID

Esempio:

client = network_create_socket_ext(network_socket_udp, 6510);

Il codice precedente crea un nuovo socket UDP sulla porta 6510.

Indietro: Collegamento in rete
