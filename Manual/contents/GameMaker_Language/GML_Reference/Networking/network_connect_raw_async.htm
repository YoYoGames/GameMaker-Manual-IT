rete_connect_raw_async

Con questa funzione potete inviare una richiesta di connessione a un server. La funzione prende il socket id attraverso cui connettersi (vedi network_create_socket()) e richiede di fornire l'indirizzo IP a cui connettersi (un string) così come la porta a cui connettersi, e se la connessione fallisce verrà restituito un valore inferiore a 0. La differenza tra questa funzione e network_connect_async()è che questa può connettersi a qualsiasi server e non fa nulla ai dati grezzi, il che significa che dovete implementare voi stessi i protocolli alla fine di server. Si noti che questa funzione è asincrona, generando un evento Asynchronous Networking del tipo network_type_non_blocking_connect.

Sintassi:

network_connect_raw_async(socket, url, port);

ArgomentoTipo Descrizione
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Ritorna:

Esempio:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

Il codice di cui sopra creerà un nuovo socket TCP poi tenterà di creare una connessione asincrona "grezza" attraverso di esso all'URL dato sulla porta 6510.
