distruzione_della_rete

Con questa funzione è possibile rimuovere una connessione alla presa di rete dal vostro gioco.

NOTA: Questa funzione non funzionerà se usata in un progetto che gira sul target HTML5.

Sintassi:

network_destroy(socket);

ArgomentoTipo Descrizione
socket The id of the socket to remove.

Ritorna:

Esempio:

if !os_is_network_connected()
{
    network_destroy(sock);
}

Il codice qui sopra controllerà per vedere se c'è una connessione dati e se non ne viene trovata nessuna, distruggerà il socket con l'id "sock".
