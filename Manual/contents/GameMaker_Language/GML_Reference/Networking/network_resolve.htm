risoluzione della rete

Questa funzione restituisce l'indirizzo IP dell'URL dato.

Sintassi:

network_resolve(url);

ArgomentoTipo Descrizione
url The URL to get the IP of (a string).

Ritorna:

Esempio:

game_ip = network_resolve("www.macsweeneygames.com");

Il codice di cui sopra restituirà l'indirizzo IP dell'URL dato e lo memorizzerà nella variabile "game_ip".
