connessione_alla_rete

Con questa funzione è possibile inviare una richiesta di connessione a un sito server. La funzione prende l'id del socket attraverso il quale connettersi (vedere network_create_socket()) e richiede di fornire l'indirizzo IP a cui connettersi (un string) e la porta attraverso la quale connettersi. La connessione utilizza un protocollo speciale che assicura che solo i giochi di GameMaker si connettano tra loro; tuttavia, se avete bisogno di connettervi a un server che non è un gioco di GameMaker, potete usare network_connect_raw(). Si noti che per impostazione predefinita la funzione è sincrona, il che significa che il gioco potrebbe apparire "bloccato" durante la connessione. È possibile impostare un valore di timeout per la connessione o, in alternativa, rendere la creazione asincrona, utilizzando la funzione network_set_config(), o in alternativa utilizzare la funzione network_connect_async().

NOTA Non è possibile utilizzare questa funzione su HTML5. Per i WebSocket, utilizzare la funzione Async.

Sintassi:

network_connect(socket, url, port);

Argomento Tipo Descrizione
socket ID della presa di rete The id of the socket to use.
url Stringa The URL or IP to connect to (a string).
port Reale The port to connect to.

Restituzione:

Real or Network Socket ID

Esempio:

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

Il codice precedente creerà un nuovo socket TCP e tenterà di connettersi attraverso di esso all'indirizzo IP indicato sulla porta 6510.
