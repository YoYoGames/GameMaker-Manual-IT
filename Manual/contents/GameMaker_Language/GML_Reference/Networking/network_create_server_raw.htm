rete_creare_server_raw

Questa funzione è usata per creare una nuova rete server per il tuo gioco, usando uno dei protocolli di connessione permessi (vedi le costanti elencate sotto). Voi fornite il tipo server, poi gli date una porta da usare, e infine il numero di connessioni massime che dovrebbero essere permesse in ogni momento a server (notate che questo numero dipende da voi, ma troppi client connessi satureranno la rete o la CPU del dispositivo non sarà in grado di gestire l'elaborazione di quel numero di giocatori, quindi usate con cura). La funzione restituisce un id unico che dovrebbe essere usato memorizzato in una variabile e utilizzato per identificare il server in tutte le ulteriori funzioni di rete, o un valore inferiore a 0 se la connessione fallisce.

Poiché questo crea un server"grezzo", non accetterà né userà nessuna delle intestazioni di dati costruite in GameMaker per la comunicazione, e quindi dovreste usare le funzioni network_send_raw() e network_send_udp_raw() per inviare dati non formattati al server creato.

Sintassi:

network_create_server_raw(type, port, max_client);

ArgomentoTipo Descrizione
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

Ritorna:

Network Socket ID

Esempio:

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server_raw(network_socket_tcp, port, 32);
}

Il codice di cui sopra cercherà di creare un server usando TCP attraverso la porta 6510. Se quella porta non è disponibile, allora loop cercherà tra le porte per trovarne una che lo sia.
