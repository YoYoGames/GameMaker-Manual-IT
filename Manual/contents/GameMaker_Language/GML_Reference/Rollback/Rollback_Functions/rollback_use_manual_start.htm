rollback_uso_avvio_manuale

Per impostazione predefinita, una partita multigiocatore inizia automaticamente quando tutti i giocatori si sono uniti.

La chiamata a questa funzione disabilita il comportamento di avvio automatico, per cui il gioco attende che l'utente lo avvii manualmente.

Se utilizzata, questa funzione deve essere eseguita prima della funzione di unione/creazione.

Sintassi:

rollback_use_manual_start();

Restituzione:

N/A

Esempio:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

Il codice sopra riportato abilita l'avvio manuale, quindi si unisce o crea una nuova partita multigiocatore.
