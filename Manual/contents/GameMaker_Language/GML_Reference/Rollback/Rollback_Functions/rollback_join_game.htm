rollback_join_game

Questa funzione tenta di unirsi a una partita. È necessario trovarsi su un URL che contenga l'ID del sito virtuale room a cui unirsi.

Se è stato unito un room, restituisce true; se non è stato unito nessun gioco, restituisce false.

Quando un nuovo giocatore si unisce a una partita in cui si è già presenti, viene attivato l'evento Rollback. Per ulteriori informazioni, vedere Eventi di rollback.

L'argomento dry_run è opzionale e, se impostato a true, consente di eseguire la funzione senza unirsi effettivamente a una partita. Viene utilizzato per verificare se è possibile partecipare a una partita senza effettivamente unirsi ad essa.

Sintassi:

rollback_join_game([dry_run]);

Argomento Tipo Descrizione
dry_run Booleano Disabled by default (false). Quando è abilitata(true), la funzione non si unisce alla partita, ma restituisce semplicemente true se è possibile unirsi a una partita, o false in caso contrario.

In questo modo è possibile utilizzare questa funzione per verificare solo se il giocatore è stato invitato.

Restituzione:

Boolean

Esempio:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

Il codice qui sopra definisce un giocatore object, e poi tenta di unirsi a una partita. Se non si unisce, crea una nuova partita, con un massimo di 2 giocatori e con il Sync Test disabilitato.

Per un esempio esteso, vedere Creare un gioco multigiocatore.
