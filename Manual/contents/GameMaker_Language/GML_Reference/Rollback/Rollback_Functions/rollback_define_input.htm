rollback_define_input

Questa funzione viene utilizzata per definire gli input personalizzati per tutti i giocatori. Prende un struct che deve contenere tutti gli input con le relative costanti di input assegnate.

Per sapere come assegnare gli ingressi personalizzati, vedere Controlli personalizzati.

Se utilizzata, questa funzione deve essere eseguita prima della funzione di unione/creazione.

Sintassi:

rollback_define_input(input_struct);

Argomento Tipo Descrizione
input_struct Struttura A struct containing input names as variables, and their assigned inputs as values.

Restituzione:

N/A

Esempio:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

Il codice precedente definisce gli input personalizzati per tutti i giocatori. Gli ultimi due input utilizzano più controlli, elencandoli in un array.

Per un esempio esteso, vedere Definizione degli ingressi.
