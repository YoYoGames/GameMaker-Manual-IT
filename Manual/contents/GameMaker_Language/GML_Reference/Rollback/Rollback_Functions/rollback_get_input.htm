rollback_get_input

Questa funzione restituisce un struct contenente i valori di input di un giocatore. Richiede un argomento opzionale che specifica l'ID del giocatore per il quale verranno restituiti gli input.

Quando viene utilizzata in un'istanza di giocatore creata da rollback_define_player()non è necessario specificare l'ID del giocatore, quindi è possibile chiamare questa funzione senza alcun argomento.

Quando viene utilizzato in altre istanze o in istanze di giocatori create manualmente, è necessario specificare un ID giocatore.

Per informazioni su quali ingressi vengono restituiti e su come possono essere modificati, vedere Definizione degli ingressi.

Sintassi:

rollback_get_input(player_id);

Argomento Tipo Descrizione
player_id Reale OPTIONAL L'ID del giocatore per il quale verranno restituiti gli input.

Restituzione:

Struct

Esempio:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

Il codice qui sopra ottiene l'input struct in un'istanza del giocatore creata automaticamente dal sistema Rollback. In base ai valori restituiti, sposta l'istanza di 2 pixel in ciascuna direzione.

Per un esempio esteso, vedere Creazione di un gioco multigiocatore.
