rollback_start_game

Questa funzione forza l'avvio della partita multigiocatore in corso, prima che tutti i giocatori si siano uniti. È necessario creare prima una partita multigiocatore utilizzando rollback_create_game().

Normalmente il sistema attende che tutti i giocatori si uniscano prima di avviare automaticamente la partita. Questa funzione può essere utilizzata per avviare la partita in anticipo, prima che tutti i giocatori si siano uniti.

È possibile chiamare rollback_use_manual_start() per far sì che Rollback attenda questa funzione anche dopo che tutti i giocatori si sono uniti.

La chiamata di questa funzione attiva l'evento Avvio rollback.

Sintassi:

rollback_start_game();

Restituzione:

N/A

Esempio:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Se il gioco Rollback non è in corso e si preme Invio, si avvia il gioco.

Questo permette al giocatore di iniziare la partita manualmente prima che tutti i giocatori si uniscano.
