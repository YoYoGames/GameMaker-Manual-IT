rollback_use_random_input

Quando si gioca una partita utilizzando Sync Test, tutti i giocatori "remoti" ricevono valori casuali per i loro input definiti, come metodo di base per testare. È possibile disabilitare questo comportamento chiamando questa funzione con false come argomento e abilitarlo nuovamente specificando true.

Sintassi:

rollback_use_random_input(enabled);

Argomento Tipo Descrizione
enabled Booleano Specifies whether random input should be enabled (true) o disabilitato(false). Per impostazione predefinita, è abilitato.

Restituzione:

N/A

Esempio:

rollback_use_random_input(false);

Il codice precedente disabilita l'immissione casuale durante il test di sincronizzazione.
