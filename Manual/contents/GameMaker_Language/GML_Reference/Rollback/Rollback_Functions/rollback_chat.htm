rollback_chat

Questa funzione viene utilizzata per inviare un messaggio string ad altri giocatori.

Si specifica un string da inviare, con una dimensione massima di 128 byte.

Se il secondo argomento non è specificato, il messaggio viene inviato a tutti i giocatori, compreso il mittente.

Nel secondo parametro è possibile specificare l'ID del giocatore che riceverà il messaggio.

Come si usa?

Questa funzione è stata pensata per essere utilizzata "localmente" e quindi dovrebbe essere il risultato di un input locale, non di un input di Rollback.

Vedere l'esempio di utilizzo in fondo. Ascoltare i messaggi ricevuti nell'evento Rollback Event.

Sintassi:

rollback_chat(message, [to]);

Argomento Tipo Descrizione
message Stringa The message to send, as a string.
to Reale OPTIONAL L'ID del giocatore a cui verrà inviato il messaggio. Se non si specifica questo parametro o se si specifica -1, il messaggio viene inviato a tutti i giocatori.

Restituzione:

N/A

Esempio:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Se il tasto invio viene premuto localmente, il valore keyboard_string viene inviato a tutti i giocatori. Il sito string viene quindi cancellato per consentire la digitazione di un nuovo messaggio.

In questo modo si attiverà l'evento Rollback quando il messaggio sarà ricevuto da tutti i client (compreso il mittente).

Evento Rollback

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
