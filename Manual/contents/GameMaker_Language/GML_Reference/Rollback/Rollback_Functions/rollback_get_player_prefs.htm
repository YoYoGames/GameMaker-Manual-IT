rollback_get_player_prefs

Questa funzione restituisce le preferenze impostate per l'ID giocatore indicato.

Se l'ID del giocatore non è specificato, la funzione deve essere chiamata all'interno di un'istanza del giocatore, nel qual caso verranno restituite le preferenze di quel giocatore.

Si può anche utilizzare la variabile di istanza player_prefs.

Questa funzione restituisce le ultime preferenze che sono state sincronizzate per il giocatore dato, quindi l'impostazione delle preferenze non aggiornerà immediatamente le preferenze restituite da questa funzione.

Si prega di leggere: Preferenze del giocatore

Sintassi:

rollback_get_player_prefs([player_id]);

Argomento Tipo Descrizione
player_id Reale OPTIONAL L'ID del giocatore per il quale vengono restituite le preferenze.

Restituzione:

Any

Esempio:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

Il codice precedente ottiene le preferenze impostate per il giocatore corrente e applica una preferenza a una variabile di istanza.
