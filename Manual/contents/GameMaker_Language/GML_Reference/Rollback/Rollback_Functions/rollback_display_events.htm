rollback_display_events

Per impostazione predefinita, quando si esegue il gioco, vengono visualizzati i messaggi di debug nell'angolo in alto a sinistra della partita, che mostrano gli eventi multigiocatore che si verificano. È possibile disattivare questi messaggi chiamando questa funzione con false come argomento e riattivarli specificando true.

Sintassi:

rollback_display_events(enabled);

Argomento Tipo Descrizione
enabled Booleano Specifies whether event display should be enabled (true) o disabilitato(false). Per impostazione predefinita, è abilitato.

Restituzione:

N/A

Esempio:

rollback_display_events(false);

Il codice precedente ha disabilitato la visualizzazione degli eventi.
