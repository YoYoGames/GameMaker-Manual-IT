rollback_use_player_prefs

Questa funzione abilita le preferenze dei giocatori e consente di impostare le preferenze predefinite per tutti i giocatori.

Le preferenze predefinite possono essere di qualsiasi tipo. Se non si specificano le preferenze predefinite, tutti i giocatori devono avere le preferenze impostate prima che la partita possa iniziare.

Richiamando questa funzione si disabilita l'avvio automatico, per cui è necessario avviare il gioco manualmente. Vedere: Avvio manuale o automatico

Se utilizzata, questa funzione deve essere eseguita prima della funzione di unione/creazione.

Si prega di leggere: Preferenze del giocatore

Sintassi:

rollback_use_player_prefs([default]);

Argomento Tipo Descrizione
default Qualsiasi OPTIONAL Le preferenze predefinite da impostare per tutti i giocatori.

Restituzione:

N/A

Esempio:

rollback_use_player_prefs({
carattere: 0,
colore: c_bianco
});

Il codice sopra riportato abilita le preferenze del giocatore e imposta le preferenze predefinite in struct.
