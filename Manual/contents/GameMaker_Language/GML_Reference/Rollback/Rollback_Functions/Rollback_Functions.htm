Funzioni di rollback

Le seguenti funzioni sono utilizzate per avviare, partecipare o abbandonare una partita multigiocatore:

  • rollback_join_game
  • rollback_create_game
  • rollback_start_game
  • rollback_leave_game

Le seguenti funzioni sono utilizzate per definire e ottenere informazioni sul gioco:

  • Set
    • rollback_define_player
    • rollback_define_input
    • rollback_define_mock_input
    • rollback_define_input_frame_delay
    • rollback_uso_avvio_manuale
    • rollback_use_player_prefs
    • rollback_set_player_prefs
  • Ottenere
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs

Le seguenti funzioni sono disponibili per l'uso durante una partita attiva:

  • rollback_chat
  • rollback_sync_on_frame

Le seguenti funzioni sono utilizzate per i test o per abilitare/disabilitare le funzioni relative ai test:

  • rollback_use_random_input
  • rollback_display_events
