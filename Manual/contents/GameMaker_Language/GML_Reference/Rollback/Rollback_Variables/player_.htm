nome_giocatore

Questa variabile memorizza il nome dell'istanza del giocatore corrente.

Questa variabile esiste solo nelle istanze di giocatore create da rollback_define_player()ed è diversa per ogni giocatore.

Igiocatori ospiti restituiranno un nome segnaposto.

Sintassi:

player_name;

Restituzione:

String

Esempio:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Disegna il nome del giocatore 100px sopra la sua istanza, con un allineamento al centro.

Indietro: Variabili di rollback
Avanti: player_type
