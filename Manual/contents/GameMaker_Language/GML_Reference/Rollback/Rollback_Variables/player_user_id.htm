ID_giocatore

Questa variabile memorizza l'ID GX.games dell'istanza del giocatore corrente, da utilizzare per le chiamate API dirette al server.

Questa variabile esiste solo nelle istanze di giocatore create da rollback_define_player()ed è diversa per ogni giocatore.

Igiocatori ospiti avranno un string vuoto assegnato a questa variabile.

Sintassi:

player_user_id;

Restituzione:

String

Indietro: Variabili di rollback
Avanti: player_prefs
