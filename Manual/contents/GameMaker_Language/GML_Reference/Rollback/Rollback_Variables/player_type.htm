tipo_giocatore

Questa variabile indica se l'istanza del giocatore è un utente GX.games registrato o un ospite. Può restituire una qualsiasi delle seguenti stringhe:

Stringa Descrizione
"Guest" Il giocatore non ha effettuato l'accesso a GX.games
"User" Il giocatore è iscritto a GX.games

Gli utenti ospiti ricevono avatar e nomi segnaposto.

Questa variabile esiste solo nelle istanze di giocatore create da rollback_define_player()ed è diversa per ogni giocatore.

Sintassi:

player_type;

Restituzione:

String

Indietro: Variabili di rollback
Avanti: player_user_id
