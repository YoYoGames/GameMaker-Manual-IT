player_prefs

Questa variabile memorizza le preferenze impostate per il giocatore corrente. Viene assegnata alle istanze di giocatore create con il metodo rollback_define_player().

Si prega di leggere: Preferenze del giocatore

Sintassi:

player_prefs;

Restituzione:

Any

Esempio:

image_index = player_prefs.character;

Il codice precedente legge una preferenza e imposta una variabile di istanza nel giocatore.
