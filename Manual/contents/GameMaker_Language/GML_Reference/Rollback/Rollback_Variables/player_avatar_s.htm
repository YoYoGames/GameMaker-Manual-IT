player_avatar_sprite

Questa variabile memorizza l'indirizzo sprite dell'istanza del giocatore corrente.

Questa variabile esiste solo nelle istanze di giocatore create da rollback_define_player()ed è diversa per ogni giocatore.

Igiocatori ospiti visualizzeranno un segnaposto sprite.

Sintassi:

player_avatar_sprite;

Restituzione:

Sprite Asset

Esempio:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Visualizza l'avatar dell'istanza del giocatore che esegue il codice, in un'area 64x64, 256px sopra l'istanza.
