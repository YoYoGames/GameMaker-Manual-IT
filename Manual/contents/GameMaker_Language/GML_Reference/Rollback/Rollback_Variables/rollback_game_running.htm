rollback_gioco_in_esecuzione

Questa variabile globale indica se la partita di rollback è attualmente in corso. Se è true, significa che la partita di rollback è stata avviata ed è in corso. In tutti gli altri casi, sarà false.

Sintassi:

rollback_game_running

Restituzione:

Boolean

Esempio:

if (!rollback_game_running) return;

x += move_x;
y += move_y;

Il codice di cui sopra verrebbe eseguito nell'evento Step di un giocatore object. Quando la partita di rollback non è in corso, esce dall'evento, quindi il giocatore non può essere controllato finché la partita di rollback non è iniziata.
