Rollback Multiplayer

Rollback multiplayer in GameMaker consente di impostare e creare rapidamente un gioco multiplayer in rete per l'obiettivo Opera GX.

Questa funzionalità utilizza i server GX per connettere i giocatori e sincronizzare gli input, quindi lo sviluppatore non deve configurare server per avviare una nuova partita online.

Mentre la funzionalità online può essere utilizzata solo con il target Opera GX, è possibile utilizzare "Sync Test" per testare il gioco multigiocatore in locale, con input casuali o finti applicati agli altri giocatori. Sync Test funziona sui target Windows e macOS, oltre che su Opera GX.
