annullamento della chiamata

Questa funzione annulla una funzione di callback avviata in precedenza da call_later.

Sintassi:

call_cancel(handle);

Argomento Tipo Descrizione
handle ID della fonte temporale The handle to a Time Source restituito da call_later.

Restituzione:

N/A

Esempio:

call_cancel(handle);

Indietro: call_later
Avanti: time_source_create
