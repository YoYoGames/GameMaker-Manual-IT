tempo_sorgente_arresto

Questa funzione arresta la sorgente di tempo data e ne azzera il timer. Non è possibile arrestare una sorgente di tempo che non è stata avviata o che si trova in uno stato finito.

Sintassi:

time_source_stop(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per fermarsi

Restituzione:

N/A

Esempio:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

Il codice sopra riportato arresta una sorgente di tempo se non è già stata arrestata.
