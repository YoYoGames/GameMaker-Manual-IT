time_source_get_parent

Questa funzione restituisce l'ID ID sorgente tempo del genitore della sorgente temporale indicata.

Il valore restituito può essere l'ID di una fonte temporale personalizzata o di una delle fonti temporali integrate, a seconda di come è stata creata la fonte temporale.

Sintassi:

time_source_get_parent(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per ottenere il genitore di

Restituzione:

Time Source ID

Esempio:

var _parent = time_source_get_parent(time_source);

Questo codice ottiene il genitore di una sorgente temporale e lo memorizza in una variabile locale.
