time_source_get_reps_remaining

Questa funzione restituisce il numero di ripetizioni rimaste fino all'arresto completo della sorgente di tempo indicata.

La funzione restituisce undefined se la sorgente di tempo data funziona a tempo indeterminato o se si tratta di una sorgente di tempo incorporata.

Sintassi:

time_source_get_reps_remaining(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per ottenere informazioni per

Restituzione:

Real

Indietro: Fonti di tempo
Avanti: time_source_get_state
