time_source_get_units

Questa funzione restituisce le unità di misura impostate per la sorgente di tempo indicata. Il valore del periodo della sorgente temporale è espresso in queste unità.

Sintassi:

time_source_get_units(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per ottenere le unità per

Restituzione:

Time Source Unit Constant

Indietro: Fonti di tempo
Avanti: time_bpm_to_seconds
