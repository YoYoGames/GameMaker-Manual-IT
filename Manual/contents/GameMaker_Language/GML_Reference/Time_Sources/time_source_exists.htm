time_source_exists

Questa funzione restituisce se l'ID dato è una fonte temporale valida.

Se la Sorgente del tempo è stata distrutta, la funzione restituirà false, ma se la Sorgente del tempo è stata fermata o è semplicemente scaduta, continuerà a esistere, il che significa che la funzione restituirà true.

Sintassi:

time_source_exists(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per controllare

Restituzione:

Boolean

Esempio:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Questo codice controlla se esiste un'origine temporale. In caso affermativo, controlla se è stato premuto il tasto Spazio.

In questo caso, distrugge la Sorgente del Tempo.
