tempo_fonte_distruzione

Questa funzione distrugge la fonte temporale indicata.

È possibile specificare se eliminare anche tutte le sorgenti temporali figlie.

Sintassi:

time_source_destroy(id, [destroy_tree]);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source distruggere
destroy_tree Booleano OPTIONAL Se distruggere anche le sorgenti temporali figlio, è false per impostazione predefinita

Restituzione:

N/A

Esempio:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Il codice dell'evento Create crea una nuova sorgente di tempo che scade una volta al secondo e si ripete all'infinito.

Il codice dell'evento Clean Up distrugge la sorgente temporale, in quanto non sarà più necessaria dopo la distruzione dell'istanza.

Indietro: Fonti di tempo
Avanti: time_source_start
