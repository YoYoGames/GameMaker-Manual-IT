chiamata_più_tardi

Questa funzione attiva una funzione di callback dopo il tempo specificato o dopo un certo intervallo. Lo fa creando una sorgente di tempo (vedere Sorgenti di tempo).

Per annullare la funzione di callback, vedere call_cancel.

IMPORTANTE Sebbene questa funzione restituisca un ID effettivo della sorgente di tempo, questo ID non può essere usato con le funzioni di sorgente di tempo, poiché le sorgenti di tempo usate da call_later e call_cancel sono "invisibili": non appaiono come figli di alcuna sorgente di tempo incorporata e non possono essere usate con altre funzioni di sorgente di tempo. Tuttavia, sono incluse nel conteggio delle fonti temporali restituito da debug_event("ResourceCounts").

Argomenti

Di seguito sono riportati gli argomenti di questa funzione:

Periodo

Si tratta di un periodo di tempo che può essere espresso in secondi o in fotogrammi, a seconda dell'unità specificata nell'argomento successivo.

Quando si utilizzano i frame come unità di misura, il periodo deve essere un numero intero. I valori non interi vengono arrotondati per difetto (floored), con l'eccezione dei valori inferiori a 1, che vengono arrotondati per eccesso.

Unità

È l'unità in cui è espresso il periodo e può essere time_source_units_seconds o time_source_units_frames.

Vedere: Unità di origine del tempo

È possibile utilizzare un valore di battiti al minuto utilizzando time_bpm_to_seconds().

Richiamo

È necessario specificare un "callback" method che viene richiamato quando la sorgente temporale scade.

Anello

Se la fonte temporale deve essere loop.

Sintassi:

call_later(period, unit, callback, [loop=false]);

Argomento Tipo Descrizione
period Reale The period that the Time Source per, nelle unità date
units Sorgente di tempo Unità Costante The units that the given period is in
callback Metodo o Funzione Script The method or script function to call after the specified time
loop Booleano OPTIONAL Se eseguire il loop dell'origine temporale (l'impostazione predefinita è false)

Restituzione:

Time Source ID

Esempio 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

In questo esempio, si crea una funzione di callback che mostra un messaggio di debug dopo cinque secondi. Dopodiché, la sorgente temporale sottostante viene distrutta.

Esempio 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

In questo esempio, creiamo una funzione di callback che crea un nuovo nemico in una posizione casuale di room. Questa funzione viene poi eseguita ogni 10 secondi passandola a call_later. L'ultimo parametro è impostato a vero in modo che la funzione venga richiamata ogni 10 secondi.

Utilizzare call_cancel con l'handle restituito da call_later per interrompere la sorgente temporale e le chiamate al callback.
