tempo_fonte_inizio

Questa funzione avvia la sorgente temporale data, cambiando il suo stato in time_source_state_active.

La sorgente di tempo specificata può essere una nuova sorgente di tempo che non è mai stata avviata, che è stata messa in pausa, fermata o scaduta senza ripetizioni.

Questa funzione esegue un "soft-reset" della sorgente di tempo indicata, riportando il tempo di scadenza e le ripetizioni rimanenti ai valori configurati.

Sintassi:

time_source_start(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per iniziare

Restituzione:

N/A

Esempio:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

In questo esempio, vogliamo che l'istanza si distrugga 300 fotogrammi dopo.

Il codice crea prima un metodo che chiama semplicemente instance_destroy().

Crea quindi una sorgente di tempo, che eredita dalla sorgente di tempo del gioco. Imposta il suo periodo a 300 fotogrammi.

Infine, avvia la Sorgente tempo.

Indietro: Fonti di tempo
Avanti: time_source_stop
