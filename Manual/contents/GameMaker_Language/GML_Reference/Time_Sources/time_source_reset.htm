tempo_fonte_reset

Questa funzione resetta la sorgente di tempo data, azzerando il suo timer e le informazioni di ripetizione.

Dopo aver richiamato questa funzione, è necessario avviare nuovamente la Sorgente tempo per far partire il conto alla rovescia.

Sintassi:

time_source_reset(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per azzerare

Restituzione:

N/A

Esempio:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Questo codice azzera la sorgente di tempo data ogni volta che si avvia un nuovo room e quindi avvia nuovamente la sorgente di tempo.
