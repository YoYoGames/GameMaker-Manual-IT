time_source_get_children

Per la sorgente temporale data, questa funzione restituisce una matrice contenente i file ID sorgente temporaledei suoi figli.

Sintassi:

time_source_get_children(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per ottenere i figli di

Restituzione:

Array of Time Source ID

Esempio:

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i ++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Questo codice esamina tutti i figli di una Sorgente del tempo e li distrugge uno per uno.

Alla fine, distrugge la sorgente temporale madre.
