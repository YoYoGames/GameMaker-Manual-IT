time_source_get_time_remaining

Questa funzione restituisce il tempo rimanente fino alla scadenza dell'origine temporale indicata ed esegue il suo metodo di callback.

Questo valore temporale sarà espresso nelle unità di misura impostate per la sorgente temporale (secondi o fotogrammi).

Sintassi:

time_source_get_time_remaining(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per ottenere informazioni per

Restituzione:

Real

Indietro: Fonti di tempo
