tempo_fonte_ripresa

Questa funzione riprende la sorgente temporale data e i suoi figli. La sorgente di tempo data deve essere stata precedentemente messa in pausa con l'opzione time_source_pause().

Sintassi:

time_source_resume(id);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per riprendere

Restituzione:

N/A

Esempio:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

Quando si preme il tasto Spazio, questo codice ottiene lo stato di una sorgente temporale.

Quando lo stato è attivo, mette in pausa la Sorgente tempo e quando è in pausa la riprende.
