fonte_tempo_riconfigura

Questa funzione serve a modificare le proprietà principali di una sorgente temporale, senza doverne creare una completamente nuova.

Si specifica un'origine temporale esistente e si impostano le proprietà che sono impostate anche in time_source_create()tranne che nel genitore. Per informazioni dettagliate su queste proprietà, leggere questa pagina.

La sorgente di tempo specificata verrà azzerata e disattivata e dovrà essere riavviata.

Sintassi:

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

Argomento Tipo Descrizione
id ID della fonte temporale The Time Source per riconfigurare
period Reale The period that the Time Source per, nelle unità date
units Sorgente di tempo Unità Costante The units that the given period is in
callback Metodo The method to call when the Time Source Scade
args Array OPTIONAL Un array contenente gli argomenti da passare al metodo
repetitions Reale OPTIONAL Il numero di volte in cui la sorgente temporale deve ripetersi, o -1 per una ripetizione indefinita.
expiry_type Sorgente di tempo Costante di scadenza OPTIONAL Se la sorgente temporale scade nel fotogramma più vicino alla sua scadenza o nel fotogramma successivo.

Restituzione:

N/A

Esempio:

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

Crea una nuova funzione che modifica il ritardo di spawn utilizzato per i nemici del gioco.

Supponendo che il gioco utilizzi una fonte di tempo chiamata obj_game.spawn_time_source per generare i nemici, tale fonte di tempo dovrà essere aggiornata una volta modificato il ritardo di spawn.

Questa funzione fa esattamente questo, riconfigurando la sorgente di tempo con il nuovo ritardo e riavviandola.
