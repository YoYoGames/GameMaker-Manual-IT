valutazione del negozio

Questa funzione apre un dialogo dipendente dal sistema operativo in cui si può chiedere all'utente di pubblicare una valutazione o un commento su una determinata pagina. Puoi definire il testo che deve apparire nel dialogo, così come il testo che vuoi far apparire sui due pulsanti e l'URL dove il commento deve essere postato.

NOTA: Questa funzione è solo per gli obiettivi Android e iOS.

IMPORTANTE! Su iOS, quando si usa TestFlight, in fase di sviluppo la richiesta verrà visualizzata ma non sarà possibile fare nulla. Una volta che la tua app è nell'App Store, questo pulsante diventa attivo e l'utente ha la possibilità di scrivere una recensione o lasciare una valutazione. Nota che se distribuisci il gioco attraverso TestFlight, questa finestra potrebbe anche non essere mostrata.

Sintassi:

shop_leave_rating(text, yes_string, no_string, url)

Argomento Tipo Descrizione
text Stringa Text that appears on the dialog.
yes_string Stringa Text that appears on the "yes" button.
no_string Stringa Text that appears on the "no" button.
url Stringa The full URL where the comment has to be sent.

Ritorna:

N/A

Esempio:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

Il codice qui sopra chiederà all'utente di lasciare un commento se la variabile "timer" ha contato fino a 0.
