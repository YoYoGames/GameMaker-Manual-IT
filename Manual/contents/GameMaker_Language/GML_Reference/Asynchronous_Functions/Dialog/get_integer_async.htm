get_integer_async

Questa funzione apre una finestra e visualizza il messaggio e uno spazio per l'utente per inserire un valore (che conterrà il valore di default fornito per iniziare). Questa è una funzione asincrona, e come tale GameMaker non blocca il dispositivo su cui viene eseguita in attesa di una risposta, ma piuttosto continua a eseguire gli eventi normalmente. Una volta che l'utente ha digitato il suo string e premuto il pulsante "Ok", un evento asincrono Dialogo che, solo per la durata di quell'evento, avrà una mappa DS memorizzata nella variabile async_load.

Questa mappa conterrà le tre chiavi, "id", "status", e "value". "id" è il valore che è stato restituito dalla funzione quando è stata chiamata, lo "status" sarà o true per il pulsante "Ok" premuto, o false se il messaggio è stato annullato (dove applicabile, poiché non tutte le piattaforme di destinazione permettono di annullare i messaggi). Infine "value" restituirà l'intero che l'utente ha inserito come string (un string "" vuoto sarà restituito se non è stato dato alcun input).

NOTA Questa funzione è solo per debug e teste non dovrebbe essere usata nei giochi rilasciati. A tal fine dovreste creare un'interfaccia utente personalizzata per ricevere input dai giocatori in modo da avere il controllo completo su come le finestre di dialogo appaiono e si comportano.

Sintassi:

get_integer_async(string, default);

Argomento Tipo Descrizione
string Stringa The message to show in the dialog.
default Real The default value.

Ritorna:

Async Request ID

Esempio esteso:

L'evento di pressione sinistra del mouse (per esempio) di object che mostra il messaggio avrebbe il seguente codice:

msg = get_integer_async("How old are you?", 0);

Quanto sopra mostrerà un messaggio che richiede che l'utente inserisca un string e prema "Okay". L'id della funzione è memorizzato nella variabile "msg" e sarà usato nell'evento asincrono Dialogs come mostrato sotto:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
    if ds_map_find_value(async_load, "status")
    {
        global.Age = ds_map_find_value(async_load, "value");
    }
}

Il codice di cui sopra controlla la chiave "id" del ritorno DS Map contro il valore memorizzato nella variabile "msg". Se sono uguali, controlla se è stato premuto "Ok" (piuttosto che la finestra è stata chiusa/annullata) e se ritorna true imposta una variabile globale dall'intero restituito.
