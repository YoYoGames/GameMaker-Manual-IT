mostra_messaggio_async

Questa funzione apre una finestra e mostra all'utente il messaggio che definite nella funzione. Questa è una funzione asincrona, e come tale GameMaker non blocca il dispositivo su cui viene eseguita in attesa di una risposta, ma piuttosto continua a eseguire gli eventi normalmente. Una volta che l'utente ha premuto il pulsante "Ok", un evento asincrono Dialogo che, solo per la durata di quell'evento, avrà una ds_map memorizzata nella variabile async_load.

Questa mappa conterrà le due chiavi, "id" e "status", con "id" che è il valore che è stato restituito dalla funzione quando è stata chiamata, e lo "status" che è o true per il pulsante "Ok" premuto, o false se il messaggio è stato cancellato (se disponibile, poiché non tutte le piattaforme di destinazione permettono la cancellazione dei dialoghi).

NOTA Questa funzione è solo per debug e teste non dovrebbe essere usata nei giochi rilasciati. A tal fine dovreste creare un'interfaccia utente personalizzata per ricevere input dai giocatori in modo da avere il controllo completo su come le finestre di dialogo appaiono e si comportano.

Sintassi:

show_message_async(string);

Argomento Tipo Descrizione
string Stringa The message to show to the user.

Ritorna:

Async Request ID

Esempio esteso:

L'evento di pressione sinistra del mouse (per esempio) di object che mostra il messaggio avrebbe il seguente codice:

msg = show_message_async("You will now be taken to the store");

Quanto sopra mostrerà un messaggio con il dato string. L'id del messaggio è memorizzato nella variabile "msg" e sarà usato nell'evento asincrono Dialogs come mostrato di seguito:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if (i_d == msg)
{
    if (ds_map_find_value(async_load, "status"))
    {
        url_open("https://play.google.com/store");
    }
}

Il codice qui sopra controlla la chiave "id" del DS Map restituito contro il valore memorizzato nella variabile "msg". Se sono uguali, controlla se è stato premuto il pulsante "Ok" (piuttosto che la finestra è stata chiusa/annullata) e se ritorna true apre un url.
