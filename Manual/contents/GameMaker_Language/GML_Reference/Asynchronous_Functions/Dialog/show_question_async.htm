mostra_domanda_async

Questa funzione apre una finestra e mostra all'utente la domanda definita nella funzione. Questa è una funzione asincrona, e come tale GameMaker non blocca il dispositivo su cui viene eseguita in attesa di una risposta, ma piuttosto continua a eseguire gli eventi normalmente. La funzione ha due pulsanti che mostrano "Sì" e "No", e una volta che l'utente ne ha premuto uno, viene attivato un evento asincrono Dialog che, solo per la durata di quell'evento, avrà una mappa DS memorizzata nella variabile async_load.

Questa mappa conterrà le due chiavi, "id", e "status". "id" è il valore che è stato restituito dalla funzione quando è stata chiamata, mentre lo "status" sarà o true o false per "Yes" e "No" rispettivamente.

NOTA Questa funzione è solo per debug e teste non dovrebbe essere usata nei giochi rilasciati. A tal fine dovreste creare un'interfaccia utente personalizzata per ricevere input dai giocatori in modo da avere il controllo completo su come le finestre di dialogo appaiono e si comportano.

Sintassi:

show_question_async(string);

Argomento Tipo Descrizione
string Stringa The question to ask to the user.

Ritorna:

Async Request ID

Esempio esteso:

L'evento di pressione sinistra del mouse (per esempio) di object che mostra il messaggio avrebbe il seguente codice:

msg = show_question_async("Do you want to buy some armour for " + string(armour[0, 5]) + "coins?");

Quanto sopra mostrerà una domanda con il dato string, richiedendo che l'utente prema "yes" o "No". L'id della funzione è memorizzato nella variabile "msg" e sarà usato nell'evento Asynchronous Dialog come mostrato di seguito:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if (i_d == msg)
{
    if (ds_map_find_value(async_load, "status"))
    {
        coins -= armour[0, 5];
        global.protection += armour[0, 0];
    }
}

Il codice qui sopra controlla la chiave "id" del DS Map restituito contro il valore memorizzato nella variabile "msg". Se sono uguali, controlla se uno dei due pulsanti è stato premuto e se restituisce true allora dedurrà un valore da una variabile e aggiungerà anche un valore a una variabile globale.
