salva_stringa_nuvola

Questa funzione impegnerà un string nel servizio cloud scelto per la memorizzazione. La funzione restituirà un valore id unico che dovrebbe poi essere usato nell'evento asincrono appropriato per identificare la mappa DS che viene restituita come "richiamo" dal servizio cloud. L'indirizzo string dovrebbe contenere tutte le informazioni che è necessario salvare per il gioco, poiché è possibile memorizzare solo un singolo "blob di dati" sul cloud, ed eseguire nuovamente questa funzione sovrascriverà qualsiasi valore precedentemente memorizzato (così come l'utilizzo della cloud_file_save() funzione). La descrizione dovrebbe essere un breve string di informazioni che descrive il salvataggio, ad esempio: "Level2, Stage2".

Per ulteriori informazioni sui dati asincroni restituiti, vedere la funzione cloud_synchronise().

Sintassi:

cloud_string_save(string, description);

ArgomentoTipo Descrizione
string The data string to be uploaded.
description A brief description of the data being stored.

Ritorna:

Async Request ID

Esempio:

var t_str = "";
for (var i = 0; i < 10; i++;)
{
    t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

Il codice di cui sopra crea un string dai valori memorizzati nell'array globale "Highscores" e poi scrive questo string al servizio cloud per la memorizzazione, così come lo scrive in un file per la memorizzazione locale.
