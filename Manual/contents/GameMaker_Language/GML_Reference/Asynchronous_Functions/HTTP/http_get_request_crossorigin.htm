http_get_request_crossorigin

Questa funzione può essere usata per ottenere il tipo cross-origin impostato per i giochi HTML5 e restituirà un string (su tutte le altre piattaforme verrà restituito un string vuoto "").

Sintassi:

http_get_request_crossorigin();

Ritorna:

Esempio:

if string_lower(http_get_request_crossorigin()) != "use-credentials"
{
    http_set_request_crossorigin("use-credentials");
}
sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0);

Il codice di cui sopra controlla prima il tipo di origine incrociata attualmente impostato e se non è "use-credentials" allora viene impostato su "use-credentials" e poi viene aggiunto un sprite da un file.

Indietro: HTTP
Il prossimo: http_set_request_crossorigin
