buffer_load_ext

Questa funzione caricherà i dati di buffer che sono stati precedentemente salvati usando le buffer_save() in un buffer esistente. Si dà l'ID del file buffer creato in precedenza in cui caricare, poi il file buffer salvato da caricare, e infine l'offset dall'inizio di buffer (in byte) in cui si desidera caricare i dati.

Si prega di leggere la pagina buffer_load() per le note specifiche della piattaforma.

Sintassi:

buffer_load_ext(buffer, filename, offset);

ArgomentoTipo Descrizione
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

Ritorna:

Esempio:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

Il codice qui sopra otterrà prima la posizione della fine di buffer indicizzata nella variabile "player_buffer" e poi caricherà i dati dal dato in quella posizione (nota che questo esempio funzionerà solo con i tipi "grow" o "wrap" buffer ).
