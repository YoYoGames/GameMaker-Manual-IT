buffer_fill

Questa funzione può essere usata per riempire un buffer precedentemente creato con un dato data type e un valore. I dati con cui riempite il buffer devono essere in accordo con l'argomento "type" di questa funzione, il che significa che non potete provare a riempire con un string e usare il tipo unsigned 16bit integer, per esempio. Le costanti di tipo sono le stesse usate dalle funzioni buffer_read() e buffer_write() funzioni. La "dimensione" è la dimensione del buffer (in byte) che volete riempire, mentre l'offset è il valore di offset (anch'esso in byte) dall'inizio del buffer da cui iniziare il riempimento.

Sintassi:

buffer_fill(buffer, offset, type, value, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Ritorna:

Esempio:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Il codice qui sopra trova l'inizio del buffer con l'id memorizzato nella variabile "buff" e vi scrive una serie di valori interi firmati a 16 bit.
