buffer_poke

Con la funzione buffer_write() potete scrivere dati sul dato buffer nella posizione di "ricerca" corrente, con ogni pezzo di dati che fa avanzare questa posizione dei byte scritti o letti. Tuttavia, potrebbe essere necessario per voi cambiare un dato pezzo di dati senza voler cambiare la posizione di ricerca corrente, ed è allora che usereste questa funzione. Dovete semplicemente fornire alla funzione un indice buffer, e poi la posizione di offset dall'inizio di buffer (in byte) all'interno di quel buffer su cui scrivere, così come data type e il valore da scrivere.

Sintassi:

buffer_poke(buffer, offset, type, value);

Argomento Tipo Descrizione
buffer ID del buffer The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to write the given data to.
type Buffer Data Type Costante The type of data that is to be written to the buffer (see the list of constants here).
value Variabile The data to add to the buffer, in accordance with the type specified.

Ritorna:

N/A

Esempio:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

Il codice qui sopra aggiungerà il valore della componente blu del colore usato per la fusione dell'immagine nel buffer indicizzato nella variabile "buff", alla terza posizione nel buffer e come valore senza segno a 8 bit.
