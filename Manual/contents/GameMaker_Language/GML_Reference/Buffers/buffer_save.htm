salvataggio del buffer

Con questa funzione potete salvare il contenuto di un buffer in un file, pronto per essere riletto in memoria usando la buffer_load() funzione.

NOTA: su HTML5 il contenuto di buffer sarà salvato come base64 encoded strings quando si usa questa funzione.

Sintassi:

buffer_save(buffer, filename);

ArgomentoTipo Descrizione
buffer The index of the buffer to save.
filename The name of the file to save as.

Ritorna:

Esempio:

buffer_save(buff, "Player_Save.sav");

Salva il contenuto corrente di buffer con l'id memorizzato nella variabile "buff" in un file.

Indietro: Buffer
Il prossimo: buffer_save_ext
