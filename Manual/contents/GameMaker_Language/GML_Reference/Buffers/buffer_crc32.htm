buffer_crc32

Questa funzione prende i dati in ingresso da un buffer e restituisce un hash crc32 checksum. Si specifica l'ID buffer del buffer da usare, poi un valore di offset (in byte) per dove iniziare, e poi una dimensione (di nuovo in byte) per la regione da hashtare, e la funzione restituirà un valore intero a 32 bit per quella regione.

Sintassi:

buffer_crc32(buffer, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Ritorna:

Esempio:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

Il codice di cui sopra creerà un valore di checksum crc32 per i dati completi memorizzati in buffer indicizzati dalla variabile "buff", e memorizzerà il valore di hash intero restituito nella variabile "check_val".
