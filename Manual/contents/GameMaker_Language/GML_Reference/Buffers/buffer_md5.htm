buffer_md5

In crittografia, MD5 (Message-Digest algorithm 5) è una funzione di hashing crittografico ampiamente utilizzata con un valore di hash a 128 bit ed è stata impiegata in un'ampia varietà di applicazioni di sicurezza. È anche comunemente usato per controllare l'integrità dei dati. Questa funzione prenderà i dati di input da un dato buffer (o parte di buffer) e restituirà l'hash MD5 esadecimale di 32 caratteri che è unico per quei dati. In questo modo è possibile generare una chiave sicura che può essere memorizzata e utilizzata per controllare l'integrità delle informazioni inviate a (o ricevute da) un server esterno (per esempio). Quando si applica a buffers usando questa funzione, si deve specificare l'id buffer del buffer da usare, poi un valore di offset (in byte) per dove iniziare, e poi una dimensione (di nuovo in byte) per la regione da hashare.

NOTA: MD5 non è completamente sicuro e può essere rotto. Vedi questa pagina per maggiori informazioni.

Sintassi:

buffer_md5(buffer, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Ritorna:

Esempio:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

Il codice di cui sopra creerà un hash md5 per i dati completi memorizzati in buffer indicizzati dalla variabile "buff", e memorizzerà l'hash restituito nella variabile "check_string".
