buffer_sizeof

Questa funzione restituisce la dimensione (in byte) del dato Buffer Data Type Constant (elencato qui).

Sintassi:

buffer_sizeof(type);

Argomento Tipo Descrizione
type Buffer Data Type Costante The type of data that is to be checked (see the list of constants here).

Ritorna:

Real

Esempio:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Il codice di cui sopra calcola prima la dimensione del buffer da creare moltiplicando l'unsigned 8bit data type per 12 (dato che useremo il buffer per contenere 12 pezzi di dati), e poi usa questo valore per impostare un buffer fisso alla dimensione corretta.

Indietro: Buffer
Il prossimo: buffer_md5
