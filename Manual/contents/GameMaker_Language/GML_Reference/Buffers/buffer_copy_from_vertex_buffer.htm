buffer_copy_from_vertex_buffer

Questa funzione può essere usata per copiare alcuni (o tutti) i dati dei vertici memorizzati in un vertice buffer in un buffer regolare precedentemente creato. Quando si copia da un vertice buffer in un regolare buffer con questa funzione, entrambi i buffers devono essere stati precedentemente creati (usando i tasti vertex_create_buffer() e buffer_create() per esempio). Potete specificare l'intervallo di dati dei vertici che desiderate copiare in buffer, dove il vertice iniziale può essere ovunque tra 0 e il numero di vertici -1, e potete dare il numero di vertici da quel punto in poi da copiare. Potete usare la funzione vertex_get_number() sul vertice buffer per ottenere il numero totale di vertici memorizzati. Infine date l'indice buffer in cui copiare i dati del vertice, così come un offset di dati per definire la posizione in cui copiare i dati del vertice nella destinazione buffer.

Sintassi:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argomento Tipo Descrizione
vertex_buffer Vertex Buffer ID The index of the vertex buffer to copy from.
start_vertex Real The starting vertex.
num_vertices Real The total number of vertices to use.
dest_buffer ID del buffer The index of the buffer to copy to.
dest_offset Real The offset position to copy the data to (in bytes).

Ritorna:

N/A

Esempio:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Il codice di cui sopra copierà i dati dei vertici memorizzati nel vertice buffer indicizzato nella variabile "model_buffer", e poi li incollerà nel buffer indicizzato nella variabile "player_buffer".
