comprimere il buffer

Con questa funzione potete comprimere parte (o tutto) di un buffer usando la compressione zlib. Si fornisce l'ID del buffer da comprimere (come restituito da buffer_create()), l'offset all'interno di buffer da usare in byte, e la dimensione dei dati di buffer da comprimere (anch'essa in byte). La funzione restituirà un nuovo valore ID buffer per il buffer compresso, o un valore inferiore a 0 se è fallita per qualsiasi motivo. Questa funzione non altera il buffer originale.

Sintassi:

buffer_compress(buffer, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Ritorna:

Buffer ID

Esempio:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Il codice qui sopra creerà un buffer poi lo popolerà con i dati di un string. Questo buffer è poi compresso e salvato, e sia il sorgente che il compresso buffers sono cancellati
