ridimensionamento del buffer

Con questa funzione potete ridimensionare un dato buffer per essere della dimensione (in byte) che specificate.

Sintassi:

buffer_resize(buffer, newsize);

ArgomentoTipo Descrizione
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Ritorna:

Esempio:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

Il codice qui sopra controllerà la dimensione del buffer indicizzato nella variabile "buff" e se è inferiore al valore dato, il buffer viene ridimensionato.

Indietro: Buffer
Il prossimo: buffer_sizeof
