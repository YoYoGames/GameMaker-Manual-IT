buffer_sha1

In crittografia, SHA-1 è una funzione di hashing crittografico progettata dalla United States National Security Agency ed è impiegata in diverse applicazioni e protocolli ampiamente utilizzati come il popolare Git dove è usato per controllare le modifiche ai file. Questa funzione prende i dati di input da un buffer e restituisce un message digest a 160 bit in formato ASCII. In questo modo è possibile generare una chiave sicura che può essere memorizzata e utilizzata per controllare l'integrità delle informazioni inviate a (o ricevute da) un server esterno (per esempio). Quando si applica a buffers usando questa funzione, si deve specificare l'id buffer del buffer da usare, poi un valore di offset (in byte) per dove iniziare, e poi una dimensione (di nuovo in byte) per la regione da sottoporre ad hash.

NOTA: SHA-1 non è completamente sicuro e può essere rotto. Vedi questa pagina per maggiori informazioni.

Sintassi:

buffer_sha1(buffer, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Ritorna:

Esempio:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

Il codice di cui sopra creerà un hash sha1 per i dati completi memorizzati in buffer indicizzati dalla variabile "buff", e memorizzerà l'hash restituito nella variabile "check_string".
