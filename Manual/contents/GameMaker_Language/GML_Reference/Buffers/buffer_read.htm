buffer_read

Questa funzione può essere utilizzata per leggere i dati da un sito buffer precedentemente creato. Il valore di ritorno dipende dal tipo di dati che si sta leggendo, che di per sé è definito dalle seguenti costanti:

Costante Descrizione
buffer_u8 Un numero intero a 8 bit senza segno. È un valore positivo compreso tra 0 e 255.
buffer_s8 Un numero intero firmato a 8 bit. Può essere un valore positivo o negativo compreso tra -128 e 127 (0 è considerato positivo).
buffer_u16 Un numero intero a 16 bit senza segno. È un valore positivo compreso tra 0 e 65.535.
buffer_s16 Un numero intero firmato a 16 bit. Può essere un valore positivo o negativo compreso tra -32.768 e 32.767 (0 è considerato positivo).
buffer_u32 Un numero intero a 32 bit senza segno. Si tratta di un valore positivo compreso tra 0 e 4.294.967.295.
buffer_s32 Un numero intero firmato a 32 bit. Può essere un valore positivo o negativo compreso tra -2.147.483.648 e 2.147.483.647 (0 è considerato positivo).
buffer_u64 Un intero a 64 bit senza segno.
buffer_f16 Un float a 16 bit. Può essere un valore positivo o negativo compreso nell'intervallo +/- 65504. (Attualmente non supportato!)
buffer_f32 Un float a 32 bit. Può essere un valore positivo o negativo compreso nell'intervallo +/-16777216.
buffer_f64 Un float a 64 bit.
buffer_bool Un valore booleano. Può essere solo 1 o 0 (true o false).
buffer_string Un string di qualsiasi dimensione.
buffer_text Un string di qualsiasi dimensione, senza il carattere finale null.

Se la funzione ha successo, restituirà un valore del tipo indicato, ma se fallisce causerà un errore di esecuzione.

NOTA L'utilizzo di data type non corretto per i dati da leggere comporta la restituzione di valori errati.

Sintassi:

buffer_read(buffer, type)

Argomento Tipo Descrizione
buffer ID buffer The index of the buffer to read from.
type Buffer Data Type Costante The type of data that is to be read from the buffer (see the list of constants above).

Restituzione:

Real, Boolean or String

Esempio:

var cmd = buffer_read(buff, buffer_s16);

Il codice precedente legge dal sito buffer con l'id memorizzato nella variabile "buff" un valore firmato a 16 bit nella variabile locale "cmd".
