buffer_save_ext

Con questa funzione potete salvare parte del contenuto di un buffer in un file, pronto per essere riletto in memoria usando la buffer_load() funzione. L'"offset" definisce la posizione iniziale all'interno del buffer per il salvataggio (in byte), e la "dimensione" è la dimensione dell'area buffer da salvare da quell'offset in poi (sempre in byte).

Sintassi:

buffer_save_ext(buffer, filename, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Ritorna:

Esempio:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Salva parte del contenuto corrente di buffer con l'id memorizzato nella variabile "buff" in un file.

Indietro: Buffer
Il prossimo: buffer_save_async
