cercare il buffer

Questa funzione può essere usata per muoversi attraverso un buffer, trovando l'inizio, la fine, o una posizione relativa a quella che è stata usata per ultima durante la lettura o la scrittura dei dati. Il valore "offset" è l'offset (in byte) da aggiungere alla posizione di ricerca data, per esempio, se la base è relativa e l'offset è 4, allora la posizione buffer si sposterà lungo 4 byte dalla sua posizione attuale. Si prega di notare quanto segue:

  • Potete usare valori negativi per l'offset per cercare all'indietro attraverso buffer così come valori positivi.
  • Se buffer è del tipo "a capo" e si sfasa oltre la fine di buffer, anche la posizione di ricerca sarà a capo.
  • Se buffer non è del tipo "a capo", la ricerca si blocca all'inizio o alla fine di buffer, anche quando l'offset porterebbe la ricerca fuori dai limiti di buffer.

Le seguenti costanti sono accettate come argomento "base" per cercare di:

Costante di ricerca del buffer
Costante Descrizione
buffer_seek_start L'inizio del buffer
buffer_seek_relative Una posizione relativa alla posizione corrente di lettura/scrittura
buffer_seek_end La fine del buffer

Sintassi:

buffer_seek(buffer, base, offset);

Argomento Tipo Descrizione
buffer ID del buffer The index of the buffer to use.
base Real The base position to seek.
offset Real The data offset value.

Ritorna:

N/A

Esempio:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Il codice qui sopra trova l'inizio del buffer con l'id memorizzato nella variabile "buff" e vi scrive una serie di valori interi firmati a 16 bit.
