buffer_peek

Con la funzione buffer_read() potete leggere i dati dal dato buffer nella posizione di "ricerca" corrente, con ogni pezzo di dati letto che avanza questa posizione per i byte letti o scritti. Tuttavia, potrebbe essere necessario per voi leggere un dato pezzo di dati senza voler cambiare la posizione di ricerca corrente, ed è allora che usereste questa funzione. Dovete semplicemente fornire alla funzione un id buffer, e poi una posizione di offset (dall'inizio di buffer ) all'interno di quel buffer da cui leggere, così come il data type che volete leggere.

NOTA: L'utilizzo di un data type non corretto per i dati da leggere risulterà in valori errati!

Se la funzione ha successo, verrà restituito un valore (reale, string, booleano, ecc...) a seconda di data type, mentre un fallimento restituirà semplicemente 0.

Sintassi:

buffer_peek(buffer, offset, type);

Argomento Tipo Descrizione
buffer ID del buffer The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to read the given data from.
type Buffer Data Type Costante The type of data that is to be read from the buffer (see the list of constants here).

Ritorna:

Real, Boolean, String or 0 (if it fails)

Esempio:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Il codice di cui sopra otterrà tre valori da tre diverse posizioni all'interno di buffer indicizzate nella variabile "buff" e poi userà quei valori per impostare la miscela dell'immagine dell'istanza.
