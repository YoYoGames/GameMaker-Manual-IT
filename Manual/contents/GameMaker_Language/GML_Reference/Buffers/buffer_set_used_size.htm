buffer_set_used_size

Questa funzione è principalmente per l'uso all'interno di extensions, e permette di impostare la dimensione "usata" del dato buffer, che è il numero di byte che sono stati scritti in esso.

Quando si scrivono dati su un buffer da un extension, GameMaker non sa quanto di buffer è stato riempito dal codice extension e non è in grado di leggere quei dati. Questa funzione può essere chiamata da extension per dire al motore quanti byte di dati è stato scritto su buffer, in modo che il motore possa leggere quei dati.

Sintassi:

buffer_set_used_size(buffer, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Ritorna:

Esempio:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Le funzioni di cui sopra sarebbero chiamate in un extension per scrivere alcuni dati in un GameMaker buffer (attraverso il suo indirizzo di memoria), e poi impostare il numero di byte che sono stati scritti in esso in modo che il motore possa leggere quei dati.
