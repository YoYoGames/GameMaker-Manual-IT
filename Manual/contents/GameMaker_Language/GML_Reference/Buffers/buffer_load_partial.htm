buffer_load_parziale

Questa funzione caricherà alcuni dei dati di buffer che sono stati precedentemente salvati usando le buffer_save() in un buffer esistente. Voi date l'id del buffer precedentemente creato in cui caricare, poi il file buffer salvato da caricare, e poi l'offset dall'inizio del buffer (in byte) da cui volete caricare i dati. I seguenti argomenti servono per impostare la lunghezza dei dati di buffer (in byte) dal punto di offset iniziale che si desidera caricare e il punto di offset a cui caricare i dati in buffer (di nuovo, in byte).

Si prega di leggere la pagina buffer_load() per le note specifiche della piattaforma.

Sintassi:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

ArgomentoTipo Descrizione
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Ritorna:

N/A

Esempio:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

Il codice di cui sopra creerà un nuovo "grow" buffer e poi vi caricherà una parte dei dati salvati nel file "save.dat".
