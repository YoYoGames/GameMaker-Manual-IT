buffer_base64_encode

Questa funzione convertirà i dati dal dato buffer in un formato base64 codificato string. Questo è uno schema di codifica comunemente usato che è spesso utilizzato per qualsiasi media che deve essere memorizzato o trasferito su Internet come testo, e rende l'output l'output illeggibile all'occhio umano. Per usarlo è necessario specificare un buffer già creato, il valore di offset (che è il punto all'interno del buffer in cui si desidera iniziare la codifica) così come la dimensione, in byte, della memoria buffer da codificare.

Sintassi:

buffer_base64_encode(buffer, offset, size);

ArgomentoTipo Descrizione
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Ritorna:

Esempio:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Il codice di cui sopra creerà la codifica dei dati completi memorizzati in buffer indicizzati dalla variabile "buff", e memorizzerà il string restituito nella variabile locale "b_str".
