buffer_async_group_option

Con questa funzione è possibile impostare alcune opzioni specifiche della piattaforma per il gruppo buffer che viene salvato/caricato. Le opzioni disponibili sono le seguenti:

  • "subtitle" o "slottitle" - Il valore di questa opzione sarebbe un string, che sarà mostrato all'utente quando gestisce i suoi dati di salvataggio nel SO. Questo è importante solo quando si salvano i dati, non quando si caricano di nuovo.
  • "showdialog" - Il valore di questa opzione è un booleano true o false. Se impostato su true, quando si salva o si carica GameMaker mostrerà l'interfaccia utente di sistema, altrimenti farà un quicksave/quickload senza mostrare alcuna interfaccia. Normalmente è necessario solo se si supportano più slot di salvataggio per permettere all'utente di scegliere uno slot, ma se si supporta solo uno slot per utente, impostatelo su false.
  • "savepadindex" - Per questa opzione devi specificare l'indice del pad del giocatore che sta salvando o caricando e il sistema scriverà e leggerà i dati nella cartella di salvataggio di questo giocatore.
  • "saveslotsize" - Questa opzione richiede che specifichiate la dimensione effettiva in byte che volete salvare (così potete fare un buffer_seek() e buffer_tell() per ottenerlo, per esempio). Nota che non è obbligatorio fornire questo valore, poiché tutti i salvataggi sono pre-assegnati ad uno spazio minimo, che di solito varia con la piattaforma di destinazione.
  • "ps_create_backup" - (solo per PlayStation) - Impostando questo a true ci si assicura che i dati di salvataggio vengano sottoposti a backup quando la directory di salvataggio viene smontata (cioè quando viene chiamata la funzione buffer_async_group_end() ). Vedere la documentazione della console per maggiori informazioni.

IMPORTANTE! Questa funzione è attualmente valida solo per i moduli target PSVita, PS4 e XBox One. Su tutti gli altri target non farà nulla.

Sintassi:

buffer_async_group_option(option, value);

Argomento Tipo Descrizione
option Stringa The option to set.
value Reale o Stringa The value to set (can be a real or string, depending on the option).

Ritorna:

N/A

Esempio:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Il codice qui sopra inizia un gruppo buffer poi imposta le opzioni del gruppo prima di impostare 4 file da salvare in modo asincrono. La definizione del gruppo viene poi terminata (a quel punto inizierà il salvataggio).
