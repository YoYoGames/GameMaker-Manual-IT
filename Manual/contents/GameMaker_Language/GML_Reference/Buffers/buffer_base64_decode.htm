buffer_base64_decode

Con questa funzione è possibile decodificare un string codificato in base64 (creato utilizzando la funzione buffer_base64_encode() ) in un buffer. Questa funzione creerà il buffer (come un buffer "grow" allineato a 1 byte) e restituirà l'indice unico per il buffer che dovrebbe essere usato in tutte le ulteriori chiamate di funzione.

NOTA: è importante che rimuoviate qualsiasi risorsa creata dinamicamente come questa dalla memoria quando non ne avete più bisogno per evitare perdite di memoria, quindi quando avete finito con il buffer che avete creato dovreste liberarlo di nuovo usando buffer_delete().

Sintassi:

buffer_base64_decode(string);

ArgomentoTipo Descrizione
string The base64 encoded string to decode

Ritorna:

Buffer ID

Esempio:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

Il codice qui sopra aprirà un file ini e poi leggerà un string da esso nella funzione di decodifica. Questa funzione restituirà un indice buffer, che viene memorizzato nella variabile "buff", contenente i dati precedentemente codificati e salvati. Il file ini viene quindi chiuso.
