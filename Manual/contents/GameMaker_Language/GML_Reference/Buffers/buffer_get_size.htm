buffer_get_size

Con questa funzione potete ottenere la dimensione del dato buffer in byte.

Sintassi:

buffer_get_size(buffer);

Argomento Tipo Descrizione
buffer ID del buffer The index of the buffer to get the size of.

Ritorna:

Real

Esempio:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

Il codice di cui sopra creerà un nuovo buffer e memorizzerà il suo indice nella variabile locale "_temp", con la dimensione di questo nuovo buffer uguale a quella del buffer indicizzato nella variabile "player_data".
