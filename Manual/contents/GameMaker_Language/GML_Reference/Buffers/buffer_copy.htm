buffer_copy

Questa funzione può essere usata per copiare un segmento (o tutti) i dati memorizzati in un buffer in un altro. Quando si usano due buffers e si copia da uno all'altro, entrambi i buffers devono essere stati precedentemente creati usando la funzione buffer_create() (per esempio), ed è possibile specificare un offset di dati (in byte) per il punto di inizio dei dati da copiare dal sorgente buffer rispetto all'inizio del buffer, così come un altro offset di dati per definire la posizione in cui copiare i dati in nella destinazione buffer.

NOTA: non è possibile copiare sullo stesso buffer.

Sintassi:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

ArgomentoTipo Descrizione
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

Ritorna:

Esempio:

buffer_copy(buff1, 0, 2048, buff2, 2048);

Il codice di cui sopra copierà i dati memorizzati in buffer indicizzati nella variabile "buff1", e poi li incollerà in buffer indicizzati nella variabile "buff2", ma sfalsati di 2048 byte.
