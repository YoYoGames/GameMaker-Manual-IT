buffer_decompressione

Con questa funzione potete decomprimere un buffer precedentemente compresso usando la compressione zlib. Tu fornisci l'ID buffer da decomprimere, e la funzione restituirà un nuovo ID buffer che contiene i dati non compressi. Se la decompressione è fallita (per esempio, state fornendo un buffer che non è stato compresso) allora la funzione restituirà un valore inferiore a 0.

Sintassi:

buffer_decompress(buffer);

ArgomentoTipo Descrizione
buffer The index of the buffer to decompress.

Ritorna:

Buffer ID

Esempio:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Il codice qui sopra caricherà prima un buffer salvato, poi lo decomprimerà e infine leggerà i dati di string dal buffer decompresso in una variabile globale.

Indietro: Buffer
