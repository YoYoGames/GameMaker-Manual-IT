ds_priority_find_min

Con questa funzione potete trovare il valore memorizzato nella priorità queue con la priorità più bassa, e se più di un valore ha la stessa priorità, allora ognuno di loro potrebbe essere restituito in qualsiasi ordine. Tuttavia, a differenza di ds_priority_delete_min()questa funzione non rimuove il valore da queue.

Sintassi:

ds_priority_find_min(id);

ArgomentoTipo Descrizione
id The id of the priority queue to use.

Ritorna:

Real or String

Esempio:

if ai_move
{
    script_execute(ds_priority_find_min(ai_priority));
}

Il codice qui sopra controlla una variabile di istanza e se restituisce true eseguirà una funzione script indicizzata nella priorità queue con il valore di priorità più basso.
