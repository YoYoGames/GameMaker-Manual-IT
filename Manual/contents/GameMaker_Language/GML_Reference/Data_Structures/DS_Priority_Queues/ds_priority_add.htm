ds_priority_add

Con questa funzione potete aggiungere un valore (un numero reale o un string) a una priorità queue, assegnandogli allo stesso tempo un valore di priorità.

Sintassi:

ds_priority_add(id, val, priority);

Argomento Tipo Descrizione
id ID di priorità DS The id of the priority queue to add to.
val Variabile The value to add to the priority queue.
priority Real The priority of the value to add.

Ritorna:

N/A

Esempio:

ds_priority_add(ai_priority, AI_Search, 5);

Il codice qui sopra aggiunge una funzione script alla priorità queue indicizzata nella variabile "ai_priority" e le assegna una priorità di 5.

Indietro: Code di priorità DS
