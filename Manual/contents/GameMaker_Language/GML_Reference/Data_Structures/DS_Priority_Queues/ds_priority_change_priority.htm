ds_priority_change_priority

Questa funzione prenderà un valore dato e cambierà la sua priorità all'interno della priorità di riferimento queue. Il valore dato dovrebbe già esistere nella priorità queue.

Sintassi:

ds_priority_change_priority(id, val, priority);

Argomento Tipo Descrizione
id ID di priorità DS The id of the priority queue to change.
val Variabile The value to change the priority of.
priority Real The new priority of the value.

Ritorna:

N/A

Esempio:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

Il codice qui sopra controlla una variabile globale e se è al di sotto di un certo valore cambierà la priorità dell'indice della funzione script contenuta nella priorità queue.
