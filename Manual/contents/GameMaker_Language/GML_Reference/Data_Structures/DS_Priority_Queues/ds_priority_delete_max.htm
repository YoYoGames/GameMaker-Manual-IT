ds_priority_delete_max

Questa funzione restituirà il valore che ha la priorità più alta in queue e poi rimuoverà il valore (e la priorità) dalla struttura dati. Se più di un valore ha la stessa priorità, allora uno qualsiasi di essi potrebbe essere restituito in qualsiasi ordine, ma tutti gli altri valori con la stessa priorità saranno ancora in queue.

Sintassi:

ds_priority_delete_max(id);

ArgomentoTipo Descrizione
id The id of the priority queue to use.

Ritorna:

(Data type stored in the priority)

Esempio:

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

Il codice precedente controlla una variabile di istanza e se restituisce true eseguirà un script indicizzato nella priorità queue con il valore di priorità più alto e poi rimuoverà quel script dal queue.
