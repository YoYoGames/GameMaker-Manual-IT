ds_priority_copy

Questa funzione può essere usata per copiare il contenuto di una priorità queue in un'altra. Si noti che questo NON rimuove i contenuti dalla priorità originale queue, né distrugge la priorità originale queue. Quando si usa questa funzione, la priorità queue in cui si sta copiando deve essere stata creata in precedenza e se conteneva elementi prima della copia, questi saranno cancellati per primi (il che significa che queste informazioni saranno perse).

Sintassi:

ds_priority_copy(id, source);

ArgomentoTipo Descrizione
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Ritorna:

Esempio:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

La funzione precedente crea una nuova istanza e poi in quell'istanza crea una nuova priorità DS queue e copia il contenuto della priorità queue nell'istanza che esegue il blocco di codice, nella priorità dell'istanza appena creata queue.

Indietro: Code di priorità DS
