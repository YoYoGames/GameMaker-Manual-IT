ds_priority_delete_min

Questa funzione restituirà il valore che ha la priorità più bassa in queue e poi rimuoverà il valore (e la priorità) dalla struttura dati. Se più di un valore ha la stessa priorità, allora uno qualsiasi di essi potrebbe essere restituito in qualsiasi ordine, ma tutti gli altri valori con la stessa priorità saranno ancora in queue.

Sintassi:

ds_priority_delete_min(id);

ArgomentoTipo Descrizione
id The id of the priority queue to use.

Ritorna:

(Data type stored in the priority)

Esempio:

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

Il codice qui sopra controlla una variabile di istanza e se restituisce true eseguirà una funzione script indicizzata nella priorità queue con il valore di priorità più basso e poi rimuoverà quella script dalla queue.
