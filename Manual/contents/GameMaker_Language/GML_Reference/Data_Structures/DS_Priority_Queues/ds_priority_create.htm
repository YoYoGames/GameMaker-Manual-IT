ds_priority_create

Questa funzione creerà una nuova struttura dati di priorità queue e restituirà il valore dell'indice. Questo valore dovrebbe essere memorizzato in una variabile e utilizzato in tutte le ulteriori chiamate di funzioni relative alla priorità queue.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere usato da una nuova struttura creata in seguito.

Sintassi:

ds_priority_create();

Ritorna:

DS Priority ID

Esempio:

p_queue = ds_priority_create();

Questo creerà una nuova priorità queue e assegnerà il suo indice id alla variabile di istanza "p_queue".

Indietro: Code di priorità DS
Il prossimo: ds_priority_destroy
