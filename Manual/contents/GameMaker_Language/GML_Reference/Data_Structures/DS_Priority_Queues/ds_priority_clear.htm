ds_priority_clear

Con questa funzione potete cancellare tutti i dati dalla data struttura di priorità queue. Questo NON distrugge la struttura dati (per quello dovreste usare ds_priority_destroy()) cancella solo tutti i dati da essa e restituisce una priorità vuota queue.

Sintassi:

ds_priority_clear(id);

ArgomentoTipo Descrizione
id The id of the data structure to clear.

Ritorna:

Esempio:

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella la priorità DS queue indicizzata nella variabile "command_queue", imposta un allarme e resetta la variabile a 0.

Indietro: Code di priorità DS
Il prossimo: ds_priority_empty
