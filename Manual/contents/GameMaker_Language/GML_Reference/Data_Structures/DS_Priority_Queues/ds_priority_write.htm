ds_priority_write

Questa funzione restituisce un string che può poi essere memorizzato o trasferito in un'altra struttura di dati utilizzando la ds_priority_read() funzione.

NOTA: Il string restituito non è un string leggibile dall'uomo, ma piuttosto un dump del contenuto della struttura dei dati

Sintassi:

ds_priority_write(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

Il codice di cui sopra apre un file ini e poi scrive un string contenente le informazioni memorizzate nella priorità DS queue indicizzata nella variabile "p_queue". La priorità queue viene poi cancellata e il file ini chiuso.

Indietro: Code di priorità DS
