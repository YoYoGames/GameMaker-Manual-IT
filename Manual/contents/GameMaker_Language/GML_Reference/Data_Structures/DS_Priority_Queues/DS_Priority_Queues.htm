Code di priorità DS

Una priorità DS queue è una struttura di dati speciale simile a una DS queuesolo che qui il queue di valori è ordinato dalle priorità (sempre numeri reali) che i valori sono stati assegnati. Questa struttura può essere molto utile per costruire classifiche o liste di informazioni in cui l'importanza di ogni voce può essere ponderata.

NOTA: Come tutte le risorse dinamiche, le strutture dati occupano memoria e quindi dovrebbero sempre essere distrutte quando non sono più necessarie per evitare perdite di memoria che rallentano e alla fine fanno crashare il gioco.

NOTA: Se avete bisogno di controllare se una struttura di dati esiste, potete usare la funzione ds_exists() funzione.

Esistono le seguenti funzioni che si occupano delle code di priorità DS:

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
