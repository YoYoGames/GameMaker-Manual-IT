ds_priority_size

Questa funzione restituisce la "dimensione" della priorità queue, cioè: il numero di elementi che sono stati prioritarizzati in essa.

Sintassi:

ds_priority_size(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if !ds_priority_empty(control_priority)
{
    num = ds_priority_size(control_priority);
}

Il codice qui sopra controlla una priorità DS queue per vedere se è vuota o no, e se non lo è, ottiene il numero di elementi che contiene e memorizza il valore in una variabile.
