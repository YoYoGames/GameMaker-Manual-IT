ds_priority_find_priority

Con questa funzione potete recuperare la priorità di qualsiasi valore dato. Se il valore non esiste nella priorità queue allora verrà restituito undefined.

Sintassi:

ds_priority_find_priority(id, val);

ArgomentoTipo Descrizione
id The id of the priority queue to use.
val The value to find the priority of.

Ritorna:

Real or undefined

Esempio:

p = ds_priority_find_priority(ai_priority, "intelligence");

Il codice precedente memorizzerà la priorità restituita per il valore dato nella variabile di istanza "p".
