ds_list_read

Con questa funzione si può ricreare una lista DS salvata (una che è stata precedentemente scritta come string usando ds_list_write()). Dovete prima creare una nuova lista DS in cui leggere il string, e se la lista DS esiste già e ha informazioni memorizzate in essa, allora questa sarà cancellata prima della lettura. Questa funzione è di vitale importanza quando si creano meccanismi di salvataggio/caricamento per il gioco.

Notate che se il DS specificato string è stato scritto da GameMaker: Studio 1.2.x runtime (o più vecchio), dovreste specificare l'argomento opzionale "legacy", impostandolo su true poiché il formato string è cambiato dopo.

Sintassi:

ds_list_read(id, str [, legacy]);

Argomento Tipo Descrizione
id ID elenco DS The id of the data structure to read into.
str Stringa The string to read from.
legacy Booleano OPTIONAL Può essere true o false o omesso completamente.

Ritorna:

N/A

Esempio:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Lists", "0", "");
if (str != "")
{
    ds_list_read(list, str);
}
ini_close();

Il codice qui sopra crea una lista e memorizza l'indice nella variabile "list". Poi apre un file ini e legge un string da quello, controllando per assicurarsi che il string non venga prima restituito come vuoto. Questo string viene poi letto nella lista appena creata ds_list.

Indietro: Elenchi DS
