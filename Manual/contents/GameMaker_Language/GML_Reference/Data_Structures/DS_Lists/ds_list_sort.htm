ds_list_sort

Con questa funzione potete ordinare tutti i valori all'interno di una lista, sia in ordine crescente che decrescente. Se la lista contiene strings, questi saranno ordinati alfabeticamente, in base all'alfabeto inglese di 26 lettere.

Sintassi:

ds_list_sort(id, ascend);

ArgomentoTipo Descrizione
id The id of the list to sort.
ascend Whether the values should be ascending (true) o discendente (false).

Ritorna:

Esempio:

if newgame
{
    ds_list_sort(name_list, true);
}

Il codice qui sopra ordinerà la lista indicizzata nella variabile "name_list" se la variabile "newgame" è segnalata come true.
