ds_list_empty

Con questa funzione potete controllare la lista DS data per vedere se è vuota (restituisce true) o no (restituisce false).

Sintassi:

ds_list_empty(id);

Argomento Tipo Descrizione
id Elenco DS id The id of the data structure to check.

Ritorna:

Esempio:

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella la lista DS indicizzata nella variabile "command_list", imposta un allarme e resetta la variabile a 0.

Indietro: Elenchi DS
Il prossimo: ds_list_size
