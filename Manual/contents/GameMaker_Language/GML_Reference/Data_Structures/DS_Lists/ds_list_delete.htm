ds_list_delete

Con questa funzione potete rimuovere il valore memorizzato in una posizione specifica all'interno della lista.

Sintassi:

ds_list_delete(id, pos);

ArgomentoTipo Descrizione
id The id of the list to change.
pos Where in the list to delete the value.

Ritorna:

Esempio:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

Il codice qui sopra controlla la dimensione di una lista DS e se è più grande di dieci, loops attraverso la lista rimuovendo il valore superiore (posizione 0) finché la lista ha solo 10 voci.
