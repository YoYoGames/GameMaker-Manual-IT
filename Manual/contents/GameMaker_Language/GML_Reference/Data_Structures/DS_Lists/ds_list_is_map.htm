ds_list_is_map

Con questa funzione potete controllare se una mappa DS è memorizzata nella posizione data all'interno di una lista DS. Se la posizione data contiene un ID di mappa DS, allora la funzione restituirà true altrimenti restituirà false.

Nota che questo rileverà solo le mappe che sono state marcate manualmente usando la funzione ds_list_mark_as_map() .

Sintassi:

ds_list_is_map(id, pos);

ArgomentoTipo Descrizione
id The id of the list to check.
pos The position within the list to check.

Ritorna:

Esempio:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_map(ships, i)
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Il codice di cui sopra loops attraverso una lista DS e controlla se una qualsiasi delle voci contiene ID di mappa. Se lo fanno, allora queste mappe vengono distrutte, e poi la lista principale viene distrutta dopo che loop ha finito.
