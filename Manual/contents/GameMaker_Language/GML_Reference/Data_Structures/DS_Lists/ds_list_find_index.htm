ds_list_find_index

Con questa funzione è possibile verificare la presenza di un valore nell'elenco dato e restituire la posizione di tale valore all'interno dell'elenco. Si noti che se nell'elenco ci sono più voci con lo stesso valore, può essere restituita la posizione di una qualsiasi di esse e che se il valore non esiste, viene restituito -1.

Sintassi:

ds_list_find_index(id, val);

Argomento Tipo Descrizione
id ID elenco DS The id of the list to use.
val Variabile The value to find.

Restituzione:

Real

Esempio:

pos = ds_list_find_index(list, "Player1");

Il codice precedente controlla l'elenco indicizzato nella variabile "list" per il valore "Player1" e memorizza la posizione restituita nella variabile "pos".
