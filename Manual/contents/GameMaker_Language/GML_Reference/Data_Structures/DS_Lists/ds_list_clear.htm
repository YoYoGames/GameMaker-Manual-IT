ds_list_clear

Con questa funzione potete cancellare tutti i dati dalla struttura dati della lista data. Questo NON distrugge la struttura dei dati (per quello dovreste usare ds_list_destroy()) cancella solo tutti i dati da essa e rende la lista vuota (dimensione zero). Notate che la cancellazione di una lista de-referenzierà qualsiasi struttura dati memorizzata in essa, causando una perdita di memoria, quindi dovreste passare attraverso la lista e distruggere tutte le voci della struttura dati manualmente prima di cancellarla per evitare questo. L'unica volta che questo non è necessario è quando hai contrassegnato qualsiasi elemento della lista come un'altra lista DS o come una mappa DS, nel qual caso questi elementi saranno distrutti (non cancellati!) e la loro memoria ripulita automaticamente quando il genitore viene cancellato.

Sintassi:

ds_list_clear(id);

ArgomentoTipo Descrizione
id The id of the data structure to clear.

Ritorna:

Esempio:

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella la lista DS indicizzata nella variabile "command_list", imposta un allarme e resetta la variabile a 0.

Indietro: Elenchi DS
Il prossimo: ds_list_empty
