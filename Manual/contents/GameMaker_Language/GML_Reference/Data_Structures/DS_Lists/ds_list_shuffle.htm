ds_list_shuffle

Con questa funzione potete mescolare una lista, che riordinerà tutti i valori dei componenti in posizioni casuali da quelle in cui sono stati originariamente aggiunti alla lista.

NOTA: Questa funzione mischierà gli elementi della lista nelle stesse posizioni ogni volta che il gioco viene eseguito di nuovo a causa del fatto che GameMaker genera lo stesso seme casuale iniziale ogni volta per rendere debugging codice un compito molto più facile. Per evitare questo comportamento usate randomise() all'inizio del vostro gioco. Questo è vero solo quando si testa e debugging il gioco, poiché il pacchetto eseguibile finale non mostrerà questo comportamento e sarà casuale ad ogni esecuzione.

Sintassi:

ds_list_shuffle(id);

ArgomentoTipo Descrizione
id The id of the list to shuffle.

Ritorna:

Esempio:

if restart
{
    ds_list_shuffle(card_list);
}

Il codice qui sopra rimescolerà la lista indicizzata nella variabile "card_list" se la variabile "restart" è segnalata come true.
