ds_list_create

Questa funzione creerà una nuova struttura dati della lista e restituirà il valore dell'indice. Questo valore dovrebbe essere memorizzato in una variabile e utilizzato in tutte le ulteriori chiamate di funzioni relative alla lista. Notate che se provate ad accedere ad un valore nella lista prima che sia stata popolata - cioè se la lista è "vuota" - allora il valore di ritorno potrebbe essere 0 poiché internamente le prime voci della lista sono impostate al momento della creazione per minimizzare i problemi di prestazioni quando si aggiungono inizialmente elementi alla lista (sebbene la ds_list_size() restituirà comunque 0 su una lista appena creata). Se ci si vuole assicurare che la lista sia "veramente" vuota alla creazione, allora si dovrebbe chiamare ds_list_clear() dopo aver creato la lista, il che significa che qualsiasi valore restituito per gli slot della lista non riempiti sarà undefined.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere usato da una nuova struttura creata in seguito.

Sintassi:

ds_list_create();

Ritorna:

DS List ID

Esempio:

list = ds_list_create();

Questo creerà una nuova lista e assegnerà il suo indice id alla variabile di istanza "list".
