ds_list_find_value

Con questa funzione potete controllare la posizione della lista data e verrà restituito il valore contenuto nella lista per quella posizione. Notate che se date una posizione che è al di fuori della dimensione della lista data (ad esempio: posizione 11 in una lista di 10 valori) allora la funzione potrebbe restituire undefined o 0. Questo perché quando create la lista, internamente le prime voci della lista sono impostate a 0 per minimizzare i problemi di prestazioni quando inizialmente si aggiungono elementi alla lista (sebbene la ds_list_size() restituirà comunque 0 su una lista appena creata). Se ci si vuole assicurare che la lista sia "veramente" vuota alla creazione, allora si dovrebbe chiamare ds_list_clear() dopo aver creato la lista, il che significherà che qualsiasi valore restituito per gli slot della lista non riempiti sarà undefined, che si può poi controllare costantemente usando la funzione is_undefined() funzione.

Sintassi:

ds_list_find_value(id, pos);

Argomento Tipo Descrizione
id ID elenco DS The id of the list to use.
pos Real The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

Ritorna:

Variable or undefined

Esempio:

val = ds_list_find_value(list, ds_list_size(list) - 1);

Il codice precedente controlla la lista indicizzata nella variabile "list" all'ultima posizione della lista e memorizza il valore restituito nella variabile "val".
