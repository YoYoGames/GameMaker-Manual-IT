ds_list_set

Questa funzione può essere usata per impostare una voce della lista aggiunta in precedenza. Tu dai l'ID della lista (come restituito quando hai creato la lista) e la posizione all'interno della lista da impostare così come il valore a cui impostarla. Nota che se la voce da impostare è al di fuori dei limiti della lista (ad esempio, imposti la voce 20 della lista ma la lista corrente contiene solo 10 voci) allora la lista sarà riempita fino alla posizione data e ogni voce sarà impostata a 0. Questa funzione è la stessa che usa l'accessor DS list.

Sintassi:

ds_list_set(id, pos, val);

Argomento Tipo Descrizione
id ID elenco DS The id of the list to add to.
pos Variabile The position within the list to set.
[val2, ... max_val] Variabile The value to set in the list.

Ritorna:

N/A

Esempio:

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

Il codice qui sopra aggiungerà il valore memorizzato nella variabile "score" nella lista indicizzata nella variabile "sc_list".

Indietro: Elenchi DS
