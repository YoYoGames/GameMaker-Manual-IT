ds_list_add

Questa funzione può essere usata per aggiungere un nuovo valore (reale o string) alla lista, che verrà aggiunto alla fine. La funzione può prendere ulteriori argomenti opzionali (quanti ne volete), permettendovi di aggiungere più valori consecutivamente alla lista in una sola chiamata.

Sintassi:

ds_list_add(id, val1 [, val2, ... max_val]);

Argomento Tipo Descrizione
id ID elenco DS The id of the list to add to.
val Variabile The value to add to the list.
[val2, ... max_val] Variabile Optional values to be added to the list.

Ritorna:

N/A

Esempio:

ds_list_add(sc_list, score);

Il codice qui sopra aggiungerà il valore memorizzato nella variabile "score" nella lista indicizzata nella variabile "sc_list".

Indietro: Elenchi DS
Il prossimo: ds_list_set
