ds_list_copy

Con questa funzione potete copiare il contenuto di una lista in un'altra. Entrambe le liste devono essere state create in precedenza e se la lista in cui si sta copiando ha già delle informazioni al suo interno, questa lista verrà cancellata per prima. Il risultato finale sono due liste indipendenti che contengono le stesse informazioni.

Sintassi:

ds_list_copy( id, source );

ArgomentoTipo Descrizione
id The id of the list being copied to.
source The id of the list to be copied from.

Ritorna:

Esempio:

if !ds_list_empty(main_list)
{
    old_list = ds_list_create();
    ds_list_copy(old_list, main_list);
    ds_list_clear(main_list);
}

Il codice qui sopra controlla una lista DS per vedere se è vuota. Se non è vuota viene copiata in un'altra lista DS (che è stata creata in precedenza) e poi la lista originale viene cancellata.

Indietro: Elenchi DS
