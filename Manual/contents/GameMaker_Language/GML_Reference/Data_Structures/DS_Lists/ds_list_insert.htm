ds_list_insert

Questa funzione aggiungerà il valore dato nella lista nella posizione data. Se la lista contiene più valori dopo la posizione data, la loro posizione sarà spostata in alto di uno per fare room rendendo la lista più grande di uno.

Sintassi:

ds_list_insert(id, pos, val);

Argomento Tipo Descrizione
id ID elenco DS The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variabile The value to add to the list.

Ritorna:

N/A

Esempio:

ds_list_insert(list, 9, score);

Il codice qui sopra aggiungerà il valore memorizzato nella variabile "score" nella decima posizione della lista indicizzata dalla variabile "list" (le liste sono contate da 0, quindi una lista di dieci valori è numerata da 0 a 9).

Indietro: Elenchi DS
