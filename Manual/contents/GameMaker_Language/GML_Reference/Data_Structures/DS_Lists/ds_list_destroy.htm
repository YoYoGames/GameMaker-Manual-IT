ds_list_destroy

Questa funzione rimuoverà la data-struttura della lista data dalla memoria, liberando le risorse che stava usando e rimuovendo tutti i valori che conteneva. Questa funzione dovrebbe sempre essere usata quando avete finito di usare la ds_list per prevenire perdite di memoria che possono rallentare e mandare in crash il vostro gioco. Notate che distruggere una lista de-referenzia qualsiasi struttura di dati memorizzata in essa dando una perdita di memoria, quindi dovreste passare attraverso la lista e distruggere tutte le voci della struttura di dati manualmente prima di distruggerla per prevenire questo. L'unica volta che questo non è necessario è quando avete contrassegnato qualsiasi elemento della lista come un'altra lista DS o come una mappa DS, nel qual caso questi elementi saranno distrutti e la loro memoria pulita automaticamente.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere utilizzato da una nuova struttura creata in seguito, quindi si consiglia di impostare sempre la variabile che conteneva l'indice DS a -1 dopo la distruzione.

Sintassi:

ds_list_destroy(id);

ArgomentoTipo Descrizione
id The id of the data structure to remove.

Ritorna:

Esempio:

if (lives == 0)
{
    ds_list_destroy(AI_list);
    AI_list = -1;
    room_goto(rm_Menu);
}

Il codice qui sopra controllerà il valore della variabile globale incorporata "lives" e se è 0, distrugge la lista DS indicizzata nella variabile "AI_list" e poi cambia rooms.
