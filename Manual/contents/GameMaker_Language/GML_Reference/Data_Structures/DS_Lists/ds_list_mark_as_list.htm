ds_list_mark_as_list

Questa funzione "segnerà" (o "flag") una data posizione all'interno di una lista DS precedentemente creata come contenente un'altra lista DS. Questa funzionalità è necessaria quando si codifica JSON strings (che si può creare usando json_encode()), ma può anche essere utile quando si annidano le strutture di dati, poiché gli elementi marcati in questo modo saranno automaticamente smaltiti (distrutti) quando la lista DS madre viene distrutta. Questo significa che non devi passare manualmente attraverso il contenuto della lista e distruggere le strutture dati marcate individualmente prima di distruggere la lista "madre". Tuttavia, se si elimina la posizione dell'elenco individualmente, l'elenco DS che conteneva non verrà smaltito e si dovrebbe chiamare la funzione appropriata di distruzione dell'elenco DS prima di eliminare l'elemento dalla posizione dell'elenco padre. Nota anche che se chiami la funzione ds_list_clear() su una lista, qualsiasi elemento contrassegnato come lista verrà distrutto anch'esso quando la lista verrà cancellata.

Vale anche la pena notare che se la posizione che è stata marcata cambia all'interno della lista, il "marchio" si sposterà con essa, quindi se avete segnato la posizione 3 nella lista (per esempio) e poi inserite altri 2 elementi sotto di essa in modo che si sposti fino alla posizione 5, sarà ancora marcata come lista.

Sintassi:

ds_list_mark_as_list(id, pos);

ArgomentoTipo Descrizione
id The id of the list to mark.
pos The position within the list to mark.

Ritorna:

DS List ID or -1 (if it fails)

Esempio:

var j_list = ds_list_create();
var sub_list = ds_list_create();
ds_list_add(sub_list, health);
ds_list_add(sub_list, lives);
ds_list_add(sub_list, score);
ds_list_add(j_list, sub_list);
ds_list_mark_as_list(j_list, 0);

Il codice qui sopra crea due liste DS, poi ne popola una con vari valori dalle variabili globali. Questa lista viene poi aggiunta alla seconda lista, e la posizione "marcata" come tale in modo che possa essere codificata correttamente in seguito.
