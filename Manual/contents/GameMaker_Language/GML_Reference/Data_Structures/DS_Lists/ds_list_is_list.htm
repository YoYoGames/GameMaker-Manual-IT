ds_list_is_list

Con questa funzione potete controllare se un'altra lista DS è memorizzata nella posizione data all'interno di una lista DS. Se la posizione data contiene un ID di lista DS, allora la funzione restituirà true altrimenti restituirà false.

Si noti che questo rileverà solo le liste che sono state marcate manualmente usando la funzione ds_list_mark_as_list() .

Sintassi:

ds_list_is_list(id, pos);

ArgomentoTipo Descrizione
id The id of the list to check.
pos The position within the list to check.

Ritorna:

Esempio:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Il codice di cui sopra loops attraverso una lista DS e controlla se una qualsiasi delle voci contiene altri ID di liste. Se lo fanno, allora queste liste vengono distrutte, e poi la lista principale viene distrutta dopo che loop ha finito.
