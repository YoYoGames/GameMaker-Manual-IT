ds_map_secure_load

Questa funzione caricherà una mappa DS precedentemente salvata dal file dato. Questo file deve essere stato precedentemente creato utilizzando la funzione ds_map_secure_save() e, una volta caricato, la funzione restituirà l'indice della mappa DS che è stata creata dai dati caricati. Questo indice di mappa DS deve essere memorizzato in una variabile e utilizzato per tutte le ulteriori chiamate di funzione a questa mappa. Si noti che se la mappa DS caricata è stata salvata con un array come uno dei valori chiave, questo array sarà stato convertito in una lista DS al momento del caricamento.

IMPORTANTE! Una delle caratteristiche di un file salvato in modo sicuro è che è bloccato al dispositivo su cui è stato creato, quindi non è possibile caricare un file salvato su un dispositivo in un progetto in esecuzione su un altro dispositivo.

Sintassi:

ds_map_secure_load(filename);

ArgomentoTipo Descrizione
filename The name of the file to load the map data from.

Ritorna:

DS Map ID

Esempio:

p_map = ds_map_secure_load("p_data.dat");

Il codice qui sopra caricherà una mappa DS salvata in modo sicuro e memorizzerà il suo valore di indice in una variabile per uso futuro.
