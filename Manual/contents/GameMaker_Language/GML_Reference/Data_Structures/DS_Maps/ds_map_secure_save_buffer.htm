ds_map_secure_save_buffer

Questa funzione salverà una mappa DS creata in precedenza su un buffer. Voi fornite il valore ID della mappa DS (come restituito dalla funzione ds_map_create()) e l'ID del sito buffer su cui scrivere (come restituito dalla funzione buffer_create()). Si noti che se la mappa DS da salvare contiene un array, questo verrà convertito in una lista DS al momento del salvataggio.

IMPORTANTE! Il file di mappa DS salvato in modo sicuro può essere caricato solo sul dispositivo che lo ha creato, e se si cerca di caricare un file che è stato salvato su un dispositivo diverso, allora non funzionerà.

NOTA: Questa funzione non è supportata su HTML5 e UWP.

Sintassi:

ds_map_secure_save_buffer(filename);

Argomento Tipo Descrizione
id ID della mappa DS The DS map ID value.
buffer ID del buffer The buffer to save to.

Ritorna:

N/A

Esempio:

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

Il codice qui sopra creerà un buffer e una mappa DS, poi popolerà la mappa con alcuni valori e la scriverà su buffer prima di cancellare la mappa.
