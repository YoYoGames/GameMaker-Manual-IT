ds_map_empty

Questa funzione restituisce semplicemente false se la mappa DS specificata (creata in precedenza) contiene delle coppie chiave/valore, o true se non ne contiene.

Sintassi:

ds_map_empty(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if ds_map_empty(inventory)
{
    weight = 0;
}

Il codice qui sopra controlla se la mappa DS indicizzata nella variabile "inventory" ha delle coppie chiave/valore e se non ne ha, imposta la variabile "weight" a 0.

Indietro: Mappe DS
Il prossimo: ds_map_size
