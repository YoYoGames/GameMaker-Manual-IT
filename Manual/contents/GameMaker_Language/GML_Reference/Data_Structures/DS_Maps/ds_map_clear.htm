ds_map_clear

Questa funzione cancellerà la mappa DS (precedentemente creata) da tutte le coppie chiave/valore, ma non cancellerà la mappa DS stessa dalla memoria. Per questo dovreste usare la funzione ds_map_destroy(). Notate che la cancellazione di una mappa de-referenzierà qualsiasi struttura dati memorizzata nella mappa, causando una perdita di memoria, quindi dovreste passare attraverso di essa e distruggere manualmente tutti gli elementi della struttura dati prima di cancellare la mappa per evitare questo. L'unica volta che questo non è necessario è quando hai contrassegnato qualsiasi elemento nella mappa come una lista DS o come un'altra mappa DS, nel qual caso questi elementi saranno distrutti (non cancellati!) e la loro memoria ripulita automaticamente quando la mappa madre viene cancellata.

Sintassi:

ds_map_clear(id);

ArgomentoTipo Descrizione
id The id of the data structure to clear.

Ritorna:

Esempio:

if global.new_game
{
    ds_map_clear(inventory);
}

La funzione di cui sopra controllerà una variabile globale per vedere se è vera e, se lo è, allora la mappa DS indicizzata nella variabile di istanza "inventory" sarà cancellata da tutte le coppie chiave/valore (ma non viene distrutta).
