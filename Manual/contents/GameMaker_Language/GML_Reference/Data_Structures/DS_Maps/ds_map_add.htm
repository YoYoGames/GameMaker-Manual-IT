ds_map_add

Questa funzione dovrebbe essere usata per aggiungere insiemi di coppie chiave/valore nella mappa DS specificata. Potete controllare questa funzione per vedere se ha avuto successo o meno (restituirà true in caso di successo o false altrimenti), poiché potrebbe fallire se esiste già la stessa chiave nella mappa DS o se si specifica una mappa DS inesistente come ID della mappa da aggiungere. Le chiavi e i valori che fornite possono essere costituiti da qualsiasi combinazione di data types, quindi tutti i seguenti - e più - sono accettabili (anche se, in pratica, si userebbe più comunemente un string per la chiave):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

Si può anche aggiungere a una mappa usando l'accessor "?", come mostrato qui sotto:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

NOTA: A differenza di altre strutture dati in GameMaker questa chiave non andrà all'inizio (né alla fine) della mappa DS, ma piuttosto andrà semplicemente nella mappa DS da qualche parte.

Sintassi:

ds_map_add(id, key, val);

ArgomentoTipo Descrizione
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

Ritorna:

Esempio:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Questo creerà una nuova mappa, assegnando il suo id alla variabile "inventario". Aggiunge poi due nuove chiavi alla mappa, "hp potion" e "gold", e imposta i loro valori iniziali come 1 e 100.

Indietro: Mappe DS
Il prossimo: ds_map_clear
