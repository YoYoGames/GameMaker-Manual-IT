ds_map_secure_load_buffer

Questa funzione caricherà una mappa DS sicura salvata da un buffer. Dovete aver precedentemente caricato il buffer in memoria (usando buffer_load()) e quindi passarlo in questa funzione restituirà una mappa DS popolata con il contenuto di buffer. Si noti che buffer deve essere stato creato usando la funzione ds_map_secure_save_buffer() perché questo funzioni correttamente, e si noti anche che se la mappa DS caricata contiene un array, questo sarà convertito in una lista DS al suo posto al momento del caricamento.

IMPORTANTE! Una delle caratteristiche di un file salvato in modo sicuro è che è bloccato al dispositivo su cui è stato creato, quindi non è possibile caricare un file salvato su un dispositivo in un progetto in esecuzione su un altro dispositivo.

NOTA: Questa funzione non è supportata su HTML5 e UWP.

Sintassi:

ds_map_secure_load_buffer(buffer);

Argomento Tipo Descrizione
buffer ID del buffer The buffer ID of the buffer to load the map data from.

Ritorna:

DS Map ID

Esempio:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Il codice qui sopra caricherà una mappa DS salvata in modo sicuro da un buffer e memorizzerà il suo valore di indice in una variabile per uso futuro.
