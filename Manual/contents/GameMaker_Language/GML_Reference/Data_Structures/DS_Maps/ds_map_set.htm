ds_map_set

Con questa funzione potete impostare il valore di una chiave all'interno di una data mappa DS. Si fornisce il valore ID della mappa DS (come restituito dalla funzione ds_map_create()), poi date la chiave che volete impostare e il valore a cui impostarla. Le chiavi possono essere interi o strings, e se la chiave data non esiste allora verrà creata per voi e impostata al valore. Questa funzione è la stessa che usa l'accessor DS map per impostare/creare una coppia chiave/valore della mappa. La funzione non restituisce nulla, quindi se avete bisogno di controllare se il valore della chiave è stato sostituito o se è stata creata una nuova chiave, allora dovreste usare la funzione ds_map_replace().

Sintassi:

ds_map_set(id, key, value)

ArgomentoTipo Descrizione
id The id of the map to use.
key The key to set.
value The value to set the key to.

Ritorna:

Esempio:

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

Il codice qui sopra controllerà se la chiave data esiste e se non esiste, allora viene creata e impostata.
