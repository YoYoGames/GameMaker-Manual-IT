ds_map_read

Questa funzione prenderà un string che è stato precedentemente creato dalla funzione ds_map_write() e lo leggerà in una mappa DS creata in precedenza. Se la mappa in cui viene letto string contiene delle coppie chiave/valore, queste verranno cancellate prima che la mappa salvata venga ricostruita.

Notate che se il DS specificato string è stato scritto da GameMaker: Studio 1.2.x runtime (o più vecchio), dovreste specificare l'argomento opzionale "legacy", impostandolo su true poiché il formato string è cambiato dopo.

Sintassi:

ds_map_read(id, str, [legacy]);

Argomento Tipo Descrizione
id ID della mappa DS The id of the data structure to read the string into
str Stringa The string to read
legacy Booleano OPTIONAL Può essere true o false o omesso completamente.

Ritorna:

N/A

Esempio:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

Il codice qui sopra crea una nuova mappa DS e memorizza il suo indice id nella variabile "inventory". Poi apre un file ini e legge un string da quel file nella variabile temporanea "t_string". Infine, controlla per assicurarsi che il string sia valido (non il valore ini predefinito di "") e se lo è legge il string nella mappa DS appena creata prima di chiudere nuovamente l'ini.
