ds_map_values_to_array

Con questa funzione è possibile recuperare tutti i valori che una mappa DS contiene per ogni chiave della mappa. Si fornisce l'ID della mappa DS da cui ottenere le chiavi (come restituito da ds_map_create()) e la funzione restituirà un array in cui ogni voce dell'array è un valore della mappa DS (ad esempio, se la mappa ha 10 chiavi, verrà restituito un array di 10 voci con ogni voce corrispondente a un valore per una chiave). La funzione ha un secondo argomento opzionale, in cui è possibile fornire un array creato dall'utente; in questo caso, i dati dei valori delle chiavi della mappa verranno aggiunti ai dati esistenti nell'array. Si noti che la funzione modificherà direttamente l'array fornito, ma restituirà anche un riferimento ad esso (o un riferimento a un nuovo array se non viene fornito).

Sintassi:

ds_map_values_to_array(id, [array])

Argomento Tipo Descrizione
id ID mappa DS The id of the map to use.
[array] Array OPTIONAL L'array a cui aggiungere i valori della mappa DS.

Restituzione:

Array

Esempio:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

Questo metodo recupera tutti i valori da una mappa DS per ogni chiave della mappa e li memorizza in una variabile locale come array. Quindi, loops sfoglia l'array e aggiunge ciascuno dei valori a una variabile locale, che viene poi utilizzata per visualizzare il totale di tutti i valori.
