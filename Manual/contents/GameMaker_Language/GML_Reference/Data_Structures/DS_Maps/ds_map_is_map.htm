ds_map_is_map

Con questa funzione potete controllare se una mappa DS è memorizzata nella chiave di mappa data. Se la chiave data contiene un ID di mappa DS, allora la funzione restituirà true altrimenti restituirà false.

Si noti che questo rileverà solo le mappe che sono state aggiunte usando la funzione ds_map_add_map() .

Sintassi:

ds_map_is_map(id, key)

ArgomentoTipo Descrizione
id The id of the map to use.
key The key to replace.

Ritorna:

Esempio:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Il codice sopra loops attraverso una mappa DS e controlla se una qualsiasi delle chiavi all'interno di essa è per altre mappe DS. Se lo sono, allora la mappa DS memorizzata viene distrutta, e alla fine di loop anche la mappa DS principale viene distrutta.
