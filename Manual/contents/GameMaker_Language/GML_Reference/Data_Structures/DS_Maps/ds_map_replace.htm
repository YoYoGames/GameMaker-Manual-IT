ds_map_replace

Con questa funzione potete cambiare il valore della chiave data all'interno di una mappa DS. Si fornisce l'indice della mappa (come restituito dalla funzione ds_map_create()) e poi la chiave da sostituire - o un string o un intero - e il valore con cui sostituire il valore della chiave. Se la chiave data non esiste allora verrà creata per voi, e se esiste allora il valore corrente verrà sostituito con il nuovo valore. La funzione restituirà true se la chiave esiste e il valore viene sostituito, e false se la chiave non esiste e una nuova chiave è stata creata con il valore.

Sintassi:

ds_map_replace( id, key, val );

ArgomentoTipo Descrizione
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Ritorna:

Esempio:

ds_map_replace(inventory, "torso", 55);

Il codice qui sopra cerca la mappa DS per la chiave "torso" e quando la trova (o viene creata se non esiste) il valore corrente viene sostituito con quello specificato.
