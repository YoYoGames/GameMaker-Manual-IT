ds_map_add_list

Con questa funzione potete assegnare una lista DS (creata in precedenza) a una chiave all'interno della mappa DS data. Questa funzione è progettata per la creazione di mappe compatibili con JSON che verrebbero poi codificate usando json_encode() e dovrebbe essere usata solo insieme a questa funzionalità. Se una mappa DS ha una lista aggiunta in questo modo, distruggere la mappa padre distruggerà anche le liste contenute e libererà la loro memoria, e chiamando ds_map_clear() sulla mappa genitore distruggerà e pulirà anche qualsiasi lista contrassegnata.

Sintassi:

ds_map_add_list(id, key, value)

Argomento Tipo Descrizione
id ID della mappa DS The id of the map to use.
key Stringa The key for the added list.
value ID elenco DS The id of the list to add.

Ritorna:

N/A

Esempio:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Il codice di cui sopra creerà una lista e la popolerà con i vari valori delle variabili globali. Questa lista viene poi "annidata" all'interno di una mappa DS, e la mappa viene poi codificata in un JSON string, prima che la mappa venga distrutta, rimuovendo dalla memoria la mappa e tutte le liste che contiene.
