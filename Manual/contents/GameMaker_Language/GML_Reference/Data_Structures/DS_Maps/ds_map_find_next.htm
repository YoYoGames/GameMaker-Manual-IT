ds_map_find_next

Questa funzione restituisce la prossima chiave memorizzata nella mappa DS dopo quella specificata nella funzione. Questo può essere utile se dovete iterare attraverso la mappa DS alla ricerca di qualcosa, ma dovrebbe essere evitato se possibile perché può essere lento. Se tale chiave non esiste, la funzione restituisce undefined. Dovreste sempre controllare questo usando la funzione is_undefined() funzione.

Sintassi:

ds_map_find_next(id, key);

ArgomentoTipo Descrizione
id The id of the map to use.
key The key to find the next one to.

Ritorna:

Variable or undefined

Esempio:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

Il codice qui sopra crea alcune variabili temporanee e poi ottiene la dimensione della mappa DS e trova la prima chiave memorizzata dal computer nella mappa. Poi usa un for loop per iterare attraverso la mappa DS alla ricerca del valore chiave "oro". Se lo trova, estrae il loop.
