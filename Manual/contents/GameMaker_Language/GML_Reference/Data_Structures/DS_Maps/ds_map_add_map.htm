ds_map_add_map

Con questa funzione potete assegnare una mappa DS (creata in precedenza) a una chiave all'interno della mappa DS data. Questa funzione è progettata per la creazione di mappe compatibili con JSON che verrebbero poi codificate usando json_encode() e dovrebbe essere usata solo insieme a questa funzionalità. Se una mappa DS ha un'altra mappa aggiunta in questo modo, allora distruggere la mappa padre distruggerà anche le mappe contenute e libererà la loro memoria, e chiamando ds_map_clear() sulla mappa padre distruggerà e pulirà anche qualsiasi mappa contrassegnata.

Sintassi:

ds_map_add_map(id, key, value)

Argomento Tipo Descrizione
id ID della mappa DS The id of the map to use.
key Stringa The key for the added map.
value ID della mappa DS The id of the map to add.

Ritorna:

N/A

Esempio:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Il codice qui sopra creerà due mappe DS, e poi ne popolerà una con una lista e un array prima di aggiungerla nella seconda, che viene poi codificata in un JSON string. La mappa viene poi distrutta per rimuoverla, e qualsiasi altra mappa o lista che contiene, dalla memoria.
