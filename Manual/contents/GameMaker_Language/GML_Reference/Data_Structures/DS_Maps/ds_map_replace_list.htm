ds_map_replace_list

Con questa funzione potete sostituire una lista DS che è stata memorizzata nella "chiave" data con un'altra lista che è stata creata in precedenza. Questa funzione è progettata per la creazione di mappe compatibili con JSON che verrebbero poi codificate utilizzando json_encode() e dovrebbe essere usata solo in combinazione con questa funzionalità.

Sintassi:

ds_map_replace_list(id, key, value)

Argomento Tipo Descrizione
id ID della mappa DS The id of the map to use.
key Stringa The key to replace.
value ID elenco DS The id of the ds_list to use to replace the one previously stored in the given key.

Ritorna:

N/A

Esempio:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

Il codice qui sopra creerà un DS List e lo popolerà con i valori di varie variabili globali prima di sostituire una lista precedentemente memorizzata in DS Map "j_map".

Indietro: Mappe DS
Il prossimo: ds_map_replace_map
