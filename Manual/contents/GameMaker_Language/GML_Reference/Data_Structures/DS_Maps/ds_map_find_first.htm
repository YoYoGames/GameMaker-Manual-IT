ds_map_find_first

Questa funzione restituisce la prima chiave memorizzata nella mappa DS. Questa non è la prima chiave nell'ordine in cui le hai aggiunte! Le mappe DS non sono memorizzate in forma lineare, per questo usate DS list, quindi tutto ciò che fa è trovare la prima chiave come memorizzata dal computer. Questo può essere utile se devi iterare attraverso la mappa DS alla ricerca di qualcosa, ma dovrebbe essere evitato se possibile perché può essere lento.

Si noti che questa funzione restituisce undefined se la mappa DS data è vuota.

Sintassi:

ds_map_find_first(id);

ArgomentoTipo Descrizione
id The id of the map to use.

Ritorna:

Variable or undefined

Esempio:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

Il codice qui sopra crea alcune variabili temporanee e poi ottiene la dimensione della mappa DS e trova la prima chiave memorizzata dal computer nella mappa. Poi usa un for loop per iterare attraverso la mappa DS alla ricerca del valore chiave "oro". Se lo trova, estrae il loop.
