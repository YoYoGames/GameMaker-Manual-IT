ds_map_copy

Potete usare questa funzione per copiare il contenuto di una mappa in un'altra che avete precedentemente creato usando ds_map_create(). Se la mappa DS su cui si sta copiando non è vuota, allora questa funzione la cancellerà prima di copiare. La mappa DS originale rimane invariata da questo processo.

Sintassi:

ds_map_copy(id, source);

ArgomentoTipo Descrizione
id The id of the map you are copying to
source The id of the map you are copying from

Ritorna:

Esempio:

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

Il codice qui sopra creerà una nuova mappa e la assegnerà alla variabile "inventory_2". Poi copierà il contenuto della mappa DS indicizzata nella variabile "inventory_1" in questa nuova mappa.

Indietro: Mappe DS
Il prossimo: ds_map_replace
