ds_map_delete

Con questa funzione potete rimuovere qualsiasi chiave data (e il suo valore corrispondente) dalla mappa DS data, creata in precedenza.

Sintassi:

ds_map_delete(id, key);

Argomento Tipo Descrizione
id ID della mappa DS The id of the map to change.
key Stringa The key to delete (along with its associated value).

Ritorna:

N/A

Esempio:

ds_map_delete(inventory, "shield");

Il codice qui sopra cancellerà la chiave "shield" (e il valore a cui è abbinata) dalla mappa DS (inventario).

Indietro: Mappe DS
Il prossimo: ds_map_empty
