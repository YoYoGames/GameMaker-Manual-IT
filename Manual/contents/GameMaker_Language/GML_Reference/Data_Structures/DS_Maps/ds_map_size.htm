ds_map_size

Con questa funzione potete trovare quante coppie chiave/valore contiene la mappa DS (creata in precedenza).

Sintassi:

ds_map_size(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

Il codice qui sopra controllerà la dimensione della mappa DS (cioè: numero di coppie chiave/valore) e se è maggiore di 49 imposta la variabile "full" su true.
