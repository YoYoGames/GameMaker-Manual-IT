ds_map_find_value

Con questa funzione potete ottenere il valore da una chiave specificata. I valori di input della funzione sono la mappa DS (precedentemente creata) da usare e la chiave da controllare.

NOTA: se non esiste una tale chiave, la funzione restituirà undefined. Dovreste sempre controllare questo utilizzando la is_undefined() funzione

Sintassi:

ds_map_find_value(id, key)

ArgomentoTipo Descrizione
id The id of the map to use.
key The key to find.

Ritorna:

Variable or undefined

Esempio:

amount = ds_map_find_value(inventory, "food");

Oppure, usando l'accessor della mappa "?":

amount = inventory[? "food"];

Il codice qui sopra otterrà il valore della chiave "food" e lo memorizzerà nella variabile "amount".
