ds_map_is_list

Con questa funzione potete controllare se una lista DS è memorizzata nella chiave della mappa data. Se la chiave data contiene un ID di lista DS, allora la funzione restituirà true altrimenti restituirà false.

Si noti che questo rileverà solo le liste che sono state aggiunte usando la funzione ds_map_add_list() .

Sintassi:

ds_map_is_list(id, key)

ArgomentoTipo Descrizione
id The id of the map to use.
key The key to check.

Ritorna:

Esempio:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if ds_map_is_list(inventory, key)
    {
        ds_list_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Il codice sopra loops attraverso una mappa DS e controlla se una qualsiasi delle chiavi all'interno di essa è per una lista DS. Se lo sono, allora la lista DS viene distrutta, e alla fine di loop anche la mappa DS viene distrutta.

Indietro: Mappe DS
Prossimo: ds_map_is_map
