ds_map_replace_map

Con questa funzione potete sostituire un DS Map che è stato memorizzato nella "chiave" data con un'altra mappa che è stata creata in precedenza. Questa funzione è progettata per la creazione di mappe compatibili con JSON che verrebbero poi codificate utilizzando json_encode() e dovrebbe essere usata solo in combinazione con questa funzionalità.

Sintassi:

ds_map_replace_map(id, key, value)

Argomento Tipo Descrizione
id ID della mappa DS The id of the ds_map to use.
key Stringa The key to replace.
value ID della mappa DS The id of the ds_map to use to replace the one previously stored in the given key.

Ritorna:

N/A

Esempio:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Il codice qui sopra creerà un DS Map e lo popolerà con un array e un DS List prima di sostituire una mappa precedentemente memorizzata nel DS Map "j_map".
