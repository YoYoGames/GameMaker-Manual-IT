ds_map_exists

Questa funzione restituisce true se la chiave specificata esiste nella mappa DS (precedentemente creata), e false se non esiste.

Sintassi:

ds_map_exists(id, key);

Argomento Tipo Descrizione
id ID della mappa DS The id of the data structure to check
key Stringa The key to check for

Ritorna:

Boolean

Esempio:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

Il codice qui sopra controllerà la mappa DS indicizzata nella variabile "inventory" per la chiave "potions" e se non esiste la aggiungerà alla mappa.

Indietro: Mappe DS
