ds_map_keys_to_array

Con questa funzione potete recuperare tutte le chiavi che una mappa DS contiene. Si fornisce l'ID della mappa DS da cui ottenere le chiavi (come restituito da ds_map_create()) e la funzione restituirà un array dove ogni voce dell'array è una chiave della mappa DS. La funzione ha un secondo argomento opzionale in cui potete fornire un array che avete creato, nel qual caso i dati della chiave della mappa saranno aggiunti a qualsiasi dato esistente nell'array. Notate che la funzione modificherà direttamente l'array fornito, ma restituirà anche un riferimento ad esso (o un riferimento ad un nuovo array se non ne viene fornito nessuno).

Sintassi:

ds_map_keys_to_array(id, [array])

Argomento Tipo Descrizione
id ID della mappa DS The id of the map to use.
[array] Array OPTIONAL L'array a cui aggiungere le chiavi della mappa DS.

Ritorna:

Array

Esempio:

map_keys = ds_map_keys_to_array(inventory);

Il codice qui sopra recupera le chiavi di una mappa DS e poi le memorizza come un array per un uso futuro.
