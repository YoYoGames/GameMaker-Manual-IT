ds_map_secure_save

Questa funzione salverà il contenuto della mappa DS data in un file che è collegato al dispositivo su cui è stato creato (il che significa che non può essere letto se trasferito su qualsiasi altro dispositivo). Il file stesso può avere quasi qualsiasi extension (per esempio, *.dat, *.json, *.bin, ecc...) e sarà offuscato e memorizzato nella memoria locale sulla piattaforma di destinazione. È quindi possibile ricaricare la ds_map utilizzando la funzione ds_map_secure_load(). Si noti che se la mappa DS che viene salvata contiene un array, questo array sarà invece convertito in una lista DS al momento del salvataggio.

IMPORTANTE! Una delle caratteristiche di un file salvato in modo sicuro è che è bloccato al dispositivo su cui è stato creato, quindi non è possibile caricare un file salvato su un dispositivo in un progetto in esecuzione su un altro dispositivo.

Sintassi:

ds_map_secure_save(map, filename);

ArgomentoTipo Descrizione
map The id of the data structure to use
filename The name of the file to save the map to

Ritorna:

Esempio:

ds_map_secure_save(purchase_map, "p_data.dat");

Il codice qui sopra salverà la mappa DS indicizzata nella variabile "p_data" nel file dato per un successivo recupero.
