ds_map_find_last

Questa funzione restituisce l'ultima chiave memorizzata nella mappa DS. Questa non è l'ultima chiave nell'ordine in cui l'avete aggiunta! Le mappe DS non sono memorizzate in forma lineare, per questo usate le liste DS, quindi tutto ciò che fa è trovare l'ultima chiave come memorizzata dal computer. Questo può essere utile se dovete iterare attraverso la mappa DS alla ricerca di qualcosa, ma dovrebbe essere evitato se possibile perché può essere lento.

Si noti che questa funzione restituisce undefined se la mappa DS data è vuota.

Sintassi:

ds_map_find_last(id);

ArgomentoTipo Descrizione
id The id of the map to use.

Ritorna:

Variable or undefined

Esempio:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

Il codice qui sopra crea alcune variabili temporanee e poi ottiene la dimensione della mappa DS e trova l'ultima chiave memorizzata dal computer nella mappa. Poi usa un for loop per iterare a ritroso la mappa DS alla ricerca del valore chiave "oro". Se lo trova, fa uscire il loop.
