ds_map_write

Questa funzione trasformerà i dati della mappa DS della mappa specificata nel formato string che può poi essere scritto in un file *.ini o *.txt per una facile memorizzazione. Questo string può poi essere riletto successivamente in una nuova ds_map usando ds_map_read().

NOTA: Il string restituito non è un string leggibile dall'uomo, ma piuttosto un dump del contenuto della struttura dei dati.

Sintassi:

ds_map_write(id);

ArgomentoTipo Descrizione
id The id of the data structure to use

Ritorna:

Esempio:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Il codice qui sopra apre un file ini pronto per essere scritto. Poi usa ds_map_write() per generare un string che viene memorizzato nella variabile temporanea "t_string". Infine, scrive quel string nel file ini prima di chiuderlo.

Indietro: Mappe DS
Il prossimo: ds_map_destroy
