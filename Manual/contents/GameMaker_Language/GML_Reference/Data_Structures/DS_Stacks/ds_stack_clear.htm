ds_stack_clear

Con questa funzione potete cancellare tutti i dati dalla struttura dati dello stack dato. Questo NON distrugge la struttura dati (per quello dovreste usare ds_stack_destroy()) cancella solo tutti i dati da essa e restituisce uno stack vuoto.

Sintassi:

ds_stack_clear(id);

ArgomentoTipo Descrizione
id The id of the data structure to clear.

Ritorna:

Esempio:

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella lo stack DS indicizzato nella variabile "AI_stack", imposta un allarme e resetta la variabile a 0.

Indietro: ds_stack_create
Il prossimo: ds_stack_empty
