ds_stack_size

Questa funzione restituisce la "dimensione" della pila, cioè il numero di elementi che sono stati spinti su di essa.

Sintassi:

ds_stack_size(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if !ds_stack_empty(control_stack)
{
    num = ds_stack_size(control_stack);
}

Il codice qui sopra controlla uno stack DS per vedere se è vuoto o no, e se non lo è, ottiene il numero di elementi che contiene e memorizza il valore in una variabile.

Indietro: ds_stack_create
Il prossimo: ds_stack_copy
