ds_stack_push

Questa funzione spingerà (aggiungerà) un valore, che può essere di qualsiasi data type, in cima allo stack. La funzione può prendere altri 14 argomenti opzionali (per un totale di 15 possibili aggiunte), permettendovi di spingere più valori consecutivamente allo stack in una singola chiamata.

Sintassi:

ds_stack_push(id, val [, val2, ... val15]);

ArgomentoTipo Descrizione
id The id of the data structure to push onto.
val The value to push onto the stack.
[val2, ... val13] Optional values to be added to the stack.

Ritorna:

Esempio:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

Il codice qui sopra crea un nuovo stack DS e memorizza il suo indice nella variabile "move_stack". Poi spinge un certo numero di valori sullo stack per un uso futuro.

Indietro: ds_stack_create
Il prossimo: ds_stack_read
