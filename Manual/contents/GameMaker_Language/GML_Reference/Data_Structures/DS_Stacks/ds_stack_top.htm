ds_stack_top

Questa funzione leggerà solo il primo valore dello stack (quello che è "in cima"). Non farà il pop del valore, il che significa che può ancora essere letto in futuro da questa funzione o dal ds_stack_pop(). Se lo stack è vuoto, la funzione restituirà la costante undefined, altrimenti restituirà il valore reale o string contenuto nello stack.

Sintassi:

ds_stack_top(id);

ArgomentoTipo Descrizione
id The id of the data structure to read from.

Ritorna:

Variable (Data type value that is stored in the stack) or undefined

Esempio:

num = ds_stack_top(control_stack);

Il codice precedente leggerà il valore dallo stack indicizzato nella variabile "control_stack" e memorizzerà il valore di ritorno nella variabile "num".
