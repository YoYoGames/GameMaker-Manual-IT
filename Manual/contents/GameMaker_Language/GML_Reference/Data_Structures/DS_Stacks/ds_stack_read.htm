ds_stack_read

Con questa funzione si può ricreare uno stack DS salvato (uno che è stato precedentemente scritto come string usando ds_stack_write()). Dovete prima creare una nuova pila DS in cui leggere il string, e se la pila DS esiste già e ha informazioni memorizzate in essa, allora questa sarà cancellata prima della lettura. Questa funzione è di vitale importanza quando si creano meccanismi di salvataggio/caricamento per il gioco.

Notate che se il DS specificato string è stato scritto da GameMaker: Studio 1.2.x runtime (o più vecchio), dovreste specificare l'argomento opzionale "legacy", impostandolo su true poiché il formato string è cambiato dopo.

Sintassi:

ds_stack_read(id, str [, legacy]);

Argomento Tipo Descrizione
id ID della pila DS The id of the data structure to read into.
str Stringa The string to read from.
legacy Booleano OPTIONAL Può essere true o false o omesso completamente.

Ritorna:

N/A

Esempio:

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
    ds_stack_read(stack, str);
}
ini_close();

Il codice qui sopra crea uno stack e memorizza l'indice nella variabile "stack". Poi apre un file ini e legge un string da quello, controllando per assicurarsi che il string non venga prima restituito come vuoto. Questo string viene poi letto nello stack DS appena creato.

Indietro: ds_stack_create
Il prossimo: ds_stack_write
