ds_stack_write

Questa funzione restituisce un string che può poi essere memorizzato o trasferito in un'altra struttura di dati utilizzando la ds_stack_read() funzione.

NOTA: Il string restituito non è un string leggibile dall'uomo, ma piuttosto un dump del contenuto della struttura dei dati

Sintassi:

ds_stack_write(id);

ArgomentoTipo Descrizione
id The id of the data structure to write.

Ritorna:

Esempio:

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

Il codice qui sopra apre un file ini e poi scrive un string contenente le informazioni memorizzate nello stack DS indicizzato nella variabile "stack". Lo stack viene poi cancellato e il file ini chiuso.
