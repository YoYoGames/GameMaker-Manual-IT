ds_stack_copy

Questa funzione può essere usata per copiare il contenuto di uno stack in un altro. Notate che questo NON rimuove il contenuto della pila originale, né la distrugge. Quando si usa questa funzione lo stack in cui viene copiato deve essere stato creato in precedenza e se conteneva degli elementi prima della copia, questi saranno cancellati per primi (il che significa che queste informazioni saranno perse).

Sintassi:

ds_stack_copy(id, source);

Argomento Tipo Descrizione
id ID della pila DS The id of the new stack.
source ID della pila DS The original stack to copy from.

Ritorna:

N/A

Esempio:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
}

La funzione di cui sopra crea una nuova istanza e poi in quell'istanza crea un nuovo stack DS e copia il contenuto dello stack nell'istanza che esegue il blocco di codice, nello stack dell'istanza appena creata.

Indietro: ds_stack_create
Il prossimo: ds_stack_top
