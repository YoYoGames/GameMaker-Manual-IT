ds_grid_create

Con questa funzione potete creare una nuova struttura di dati DS grid della larghezza e dell'altezza delle celle specificate. Questa funzione restituisce un id che deve essere usato in tutte le altre funzioni che trattano questa griglia DS.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere usato da una nuova struttura creata in seguito.

Sintassi:

ds_grid_create(w, h);

ArgomentoTipo Descrizione
w The width of the grid to be created.
h The height of the grid to be created.

Ritorna:

DS Grid ID

Esempio:

mygrid = ds_grid_create(10, 10)

Questo crea una griglia alta 10 celle e larga 10 celle.

Indietro: Griglie DS
Il prossimo: ds_grid_destroy
