ds_grid_get_mean

Questa funzione può essere utilizzata per trovare il valore medio di tutte le celle che si trovano all'interno della regione definita di una griglia (tutti i valori delle celle vengono sommati e poi divisi per il numero totale di celle che compongono la regione), come mostrato nell'immagine sottostante:

Sintassi:

ds_grid_get_mean(index, x1, y1, x2, y2);

ArgomentoTipo Descrizione
index The index of the grid.
x1 The left cell column of the region.
y1 The top cell row of the region.
x2 The right cell column of the region.
y2 The bottom cell row of the region.

Ritorna:

Real or String

Esempio:

val = ds_grid_get_mean(grid, 0, 0, 5, 5)

Il codice precedente imposterà la variabile "val" al valore medio contenuto nella regione data della griglia DS indicizzata nella variabile "grid".

Indietro: Griglie DS
Il prossimo: ds_grid_get_min
