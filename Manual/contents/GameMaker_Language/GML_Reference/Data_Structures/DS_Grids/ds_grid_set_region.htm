ds_grid_set_region

Questa funzione può essere utilizzata per impostare una regione rettangolare di una data griglia su un valore specificato (che può essere un reale o un string) come illustrato dall'immagine qui sotto:

Sintassi:

ds_grid_set_region(index, x1, y1, x2, y2, val);

ArgomentoTipo Descrizione
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to set the region cells to.

Ritorna:

Esempio:

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

Questo imposterà tutte le celle nella regione della griglia indicizzata nella variabile "griglia" da (5,5) a (10,10) a 99.
