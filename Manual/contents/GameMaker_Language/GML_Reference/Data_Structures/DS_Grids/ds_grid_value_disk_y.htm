ds_grid_value_disk_y

Con questa funzione potete ottenere la coordinata y (all'interno della regione data a forma di disco della griglia) del valore che state cercando. Si dà l'indice della griglia DS (come restituito da ds_grid_create()) insieme alle posizioni x/y per la cella centrale del disco. Poi date il raggio (come valore intero) intorno alla cella centrale da cercare, prima di fornire il valore da cercare. Se il valore da cercare non esiste, allora la funzione restituirà -1.

Sintassi:

ds_grid_value_disk_y(index, xm, ym, r, val);

ArgomentoTipo Descrizione
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

Ritorna:

Esempio:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Il codice qui sopra controlla una griglia DS per un valore specifico all'interno di una regione del disco. se viene trovato, memorizza la posizione x e y del valore in due variabili per un uso successivo.
