ds_grid_destroy

Questa funzione rimuoverà la data-struttura della griglia data dalla memoria, liberando le risorse che stava usando e rimuovendo tutti i valori che conteneva. Questa funzione dovrebbe sempre essere usata quando avete finito di usare la griglia DS per evitare perdite di memoria che possono rallentare e mandare in crash il vostro gioco.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere utilizzato da una nuova struttura creata in seguito, quindi si consiglia di impostare sempre la variabile che conteneva l'indice DS a -1 dopo la distruzione.

Sintassi:

ds_grid_destroy(index);

ArgomentoTipo Descrizione
index This index of the grid to destroy.

Ritorna:

Esempio:

if (lives == 0)
{
    ds_grid_destroy(Wall_Grid);
    Wall_Grid = -1;
    room_goto(rm_Menu);
}

Il codice qui sopra controllerà il valore della variabile globale incorporata "lives" e se è 0, distrugge la griglia DS indicizzata nella variabile "Wall_Grid" e poi cambia rooms.
