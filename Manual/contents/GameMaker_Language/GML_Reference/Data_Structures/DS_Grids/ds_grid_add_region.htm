ds_grid_add_region

Questa funzione può essere utilizzata per aggiungere un dato valore (reale o string) a tutti i valori delle celle che si trovano all'interno dell'area definita di una griglia. Il valore da aggiungere deve essere dello stesso tipo di quello contenuto nelle celle della griglia, cioè: non si può aggiungere un string a un reale o viceversa, e per strings questo corrisponde alla concatenazione.

Sintassi:

ds_grid_add_region(index, x1, y1, x2, y2, val);

ArgomentoTipo Descrizione
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to add to the region cells.

Ritorna:

Esempio:

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

Questo aggiungerebbe "." a tutti i strings contenuti nelle celle all'interno della regione definita della griglia DS a cui fa riferimento la variabile "grid".
