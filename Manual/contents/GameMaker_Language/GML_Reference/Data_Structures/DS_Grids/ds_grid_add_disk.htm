ds_grid_add_disk

Questa funzione può essere utilizzata per aggiungere un dato valore (reale o string) a tutti i valori delle celle che si trovano all'interno dell'area disco definita di una griglia. Il valore da aggiungere deve essere dello stesso tipo di quello contenuto nelle celle della griglia, cioè: non si può aggiungere una stringa a un reale o viceversa, e per strings questo corrisponde alla concatenazione.

Sintassi:

ds_grid_add_disk(index, xm, ym, r, val);

ArgomentoTipo Descrizione
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to add to the cells within the disk.

Ritorna:

Esempio:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Questo aggiungerebbe 2 a tutti i valori contenuti nelle celle all'interno dell'area disco definita della griglia DS a cui fa riferimento la variabile "grid".

Indietro: Griglie DS
