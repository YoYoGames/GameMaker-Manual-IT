ds_grid_get

Questa funzione può essere usata per ottenere il valore da qualsiasi cella all'interno della griglia DS data. Se si passano coordinate di griglia non valide alla funzione, allora il valore restituito sarà undefined e un errore sarà mostrato nella finestra di output.

Sintassi:

ds_grid_get(index, x, y);

ArgomentoTipo Descrizione
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Ritorna:

Variable

Esempio:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Il codice qui sopra seleziona una cella casuale dalla griglia DS indicizzata nella variabile "grid" e memorizza il suo valore nella variabile "val".

Indietro: Griglie DS
