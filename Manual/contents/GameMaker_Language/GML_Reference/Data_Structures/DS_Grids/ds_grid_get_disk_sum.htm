ds_grid_get_disk_sum

Questa funzione può essere utilizzata per sommare tutti i valori di tutte le celle che si trovano all'interno dell'area disco definita di una griglia, come mostrato nell'immagine qui sotto:

Sintassi:

ds_grid_get_disk_sum(index, xm, ym, r);

ArgomentoTipo Descrizione
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Ritorna:

Real or String

Esempio:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

Il codice qui sopra imposterà la variabile "val" alla somma di tutti i valori contenuti nel disco dato della griglia DS indicizzata nella variabile "grid".
