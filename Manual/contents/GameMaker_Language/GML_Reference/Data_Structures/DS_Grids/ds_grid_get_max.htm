ds_grid_get_max

Questa funzione può essere utilizzata per trovare il valore massimo per tutte le celle che si trovano all'interno della regione definita di una griglia, come mostrato nell'immagine qui sotto:

Sintassi:

ds_grid_get_max(index, x1, y1, x2, y2);

ArgomentoTipo Descrizione
index The index of the grid.
x1 The left cell column of the region.
y1 The top cell row of the region.
x2 The right cell column of the region.
y2 The bottom cell row of the region.

Ritorna:

Real or String

Esempio:

val = ds_grid_get_max(grid, 0, 0, 5, 5)

Il codice precedente imposterà la variabile "val" al valore massimo contenuto nella regione data della griglia DS indicizzata nella variabile "grid".
