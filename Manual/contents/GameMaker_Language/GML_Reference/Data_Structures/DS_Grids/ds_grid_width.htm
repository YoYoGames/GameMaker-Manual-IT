ds_grid_width

Questa funzione restituisce la larghezza della griglia data. Questo valore è il numero di celle che la griglia ha lungo l'asse x ed è sempre un numero intero, come mostrato nell'immagine qui sotto:

Sintassi:

ds_grid_width(index);

ArgomentoTipo Descrizione
index This index of the grid to find the width of.

Ritorna:

Esempio:

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

Il codice di cui sopra si collegherà a loop attraverso la griglia DS indicizzata nella variabile "grid" e se il valore trovato in qualsiasi cella specifica è uguale a 1, creerà un'istanza di "obj_Wall" nella posizione appropriata all'interno di room.
