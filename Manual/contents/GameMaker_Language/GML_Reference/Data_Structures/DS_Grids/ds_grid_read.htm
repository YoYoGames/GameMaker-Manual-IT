ds_grid_read

Questa funzione può essere usata per convertire un string che è stato creato precedentemente dalla funzione ds_grid_write() in una griglia DS. La griglia DS deve essere stata creata in precedenza (vedi l'esempio qui sotto).

Notate che se il DS specificato string è stato scritto da GameMaker: Studio 1.2.x runtime (o più vecchio), dovreste specificare l'argomento opzionale "legacy", impostandolo su true poiché il formato string è cambiato dopo.

Sintassi:

ds_grid_read(index, string [, legacy]);

Argomento Tipo Descrizione
index ID griglia DS The index of the grid to read.
string Stringa The string to read into the DS grid.
legacy Booleano OPTIONAL Può essere true o false o omesso completamente.

Ritorna:

N/A

Esempio:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

Il codice qui sopra crea una griglia DS basata sulla dimensione di room (ogni quadrato di pixel 32x32 rappresenta una cella della griglia) e poi legge un set di dati della griglia precedentemente salvato da un file ini nella nuova griglia DS.

Indietro: Griglie DS
Il prossimo: ds_grid_write
