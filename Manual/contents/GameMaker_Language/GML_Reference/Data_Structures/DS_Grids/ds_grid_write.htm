ds_grid_write

Questa funzione può essere utilizzata per convertire la ds_grid data in un string, che può poi essere memorizzato in un file esterno (per esempio). È possibile leggere il string restituito da questa funzione in una ds_grid utilizzando la funzione ds_grid_read().

NOTA Il string restituito non è un string leggibile dall'uomo, ma piuttosto un dump del contenuto della struttura dei dati.

Sintassi:

ds_grid_write(index);

ArgomentoTipo Descrizione
index The index of the grid to write.

Ritorna:

Esempio:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Il codice qui sopra aprirà un file ini (creandolo se non esiste già) e poi scriverà la ds_grid data come string a quel file.
