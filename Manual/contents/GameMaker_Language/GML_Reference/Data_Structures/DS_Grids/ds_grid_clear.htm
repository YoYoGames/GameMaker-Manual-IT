ds_grid_clear

Questa funzione può essere usata per cancellare una determinata griglia DS ad un valore specifico. Tutte le celle all'interno della griglia conterranno quindi questo valore, che può essere un numero reale o un string. L'immagine qui sotto illustra come funziona:

Sintassi:

ds_grid_clear(index, val);

ArgomentoTipo Descrizione
index This index of the grid to clear.
val The new value for all grid cells.

Ritorna:

Esempio:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Il codice qui sopra ridimensiona la griglia DS indicizzata nella variabile globale "Grid" e poi la cancella in modo che ogni cella abbia il valore -1.

Indietro: Griglie DS
