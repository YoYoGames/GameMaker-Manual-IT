ds_grid_add

Questa funzione può essere usata per aggiungere un dato valore (reale o string) al valore della cella data all'interno della griglia. Il valore da aggiungere deve essere dello stesso tipo di quello contenuto nella cella della griglia, cioè: non si può aggiungere un string a un reale o viceversa, e per strings questo corrisponde alla concatenazione.

Sintassi:

ds_grid_add(index, x, y, val);

ArgomentoTipo Descrizione
index The index of the grid.
x The x position of the cell in the grid.
y The y position of the cell in the grid.
val The value to add to the cell.

Ritorna:

Esempio:

ds_grid_add(grid, 5, 5, 6)

Questo aggiungerebbe 6 alla cella data all'interno della griglia DS a cui fa riferimento la variabile "grid".

Indietro: Griglie DS
