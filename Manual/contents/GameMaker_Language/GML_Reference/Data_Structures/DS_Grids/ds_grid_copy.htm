ds_grid_copy

Con questa funzione potete copiare il contenuto di una griglia in un'altra. Entrambe le griglie devono essere state create in precedenza utilizzando la funzione ds_grid_create() funzione.

Sintassi:

ds_grid_copy(destination, source);

Argomento Tipo Descrizione
destination ID griglia DS This index of the grid to copy to.
source ID griglia DS This index of the grid to copy from.

Ritorna:

N/A

Esempio:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

Il codice qui sopra crea una nuova griglia DS, basata sulla larghezza e l'altezza di una griglia creata in precedenza, poi copia le informazioni dalla griglia precedente a quella nuova. Infine cancella la vecchia griglia in modo che tutte le celle abbiano un valore di -1.

Indietro: Griglie DS
