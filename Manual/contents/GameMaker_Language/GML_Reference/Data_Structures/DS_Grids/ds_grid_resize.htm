ds_grid_resize

Con questa funzione potete ridimensionare la griglia DS data per avere una larghezza e/o un'altezza diversa. Se la dimensione della griglia è più grande della griglia corrente, le nuove celle avranno un valore base di 0, e se la dimensione è più piccola allora i valori contenuti nelle celle che non sono più all'interno della nuova dimensione saranno persi. Tutte le altre celle saranno lasciate intatte.

Sintassi:

ds_grid_resize(index, w, h);

ArgomentoTipo Descrizione
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

Ritorna:

Esempio:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Il codice qui sopra ridimensiona la griglia DS indicizzata nella variabile globale "Grid" e poi la cancella in modo che ogni cella abbia il valore -1.

Indietro: Griglie DS
Il prossimo: ds_grid_clear
