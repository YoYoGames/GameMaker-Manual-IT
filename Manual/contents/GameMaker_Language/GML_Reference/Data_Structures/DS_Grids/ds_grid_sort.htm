ds_grid_sort

Questa funzione può essere usata per ordinare una griglia DS in base ai valori di una data colonna (proprio come si ordinano i file per data, dimensione ecc... nell'esploratore di file del sistema operativo). L'immagine seguente mostra un esempio:

Sintassi:

ds_grid_sort(index, column, ascending);

ArgomentoTipo Descrizione
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true), o dal più alto al più basso (false).

Ritorna:

Esempio:

ds_grid_sort(grid, 3, false)

Questo prenderebbe tutti i valori della griglia DS indicizzati nella variabile "grid" e li ordinerebbe secondo i valori trovati nella terza colonna della griglia (come mostrato nell'immagine sopra).
