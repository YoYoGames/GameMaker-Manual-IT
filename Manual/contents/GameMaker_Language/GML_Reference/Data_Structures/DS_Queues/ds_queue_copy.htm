ds_queue_copy

Questa funzione può essere usata per copiare il contenuto di un queue in un altro. Si noti che questo NON rimuove il contenuto dal queue originale, né distrugge il queue originale. Quando si usa questa funzione, il sito queue in cui si sta copiando deve essere stato creato in precedenza e se conteneva qualche elemento prima della copia, allora questi saranno cancellati per primi (il che significa che queste informazioni andranno perse).

Sintassi:

ds_queue_copy(id, source);

Argomento Tipo Descrizione
id DS Queue ID The id of the new queue.
source DS Queue ID The original queue to copy from.

Ritorna:

N/A

Esempio:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

La funzione precedente crea una nuova istanza e poi in quell'istanza crea una nuova coda DS e copia il contenuto di queue nell'istanza che esegue il blocco di codice, nell'istanza appena creata queue.
