ds_queue_destroy

Questa funzione rimuoverà la data struttura queue dalla memoria, liberando le risorse che stava usando e rimuovendo tutti i valori che conteneva. Questa funzione dovrebbe sempre essere usata quando avete finito di usare la coda DS per prevenire perdite di memoria che possono rallentare e mandare in crash il vostro gioco.

IMPORTANTE! Quando create una struttura di dati, il valore dell'indice per identificarla è un valore intero che parte da 0. Questo significa che strutture di dati di tipo diverso possono avere lo stesso valore di indice, quindi nel dubbio dovreste usare la ds_exists() prima di accedervi. Notate anche che gli indici vengono riutilizzati, quindi un valore di indice di una struttura dati distrutta può essere utilizzato da una nuova struttura creata in seguito, quindi si consiglia di impostare sempre la variabile che conteneva l'indice DS a -1 dopo la distruzione.

Sintassi:

ds_queue_destroy(id);

ArgomentoTipo Descrizione
id The id of the data structure to remove.

Ritorna:

Esempio:

if (lives == 0)
{
    ds_queue_destroy(AI_queue);
    AI_queue = -1;
    room_goto(rm_Menu);
}

Il codice qui sopra controllerà il valore della variabile globale incorporata "lives" e se è 0, distrugge la coda DS indicizzata nella variabile "AI_queue" e poi cambia rooms.
