ds_queue_clear

Con questa funzione potete cancellare tutti i dati dalla data queue data-structure. Questo NON distrugge la struttura dati (per quello dovreste usare ds_queue_destroy()) cancella solo tutti i dati da essa e restituisce un queue vuoto.

Sintassi:

ds_queue_clear(id);

ArgomentoTipo Descrizione
id The id of the data structure to clear.

Ritorna:

Esempio:

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella la coda DS indicizzata nella variabile "command_queue", imposta un allarme e resetta la variabile a 0.

Indietro: Code DS
