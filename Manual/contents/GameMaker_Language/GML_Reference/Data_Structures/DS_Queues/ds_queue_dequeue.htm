ds_queue_dequeue

Questa funzione rimuoverà il valore di testa dal DS queue, rimuovendolo dal queue e restituendo il valore da memorizzare in una variabile. Se il queue è vuoto, allora la funzione restituirà la costante undefined, altrimenti restituirà il valore contenuto nel queue.

Sintassi:

ds_queue_dequeue(id);

ArgomentoTipo Descrizione
id The id of the queue to dequeue from.

Ritorna:

Variable (Data type value stored in the queue) or undefined

Esempio:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

Il codice di cui sopra controlla la coda DS indicizzata nella variabile "move_queue" per vedere se è vuota, e se non lo è, allora de-clude i due valori dalla testa del queue e li usa per impostare una direzione per il movimento.
