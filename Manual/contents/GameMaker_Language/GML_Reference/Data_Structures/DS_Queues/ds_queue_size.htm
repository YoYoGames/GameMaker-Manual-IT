ds_queue_size

Questa funzione restituisce la "dimensione" di queue, cioè: il numero di elementi che sono stati accodati su di esso.

Sintassi:

ds_queue_size(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if !ds_queue_empty(control_queue)
{
    num = ds_queue_size(control_queue);
}

Il codice qui sopra controlla una coda DS per vedere se è vuota o no, e se non lo è, ottiene il numero di elementi che contiene e memorizza il valore in una variabile.
