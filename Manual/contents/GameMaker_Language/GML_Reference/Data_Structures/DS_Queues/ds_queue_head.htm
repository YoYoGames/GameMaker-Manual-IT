ds_queue_head

Questa funzione leggerà solo il primo valore di queue (quello che è "in testa"). Non rimuoverà il valore, il che significa che può ancora essere letto in futuro da questa funzione o dal ds_queue_dequeue(). Se queue è vuoto, la funzione restituirà la costante undefined, altrimenti restituirà il valore reale o string contenuto in queue.

Sintassi:

ds_queue_head(id);

ArgomentoTipo Descrizione
id The id of the data structure to read from.

Ritorna:

(Data type value stored in the queue)

Esempio:

num = ds_queue_head(control_queue);

Il codice precedente leggerà il valore da queue indicizzato nella variabile "control_queue" e memorizzerà il valore di ritorno nella variabile "num".
