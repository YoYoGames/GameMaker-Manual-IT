ds_queue_enqueue

Questa funzione aggiunge un valore (reale o string) alla coda della coda DS. La funzione può accettare altri 14 argomenti opzionali (per un totale di 15 possibili aggiunte), permettendovi di aggiungere più valori consecutivamente alla coda di queue in una singola chiamata.

Sintassi:

ds_queue_enqueue(id, val [, val2, ... val15]);

ArgomentoTipo Descrizione
id The id of the queue to add to.
val The value to add to the queue.
[val2, ... val15] Optional values to be added to the queue.

Ritorna:

Esempio:

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

Il codice qui sopra crea una nuova coda DS e memorizza il suo indice nella variabile "move_queue". Poi spinge un certo numero di valori su queue per un uso futuro.

Indietro: Code DS
