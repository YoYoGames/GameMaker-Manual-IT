ds_queue_empty

Con questa funzione potete controllare la coda DS data per vedere se è vuota (restituisce true) o no (restituisce false).

Sintassi:

ds_queue_empty(id);

ArgomentoTipo Descrizione
id The id of the data structure to check.

Ritorna:

Esempio:

if (count == 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Il codice qui sopra controlla una variabile per vedere se ha raggiunto un valore specifico e se lo ha fatto cancella la coda DS indicizzata nella variabile "command_queue", imposta un allarme e resetta la variabile a 0.

Indietro: Code DS
