ds_exists

Con questa funzione potete controllare se esiste una struttura di dati del tipo dato. Voi fornite il valore "index" (tenuto in una variabile) e il "type" DS, che può essere una qualsiasi delle costanti elencate di seguito, e la funzione restituisce true se la struttura dati esiste e false altrimenti.

DS Tipo costante
Costante Descrizione
ds_type_map Una struttura di dati a forma di mappa
ds_type_list Una struttura dati a lista
ds_type_stack Una struttura dati a pila
ds_type_grid Una struttura dati a griglia
ds_type_queue Una struttura di dati della coda
ds_type_priority Una struttura di dati di priorità

NOTA Non potete usare questa funzione per controllare il tipo di una struttura di dati, poiché lo stesso numero di indice può essere usato da più strutture di dati di tipo diverso.

Sintassi:

ds_exists(ind, type);

Argomento Tipo Descrizione
ind Qualsiasi ID DS The variable index to check for the data structure
type DS Tipo costante The type of data structure to check for (see the list of constants above)

Ritorna:

Boolean

Esempio:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

Il codice qui sopra controlla la variabile (precedentemente inizializzata) "ai_grid" per vedere se indicizza una struttura dati di tipo DS grid, e se non lo fa allora ne crea una e memorizza il suo indice nella variabile.

Indietro: Strutture di dati
