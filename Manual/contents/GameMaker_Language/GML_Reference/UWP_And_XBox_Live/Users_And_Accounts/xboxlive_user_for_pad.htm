xboxlive_user_for_pad

Questa funzione restituisce il puntatore dell'ID utente associato al valore dell'indice del gamepad dato, o pointer_null se nessun utente è associato ad esso. Si noti che questa funzione dovrebbe essere usata solo con gli indici del gamepad, e non con xboxlive_get_user_count().

Sintassi:

xboxlive_user_for_pad(pad_index);

ArgomentoTipo Descrizione
pad_index The index (an integer) of the gamepad slot to check

Ritorna:

Xbox User ID or pointer_null

Esempio:

for(var i = 0; i < 11; ++i;)
{
    var u_id = xboxlive_user_for_pad(i);
    if (u_id != pointer_null)
    {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
    }
    else
    {
        pad_name[i] = "Press Play";
    }
}

Il codice qui sopra loops attraverso tutti gli slot pad disponibili e controlla se c'è un puntatore ID utente associato ad esso. Se c'è, viene impostato un array con il nome dell'utente, altrimenti l'array viene impostato su un testo predefinito.
