xboxlive_generare_il_giocatore_session_id

Questa funzione genererà un unico string (contenente un GUID) per la sessione corrente del giocatore. Questo può poi essere usato con la funzione xboxlive_fire_event().

Sintassi:

xboxlive_generate_player_session_id();

Ritorna:

Esempio:

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Il codice qui sopra controlla per vedere se un utente è iscritto a XBox Live e se lo è, il codice otterrà la gamertag dell'utente e la memorizzerà in una variabile globale per uso futuro. Se l'utente non è registrato, il codice controlla se è in procinto di registrarsi, nel qual caso non succederà nulla, e se non lo è, allora aprirà il selezionatore di account, invitandolo a registrarsi.
