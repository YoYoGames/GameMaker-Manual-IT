xboxlive_user_is_active

Con questa funzione potete controllare se l'ID utente dato è nella lista degli utenti che stanno usando la console, e la funzione restituirà true se lo è, o false altrimenti. Potete ottenere un puntatore all'ID utente con la funzione xboxlive_get_user().

Sintassi:

xboxlive_user_is_active(user_id);

ArgomentoTipo Descrizione
user_id The ID pointer of the user to check.

Ritorna:

Esempio:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    var _uid = xboxlive_get_user(i);
    if xboxlive_user_is_active(_uid)
    {
        global.Player_ID[global.PlayerNum++] = _uid;
    }
}

Il codice qui sopra loops attraverso gli account utente e poi controlla se qualcuno di loro è attivo. Se lo sono, il loro ID utente viene aggiunto in un array globale.
