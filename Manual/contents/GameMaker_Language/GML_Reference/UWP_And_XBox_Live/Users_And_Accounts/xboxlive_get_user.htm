xboxlive_get_user

Con questa funzione potete recuperare il puntatore dell'ID utente per l'utente indicizzato. Se l'utente non esiste, la funzione restituirà invece la costante pointer_null. Potete trovare il numero di utenti attualmente connessi con la funzione xboxlive_get_user_count().

IMPORTANTE: Questa funzione dovrebbe essere usata solo con xboxlive_get_user_count() - non usarla con i gamepad. Usate invece xboxlive_user_for_pad().

Sintassi:

xboxlive_get_user(index);

Argomento Tipo Descrizione
index Real The index (an integer) to get the User ID from.

Ritorna:

Xbox User ID or pointer_null

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Il precedente loops attraverso tutti gli utenti registrati e memorizza il loro puntatore ID unico in un array.
