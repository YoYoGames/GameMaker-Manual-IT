xboxlive_pad_count_per_utente

Con questa funzione potete scoprire quanti "slot" di pad sono attivi per l'utente corrente (vedi xboxlive_pad_for_user() per ulteriori dettagli).

Sintassi:

xboxlive_pad_count_for_user(user_id, slot);

ArgomentoTipo Descrizione
user_id The user ID (a pointer) to check

Ritorna:

Esempio:

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

Il codice di cui sopra memorizza il numero di pad associati al puntatore dell'ID utente dato.
