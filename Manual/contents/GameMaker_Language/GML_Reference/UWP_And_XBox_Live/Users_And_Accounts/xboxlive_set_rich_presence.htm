xboxlive_set_rich_presence

Questa funzione imposterà la ricca presenza string per l'utente dato. Una Rich Presence string mostra l'attività di gioco di un utente dopo il nome del gioco a cui l'utente sta giocando, separato da un trattino. Questo string viene visualizzato sotto la Gamertag di un giocatore nella lista "Friends & Clubs" così come nel profilo utente Xbox Live del giocatore.

Quando si usa questa funzione è necessario fornire il puntatore ID utente per l'utente, e poi si può segnalare l'utente come attualmente attivo nel gioco o no (usando true/false). Se impostato su false, allora la ricca presenza string sarà aggiunta con "/afk" o qualcosa di appropriato. L'argomento successivo è il rich presence string ID da mostrare, e poi finalmente si può (opzionalmente) fornire un service_config_id string. Questo argomento è opzionale, e quando non viene fornito, usa lo SCID specificato nelle Opzioni di gioco Xbox.

Sintassi:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Argomento Tipo Descrizione
user_id ID utente Xbox The ID pointer of the user to check.
is_user_active Booleano Flag the user as active or not.
rich_presence_string Stringa The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Stringa OPTIONAL Questo è il config_id string per il gioco.

Ritorna:

N/A

Esempio:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

Il codice di cui sopra ottiene il puntatore ID utente per l'utente assegnato allo slot gamepad [0], e poi imposta la ricca presenza string per quell'utente.
