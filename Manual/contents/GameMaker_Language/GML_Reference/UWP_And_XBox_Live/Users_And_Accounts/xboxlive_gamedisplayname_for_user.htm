xboxlive_gamedisplayname_for_user

Con questa funzione potete recuperare il nome del display - come un string - dal puntatore ID utente dato. Notate che se l'utente è locale allora il valore restituito è semplicemente un string del suo nome visualizzato, ma se si tratta di un utente remoto allora il string sarà la sua gamertag che può essere usata, per esempio, per il muting/unmuting nella chat vocale.

Sintassi:

xboxlive_gamedisplayname_for_user(user_id);

ArgomentoTipo Descrizione
user_id The ID pointer of the user to check.

Ritorna:

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

Il codice precedente ottiene il puntatore all'ID utente per ogni account utente disponibile e poi memorizza il puntatore insieme al nome visualizzato in due array.
