xboxlive_get_user_count

Con questa funzione puoi trovare il numero totale di utenti attualmente iscritti al sistema XBox. Il valore di ritorno sarà un valore intero, da 0 - N, dove N è il numero di utenti registrati -1.

Sintassi:

xboxlive_get_user_count();

Ritorna:

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Il precedente loops attraverso tutti gli utenti registrati e memorizza il loro puntatore ID unico in un array.
