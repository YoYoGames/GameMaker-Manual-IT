xboxlive_pad_per_utente

Gli utenti Xbox possono avere vari pad assegnati al loro ID utente, e ognuno di essi riempie uno "slot". Questi slot sono numerati da 0, e possono contenere qualsiasi indice di pad, quindi lo slot 0 potrebbe essere assegnato al pad 3 e lo slot 1 potrebbe essere assegnato al pad 2. Questa funzione può essere usata per ottenere il pad assegnato a qualsiasi slot, ed è particolarmente utile nel caso in cui un utente lasci un gamepad senza input per un certo periodo di tempo, perché si spegnerà da solo switch, quindi è possibile controllare qualsiasi ID utente usando questa funzione per assicurarsi che abbia un gamepad attivo, e se non lo ha, reagire di conseguenza (per esempio) mettendo in pausa il gioco. Se il pad è attivo, verrà restituito il suo valore di indice, ma se non lo è, la funzione restituirà -1.

Notate che se un pad in qualsiasi slot diventa inattivo, gli slot successivi "scivoleranno" nella posizione vacante, quindi se avete tre pad attivi assegnati agli slot 0, 1 e 2, e il pad nello slot 0 è inattivo, gli slot 1 e a diventeranno ora gli slot 0 e 1, quindi lo slot 0 dovrebbe sempre contenere un pad attivo a meno che tutti i pad siano spenti (nel qual caso tornerà -1).

Sintassi:

xboxlive_pad_for_user(user_id, slot);

Argomento Tipo Descrizione
user_id ID utente Xbox The User ID (a pointer) to check
slot Real The index (an integer) of the gamepad slot to check

Ritorna:

Xbox Pad ID

Esempio:

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++;)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

Il codice di cui sopra controlla lo slot 0 per il puntatore ID utente dato per vedere se sono stati assegnati dei gamepad, e se non lo sono ne viene assegnato uno.
