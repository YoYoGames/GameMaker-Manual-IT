xboxlive_user_is_remote

Questa funzione controllerà l'ID utente dato e restituirà true se il giocatore è un giocatore remoto, o false altrimenti.

Sintassi:

xboxlive_user_is_remote(user_id);

ArgomentoTipo Descrizione
user_id The ID pointer of the user to check.

Ritorna:

Esempio:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    var _uid = xboxlive_get_user(i);
    if (xboxlive_user_is_active(_uid))
    {
        if (!xboxlive_user_is_remote(_uid))
        {
            global.Remote_Player_ID[global.PlayerNum++] = _uid;
        }
    }
}

Il codice qui sopra loops attraverso gli account utente e poi controlla se qualcuno di loro è attivo. Se lo sono, il loro ID utente viene aggiunto in un array globale, solo se non sono utenti remoti.
