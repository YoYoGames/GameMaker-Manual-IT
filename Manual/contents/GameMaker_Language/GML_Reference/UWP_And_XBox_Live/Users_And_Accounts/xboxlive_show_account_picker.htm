xboxlive_show_account_picker

La funzione aprirà la schermata XBox Account Picker in modo che l'utente possa registrarsi se non è già registrato (se lo è, la funzione non farà nulla). Si noti che questa funzione è applicabile solo quando si usa la build UWP su PC e non su Xbox, poiché per lanciare un gioco su Xbox l'utente deve aver già effettuato l'accesso.

Sintassi:

xboxlive_show_account_picker();

Ritorna:

Esempio:

if !xboxlive_user_is_signed_in()
{
    if (!xboxlive_user_is_signing_in())
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Il codice qui sopra controlla per vedere se un utente è iscritto a XBox Live e se lo è, il codice otterrà la gamertag dell'utente e la memorizzerà in una variabile globale per uso futuro. Se l'utente non è registrato, allora il codice controlla se è in procinto di registrarsi, nel qual caso non succederà nulla, e se non lo è, allora aprirà il selezionatore di account, invitandolo a registrarsi.
