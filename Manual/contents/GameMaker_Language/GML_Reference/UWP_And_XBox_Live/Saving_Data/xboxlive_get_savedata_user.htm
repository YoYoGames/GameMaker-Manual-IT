xboxlive_get_savedata_user

Questa funzione restituisce il puntatore dell'ID utente attualmente associato al salvataggio del file (o la costante pointer_null se nessun ID utente è attualmente utilizzato). Vedere xboxlive_set_savedata_user() per ulteriori dettagli.

Sintassi:

xboxlive_get_savedata_user();

Ritorna:

Xbox User ID or pointer_null

Esempio:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Il codice qui sopra controlla per vedere se un utente è attualmente assegnato come obiettivo di salvataggio, e se non lo è allora viene assegnato.
