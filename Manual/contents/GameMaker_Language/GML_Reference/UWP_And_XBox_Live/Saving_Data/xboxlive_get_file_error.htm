xboxlive_get_file_error

Questa funzione restituirà una costante (mostrata qui sotto) per aiutare il debug del motivo per cui una funzione di file è fallita, in modo che possiate prendere le misure appropriate nel vostro codice di gioco. Quando una delle funzioni di file fallisce, potete chiamare la funzione per scoprire esattamente qual è stato l'errore.

NOTA: Quando si usano le funzioni asincrone di salvataggio/caricamento, la async_load avrà una chiave extra, "error" che conterrà anche una delle costanti (mostrata qui sotto).

Sintassi:

xboxlive_get_file_error();

Ritorna:

Xbox File Error Constant

Costante di errore del file Xbox
Costante Descrizione
xboxlive_fileerror_outoflocalstorage Se questo si verifica durante il salvataggio, significa che tutto lo spazio disponibile per il salvataggio del gioco è stato utilizzato, e l'utente ha rifiutato l'offerta (presentata dal sistema operativo) di liberare spazio. Il gioco dovrebbe confermare all'utente che intende continuare a giocare senza salvare. Se l'utente vuole ancora essere in grado di salvare, il gioco dovrebbe riprovare l'operazione di salvataggio.
xboxlive_fileerror_quotaexceeded Se questo accade, il gioco dovrebbe avvisare l'utente che non può salvare altri dati fino a quando non si libera dello spazio, e poi presentargli un modo per farlo.
xboxlive_fileerror_noerror La funzione di salvataggio o caricamento dei file è stata eseguita senza problemi.

Esempio:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

Il codice di cui sopra controlla se c'è stato un errore nel salvataggio e se è così il codice di errore viene stampato nella console.
