xboxlive_set_savedata_user

Questa funzione specifica che le future operazioni sui file che operano nell'area di salvataggio del gioco (cioè tutte le scritture di file e le letture di file che non sono nell'area del bundle) saranno associate al puntatore ID utente specificato. Questo può essere chiamato tutte le volte che è necessario per reindirizzare i dati di salvataggio all'utente appropriato, oppure si può usare la costante pointer_null per salvare nell'area di memorizzazione generica della macchina.

Sintassi:

xboxlive_set_savedata_user(user_id);

Argomento Tipo Descrizione
user_id ID utente Xbox o puntatore_nullo The user ID (a pointer) to set for saving, or pointer_null

Ritorna:

N/A

Esempio:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Il codice qui sopra controlla per vedere se un utente è attualmente assegnato come obiettivo di salvataggio, e se non lo è allora viene assegnato.
