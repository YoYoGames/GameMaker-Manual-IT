è vincolato

Questa funzione restituisce true se il gioco è in esecuzione in una modalità con risorse limitate o false se è in esecuzione normalmente. La modalità a risorse limitate può essere attivata quando il gioco non ha il focus, e in questa modalità il gioco ha risorse GPU ridotte. La risposta a questo dovrebbe normalmente essere quella di mettere in pausa il gioco (dato che l'utente non ha più il controllo). Inoltre, disabilitare qualsiasi effetto particolarmente impegnativo può essere una buona idea se si osserva un rallentamento in questa modalità. Si noti che, a differenza della funzione uwp_is_suspending() questa funzione restituirà true continuamente per tutto il tempo in cui il gioco è in modalità vincolata (e non solo una volta quando accade per la prima volta).

Sintassi:

uwp_is_constrained();

Ritorna:

Esempio:

if xboxliveis_constrained()
{
    if (!instance_exists(obj_PauseMenu))
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

Il codice qui sopra controlla se l'app è vincolata e se lo è crea un'istanza di un object (usato per mettere in pausa il gioco).
