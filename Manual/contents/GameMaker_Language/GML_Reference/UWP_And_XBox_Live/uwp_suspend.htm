uwp_sospendi

Questa funzione indica al sistema che l'app ha finito di salvare i dati in seguito al rilevamento di una richiesta di sospensione. Chiamare questa funzione è un requisito per l'invio di Xbox One, e dovrebbe sempre essere usato insieme alla funzione uwp_is_suspending()come mostrato nell'esempio qui sotto.

Sintassi:

uwp_suspend();

Ritorna:

Esempio:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Il codice di cui sopra controlla se l'app sta andando in sospensione e se lo è chiama un script per salvare i dati di gioco prima di sospendere il gioco.
