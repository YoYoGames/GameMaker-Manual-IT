xboxlive_matchmaking_stop

Questa funzione può essere usata per terminare una sessione di matchmaking per l'utente dato.

Sintassi:

xboxlive_matchmaking_stop(user_id);

ArgomentoTipo Descrizione
user_id The user ID pointer to use

Ritorna:

Esempio:

if !xboxlive_user_is_signed_in(user_id[0])
{
    xboxlive_matchmaking_stop(user_id[0]);
}

Il codice di cui sopra terminerà la sessione di matchmaking per l'utente dato.
