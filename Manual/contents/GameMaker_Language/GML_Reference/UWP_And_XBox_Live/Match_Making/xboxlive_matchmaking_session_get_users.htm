xboxlive_matchmaking_session_get_users

Questa funzione crea e popola una mappa DS con i dettagli degli utenti nella sessione specificata, o -1 se c'è un errore. È possibile ottenere il puntatore all'ID della sessione dalla async_load mappa DS che viene restituita nell'evento sociale asincrono quando si è creata o trovata una sessione (vedere xboxlive_matchmaking_create() per maggiori dettagli). La mappa DS restituita avrà le seguenti coppie chiave/valore:

  • "num_results" - contiene il numero di utenti nella sessione
  • "userId<indice>" - contiene l'ID dell'utente (<indice> è un valore da 0 a "num_results" - 1)
  • "userIsOwner<index>" - contiene 1 se l'utente è l'host, altrimenti 0 (<index> è un valore da 0 a "num_results" - 1)

Sintassi:

xboxlive_matchmaking_session_get_users(session_id);

ArgomentoTipo Descrizione
session_id The session ID pointer to use

Ritorna:

Async Request ID

Esempio:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

Quanto sopra recupererà i dati utente per tutti gli utenti che partecipano alla sessione di matchmaking e imposterà una variabile globale all'ID del proprietario della sessione.
