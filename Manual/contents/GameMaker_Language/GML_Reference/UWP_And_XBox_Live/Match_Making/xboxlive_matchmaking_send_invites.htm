xboxlive_matchmaking_send_invites

Questa funzione fa apparire la finestra di dialogo del sistema Xbox Live per permettere a un utente di inviare inviti alla sua partita. Si specifica l'utente locale che dovrebbe controllare la finestra di dialogo (e la cui lista di amici sarà usata dalla finestra di dialogo), il valore unico dell'ID di sessione per la sessione a cui si vuole invitare la gente, e del testo predefinito da mostrare nella finestra di dialogo. Si ottiene l'ID di sessione dalla mappa async_load mappa DS generata dalla funzione xboxlive_matchmaking_create().

La funzione restituirà -1 se gli argomenti non sono validi o se l'utente non è iscritto a Xbox Live, o un valore maggiore di 0 altrimenti.

Sintassi:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

ArgomentoTipo Descrizione
from_user The local user ID for controlling the dialogue.
session_to_invite_into The unique session ID for the session to join.
text_to_add_to_invite A short text to add to the invitation.

Ritorna:

Esempio:

if gamepad_button_check_pressed(0, gp_face1)
{
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

Il codice di cui sopra controlla la pressione di un pulsante del gamepad e se ne viene rilevato uno, viene aperta la finestra di dialogo di Xbox Live per gli inviti alla sessione.
