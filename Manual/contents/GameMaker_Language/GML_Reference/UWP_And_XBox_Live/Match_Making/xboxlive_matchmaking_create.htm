xboxlive_matchmaking_creare

Questa funzione creerà una sessione multigiocatore utilizzando il modello di sessione, il matchmaking hopper e il modello Secure Device Association, che dovreste aver creato in precedenza nella dashboard XDP. L'ID utente che si può recuperare per un particolare game pad chiamando la funzione xboxlive_user_for_pad() e la visibilità sarà una delle seguenti costanti:

Costante di visibilità di Xbox Match
Costante Descrizione
xboxlive_match_visibility_open specifica che la sessione può essere raggiunta dagli altri.
xboxlive_match_visibility_private specifica che la sessione non è visibile agli utenti che non sono membri della sessione.
xboxlive_match_visibility_usetemplate specifica che dovrebbe essere usato il valore usato nel modello di sessione (questo dovrebbe essere usato normalmente, poiché non si può sovrascrivere il valore di un modello se ne è stato impostato uno in XDP).

La funzione restituisce un valore unico di ID della richiesta, che può essere utilizzato per identificare il corretto evento sociale asincrono per questa funzione. Questo evento sarà attivato quando la sessione è stata creata e conterrà una DS Map nella variabile async_load con le seguenti coppie chiave/valore:

  • "requestid" - contiene l'ID della richiesta come restituito dalla funzione chiamante
  • "status" - contiene il string "session_created" per informare su quale tipo di evento è stato attivato.
  • "sessionid" - contiene l'ID della sessione, o -1 in caso di fallimento
  • "error" - contiene 0 in caso di successo, o -1 in caso di fallimento
  • "correlationid" - l'handle di sessione per la correlazione con la funzione xboxlive_matchmaking_join_session()

Sintassi:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Argomento Tipo Descrizione
user_id ID utente Xbox The user ID pointer to use
visibility Costante di visibilità di Xbox Match One of the constants listed above
template Stringa The name of the session template
hopper Stringa The name of the matchmaking hopper
sdatemplate Stringa The name of the secure device association template
matchattributes Real OPTIONAL Gli attributi della partita

Ritorna:

Async Request ID

Esempio:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Quanto sopra recupererà l'ID utente per l'utente sul gamepad dato e poi inizializzerà una sessione di matchmaking per lui.
