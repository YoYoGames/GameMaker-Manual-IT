xboxlive_matchmaking_join_invite

Questa funzione può essere chiamata per elaborare un invito ad unirsi, una volta che si è chiamato con successo xboxlive_matchmaking_start(). Si fornisce lo UserID locale e i valori dell'invito (ID e ID host) che sono stati restituiti quando l'invito è stato ricevuto, insieme al nome del modello di sessione che è stato creato (un string). Si unirà quindi l'utente alla sessione a cui è stato invitato e si riceveranno gli eventi sociali asincroni come se si fosse chiamato xboxlive_matchmaking_find().

Sintassi:

xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name);

ArgomentoTipo Descrizione
user_who_received_invite The local user ID pointer.
invitation_id The invitation ID.
invitation_host The invitation host ID.
sda_template_name The session template name.

Ritorna:

Esempio:

var userID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate);

Quanto sopra ottiene l'ID utente per il gamepad dato e poi tenta di unirsi alla sessione a cui sono stati invitati.

Indietro: Fare gli abbinamenti
Il prossimo: xboxlive_matchmaking_start
