xboxlive_matchmaking_start

Questa funzione deve essere chiamata prima di iniziare qualsiasi sessione multigiocatore per l'utente. Prende il puntatore ID utente, che potete recuperare per un particolare game pad chiamando la funzione xboxlive_user_for_pad() e inizializzerà l'API multigiocatore.

Sintassi:

xboxlive_matchmaking_start(user_id);

ArgomentoTipo Descrizione
user_id The user ID pointer to use

Ritorna:

Esempio:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

Il codice di cui sopra ottiene l'ID utente per l'indice del pad dato e poi inizializza le capacità multiplayer.

Indietro: Fare gli abbinamenti
Il prossimo: xboxlive_matchmaking_stop
