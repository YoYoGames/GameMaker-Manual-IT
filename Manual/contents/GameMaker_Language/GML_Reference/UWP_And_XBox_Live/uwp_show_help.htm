Mostra aiuto

Questa funzione lancia l'applicazione di aiuto di Xbox One che a sua volta visualizzerà il file di aiuto del vostro gioco (che è un file in formato HTML che avete precedentemente caricato sul portale degli sviluppatori XBox). Notate che un ID utente è specificato quando si chiama questa funzione in modo che le informazioni locali dell'utente possano essere utilizzate quando viene visualizzato. È possibile recuperare l'ID utente con le funzioni xboxlive_get_user(), xboxlive_get_activating_user(), o uwp_license_trial_user().

Sintassi:

uwp_show_help(user_id);

ArgomentoTipo Descrizione
user_id The User ID pointer to open the helpfile for.

Ritorna:

Esempio:

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

Il codice qui sopra controlla se viene premuto un pulsante e poi controlla l'applicazione per vedere se è in esecuzione con una licenza di prova. Se lo è, allora ottiene l'ID utente per la licenza e apre il file di aiuto per quell'utente.

Indietro: UWP e XBox Live
Il prossimo: uwp_license_trial_version
