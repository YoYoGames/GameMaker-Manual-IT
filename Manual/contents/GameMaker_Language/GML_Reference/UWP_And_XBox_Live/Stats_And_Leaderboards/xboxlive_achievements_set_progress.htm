xboxlive_achievements_set_progress

Questa funzione può essere usata per aggiornare il progresso di una conquista. Si fornisce l'ID utente come restituito, per esempio, dalla funzione xboxlive_get_user()e poi la stringa della conquista (questo è l'id della conquista come assegnato nell'XDP/Windows Dev Center quando è stata creata). Infine si imposta il valore di progresso su un valore da 0 a 100. Si noti che il sistema delle conquiste rifiuterà gli aggiornamenti che sono inferiori all'attuale valore di progresso corrente.

Sintassi:

xboxlive_achievements_set_progress(user_id, achievement, progress);

ArgomentoTipo Descrizione
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

Ritorna:

Esempio esteso:

Quello che segue è un esempio esteso di come questa funzione può essere usata. Per cominciare, la chiamereste in qualche evento come Room Start o Create:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Il codice di cui sopra verrebbe chiamato per ottenere una lista di tutte le posizioni della classifica globale per il gioco, e genererà una chiamata all'evento asincrono sociale che tratteremo nel modo seguente:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Il codice qui sopra crea un valore percentuale basato su alcune variabili globali e poi lo usa per impostare il progresso della conquista per la conquista "Game_Completed".
