xboxlive_achievement_load_friends

Questa funzione invierà una richiesta a server per informazioni su tutti gli amici degli utenti connessi e attiverà un callback Social Asynchronous Event che contiene la async_load popolata con le relative coppie chiave/valore. La chiave id di questo DS Map è usata per identificare il callback corretto (ci può essere più di una funzione di trigger per ogni dato evento asincrono), e sarà accoppiata con la costante xboxlive_achievement_friends_info così come una serie di altre coppie chiave/valore per ogni amico. Il contenuto esatto della mappa è mostrato di seguito:

  • "id" - Per questa funzione dovrebbe essere xboxlive_achievement_friends_info
  • "FriendN" - Il nome dell'amico, dove "N" è un valore intero corrispondente alla sua posizione nella lista degli amici.
  • "FriendidN" - L'id utente unico dell'amico, "N".

Sintassi:

xboxlive_achievement_load_friends();

Ritorna:

N/A

Esempio esteso:

Il seguente codice verrebbe probabilmente chiamato dopo che il giocatore ha effettuato l'accesso al proprio account di gioco per ottenere una lista di tutti gli amici dell'utente:

xboxlive_achievement_load_friends();

Questo invierà una richiesta per le informazioni sugli amici degli utenti e genererà un callback asincrono con la speciale mappa async_load DS contenente i seguenti dati:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_friends_info
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

Il codice di cui sopra controlla il DS Map restituito nell'evento sociale asincrono e se il suo "id" corrisponde alla costante da controllare, allora loops attraverso la mappa memorizzando tutti i diversi valori in una serie di array.
