xboxlive_stats_delete_stat

Questa funzione può essere usata per cancellare una stat dal gestore di stat per l'ID utente dato. Si fornisce l'ID utente come restituito (per esempio) dalla funzione xboxlive_get_user()poi la stat string da cancellare. Questo cancella il valore della stat e la rimuove dal gestore delle stat, il che significa che non sarà più restituita dalle funzioni xboxlive_stats_get_stat_names() o xboxlive_stats_get_stat().

La funzione restituirà -1 se c'è stato un errore o l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo

Sintassi:

xboxlive_stats_delete_stat(user_id, stat);

ArgomentoTipo Descrizione
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

Ritorna:

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

Il codice di cui sopra loops attraverso gli utenti connessi e cancella la stat specificata dal gestore di stat per ognuno di essi.
