xboxlive_stats_get_stat

Questa funzione può essere usata per recuperare un singolo valore di stat dal gestore di stat per un dato utente. Si fornisce l'ID utente come restituito (per esempio) dalla funzione xboxlive_get_user()e poi la stat string come definita quando l'avete creata usando una delle funzioni xboxlive_stats_set_stat_*. Il valore di ritorno può essere o un string o un reale (a seconda della stat da controllare) o la costante GML undefined se la stat non esiste o c'è stato un errore.

Sintassi:

xboxlive_stats_get_stat(user_id, stat);

ArgomentoTipo Descrizione
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Ritorna:

Real, String or undefined

Esempio:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Il codice qui sopra controlla una variabile e se è true, allora controllerà il valore della statistica "PercentDone". Se questo valore è inferiore a 100, viene generato un valore e la statistica viene impostata su quel valore.
