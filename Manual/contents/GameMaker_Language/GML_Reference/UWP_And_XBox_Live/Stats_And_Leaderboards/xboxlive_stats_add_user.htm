xboxlive_stats_add_user

Questa funzione può essere usata per aggiungere un dato puntatore ID utente al gestore delle statistiche. Questo deve essere fatto prima di usare qualsiasi altra funzione di statistiche per sincronizzare automaticamente il gioco con XBox Live server e recuperare gli ultimi valori. Si fornisce l'ID utente come restituito da (per esempio) la funzione xboxlive_get_user()e la funzione restituirà -1 se c'è stato un errore o se l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo.

La funzione genererà un callback che innescherà un evento asincrono di sistema. Questo evento avrà la speciale mappa DS async_load che dovrebbe poi essere analizzato per le seguenti chiavi:

  • "id" - Manterrà la costante achievement_stat_event
  • "eventname" - Conterrà il string "LocalUserAdded"
  • "userid" - L'ID utente associato alla richiesta
  • "error" - 0 se successo, qualche altro valore se c'è stato un errore
  • "errormessage" - Un string con un messaggio di errore, se disponibile

Sintassi:

xboxlive_stats_add_user(user_id);

ArgomentoTipo Descrizione
user_id The user ID (a pointer) to add

Ritorna:

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
}

Il codice di cui sopra otterrà il numero di account utente disponibili, poi loop attraverso di essi e assegnerà l'ID dell'account a un array, oltre a registrare l'utente con il gestore delle statistiche.
