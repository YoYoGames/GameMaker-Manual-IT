xboxlive_achievement_load_leaderboard

Questa funzione invierà una richiesta a server per ottenere informazioni sulla classifica data. La classifica deve essere stata creata in precedenza sulla dashboard dello sviluppatore del vostro gioco e l'ID dato è quello che dovete fornire come primo argomento della funzione. Si attiverà un callback Social Asynchronous Event che contiene la async_load mappa popolata con le relative coppie chiave/valore. La chiave id di questo DS Map è usata per identificare il callback corretto (ci può essere più di una funzione di trigger per ogni dato evento asincrono), e sarà accoppiata con la costante xboxlive_achievement_leaderboard_info così come una serie di altre coppie chiave/valore per ogni giocatore. Il contenuto esatto della mappa è mostrato di seguito:

  • "id" - Per questa funzione dovrebbe essere xboxlive_achievement_leaderboard_info
  • "leaderboardid" - L'ID univoco per la leaderboard come definito nella dashboard del fornitore.
  • "numentries" - Il numero di voci nella classifica che hai ricevuto.
  • "PlayerN" - Il nome del giocatore, dove "N" è un valore intero corrispondente alla sua posizione nella lista delle voci della classifica.
  • "PlayeridN" - L'id utente unico del giocatore, "N".
  • "RankN" - Il rango del giocatore "N" all'interno della classifica.
  • "ScoreN" - Il punteggio del giocatore "N".

Puoi specificare l'esatta porzione di classifica da mostrare dando un valore di indice minimo e massimo, così come richiedere che i risultati siano filtrati per recuperare solo i punteggi dei tuoi amici o quelli postati da tutti i giocatori usando le seguenti costanti di filtro:

Xbox Live Achievement filtro costante
Costante Descrizione
xboxlive_achievement_filter_friends_only  Ottieni solo i punteggi dei giocatori amici.
xboxlive_achievement_filter_all_players  Ottieni tutti i punteggi per tutti i giocatori.

Sintassi:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argomento Tipo Descrizione
ident Stringa The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Xbox Live Achievement filtro costante Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Ritorna:

N/A

Esempio esteso:

Il seguente codice verrebbe probabilmente chiamato dopo che il giocatore ha effettuato l'accesso al proprio account di gioco per ottenere una lista di tutte le posizioni in classifica degli amici per il gioco:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Questo invierà una richiesta per le informazioni sulla classifica data e genererà un callback asincrono con la mappa speciale async_load DS contenente i seguenti dati:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_leaderboard_info
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

Il codice di cui sopra controlla il DS Map restituito nel Social Asynchronous Event e se il suo "id" corrisponde alla costante da controllare, controlla poi per assicurarsi che la leaderboard inviata corrisponda alla chiamata originale della funzione prima di eseguire il looping attraverso la mappa e memorizzare tutti i diversi valori in una serie di array.
