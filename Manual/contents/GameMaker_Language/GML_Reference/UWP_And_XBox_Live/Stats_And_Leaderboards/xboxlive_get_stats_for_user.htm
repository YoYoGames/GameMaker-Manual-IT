xboxlive_get_stats_for_user

Questa funzione può essere usata per recuperare dati su statistiche specifiche da Xbox Live servers. Si fornisce l'ID utente come restituito dalla funzione xboxlive_get_user()poi il vostro ID di configurazione del servizio di gioco (come mostrato sulla console XDP), e infine le statistiche richieste. Potete richiedere fino a un massimo di 14 statistiche, ma questo non garantisce che otterrete 14 risultati, poiché c'è un limite alla lunghezza totale della richiesta e quindi il numero massimo di statistiche dipende dalla lunghezza dei nomi delle statistiche stesse.

La funzione restituirà 0 se la richiesta stat è stata inviata o -1 se c'è stato un errore, e il callback innescherà un evento asincrono di sistema. Questo evento avrà la speciale mappa DS async_load che dovrebbe poi essere analizzato per la seguente chiave:

  • "event_type" - dovrebbe contenere il string "stat_result" se l'evento è stato attivato da questa funzione

Se il tipo di evento si riferisce a questa funzione, allora ci sarà anche la seguente chiave aggiuntiva:

  • "user" - contiene l'ID utente per le statistiche richieste.

Ci può anche essere una serie extra di coppie chiave-valore, dove la chiave è il nome della stat richiesta e il valore è il valore di quella stat su server. Si noti che se una stat non è mai stata creata per questo utente (perché non ha lanciato gli eventi che la impostano), nessun valore può essere restituito per quella stat (ds_map_exists() può essere usato per controllare le statistiche nella mappa async_load ).

Se la richiesta fallisce perché la lunghezza della richiesta è troppo grande, ci dovrebbe essere un messaggio nell'output della console che indica il codice di errore:

xboxlive_get_stats_for_user - exception occurred getting results - 0x80190190

Quando questo accade, la mappa async di DS dovrebbe avere una chiave "succeeded" con un valore di "0", e in questo caso si dovrebbe tentare di recuperare meno statistiche in una singola chiamata.

Sintassi:

xboxlive_get_stats_for_user(user_id, serviceconfig_id, statname1, …);

Argomento Tipo Descrizione
user_id ID utente Xbox The user ID pointer.
serviceconfig_id Stringa The service config file ID
statname1 (2, 3, etc...) Corde The stat names to retrieve the information for.

Ritorna:

Real

Esempio:

var _uid = xboxlive_get_user(0);
var _configid = "00000000-0000-0000-0000-000000000000";
xboxlive_get_stats_for_user(xbuser, _configid, "GameProgress", "CurrentMode");

Il codice qui sopra ottiene l'ID dell'utente e poi lo usa per richiedere informazioni su statistiche specifiche per quell'utente.
