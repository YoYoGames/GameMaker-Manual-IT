xboxlive_stats_flush_user

Questa funzione può essere usata per scaricare i dati delle statistiche per un dato utente dal gestore delle statistiche, al live server, assicurando che server sia aggiornato con i valori attuali. Secondo la documentazione della XBox, gli sviluppatori dovrebbero fare attenzione a non chiamare questa funzione troppo spesso, poiché la XBox limiterà le richieste, e il sistema operativo della XBox provvederà comunque automaticamente al flush delle statistiche ogni 5 minuti circa.

NOTA: La rimozione dell'utente può restituire un errore se le statistiche che sono state impostate sull'utente non sono valide (come i nomi delle statistiche che contengono caratteri numerici non alfa).

Per utilizzare la funzione, si fornisce l'ID utente come restituito (per esempio) dalla funzione xboxlive_get_user()e un valore di proprità (che sarà 1 per alta priorità e 0 per bassa priorità). La funzione restituirà -1 se c'è stato un errore o l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo. La funzione genererà anche un callback che innescherà un evento asincrono di sistema. Questo evento avrà la speciale mappa DS async_load che dovrebbe poi essere analizzata per le seguenti chiavi:

  • "id" - Manterrà la costante achievement_stat_event
  • "eventname" - terrà il string "StatisticUpdateComplete"
  • "userid" - L'ID utente associato alla richiesta
  • "error" - 0 se successo, qualche altro valore se c'è stato un errore
  • "errormessage" - Un string con un messaggio di errore, se disponibile

Nota che le statistiche saranno scaricate automaticamente per te quando rimuovi un utente dal gestore delle statistiche. Vedere xboxlive_stats_remove_user() per maggiori informazioni.

Sintassi:

xboxlive_stats_flush_user(user_id, priority);

Argomento Tipo Descrizione
user_id ID utente Xbox The user ID (a pointer) to use
priority Real The prority value (0 or 1)

Ritorna:

Real

Esempio:

for(var i = 0; i < array_length(user_id); ++i;)
{
    xboxlive_stats_flush_user(user_id[i], 0);
}

Il codice di cui sopra loops attraverso gli ID degli account utente memorizzati e poi esegue il flush dei dati dello stat manager per ogni utente.
