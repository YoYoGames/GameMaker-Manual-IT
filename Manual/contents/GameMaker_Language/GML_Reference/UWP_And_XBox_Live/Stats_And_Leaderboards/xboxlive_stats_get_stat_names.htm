xboxlive_stats_get_stat_names

Questa funzione può essere usata per recuperare tutte le statistiche definite dal gestore delle statistiche per un dato utente. Si fornisce l'ID dell'utente come restituito (per esempio) dalla funzione xboxlive_get_user()e la funzione restituirà un array di strings contenente le statistiche per l'utente. Se si verifica un errore o l'utente non ha statistiche, l'array verrà comunque restituito ma avrà un numero di elementi pari a 0.

Sintassi:

xboxlive_stats_get_stat_names(user_id);

ArgomentoTipo Descrizione
user_id The user ID pointer to get the stat names of

Ritorna:

Esempio:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Il codice qui sopra recupera tutte le statistiche definite per il puntatore dell'ID utente memorizzato nella variabile data, e poi loops attraverso l'array e cancella la stat specificata dal gestore delle statistiche.
