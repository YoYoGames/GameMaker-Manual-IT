xboxlive_stats_set_stat_string

Questa funzione può essere usata per impostare una stat su string per l'ID utente dato. Si fornisce l'ID utente come restituito (per esempio) dalla funzione xboxlive_get_user()poi la stat string da impostare(se la stat string non esiste già, verrà creata una nuova stat e impostata al valore dato) e poi la stat string su cui impostare la stat. Notate che il nome della stat deve essere solo alfanumerico, senza simboli o spazi, ma il valore string che state impostando può avere spazi e caratteri speciali.

Quando si imposta la stat string, qualsiasi string precedente sarà sovrascritto, quindi sta a voi determinare se la stat string deve essere aggiornata o meno confrontando la stat corrente string con quella nuova prima di impostarla.

La funzione restituirà -1 se c'è stato un errore o l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo.

Sintassi:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argomento Tipo Descrizione
user_id ID utente Xbox The user ID pointer to set the stat for
stat Stringa The statistic to set (a string)
value Stringa The value to set the stat to (a string)

Ritorna:

Real

Esempio:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Il codice precedente imposta la stat "LevelName" al string memorizzato nella variabile globale data.
