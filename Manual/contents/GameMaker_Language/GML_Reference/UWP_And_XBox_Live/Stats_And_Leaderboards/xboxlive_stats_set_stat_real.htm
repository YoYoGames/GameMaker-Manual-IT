xboxlive_stats_set_stat_real

Questa funzione può essere usata per impostare il valore di una stat per l'ID utente dato. Si fornisce l'ID utente come restituito (per esempio) dalla funzione xboxlive_get_user()poi la stat string da impostare(se la stat string non esiste già, verrà creata una nuova stat e impostata al valore dato) e un valore (un reale) a cui impostare la stat. Si noti che il nome della stat deve essere solo alfanumerico, senza simboli o spazi.

Quando si imposta il valore della statistica, qualsiasi valore precedente sarà sovrascritto, quindi sta a voi determinare se il valore della statistica deve essere aggiornato o meno (cioè controllare che il punteggio più alto sia effettivamente il più alto) confrontando il valore attuale della statistica con quello nuovo prima di impostarlo.

La funzione restituirà -1 se c'è stato un errore o l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo.

Sintassi:

xboxlive_stats_set_stat_real(user_id, stat, value);

ArgomentoTipo Descrizione
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

Ritorna:

Esempio:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Il codice qui sopra controlla una variabile e se è true, allora controllerà il valore della statistica "PercentDone". Se questo valore è inferiore a 100, viene generato un valore e la statistica viene impostata su quel valore.
