xboxlive_stats_remove_user

Questa funzione può essere usata per rimuovere (unregister) un dato utente dal gestore delle statistiche, eseguendo un flush dei dati statistici al live server. Secondo la documentazione della XBox, il gioco non deve rimuovere l'utente dal gestore di statistiche, poiché il sistema operativo della XBox provvederà comunque a lavare periodicamente le statistiche. Si fornisce l'ID utente come restituito da (per esempio) la funzione xboxlive_get_user()e la funzione restituirà -1 se c'è stato un errore o se l'ID utente non è valido, o qualsiasi altro valore se la funzione è stata chiamata con successo.

NOTA: La rimozione dell'utente può restituire un errore se le statistiche che sono state impostate sull'utente non sono valide (come i nomi delle statistiche che contengono caratteri numerici non alfa).

La funzione genererà un callback che innescherà un evento asincrono di sistema. Questo evento avrà la speciale mappa DS async_load che dovrebbe poi essere analizzato per le seguenti chiavi:

  • "id" - Manterrà la costante achievement_stat_event
  • "eventname" - Conterrà il string "LocalUserRemoved"
  • "userid" - L'ID utente associato alla richiesta
  • "error" - 0 se successo, qualche altro valore se c'è stato un errore
  • "errormessage" - Un string con un messaggio di errore, se disponibile

Si noti che se si vuole scaricare i dati delle statistiche nel live server in qualsiasi momento senza rimuovere l'utente, si può usare la funzione xboxlive_stats_flush_user().

Sintassi:

xboxlive_stats_remove_user(user_id);

ArgomentoTipo Descrizione
user_id The user ID (a pointer) to remove

Ritorna:

Esempio:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

Il codice di cui sopra loop attraverso gli ID degli account utente memorizzati e rimuove (disregistra) ogni utente dal gestore delle statistiche.
