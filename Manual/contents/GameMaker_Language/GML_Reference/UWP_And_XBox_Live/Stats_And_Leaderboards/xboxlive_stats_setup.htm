xboxlive_stats_setup

Questa funzione deve essere chiamata prima di poter utilizzare qualsiasi altra funzione di statistiche Xbox Live, e semplicemente inizializza le librerie necessarie sul sistema. L'argomento "user_id" è l'ID utente grezzo restituito dalla funzione xboxlive_get_user()mentre "service_config" e "title_id" è l'ID unico per il vostro gioco sull'Xbox Live Dev Center.

Questa funzione è solo per l'uso con le statistiche Event-Based (2013).

Sintassi:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argomento Tipo Descrizione
user_id ID utente Xbox The ID pointer of the user to check.
service_config_id Stringa This is the config_id string for the game.
title_id Stringa The unique ID for your game on the Xbox Dev Center.

Ritorna:

N/A

Esempio:

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Il codice qui sopra inizializza il sistema di statistiche per l'ID utente dato.
