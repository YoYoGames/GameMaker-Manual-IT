è stato chiuso da un utente

Con questa funzione si può scoprire se il gioco è stato precedentemente chiuso dall'utente mentre giocava (cioè è stato terminato correttamente e non dal sistema). Se lo è stato allora la funzione restituirà true, altrimenti restituirà false.

Sintassi:

uwp_was_closed_by_user();

Ritorna:

Esempio:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

Il codice di cui sopra controlla se l'app è stata precedentemente terminata dall'utente, e se è così, imposta una variabile globale.

Indietro: UWP e XBox Live
