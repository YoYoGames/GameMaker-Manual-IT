tempo di prova della licenza rimanente

Questa funzione restituisce il numero approssimativo di secondi fino alla scadenza di una prova a tempo. Il valore restituito da questa funzione non ha senso se la prova non è a tempo o se il gioco non è in modalità prova.

Sintassi:

uwp_license_trial_time_remaining();

Ritorna:

Esempio:

if uwp_license_trial_version()
{
    var secs = uwp_license_trial_time_remaining() mod 60;
    var mins = uwp_license_trial_time_remaining() div 60;
    var hours = mins div 60;
    if secs < 10 secs = "0" + string(secs) else secs = string(secs);
    if mins > 60 mins -= (hours * 60);
    if mins < 10 mins = "0" + string(mins) else mins = string(mins);
    if hours > 9 hours = "9" else hours = string(hours);
    time_string = hours + ":" + mins + ":" + secs;
}

Il codice qui sopra controlla se il gioco viene eseguito con una licenza di prova e se lo è crea un string con il tempo fino alla scadenza della licenza.
