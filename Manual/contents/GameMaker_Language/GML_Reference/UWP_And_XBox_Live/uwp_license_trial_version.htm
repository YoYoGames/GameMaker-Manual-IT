versione di prova della licenza

Questa funzione può essere usata per controllare se il gioco è sotto una licenza di prova o no. Se lo è, la funzione restituisce true, o false altrimenti.

Sintassi:

uwp_license_trial_version();

Ritorna:

Esempio:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

Il codice qui sopra controlla se l'app è sotto una licenza di prova e, in caso affermativo, imposta una variabile globale.

Indietro: UWP e XBox Live
Il prossimo: uwp_license_trial_user
