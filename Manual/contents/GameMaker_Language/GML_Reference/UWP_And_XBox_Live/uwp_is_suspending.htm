è in sospeso

Questa funzione restituirà true per un singolo frame (e false altrimenti) in modo simile a os_is_paused(). Il gioco ha quindi un secondo per salvare lo stato di gioco prima di essere potenzialmente terminato dal sistema. Una volta che il gioco ha finito di salvare i dati dovrebbe chiamare uwp_suspend() per sospendere in modo pulito. Il gioco dovrebbe probabilmente anche entrare in uno stato di pausa in modo che se viene ripreso dal sistema (se non è stato completamente terminato) il giocatore non viene ributtato direttamente in una situazione di gioco.

Sintassi:

uwp_is_suspending();

Ritorna:

Esempio:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Il codice di cui sopra controlla se l'app sta andando in sospensione e se lo è chiama un script per salvare i dati di gioco prima di sospendere il gioco.

Indietro: UWP e XBox Live
Il prossimo: uwp_is_constrained
