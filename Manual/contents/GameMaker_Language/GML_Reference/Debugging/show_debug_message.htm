mostra messaggio di debug

Questa funzione crea un messaggio di debug personalizzato che viene mostrato nella finestra del compilatore all'indirizzo runtime. Notate che il messaggio può essere sia un string che un numero reale, ma se avete bisogno di entrambi allora il numero dovrà essere convertito prima in string usando la funzione string() (vedere l'esempio qui sotto) e che se il numero ha più di due cifre decimali allora si dovrebbe usare string_format() per mostrarle, poiché per default i decimali sono arrotondati alle due cifre decimali più vicine (quindi "1.2468" verrebbe mostrato come "1.25" nella finestra di output).

I messaggi di debug mostrati con questa funzione saranno mostrati nella finestra di output del compilatore nella parte inferiore di IDE così come nella vista grafica di debugger quando si esegue il gioco in modalità debug. Se volete vedere i messaggi solo in modalità Debug, allora probabilmente dovreste usare debug_event() invece.

Sintassi:

show_debug_message(string);

ArgomentoTipo Descrizione
string The custom debug message to show.

Ritorna:

Esempio:

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

Il codice di cui sopra controlla se esiste un'istanza e se non esiste, viene mostrato un messaggio di debug nella finestra di compilazione e il gioco viene terminato.
