get_stringa

Viene creata una finestra pop-up con un messaggio standard e un pulsante "Ok", che richiede all'utente di immettere un valore string. La funzione restituisce il valore immesso string, o il valore predefinito se non è stato immesso nulla.

NOTA Questa funzione può essere usata solo per il debug sul target Windows, ma è deprecata su tutti gli altri target.

Sintassi:

get_string(str, def);

Argomento Tipo Descrizione
str Stringa The string to show in the pop-up message.
def Stringa The default string in the text box.

Restituzione:

String

Esempio:

global.test_name = get_string("Test highscore name:", "Anonymous");

Il codice precedente chiederà all'utente di indicare un nome, che verrà memorizzato nella variabile globale "test_name". Se non viene inserito nulla e l'utente si limita a premere "Ok", verrà restituito il valore predefinito "Anonimo".
