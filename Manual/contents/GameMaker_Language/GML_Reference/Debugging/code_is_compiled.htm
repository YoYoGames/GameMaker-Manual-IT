codice_è_compilato

Questa funzione restituirà true se la piattaforma compila al di fuori della macchina virtuale, come per le piattaforme YYC e JS.

Sintassi:

code_is_compiled();

Ritorna:

Esempio:

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

Il codice qui sopra controlla se il gioco è stato compilato usando il compilatore YoYo e mostra un messaggio di debug se lo è stato.

Indietro: Debugging
Il prossimo: fps
