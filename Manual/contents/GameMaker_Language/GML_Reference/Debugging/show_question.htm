mostra_domanda

Questa funzione crea una casella di messaggio a comparsa con due pulsanti per "Sì" e "No". Restituisce true o false a seconda di quale dei due pulsanti viene premuto dall'utente.

NOTA Questa funzione può essere usata solo per il debug sul target Windows, ma è deprecata su tutti gli altri target.

Sintassi:

show_question(str);

Argomento Tipo Descrizione
str Stringa The string to show in the pop-up question.

Restituzione:

Boolean

Esempio:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

Il codice sopra riportato controlla il punteggio e se è superiore a 500, chiede all'utente se desidera continuare o meno e se viene cliccato il pulsante "sì" si passa a un altro room, ma se viene selezionato il pulsante "no" il gioco termina.

Indietro: Debug
Il prossimo: show_debug_message
