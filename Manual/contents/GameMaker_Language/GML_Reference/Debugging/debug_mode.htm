modalità di debug

Questa variabile di sola lettura restituisce true quando il gioco viene giocato in modalità debug e false quando viene giocato normalmente.

Sintassi:

debug_mode

Ritorna:

Esempio:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

Il codice qui sopra apre un file ini diverso a seconda del valore della variabile di sola lettura debug_mode.
