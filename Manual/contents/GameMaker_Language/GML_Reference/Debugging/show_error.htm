mostra_errore

Questa funzione mostrerà un string personalizzato come messaggio di errore.

IMPORTANTE Il secondo argomento esiste solo per la compatibilità all'indietro con i vecchi progetti e non avrà alcun effetto sul dialogo di errore mostrato.

NOTA Questa funzione è solo per il debug.

Sintassi:

show_error(str, abort);

Argomento Tipo Descrizione
str Stringa The string to show in the pop-up message.
abort Booleano UNUSED Se l'errore deve interrompere il gioco (true) o permettere al giocatore di ignorarlo (false).

Ritorna:

N/A

Esempio:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

Il codice qui sopra controllerà se il corrente room è rm_Dungeon, e se non lo è, mostrerà un messaggio di errore.
