mostra_messaggio

Questa funzione crea una casella di messaggio a comparsa che visualizza l'indirizzo string e un pulsante "Ok" per chiuderla.

NOTA Questa funzione può essere usata solo per il debug sul target Windows, ma è deprecata su tutti gli altri target.

Sintassi:

show_message(str);

Argomento Tipo Descrizione
str Stringa The string to show in the pop-up message.

Restituzione:

N/A

Esempio:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

Il codice precedente loop scorrerà i valori memorizzati nell'array "inv" e li aggiungerà alla variabile "tot" prima di mostrare un messaggio con il totale.
