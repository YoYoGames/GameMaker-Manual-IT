stringa_pos

Questa funzione restituisce la posizione del carattere di un'istanza di una sottostringa all'interno di un string, cercando dall'inizio del string (per cercare dalla fine, usare la funzione string_last_pos()). La funzione restituirà 0 se non viene trovata affatto, o la posizione del primo carattere della sottostringa se viene trovata. Tenete a mente che per il supporto legacy strings sono indicizzati da 1, quindi 1 è la prima posizione in string, non 0 come ci si potrebbe aspettare. Un uso per questo è per filtrare le parole che possono essere considerate offensive, o per trovare il posto corretto per inserire del testo in un altro string.

Sintassi:

string_pos(substr, str);

ArgomentoTipo Descrizione
substr The substring to look for in the string.
str The string.

Ritorna:

Esempio:

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

Il codice qui sopra cerca una virgola in string memorizzata nella variabile "text", e se la trova inserisce la sottostringa "name" in quella posizione.
