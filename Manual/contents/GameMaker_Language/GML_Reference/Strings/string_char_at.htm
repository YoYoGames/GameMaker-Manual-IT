stringa_char_at

Potete usare questa funzione per restituire un carattere specifico in una posizione specifica all'interno di un string, con l'indice che parte da 1 per il primo carattere. Se non viene trovato alcun carattere o il string è più corto del valore dell'indice dato, viene restituito un string vuoto "", tuttavia se l'indice dato è uguale o inferiore a 0, allora viene restituito il primo carattere del string.

Sintassi:

string_char_at(str, index);

ArgomentoTipo Descrizione
str The string to check.
index The position to get the character from.

Ritorna:

Esempio:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Questo imposterà str2 al settimo carattere ("H" conta come il 1°) in string str1, in questo caso "W".
