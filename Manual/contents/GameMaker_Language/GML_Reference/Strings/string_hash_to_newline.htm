string_hash_to_newline

Questa funzione restituisce un string dove il simbolo "#" è stato convertito in una nuova linea. Quindi un string formattato come:

"Hello#World"

Verrebbe stampato sullo schermo come:

Hello
World

IMPORTANTE! Questa funzione è fornita per la compatibilità di importazione tra GameMaker e le versioni precedenti del software e come tale non dovrebbe essere usata se non quando necessario per replicare funzionalità obsolete. Invece dovreste usare i caratteri di escape per segnare una nuova linea. Per ulteriori informazioni sulla compatibilità di importazione vedere qui.

Sintassi:

string_hash_to_newline(string);

ArgomentoTipo Descrizione
string The string to convert over multiple lines

Ritorna:

Esempio:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Il codice qui sopra converte il string con il simbolo dell'hash in un string diviso su due righe e lo stampa sullo schermo.
