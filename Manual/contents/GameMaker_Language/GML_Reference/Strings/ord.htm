ord

Questa funzione prende in input un singolo carattere string e restituisce il valore Unicode (UTF8) per quel carattere. Notate che quando viene usata con le funzioni keyboard_check* l'input string può essere solo un carattere in lunghezza e può essere solo un numero da 0 a 9 o un carattere romano maiuscolo dalla A alla Z.

Sintassi:

ord(string);

Argomento Tipo Descrizione
string Stringa The string with which to find the Unicode value.

Ritorna:

Real

Esempio:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Questo sposterà l'istanza chiamante di quattro pixel verso l'alto se il tasto W viene tenuto premuto.

Indietro: Corde
Il prossimo: real
