stringa

Con questa funzione potete trasformare qualsiasi numero reale in un string. Se il numero reale è un intero, sarà salvato senza decimali, altrimenti sarà salvato con due decimali. Se avete bisogno di più cifre decimali, allora usate la funzione string_format(). Notate anche che l'uso di questa funzione su una variabile che memorizza un array, una struttura dati o una struct convertirà il contenuto di queste variabili in un string che può poi essere inviato alla console o salvato in un file per debugging.

Sintassi:

string(val);

Argomento Tipo Descrizione
val Real The real number to be turned into a string.

Ritorna:

String

Esempio:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

Il codice qui sopra usa la funzione string per disegnare entrambi i numeri reali e strings insieme (draw accetta solo o un string o un reale, ma non entrambi).
