stringa_larghezza_ext

Questa funzione restituisce la larghezza massima (in pixel) dell'input string, tenendo conto della separazione delle linee e della larghezza dell'interruzione di linea (che è definita come il numero di pixel che string può occupare prima che venga inserita un'interruzione di linea). È molto comodo per calcolare le distanze tra gli elementi di testo in base alla larghezza massima di un string che è diviso su più righe come sarebbe disegnato con draw_text_ext() utilizzando l'attuale definizione di font. Separation e width possono essere impostati a -1 per ottenere la spaziatura predefinita.

Sintassi:

string_width_ext(string, sep, w);

ArgomentoTipo Descrizione
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Ritorna:

Esempio:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Il codice qui sopra otterrà la larghezza del dato string, tenendo conto della separazione delle linee e della larghezza dell'interruzione di linea, e poi disegnerà due linee di testo, usando la larghezza totale restituita string come separatore.
