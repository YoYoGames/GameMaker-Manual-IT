stringa_inferiore

Con questa funzione potete forzare un string a contenere solo caratteri minuscoli.

NOTA Questa funzione rileva solo le 26 lettere dell'alfabeto inglese dalla A alla Z.

Sintassi:

string_lower(string);

ArgomentoTipo Descrizione
string The string to convert to lowercase.

Ritorna:

Esempio:

str1 = "Hello World";
str2 = string_lower(str1);

Il codice qui sopra imposterà str2 su "hello world".
