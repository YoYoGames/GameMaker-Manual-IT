Appunti_ha_testo

Questa funzione restituisce true se gli appunti contengono testo o false se non ne contengono.

Sintassi:

clipboard_has_text();

Restituzione:

Boolean

Esempio:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Il codice precedente controlla la presenza di testo negli appunti e, se lo contiene, viene letto come string nella variabile "str". Infine, la clipboard viene cancellata impostando un string vuoto.

Indietro: Corde
Il prossimo: clipboard_get_text
