lettere_stringa

Con questa funzione puoi rimuovere tutti i caratteri che non sono classificati come lettere. Questo è utile per impedire alle persone di inserire caratteri indesiderati in una voce di testo (come inserire "#" per forzare una nuova riga).

NOTA Questa funzione rileva e restituisce solo le 26 lettere dell'alfabeto inglese dalla A alla Z (maiuscole o minuscole).

Sintassi:

string_letters(string);

ArgomentoTipo Descrizione
string The string to convert to letters.

Ritorna:

Esempio:

username = string_letters(username);

Il codice di cui sopra imposterà la variabile "username" per contenere solo i caratteri di lettera dell'originale string (id l'originale string era "MacSw33ny" il string restituito sarebbe "MacSwny").
