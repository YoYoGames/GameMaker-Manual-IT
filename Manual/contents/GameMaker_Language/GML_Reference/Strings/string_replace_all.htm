stringa_sostituisci_tutto

Potete usare questa funzione per analizzare un string alla ricerca di parti specifiche, che possono poi essere sostituite dal nuovo string che avete specificato in tutti i posti in cui si presentano.

Sintassi:

string_replace_all(str, substr, newstr);

ArgomentoTipo Descrizione
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Ritorna:

Esempio:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Questo imposterà str2 su str1, ma con le sue istanze di "x" sostituite da "l", con il risultato che str2 sarà "Hello World".
