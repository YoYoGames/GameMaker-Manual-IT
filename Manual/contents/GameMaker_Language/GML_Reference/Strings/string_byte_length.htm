lunghezza_stringa_byte

Questa funzione restituisce il numero di byte in un string. A causa del fatto che sono tenuti come UTF8, questo non sarà uguale al numero di caratteri in string.

Sintassi:

string_byte_length(string);

Argomento Tipo Descrizione
string Stringa The string to measure the number of bytes of.

Restituzione:

Real

Esempio:

bytesize = string_byte_length("Hello World");

Questo imposta bytesize al numero di byte di "Hello World".

Indietro: Corde
