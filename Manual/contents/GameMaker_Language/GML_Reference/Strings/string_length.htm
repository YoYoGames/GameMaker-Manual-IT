lunghezza_stringa

Questa funzione restituisce il numero di caratteri che compongono un dato string. Può essere utile per capire quando limitare la lunghezza dei caratteri di una voce di testo personalizzata (ad esempio, limitando il nome di un giocatore a 10 caratteri). Si ricordi che questo parametro è diverso da string_width() in quanto misura il numero di caratteri di string, non la sua larghezza in pixel disegnata sullo schermo.

Sintassi:

string_length(string);

Argomento Tipo Descrizione
string Stringa The string to measure the number of characters of.

Restituzione:

Real

Esempio:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

Questo controllerà se il nome string è maggiore di dieci caratteri e, in caso affermativo, copierà e utilizzerà solo i primi dieci caratteri.
