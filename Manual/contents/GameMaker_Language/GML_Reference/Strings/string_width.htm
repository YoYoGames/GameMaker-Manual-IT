larghezza_stringa

Questa funzione restituisce la larghezza (in pixel) dell'input string, tenendo conto di eventuali interruzioni di riga che il testo può avere. È molto utile per calcolare le distanze tra gli elementi di testo in base alla larghezza totale delle lettere che compongono la stringa come verrebbe disegnata con draw_text() usando il correntemente definito font.

Sintassi:

string_width(string);

ArgomentoTipo Descrizione
string The string to measure the width of.

Ritorna:

Esempio:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

Il codice qui sopra otterrà la larghezza del dato string e poi disegnerà due linee di testo, usando la larghezza restituita string come separatore.
