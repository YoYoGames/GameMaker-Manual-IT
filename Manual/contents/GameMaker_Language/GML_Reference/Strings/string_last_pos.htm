stringa_ultima_posizione

Questa funzione restituisce la posizione del carattere di un'istanza di una sottostringa all'interno di un string, cercando dalla fine del string all'inizio (quindi l'inverso di string_pos(). La funzione restituirà 0 se la ricerca string non viene trovata, o la posizione del primo carattere della ricerca string se viene trovata. Tenete a mente che per il supporto legacy strings sono indicizzati da 1, quindi 1 è la prima posizione in string, non 0 come ci si potrebbe aspettare.

Sintassi:

string_last_pos(substr, str);

ArgomentoTipo Descrizione
substr The substring to look for in the string.
str The string.

Ritorna:

Esempio:

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

Il codice qui sopra cerca nel sito string memorizzato nella variabile "text" una virgola, iniziando la ricerca dalla fine di string. Se ne trova una, inserisce la sottostringa "name" in quella posizione.
