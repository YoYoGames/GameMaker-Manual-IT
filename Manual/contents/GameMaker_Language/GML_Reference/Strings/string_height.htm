altezza_stringa

Questa funzione restituisce l'altezza (in pixel) dell'input string, tenendo conto della separazione delle linee e di qualsiasi interruzione di linea che il testo può avere. È molto utile per calcolare le distanze tra gli elementi di testo in base alla più alta delle lettere che compongono il sito string come verrebbe disegnato con draw_text() usando l'attuale definizione di font.

Sintassi:

string_height(string);

ArgomentoTipo Descrizione
string The string to measure the height of.

Ritorna:

Esempio:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

Il codice qui sopra otterrà l'altezza del dato string e poi disegnerà due linee di testo, usando l'altezza restituita string come separatore.
