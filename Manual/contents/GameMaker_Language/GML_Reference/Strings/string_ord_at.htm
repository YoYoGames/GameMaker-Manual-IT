stringa_ord_at

Potete usare questa funzione per restituire un codice di carattere specifico in una posizione specifica all'interno di un string, con l'indice che parte da 1 per il primo carattere. Se non viene trovato alcun carattere o il string è più corto del valore dato all'indice, viene restituito -1.

Sintassi:

string_ord_at(str, index);

ArgomentoTipo Descrizione
str The string to check.
index The position to get the character code from.

Ritorna:

Esempio:

str = "Hello World";
char_code = string_ord_at(str, 7);

Questo otterrà il codice del carattere per il settimo carattere (dove "H" conta come primo) in string "str" e lo memorizzerà nella variabile "char_code".
