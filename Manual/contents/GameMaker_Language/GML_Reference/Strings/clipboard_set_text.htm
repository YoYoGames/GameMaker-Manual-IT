Appunti_set_testo

Questa funzione imposta gli appunti in modo che contengano l'indirizzo string definito. È possibile impostarla su un string "" vuoto per svuotare effettivamente gli appunti dal testo.

Sintassi:

clipboard_set_text(string);

Argomento Tipo Descrizione
string Stringa The text (as a string) to set the clipboard to hold.

Restituzione:

N/A

Esempio:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Il codice precedente controlla la presenza di testo negli appunti e, se lo contiene, viene letto come string nella variabile "str". Infine, la clipboard viene cancellata impostando un string vuoto.
