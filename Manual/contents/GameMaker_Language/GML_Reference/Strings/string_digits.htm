cifre della stringa

Potete usare questa funzione per analizzare un dato string e ottenere qualsiasi numero da esso. Per esempio, diciamo che avete questo testo - "Ho 81 anni". Con questa funzione otterreste un ritorno string di "81".

Sintassi:

string_digits(string);

ArgomentoTipo Descrizione
string The string to get the digits from.

Ritorna:

Esempio:

var t_str = string_digits(input_str);
age = real(t_str);

Il codice di cui sopra prenderà l'input string, lo spoglierà di tutti i caratteri diversi dai numeri e poi imposterà la variabile age per contenere il valore numerico reale del ritorno string (così, se l'input string fosse - per esempio - "Ho 18 anni", la funzione restituirebbe "18").
