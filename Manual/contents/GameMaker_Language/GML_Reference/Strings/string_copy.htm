copia_stringa

Con questa funzione potete facilmente selezionare un certo numero di caratteri da un string da copiare in un altro. Il primo carattere in un string è sempre indicizzato come 1 e non 0 come ci si potrebbe aspettare, quindi per copiare (per esempio) i primi cinque caratteri di string si avrebbe string_copy(str, 1, 5). Un ulteriore esempio è fornito di seguito.

Sintassi:

string_copy(str, index, count);

ArgomentoTipo Descrizione
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Ritorna:

Esempio:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

Il codice di cui sopra permetterà al giocatore di inserire un string attraverso la tastiera che viene poi memorizzato nella variabile name. Questa variabile viene poi controllata per vedere se supera la lunghezza massima di 15 caratteri e se lo fa, la keyboard_string viene sostituita da una copia di 15 caratteri di name string. Questo limita effettivamente l'input del giocatore a 15 caratteri.
