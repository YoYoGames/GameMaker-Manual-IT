ripetizione della stringa

Questa funzione restituisce semplicemente lo stesso string ripetuto un dato numero di volte su se stesso.

Sintassi:

string_repeat(str, count);

ArgomentoTipo Descrizione
str The string to repeat.
count The number of times to repeat the string.

Ritorna:

Esempio:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Il codice qui sopra imposterà str2 su "Hello WorldHello WorldHello World".
