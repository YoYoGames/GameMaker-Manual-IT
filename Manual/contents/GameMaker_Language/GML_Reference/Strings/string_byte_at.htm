stringa_byte_at

Restituisce il valore grezzo del byte come valore reale in una data posizione nel dato string.

Sintassi:

string_byte_at(str, index);

ArgomentoTipo Descrizione
str The string to check.
index The position to get the byte from.

Ritorna:

Esempio:

newbyte = string_byte_at("Hello World", 5);

Questo imposterà newbyte al valore grezzo del byte della sesta lettera di "Hello World".
