stringa_set_byte_at

Questa funzione imposta un byte direttamente in un string (basato sul formato UTF8) e restituisce una copia di string con le modifiche.

NOTA Questa funzione è incredibilmente lenta, quindi considerate attentamente se è necessaria e dove la usate.

Sintassi:

string_set_byte_at(str, pos, byte);

ArgomentoTipo Descrizione
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

Ritorna:

Esempio:

str = string_set_byte_at("hello", 2, 97);

Il codice di cui sopra cambierebbe il valore del byte della seconda lettera in string, e quindi imposterebbe la variabile str per contenere "hallo".
