numero di stringhe

Questa funzione restituirà la quantità di volte che la data sottostringa appare all'interno di uno specifico string. In questo modo potete controllare quante volte una singola lettera o una frase è ripetuta in una sezione di testo memorizzata.

Sintassi:

string_count(substr, str);

ArgomentoTipo Descrizione
substr The substring to check the string for.
str The string to check.

Ritorna:

Esempio:

str1 = "Hello World";
ocount = string_count( "o", str1 );

Questo imposterà la variabile ocount al numero di "o" in str1, in questo caso 2.
