cifre della stringa

Questa funzione restituisce una copia di un dato string con tutto tranne le lettere e le cifre rimosse, il che significa che può essere usato per rimuovere qualsiasi carattere indesiderato (come "#" o "?") da, per esempio, un nome di login o una password.

NOTA Questa funzione restituisce solo le 26 lettere dell'alfabeto inglese dalla A alla Z (maiuscole o minuscole) e i numeri da 0 a 9.

Sintassi:

string_lettersdigits(string);

ArgomentoTipo Descrizione
string The string to convert to letters and digits.

Ritorna:

Esempio:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}
