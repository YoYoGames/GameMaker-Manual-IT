cancella stringa

Questa funzione è usata per rimuovere una parte del dato string. Si fornisce l'input string, la posizione iniziale all'interno di quel string da cui rimuovere i caratteri (l'indice inizia da 1) e il numero di caratteri da rimuovere. La funzione restituisce un nuovo string senza quella parte.

Sintassi:

string_delete(str, index, count);

ArgomentoTipo Descrizione
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Ritorna:

String

Esempio:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Questo imposterà str2 su "Hello World", poiché rimuove la "o" extra da "Hello" (che è presente in posizione 5).
