chr

Questa funzione restituisce un string contenente il carattere che si riferisce al codice Unicode di input per la visualizzazione. Questo carattere dipende dalla pagina di codice del set di caratteri del disegno corrente fonts e se non è impostato font, userà la pagina di codice predefinita per la macchina.

Sintassi:

chr(val);

ArgomentoTipo Descrizione
val The Unicode code value to get the string from

Ritorna:

Esempio:

mystring = chr(53) + chr(48);

Questo imposterebbe mystring a "50" (un string, non un intero) poiché chr(53) è "5" e chr(48) è "0".
