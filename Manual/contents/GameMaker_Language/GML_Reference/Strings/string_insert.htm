inserimento stringa

Con questa funzione potete creare un nuovo string composto da due strings, dove uno è stato inserito nell'altro in una determinata posizione. Può essere utile, per esempio, per aggiungere un nome utente in un testo predefinito e far sentire il giocatore del vostro gioco più coinvolto nell'azione. Tenete a mente, quando calcolate la posizione in cui inserire, che strings è indicizzato a partire da 1, quindi il primo carattere in un string è 1 e non 0 come ci si potrebbe aspettare.

Sintassi:

string_insert(substr, str, index);

ArgomentoTipo Descrizione
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

Ritorna:

Esempio:

str2 = string_insert(username, "Hello, , how are you?", 8);

Questo inserirà il string nella variabile "username" nella frase data con il risultato string che assomiglia a questo: "Hello, NAME, how are you?".

Indietro: Corde
Il prossimo: string_length
