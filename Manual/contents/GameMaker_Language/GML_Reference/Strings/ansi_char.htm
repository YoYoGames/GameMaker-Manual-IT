ansi_char

Questa funzione restituisce un string contenente il carattere con il valore grezzo BYTE impostato. Questo non sarà, e non dovrebbe essere, visualizzato, ma salverà correttamente su disco per l'uso nella codifica.

Sintassi:

ansi_char(val);

ArgomentoTipo Descrizione
val The raw byte value.

Ritorna:

(Single character)

Esempio:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

Il codice qui sopra crea un string da dati byte grezzi e li scrive in un file (precedentemente aperto).
