sostituzione di stringhe

Potete usare questa funzione per analizzare un string cercando una parte specifica, che può poi essere sostituita dal nuovo string che avete specificato.

Sintassi:

string_replace(str, substr, newstr);

ArgomentoTipo Descrizione
str The string to be copied.
substr The substring within the string to be replaced.
newstr The new substring to replace the previous one.

Ritorna:

Esempio:

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Questo imposterà str2 su str1, ma con la sua istanza di "Earth" sostituita da "World", con il risultato che str2 sarà "Hello World".
