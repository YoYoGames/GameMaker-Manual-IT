altezza_stringa_ext

Questa funzione restituisce l'altezza (in pixel) dell'input string, tenendo conto della separazione delle linee e della larghezza dell'interruzione di linea (che è definita come il numero di pixel che string può occupare prima che venga inserita un'interruzione di linea). È molto utile per calcolare le distanze tra gli elementi di testo in base alla più alta delle lettere che compongono string, nonché la lunghezza effettiva di string stesso quando è distribuito su più righe, come verrebbe disegnato con draw_text_ext() usando l'attuale definizione di font. Separazione e larghezza possono essere impostate a -1 per ottenere la spaziatura predefinita.

Sintassi:

string_height_ext(string, sep, w);

ArgomentoTipo Descrizione
string The string to measure the height of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Ritorna:

Esempio:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

Il codice qui sopra otterrà l'altezza del dato string, tenendo conto della separazione delle linee e della larghezza dell'interruzione di linea, e poi disegnerà due linee di testo, usando l'altezza totale restituita string come separatore.
