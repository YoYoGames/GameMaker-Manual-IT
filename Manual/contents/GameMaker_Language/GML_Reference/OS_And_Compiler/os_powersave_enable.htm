salva-persona_automatico

Con questa funzione potete attivare o disattivare le funzioni di risparmio energetico del dispositivo. Questo è importante perché alcuni giochi (per esempio quelli che usano le funzioni di inclinazione) potrebbero non generare eventi che il sistema operativo possa interpretare come input dell'utente e quindi spegnere lo schermo o uscire dal gioco. Impostando questa funzione su false è possibile disabilitare le funzioni di risparmio energetico e garantire che lo schermo (e il gioco) siano sempre funzionanti.

NOTA: Questo è limitato agli obiettivi iOS e Android.

Sintassi:

os_powersave_enable(flag)

ArgomentoTipo Descrizione
flag Set to true o false per abilitare o disabilitare le funzioni di risparmio energetico.

Ritorna:

Esempio:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

Il codice di cui sopra controlla il tipo di sistema operativo e se è o Android o un iOS allora le funzioni di risparmio energetico sono disattivate.
