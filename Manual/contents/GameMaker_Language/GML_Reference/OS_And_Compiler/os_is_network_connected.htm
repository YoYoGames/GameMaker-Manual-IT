è collegato alla rete

Con questa funzione potete controllare e vedere se il vostro dispositivo ha attualmente una connessione internet e restituirà true se ce l'ha, o false se non ce l'ha, e a seconda del sistema operativo, potrebbe tentare di effettuare una connessione prima di restituire un valore.

La funzione ha un argomento opzionale per abilitare/disabilitare il tentativo di stabilire una connessione quando viene chiamata, e questa opzione è attualmente solo per il target Nintendo Switch. Quando è impostata su true la funzione tenterà di stabilire una connessione, e quando è impostata su false sarà semplicemente limitata a controllare la connessione. Notate che il tentativo di stabilire una connessione può causare un blocco del sistema operativo Switch per alcuni secondi.

NOTA Questa funzione controlla l'API interna del dispositivo che controlla le connessioni e quindi può restituire true se c'è una connessione bluetooth, una connessione Wi-Fi, o anche solo una normale connessione di rete che permette l'accesso a internet.

Sintassi:

os_is_network_connected([attempt_connection]);

Argomento Tipo Descrizione
attempt_connection Booleano OPTIONAL Solo per l'obiettivo Nintendo Switch. Impostare su true per tentare una connessione di livello OS quando viene chiamata.

Ritorna:

Boolean

Esempio:

if os_is_network_connected()
{
    global.connected = true;
}

Il codice di cui sopra controlla se il dispositivo ha una connessione a internet e se è così imposta una variabile globale.

Indietro: Sistema operativo e compilatore
