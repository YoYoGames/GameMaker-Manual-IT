esterno_libero

Questa funzione libera la memoria associata a dll o dylib con il nome dato. Questo dovrebbe essere fatto ogni volta che il file in questione non è più necessario nel gioco, normalmente (per esempio) in un evento di fine gioco.

Sintassi:

external_free(id);

Argomento Tipo Descrizione
id Funzione esterna The name of the dll o dylib che si vuole liberare

Ritorna:

N/A

Esempio:

external_free("MyDLL.dll");

Il codice dell'esempio precedente libererà la memoria associata alla dll data.
