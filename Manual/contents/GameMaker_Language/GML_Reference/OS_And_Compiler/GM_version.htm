GM_versione

Questa costante tiene il numero di versione come definito nelle Opzioni di gioco per ogni piattaforma di destinazione. Il valore è memorizzato come string.

Sintassi:

GM_version;

Ritorna:

Esempio:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

Il codice qui sopra prende la data GM e le costanti di versione e le disegna sullo schermo.

Indietro: Sistema operativo e compilatore
Il prossimo: GM_runtime_version
