controllare le autorizzazioni

Con questa funzione potete richiedere un permesso specifico. Si fornisce il permesso da richiedere come string usando il formato "android.permission.<permission>", quindi per richiedere il permesso RECORD_AUDIO (per esempio) si chiama

os_request_permission("android.permission.RECORD_AUDIO");

La funzione restituirà un evento asincrono di sistema in cui la mappa DS incorporata conterrà le seguenti coppie chiave/valore async_load DS conterrà le seguenti coppie chiave/valore:

  • "tipo": Questo è il tipo di evento che è stato attivato e il valore sarà il string "permission_request_result".
  • "<permesso>": Questo sarà il permesso richiesto dalla funzione come un string, che conterrà una delle seguenti costanti:
Costante Descrizione
os_permission_granted Questo indica che il permesso è stato concesso
os_permission_denied Questo indica che il permesso non è stato concesso

Vale la pena notare che i seguenti permessi sono supportati nativamente da GameMaker, ma sono considerati "pericolosi" da Google e come tali devono essere esplicitamente richiesti (si noti anche che alcuni permessi possono essere richiesti utilizzando le Opzioni di gioco Android senza la necessità di questa funzione):

  • android.permission.WRITE_EXTERNAL_STORAGE
  • android.permission.READ_PHONE_STATE
  • android.permission.RECORD_AUDIO

Per ulteriori informazioni sui permessi delle app, si prega di consultare la documentazione di Android.

IMPORTANTE! Questa funzione è solo per il target Android.

Sintassi:

os_request_permission(permission)

Argomento Tipo Descrizione
permission Stringa The permission to request (a string)

Ritorna:

N/A

Esempio:

if (os_type == os_android)
{
    if (os_check_permission("android.permission.INTERNET") == os_permission_denied)
    {
        os_request_permission("android.permission.INTERNET");
    }
}

Il codice di cui sopra controlla il tipo di sistema operativo e se è Android, esegue un controllo sui permessi e se il permesso "INTERNET" non è ancora stato concesso, lo richiede.
