os_device

Questa variabile di sola lettura contiene uno dei vari valori costanti elencati di seguito per dirvi su quale dispositivo state attualmente eseguendo il gioco. Notate che questa variabile è deprecata in favore della funzione os_get_info() che restituisce informazioni più accurate sul dispositivo che esegue il gioco.

Sintassi:

os_device

Ritorna:

Device Type Constant

Costante del tipo di dispositivo
Costante Descrizione
device_ios_ipad iPad
device_ios_ipad_retina iPad più recente con display Retina di dimensioni 2048 x 1536
device_ios_iphone6 iPhone6 con dimensioni del display 1334 x 750
device_ios_iphone6plus iPhone 6 più grande con display 1920 x 1080
device_ios_iphone5 iPhone5 con dimensioni del display 640 x 1136)
device_ios_iphone Vecchio iPhone/iPod Touch (schermo 480 x 320) o telefono Android
device_ios_iphone_retina iPhone/iPod Touch più recente con display Retina di 960 x 640
device_emulator Il dispositivo è in realtà un emulatore (Windows Phone o Android)
device_tablet Tablet Android
device_ios_unknown Sconosciuto o non iOS

Esempio:

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

Il codice qui sopra controlla se il gioco è in esecuzione in un browser o meno e imposta una variabile globale su un valore che dipende dal risultato del controllo.
