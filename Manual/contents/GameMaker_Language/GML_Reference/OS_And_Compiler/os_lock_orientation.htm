orientamento del blocco di sicurezza

Con questa funzione puoi "bloccare" il tuo dispositivo all'orientamento corrente fino al momento in cui lo "liberi" per permettere di nuovo tutti gli orientamenti abilitati dalle Opzioni di gioco per quella piattaforma di destinazione. Nota che probabilmente vorrai confermare che l'orientamento è quello desiderato prima di bloccarlo.

Sintassi:

os_lock_orientation(flag)

ArgomentoTipo Descrizione
flag Set to true o false per abilitare o disabilitare il blocco dell'orientamento.

Ritorna:

Esempio:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

Il codice di cui sopra controlla il tipo di SO e se è o Android o un iOS allora il bloccaggio dell'orientamento è segnalato come true.
