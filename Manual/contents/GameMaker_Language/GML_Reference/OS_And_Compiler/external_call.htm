chiamata_esterna

Se avete creato una chiamata a una funzione esterna a dll o dylib usando external_define()potete usare questa funzione per chiamarla. Fornite il nome della funzione definita in precedenza e ciascuno degli argomenti che richiede (ogni argomento deve essere del tipo corretto, reale o string) e la funzione restituisce il risultato della chiamata esterna.

Sintassi:

external_call(id, args[0...15]);

Argomento Tipo Descrizione
id Funzione esterna The name of the function that you want to call
args[0...10] Reale o stringa The different arguments that you want to pass through to the external function

Ritorna:

Variable (the type of value returned will depend on the defined function)

Esempio:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Il codice dell'esempio precedente chiama una funzione esterna definita in precedenza e memorizza il valore restituito in una variabile locale.
