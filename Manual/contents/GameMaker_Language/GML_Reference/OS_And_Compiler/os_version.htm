os_version

Questa variabile vi dirà il numero di versione del sistema operativo che sta eseguendo il vostro gioco. Per esempio, se lo state eseguendo su Windows 10, os_version sarà uguale a 655360.

NOTA: Se avete bisogno di ulteriori informazioni sul sistema operativo Windows potete usare il environment_get_variable().

Sintassi:

os_version

Ritorna:

Esempio:

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

Il codice di cui sopra controlla il tipo di os e il numero di versione e sono entrambi corretti, quindi la variabile globale è impostata su 1.
