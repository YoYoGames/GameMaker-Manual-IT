parametro_stringa

I parametri della riga di comando sono quei comandi extra che potete aggiungere a un exe per cambiare il modo in cui il programma viene eseguito e con questa funzione potete ottenere il parametro della riga di comando scelto come string. Potete trovare il numero di parametri per il gioco corrente usando la funzione parameter_count(), dove il primo parametro ha indice 1 e l'ultimo ha l'indice restituito dalla funzione (un valore di 0 è speciale in quanto è il nome del file dell'eseguibile del gioco, incluso il path). Va notato che questa funzione funziona per la piattaforma HTML5, recuperando i parametri url.

Sintassi:

parameter_string(n);

Ritorna:

Esempio:

var p_num;
p_num = parameter_count();
if p_num > 0
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

Il codice di cui sopra otterrà il numero di parametri della linea di comando, e se ce n'è 1 o più, loop e li memorizzerà come strings in un array.
