os_get_config

Questa funzione restituisce il nome (come string) della configurazione attualmente selezionata per il vostro gioco. Per maggiori informazioni sulle configurazioni, vedi la sezione Configurazioni.

Sintassi:

os_get_config()

Ritorna:

Esempio:

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

Il codice qui sopra controllerà per vedere quale configurazione è in uso e se è quella chiamata "Free_Version", gli annunci saranno abilitati nel gioco.
