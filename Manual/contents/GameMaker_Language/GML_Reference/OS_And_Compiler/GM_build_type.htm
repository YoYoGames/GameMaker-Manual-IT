Tipo_di_costruzione_GM

Questa costante di string contiene il tipo di compilazione: "exe" per un eseguibile creato o "run" per un'esecuzione di prova avviata da IDE.

Sintassi:

GM_build_type;

Restituzione:

String

Esempio:

draw_text(32, 32, GM_build_type);

Il codice sopra riportato ricava il tipo di compilazione dell'eseguibile in esecuzione.

Indietro: Sistema operativo e compilatore
Il prossimo: GM_version
