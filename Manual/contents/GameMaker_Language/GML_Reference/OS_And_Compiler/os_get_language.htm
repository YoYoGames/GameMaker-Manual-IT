os_get_language

Questa funzione restituisce un string con il Codice Lingua di due lettere per il sistema operativo che sta eseguendo il gioco, come stabilito dallo standard ISO639. Se l'informazione non è disponibile, terrà semplicemente un string vuoto "", o "en" per la lingua "inglese". Notate che alcune lingue hanno anche un codice regionale pertinente, quindi per distinguere tra diverse regioni dello stesso paese usate la funzione os_get_region().

La tabella seguente mostra un esempio di alcuni dei principali codici linguistici a due lettere definiti da ISO 639:

Lingua Codice
Arabo ar
Cinese zh
Danese da
Inglese it
Francese fr
Tedesco de
Greco el
Italiano it
Giapponese ja
Norvegese no
Polacco pl
Portoghese pt
Russo ru
Spagnolo es
Svedese sv

NOTA: questo non è il codice del paese della località che viene restituito, ma il codice della lingua del sistema operativo.

Sintassi:

os_get_language()

Ritorna:

Esempio:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

Il codice qui sopra controlla la lingua del sistema operativo e apre un diverso file *.ini a seconda del valore restituito.
