Versione_runtime_gm

Questa costante contiene il numero di versione di runtime definito nelle Preferenze come runtime utilizzato per costruire il progetto. Il valore è memorizzato come string.

Sintassi:

GM_runtime_version;

Restituzione:

String

Esempio:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

Il codice precedente prende le costanti GM date, version e runtime e le disegna sullo schermo.
