os_get_region

Questa funzione restituisce un string con il codice regionale di due o tre lettere per il sistema operativo che sta eseguendo il gioco, come stabilito dallo standard ISO3166-1. Se l'informazione non è disponibile, terrà semplicemente un string vuoto "".

NOTA: Questo non è il codice regionale della località che viene restituito, ma il codice della lingua regionale del sistema operativo.

Sintassi:

os_get_region()

Ritorna:

Esempio:

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

Il codice di cui sopra controlla la lingua del sistema operativo e, se è cinese, controlla la regione del sistema operativo, aprendo un diverso file *.ini a seconda dei valori restituiti.
