ambiente_get_variabile

Questa funzione restituisce il valore (un string) della variabile d'ambiente con il nome dato (anche un string). Potete ottenere le variabili d'ambiente disponibili su macOS e Ubuntu (Linux) digitando "env" nell'applicazione terminale, e per informazioni sulle variabili d'ambiente Windows, se state usando il prompt dei comandi allora digitate "echo %PATH%", e usando PowerShell è "ls env". Si noti che sia su macOS che su Ubuntu (Linux) la variabile d'ambiente "HOME" restituirà la variabile d'ambiente "~/" path che mappa a "/Users/<username>" su macOS e "/home/<username>" su Ubuntu (Linux).

NOTA: Questa funzione è solo per Windows, macOS e Ubuntu (Linux).

Sintassi:

environment_get_variable(name);

ArgomentoTipo Descrizione
name The name (a string) of the environment variable to check.

Ritorna:

Esempio:

e_str = environment_get_variable("APPDATA");

Il codice di cui sopra restituirà l'intero path per la variabile d'ambiente "%appdata%", che normalmente è "C:\Users\{username}\AppData\Roaming".
