lunghezza_dir_x

Questa funzione è usata per ottenere la componente x di una posizione "len" pixel dal punto di partenza e nella direzione "dir". Se immaginate un cerchio intorno alla vostra istanza, e poi immaginate un punto in qualsiasi punto di quel cerchio, per muovervi verso quel punto dobbiamo spostare object di tanti pixel in quella direzione... quindi questa funzione (se usata con lengthdir_y()) ottiene la posizione di quel punto sul cerchio per essere usata nel codice dall'istanza. Vedere l'immagine qui sotto per i dettagli:

Sintassi:

lengthdir_x(len, dir);

ArgomentoTipo Descrizione
len The length away of the point to return.
dir The direction of the point to return.

Ritorna:

Esempio:

var _xx = x + lengthdir_x(64, image_angle);
var _yy = y + lengthdir_y(64, image_angle);
instance_create_layer(_xx, _yy, "Bullets", obj_bullet);

Questo creerà un'istanza di proiettile a (_xx, _yy), che sarà a 64 pixel dall'istanza madre nella direzione dell'angolo dell'immagine.
