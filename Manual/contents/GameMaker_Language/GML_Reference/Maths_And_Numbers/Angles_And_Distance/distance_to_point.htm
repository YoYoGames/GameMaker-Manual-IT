distanza dal punto

Questa funzione calcola la distanza dal bordo del riquadro di delimitazione dell'istanza chiamante alla posizione x/y specificata in room, con il valore di ritorno in pixel. Si noti che se l'istanza chiamante object non ha sprite o nessuna maschera definita, i risultati non saranno corretti.

Sintassi:

distance_to_point(x, y);

ArgomentoTipo Descrizione
x The x position to check.
y The y position to check.

Ritorna:

Esempio:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    canshoot = true;
}

Il codice di cui sopra controllerà la distanza dalla posizione x/y del giocatore object e se è inferiore al valore memorizzato nella variabile "range" la variabile "canshoot" è impostata su true.
