distanza dall'oggetto

Questa funzione calcola la distanza dal bordo del rettangolo di delimitazione dell'istanza chiamante al bordo più vicino dell'istanza più vicina di object specificata. L'indirizzo object può essere un indice object o un ID di un'istanza specifica così come parola chiave l'altro, e la distanza viene restituita in pixel. Si noti che se uno dei due objects non ha nessun sprite o nessuna maschera definita, i risultati non saranno corretti.

Sintassi:

distance_to_object(obj);

Argomento Tipo Descrizione
obj ID dell'asset dell'oggetto o dell'istanza The object to check for.

Ritorna:

Real

Esempio:

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

Il codice di cui sopra controllerà la distanza dal giocatore object e se è inferiore al valore memorizzato nella variabile "range" la variabile "canshoot" è impostata su true.
