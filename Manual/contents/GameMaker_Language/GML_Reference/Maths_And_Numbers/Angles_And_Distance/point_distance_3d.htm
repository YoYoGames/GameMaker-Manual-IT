punto_distanza_3d

Questa funzione prende le componenti fornite del vettore e restituisce la lunghezza (distanza) del vettore. Funziona esattamente allo stesso modo di point_distance() ma con l'aggiunta di fattorizzare il valore z (profondità) per l'uso nello spazio 3D.

Sintassi:

point_distance_3d(x1, y1, z1, x2, y2, z2);

ArgomentoTipo Descrizione
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
z1 The z coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector
z2 The z coordinate of the second component of the vector

Ritorna:

Esempio:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
    ex = inst.x;
    ey = inst.y;
    ez = inst.z;
    if point_distance_3d(x, y, z, ex, ey, ez) < 200
    {
        instance_create_layer(x, y, "Bullets", obj_Missile)
    }
}

Il codice di cui sopra otterrà le coordinate x, y e z del nemico più vicino e poi le userà per controllare la distanza (lunghezza) del vettore formato da esse e le coordinate del giocatore. Se il valore è inferiore a 200, il giocatore object creerà un'istanza di "obj_Missile".
