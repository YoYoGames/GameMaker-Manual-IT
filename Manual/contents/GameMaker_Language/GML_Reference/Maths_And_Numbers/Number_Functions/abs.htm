abs

Questa funzione restituisce il valore assoluto dell'argomento in ingresso, quindi se è un valore positivo allora rimarrà lo stesso, ma se è negativo sarà moltiplicato per -1 per renderlo positivo.

Sintassi:

abs(val)

ArgomentoTipo Descrizione
val The number to turn absolute.

Ritorna:

Esempio:

x += abs (x - mouse_x)

Questo aggiungerà una quantità pari al valore assoluto della posizione x corrente dell'istanza e la posizione x del mouse.
