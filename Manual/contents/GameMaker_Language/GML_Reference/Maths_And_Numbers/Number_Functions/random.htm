casuale

Questa funzione è buona per le probabilità in cui non è necessario restituire un intero (numero intero). Per esempio, random(100) restituirà un valore da 0 a 99, ma quel valore può essere 22,56473! Potete anche usare numeri reali e non interi in questa funzione come questa - random(0.5), che restituirà un valore tra 0 e 0,499999999.

NOTA: Questa funzione restituirà lo stesso valore ogni volta che il gioco viene eseguito di nuovo a causa del fatto che GameMaker genera lo stesso seme casuale iniziale ogni volta per rendere debugging codice un compito molto più facile. Per evitare questo comportamento usate randomise() all'inizio del vostro gioco.

Sintassi:

random(n);

ArgomentoTipo Descrizione
n The upper range from which the random number will be selected.

Ritorna:

Esempio:

if (random(10) >= 9)
{
    score += 100;
}

Questo produrrà circa una possibilità su dieci di aggiungere 100 al punteggio.
