potenza

Questo restituirà il valore di un numero moltiplicato per se stesso un numero "n" di volte. Per esempio, power(5,3) moltiplicherà 5 per se stesso 3 volte e restituirà 125, che è come dire 5*5*5=125. Si noti che il valore "x" (il numero da cambiare) non può essere un valore negativo.

Sintassi:

power(x, n);

ArgomentoTipo Descrizione
x The number to change.
n How many times to multiply x by itself.

Ritorna:

Esempio:

score += power(dmg, 3);

Questo aggiungerà il valore tenuto nella variabile "dmg" alla potenza di 3 alla variabile punteggio.
