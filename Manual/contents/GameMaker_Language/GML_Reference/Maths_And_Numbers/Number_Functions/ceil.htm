ceil

Questa funzione prende qualsiasi numero reale e lo arrotonda all'intero più vicino. Bisogna fare attenzione con questa funzione perché un errore comune è quello di usarla per arrotondare un valore casuale e aspettarsi che sia sempre maggiore di 1, cioè:

int = ceil(random(5));

Ora, ci si aspetterebbe che questo codice dia sempre un numero intero tra 1 e 5, ma questo potrebbe non essere sempre il caso perché c'è una piccolissima possibilità che la funzione random restituisca 0, e arrotondando 0 si ottiene ancora 0. Questa è una remota possibilità ma dovrebbe essere presa in considerazione quando si usa questa funzione.

Sintassi:

ceil(val);

ArgomentoTipo Descrizione
x The number to change.

Ritorna:

Esempio:

val = ceil( 3.4 );

Questo imposterà val a 4.
