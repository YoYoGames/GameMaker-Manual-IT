math_get_epsilon

Questa funzione restituisce il valore attuale di epsilon per la piattaforma di destinazione. Per maggiori informazioni su epsilon, consultate la funzione math_set_epsilon().

Sintassi:

math_get_epsilon();

Ritorna:

Esempio:

var e = math_get_epsilon();
if (e != 0.000001)
{
    math_set_epsilon(0.000001);
}

Questo recupererà il valore attuale di epsilon e lo memorizzerà in una variabile locale, che sarà poi controllata e un nuovo epsilon impostato se necessario.
