pavimento

Restituisce il pavimento di n, cioè n arrotondato per difetto a un intero. Questo è simile alla funzione round() ma arrotonda solo per difetto, indipendentemente dal valore decimale, quindi floor(5.99999) restituirà 5, così come floor(5.2), floor(5.6457) ecc...

Sintassi:

floor(n);

ArgomentoTipo Descrizione
n The number to floor.

Ritorna:

Esempio:

val = floor( 3.9 );

Questo imposterà val a 3.
