morsetto

Con questa funzione è possibile mantenere un valore di ingresso tra un intervallo specificato.

Sintassi:

clamp(val, min, max)

ArgomentoTipo Descrizione
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Ritorna:

Esempio:

speed = clamp(speed, 1, 10);

Il codice di cui sopra bloccherà la velocità in modo che non scenda mai sotto 1 o vada oltre 10.
