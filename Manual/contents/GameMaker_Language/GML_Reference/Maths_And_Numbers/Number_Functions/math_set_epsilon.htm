math_set_epsilon

L'epsilon è un valore utilizzato per determinare se due numeri soggetti a errore di arrotondamento sono abbastanza vicini da essere considerati "uguali". È utile quando si ha a che fare con la matematica in virgola mobile, poiché può ridurre gli "errori di arrotondamento" che fanno sì che certe operazioni restituiscano valori che appaiono errati o contrari a quanto ci aspettiamo. Per esempio, potremmo aver aggiunto un valore all'indice dell'immagine di un sprite e aspettarci che il risultato sia un singolo intero, ma a causa della natura della matematica in virgola mobile, il valore finale effettivo finisce per essere qualcosa come 5.0000002, quindi quando abbiamo il seguente controllo:

if image_index == 5
{
    //do something
}

Il codice non si comporta come previsto e restituisce false. Tuttavia, se impostiamo il valore di epsilon a 0.000001, il valore di image_index sarà arrotondato al numero reale più vicino che è +/- 0.000001 del valore originale, rendendo il confronto di cui sopra vero.

Il valore epsilon sarà usato per tutti i seguenti operatori interi:

  • < : Meno di
  • > : Maggiore di
  • == : Uguale a
  • <= : Inferiore o uguale a
  • >= : Maggiore o uguale a
  • != : Non uguale a

Si noti che impostando un valore di epsilon pari a 0 si disabilitano tutti gli arrotondamenti, e utilizzando un valore di 1 si ottiene un errore.

Sintassi:

math_set_epsilon(epsilon);

Argomento Tipo Descrizione
epsilon Real The new epsilon value (from 0 to 0.999999999).

Ritorna:

Real

Esempio:

math_set_epsilon(0.0001);

Questo imposterà il valore epsilon per tutte le ulteriori operazioni in virgola mobile.
