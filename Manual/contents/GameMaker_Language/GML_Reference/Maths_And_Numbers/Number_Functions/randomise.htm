randomizzare

Questa funzione imposta il seme su un valore casuale. Se avete bisogno di mantenere un valore costante per un certo numero di esecuzioni di un gioco, dovreste usare random_set_seed(). Si prega di notare che quando si utilizzano le funzioni di numero casuale in GameMaker, il seme iniziale è sempre lo stesso, in quanto questo rende molto più facile rintracciare gli errori e debugging. Se desiderate testare con il true random, dovreste chiamare questa funzione all'inizio del vostro gioco. La funzione restituisce il nuovo valore del seme randomizzato (un intero senza segno a 32 bit).

Sintassi:

randomise();
// or
randomize();

Ritorna:

(unsigned 32 bit value)

Esempio:

randomise();

Il codice di cui sopra randomizzerà il seme.
