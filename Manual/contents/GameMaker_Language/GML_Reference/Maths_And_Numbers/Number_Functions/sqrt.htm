sqrt

Se si moltiplica un numero per se stesso, si ottiene il quadrato di quel numero, ma a volte si vuole fare il contrario e ottenere la radice quadrata di un numero. Quindi, per trovare quale numero è stato moltiplicato con se stesso per ottenere un qualsiasi valore positivo, usiamo questa funzione. Per esempio: sqrt(9) restituirà 3 poiché 3*3=9.

Sintassi:

sqrt(val);

ArgomentoTipo Descrizione
val The number to get the square root of.

Ritorna:

Esempio:

num = sqrt(exp);

Il codice qui sopra imposterà la variabile "num" per contenere la radice quadrata del valore contenuto in "exp".
