ln

Il logaritmo naturale ln(n) è la quantità di tempo necessaria per raggiungere un certo livello di crescita continua, dove n è il livello raggiunto. Quindi, se vogliamo scoprire di quante unità di tempo abbiamo bisogno per ottenere 20 di crescita, useremo ln(20) che restituisce 2,99 unità di tempo per ottenere quella quantità di crescita.

Sintassi:

ln(n);

ArgomentoTipo Descrizione
n The input value.

Ritorna:

Esempio:

alarm[0] = ln(age) * room_speed;

Il codice qui sopra usa il logaritmo naturale del valore memorizzato nella variabile "age" per impostare un allarme.
