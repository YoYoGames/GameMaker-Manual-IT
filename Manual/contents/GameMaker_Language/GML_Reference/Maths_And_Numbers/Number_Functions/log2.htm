log2

Questa funzione fondamentalmente chiede "quanti 2 bisogna moltiplicare per ottenere n? Un esempio comune di utilizzo nella programmazione sarebbe quello di calcolare il numero di bit che sono necessari per rappresentare un numero. Per fare questo calcoliamo il log2(x) del numero, lo arrotondiamo per difetto e aggiungiamo 1 - per esempio log2(100) restituisce 6.643856, che arrotondato per difetto è 6, al quale aggiungiamo uno e ci dà 7. Quindi abbiamo bisogno di 7 bit per rappresentare 100. Può anche essere usato per descrivere la crescita esponenziale o il decadimento, ed è spesso chiamato logaritmo binario.

Sintassi:

log2(n);

ArgomentoTipo Descrizione
n The input value.

Ritorna:

Esempio:

colourbits = floor(log2(colour)) - 1;

Il codice precedente memorizza il numero di bit necessari per rappresentare il valore memorizzato nella variabile "colore".
