matrice_moltiplica

Con questa funzione potete moltiplicare insieme due matrici per creare una nuova matrice di trasformazione. La funzione restituisce l'indice della nuova matrice che dovrebbe essere memorizzato in una variabile per uso futuro.

NOTA: Non potete usare una costante di matrice come argomento con questa funzione, quindi se volete moltiplicare la matrice di vista (per esempio) con una matrice personalizzata che avete costruito, dovete prima chiamare matrix_get() e assegnare i valori della matrice di vista a una matrice variabile, poi moltiplicarla per la vostra matrice personalizzata, e poi impostare la matrice scelta (vista, proiezione o mondo).

Sintassi:

matrix_multiply(matrix1, matrix2);

ArgomentoTipo Descrizione
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

Ritorna:

Matrix Array

Esempio:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

Il codice di cui sopra otterrà la matrice di vista corrente, poi la moltiplicherà con una matrice personalizzata e poi userà l'indice della matrice risultante per impostare nuovamente la matrice di vista.
