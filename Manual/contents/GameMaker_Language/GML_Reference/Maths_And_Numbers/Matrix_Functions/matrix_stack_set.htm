matrice_stack_set

Questa funzione sovrascrive l'attuale cima dello stack di matrici con la matrice specificata.

Sintassi:

matrix_stack_set(matrix);

ArgomentoTipo Descrizione
matrix The matrix index to use.

Ritorna:

Esempio:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

Il codice di cui sopra costruirà una nuova matrice e memorizzerà l'indice della matrice risultante nella variabile "m" prima di sostituire la parte superiore dello stack della matrice con essa.
