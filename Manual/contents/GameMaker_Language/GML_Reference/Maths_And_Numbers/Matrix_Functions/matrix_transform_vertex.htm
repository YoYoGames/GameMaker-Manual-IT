matrice_trasformazione_vertex

Questa funzione trasforma un vettore con una matrice. Si fornisce l'ID della matrice di trasformazione (come restituito dalla funzione matrix build()), così come i valori x, y e z del vettore da trasformare. La funzione restituirà una matrice di 3 elementi dove:

  • array[0] = x
  • array[1] = y
  • array[2] = z.

Sintassi:

matrix_transform_vertex(matrix, x, y, z);

ArgomentoTipo Descrizione
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

Ritorna:

(3 elements)

Esempio:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

Il codice precedente trasforma i valori dati usando la matrice memorizzata nella variabile "t_matrix" e li memorizza nell'array "verts".
