matrice_set

Questa funzione prende un array di 16 valori, corrispondenti a un dato tipo di matrice 4x4, dove gli elementi [0 - 3] sarebbero la riga 1, gli elementi [4 -7] sarebbero la riga 2, ecc. (vedi l'immagine nella pagina principale). Potete creare una matrice di questo tipo usando l'opzione matrix_build() o matrix_get() o semplicemente costruendo la matrice da soli e passandola nella funzione. I tipi di matrice disponibili sono vista, proiezione e mondo, per i quali usereste una delle seguenti costanti:

Costante Descrizione
matrix_view La matrice della vista corrente
matrix_projection La matrice di proiezione corrente
matrix_world La matrice del mondo attuale

Sintassi:

matrix_set(type, matrix);

Argomento Tipo Descrizione
type Costante del tipo di matrice The type of matrix to get the values of (see the constants listed above)
matrix Matrice Array The matrix data as an array

Ritorna:

N/A

Esempio:

matrix_set(matrix_world, m_array);

Il codice qui sopra imposterà i valori della matrice del mondo corrente a quelli memorizzati nella matrice "m_array".
