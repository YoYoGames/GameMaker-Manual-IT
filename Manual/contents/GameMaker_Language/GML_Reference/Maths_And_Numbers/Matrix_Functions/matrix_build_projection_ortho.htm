matrix_build_projection_ortho

Questa funzione costruisce una matrice di proiezione ortografica basata sui parametri specificati elencati di seguito (questo è il metodo di proiezione di default utilizzato quando si crea un room in GameMaker senza cambiare nulla). A volte è necessario switch da una proiezione prospettica a una proiezione ortografica, ed è a questo che serve questa funzione. È tipicamente usata per disegnare una sovrapposizione, per esempio, per mostrare la colonna sonora o altri aspetti, poiché questo dà una visione "piatta" degli elementi disegnati (cioè: nessuna prospettiva) in un gioco 3D. Vedi l'immagine qui sotto per avere un'idea della differenza tra le viste ortografiche e prospettiche.

NOTA: Potresti anche aver bisogno di disattivare temporaneamente switch la rimozione delle superfici nascoste se vuoi che le informazioni vengano disegnate indipendentemente dal valore corrente della profondità.

Sintassi:

matrix_build_projection_ortho(width, height, znear, zfar);

ArgomentoTipo Descrizione
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

Ritorna:

Matrix Array

Esempio:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Il codice qui sopra crea una nuova matrice look-at e una matrice ortografica, memorizza i loro ID in variabili locali e poi li usa per impostare le matrici di vista e proiezione per la telecamera assegnata alla porta di vista[0].
