matrice_stack_push

Questa funzione è usata per spingere una nuova matrice in cima alla pila di matrici corrente. Per prima cosa moltiplica la matrice data con la matrice attualmente in cima alla pila, e poi spinge la matrice risultante nella pila.

Questa funzione è utile per applicare più trasformazioni di matrici alle vostre immagini senza dover moltiplicare manualmente varie matrici insieme.

Per esempio, potete avere una matrice di base spinta sullo stack che disegna la grafica in una certa area rettangolare (diciamo una finestra). Dopo aver disegnato della grafica usando quella matrice, potete spingere un'altra matrice sullo stack per disegnare la grafica all'interno di una sottoarea (come un pulsante nella finestra). Dopo aver disegnato qualcosa in quella sottoarea, potete chiamare matrix_stack_pop() per rimuovere la sua matrice dallo stack e continuare a disegnare grafica nell'area principale della finestra.

Sintassi:

matrix_stack_push(matrix);

ArgomentoTipo Descrizione
matrix The matrix to push to the stack

Ritorna:

Esempio:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Il codice di cui sopra costruirà due nuove matrici e poi le spingerà sullo stack delle matrici. La matrice del mondo viene quindi impostata in cima allo stack, viene disegnato un sprite e le trasformazioni vengono poi estratte dallo stack.
