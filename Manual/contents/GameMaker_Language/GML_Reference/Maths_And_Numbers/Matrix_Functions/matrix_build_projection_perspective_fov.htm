matrix_build_projection_perspective_fov

Questa funzione costruisce una matrice di proiezione prospettica basata sul campo visivo, utilizzando i parametri specificati elencati di seguito.

Sintassi:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);

ArgomentoTipo Descrizione
fov The angle of the field of view.
aspect The aspect ratio of the field of view.
znear The near clipping plane.
zfar The far clipping plane.

Ritorna:

Matrix Array

Esempio:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Il codice di cui sopra crea una matrice di proiezione del campo visivo che viene poi memorizzata in una variabile. Questa matrice viene poi utilizzata per impostare la telecamera assegnata alla porta di vista[0].
