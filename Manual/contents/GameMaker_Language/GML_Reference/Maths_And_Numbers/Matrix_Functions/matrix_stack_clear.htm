matrice_stack_clear

Questa funzione cancella lo stack della matrice.

Sintassi:

matrix_stack_clear();

Ritorna:

Esempio:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

Il codice di cui sopra controlla lo stack della matrice per vedere se è vuoto e se non lo è lo cancella.
