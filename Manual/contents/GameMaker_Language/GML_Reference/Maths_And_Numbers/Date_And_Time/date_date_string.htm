data_data_stringa

Con questa funzione potete creare un string contenente la data data data, formattata come giorno/mese/anno.

Sintassi:

date_date_string(date);

ArgomentoTipo Descrizione
date The date to use.

Ritorna:

Esempio:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Questo imposterà "str" per contenere un string formattato della data e dell'ora corrente come mostrato dal sistema.
