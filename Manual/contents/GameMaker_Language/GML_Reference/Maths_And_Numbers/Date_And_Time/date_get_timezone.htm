data_get_timezone

Questa funzione ottiene il fuso orario di base utilizzato per tutte le altre funzioni di data e ora. Questo fuso orario può essere locale (come impostato dal sistema) o UTC, e la funzione restituirà una delle seguenti costanti:

Costante di fuso orario
Costante Descrizione
timezone_local utilizzare il fuso orario locale impostato dal sistema
timezone_utc utilizzare il tempo universale coordinato

Sintassi:

date_get_timezone();

Ritorna:

Time Zone Constant

Esempio:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Questo codice controlla l'impostazione del fuso orario di base per il gioco e se non è UTC lo cambia.
