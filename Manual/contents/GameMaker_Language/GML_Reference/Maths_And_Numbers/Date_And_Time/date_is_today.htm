data_è_oggi

Questa funzione restituirà true se il valore datetime dato è il giorno in cui viene controllato (cioè oggi), o false altrimenti. Questa può essere una funzione utile per cose come gli Easter Eggs nei vostri giochi, o per sbloccare contenuti stagionali. Notate che questa funzione sarà influenzata dal fuso orario impostato (il default è l'ora locale) che potete cambiare usando la funzione date_set_timezone() funzione.

Sintassi:

date_is_today(date);

ArgomentoTipo Descrizione
date The datetime to use.

Ritorna:

Esempio:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

Il codice di cui sopra controllerà il datetime memorizzato nella variabile globale "Halloween" per vedere se è oggi, e se lo è imposta un'altra variabile globale ad un nuovo valore.
