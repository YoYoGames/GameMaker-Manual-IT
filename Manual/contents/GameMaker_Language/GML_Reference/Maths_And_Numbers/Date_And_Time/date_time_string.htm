data_time_string

Con questa funzione è possibile creare un string contenente l'ora data, formattata per il sistema o il dispositivo che sta eseguendo il gioco quando la funzione viene chiamata.

Sintassi:

date_time_string( date );

ArgomentoTipo Descrizione
date The datetime to use.

Ritorna:

Esempio:

str = date_time_string(date_current_datetime());

Questo imposterà la variabile data a qualcosa come "11:36.00" a seconda delle impostazioni di sistema per la visualizzazione di data/ora e la data e l'ora correnti.
