ora_corrente

Questa variabile di sola lettura restituisce il numero di millisecondi che sono passati dall'inizio del gioco.

Sintassi:

current_time;

Ritorna:

Esempio:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

Il codice qui sopra controlla se sono passati più di 10 minuti prima di fare una domanda all'utente.

Indietro: Data e ora
Il prossimo: current_second
