extension_get_option_value

Questa funzione restituisce il valore dell'opzione data nel sito extension con il nome dato.

La funzione restituisce undefined se il nome extension fornito non è valido.

Sintassi:

extension_get_option_value(extension_name, option_name);

Argomento Tipo Descrizione
extension_name Stringa The name of your extension asset as a string
option_name Stringa The name of the option to read

Restituzione:

Any

Esempio:

var _enabled = extension_get_option_value("MyExtension", "enabled");

Questo codice ottiene il valore dell'opzione enabled nella MyExtension extension.

Indietro: Estensioni
