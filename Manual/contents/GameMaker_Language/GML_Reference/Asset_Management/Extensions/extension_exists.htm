estensione_esiste

Questa funzione indica se un extension con il nome dato esiste(true) o no(false).

Sintassi:

extension_exists(extension_name);

Argomento Tipo Descrizione
extension_name Stringa The name of your extension asset as a string

Restituzione:

Boolean

Esempio:

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

Questo codice controlla se l'estensione MyExtension extension esiste e, in caso affermativo, ottiene le sue opzioni come struct e legge un valore da essa.

Indietro: Estensioni
Avanti: extension_get_options
