tag_get_assets

Con questa funzione potete recuperare i nomi di tutti i assets a cui sono stati assegnati il tag o i tag dati. Si fornisce un singolo tag string o un array, dove ogni elemento dell'array è un tag string. La funzione restituirà un array dove ogni voce è il nome (come string) del asset con il tag dato. Se avete bisogno dell'indice unico per il asset, allora potete usare la funzione asset_get_index() insieme al nome restituito. Se non ci sono assets con i tag dati, o se c'è un errore (ad esempio: i tag dati non esistono), allora verrà restituito un array vuoto.

Sintassi:

tag_get_assets(tags);

Argomento Tipo Descrizione
tags Stringa o matrice di stringhe A single asset tag string or an array with various asset tags.

Ritorna:

Array

Esempio:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

Il codice qui sopra crea una lista, poi recupera i nomi di tutti i assets con il tag "background". loops attraverso l'array di nomi restituiti, controllando se qualcuno di loro è sprite assets, e se lo sono allora il valore dell'indice unico per asset viene aggiunto alla lista per uso futuro.
