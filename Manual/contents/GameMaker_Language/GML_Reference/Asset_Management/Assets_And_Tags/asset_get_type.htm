tipo di asset_get

Con questa funzione potete ottenere il tipo di asset a cui si fa riferimento dal suo nome (un string). Verrà restituita una delle costanti elencate di seguito.

Sintassi:

asset_get_type(name);

Argomento Tipo Descrizione
name Stringa The name of the game asset to get the type of (a string).

Ritorna:

Asset Type Constant

Costante Descrizione
asset_object Il nome dato si riferisce a un oggetto.
asset_sprite Il nome dato si riferisce a uno sprite.
asset_sound Il nome dato si riferisce a un suono.
asset_room Il nome dato si riferisce a un room.
asset_tiles Il nome dato si riferisce a un insieme di piastrelle.
asset_path Il nome dato si riferisce a un percorso.
asset_script Il nome dato si riferisce a un script.
asset_font Il nome dato si riferisce a un carattere.
asset_timeline Il nome dato si riferisce a una linea temporale.
asset_shader Il nome dato si riferisce a uno shader.
asset_animationcurve Il nome dato si riferisce a una curva di animazione.
asset_sequence Il nome dato si riferisce a una sequenza.
asset_unknown Il nome dato si riferisce a un asset che o non esiste, o non è uno di quelli sopra elencati.

Esempio:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

Il codice precedente controlla un nome creato dinamicamente asset per vedere se asset è del tipo corretto. Se non lo è, allora verrà mostrato un messaggio di debug, altrimenti il nome asset viene usato per assegnare il asset all'istanza.
