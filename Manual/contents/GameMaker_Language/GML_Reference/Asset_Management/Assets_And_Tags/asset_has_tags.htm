asset_has_tags

Con questa funzione potete controllare se uno o più tag strings è assegnato a qualsiasi asset dal browser asset. Si fornisce o il nome asset (come string) o il suo indice asset, così come un singolo tag string o un array dove ogni elemento è un singolo tag string. Se si fornisce un valore di indice asset, allora è necessario fornire l'argomento opzionale asset type (una costante), poiché assets di tipi diversi possono avere lo stesso indice, anche se non possono avere lo stesso nome. I tipi di asset disponibili sono elencati nella tabella sottostante:

Costante Descrizione
asset_object Il nome dato si riferisce a un oggetto.
asset_sprite Il nome dato si riferisce a uno sprite.
asset_sound Il nome dato si riferisce a un suono.
asset_room Il nome dato si riferisce a un room.
asset_tiles Il nome dato si riferisce a un insieme di piastrelle.
asset_path Il nome dato si riferisce a un percorso.
asset_script Il nome dato si riferisce a un script.
asset_font Il nome dato si riferisce a un carattere.
asset_timeline Il nome dato si riferisce a una linea temporale.
asset_shader Il nome dato si riferisce a uno shader.
asset_animationcurve Il nome dato si riferisce a una curva di animazione.
asset_sequence Il nome dato si riferisce a una sequenza.

Se la funzione ha successo e tutti i tag dati sono presenti per asset allora restituirà true altrimenti restituirà false. Se hai bisogno di controllare uno qualsiasi di una selezione di tag piuttosto che tutti i tag, usa la funzione asset_has_any_tag().

Sintassi:

asset_has_tags(name_or_index, tags, [asset_type]);

Argomento Tipo Descrizione
name_or_index Stringa o risorsa The name of the asset (a string) or its index value (an integer).
tags Stringa o matrice di stringhe A single asset tag string or an array with various asset tags.
[asset_type] Tipo di attività Costante OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

Ritorna:

Boolean

Esempio:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "level_" + string(global.Level);
_a[2] = "boss";
if asset_has_tags(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

Il codice qui sopra creerà un array di tag e poi controllerà per vedere se tutti sono applicati al dato object, e se lo sono creerà un'istanza di un altro object.
