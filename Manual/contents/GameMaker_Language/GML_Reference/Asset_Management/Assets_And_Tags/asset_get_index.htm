indice_di_capitale

Potete usare questa funzione per ottenere l'indice unico di identificazione di un gioco asset dal suo nome. Se il asset non viene trovato, la funzione restituirà un valore di -1, altrimenti restituirà l'id dell'indice unico per il asset controllato. Questo id può poi essere usato in altre funzioni come qualsiasi altro valore di indice, come il sprite_index o il path_index, per esempio. Si prega di notare che anche se questa funzione può essere utilizzata per fare riferimento a assets da strings (vedere l'esempio qui sotto) dovreste sempre assicurarvi che asset esista prima di utilizzarla, altrimenti potreste ottenere errori che faranno crashare il vostro gioco.

Sintassi:

asset_get_index(name);

ArgomentoTipo Descrizione
name The name of the game asset to get the index of (a string).

Ritorna:

Asset (any asset type)

Esempio:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));

if (object_exists(_obj))
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

Il codice di cui sopra otterrà un indice object da un string, e se quell'indice esiste, creerà un'istanza di object nel gioco.
