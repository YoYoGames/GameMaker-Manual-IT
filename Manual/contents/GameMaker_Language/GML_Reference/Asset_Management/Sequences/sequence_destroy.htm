distruggere la sequenza

Con questa funzione potete distruggere un sequence object che è stato creato dinamicamente. Si fornisce o il sequence object struct (come restituito dalla funzione sequence_create()) o l'ID di sequence (come restituito dalla funzione layer_sequence_get_sequence() o dall'istanza sequence struct proprietà sequence). Questa funzione dovrebbe essere usata ogni volta che un sequence creato dinamicamente non è più necessario per liberare la memoria ad esso associata.

Sintassi:

sequence_destroy(sequence_struct_or_id);

Argomento Tipo Descrizione
sequence_struct_or_id Sequence Asset o Sequence Object Struct The sequence object struct or ID to destroy

Ritorna:

N/A

Esempio:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Il codice precedente controlla se il dato sequence object esiste e se esiste viene distrutto.
