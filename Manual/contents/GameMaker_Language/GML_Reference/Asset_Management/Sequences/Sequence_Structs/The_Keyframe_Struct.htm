La struttura del keyframe

Ogni traccia (come definito nella pagina che copre le strutture di traccia) avrà una o più strutture keyframe assegnate ad essa (che si possono ottenere dalla proprietà keyframes della traccia struct), e ognuna avrà le seguenti proprietà:

Sequenza Keyframe Struttura
Variabile Tipo Descrizione
frame Real La posizione (in fotogrammi) lungo il timeline per il keyframe. Il valore predefinito è 0.
length Real La lunghezza di keyframe. Il valore predefinito è 1, e quando impostato a valori più grandi allora la proprietà della traccia a cui si riferisce keyframe sarà mantenuta al valore iniziale per la durata della lunghezza data. Si noti che la proprietà stretch sovrascriverà questo se impostata a true.
stretch Booleano Se questa proprietà è impostata su true allora il keyframe si estende fino al prossimo keyframe per la traccia o alla fine della traccia se è l'ultimo keyframe. È possibile ottenere o impostare questo valore, e il valore predefinito è false.
channels Array di Sequenza Keyframe Strutture datis Questa proprietà permette l'accesso alla lista di strutture di dati keyframe per i canali della traccia. Quando si ottiene questa proprietà viene restituito un array di dati keyframe structs , e quando si imposta questa proprietà dovrebbe essere specificato un array di dati keyframe structs .
