La struttura dei dati dei fotogrammi chiave

Quando si definisce una struttura di keyframe per una traccia, è necessario definire anche i dati che saranno associati ad essa. Questo è composto da diversi canali, dove ogni canale è semplicemente un Keyframe Data Struct. Un canale può essere dato un tipo specifico di dati a seconda del tipo di traccia per cui state impostando i dati keyframe.

Un Keyframe Data Struct contiene le seguenti variabili:

Sequenza Keyframe Struttura dati
Variabile Tipo Descrizione
channel Real Questo è il canale a cui i dati di keyframe dovrebbero essere applicati. È un valore intero positivo che inizia da 0, e vale la pena notare che quando si creano tracce di parametri per "posizione" o "scala" keyframes, allora è necessario utilizzare valori di canale molto specifici. Questi sono: il canale 0 è la posizione X o la scala X, il canale 1 è la posizione Y o la scala Y.
spriteIndex Sprite Asset L'indice di sprite asset da usare per la traccia. Questa proprietà è disponibile solo per le tracce del tipo seqtracktype_graphic e si può ottenere o impostare.
soundIndex Suono L'indice dell'audio asset da usare per la traccia. Questa proprietà è disponibile solo per le tracce del tipo seqtracktype_audio e si può ottenere o impostare.
particleSystemIndex Particle System Asset The particle system asset used for the track. This property is only available for tracks of the type seqtracktype_particlesystem and you can get or set it.
playbackMode Sequenza Chiave Audio Costante La modalità di riproduzione del suono. Questa proprietà è disponibile solo per le tracce del tipo seqtracktype_audio e si può ottenere o impostare. La proprietà dovrebbe essere una delle costanti date nella tabella sotto questa.
curve Struttura della curva di animazione Questa proprietà richiede un animation curve struct (vedi qui per maggiori informazioni) ed è disponibile solo per le tracce del tipo seqtracktype_real. Se non viene usata nessuna curva struct allora il valore di questa proprietà sarà -1.
value Real Questa proprietà è semplicemente un valore associato al canale dati keyframe, ed è disponibile solo per le tracce del tipo seqtracktype_real quando non viene fornito curve struct. Questo può essere, per esempio, la posizione X o Y della traccia se posta all'interno di una traccia con parametro "posizione".
colour Array di Reales Questa proprietà restituisce (o richiede, se impostata) un array per il valore del colore di keyframe con il formato [A, R, G, B]. Questo è disponibile solo per le tracce del tipo seqtracktype_colour. Nota che i valori per ogni componente dovrebbero essere espressi come tra 0 e 1, dove 0 corrisponde al valore HEX #00 e 1 corrisponde al valore HEX #FF (0 - 255 come mostrato nel selezionatore di colori per le tracce a colori nell'Editor di Sequences ).
sequence Sequenza Object Struttura Questa proprietà restituisce (o richiede, se impostata) una struct object ed è disponibile solo per le tracce del tipo seqtracktype_sequence.
objectIndex Oggetto Attività Questa proprietà restituisce (o richiede, se impostata) un indice object ed è disponibile solo per le tracce del tipo seqtracktype_instance.
events Array di Stringas Questa proprietà permette di accedere agli eventi e ai messaggi di trasmissione associati ai dati keyframe struct . È possibile ottenere o impostare questa proprietà, e quando la si ottiene viene restituito un array di strings, e quando la si imposta deve essere specificato un array di strings. Per maggiori informazioni sugli eventi, consulta la sezione Sequence Events And Moments. Questa proprietà è disponibile solo per le tracce del tipo seqtracktype_message.
event Metodo Questa proprietà restituisce (o può essere impostata su) il method associato ai dati di keyframe struct . Se non è stato specificato alcun metodo o si desidera rimuovere il metodo, allora la proprietà dovrebbe essere -1. Questa proprietà è disponibile solo per le tracce del tipo seqtracktype_moment.
Qualsiasi variabile aggiuntiva elencata nella tabella(e) seguente(i)

La variabile playbackMode può essere una delle seguenti costanti:

Sequenza Chiave Audio Costante
Costante Descrizione Valore
seqaudiokey_loop Il suono sarà loop quando viene riprodotto. 0
seqaudiokey_oneshot Il suono suonerà solo una volta e poi si fermerà. 1

Dati della traccia di testo

Il sito struct conterrà le seguenti variabili aggiuntive se assegnato a una traccia di testo (seqtracktype_text):

Sequenza Keyframe Struttura dati
Variabile Tipo Descrizione
text Stringa Questo è il testo string che viene disegnato sulla traccia.
wrap Booleano Questo è un booleano che indica se il testo deve essere avvolto (true) o no (false).
alignmentH Costante di allineamento del testo della sequenza Questo è l'allineamento orizzontale del testo, e sarà uno dei Costante di allineamento del testo della sequenzadati di seguito.
alignmentV Costante di allineamento del testo della sequenza Questo è l'allineamento verticale del testo, e sarà uno dei Costante di allineamento del testo della sequenzadati di seguito.
fontIndex Font Asset Questo è il carattere Font Asset utilizzato dalla traccia di testo.

alignmentH può essere una qualsiasi delle seguenti costanti:

Costante di allineamento del testo della sequenza (alignmentH)
Costante Descrizione
seqtextkey_left Il testo sarà allineato a sinistra della cornice
seqtextkey_right Il testo sarà allineato a destra della cornice
seqtextkey_center Il testo sarà centrato orizzontalmente all'interno della cornice
seqtextkey_justify Il testo sarà giustificato all'interno della cornice

alignmentV può essere una qualsiasi delle seguenti costanti:

Costante di allineamento del testo della sequenza (alignmentV)
Costante Descrizione
seqtextkey_top Il testo sarà allineato alla parte superiore della cornice
seqtextkey_bottom Il testo sarà allineato al fondo della cornice
seqtextkey_middle Il testo sarà centrato verticalmente all'interno della cornice
