ottenere oggetti in sequenza

Con questa funzione potete recuperare un array di tutti gli indici di object che hanno istanze create all'interno del dato sequence. Si fornisce o il sequence object struct (come restituito dalla funzione sequence_create() o sequence_get()) o l'ID sequence (come restituito dalla funzione layer_sequence_get_sequence() o dall'istanza sequence struct proprietà sequence) e la funzione restituirà un array, dove ogni elemento dell'array è un object_index per il diverso objects utilizzato da sequence per creare istanze.

Sintassi:

sequence_get_objects(sequence_struct_or_id);

Argomento Tipo Descrizione
sequence_struct_or_id Sequence Asset o Sequence Object Struct The sequence object struct or ID to get the objects from.

Ritorna:

Array

Esempio:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

Il codice qui sopra ottiene il struct per un sequence object e poi recupera il objects che usa e memorizza l'array in una variabile.
