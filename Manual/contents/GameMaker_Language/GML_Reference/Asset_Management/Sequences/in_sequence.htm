in_sequenza

Questa è una variabile incorporata che fa parte delle variabili di istanza create per ogni istanza di object nel vostro gioco. Se l'istanza è controllata da un sequence, questa variabile restituirà true, altrimenti restituirà false. Questa è una variabile di sola lettura e non può essere modificata.

Si noti che questa variabile diventerà false dopo che il controllo Sequence è terminato, anche se il suo elemento Sequence esiste ancora, e diventerà true se quell'elemento Sequence viene riprodotto di nuovo.

Potete usare questa variabile nel vostro objects controllato dal giocatore (o dalla CPU) per assicurarvi che siano spostati dal codice solo quando non sono in un Sequence. Questo può rivelarsi utile per le scene tagliate, quando avete bisogno di usare le stesse istanze in un Sequence (usando la funzione sequence_instance_override_object() ) e dovete assicurarvi che non possano essere spostate dal loro solito codice mentre quella scena è attiva.

Sintassi:

in_sequence

Ritorna:

Esempio:

if (!in_sequence)
{
    x += move_x;
    y += move_y;
}

Il codice di cui sopra controlla la variabile in_sequence, e se è false (cioè l'istanza non è controllata da un sequence) allora aggiunge move_x e move_y alla posizione dell'istanza. dell'istanza, assicurandosi che si muova solo quando non è in un Sequence.
