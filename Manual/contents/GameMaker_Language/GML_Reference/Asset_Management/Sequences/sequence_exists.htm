sequenza_esiste

Con questa funzione potete controllare se un sequence object esiste o meno. Si fornisce o il sequence object struct (come restituito dalla funzione sequence_create() o sequence_get()) o l'ID di sequence (come restituito dalla funzione layer_sequence_get_sequence() o dall'istanza sequence struct proprietà sequence, o l'indice dal browser asset ) e la funzione restituirà true se il sequence object esiste o false se non esiste.

Sintassi:

sequence_exists(sequence_struct_or_id);

Argomento Tipo Descrizione
sequence_struct_or_id Sequence Asset o Sequence Object Struct The sequence to check for, can be the asset reference itself or its Sequence object struct.

Ritorna:

Boolean

Esempio:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Il codice precedente controlla se il dato sequence object esiste e se esiste viene distrutto.

Indietro: Sequenze
Il prossimo: sequence_create
