sequenza_instanza

Questa è una variabile incorporata che fa parte delle variabili di istanza create per ogni istanza object nel vostro gioco. Se l'istanza è controllata da un sequence, questa variabile conterrà la struttura dell'istanza della sequenza per il Sequence che controlla l'istanza, altrimenti sarà undefined. Questa è una variabile di sola lettura e non può essere modificata.

Si noti che questa variabile diventerà undefined dopo che il controllo di Sequence è terminato, anche se il suo elemento Sequence esiste ancora, e otterrà di nuovo struct se quell'elemento Sequence viene riprodotto.

Sintassi:

sequence_instance

Ritorna:

Sequence Instance Struct

Esempio:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

Il codice di cui sopra controlla la variabile in_sequence e se è true (cioè l'istanza è controllata da un sequence) allora cambierà la scala di velocità di quel Sequence a 2.
