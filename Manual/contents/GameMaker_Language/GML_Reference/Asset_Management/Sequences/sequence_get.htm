ottenere la sequenza

Con questa funzione potete recuperare la sequenza Object Struct da un indice sequence asset. Si fornisce l'indice asset per il sequence come definito nel browser asset, e la funzione restituirà il sequence object struct a cui si può accedere, o -1 se il sequence non esiste o l'indice asset dato non è un sequence.

Sintassi:

sequence_get(sequence_index);

Argomento Tipo Descrizione
sequence_index Sequenza Attività The sequence asset as defined in the Asset Browser.

Ritorna:

Sequence Object Struct or -1

Esempio:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

Il codice di cui sopra recupera il sequence object struct dal sequence object "seq_Loop" e poi imposta la modalità loop per il sequence (tutte le istanze sequence create successivamente useranno ora questa modalità loop ).
