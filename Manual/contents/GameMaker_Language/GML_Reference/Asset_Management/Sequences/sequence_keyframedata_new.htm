sequenza_chiaveframedata_nuova

Con questa funzione è possibile creare una nuova struttura di dati keyframe, fornendo il tipo di traccia a cui saranno applicati i dati di keyframe. L'argomento del tipo può essere una qualsiasi delle costanti del tipo di traccia della sequenza elencate in questa pagina.

La funzione restituirà una struttura di dati di keyframe della traccia che può quindi avere valori aggiunti prima di essere assegnata a una struttura di keyframe.

Sintassi:

sequence_keyframedata_new(type);

Argomento Tipo Descrizione
type Sequenza Tipo di traccia Costante The type of keyframe data to create.

Ritorna:

Sequence Keyframe Data Struct

Esempio:

myseq = sequence_create();
var mytracks = array_create(1);
mytracks[0] = sequence_track_new(seqtracktype_graphic);
var graphickeys = array_create(1);
graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic);
graphickeys[0].frame = 0;
graphickeys[0].length = 1;
graphickeys[0].stretch = true;
graphickeys[0].disabled = false;
var graphickeydata = array_create(1);
graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic);
graphickeydata[0].spriteIndex = spr_Platform;
graphickeydata[0].channel = 0;
graphickeys[0].channels = graphickeydata;
mytracks[0].name = "TestGraphicTrack";
mytracks[0].keyframes = graphickeys;
myseq.tracks = mytracks;

Il codice qui sopra crea un nuovo sequence e poi crea una traccia grafica asset e imposta alcuni dati keyframe sulla traccia. Questa traccia viene poi assegnata all'istanza.
