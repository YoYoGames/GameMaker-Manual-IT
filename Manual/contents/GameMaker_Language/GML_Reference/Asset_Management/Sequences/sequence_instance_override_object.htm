sequenza_instanza_override_oggetto

Con questa funzione potete sovrascrivere (sostituire) tutte le istanze di un object usate in un sequence con un'altra. Si fornisce l'ID dell'istanza sequence struct (come restituito quando l'istanza sequence è stata creata in room o utilizzando una delle funzioni di livello room - vedi qui), così come l'indice object (come definito nel browser asset ) per il object che si vuole sovrascrivere. Infine si fornisce un indice object o un ID di istanza da usare come object che sta per sovrascrivere il sequence (fornendo un ID di istanza si userà semplicemente il object da cui l'istanza è stata creata come sovrascrittura). Si noti che questo può essere fatto solo su istanze sequence (non sequence objects) e deve essere fatto prima che sequence inizi a suonare, altrimenti non funzionerà.

Sintassi:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argomento Tipo Descrizione
sequence_instance_struct Struttura dell'istanza della sequenza The sequence instance struct to modify.
object_id Oggetto Attività The object index of the object within the sequence to override.
instance_or_object_id ID dell'asset dell'oggetto o dell'istanza The object index or instance ID to use to override the sequence objects.

Ritorna:

N/A

Esempio:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Il codice qui sopra crea una nuova istanza di sequence sul livello dato e poi lo modifica in modo che tutte le istanze di object "obj_Trees_Winter" siano sostituite da istanze di object "obj_Trees_Summer".
