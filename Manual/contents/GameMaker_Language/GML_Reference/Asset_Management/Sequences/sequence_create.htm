creare una sequenza

Con questa funzione potete creare un nuovo sequence object al quale potete poi aggiungere delle tracce. La funzione restituisce una struct a cui potete accedere per impostare il nuovo sequence che avete creato. Il contenuto di questa struct è dettagliato in questa pagina. Il sequence object struct può quindi essere utilizzato per creare istanze del sequence su un livello room utilizzando la funzione layer_sequence_create(). Si noti che quando si crea sequence objects in questo modo si dovrebbe rimuoverli di nuovo chiamando la funzione sequence_destroy() quando non sono più necessarie, altrimenti si avrà una perdita di memoria che può rallentare ed eventualmente mandare in crash il gioco.

Sintassi:

sequence_create();

Ritorna:

Sequence Object Struct

Esempio:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

Il codice qui sopra crea un nuovo sequence object struct e imposta la sua lunghezza di riproduzione e la modalità loop.
