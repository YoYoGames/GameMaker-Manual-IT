shader_get_name

Con questa funzione potete recuperare il nome di una risorsa shader. Si fornisce il valore ID unico per il shader di cui ottenere il nome e la funzione restituirà il nome della risorsa come string.

Sintassi:

shader_get_name(shader);

ArgomentoTipo Descrizione
shader The index of the shader to get the name of.

Ritorna:

Esempio:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

Il codice qui sopra otterrà il nome del dato shader e lo disegnerà sullo schermo.
