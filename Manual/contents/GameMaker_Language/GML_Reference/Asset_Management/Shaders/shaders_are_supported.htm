gli shader sono supportati

Questa funzione farà un controllo per vedere se la piattaforma di destinazione scelta supporta shaders, restituendo true se lo fa, e false se non lo fa. È importante notare che su Android, se il progetto non ha alcuna risorsa shader definita, allora la funzione restituirà sempre false, indipendentemente dal fatto che il dispositivo supporti o meno shaders.

Sintassi:

shaders_are_supported();

Ritorna:

Esempio:

global.GFX = false;
if shaders_are_supported()
{
    if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
    {
        global.GFX = true;
    }
}

Il codice di cui sopra imposterà una variabile globale su false, e poi se la piattaforma supporta shaders ed entrambi i shaders controllati sono stati compilati correttamente, sarà impostato su true.
