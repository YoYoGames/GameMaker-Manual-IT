shader_current

Questa funzione restituisce il valore ID dell'indice di shader attualmente utilizzato per il rendering, o restituisce -1 se non viene utilizzato nessun shader.

Sintassi:

shader_current();

Ritorna:

Shader Asset or -1 (if no shader is assigned)

Esempio:

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

Il codice di cui sopra controllerà per vedere qual è l'attuale shader e se restituisce -1 (nessun shader in uso) allora viene impostato un shader.
