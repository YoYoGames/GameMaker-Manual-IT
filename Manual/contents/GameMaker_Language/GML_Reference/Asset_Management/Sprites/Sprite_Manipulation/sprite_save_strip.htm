sprite_save_strip

Questa funzione creerà un'immagine di striscia da tutte le sotto-immagini in un sprite, salvandola su disco con il nome di file specificato. Il sprite deve essere stato aggiunto a runtime (non si può salvare sprites aggiunto attraverso il IDE) e il file deve essere salvato con un *.png extension.

NOTA: A seconda della piattaforma di destinazione scelta, siete limitati su dove potete salvare e caricare i file. Vedi Gestione dei file per maggiori informazioni.

Sintassi:

sprite_save_strip(ind, filename);

Argomento Tipo Descrizione
ind Sprite Asset The index of the sprite to save.
filename Stringa The filename for the saved sprite strip.

Restituisce

N/A

Esempio:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Il codice qui sopra crea un sprite dalla superficie indicizzata nella variabile "surf", assegnando il suo indice alla variabile locale "spr_Custom", e poi usa un for loop per muoversi attraverso la superficie e catturare varie sezioni che vengono aggiunte nel sprite come sotto-immagini. questo nuovo sprite viene poi salvato come una striscia png prima di essere rimosso dalla memoria.
