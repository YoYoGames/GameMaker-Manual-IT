sprite_set_speed

Questa funzione può essere usata per impostare la velocità e il tipo di animazione di base per la risorsa sprite. Questo è normalmente impostato nello Sprite Editor, ma ci possono essere momenti in cui è necessario impostarlo manualmente per una data risorsa sprite. È necessario fornire l'indice sprite da impostare, la velocità di animazione e il tipo di velocità di animazione da applicare. Il tipo di velocità di animazione può essere una qualsiasi delle seguenti costanti:

Costante Descrizione
spritespeed_framespersecond Il sprite è stato definito con l'animazione in fotogrammi al secondo.
spritespeed_framespergameframe Il sprite è stato definito con in animazione in fotogrammi per fotogramma di gioco.


Selezionare il tipo di velocità da usare per calcolare le animazioni è molto importante, poiché c'è un'enorme differenza tra le due opzioni. I due esempi seguenti illustrano questa differenza:

  • Se avete un sprite che disegna 1 fotogramma al secondo e impostate la velocità dell'immagine a 0.5, esso disegnerà a 0.5 fotogrammi al secondo. Se il tuo frame rate di gioco è di 60 frame al secondo, allora sprite disegnerà 1 frame ogni 120 frame di gioco.
  • Se avete un sprite che disegna 1 fotogramma per ogni fotogramma di gioco e impostate la velocità dell'immagine a 0.5, esso disegnerà 0.5 fotogrammi per ogni fotogramma di gioco. Se il tuo frame rate di gioco è di 60 frame al secondo, allora sprite disegnerà 30 frame ogni 60 frame di gioco.

Potete scoprire quale era il valore di velocità usato per l'animazione sprite usando la funzione sprite_get_speed()e si può ottenere il tipo di velocità dell'animazione usando sprite_get_speed_type().

IMPORTANTE! Questa funzione influenzerà la risorsa sprite, quindi qualsiasi cambiamento fatto con questa funzione influenzerà tutte le istanze che stanno usando sprite e anche tutte le istanze future.

Sintassi:

sprite_set_speed(index, speed, type)

Argomento Tipo Descrizione
index Sprite Asset The index of the sprite to set the animation speed of
speed Real The speed value to set
type Costante di velocità dello sprite The speed type to use

Ritorna:

Constant (see above)

Esempio:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Il codice precedente imposta la velocità di animazione di base per il sprite assegnato all'istanza a 10 fotogrammi al secondo.
