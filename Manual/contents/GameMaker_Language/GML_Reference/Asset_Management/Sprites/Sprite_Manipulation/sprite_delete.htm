sprite_delete

Questa funzione cancellerà un sprite dal gioco, liberando la memoria che gli era stata riservata. Questa è una funzione molto importante per quei momenti in cui è necessario creare e modificare sprites da fonti esterne (come caricare un sprite da un file con sprite_add() o duplicare un sprite usando sprite_duplicate()) e dovrebbe sempre essere usata per rimuovere quei assets che non sono più necessario per un gioco, o per cancellare un asset indicizzato da una variabile prima di riassegnare un altro asset a quella variabile. Questo non può essere usato per cancellare sprites che sono inclusi nel gioco come parte di assets nel Browser Asset .

Sintassi:

sprite_delete(index);

ArgomentoTipo Descrizione
index The index of the sprite to be deleted.

Restituisce

Esempio:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

Il codice di cui sopra crea una variabile locale e poi memorizza l'indice del sprite creato dalla superficie dell'applicazione. Questo sprite viene poi fuso con il asset indicizzato nella variabile "spr_Player" prima di essere nuovamente rimosso dalla memoria.
