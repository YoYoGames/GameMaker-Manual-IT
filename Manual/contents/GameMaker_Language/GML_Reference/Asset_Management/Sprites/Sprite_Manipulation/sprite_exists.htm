sprite_exists

Questa funzione restituisce se un sprite con l'indice specificato esiste o meno nel progetto in corso.

Sintassi:

sprite_exists(index);

ArgomentoTipo Descrizione
index The index of the sprite to check.

Restituisce

Esempio:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

Il codice precedente controlla un array per vedere se contiene un indice sprite valido e se è così assegna quel sprite all'istanza, ma se non lo fa, assegna un sprite dalle risorse incluse.
