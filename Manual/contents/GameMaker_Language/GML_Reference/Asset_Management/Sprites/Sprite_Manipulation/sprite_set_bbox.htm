sprite_set_bbox

Questa funzione può essere usata per impostare i valori del rettangolo di selezione per un sprite. Si fornisce l'indice sprite da usare e poi i valori sinistro, superiore, destro e inferiore per le posizioni del rettangolo di selezione. Le posizioni sono valori assoluti, dove la posizione (0, 0) corrisponde all'angolo in alto a sinistra di sprite, indipendentemente dall'offset di sprite, da qualsiasi pixel "vuoto" che sprite può avere, o da dove viene disegnato in room.

NOTA Questa funzione influenza la risorsa sprite in modo che tutte le ulteriori istanze con questo sprite avranno lo stesso riquadro di delimitazione.

Sintassi:

sprite_set_bbox(ind, left, top, right, bottom);

ArgomentoTipo Descrizione
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Restituisce

Esempio:

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

Il codice di cui sopra controllerà la modalità del riquadro di delimitazione di sprite assegnato a sprite_index, e se è impostato su manual allora avrà il suo riquadro di delimitazione cambiato.
