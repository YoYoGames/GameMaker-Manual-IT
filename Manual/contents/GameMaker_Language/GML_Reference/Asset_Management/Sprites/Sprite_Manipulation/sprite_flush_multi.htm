sprite_flush_multi

Con questa funzione potete rimuovere un qualsiasi numero di pagine texture per il dato sprites dalla memoria texture. Fornite gli indici sprite come un array e le pagine texture a cui sono assegnate saranno rimosse dalla memoria texture. Notate che se una delle pagine di texture è usata altrove in room (da un'altra istanza sprite o da uno sfondo, ecc...) potreste avere un piccolo impatto sulle prestazioni poiché la pagina è immediatamente ricaricata di nuovo in memoria, quindi dovreste fare attenzione quando usate questa funzione. Si noti che la funzione restituirà -1 se il flush non è supportato per le risorse scelte, oppure restituirà 0 se tutto funziona correttamente.

Sintassi:

sprite_flush_multi(array);

ArgomentoTipo Descrizione
array Array with the sprite indices to flush

Ritorna:

(-1 or 0)

Esempio:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Il codice di cui sopra crea un array dove ogni elemento contiene un indice sprite. Questo array viene poi usato per cancellare questi sprite textures dalla memoria.
