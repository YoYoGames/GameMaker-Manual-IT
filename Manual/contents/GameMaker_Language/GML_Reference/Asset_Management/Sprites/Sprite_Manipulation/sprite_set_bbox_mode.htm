sprite_set_bbox_mode

Questa funzione può essere utilizzata per impostare la modalità di delimitazione di un sprite. Si fornisce l'indice sprite e la modalità da utilizzare, che dovrebbe essere una delle seguenti costanti:

Costante di modalità del Bounding Box
Costante Descrizione
bboxmode_automatic Automatico - Il riquadro di delimitazione sarà calcolato automaticamente, in base alle impostazioni di tolleranza per lo sprite
bboxmode_fullimage Immagine completa - Il riquadro di delimitazione sarà impostato per utilizzare l'intera larghezza e altezza di sprite, indipendentemente dalla tolleranza e dai pixel "vuoti".
bboxmode_manual Manuale - Il riquadro di delimitazione viene impostato manualmente su valori definiti dall'utente, il che richiede l'uso della funzione sprite_set_bbox()

NOTA Questa funzione influenza la risorsa sprite in modo che tutte le ulteriori istanze con questo sprite avranno la stessa modalità di bounding box.

Sintassi:

sprite_set_bbox_mode(ind, mode);

Argomento Tipo Descrizione
ind Sprite Asset The index of the sprite to change the mode of.
mode Costante di modalità del Bounding Box The mode to set (a constant).

Restituisce

N/A

Esempio:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Il codice di cui sopra controlla la modalità bbox per l'attuale sprite e se non è automatic, allora è impostato su quel valore.
