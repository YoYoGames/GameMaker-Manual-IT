sprite_set_nineslice

Questa funzione è usata per applicare una struttura Nine Slice a un sprite. Si fornisce l'indice sprite e la struttura Nine Slice struct da applicare, che può essere recuperata usando sprite_nineslice_create() o sprite_get_nineslice().

NOTA Questa funzione ha effetto sulla risorsa sprite, quindi qualsiasi modifica apportata con questa funzione avrà effetto su tutte le istanze che stanno usando questo sprite e anche su tutte le istanze future.

Sintassi:

sprite_set_nineslice(ind, nineslice);

Argomento Tipo Descrizione
ind Sprite Asset The index of the sprite to modify
nineslice Strutturazione a nove fette The Nine Slice struct to apply

Ritorna:

N/A

Esempio:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Il codice qui sopra crea un nuovo Nine Slice struct, lo abilita e imposta gli offset per le guide. Il struct viene poi applicato ad un sprite, cambiando le sue proprietà Nine Slice.
