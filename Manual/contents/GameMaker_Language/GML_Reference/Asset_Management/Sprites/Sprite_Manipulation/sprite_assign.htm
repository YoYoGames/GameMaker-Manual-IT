sprite_assign

Questa funzione prende due indici di sprite precedentemente creati (o inclusi) e copia l'immagine da uno all'altro. In questo modo si può copiare (o "clonare") un sprite in un altro indice. Nota che non puoi copiare in una risorsa di gioco. Dovete aver creato il sprite su cui copiare in precedenza usando l'opzione sprite_add() o sprite_duplicate() funzioni.

Sintassi:

sprite_assign(index, sprite);

ArgomentoTipo Descrizione
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Restituisce

Esempio:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Il codice qui sopra controlla per vedere se la variabile globale "Player_Sprite" contiene un sprite e se lo fa usa sprite_assign per cambiarlo con uno che è stato caricato da un file esterno. Se non contiene un sprite ne viene caricato uno e il suo indice viene memorizzato in quella variabile.
