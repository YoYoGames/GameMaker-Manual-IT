elenco scheletro_bone

Con questa funzione si può popolare una lista DS (pre-creata) con tutti i nomi delle ossa usate come parte dell'animazione scheletrica sprite. I nomi saranno strings e potranno poi essere utilizzati nelle altre funzioni delle ossa dell'animazione dello scheletro per questi tipi di sprite.

Sintassi:

skeleton_bone_list(sprite, list);

Argomento Tipo Descrizione
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID elenco DS The ID of the DS list to populate with the bone names.

Ritorna:

N/A

Esempio:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

Il codice di cui sopra crea una lista DS e la popola con i nomi delle ossa per un uso successivo.
