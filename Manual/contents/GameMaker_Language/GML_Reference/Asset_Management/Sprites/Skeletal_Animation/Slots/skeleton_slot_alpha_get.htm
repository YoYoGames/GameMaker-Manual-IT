scheletro_slot_alpha_get

Con questa funzione potete ottenere il valore alfa di uno slot allegato. Si fornisce il nome dello slot (un string) e la funzione restituisce un valore reale per l'alfa tra 0 e 1.

Sintassi:

skeleton_slot_alpha_get(slot);

Argomento Tipo Descrizione
slot Stringa The slot name to check, a string

Ritorna:

Real

Esempio:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Il codice qui sopra crea una lista DS e poi la popola con i nomi degli slot trovati nella posizione del mouse in room. Poi loops attraverso la lista degli slot e reimposta il colore per gli slot trovati al bianco con un alpha di 1 se l'alpha non è già 1.
