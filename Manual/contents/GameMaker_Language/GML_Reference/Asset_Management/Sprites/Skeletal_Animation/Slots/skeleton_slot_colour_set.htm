set di colori per le scanalature dello scheletro

Con questa funzione potete impostare il colore e l'alfa di uno slot di attacco, così tutti i sprites attaccati ad esso saranno disegnati con questi valori di fusione. Tenete presente che l'istanza sprite può avere un colore di fusione e un'impostazione alfa (image_blend e image_alpha), così come l'allegato (vedi la funzione skeleton_attachment_create_colour()), e quindi il colore finale e l'alfa che l'allegato assegnato sprite per lo slot avrà sarà un composto di tutti questi valori.

Sintassi:

skeleton_slot_colour_set(slot, colour, alpha);

Argomento Tipo Descrizione
slot Stringa The slot name to set, a string
colour Colore The colour to set, either an integer, a constant, or a hex value.
alpha Real The alpha to set from 0 to 1.

Ritorna:

N/A

Esempio:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Il codice qui sopra crea una lista DS e poi la popola con i nomi degli slot trovati nella posizione del mouse in room. Poi loops attraverso la lista degli slot e reimposta il colore per gli slot trovati al bianco con un alpha di 1 se il colore non è già bianco.
