scheletro_slot_data

Con questa funzione è possibile popolare una lista DS (pre-creata) con dati per ciascuno degli slot di attacco disponibili che un'animazione contiene. Questi dati sono restituiti nel forum di una mappa DS che contiene le seguenti coppie chiave/valore:

  • "nome": Il nome dello slot.
  • "osso": Il nome dell'osso.
  • "allegato": Il nome dell'allegato se ne viene usato uno o "(nessuno)" se non ce n'è uno assegnato.

I valori per ogni chiave saranno strings e possono poi essere usati nelle altre funzioni di attacco dello scheletro per questi tipi di sprite. Notate che la mappa DS creata non viene distrutta, quindi dovrete loop attraverso la lista DS e distruggere ciascuna delle mappe create da soli.

Sintassi:

skeleton_slot_data(sprite, list);

Argomento Tipo Descrizione
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the data from.
list ID elenco DS The ID of the DS list to populate with the DS maps.

Ritorna:

N/A

Esempio:

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);

for (var i = 0; i < ds_list_size(list); i++)
{
    var map = list[| i];
    if open
    {
        if map[? "attachment"] == "(none)"
        {
            open = false;
            slot_name = map[? "name"];
        }
    }

    ds_map_destroy(map);
}

ds_list_destroy(list);

Il codice di cui sopra crea una lista DS e poi la popola con i dati degli slot per l'istanza sprite. Questi dati vengono poi analizzati per estrarre le singole mappe DS con i dati degli slot. Questo viene poi controllato per vedere se c'è uno slot vuoto, e se è così alla variabile "slot_name" viene assegnato il nome dello slot vuoto prima che la mappa DS venga distrutta. Infine si distrugge la lista DS poiché non è più necessaria.
