elenco degli scheletri

Con questa funzione è possibile popolare una lista (pre-creata) DS con tutti i nomi degli slot creati come parte dell'animazione scheletrica sprite. I nomi saranno strings e potranno poi essere utilizzati nelle altre funzioni di slot di animazione dello scheletro per questi tipi di sprite.

Sintassi:

skeleton_slot_list(sprite, list);

Argomento Tipo Descrizione
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID elenco DS The ID of the DS list to populate with the bone names.

Ritorna:

N/A

Esempio:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

Il codice di cui sopra crea una lista DS e la popola con i nomi degli slot per un uso successivo.
