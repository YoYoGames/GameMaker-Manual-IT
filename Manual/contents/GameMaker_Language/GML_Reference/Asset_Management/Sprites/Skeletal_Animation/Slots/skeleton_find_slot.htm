scheletro_find_slot

Con questa funzione è possibile trovare quali slot si trovano in una determinata posizione spazio-ambiente nel sito Spine sprite associato all'istanza corrente. Si crea un elenco DS e si fornisce il suo ID insieme a una posizione x/y da controllare e l'elenco verrà popolato con il nome string per ciascuno degli slot di attacco disponibili per l'sprite (comprese eventuali modifiche di attacco). Si noti che l'elenco è sempre ordinato in ordine decrescente a partire dallo slot più in alto.

Sintassi:

skeleton_find_slot(x, y, list);

Argomento Tipo Descrizione
x Real The x position in the room to check.
y Real The y position in the room to check.
list ID elenco DS The ID of the DS list to populate with the DS maps.

Ritorna:

N/A

Esempio:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Il codice qui sopra crea una lista DS e poi la popola con i nomi degli slot trovati nella posizione del mouse in room.
