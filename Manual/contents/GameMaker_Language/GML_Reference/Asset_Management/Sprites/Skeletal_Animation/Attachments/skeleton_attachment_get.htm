attacco_scheletro_get

Un'animazione scheletrica sprite può avere altri sprites aggiunti come allegati, con questi sprites aggiunti ad uno slot nominato (il nome è dato quando si crea lo slot dell'allegato nel vostro programma di animazione) e saranno disegnati insieme all'animazione dell'attuale sprite. Con questa funzione è possibile ottenere il nome (come string) dell'allegato per il dato slot dell'attuale sprite assegnato. Notate che gli allegati sprites sono referenziati attraverso la loro stringa di nome come assegnata in Spine, o quando avete chiamato skeleton_attachment_create().

Sintassi:

skeleton_attachment_get(slot);

Argomento Tipo Descrizione
slot Stringa The slot name (a string) to get the attachment of.

Ritorna:

String

Esempio:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Il codice di cui sopra controllerebbe il nome dell'allegato attualmente assegnato per lo slot chiamato "slot_leftHand" e se viene restituito un string vuoto, viene allegato un nuovo sprite.

Indietro: Allegati
