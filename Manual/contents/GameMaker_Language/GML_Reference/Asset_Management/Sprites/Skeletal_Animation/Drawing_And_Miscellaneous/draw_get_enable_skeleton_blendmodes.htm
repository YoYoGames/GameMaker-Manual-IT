draw_get_enable_skeleton_blendmodes

Questa funzione restituisce se le modalità di fusione per slot per lo scheletro sprites sono abilitate(true) o disabilitate(false).

Sintassi:

draw_get_enable_skeleton_blendmodes()

Ritorna:

Boolean

Esempio:

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

Il codice qui sopra abilita le modalità di fusione per slot per lo scheletro sprites, se sono disabilitate.
