scheletro_get_minmax

Questa funzione restituirà un array con il valore totale del bounding box per tutti i singoli bounding box assegnati a un'animazione skeleton sprite. L'array restituito avrà 4 elementi:

  • [0] - la posizione x minima per tutti i box di delimitazione
  • [1] - la posizione y minima per tutti i box di delimitazione
  • [2] - la posizione x massima per tutti i box di delimitazione
  • [3] - la posizione y massima per tutti i box di delimitazione

Sintassi:

skeleton_get_minmax();

Ritorna:

Array (4 elements: xMin, yMin, xMax, yMax)

Esempio:

var frame = skeleton_animation_get_frame(0);
if (frame > 0 && frame < 2)
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Il codice di cui sopra controllerà il fotogramma corrente dell'animazione assegnata alla traccia 0 e poi farà scoppiare alcune particelle da un punto al centro del bounding box totale per il sprite.
