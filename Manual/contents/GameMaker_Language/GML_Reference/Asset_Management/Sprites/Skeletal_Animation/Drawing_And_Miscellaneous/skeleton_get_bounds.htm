scheletro_get_bounds

Questa funzione restituirà un array di valori associati a qualsiasi box di contorno dato per l'animazione dello scheletro attualmente assegnata sprite. Si fornisce il numero di indice per il rettangolo di selezione di cui ottenere i dettagli (è possibile recuperare il numero totale di rettangoli di selezione per il sprite utilizzando la funzione skeleton_get_num_bounds()) e la funzione restituirà un array con i seguenti elementi:

  • [0] - il numero di punti sul riquadro di delimitazione (un reale)
  • [1] - il nome del riquadro di delimitazione (un string)
  • [2] - la posizione x del primo punto
  • [3] - la posizione y del primo punto
  • [ecc...] - ulteriori dati di posizione x/y fino al numero di punti sul riquadro di delimitazione

Sintassi:

skeleton_get_bounds(index);

Argomento Tipo Descrizione
index Real The bounding box index to get the values of.

Ritorna:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

Esempio:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

Il codice di cui sopra si occuperà di loop attraverso ciascuno dei riquadri di delimitazione associati al corrente sprite e poi disegnerà quei dati come un string sullo schermo.
