skeleton_collision_draw_set

Con questa funzione, è possibile attivare (true) o disattivare (false) il disegno dei dati di collisione per l'animazione scheletrica corrente sprite utilizzata dall'istanza. Se questo è attivato, il bounding box e la maschera di collisione precisa saranno disegnati come contorni intorno al sprite.

Questa funzione funziona solo quando il disegno dello scheletro sprite è gestito da object. Per disegnare il rettangolo di delimitazione delle collisioni quando si disegna uno scheletro sprite manualmente, usare draw_skeleton_collision().

Sintassi:

skeleton_collision_draw_set(flag);

Argomento Tipo Descrizione
flag Booleano Set to true per attivare il disegno e false per disattivarlo.

Ritorna:

N/A

Esempio:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

Il codice qui sopra controlla se il gioco viene eseguito in modalità debug e se lo è, altera i dati di collisione scheletrica per l'istanza da mostrare.
