scheletro_pelle_get

Con l'animazione scheletrica sprites, è possibile assegnare textures separati (chiamati "skin") all'animazione, utilizzando così un'animazione per più cose diverse. Questa funzione restituisce il nome della pelle (come string) che è attualmente assegnata all'animazione scheletrica sprite che la vostra istanza sta usando. Il nome restituito è quello che avete impostato quando avete creato il sprite nel vostro programma di animazione.

Sintassi:

skeleton_skin_get();

Ritorna:

Esempio:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Il codice di cui sopra controllerebbe la skin attualmente assegnata all'animazione e se è "skin_Enemy1", sceglie e imposta una nuova skin da una delle tre opzioni.

Indietro: Pelli
