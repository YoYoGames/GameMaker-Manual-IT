elenco scheletro_pelle

Con questa funzione è possibile popolare una lista (pre-creata) DS con tutti i nomi delle skin incluse come parte dell'animazione scheletrica sprite. I nomi saranno strings e potranno poi essere utilizzati nelle altre funzioni di skin di animazione dello scheletro per questi tipi di sprite.

Sintassi:

skeleton_skin_list(sprite, list);

Argomento Tipo Descrizione
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID elenco DS The ID of the DS list to populate with the animation names.

Ritorna:

N/A

Esempio:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Il codice di cui sopra crea una lista DS e la popola con i nomi delle skin. Una casuale viene poi scelta e applicata all'istanza prima che la lista venga distrutta.
