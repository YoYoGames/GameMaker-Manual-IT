scheletro_skin_set

Un'animazione scheletrica sprite può essere disegnata con una varietà di textures (chiamata "skin"), permettendovi di usare una singola animazione per più scopi diversi. Per esempio, potete avere un'animazione generica di camminata e questa può essere usata dai vostri personaggi NPC, nemici, ecc... in un gioco semplicemente cambiando la skin con cui è disegnata. Queste skin devono essere state precedentemente create e nominate nel vostro programma di animazione, e questo nome viene passato nella funzione (come string) per impostare sprite ad usare quella skin quando viene disegnata.

Sintassi:

skeleton_skin_set(skinname);

ArgomentoTipo Descrizione
skinname The name (a string) of the skin to use.

Ritorna:

Esempio:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Il codice di cui sopra controllerebbe la skin attualmente assegnata all'animazione e se è "skin_Enemy1", sceglie e imposta una nuova skin da una delle tre opzioni.
