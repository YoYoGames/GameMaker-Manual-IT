scheletro_animazione_get_ext

Una singola animazione scheletrica sprite può avere vari set di animazione, e questi set possono essere assegnati a diverse tracce in modo da poter "mescolare e abbinare" i set di animazione. Questa funzione restituirà il nome del set di animazione attualmente utilizzato dal numero di traccia dato (come impostato dalla funzione skeleton_animation_set_ext).

Sintassi:

skeleton_animation_get_ext(track);

Argomento Tipo Descrizione
track Real The track number to get the animation set name of.

Ritorna:

String

Esempio:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

Il codice di cui sopra cambierà il set di animazione usato dalla traccia 1 in "bodyfight" se non lo è già.
