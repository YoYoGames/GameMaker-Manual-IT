set_animazione_scheletro

Una volta assegnata un'animazione scheletrica sprite alla propria istanza, è possibile definire quale set di animazioni deve utilizzare in qualsiasi momento con questa funzione. Quando avete creato il vostro sprite (in Spine) avrete definito più set di animazione per diversi stati e avrete dato a ciascuno di essi un nome appropriato. Questi nomi saranno utilizzati per indicare a GameMaker quale set di animazioni utilizzare in qualsiasi momento del gioco. Ad esempio, se il vostro sprite è un gioco di piattaforme, potreste avere un set di "salto", un set di "corsa" e un set di "inattività", tutti contenuti nell'unico sprite e potete switch passare da un set all'altro nel vostro gioco usando questa funzione.
L'animazione viene riprodotta in loop, come da impostazione predefinita.

Si noti che l'impostazione di una nuova animazione in questo modo reimposta il parametro image_index a 0, facendo ripartire la nuova animazione dall'inizio della sua animazione loop. Se questo non è il comportamento desiderato, si può impostare un valore di "mix" usando il comando skeleton_animation_mix() e quando si imposta una nuova animazione, questa verrà interpolata in modo uniforme con quella precedente. Si dovrebbe anche assicurarsi di avere un keyframe all'inizio di ogni animazione, per evitare che i dati vengano trattenuti dalle animazioni precedenti, a meno che non si intenda usare tracce di animazione separate.

Sintassi:

skeleton_animation_set(animname, [loop]);

Argomento Tipo Descrizione
animname Stringa The name (a string) of the animation set to use.
loop Booleano Se l'animazione deve essere loop o meno (il valore predefinito è true)

Restituzione:

N/A

Esempio:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

Il codice precedente cambierà il set di animazione in uso in quello "salto" quando si preme il tasto spazio, ma solo se il set correntemente usato non è già "salto".
