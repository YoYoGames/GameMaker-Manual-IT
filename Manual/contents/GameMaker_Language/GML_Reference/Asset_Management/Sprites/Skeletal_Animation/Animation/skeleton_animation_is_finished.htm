scheletro_animazione_è_finito

Con questa funzione è possibile verificare se l'animazione scheletrica corrente non in loop ha terminato la riproduzione sulla traccia indicata.

Sintassi:

skeleton_animation_is_finished(track);

Argomento Tipo di dati Descrizione
traccia Reale L'indice della traccia di animazione

Restituzione:

Boolean

Esempio:

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

Il codice qui sopra controlla innanzitutto se l'animazione sulla traccia 1 ha terminato la riproduzione e memorizza il risultato in una variabile locale _finished. Poi mostra un messaggio di debug nel caso in cui l'animazione abbia terminato la riproduzione.
