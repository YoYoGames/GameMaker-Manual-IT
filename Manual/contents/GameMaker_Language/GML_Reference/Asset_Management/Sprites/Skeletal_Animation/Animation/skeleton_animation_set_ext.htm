skeleton_animation_set_ext

Quando si imposta un set di animazione per l'animazione scheletrica sprite, gli viene assegnata la traccia 0 internamente. Tuttavia, è possibile assegnare altri set ad altre tracce (1 e oltre) e in questo modo far sì che sprite riproduca due (o più) animazioni contemporaneamente. Si noti che sprite dovrebbe essere stato progettato nel programma di animazione tenendo presente questo aspetto, in modo che l'animazione "base" nella traccia 0 abbia keyframes solo per alcune ossa, mentre le tracce successive abbiano keyframes per le altre. Ad esempio, se si dispone di un'animazione di "camminata" e le sue keyframes influiscono solo sulle gambe sprite, si può avere una serie di animazioni della testa e delle braccia che si possono impostare rispettivamente nelle tracce 1 e 2, consentendo di avere un movimento di base e di adattarlo in tempo reale durante il gioco.
L'animazione viene riprodotta in loop, come da impostazione predefinita.

Sintassi:

skeleton_animation_set_ext(animname, track, [loop]);

Argomento Tipo Descrizione
animname Stringa The name (a string) of the animation set to use.
track Reale The track for the animation set to use.
loop Booleano Se l'animazione deve essere loop o meno (il valore predefinito è true)

Restituzione:

N/A

Esempio:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

Il codice sopra riportato cambierà il set di animazione utilizzato dalla traccia 1 nel set "bodyfight" o nel set "bodywalk" quando si preme il pulsante sinistro del mouse.
