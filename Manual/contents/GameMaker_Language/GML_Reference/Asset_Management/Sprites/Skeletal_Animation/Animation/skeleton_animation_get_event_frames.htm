skeleton_animation_get_event_frames

Questa funzione può essere utilizzata per recuperare tutti i fotogrammi dell'evento dato, nell'animazione data. Si fornisce il nome dello scheletro dell'animazione (come string, come definito nel programma usato per creare l'animazione, o come restituito dalla funzione skeleton_animation_get()) e il nome dell'evento. La funzione restituisce un array con i fotogrammi dell'evento.

Se il nome dell'evento specificato non esiste, la funzione restituirà un array con -1 come primo (e unico) elemento.

Sintassi:

skeleton_animation_get_event_frames(anim_name, event_name);

Argomento Tipo Descrizione
anim_name Stringa The animation name to get the frames of.
event_name Stringa The event name to get the frames for.

Restituzione:

Array of Reals

Esempio:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

Il codice precedente ottiene i fotogrammi per l'evento "Passo". Se viene restituito un array con i fotogrammi validi, loops lo attraversa e controlla se il fotogramma corrente è uguale a uno qualsiasi dei fotogrammi dell'array. In tal caso, riproduce l'effetto sonoro del passo e interrompe loop.
