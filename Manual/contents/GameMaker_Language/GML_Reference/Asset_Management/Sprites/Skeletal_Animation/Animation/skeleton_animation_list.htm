lista_animazione_scheletro

Con questa funzione si può popolare una lista DS (pre-creata) con tutti i nomi delle animazioni incluse come parte dell'animazione scheletrica sprite. I nomi saranno strings e potranno poi essere utilizzati nelle altre funzioni di animazione per questi tipi di sprite.

Sintassi:

skeleton_animation_list(sprite, list);

Argomento Tipo Descrizione
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID elenco DS The ID of the DS list to populate with the animation names.

Ritorna:

N/A

Esempio:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

Il codice di cui sopra crea una lista DS di tutti i nomi delle animazioni per sprite usate dall'istanza. Poi loops attraverso questi e imposta il valore di mix per tutti loro con la prima animazione a 0.5.

Indietro: Animazione
Il prossimo: skeleton_animation_clear
