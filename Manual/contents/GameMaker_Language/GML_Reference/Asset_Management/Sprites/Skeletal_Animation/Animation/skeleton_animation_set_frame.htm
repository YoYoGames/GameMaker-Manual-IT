skeleton_animation_set_frame

Questa funzione imposta l'animazione assegnata alla traccia data al fotogramma fornito. L'indice del fotogramma deve essere un valore reale compreso tra 0 e il numero di fotogrammi dell'animazione (che si può ottenere usando la funzione skeleton_animation_get_frames). Se si fornisce un valore al di fuori di questo intervallo, l'animazione verrà ripetuta per compensare la differenza, ad esempio se l'animazione ha 5 fotogrammi e si imposta il fotogramma a 8, il fotogramma effettivo mostrato sarà 3.

Sintassi:

skeleton_animation_set_frame(track, index);

Argomento Tipo Descrizione
track Reale The animation track to set the frame of.
index Reale The new index for the animation frame.

Restituzione:

N/A

Esempio:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

Il codice precedente imposta l'indice dei fotogrammi per l'animazione nella traccia 0 alla velocità negativa dell'immagine quando la variabile globale "pause" è true (essenzialmente mettendo in pausa l'animazione).
