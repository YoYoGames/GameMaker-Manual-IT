scheletro_animazione_chiaro

Questa funzione cancella la traccia di animazione specificata da tutte le animazioni, pronta per essere riassegnata.

Sintassi:

skeleton_animation_clear(track);

Argomento Tipo Descrizione
track Real The animation track to clear.

Ritorna:

N/A

Esempio:

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

Il codice di cui sopra cancellerà la traccia 1 dell'animazione se viene premuto il pulsante destro del mouse.

Indietro: Animazione
Il prossimo: skeleton_animation_get_frames
