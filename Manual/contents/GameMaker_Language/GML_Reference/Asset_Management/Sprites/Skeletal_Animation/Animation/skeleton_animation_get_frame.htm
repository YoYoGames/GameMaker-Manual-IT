scheletro_animazione_get_frame

Questa funzione restituisce il numero di fotogramma dell'animazione sulla traccia di animazione specificata. La funzione restituirà -1 se nessuna animazione è assegnata alla traccia specifica indicata.

Sintassi:

skeleton_animation_get_frame(track);

Argomento Tipo Descrizione
track Real The animation track to get the frame of.

Ritorna:

Real

Esempio:

var frame = skeleton_animation_get_frame(0);
if (frame > 0 && frame < 2)
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Il codice di cui sopra controllerà il fotogramma corrente dell'animazione assegnata alla traccia 0 e poi farà scoppiare alcune particelle da un punto al centro del bounding box totale per il sprite.
