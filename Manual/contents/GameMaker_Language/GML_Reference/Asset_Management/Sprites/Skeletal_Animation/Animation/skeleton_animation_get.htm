scheletro_animazione_get

Con questa funzione potete ottenere il set di animazione corrente utilizzato dalla vostra animazione scheletrica sprite. Il valore di ritorno è un string, che sarà il nome del set come lo avete definito nel vostro programma di animazione scheletrica.

Sintassi:

skeleton_animation_get();

Ritorna:

Esempio:

if keyboard_check_pressed(vk_space)
{
    if (skeleton_animation_get() != "jump")
    {
        skeleton_animation_set("jump");
    }
}

Il codice di cui sopra cambierà il set di animazione in uso nel set "jump" quando il tasto spazio viene premuto, ma solo se il set correntemente usato non è già "jump".
