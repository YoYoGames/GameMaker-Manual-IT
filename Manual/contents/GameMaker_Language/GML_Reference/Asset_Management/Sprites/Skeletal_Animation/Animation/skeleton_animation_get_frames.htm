scheletro_animazione_get_frames

Questa funzione può essere usata per recuperare il numero di fotogrammi che una data animazione scheletrica ha. Si fornisce il nome dell'animazione dello scheletro (come string, come definito nel programma usato per fare l'animazione, o come restituito usando la funzione skeleton_animation_get()) e la funzione restituisce i fotogrammi che ha come valore intero. La funzione restituisce 0 se l'animazione specificata non esiste.

Sintassi:

skeleton_animation_get_frames(anim_name);

Argomento Tipo Descrizione
anim_name Stringa The animation name to get the frames of.

Ritorna:

Real

Esempio:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Il codice di cui sopra otterrà il numero di fotogrammi nell'animazione e poi imposterà sprite all'ultimo fotogramma e fermerà l'animazione.
