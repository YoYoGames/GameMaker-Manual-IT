skeleton_animation_is_looping

Con questa funzione si può verificare se l'animazione corrente sulla traccia data è impostata su loop.

Sintassi:

skeleton_animation_is_looping(track);

Argomento Tipo Descrizione
traccia Reale L'indice della traccia di animazione

Restituzione:

Boolean

Esempio:

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

Il codice precedente controlla se l'animazione sulla traccia 0 è impostata su loop. In tal caso, viene visualizzato un messaggio di debug.
