scheletro_animazione_get_duration

Questa funzione restituisce il tempo necessario per l'esecuzione dell'animazione data prima di tornare all'inizio. Il valore di ritorno è in secondi.

Sintassi:

skeleton_animation_get_duration(animname);

ArgomentoTipo Descrizione
animname The name (a string) of the animation set to use.

Ritorna:

Esempio1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Il codice qui sopra imposterà il image_index al valore corretto per l'animazione scheletrica attualmente assegnata sprite.

Esempio2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Il codice qui sopra disegnerà l'animazione scheletrica data sprite usando il delta-time per impostare il fotogramma da disegnare.
