Sprites

Gli sprite sono generalmente le rappresentazioni visive di objects all'interno del gioco che state creando, anche se possono essere usati per molti altri scopi. Un sprite è una singola immagine o un insieme di immagini che, se riprodotte una dopo l'altra, assomigliano a un'animazione in loop. Sprites può essere creato in molti formati, con il "tradizionale" sprite costituito da fotogrammi di immagini pixel, importati in GameMaker come file *.png. Tuttavia è anche possibile importare sprites vettoriali che sono stati salvati nel formato *.swf, o animazioni scheletriche sprites che sono state salvate come file *.json con file di accompagnamento *.atlas e *.png (per maggiori informazioni su questi tipi di file immagine e su come importarli in GameMaker si veda la sezione Importare sprites non bitmap).

Poiché sprites è parte integrante di qualsiasi gioco, GameMaker ha un gran numero di funzioni e variabili che possono essere utilizzate per ottenere informazioni su specifici sprites, o per cambiare quelli che sono nel vostro gioco, o anche per caricare sprites personalizzati su runtime da un file immagine sorgente.
