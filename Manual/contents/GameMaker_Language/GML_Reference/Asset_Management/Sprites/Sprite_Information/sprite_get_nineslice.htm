sprite_get_nineslice

Questa funzione restituisce la struttura Nine Sl ice per un sprite, contenente tutte le sue proprietà Nine Slice come impostate nell'editor di sprite o usando sprite_set_nineslice(). Il contenuto di questa struct è dettagliato in questa pagina.

Se il sprite fornito non ha un Nine Slice struct assegnato ad esso, un nuovo struct con proprietà Nine Slice predefinite sarà creato per il sprite e restituito.

Cambiare qualsiasi valore in questo struct modificherà le proprietà Nine Slice del sprite originale, influenzando qualsiasi futura chiamata di disegno fatta con quel sprite.

Sintassi:

sprite_get_nineslice(ind);

ArgomentoTipo Descrizione
ind The index of the sprite from which the Nine Slice struct will be retrieved

Ritorna:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Esempio:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Il codice qui sopra recupera il Nine Slice struct da un sprite, abilita il Nine Slicing per esso e imposta gli offset di guida.
