sprite_get_name

Questa funzione restituirà il nome come stringa del sprite specificato. Questo nome è quello che è stato specificato per il sprite nell'albero delle risorse della finestra principale di GameMaker. Si prega di notare che questo è solo un string e non può essere utilizzato per fare riferimento al sprite direttamente - per questo è necessario l'indice dello sprite. È possibile, tuttavia, utilizzare questo string per ottenere l'indice dello sprite utilizzando il string restituito insieme alla funzione asset_get_index().

Sintassi:

sprite_get_name(index);

Argomento Tipo Descrizione
index Asset The index of the sprite to get the name of.

Restituisce

String

Esempio:

str = sprite_get_name(sprite_index);

Il codice di cui sopra otterrà il nome dell'indice sprite per l'istanza che esegue il codice e memorizzerà il valore di ritorno nella variabile "str".
