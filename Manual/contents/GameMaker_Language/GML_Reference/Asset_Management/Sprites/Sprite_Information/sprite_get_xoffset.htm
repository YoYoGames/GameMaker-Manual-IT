sprite_get_xoffset

Quando si definisce un sprite nell'editor di sprite, è necessario impostare l'origine per quel sprite. Questo è il punto in cui il sprite sarà "attaccato" o "disegnato" quando usato con un'istanza. Questa funzione restituisce l'offset relativo per l'asse x dell'origine basato sull'angolo superiore sinistro che è la posizione 0,0, con un +x che è a destra e un -x che è a sinistra di questo. L'immagine seguente illustra questo:

Sintassi:

sprite_get_xoffset(index);

ArgomentoTipo Descrizione
index The index of the sprite to find the xoffset of.

Restituisce

Esempio:

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = sprite_get_xoffset(sprite_index);
}

Il codice di cui sopra assicurerà che un'istanza sia mantenuta all'interno di room in base a sprite (quindi sprite è sempre visibile).
