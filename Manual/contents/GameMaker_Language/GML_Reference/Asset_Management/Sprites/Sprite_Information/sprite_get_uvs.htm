sprite_get_uvs

Questa funzione restituisce un array con le coordinate UV e altri dati per il texture della sottoimmagine sprite sulla pagina texture. La funzione restituisce un array con i seguenti 8 elementi:

  • [0] = sinistra
  • [1] = top
  • [2] = destra
  • [3] = fondo
  • [4] = quantità di pixel che il compilatore asset ha tagliato dal lato sinistro di sprites
  • [5] = quantità di pixel che il compilatore asset ha tagliato dal lato superiore di sprites
  • [6] = percentuale normalizzata di dati di pixel della larghezza originale sprites che è stata salvata nella pagina texture
  • [7] = percentuale normalizzata di dati di pixel dell'altezza originale sprites che è stata salvata nella pagina texture

Questa matrice può poi essere utilizzata in altre funzioni di disegno, in particolare quando si lavora in 3D o si utilizzano le funzioni primitive 2D, così come quando si lavora con le funzioni Shader.

NOTA: Questa funzione non funziona con sprites vettoriale o con l'animazione dello scheletro sprites.

Sintassi:

sprite_get_uvs(sprite, subimage);

ArgomentoTipo Descrizione
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

Restituisce

(1D, 8 elements)

Esempio:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Il codice precedente memorizzerà le coordinate UV per il dato sprite in un array locale e poi assegnerà i valori alle variabili di istanza.
