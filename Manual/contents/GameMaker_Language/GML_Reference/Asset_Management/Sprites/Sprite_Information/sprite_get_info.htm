sprite_get_info

Questa funzione serve a recuperare le informazioni per il dato sprite. Si fornisce un indice sprite (che può essere un asset aggiunto attraverso l'Asset Browser o uno sprite aggiunto a runtime) e la funzione restituisce una struct con le seguenti variabili:

Strutture di informazioni sugli sprite
Variabile Tipo Descrizione
larghezza reale Larghezza dello sprite (in pixel)
altezza reale Altezza dello sprite (in pixel)
xoffset reale L'offset/origine X dello sprite (in pixel)
yoffset reale L'offset/origine Y dello sprite (in pixel)
trasparente booleano true se sprite è contrassegnato come trasparente, altrimenti false
(Questo può essere specificato solo attraverso sprite_add() o funzioni simili e sarà false per sprites creato in IDE)
liscio booleano true se sprite è contrassegnato come liscio, altrimenti false
(Questo può essere specificato solo attraverso sprite_add() o funzioni simili, e sarà false per sprites creato nella IDE)
tipo reale Il tipo di sprite:
0 - Bitmap (Regular sprites)
1 - SWF
2 - Dorso
bbox_left reale Posizione del bordo sinistro del rettangolo di selezione (in pixel)
bbox_top reale Posizione del bordo superiore del rettangolo di selezione (in pixel)
bbox_destra reale Posizione del bordo destro del rettangolo di selezione (in pixel)
bbox_bottom reale Posizione del bordo inferiore del rettangolo di selezione (in pixel)
nome stringa Il nome dello sprite
num_subimages reale Il numero di sotto-immagini (o fotogrammi) dello sprite
maschera_uso booleano true se questo sprite utilizza una maschera di collisione, altrimenti false
num_maschere reale Il numero di maschere in questo sprite (sarà maggiore di 1 se sprite utilizza maschere per-frame)
nineslice struttura La struttura Nine Slice per questo sprite, oppure undefined se non ha dati di nove slice
messaggi array Array di messaggi di broadcast per questo sprite, dove ogni messaggio di broadcast è un struct contenente informazioni sul messaggio (maggiori informazioni in "Dati generali Sprite ")
frame_info array Array di fotogrammi per questo sprite, dove ogni fotogramma è un struct contenente informazioni sulla sua tempistica (maggiori informazioni in "Dati generali Sprite ")

Questa variabile aggiuntiva è disponibile solo per gli sprite Bitmap (normali):

Strutture di informazioni sugli sprite
Variabile Tipo Descrizione
cornici array Array di fotogrammi per questo sprite, dove ogni fotogramma è un struct contenente informazioni sulla sua texture (maggiori informazioni in "Dati Bitmap Sprite ")

Queste variabili aggiuntive sono disponibili solo per gli sprite di Spine:

Strutture di informazioni sugli sprite
Variabile Tipo Descrizione
num_atlante reale Il numero di atlanti utilizzati textures
atlante_testo array Array di ID texture utilizzati per l'atlante
premoltiplicato booleano true se questo sprite è contrassegnato come premoltiplicato, altrimenti false
nomi_animazione array Array contenente i nomi di ogni animazione di questo sprite
nomi_pelle array Array contenente i nomi di ogni pelle di questo sprite
ossa array Array contenente structs per ogni osso di questo sprite (maggiori informazioni in "Dati della colonna vertebrale Sprite ")
slot array Array contenente structs per ogni slot di questo sprite (maggiori informazioni in "Dati della colonna vertebrale Sprite ")

La funzione restituirà undefined se il dato sprite non esiste. Si noti inoltre che le informazioni restituite in questo struct devono essere considerate di sola lettura, poiché la modifica di una qualsiasi di queste variabili non avrà effetto sul sprite.

Le sezioni seguenti contengono informazioni sugli array e su structs inclusi nel file struct restituito, in base al tipo sprite:

Dati generali Sprite

Questa sezione contiene informazioni sulle variabili incluse in struct per tutti i tipi di sprite.

La variabile messages è un array che contiene informazioni sui messaggi di trasmissione esistenti nel dato sprite. Ogni messaggio di trasmissione in questo array è un struct contenente le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
telaio reale La tempistica di questo messaggio di trasmissione dall'inizio dell'animazione (in fotogrammi)
messaggio stringa La stringa del messaggio di trasmissione

La variabile frame_info è un array che contiene informazioni sui tempi dei fotogrammi dello sprite. Ogni fotogramma di questo array è un struct contenente le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
telaio reale La tempistica per l'inizio di questo fotogramma (in fotogrammi)
durata reale La durata di questo fotogramma (in fotogrammi)
indice_immagine reale L'indice dell'immagine di questa cornice

Bitmap Sprite Dati

Questa sezione contiene informazioni sulle variabili incluse in struct per Bitmap sprites (sprites che non sono Spine o di tipo SWF sprites).

La variabile frames è un array che contiene informazioni sui textures fotogrammi dello sprite. Ogni fotogramma di questo array è un struct contenente le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
x reale La posizione X di questa cornice nella pagina texture (in pixel)
y reale La posizione Y di questa cornice nella pagina texture (in pixel)
w reale La larghezza logica del riquadro (in pixel) utilizzata internamente
h reale L'altezza logica della cornice (in pixel) utilizzata internamente
struttura reale L'ID della pagina texture per questo frame
larghezza_originale reale La larghezza originale del fotogramma (in pixel)
altezza_originale reale L'altezza originale del fotogramma (in pixel)
larghezza_di_coltura reale La larghezza effettiva del riquadro sulla pagina texture dopo il ritaglio e il ridimensionamento (poiché GameMaker ritaglia automaticamente lo spazio vuoto intorno a un'immagine e la ridimensiona se non si adatta).
altezza_di_coltura reale L'altezza effettiva della cornice sulla pagina texture dopo il ritaglio e il ridimensionamento.
x_offset reale Lo spostamento X dal bordo sinistro dell'immagine originale al bordo sinistro della sezione ritagliata.
y_offset reale Lo spostamento Y dal bordo superiore dell'immagine originale al bordo superiore della sezione ritagliata.

Dati sulla colonna vertebrale Sprite

Questa sezione contiene informazioni sulle variabili incluse nel sito struct per Spine sprites.

La variabile bones è un array che contiene informazioni su tutte le ossa del dato Spine sprite. Ogni osso di questo array è un struct contenente le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
genitore stringa Il nome dell'osso genitore, o undefined se questo osso non ha un genitore.
nome stringa Il nome di questo osso
indice reale L'indice di questo osso
lunghezza reale La lunghezza di questo osso
x reale La posizione X di questo osso
y reale La posizione Y di questo osso
rotazione reale La rotazione di questo osso
scala_x reale (Interno a Spine) Valore di scala su X
scala_y reale (Interno a Spine) Valore di scala su Y
cesoia_x reale (Interno a Spine) Valore di taglio su X
cesoia_y reale (Interno a Spine) Valore di taglio su Y
modalità_di_trasformazione reale (Interno a Spine) La modalità di trasformazione

NOTA: Per ulteriori informazioni sulle variabili interne di Spine, consultare la documentazione di Spine.

La variabile slots è un array che contiene informazioni su tutti gli slot del dato Spine sprite. Ogni slot di questo array è un struct contenente le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
nome stringa Il nome dello slot
indice reale L'indice dello slot
osso stringa Il nome dell'osso dello slot, oppure "(none)" se non esiste un osso per questo slot.
attaccamento stringa Nome dell'allegato
rosso reale Componente rossa del colore dello slot (0-1)
verde reale Componente verde del colore dello slot (0-1)
blu reale Componente blu del colore dello slot (0-1)
alfa reale Componente alfa del colore dello slot (0-1)
blendMode reale (Interno a Spine) Modalità di fusione per lo slot
rosso_scuro* reale Componente rossa del colore scuro della fessura (0-1)
verde_scuro* reale Componente verde del colore scuro della fessura (0-1)
blu_scuro* reale Componente blu del colore scuro della fessura (0-1)
dark_alpha* stringa Componente alfa del colore scuro dello slot (0-1)

* NOTA: Le variabili di colore scuro sono disponibili solo se lo slot ha attivato la funzione Tint Black.

Sintassi:

sprite_get_info(index);

Argomento Tipo Descrizione
index Asset di sprite The index of the sprite to get the information for.

Restituzioni

Sprite Info Struct (or undefined)

Esempio:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

Il codice precedente ottiene le informazioni struct per l'istanza sprite, quindi controlla che non sia undefined e che il suo tipo sia 0 (cioè che sia una bitmap sprite). In tal caso, ottiene l'array dei messaggi di trasmissione per quel sprite e poi esegue un for loop per stampare ogni messaggio di trasmissione (insieme al suo frame) nel registro di output.
