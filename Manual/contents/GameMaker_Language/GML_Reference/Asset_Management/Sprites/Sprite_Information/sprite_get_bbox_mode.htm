sprite_get_bbox_mode

Questa funzione restituisce la "modalità" corrente per i calcoli del bounding box. Si fornisce l'indice sprite del sprite da controllare, e la funzione restituirà una delle costanti mostrate di seguito.

Sintassi:

sprite_get_bbox_mode(ind);

ArgomentoTipo Descrizione
ind The index of the sprite to check.

Restituisce

Bounding Box Mode Constant

Costante Descrizione
bboxmode_automatic Automatico - Il riquadro di delimitazione sarà calcolato automaticamente, in base alle impostazioni di tolleranza per lo sprite
bboxmode_fullimage Immagine completa - Il riquadro di delimitazione sarà impostato per utilizzare l'intera larghezza e altezza di sprite, indipendentemente dalla tolleranza e dai pixel "vuoti".
bboxmode_manual Manuale - Il riquadro di delimitazione è stato impostato manualmente su valori definiti dall'utente (sia nell'editor sprite, sia usando la funzione sprite_set_bbox())

Esempio:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Quanto sopra controlla la modalità bbox per l'attuale sprite e se non è automatic, allora la imposta a quel valore.
