La struttura a nove fette

Una struct Nine Slice può essere creata usando sprite_nineslice_create(), o recuperata da un sprite usando sprite_get_nineslice(). Questo struct conterrà proprietà che possono essere modificate e applicate a un sprite usando sprite_set_nineslice().

Se recuperato da un sprite, cambiare i valori in questo struct avrà effetto sul sprite originale.

Le seguenti proprietà saranno disponibili in questa struttura:

Strutturazione a nove fette
Variabile Tipo Descrizione
enabled Booleano Questa booleana sarà true se Nine Slicing è abilitato per sprite, e false se è disabilitato. Si può modificare per abilitare o disabilitare il Nine Slicing per sprite.
left Real Questo è l'offset per la guida sinistra sull'asse x, rispetto al bordo sinistro di sprite.
right Real Questo è l'offset per la guida destra sull'asse x, rispetto al bordo destro di sprite.
top Real Questo è l'offset per la guida superiore sull'asse y, rispetto al bordo superiore del sprite.
bottom Real Questo è l'offset per la guida inferiore sull'asse y, rispetto al bordo inferiore del sprite.
tilemode Array di Costante del modo Tiles Questo restituisce un array contenente le modalità Tile per le fette del bordo e del centro.

Modalità delle mattonelle

Il Tile Mode di ogni slice può essere letto o modificato usando queste costanti come indice dell'array:

Costante di bordo a nove fette
Costante Descrizione
nineslice_left La fetta di bordo sinistro
nineslice_top La fetta di bordo superiore
nineslice_right La fetta di bordo destro
nineslice_bottom La fetta di bordo inferiore
nineslice_centre La fetta centrale

La modalità Tile per una fetta sarà (o può essere impostata su) una delle seguenti costanti:

Costante del modo Tile
Costante Descrizione
nineslice_stretch La fetta sarà allungata
nineslice_repeat La fetta sarà ripetuta
nineslice_mirror La fetta sarà ripetuta con il mirroring
nineslice_blank La fetta non sarà allungata o ripetuta, risultando in un'area vuota dopo di essa
nineslice_hide La fetta non apparirà affatto

Poiché tilemode è un array, è necessario utilizzare la sintassi dell'array per cambiare il Tile Mode di una slice. L'esempio di codice qui sotto imposta il Tile Mode della slice superiore a Repeat:

Esempio:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
