sprite_index

Questa variabile restituisce l'indice dell'attuale sprite per l'istanza, o -1 se l'istanza non ha alcun sprite associato ad essa. Potete cambiarla per dare all'istanza un diverso sprite dandole il nome di un sprite dall'albero delle risorse o usando una variabile che ha un sprite caricato esternamente indicizzato in esso. Cambiare il sprite non cambia l'indice del frame attualmente visibile, quindi se cambiate il sprite sul frame numero 3, il nuovo sprite sarà disegnato con quel frame visibile (assumendo che abbia lo stesso numero di frame).

Sintassi:

sprite_index;

Ritorna:

Sprite Asset

Esempio:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

Il codice di cui sopra loop attraverso tutte le istanze di "obj_Check" controllando una linea di collisione tra loro e l'istanza che esegue il codice. Il sprite di queste istanze sarà cambiato a seconda del valore di ritorno (vero o falso) per la linea di collisione.
