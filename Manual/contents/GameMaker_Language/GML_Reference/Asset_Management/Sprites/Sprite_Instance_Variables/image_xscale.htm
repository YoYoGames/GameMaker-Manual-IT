scala_immagine

Questo valore imposta la scala orizzontale applicata al sprite che è stato assegnato all'istanza corrente. Una scala di 1 indica nessun ridimensionamento (1:1), valori più piccoli ridimensioneranno (0.5, per esempio, dimezzerà la larghezza di sprite), valori più grandi ridimensioneranno e valori negativi capovolgeranno sprite e lo ridimensioneranno a meno che il valore usato sia esattamente -1 (nel qual caso sprite è semplicemente capovolto intorno alla sua origine senza ridimensionamento).

Sintassi:

image_xscale;

Ritorna:

(single precision floating point value)

Esempio:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Il codice di cui sopra scala il sprite e poi una volta che è scalato a 5 volte la sua dimensione originale, viene creata una nuova istanza di un altro object e l'istanza viene distrutta.
