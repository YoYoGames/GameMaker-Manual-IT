sprite_xoffset

Questa variabile di sola lettura restituisce l'xoffset locale (la componente x dell'origine come definita nell'editor sprite ) del sprite che è stato assegnato all'istanza. Questo xoffset è restituito in pixel e dipenderà dal image_xscalecosì che (per esempio) se un quadrato 32x32 sprite è definito come avente l'origine x a 16, quando quel sprite è scalato di 2 questa variabile restituirà 32. Se avete bisogno dell'offset x non scalato dovreste usare sprite_get_xoffset().

Sintassi:

sprite_xoffset;

Ritorna:

Real

Esempio:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

Il codice di cui sopra controlla se il bordo sinistro di sprite dell'istanza è fuori da room e se lo è sposta l'istanza in modo che l'intero sprite sia visibile.
