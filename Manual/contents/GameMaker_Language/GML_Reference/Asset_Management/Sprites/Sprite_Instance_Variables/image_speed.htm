velocità immagine

Questa variabile determina la velocità con cui GameMaker farà scorrere le sotto-immagini per l'istanza corrente sprite. Il valore di velocità dato è un moltiplicatore, con 1 che è il valore predefinito, e impostandolo a 0,5 si dimezza la velocità dell'animazione - come impostato nello Sprite Editor o nell'Image Editor - mentre impostandolo a 2 la raddoppia. Se il sprite utilizzato non ha sotto-immagini, questa variabile non avrà alcun effetto.

Sintassi:

image_speed;

Ritorna:

(single precision floating point value)

Esempio:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

Il codice qui sopra crea un'istanza di object "obj_Explosion" e imposta la sua image_speed a 0,5 (dimezzando effettivamente la velocità di corsa).
