immagine_alfa

Questa variabile è usata per ottenere o impostare il valore alfa per il sprite. L'alfa è sempre calcolata come un valore tra 0 e 1 dove 0 è completamente trasparente e 1 è completamente opaco. Si prega di notare che per i cambiamenti in questa variabile per essere visibili, l'istanza dovrebbe avere o nessun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata utilizzando una delle funzioni di disegno estese come draw_self() o draw_sprite_ext().

Sintassi:

image_alpha;

Ritorna:

Real

Esempio:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

Il codice di cui sopra ridurrà lentamente l'image_alpha fino a raggiungere lo 0.
