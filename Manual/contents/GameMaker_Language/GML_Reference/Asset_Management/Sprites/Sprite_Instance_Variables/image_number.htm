numero immagine

Questa variabile di sola lettura può essere usata per ottenere il numero di sotto-immagini in un sprite che è stato assegnato a un'istanza (se avete bisogno del numero di sotto-immagini per un sprite diverso da quello assegnato all'istanza dovreste usare sprite_get_number()). Si noti che quando c'è (per esempio) 1 sotto-immagine questa variabile restituirà "1" ma l'image_index di quella sotto-immagine è 0.

Sintassi:

image_number;

Ritorna:

(single precision floating point value)

Esempio:

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

Il codice di cui sopra controlla per vedere se sprite si sta animando, e se lo fa controlla se l'image_index corrente è maggiore del numero di sotto-immagini e se lo è distrugge l'istanza.

Indietro: Variabili di istanza dello sprite
Il prossimo: image_speed
