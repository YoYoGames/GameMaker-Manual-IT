bbox_left

Questa variabile di sola lettura restituisce la posizione (lungo l'asse x) all'interno di room del rettangolo di delimitazione di sinistra per l'istanza, dove il rettangolo di delimitazione è definito dalla massima larghezza e altezza della maschera per l'istanza (come impostato da sprite_index o da mask_index). Anche quando un sprite ha una maschera di collisione precisa, il bounding box esiste ed è usato per certe cose, e quindi si può usare questa variabile per trovarlo. Si noti che quando l'istanza non ha sprite assegnato il valore restituito sarà lo stesso della posizione X dell'istanza.

Sintassi:

bbox_left;

Ritorna:

(integer)

Esempio:

if bbox_left < 0
{
    x = sprite_xoffset;
}

Il codice di cui sopra controlla se il bounding box dell'istanza è fuori da room e se lo è imposta l'istanza in una nuova posizione.
