altezza_sprite

Questa variabile di sola lettura restituisce l'altezza di sprite che è stata assegnata all'istanza. Questa altezza è restituita in pixel e dipenderà dal image_yscale. Se avete bisogno dell'altezza non scalata dovreste usare sprite_get_height().

Sintassi:

sprite_height;

Ritorna:

Real

Esempio:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

Il codice di cui sopra controlla l'altezza di sprite assegnata all'istanza che esegue il codice rispetto all'altezza della risorsa sprite e se non è la stessa, resetta image_yscale a 1,

Indietro: Variabili di istanza dello sprite
