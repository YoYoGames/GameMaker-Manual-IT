angolo_immagine

Questo valore imposta l'angolo (rotazione) di sprite ed è misurato in gradi, con la destra a 0º, in alto a 90º, a sinistra a 180º e in basso a 270º. Imposta questa variabile a 0 per ripristinare il disegno di sprite come definito nell'editor di sprite. Si prega di notare che per i cambiamenti in questa variabile per essere visibili, l'istanza dovrebbe avere o nessun evento di disegno (e quindi GameMaker disegnerà di default il sprite) o essere disegnata utilizzando una delle funzioni di disegno estese come draw_self() o draw_sprite_ext().

Sintassi:

image_angle;

Ritorna:

Real

Esempio:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Il codice di cui sopra ruoterà il sprite dell'istanza per puntare sempre alla posizione del mouse.
