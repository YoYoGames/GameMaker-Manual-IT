larghezza_sprite

Questa variabile di sola lettura restituisce la larghezza di sprite che è stata assegnata all'istanza. Questa larghezza è restituita in pixel e dipenderà dal image_xscale. Se avete bisogno della larghezza non scalata dovreste usare sprite_get_width().

Sintassi:

sprite_width;

Ritorna:

Real

Esempio:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Il codice di cui sopra controlla la larghezza di sprite assegnata all'istanza che esegue il codice rispetto alla larghezza della risorsa sprite e se non è la stessa, resetta image_xscale a 1,

Indietro: Variabili di istanza dello sprite
Il prossimo: sprite_height
