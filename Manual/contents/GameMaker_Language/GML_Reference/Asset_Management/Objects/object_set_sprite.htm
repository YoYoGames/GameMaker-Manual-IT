object_set_sprite

Con questa funzione potete impostare l'indice sprite di uno specifico object. Questo significa che tutte le istanze di questo object che sono create dopo che lo sprite_index è stato cambiato saranno create con questo nuovo sprite_index, mentre le istanze che sono già nel room possono non essere interessate.

Si prega di notare che questa non è una funzione di istanza! Potete impostare l'indice sprite delle singole istanze usando la variabile sprite_index e quindi avere dieci istanze tutte con un sprite diverso dal object da cui sono state create, e anche se cambiate l'indice sprite del object usando questa funzione, tutte le istanze che sono attualmente nel room rimarranno come erano, e solo le istanze create dopo aver chiamato la funzione inizieranno con il nuovo sprite.

Sintassi:

object_set_sprite( index, spr );

Argomento Tipo Descrizione
index Oggetto Attività The index of the object to change.
spr Sprite Asset The sprite to assign to the object.

Ritorna:

N/A

Esempio:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

Il codice qui sopra imposterà l'indice object "obj_Player" sprite a valori diversi a seconda del room in cui si trova attualmente l'istanza che esegue il codice. Quindi crea un'istanza di "obj_Player".
