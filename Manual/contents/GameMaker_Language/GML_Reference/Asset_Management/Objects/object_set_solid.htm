oggetto_set_solido

Con questa funzione è possibile contrassegnare uno specifico object come solido o no (per maggiori informazioni sul flag "solid" vedere la sezione sull'Object Editor). Questo significa che tutte le istanze di questo object che vengono create dopo che solid è stato cambiato saranno create con questo nuovo stato, mentre le istanze che sono già nel room potrebbero non essere interessate.

Notate che questa non è una funzione di istanza! Potete impostare il flag solido delle singole istanze usando la variabile solid e quindi avere dieci istanze tutte contrassegnate come solide anche se il object in questione ha il flag impostato a false nel IDE, e anche se cambiate il flag solido del object a true usando questa funzione, tutte le istanze che attualmente nel room rimarranno come erano, e solo le istanze create dopo aver chiamato la funzione inizieranno con il solido contrassegnato come true.

Sintassi:

object_set_solid( index, solid );

Argomento Tipo Descrizione
index Oggetto Attività The index of the object to change.
solid Booleano The new solidity of the object (true=solido, false=non solido).

Ritorna:

N/A

Esempio:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

Il codice di cui sopra controlla il flag solido di object "obj_Block" e se è falso allora viene cambiato per essere contrassegnato come vero.
