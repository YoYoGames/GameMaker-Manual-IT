nome dell'oggetto

Questa funzione restituisce il nome come stringa del object specificato. Questo nome è quello che è stato specificato per il object nel Browser Asset della finestra principale di GameMaker. Si prega di notare che questa è solo una stringa e non può essere usata per fare riferimento direttamente a object - per questo è necessario l'indice dell'oggetto. È possibile, tuttavia, utilizzare questo string per ottenere l'indice dell'oggetto utilizzando il string restituito insieme alla funzione asset_get_index().

Sintassi:

object_get_name(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

str = object_get_name(object_index);

Il codice di cui sopra otterrà il nome dell'indice object per l'istanza che esegue il codice e memorizzerà il valore di ritorno nella variabile "str".
