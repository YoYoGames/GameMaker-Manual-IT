oggetto_get_sprite

Questa funzione vi dirà se il object che state controllando ha un sprite o no, e se ce l'ha allora restituirà l'indice di quel sprite, o -1 se non ce l'ha. Notate che questa non è una funzione di istanza! Potete avere un object senza sprite mentre un'istanza di quello stesso object può averne uno e viceversa, o possono anche avere diversi sprites. Potete impostare una singola istanza sprite usando la variabile sprite_index variabile di istanza.

Sintassi:

object_get_sprite(obj);

ArgomentoTipo Descrizione
obj The index of the object to check

Ritorna:

Sprite Asset

Esempio:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

L'esempio precedente controllerà il sprite_index dell'istanza contro il sprite del object_index dell'istanza. Se non sono uguali, allora assegnerà all'istanza lo stesso sprite dell'indice object.
