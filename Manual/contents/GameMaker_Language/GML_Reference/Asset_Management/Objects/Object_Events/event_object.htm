oggetto_evento

Questa variabile di sola lettura restituisce object indice dell'istanza che sta eseguendo l'evento da controllare.

Sintassi:

event_object;

Ritorna:

Object Asset

Esempio:

global.obj = event_object;

Il codice precedente memorizza l'indice object dell'istanza che esegue l'evento in una variabile globale.

Indietro: Generazione di Object Eventi
Il prossimo: event_type
