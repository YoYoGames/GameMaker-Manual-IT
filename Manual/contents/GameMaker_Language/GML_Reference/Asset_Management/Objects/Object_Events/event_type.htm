tipo_evento

Questa variabile di sola lettura restituisce il tipo di evento attualmente in esecuzione, che può essere una delle seguenti costanti:

  • ev_creare
  • ev_destroy
  • ev_step
  • ev_allarme
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collisione
  • ev_altro
  • ev_gesture
  • ev_draw


Per una lista completa delle costanti che sono legate ai diversi eventi vedi event_perform(), e se si dovesse avere bisogno del numero dell'evento (cioè: il sotto evento), si dovrebbe controllare il event_number.

Sintassi:

event_type;

Ritorna:

Event Type Constant

Esempio:

show_debug_message("Current Event = " + string(event_type));

Il codice di cui sopra mostrerà un messaggio di debug con il tipo di evento attualmente in corso.
