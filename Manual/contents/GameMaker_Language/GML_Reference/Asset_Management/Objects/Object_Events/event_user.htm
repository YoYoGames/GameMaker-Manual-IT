utente_evento

Con questa funzione dite all'istanza di eseguire le azioni o il codice che è stato inserito in uno dei 16 eventi definiti dall'utente. Questi eventi possono essere chiamati solo in questo modo, o usando la funzione event_perform funzione.

Sintassi:

event_user(numb);

Argomento Tipo Descrizione
numb Real The number of User Event to call, between 0 and 15.

Ritorna:

N/A

Esempio:

event_user(4);

Questo chiamerebbe l'evento 4 definito dall'utente.

Indietro: Generazione di Object Eventi
Il prossimo: event_action
