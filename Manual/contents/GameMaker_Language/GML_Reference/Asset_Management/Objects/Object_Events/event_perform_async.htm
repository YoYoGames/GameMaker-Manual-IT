evento_perform_async

Questa funzione viene utilizzata per eseguire uno qualsiasi degli eventi asincroni forniti in GameMaker. Si fornisce la costante dell'evento Async (indicata nella tabella sottostante) e una mappa DS che sarà disponibile nell'evento Async chiamato nella variabile async_load.

Costante del tipo di evento asincrono
Costante Descrizione
caricamento immagine web Evento immagine caricata
ev_async_web Evento HTTP
ev_async_dialog Evento di dialogo
ev_async_web_iap Evento acquisto in-App
ev_async_web_cloud Evento cloud
ev_async_web_networking Evento di rete
ev_async_web_steam Evento del vapore
ev_async_social Evento sociale
ev_async_push_notification Evento di notifica push
ev_async_save_load Salvare/Caricare l'evento
registrazione audio ev_async_audio_recording Evento di registrazione audio
ev_async_audio_playback Evento riproduzione audio
ev_async_system_event Evento di sistema

Gli eventi non asincroni possono essere chiamati usando event_perform().

Si noti che la mappa DS specificata nel secondo argomento non deve essere distrutta manualmente, poiché sarà automaticamente distrutta dalla funzione dopo l'esecuzione dell'evento Async.

Sintassi:

event_perform_async(type, ds_map);

Argomento Tipo Descrizione
type Costante del tipo di evento asincrono The type of event to perform (see the table above).
ds_map ID della mappa DS The DS map to use as async_load nell'evento chiamato.

Ritorna:

N/A

Esempio:

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

Il codice di cui sopra crea una mappa DS e la popola con voci personalizzate da leggere nell'evento Async. Poi esegue l'evento Async Social con la mappa appena creata passata come async_load per l'evento chiamato.
