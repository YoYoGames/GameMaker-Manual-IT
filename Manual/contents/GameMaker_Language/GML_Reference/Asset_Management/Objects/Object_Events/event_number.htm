numero evento

Questa variabile di sola lettura restituisce il numero dell'evento attualmente chiamato, dove il numero si riferisce effettivamente al "sotto-evento" dell'evento, cioè: per l'evento passo il numero dell'evento potrebbe essere una qualsiasi delle costanti ev_step_normal, ev_step_begin, o ev_step_end. Per una lista completa delle costanti che sono disponibili per i sotto-eventi specifici vedi event_perform(), e se si dovesse aver bisogno di conoscere l'evento principale stesso, si dovrebbe controllare il event_type.

Sintassi:

event_number;

Ritorna:

Event Number Constant

Esempio:

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

Il codice di cui sopra potrebbe essere chiamato da un script e usato per mostrare messaggi di debug che informano su quale evento viene attualmente attivato.
