azione_evento DEPRECATO

Questa variabile di sola lettura restituisce l'indice dell'azione attualmente in esecuzione, a partire da 0 nelle versioni precedenti di GameMaker. Tuttavia, questa è ora una variabile obsoleta in GameMaker. È stata lasciata solo per il supporto legacy, e restituirà sempre 0 poiché tutte le azioni sono concatenate insieme per migliorare la velocità di esecuzione.

Sintassi:

event_action;

Ritorna:

Real

Esempio:

num = event_action;

Il codice precedente memorizza il numero dell'azione corrente nella variabile "num".

Indietro: Generazione di Object Eventi
Il prossimo: event_number
