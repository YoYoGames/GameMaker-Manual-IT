oggetto_esiste

Questa funzione restituisce se il valore specificato è un object asset (true) o no (false). Si noti che questo controlla se un object è presente nel Browser Asset, e non effettivamente nel gioco room. Per questo dovreste usare la funzione instance_exists().

Sintassi:

object_exists(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

Il codice qui sopra legge i dati da un buffer che è stato ricevuto via rete. Poiché contiene un ID object (usato per lo spawning delle istanze), dobbiamo prima assicurarci che il valore non sia invalido e che il object esista davvero. Per questo chiamiamo object_exists() in una condizione, e se questo è vero, creiamo un'istanza di quel object nella posizione ricevuta.
