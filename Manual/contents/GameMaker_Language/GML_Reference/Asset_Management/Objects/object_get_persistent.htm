oggetto_get_persistente

Questa funzione vi dirà se il object che state controllando è stato contrassegnato come "persistente" o no. Un object persistente è uno che farà sì che qualsiasi istanza di esso venga portata da room a room a meno che non venga esplicitamente distrutta. Si noti che questa non è una funzione di istanza! Quindi, si può avere un object persistente e un'istanza non persistente dello stesso object e viceversa. Potete impostare il flag persistente di una singola istanza usando la variabile persistent variabile di istanza.

Sintassi:

object_get_persistent(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

Il codice di cui sopra controllerà l'istanza in esecuzione per vedere se è persistente o no, così come controllerà l'indice object dell'istanza per vedere se è contrassegnato come persistente o no. Se l'istanza non è persistente ma l'indice object è contrassegnato come persistente, imposterà "persistent" a true per quell'istanza.
