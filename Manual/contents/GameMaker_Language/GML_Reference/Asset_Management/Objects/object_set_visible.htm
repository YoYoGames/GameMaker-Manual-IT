oggetto_set_visibile

Con questa funzione è possibile contrassegnare uno specifico object come visibile o meno (per maggiori informazioni sul flag "visible" si veda la sezione sull'Object Editor). Questo significa che tutte le istanze di questo object che vengono create dopo che visible è stato cambiato saranno create con questo nuovo stato, mentre le istanze che sono già nel room potrebbero non essere interessate.

Si prega di notare che questa non è una funzione di istanza! Potete impostare il flag di visibilità delle singole istanze usando la variabile visible e quindi avere dieci istanze tutte contrassegnate come visibili anche se il object in questione ha il flag impostato a false nel IDE, e anche se cambiate il flag visibile del object a true usando questa funzione, tutte le istanze che attualmente nel room rimarranno come erano, e solo le istanze create dopo aver chiamato la funzione inizieranno con visible contrassegnato come true.

Sintassi:

object_set_visible( index, vis );

Argomento Tipo Descrizione
index Oggetto Attività The index of the object to change.
vis Booleano The new visibility of the object (true=visibile, false=invisibile).

Ritorna:

N/A

Esempio:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

Il codice qui sopra controlla il flag visibile di object "obj_Block" e se è falso allora viene cambiato per essere contrassegnato come vero.
