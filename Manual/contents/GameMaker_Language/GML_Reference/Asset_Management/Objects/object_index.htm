indice oggetto

Questa variabile di sola lettura restituisce l'indice del object da cui è stata creata l'istanza. Questo non è lo stesso del nome object, che è un string e può essere trovato usando object_get_name()poiché questa funzione restituisce il numero di indice, che è un valore unico che GameMaker assegna ad ogni object al momento della creazione.

Sintassi:

object_index;

Ritorna:

Object Asset

Esempio:

obj_name = object_get_name(object_index);

Il codice di cui sopra userà il object_index per trovare il nome del object da cui è stata creata l'istanza corrente.
