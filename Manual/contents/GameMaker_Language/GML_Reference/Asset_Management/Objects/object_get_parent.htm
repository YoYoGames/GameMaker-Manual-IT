oggetto_get_parent

Questa funzione vi darà l'indice object di qualsiasi genitore che è stato assegnato al object specificato, o altrimenti restituisce -100 per mostrare che il object non ha un genitore assegnato ad esso, o -1 se il object controllato non esiste. Per maggiori informazioni sui genitori si veda la sezione sull'editor di oggetti.

Sintassi:

object_get_parent(obj);

ArgomentoTipo Descrizione
obj The index of the object to check

Ritorna:

Object Asset

Esempio:

par = object_get_parent(object_index);

Il codice di cui sopra otterrà il genitore dell'indice object per l'istanza che esegue il codice e memorizzerà il valore di ritorno nella variabile "par".
