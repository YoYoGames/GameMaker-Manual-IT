oggetto_get_physics

Questa funzione vi dirà se il object che state controllando è stato segnalato come "abilitato alla fisica" - nel qual caso restituirà true, - o no - nel qual caso restituirà false.

Sintassi:

object_get_physics(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

if object_get_physics(object_index)
{
    phy_active = true;
}

Il codice di cui sopra controllerà l'istanza che lo esegue per vedere se il object da cui è creato è abilitato alla fisica, e se lo è attiva la simulazione fisica per l'istanza.
