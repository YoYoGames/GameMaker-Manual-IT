object_set_mask

Con questa funzione si può impostare il mask_index di uno specifico object (per maggiori informazioni sulle maschere si veda The Object Editor). Questo significa che tutte le istanze di questo object che sono create dopo che la maschera è stata cambiata saranno create con questa nuova maschera, mentre le istanze che sono già in room potrebbero non essere interessate. Si può impostare a -1 per rimuovere una maschera sprite e quindi tornare alla maschera definita per la sprite della object (o nessuna maschera se non è stato scelto sprite ).

Si prega di notare che questa non è una funzione di istanza! Potete impostare l'indice della maschera delle singole istanze usando la variabile mask_index e quindi avere dieci istanze tutte con una maschera diversa da quella di object da cui sono state create, e anche se cambiate l'indice della maschera di object usando questa funzione, tutte le istanze che sono attualmente in room rimarranno come erano, e solo le istanze create dopo aver chiamato la funzione inizieranno con la nuova maschera.

Sintassi:

object_set_mask(index, spr);

Argomento Tipo Descrizione
index Oggetto Attività The index of the object to change.
spr Sprite Asset The new sprite to use as the object's mask.

Ritorna:

N/A

Esempio:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

Il codice qui sopra controlla il valore della variabile globale e poi cambia l'indice della maschera di object "obj_Platform" se è uguale a dieci.
