oggetto_get_mask

Questa funzione vi dirà se object che state controllando ha un indice di maschera o no, e se ce l'ha allora restituirà l'indice di quella maschera (che è un sprite asset), o -1 se non ce l'ha. Notate che questa non è una funzione di istanza! Potete avere un object senza maschera mentre un'istanza dello stesso object può averne una e viceversa, o possono anche avere maschere diverse. Potete impostare l'indice della maschera di una singola istanza usando la variabile mask_index variabile di istanza.

Sintassi:

object_get_mask(obj);

ArgomentoTipo Descrizione
obj The index of the object to check

Ritorna:

Sprite Asset

Esempio:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

L'esempio precedente controllerà l'indice della maschera dell'istanza contro la maschera dell'indice_oggetto dell'istanza. Se non sono uguali, allora assegnerà all'istanza la stessa maschera che ha l'indice object.
