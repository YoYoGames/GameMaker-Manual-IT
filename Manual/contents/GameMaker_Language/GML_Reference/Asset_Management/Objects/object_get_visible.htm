oggetto_get_visibile

Questa funzione vi dirà se il object che state controllando è stato segnalato come "visibile" (esegue il suo evento di disegno) o no (non esegue il suo evento di disegno). Notate che questa non è una funzione di istanza! Quindi, potete avere un'istanza visibile object e un'istanza invisibile dello stesso object e viceversa. Potete impostare la visibilità di una singola istanza usando la variabile visible variabile di istanza.

Sintassi:

object_get_visible(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

Il codice di cui sopra controllerà l'istanza in esecuzione per vedere se è visibile o meno, così come controllerà l'indice object dell'istanza per vedere se è contrassegnata come visibile o meno. Se l'istanza non è visibile ma l'indice object è contrassegnato come on, imposterà "visible" a true per quell'istanza.
