oggetto_get_solido

Questa funzione vi dirà se il object che state controllando è stato contrassegnato come "solido" o no. Un solido object genera un evento di collisione speciale quando si usa il sistema di collisione tradizionale (cioè: il mondo fisico è spento). Si prega di notare che questa non è una funzione di istanza! Quindi, si può avere un solido object e un'istanza normale dello stesso object e viceversa. È possibile impostare un flag di solid di un'istanza individuale usando la variabile solid variabile di istanza.

Sintassi:

object_get_solid(obj);

ArgomentoTipo Descrizione
obj The index of the object to check.

Ritorna:

Esempio:

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

Il codice di cui sopra controllerà l'istanza in esecuzione per vedere se è solida o no, così come controllerà l'indice object dell'istanza per vedere se è contrassegnato come solido o no. Se l'istanza non è solida ma l'indice object è contrassegnato come solido, imposterà "solid" a true per quell'istanza.
