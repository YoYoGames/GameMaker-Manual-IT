timeline_moment_add_script

Con questa funzione potete aggiungere dinamicamente una funzione di script a Timelines in qualsiasi "momento" all'interno di quella linea temporale, dove un "momento" è l'equivalente di un tick (o passo) di gioco. In questo modo potete creare una nuova linea temporale usando la funzione timeline_add() e aggiungere diversi comportamenti in qualsiasi punto, o semplicemente modificare una risorsa di linea temporale creata in precedenza con nuovi comportamenti. Si noti che la funzione non può richiedere argomenti aggiuntivi quando si usa questa funzione, e se la si usa per modificare un Timeline asset presente nel Browser Asset, allora tutte le istanze che usano questo timeline saranno interessate, e la modifica durerà fino alla fine del gioco (chiamando game_restart() non si resetterà nemmeno la modifica).

Sintassi:

timeline_moment_add_script(ind, step, script);

Argomento Tipo Descrizione
ind Timeline Asset The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Funzione Script The index of the script function to add into the moment.

Ritorna:

N/A

Esempio:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

Il codice qui sopra creerà una nuova linea temporale e memorizzerà il suo indice nella variabile "global.tl". Quindi aggiungerà tre scripts alla linea del tempo, scelti a caso, a intervalli di un minuto.

Indietro: Linee temporali
Il prossimo: timeline_moment_clear
