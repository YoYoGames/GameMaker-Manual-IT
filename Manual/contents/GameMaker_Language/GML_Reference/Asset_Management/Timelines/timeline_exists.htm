esiste una linea temporale

Con questa funzione potete controllare e vedere se una linea temporale esiste (restituisce true) o no (restituisce false). Questo è particolarmente utile quando si crea dinamicamente Timelines usando la funzione timeline_add() ma dovreste notare che se cercate una variabile non inizializzata (che altrimenti sarebbe assegnata all'indice di una linea temporale) verrà lanciato un errore.

Sintassi:

timeline_exists(ind);

ArgomentoTipo Descrizione
ind The index of the time line to check for.

Ritorna:

Esempio:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Il codice qui sopra controlla se una linea temporale esiste ed è indicizzata nella variabile globale "tl" e se è così cancella quella linea temporale.
