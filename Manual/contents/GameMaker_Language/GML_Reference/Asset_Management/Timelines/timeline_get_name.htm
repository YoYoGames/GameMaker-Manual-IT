nome della linea temporale

Questa funzione può essere usata per ottenere il nome di una linea temporale come string. se la linea temporale è stata creata dinamicamente usando la funzione timeline_add() il nome restituito avrà il formato formato "__newtimelineN" dove "N" è il numero della linea temporale (partendo da 0). Nota che questo è solo un string e non può essere usato per fare riferimento direttamente a timeline - per questo avresti bisogno dell'indice della linea temporale. Puoi, comunque, usare questo string per ottenere l'indice della linea temporale usando il string restituito insieme alla funzione asset_get_index().

Sintassi:

timeline_get_name(ind);

ArgomentoTipo Descrizione
ind The index of the time line to check the name of.

Ritorna:

Esempio:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

Il codice di cui sopra ottiene il nome dell'attuale timeline assegnato e lo invia alla console.

Indietro: Linee temporali
Il prossimo: timeline_add
