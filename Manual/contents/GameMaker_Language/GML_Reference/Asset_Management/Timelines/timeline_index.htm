indice_della_tempistica

Questa variabile contiene l'indice del timeline attualmente associato all'istanza. Puoi impostarlo su una particolare linea temporale per usare quella, o impostarlo a -1 per smettere di usare una linea temporale per l'istanza (se nessuna linea temporale è definita per l'istanza, viene restituito anche -1). Notate che questo non fa partire la linea temporale - per quello usate la variabile timeline_running.

Sintassi:

timeline_index;

Ritorna:

Timeline Asset

Esempio:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

Il codice di cui sopra assegnerà all'istanza che esegue il codice una linea temporale indicizzata nella variabile "global.tl" se questa linea temporale esiste.

Indietro: Linee temporali
Il prossimo: timeline_running
