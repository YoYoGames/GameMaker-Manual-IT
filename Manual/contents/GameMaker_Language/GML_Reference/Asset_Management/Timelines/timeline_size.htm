dimensione della linea temporale

Con questa funzione si può ottenere il numero totale di momenti attivi per un timeline (un momento "attivo" è uno che ha codice o GML Visual aggiunto ad esso). Questo può essere utile quando si crea un timelines dinamico usando gli elementi timeline_moment_add_script() e timeline_moment_clear() perché si può controllare se un dato timeline ha il numero corretto di momenti attivi o nessuno.

Sintassi:

timeline_size(ind);

ArgomentoTipo Descrizione
ind The index of the timeline get the information from.

Ritorna:

Esempio:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Il codice qui sopra controlla la dimensione data timeline, e se restituisce 0 (cioè: il timeline non ha momenti attivi) aggiungerà una funzione script da usare in un momento casuale all'interno del timeline indicizzato nella variabile globale "tl".
