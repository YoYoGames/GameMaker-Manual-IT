caricamento audio_gruppo_progresso

Questa funzione controlla il progresso del caricamento di un gruppo audio e restituisce un valore (approssimativo) tra 0 e 100.

Sintassi:

audio_group_load_progress(groupID);

Argomento Tipo Descrizione
groupID ID gruppo audio The index of the audio group to check (as defined in the Audio Groups Window)

Ritorna:

Real

Esempio:

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

Il codice qui sopra controlla una variabile e se restituisce true allora verrà disegnato del testo che mostra il progresso dell'audio che viene caricato.

Indietro: Gruppi audio
Il prossimo: audio_group_name
