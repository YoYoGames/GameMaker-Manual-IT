audio_set_master_gain

Con questa funzione è possibile impostare il valore assoluto per il volume globale di tutti i suoni e la musica per un ascoltatore specifico. L'indice di default dell'ascoltatore è 0, ma puoi usare la funzione audio_get_listener_info() per ottenere i diversi indici disponibili per la piattaforma di destinazione. Il valore di guadagno è basato su una scala lineare da 0 (silenzioso) a 1 (pieno volume) e influenzerà il volume relativo di tutti i suoni e la musica riprodotti dall'interno del vostro gioco attraverso quell'ascoltatore.

Sintassi:

audio_set_master_gain(listenerIndex, gain);

Argomento Tipo Descrizione
listenerIndex ID ascoltatorereale o audio The index of the listener to set the gain on.
gain Real Value for the global volume (0 to 1).

Ritorna:

N/A

Esempio:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

Il codice qui sopra loops attraverso gli ascoltatori disponibili e poi imposta il loro guadagno principale a 0,75.
