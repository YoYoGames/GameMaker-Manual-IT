audio_sound_get_track_position

Questa funzione otterrà la posizione (in secondi) all'interno del file audio per il suono da riprodurre. Il suono può essere solo una singola istanza di un suono (l'indice per i singoli suoni da riprodurre può essere memorizzato in una variabile quando si usa l'opzione audio_play_sound() o audio_play_sound_at() ).

Sintassi:

audio_sound_get_track_position(index);

Argomento Tipo Descrizione
index ID dell'istanza sonora The index of the sound to get the play position of.

Ritorna:

Real

Esempio:

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

Il codice di cui sopra controlla una traccia per ottenere la sua posizione iniziale e se non è 0 secondi la imposta a 0 secondi.
