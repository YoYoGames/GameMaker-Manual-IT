audio_resume_syncgroup

Questa funzione riprende il gruppo di sincronizzazione dato se è in riproduzione ed è stato precedentemente messo in pausa (usando la funzione audio_pause_sync_group). L'indice del gruppo è il valore restituito quando avete creato il gruppo utilizzando la funzione audio_create_sync_group().

NOTA: Questa funzionalità non è disponibile per la piattaforma di destinazione HTML5.

Sintassi:

audio_resume_sync_group(group_index);

Argomento Tipo Descrizione
group_index ID del gruppo di sincronizzazione audio The group index to resume.

Ritorna:

N/A

Esempio:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if global.Pause
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

Il codice di cui sopra controlla la pressione del tasto "P", e se ne viene rilevata una, attiva la variabile "global.Pause" e la controlla per mettere in pausa o riprendere il gruppo di sincronizzazione indicizzato nella variabile "sg".
