audio_sync_group_is_playing

Questa funzione può essere utilizzata per verificare se l'audio di un gruppo sincronizzato è in riproduzione. È necessario fornire l'ID del gruppo di sincronizzazione, come restituito dalla funzione audio_create_sync_group().

Questa funzionalità non è disponibile per la piattaforma di destinazione HTML5.

Sintassi:

audio_sync_group_is_playing(group_index);

Argomento Tipo Descrizione
group_index ID gruppo di sincronizzazione audio The group index to check.

Restituzione:

Boolean

Esempio:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

Il codice precedente controlla se il gruppo di sincronizzazione group_one è attualmente in riproduzione e, in caso affermativo, il gruppo viene fermato.
