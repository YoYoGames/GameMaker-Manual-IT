audio_sync_group_get_track_pos

Questa funzione restituisce l'attuale posizione di riproduzione del gruppo di sincronizzazione dato. L'indice del gruppo è il valore restituito quando avete creato il gruppo utilizzando la funzione audio_create_sync_group()e il valore di ritorno è il tempo in secondi in cui le tracce sono state riprodotte.

NOTA: Questa funzionalità non è disponibile per la piattaforma di destinazione HTML5.

Sintassi:

audio_sync_group_get_track_pos(group_index);

Argomento Tipo Descrizione
group_index ID del gruppo di sincronizzazione audio The group index to get the position of.

Ritorna:

Real

Esempio:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

Il codice qui sopra ottiene la posizione temporale per il gruppo di sincronizzazione indicizzato nella variabile "sg", poi usa questo valore per disegnare il tempo giocato sullo schermo.
