audio_listener_get_data

Questa funzione creerà una mappa DS e la popolerà con i valori di posizione, velocità e orientamento per l'ascoltatore dato. L'indice predefinito dell'ascoltatore è 0, ma è possibile utilizzare la funzione audio_get_listener_info() per ottenere i diversi indici disponibili per la piattaforma di destinazione. Se si fornisce un indice dell'ascoltatore errato, la funzione restituirà -1.

NOTA Siete responsabili della distruzione della mappa DS restituita utilizzando la funzione appropriata.

La mappa DS conterrà le seguenti chiavi:

  • "x" - La posizione x dell'ascoltatore
  • "y" - La posizione y dell'ascoltatore
  • "z" - La posizione z dell'ascoltatore
  • "vx" - La velocità sull'asse x dell'ascoltatore
  • "vy" - La velocità dell'asse y dell'ascoltatore
  • "vz" - La velocità dell'asse z dell'ascoltatore
  • "lookat_x" - La componente x del vettore look at dell'ascoltatore
  • "lookat_y" - La componente y del vettore look at dell'ascoltatore
  • "lookat_z" - La componente z del vettore look at dell'ascoltatore
  • "up_x" - La componente x del vettore up dell'ascoltatore
  • "up_y" - La componente y del vettore up dell'ascoltatore
  • "up_z" - La componente z del vettore up dell'ascoltatore

Sintassi:

audio_listener_get_data(index);

Argomento Tipo Descrizione
index ID ascoltatorereale o audio The listener to get the data for (default is 0).

Ritorna:

DS Map ID

Esempio:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

Il codice di cui sopra controlla il numero di ascoltatori disponibili poi loops attraverso di loro e se la loro posizione x non è 0, imposta la loro posizione a 0, 0, 0.
