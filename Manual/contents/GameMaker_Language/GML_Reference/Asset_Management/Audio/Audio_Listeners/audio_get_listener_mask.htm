audio_get_listener_mask

Questa funzione restituisce i dati bit-maschera che definiscono l'attuale maschera predefinita (globale) per gli ascoltatori audio.

Sintassi:

audio_get_listener_mask();

Ritorna:

Esempio:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

Il codice di cui sopra ottiene i dati della maschera dell'ascoltatore corrente e li confronta con i dati memorizzati in una variabile globale. Se non sono uguali, il codice controlla il numero di ascoltatori disponibili poi loops attraverso di loro e ottiene i loro bit di maschera, che sono poi combinati per creare una singola maschera che viene applicata al sistema audio per definire gli ascoltatori globali.
