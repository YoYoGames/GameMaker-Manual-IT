audio_set_listener_mask

Quando usate più ascoltatori su un sistema, potete impostare la maschera di bit per un suono e farlo sentire solo dall'ascoltatore contrassegnato. Tuttavia, potete anche impostare la maschera globale usando questa funzione e tutti i suoni riprodotti normalmente saranno ascoltati dagli ascoltatori contrassegnati da questa maschera, senza la necessità di impostare la maschera per ogni suono individualmente.

Sintassi:

audio_set_listener_mask(mask);

Argomento Tipo Descrizione
mask Real The bit-mask data to set for the listeners

Ritorna:

N/A

Esempio:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Il codice di cui sopra controlla il numero di ascoltatori disponibili poi loops attraverso di loro ottiene i loro bit di maschera, che sono poi combinati per creare una singola maschera di bit che viene applicata all'ascoltatore globale.
