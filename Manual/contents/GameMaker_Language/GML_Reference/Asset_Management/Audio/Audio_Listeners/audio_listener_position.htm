posizione dell'ascoltatore audio

Con questa funzione puoi cambiare la posizione dell'ascoltatore all'interno dello spazio audio 3D. L'immagine di esempio qui sotto mostra la posizione predefinita dell'ascoltatore nello spazio audio:

NOTASe avete più ascoltatori, usate la funzione audio_listener_set_position().

Come potete vedere, la posizione di default è (0, 0, 0) ma normalmente usereste questa funzione per spostare l'ascoltatore in giro con il giocatore object all'interno del vostro gioco e quindi cambiare il modo in cui l'audio creato dagli emettitori viene ascoltato dal giocatore, per esempio, nell'immagine qui sotto di un gioco top down, l'istanza del giocatore imposta l'ascoltatore che farà sì che l'audio dai vari emettitori "cambi" mentre il giocatore si muove nel livello:

Sintassi:

audio_listener_position(x, y, z);

Argomento Tipo Descrizione
x Real The x position of the listener (default 0).
y Real The y position of the listener (default 0).
z Real The z position of the listener (default 0).

Ritorna:

N/A

Esempio:

if speed > 0
{
    audio_listener_position(x, y, 0);
}

Il codice qui sopra controlla se la velocità dell'istanza del giocatore è superiore a 0 e se lo è aggiorna l'ascoltatore audio alla posizione x/y corrente.
