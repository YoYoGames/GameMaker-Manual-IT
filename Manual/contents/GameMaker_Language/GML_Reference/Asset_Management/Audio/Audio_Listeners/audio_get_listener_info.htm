audio_get_listener_info

Questa funzione creerà una mappa DS e la popolerà con le informazioni per l'ascoltatore dato.

NOTA L'utente è responsabile della distruzione del DS Map restituito utilizzando la funzione appropriata.

La mappa DS conterrà le seguenti chiavi:

  • "name" - Il nome dell'ascoltatore, come string, con "default" che è il nome standard dell'ascoltatore sulla maggior parte delle piattaforme di destinazione
  • "mask" - La maschera di bit per l'ascoltatore
  • "index" - Il valore unico dell'indice dell'ascoltatore

Il valore della maschera può essere usato per impostare un suono o un emettitore per suonare da più ascoltatori contemporaneamente, semplicemente usando il bitwise or "|" per generare una maschera per il suono (vedi il codice di esempio qui sotto), mentre l'indice è usato per impostare le proprietà come la posizione o la velocità per un dato ascoltatore usando funzioni come audio_listener_set_position().

Sintassi:

audio_get_listener_info(num);

Argomento Tipo Descrizione
num ID ascoltatorereale o audio The listener number to get the data for.

Ritorna:

DS Map ID

Esempio:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Il codice di cui sopra controlla il numero di ascoltatori disponibili poi loops attraverso di loro ottiene i loro bit di maschera, che sono poi combinati per creare una singola maschera di bit che viene applicata all'ascoltatore globale.
