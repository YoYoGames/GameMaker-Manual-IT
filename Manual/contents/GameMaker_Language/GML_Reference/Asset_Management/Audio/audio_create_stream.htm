creare un flusso audio

Con questa funzione potete creare un nuovo indice sonoro che può poi essere usato nelle normali funzioni audio per trasmettere l'audio direttamente da una sorgente esterna di file OGG. La funzione richiede il nome del file (che può essere un file incluso, per esempio) e restituirà il nuovo indice del suono da usare. Notate che dopo che non avete più bisogno del suono dovreste chiamare la funzione audio_destroy_stream() con l'indice del suono per rimuoverlo dalla memoria, altrimenti potreste ottenere una perdita di memoria che rallenterà ed eventualmente farà crashare il gioco.

NOTA: Questa funzionalità non è disponibile per la piattaforma di destinazione HTML5.

Sintassi:

audio_create_stream(filename);

Argomento Tipo Descrizione
filename Stringa Path to the file (OGG only) to stream the audio from.

Ritorna:

Sound Asset

Esempio:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Il codice qui sopra crea un nuovo indice di suono nella variabile "snd" dal file dato, poi riproduce questo suono.

Indietro: Audio
Il prossimo: audio_destroy_stream
