audio_stop_sound

Questa funzione fermerà il suono dato se è attualmente in riproduzione. Il suono può essere una singola istanza di un suono (l'indice per i singoli suoni in riproduzione può essere memorizzato in una variabile quando si usa la funzione audio_play_sound() o audio_play_sound_at() ) o un suono asset, nel qual caso tutte le istanze del suono dato saranno fermate.

Sintassi:

audio_stop_sound(index);

Argomento Tipo Descrizione
index ID dellarisorsa sonora o dell'istanza sonora The index of the sound to stop.

Ritorna:

N/A

Esempio:

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

Il codice qui sopra controlla la variabile globale "SFX" e se restituisce false, fermerà il suono indicizzato nella variabile "snd_Waterfall" che è attualmente in riproduzione, e se restituisce true, loop il suono.
