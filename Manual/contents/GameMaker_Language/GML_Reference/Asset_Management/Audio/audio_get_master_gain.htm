audio_get_master_gain

Con questa funzione potete ottenere il valore assoluto del volume globale di tutti i suoni e della musica per un ascoltatore specifico. L'indice predefinito dell'ascoltatore è 0, ma è possibile utilizzare la funzione audio_get_listener_info() per ottenere i diversi indici disponibili per la piattaforma di destinazione. Il valore di guadagno restituito è basato su una scala lineare da 0 (silenzioso) a 1 (pieno volume). Nota che su alcune piattaforme puoi avere un guadagno maggiore di 1, anche se un valore di 1 è considerato "pieno volume" e qualsiasi valore maggiore può introdurre il clipping audio.

Sintassi:

audio_get_master_gain(listenerIndex);

Argomento Tipo Descrizione
listenerIndex ID ascoltatorereale o audio The index of the listener to get the gain of.

Ritorna:

Real

Esempio:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

Il codice sopra loops attraverso gli ascoltatori disponibili, controllando se il loro guadagno è 1 o no, e se non lo è lo imposta a 1 per ognuno di loro.
