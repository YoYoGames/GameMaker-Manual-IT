audio_è_pausa

Questa funzione controllerà il suono dato per vedere se è attualmente in pausa. Il suono può essere una singola istanza di un suono (l'indice per i singoli suoni in riproduzione può essere memorizzato in una variabile quando si usa la funzione audio_play_sound() o audio_play_sound_at() ) o un suono asset, nel qual caso verranno controllate tutte le istanze del suono dato e se qualcuna di esse è in pausa la funzione restituirà true altrimenti restituirà false.

Sintassi:

audio_is_paused(index);

Argomento Tipo Descrizione
index ID dellarisorsa sonora o dell'istanza sonora The index of the sound to check.

Ritorna:

Boolean

Esempio:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

Il codice qui sopra controlla se il suono indicizzato nella variabile "snd_Waterfall" è attualmente in pausa e se restituisce true allora la riproduzione del suono sarà ripresa.
