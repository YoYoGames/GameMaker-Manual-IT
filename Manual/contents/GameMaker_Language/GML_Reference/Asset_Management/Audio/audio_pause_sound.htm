audio_pausa_suono

Con questa funzione potete mettere in pausa qualsiasi suono che è attualmente in riproduzione. Il suono può essere una singola istanza di un suono (l'indice per i singoli suoni in riproduzione può essere memorizzato in una variabile quando si usa la funzione audio_play_sound() o audio_play_sound_at() ) o un suono asset, nel qual caso tutte le istanze del suono dato saranno messe in pausa.

Sintassi:

audio_pause_sound(index);

Argomento Tipo Descrizione
index ID dellarisorsa sonora o dell'istanza sonora The index of the sound to pause.

Ritorna:

N/A

Esempio:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

Il codice qui sopra controlla se viene premuto il tasto "P" della tastiera e se lo rileva imposta la variabile globale "Pause" su true o false e poi mette in pausa il suono indicizzato nella variabile "snd_Waterfall" o riprende il suono dal suo stato di pausa.
