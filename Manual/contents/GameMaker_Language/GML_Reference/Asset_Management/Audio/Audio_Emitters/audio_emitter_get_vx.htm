audio_emettitore_get_vx

Questa funzione restituisce la velocità attuale lungo l'asse x per l'emettitore audio dato.

Sintassi:

audio_emitter_get_vx(emitter);

Argomento Tipo Descrizione
emitter ID emettitore audio The index of the emitter to use.

Ritorna:

Real

Esempio:

if audio_emitter_get_vx(emitter_player) != 0
{
    audio_emitter_velocity(emitter_player, 0, 0, 0);
}

Il codice qui sopra controlla l'attuale velocità x di un dato emettitore e se non è uguale a 0, viene impostato a 0.
