creare_emettitore_audio

Questa funzione crea un nuovo emettitore audio e ne restituisce l'indice. Questo indice deve essere memorizzato in una variabile per tutte le altre funzioni che si riferiscono a questo emettitore e quando non è più necessario deve essere rimosso dalla memoria usando la funzione audio_emitter_free() per evitare perdite di memoria che potrebbero mandare in crash il gioco.

L'emettitore audio viene creato con le seguenti impostazioni predefinite:

  • Un guadagno e un passo di 1
  • Una maschera ascoltatore impostata solo sull'ascoltatore predefinito (1)
  • Una posizione di (0, 0, 0)
  • Una velocità pari a 0 (vx = 0, vy = 0, vz = 0)

Sintassi:

audio_emitter_create();

Restituzione:

Audio Emitter ID

Esempio:

s_emit = audio_emitter_create();

Il codice precedente crea un nuovo emettitore audio e assegna il suo indice alla variabile "s_emit".
