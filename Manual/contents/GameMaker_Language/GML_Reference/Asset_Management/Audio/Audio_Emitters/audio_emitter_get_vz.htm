emettitore audio_get_vz

Questa funzione restituisce la velocità attuale lungo l'asse z per l'emettitore audio dato.

Sintassi:

audio_emitter_get_vz(emitter);

Argomento Tipo Descrizione
emitter ID emettitore audio The index of the emitter to use.

Ritorna:

Real

Esempio:

if audio_emitter_get_vz(emitter_player) != 0
{
    audio_emitter_velocity(emitter_player, 0, 0, 0);
}

Il codice di cui sopra controlla la velocità z corrente di un dato emettitore e se non è uguale a 0, viene impostato a 0.
