posizione dell'emettitore audio

Con questa funzione potete cambiare la posizione di un emettitore audio all'interno dello spazio audio 3D. La posizione influenzerà il suono in modi diversi a seconda di dove l'ascoltatore è posizionato all'interno dello spazio audio (la posizione predefinita è (0, 0, 0). Vedere audio_listener_position() per maggiori informazioni), così per esempio se la posizione dell'emettitore è impostata a (100, 0, 0) e l'ascoltatore corrente è a (200, 0, 0) l'audio trasmesso dall'emettitore sembrerà essere sulla sinistra del campo audio. L'immagine qui sotto mostra una rappresentazione visiva degli emettitori e delle loro posizioni relative all'ascoltatore:

Sintassi:

audio_emitter_position(emitter, x, y, z);

Argomento Tipo Descrizione
emitter ID emettitore audio The index of the emitter to change.
x Real The x position.
y Real The y position.
z Real The z position.

Ritorna:

N/A

Esempio:

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

Il codice qui sopra controlla se la velocità dell'istanza è superiore a 0 e se lo è aggiorna l'emettitore audio indicizzato nella variabile "s_emit" alla posizione x/y corrente.
