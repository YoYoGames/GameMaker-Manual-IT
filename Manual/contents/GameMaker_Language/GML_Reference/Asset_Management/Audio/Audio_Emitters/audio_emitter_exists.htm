esiste un emettitore audio

Questa funzione restituisce se un emettitore audio esiste (true) o no (false). Notate che se l'indice che cercate non è stato inizializzato in precedenza, questa funzione causerà un errore poiché sta cercando degli indici inesistenti.

Sintassi:

audio_emitter_exists(index);

Argomento Tipo Descrizione
index ID emettitore audio The index of the emitter to check the existence of.

Ritorna:

Boolean

Esempio:

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

Il codice qui sopra controlla se esiste un emettitore, indicizzato nella variabile (precedentemente inizializzata) "s_emit". Se esiste, allora un suono viene riprodotto attraverso di esso, ma se non esiste, viene creato e poi il suono viene riprodotto.
