esiste l'audio

Questa funzione restituisce se un suono esiste (true) o no (false). Notate che se l'indice che cercate non è stato inizializzato in precedenza, questa funzione causerà un errore poiché sta cercando degli indici inesistenti asset. Il suono da controllare può essere sia una singola istanza di un suono (l'indice per i singoli suoni in riproduzione può essere memorizzato in una variabile quando si usa la funzione audio_play_sound() o audio_play_sound_at() ) o un suono asset.

Sintassi:

audio_exists(index);

Argomento Tipo Descrizione
index Asset The index of the sound to check the existence of.

Ritorna:

Boolean

Esempio:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

Il codice qui sopra controlla se esiste un suono. Se esiste, viene impostato per essere riprodotto in un loop.
