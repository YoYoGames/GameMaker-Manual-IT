audio_pausa_tutto

Con questa funzione potete mettere in pausa tutti i suoni che sono attualmente in riproduzione.

Sintassi:

audio_pause_all();

Ritorna:

Esempio:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

Il codice qui sopra controlla se viene premuto il tasto "P" della tastiera e se lo rileva imposta la variabile globale "Pause" su true o false e poi mette in pausa tutti i suoni o riavvia tutti i suoni precedentemente messi in pausa.
