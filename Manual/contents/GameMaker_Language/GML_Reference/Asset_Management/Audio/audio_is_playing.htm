l'audio è in riproduzione

Questa funzione controllerà il suono dato per vedere se è attualmente in riproduzione. Il suono può essere una singola istanza di un suono (l'indice per i singoli suoni in riproduzione può essere memorizzato in una variabile quando si usa la funzione audio_play_sound() o audio_play_sound_at() ) o un suono asset, nel qual caso verranno controllate tutte le istanze del suono dato e se qualcuna di esse sta suonando la funzione restituirà true altrimenti restituirà false. Si noti che questa funzione restituirà ancora true se il suono controllato è stato precedentemente messo in pausa usando la funzione audio_pause_sound() funzione.

Sintassi:

audio_is_playing(index);

Argomento Tipo Descrizione
index ID dellarisorsa sonora o dell'istanza sonora The index of the sound to check.

Ritorna:

Boolean

Esempio:

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

Il codice qui sopra controlla se il suono indicizzato nella variabile "snd_Waterfall" è attualmente in riproduzione e se restituisce false allora il suono sarà messo in loop nella sua posizione room, con una distanza di caduta di 300 e una priorità bassa.

Indietro: Audio
Il prossimo: audio_is_paused
