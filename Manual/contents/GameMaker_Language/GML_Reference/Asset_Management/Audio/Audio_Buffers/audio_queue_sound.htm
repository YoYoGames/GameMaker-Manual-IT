audio_queue_sound

Questa funzione aggiungerà i dati di un buffer nell'audio queue che avete precedentemente creato con la funzione audio_create_play_queue(). Specificate l'indice di queue a cui aggiungere, e l'ID di buffer da cui prendere i dati così come la posizione (offset) all'interno di buffer e il numero di byte da aggiungere. Una volta che avete aggiunto l'audio da un buffer a un queue, il buffer non può essere cancellato finché non avete prima liberato il queue usando la funzione audio_free_play_queue()e le proprietà di buffer dovrebbero corrispondere a quelle di queue a cui state aggiungendo il suono.

Sintassi:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Argomento Tipo Descrizione
queueIndex Audio Queue ID The index of the queue to add to.
bufferId ID del buffer The buffer to add to the queue.
bufferOffset Real The offset within the source buffer to start from.
bufferLength Real The length of the buffer (the number of the bytes in the buffer).

Ritorna:

N/A

Esempio:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Il codice di cui sopra verrebbe chiamato nell'evento di registrazione audio asincrono e assegna alcuni dati registrati a un buffer, che viene poi aggiunto a un audio queue. Questo viene poi riprodotto.
