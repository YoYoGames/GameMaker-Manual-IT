audio_creare_buffer_sound

Con questa funzione potete creare un nuovo suono dal contenuto di un buffer. Il buffer sarà stato creato in precedenza (vedete le funzioni buffer per i dettagli su come farlo), e vi saranno stati aggiunti o caricati dati. Quindi lo passate a questa funzione con il formato dei dati (solo buffer_u8 o buffer_s16 sono attualmente supportati), la frequenza di campionamento (che può essere tra 1000hz e 48000hz), e un offset in buffer da cui prendere i dati. Devi anche fornire il numero di campioni in buffer e i canali che il suono richiede. Questi canali sono definiti da una delle seguenti costanti:

Tipo di canale audio
Costante Descrizione
audio_mono Audio mono (canale singolo).
audio_stereo Audio stereo (doppio canale).
audio_3D Audio 3D (5.1).

Notate che dopo aver creato un suono, dovreste liberare l'indice del puntatore associato ad esso quando non è più necessario usando la funzione audio_free_buffer_sound(). Se non lo fate e poi riassegnate la variabile o cambiate rooms ecc... l'ID del suono sarà perso e avrete una perdita di memoria. Notate anche che non potete cancellare il buffer se è stato creato un qualsiasi suono da esso e il suono non è stato prima liberato. Quindi dovreste prima liberare il suono (o i suoni), poi cancellare buffer. Vale anche la pena notare che aggiungere qualcosa a buffer, o cambiare la dimensione di buffer, dopo che è stato creato un suono da esso, darà risultati inaspettati e non è raccomandato - una volta che avete iniziato a creare suoni da qualsiasi buffer non dovreste manipolarlo in nessun altro modo dopo.

Sintassi:

audio_create_buffer_sound(bufferId, bufferFormat, bufferRate, bufferOffset, bufferLength, bufferChannels);

Argomento Tipo Descrizione
bufferId ID del buffer The ID of the buffer to use.
bufferFormat Buffer Data Type Costante The format of the data in the buffer (buffer_u8 o buffer_s16).
bufferRate Real The sample rate of the data in the buffer.
bufferOffset Real The offset into the buffer to read the sample data from (in bytes).
bufferLength Real The length of the buffer (the number of bytes of audio data, excluding the header).
bufferChannels Tipo di canale audio The channels to use from one of the constants listed above.

Ritorna:

Sound Asset

Esempio:

var rate = 44100;
var hertz = irandom_range(220, 880);
var samples = 44100;

var bufferId = buffer_create(rate, buffer_fast, 1);
buffer_seek(bufferId, buffer_seek_start, 0);

var num_to_write = rate / hertz;
var length = buffer_get_size(bufferId);
var val_to_write = 1;

for (var i = 0; i < (samples / num_to_write) + 1; i++)
{
    for (var j = 0; j < num_to_write; j++)
    {
        buffer_write(bufferId, buffer_u8, val_to_write * 255);
    }
    val_to_write = (1 - val_to_write);
}

soundId = audio_create_buffer_sound(bufferId, buffer_u8, rate, 0, length, audio_stereo);

Quanto sopra crea un buffer e poi lo riempie proceduralmente di dati. Questi dati vengono poi utilizzati per creare un nuovo suono, che viene memorizzato nella variabile "soundId".
