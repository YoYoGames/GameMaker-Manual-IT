audio_get_name

Questa funzione restituirà il nome di una data risorsa audio come string. Il valore "indice" può essere quello della risorsa stessa (come si vede nel Browser di Asset ) o il valore ID unico che viene dato quando si riproduce il suono usando, per esempio, audio_play_sound(). Si noti che l'string restituito non è lo stesso dell'ID della risorsa e non può essere usato per accedere alla risorsa stessa, quindi dovrebbe essere usato solo per la visualizzazione o il controllo degli errori.

Sintassi:

audio_get_name(index);

ArgomentoTipo Descrizione
index The index of the sound to check.

Ritorna:

Esempio:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

Il codice qui sopra riproduce un suono casuale scelto da tre diverse risorse sonore, poi mostra un messaggio di debug con il suo nome.
