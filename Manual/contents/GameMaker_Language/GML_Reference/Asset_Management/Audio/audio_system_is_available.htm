il sistema audio è disponibile

Questa funzione può essere usata per controllare e vedere se il sistema audio è stato inizializzato, o se il contesto audio è in esecuzione. Su tutte le piattaforme, questa funzione restituirà true immediatamente dopo l'avvio del gioco quando il motore audio è inizializzato, tranne che sul target HTML5. Su HTML5, lo stato del contesto audio può cambiare in qualsiasi momento a seconda dell'input dell'utente, del browser utilizzato e della sua versione, quindi questa funzione può essere utilizzata per controllare e vedere se l'audio può essere riprodotto o meno. Se la funzione restituisce false (cioè: lo stato del contesto audio non è in esecuzione), allora solo i suoni non in streaming possono essere riprodotti (ma non è garantito, e si dovrebbe assumere che nessun audio può essere riprodotto), mentre se restituisce true allora tutto l'audio verrà riprodotto.

Sintassi:

audio_system_is_available();

Ritorna:

Esempio:

if audio_system_is_available()
{
    if audio_is_paused(global.Music)
    {
        audio_resume_sound(global.Music)
    }
    else
    {
        if !audio_is_playing(global.Music)
        {
            global.Music = audio_play_sound(snd_Music, 0, true);
        }
    }
}
else
{
    if audio_is_playing(global.Music)
    {
        audio_pause_sound(global.Music);
    }
}

Il codice di cui sopra mette in pausa/disattiva una traccia audio a seconda che il sistema audio sia inizializzato e disponibile o meno.
