font_get_info

Questa funzione è usata per recuperare informazioni per il dato font. Si fornisce un indice font (che può essere un asset aggiunto attraverso l'Asset Browser o un font aggiunto a runtime) e la funzione restituisce una struct con le seguenti variabili:

Struttura delle informazioni sui font
Nome della variabile Tipo di dati Descrizione
ascenderOffset Real L'offset massimo dalla linea di base alla parte superiore di font (in pixel)
size Real La dimensione approssimativa di font (in pixel)
spriteIndex Sprite Asset L'indice sprite per il font se è stato creato da uno sprite, altrimenti -1
texture Puntatore -1 se il font è stato creato da un sprite, altrimenti l'ID texture del font
name Stringa Il nome del font
bold Booleano true se il font è in grassetto, altrimenti false (vedi NOTA sotto)
italic Booleano true se il font è in corsivo, altrimenti false (vedi NOTA sotto)
glyphs Strutturazione del glifo del carattere Un struct contenente informazioni per ogni glifo in font (maggiori informazioni sono date di seguito)

NOTA Le variabili bold e italic riflettono solo le impostazioni dell'utente per font, quindi potrebbero non essere accurate per fonts che appare in grassetto o in corsivo per default.

Se il font fornito non esiste, la funzione restituirà undefined.

Notate anche che cambiare i valori di una qualsiasi di queste variabili, o di qualsiasi variabile contenuta all'interno di glyphs struct, non cambierà il modo in cui font è reso, quindi queste informazioni dovrebbero essere considerate di sola lettura.

Strutture dei glifi

La variabile glyphs nel struct restituito sarà un struct a sé stante, contenente informazioni per ogni glifo incluso nel font. Ogni variabile in questa struct sarà il nome del carattere di un glifo, che conterrà un struct contenente informazioni per quel glifo. Potete immaginare questi structs annidati nel seguente formato (partendo dal principale struct):

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

Potete usare il nome del carattere di un glifo per recuperare le sue informazioni da struct, come nel seguente codice:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

Il codice qui sopra ottiene il glifo struct per il carattere "A" e lo stampa nell'output log. Potete anche sostituire il string ("A") con una variabile che contiene un string, che vi permetterà di ottenere informazioni per qualsiasi glifo attraverso quella variabile a runtime.

Il struct restituito per un glifo conterrà le seguenti variabili:

NOTA Su HTML5, i glifi struct saranno vuoti per qualsiasi file caricato fonts (come *.ttf o *.otf) poiché runtime non avrà informazioni su quei glifi.

Strutturazione del glifo del carattere
Nome della variabile Tipo di dati Descrizione
char vero Se il font è stato creato da un sprite, questo sarà l'indice dell'immagine del glifo da quel sprite, altrimenti sarà il suo numero di carattere Unicode

Nota: Tutte le variabili sotto questo non saranno presenti nel struct se il font è stato creato da uno sprite
x vero La posizione X del glifo sulla pagina texture (in texel)
y vero La posizione Y del glifo sulla pagina texture (in texel)
w vero La larghezza del glifo sulla pagina texture (in texel)
h vero L'altezza del glifo sulla pagina texture (in texel)
shift vero Il numero di pixel da spostare a destra quando si avanza al carattere successivo (può essere negativo per lo spostamento a sinistra)
offset vero Il numero di pixel per spostare orizzontalmente il rendering di questo glifo (può essere positivo o negativo)
kerning array Un array di interi contenenti informazioni sulla crenatura in coppie (o gruppi di 2). Il primo intero in una coppia è il valore Unicode per un carattere, e il secondo intero è la quantità da aggiungere al valore di spostamento di quel carattere (può essere positivo o negativo) se è preceduto dal carattere di questo glifo.

NOTA Per il file fonts caricato su runtime, i valori x e y per un glifo possono essere -1 se non è stato ancora reso, poiché non avrà una posizione nella pagina texture usata per la cache.

Sintassi:

font_get_info(font);

Argomento Tipo Descrizione
font Font Asset The font for which information should be retrieved.

Ritorna:

Font Info Struct (or undefined)

Esempio:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

Questo codice recupera informazioni per il font memorizzato nella variabile my_font. Se il valore restituito non èundefined e il dato font è in grassetto, la scala dell'istanza viene raddoppiata.
