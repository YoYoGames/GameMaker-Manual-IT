font_set_cache_size

Quando si esegue la fusione e la colorazione delle immagini, HTML5 non può farlo dinamicamente nello stesso modo in cui potrebbe essere eseguito un eseguibile. Pertanto GameMaker deve salvare temporaneamente una copia miscelata delle immagini e caricarle quando necessario. Questa funzione imposta quante copie miscelate del dato font possono essere messe in cache prima che le vecchie vengano sovrascritte. Il valore predefinito è 4.

NOTA per sprite fonts si dovrebbe usare la funzione equivalente per sprites, sprite_set_cache_size().

Sintassi:

font_set_cache_size(ind, max);

ArgomentoTipo Descrizione
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

Ritorna:

Esempio:

font_set_cache_size(fnt_MainMenu, 2);

Questo imposterà il font cache del font indicizzato nella variabile "fnt_MainMenu" a 2 copie.
