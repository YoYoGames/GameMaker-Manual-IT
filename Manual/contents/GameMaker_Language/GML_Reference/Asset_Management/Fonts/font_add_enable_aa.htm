font_add_enable_aa

Questa funzione può essere usata per abilitare o disabilitare l'anti-aliasing (AA) per fonts aggiunto usando font_add(). Questa funzione deve essere chiamata prima di aggiungere qualsiasi fonts e impostarla su true abiliterà l'AA, e impostarla su false lo disabiliterà. Per impostazione predefinita AA è abilitato. Si noti che questo non avrà alcun effetto su fonts che sono stati aggiunti prima che la funzione sia stata chiamata, e la funzione deve essere chiamata solo una volta quando font viene aggiunto, e non ogni disegno/passo che font viene utilizzato.

Sintassi:

font_add_enable_aa(enable);

ArgomentoTipo Descrizione
enable Whether to enable (true) o disabilitare (false) AA per aggiungere fonts.

Ritorna:

Esempio:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Il codice di cui sopra controlla lo stato dell'anti-aliasing per fonts aggiunto e se non è abilitato, allora lo abilitiamo.
