font_add_get_enable_aa

Questa funzione può essere usata per controllare se l'anti-aliasing (AA) è abilitato per fonts aggiunto usando font_add(). La funzione restituirà true se AA è abilitato, e false se non lo è. Si noti che AA è abilitato per impostazione predefinita, ma è possibile cambiare lo stato di AA per i fonts aggiunti utilizzando la funzione font_add_enable_aa()a patto che sia chiamata prima di aggiungere il font.

Sintassi:

font_add_get_enable_aa();

Ritorna:

Esempio:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Il codice di cui sopra controlla lo stato dell'anti-aliasing per fonts aggiunto e se non è abilitato, allora lo abilitiamo.
