aggiungere font

Questa funzione può essere utilizzata per aggiungere un font al vostro gioco da un file font presente nei file inclusi. È possibile definire la dimensione del font (in punti), così come se il font deve essere grassetto o corsivo, e si può anche definire la gamma di caratteri da includere. La funzione restituisce un valore di indice che dovrebbe essere memorizzato in una variabile, poiché questo sarà necessario in tutto il codice successivo che fa riferimento a questo font, oppure restituirà -1 se la funzione fallisce per qualsiasi motivo.

Si noti che il file font deve essere un file *.ttf o *.otf, che può anche essere utile per aggiungere fonts non standard come fonts asiatico o arabo.

ATTENZIONE! Se si include un tale font nei file inclusi, deve essere concesso in licenza per la distribuzione con il gioco.

Per il modulo di destinazione HTML5, questa funzione può essere usata solo per aggiungere Web Font, e funzionerà solo se WebGL non è abilitato - se WebGL è abilitato allora si deve usare una risorsa generale font (cioè: usando draw_set_font() insieme a un font dal Browser Asset ) o bitmap fonts (vedere font_add_sprite()).

IMPORTANTE! Quando si punta a HTML5, il testo in corsivo e in grassetto è supportato solo se il sito font asset o web font stesso ha questi caratteri inclusi. Non saranno sintetizzati dalla base font.

Quando si carica un font da un file, la dimensione del font è in pixel e i valori primo e ultimo sono ignorati, il che significa che tutti i glifi nel font saranno aggiunti. Fonts aggiunti in questo modo sono assegnati alla propria pagina texture, quindi bisogna fare attenzione quando si usa questa funzione, perché aumenterà il numero di scambi texture durante il disegno. Vale anche la pena di notare che fonts può apparire leggermente più grande quando viene disegnato, poiché i glifi possono avere parti che sono disegnate al di fuori del riquadro di delimitazione definito per font. Si dovrebbe anche essere consapevoli che c'è una limitazione di circa 200 glifi che possono essere resi in un singolo frame per un particolare font.

Per impostazione predefinita, fonts aggiunto avrà l'anti-aliasing abilitato. Tuttavia è possibile controllare lo stato di AA in qualsiasi momento utilizzando la funzione font_add_get_enable_aa() ed è possibile abilitare/disabilitare l'AA utilizzando la funzione font_add_enable_aa().

NOTA Quando caricate un font in GameMaker dovete ricordarvi di rimuoverlo (con font_delete()) quando non è più necessario, altrimenti c'è il rischio di una perdita di memoria che rallenterà ed eventualmente bloccherà il gioco.

Sintassi:

font_add(name, size, bold, italic, first, last);

Argomento Tipo Descrizione
name Stringa The path of the font file in the Included Files (*.ttf o *.otf).
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Booleano Whether the font is bold (true) o no (false).
italic Booleano Whether the font is italic (true) o no (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Ritorna:

Font Asset

Esempio:

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

Questo caricherà il file "gamefont.ttf" presente nei file inclusi nel gioco con una dimensione di 24pt. Il font sarà in grassetto e in corsivo, e la gamma di caratteri includerà lettere maiuscole e minuscole, numeri e tutta la punteggiatura comune. Dopo che il font è stato caricato, il suo ID sarà memorizzato nella variabile new_font.
