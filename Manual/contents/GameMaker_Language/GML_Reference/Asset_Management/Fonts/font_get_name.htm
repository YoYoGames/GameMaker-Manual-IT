nome_font_get

Questa funzione può essere usata per ottenere il nome (come string) che è stato dato al font quando è stato aggiunto come asset al GameMaker Asset Browser. Si prega di notare che questo è solo un string e non può essere usato per fare riferimento direttamente a font - per questo è necessario l'indice del font. È possibile, tuttavia, utilizzare questo string per ottenere l'indice del font utilizzando il string restituito insieme alla funzione asset_get_index().

Sintassi:

font_get_name(ind);

ArgomentoTipo Descrizione
ind Index of the font to check.

Ritorna:

Esempio:

fnt_Name = font_get_name(font0);

Il codice qui sopra otterrà il nome di una risorsa font come appare nel Browser Asset e lo memorizzerà come string nella variabile "fnt_Name".
