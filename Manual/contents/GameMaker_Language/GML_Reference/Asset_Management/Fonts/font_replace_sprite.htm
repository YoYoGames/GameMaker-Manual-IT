font_replace_sprite

Con questa funzione potete usare una "striscia di sprite" per sostituire una sprite font precedentemente creata asset, dove ogni sotto-immagine sarebbe un singolo simbolo o lettera. L'indice è il numero di mappa UTF8 per il primo carattere da usare e gli intervalli da usare sono gli stessi che usereste per un normale font, iniziando con "!" ma ricordate che dovreste includere solo ciò che vi serve e che dovreste usare la ord() per ottenere il corretto valore UTF8 per la lettera iniziale.

Puoi anche impostare la spaziatura per il font per essere proporzionale (true) o no (false), dove un font proporzionale è spaziato in base alla larghezza effettiva delle lettere (così la lettera "i" prende meno room della lettera "w", per esempio) mentre un font non proporzionale sarà spaziato in base alla larghezza della sub-immagine, come un font monospaziato. Infine, si può definire lo spazio da lasciare tra ogni lettera quando si scrive, e questo può essere qualsiasi valore intero, con 0 che non è uno spazio (le lettere si "toccheranno" se proporzionale).

Se la funzione fallisce, allora fallirà silenziosamente senza sostituire il dato sprite font.

Vale anche la pena notare che non ci possono essere spazi nella mappa dei caratteri, così che se, per esempio, usate '.' e '0' ma non '/', dovete includere comunque '/' o lasciare una sotto-immagine vuota tra i due caratteri usati. L'immagine qui sotto mostra un tipico layout sprite font:

Sintassi:

font_replace_sprite(ind, spr, first, prop, sep);

Argomento Tipo Descrizione
ind Font Asset The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Sprite Asset The sprite to add a font based on.
first Real The index of the first available character.
prop Booleano Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Ritorna:

N/A

Esempio:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Il codice qui sopra sostituirà il font asset indicizzato nella variabile globale "Font" con un nuovo font asset da un file immagine esterno.
