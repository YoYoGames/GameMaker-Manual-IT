font_replace_sprite_ext

Questa funzione sostituirà un sprite font che avete creato in precedenza usando la funzione font_add_sprite_ext(). Utilizza una "striscia di sprite", dove ogni sotto-immagine è un singolo simbolo o lettera, e le immagini sono mappate alle lettere in base all'argomento "string_map" della funzione. Questo argomento è un string che si può usare per dire a GameMaker in che ordine sono le sotto-immagini di sprite font e le mapperà automaticamente quando si scrive il testo. Così, per esempio, se avete una string-map di "AaBbCcDdEeFfGgHh", il vostro sprite font deve avere le sotto-immagini ordinate in questo modo.

Puoi anche impostare la spaziatura per il font per essere proporzionale (true) o no (false), dove un font proporzionale è spaziato in base alla larghezza effettiva delle lettere (così la lettera "i" prende meno room della lettera "w", per esempio) mentre un font non proporzionale sarà spaziato in base alla larghezza della sub-immagine, come un font monospaziato. Infine, si può definire lo spazio da lasciare tra ogni lettera quando si scrive, e questo può essere qualsiasi valore intero, con 0 che non è uno spazio (le lettere si "toccano" se proporzionali). La funzione restituisce un valore di indice che dovrebbe essere memorizzato in una variabile, poiché questo sarà necessario in tutti gli altri codici che fanno riferimento a questo font.

Se la funzione fallisce, allora fallirà silenziosamente senza sostituire il dato sprite font.

NOTA Questa funzione richiede che tu abbia precedentemente aggiunto un sprite font usando font_add_sprite()perché lo sostituirà e non può essere usato su fonts che sono inclusi come parte delle risorse del gioco.

Sintassi:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argomento Tipo Descrizione
font Font Asset The font to replace.
spr Sprite Asset The sprite to add a font based on.
string_map Stringa String from which sprite sub-image order is taken.
prop Booleano Set as proportional font or not.
sep Real The space to leave between each letter.

Ritorna:

N/A

Esempio:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Il codice qui sopra sostituirà un sprite font asset che è indicizzato nella variabile "global.Font" con immagini sprite indicizzate nella variabile "spr_CalcFont".
