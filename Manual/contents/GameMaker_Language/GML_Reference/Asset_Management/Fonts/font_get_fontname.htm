font_get_fontname

Con questa funzione potete ottenere il nome di sistema effettivo del dato font asset. Questa funzione restituisce una stringa e non un indice, e il nome restituito dipenderà dal font utilizzato. Per esempio, potreste avere un font asset chiamato "fnt_Main" nel Browser Asset, e il font stesso potrebbe essere il sistema Windows font Arial. In questo caso la funzione restituirà "Arial" poiché questo è il nome di sistema di font. Se avete bisogno del nome che appare nel Browser di Asset, usate font_get_name().

Il comportamento descritto sopra si applica solo a font assets aggiunto tramite IDE. Quando viene usato su un font caricato a runtime (usando font_add()), la funzione restituirà il path completo di quel file font invece del suo nome font.

Sintassi:

font_get_fontname(ind);

ArgomentoTipo Descrizione
ind Index of the font to check.

Ritorna:

String

Esempio:

fnt_Name = font_get_fontname(font0);

Il codice precedente otterrà il nome di sistema di una risorsa font e lo memorizzerà come string nella variabile "fnt_Name".
