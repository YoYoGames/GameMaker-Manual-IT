font_exists

Questa funzione restituisce se un font con l'indice specificato esiste o meno. Potete controllare gli indici di font come definiti dal Browser Asset, o fonts che sono stati aggiunti usando funzioni come font_add().

Sintassi:

font_exists(ind);

ArgomentoTipo Descrizione
ind Index of the font to check.

Ritorna:

Esempio:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Questo imposterà il disegno attivo font su fnt_Main se esiste.
