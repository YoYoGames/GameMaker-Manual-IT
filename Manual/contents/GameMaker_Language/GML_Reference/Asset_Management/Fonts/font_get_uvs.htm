font_get_uvs

Questa funzione restituisce un array con le coordinate UV per il sito font texture nella pagina texture, riempiendo l'array con i seguenti valori:

  • [0] = sinistra
  • [1] = top
  • [2] = destra
  • [3] = fondo

Questo valore può poi essere usato in altre funzioni di disegno, in particolare nel disegno generale quando si usano le primitive e le funzioni Shader.

Sintassi:

font_get_uvs(font);

ArgomentoTipo Descrizione
font The index of the font to use.

Ritorna:

Esempio:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Il codice precedente memorizzerà le coordinate UV per lo sfondo dato in un array locale e poi assegnerà i valori alle variabili di istanza.
