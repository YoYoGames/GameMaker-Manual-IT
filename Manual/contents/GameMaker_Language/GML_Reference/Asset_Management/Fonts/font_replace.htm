font_replace

Questa funzione può essere usata per sostituire un font nel vostro gioco da quelli fonts che sono installati sul sistema su cui sta girando. Si noti che il font che viene sostituito non può essere un font dal Browser Asset, e deve essere un font che è stato aggiunto in precedenza utilizzando la funzione font_add().

Quando si sostituisce un font in questo modo, si può definire la dimensione del font (in punti), così come se il font deve essere grassetto o corsivo, e si può anche definire la gamma di caratteri da includere. Se la funzione fallisce, allora fallirà silenziosamente e il dato font non verrà sostituito.

Per il modulo di destinazione HTML5, questa funzione può essere usata solo per sostituire i Web Font, e funzionerà solo se WebGL non è abilitato - se WebGL è abilitato allora si deve usare una risorsa generale font (cioè: usando draw_set_font() insieme a un font dal Browser Asset ) o bitmap fonts (vedere font_add_sprite()).

IMPORTANTE Quando si punta a HTML5, il testo in corsivo e in grassetto è supportato solo se il sito font asset o web font stesso ha questi caratteri inclusi. Non saranno sintetizzati dalla base font.

Su tutti gli altri obiettivi (non Web), è possibile utilizzare questa funzione per sostituire un font da un file. Il file deve essere incluso nel bundle del gioco usando la funzionalità Included Files di GameMaker, e deve essere un file *.ttf o *.otf formato font, utile per aggiungere fonts non standard come asiatico o arabo.

CONSIGLIO Se includete un file font con un gioco, assicuratevi che abbia una licenza appropriata per la distribuzione con il gioco.

Quando si carica un font da un file in questo modo, la dimensione del font è in pixel e i valori primo e ultimo sono ignorati, il che significa che tutti i glifi nel font saranno aggiunti. Fonts aggiunti in questo modo sono assegnati alla propria pagina texture, quindi bisogna fare attenzione quando si usa questa funzione perché aumenterà il numero di scambi texture durante il disegno. Vale anche la pena di notare che fonts può apparire leggermente più grande quando viene disegnato, poiché i glifi possono avere parti che sono disegnate al di fuori del riquadro di delimitazione definito per font. Si dovrebbe anche essere consapevoli che c'è una limitazione di circa 200 glifi che possono essere resi in un singolo frame per un particolare font.

NOTA Quando caricate un font in GameMaker dovete ricordarvi di rimuoverlo (con font_delete()) quando non è più necessario, altrimenti c'è il rischio di una perdita di memoria che rallenterà ed eventualmente bloccherà il gioco.

Sintassi:

font_replace(ind, name, size, bold, italic, first, last);

Argomento Tipo Descrizione
ind Font Asset The index of the (previously added) font to be replaced.
name Stringa The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf o *.otf file.
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Booleano Whether the font is bold (true) or not (false).
italics Booleano Whether the font is italic (true) or not (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Ritorna:

N/A

Esempio:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Questo sostituirà il font indicizzato nella variabile "myfont" con un nuovo font che ha una dimensione di 24pt, usa "Arial" e che è grassetto e corsivo. La gamma font include lettere maiuscole e minuscole, numeri e tutta la punteggiatura comune.
