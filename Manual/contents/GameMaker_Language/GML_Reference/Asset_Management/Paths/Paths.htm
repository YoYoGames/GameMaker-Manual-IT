Percorsi

GameMaker ha una vasta collezione di funzioni e variabili incorporate per usare paths all'interno del vostro gioco. Questa sezione si occupa di quelle di base che vengono utilizzate quando si ha a che fare con paths che sono state precedentemente create nel Path Editor, ma esistono altre funzioni che vi permettono di creare il vostro paths usando il codice, e di manipolarlo in vari modi.

Esistono le seguenti funzioni di base che si occupano dei percorsi:

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

Per aiutare con debugging paths, avete anche una funzione per disegnarlo sullo schermo:

  • draw_path

Ogni istanza di un object ha anche una serie di variabili incorporate progettate per facilitare l'uso di paths, e queste variabili sono elencate nella sezione seguente:

  • Variabili di percorso

Infine, è possibile manipolare il contenuto di un path così come creare nuovi paths quando il gioco è in esecuzione. Le funzioni che permettono queste e altre cose sono disponibili nella sezione seguente:

  • Manipolazione del percorso
