inizio percorso

Un path è creato da una serie di punti di definizione che sono collegati tra loro e poi utilizzati per pianificare i movimenti di un'istanza. Possono essere creati con il codice, o nell'editor Path e vengono assegnati a un'istanza da usare nel gioco. Usereste quindi questa funzione per dire alla vostra istanza quale path seguire, a quale velocità seguire il path (misurata in pixel per passo), come comportarsi quando raggiunge la fine del path, e se seguire la posizione assoluta o relativa path. Quest'ultima parte significa che o inizia e segue il path esattamente come l'hai disegnato e posizionato nell'Editor dei percorsi (assoluto), o inizia e segue il path dalla posizione in cui l'istanza è stata creata (relativo).

Come potete vedere nell'immagine qui sopra, le due istanze stanno seguendo lo stesso path iniziato con path_start(), ma una delle istanze sta seguendo l'assoluto path mentre l'altra sta seguendo il relativo path. Va notato che quando relativo è impostato su false (assoluto) l'istanza "salterà" all'inizio di path non importa in quale punto di room è posizionata.

Le costanti usate per definire il comportamento dell'istanza quando raggiunge la fine di path sono date in questa pagina.

Sintassi:

path_start(path, speed, endaction, absolute);

Argomento Tipo Descrizione
path Percorso patrimoniale The path index to start.
speed Real The speed of which to follow the path in pixels per step, negative meaning going backwards.
endaction Costante di azione di fine percorso What to do when the end of the path is reached (see the constants listed above).
absolute Booleano Whether the calling instance should follow the absolute path as it is in the editor (true) or a relative path to its current position (false).

Ritorna:

N/A

Esempio:

path_start(path, 4, path_action_reverse, 0);

Questo farà sì che l'istanza chiamante segua il path indicizzato nella variabile "path" ad una velocità di 4 pixel per passo. Seguirà il path relativamente alla propria posizione, e quando raggiungerà la fine invertirà la direzione e tornerà indietro da dove è venuto.

Indietro: Percorsi
