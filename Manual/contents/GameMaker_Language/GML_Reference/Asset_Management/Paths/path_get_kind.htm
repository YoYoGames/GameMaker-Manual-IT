percorso_get_kind

I percorsi possono essere lisci o rettilinei (un path liscio calcola un path curvo intorno ai punti che lo definiscono, mentre un path rettilineo va semplicemente dritto da un punto all'altro). Questa può essere utilizzata per scoprire se il dato path è liscio (true) o no (false).

Sintassi:

path_get_kind(index);

ArgomentoTipo Descrizione
index The index of the path to check.

Ritorna:

Esempio:

path_kind = path_get_kind(pth_Patrol);

Questo imposterà la variabile "path_kind" su true o false a seconda del tipo di path l'indice dato path è.
