percorso_get_closed

Questa funzione può essere usata per restituire se il path è segnalato come chiuso (true) o aperto (false), cioè se il path loops o se ha un inizio e una fine definitiva.

Sintassi:

path_get_closed(index);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to check.

Ritorna:

Boolean

Esempio:

state = path_get_closed(pth_Patrol);

Questo imposterà "state" su true o false a seconda dello stato chiuso di path indicizzato in "pth_Patrol".

Indietro: Percorsi
