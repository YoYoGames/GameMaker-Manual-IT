percorso_get_length

Potete usare questa funzione per ottenere la lunghezza esatta di un path in pixel. Questa non è una lunghezza approssimativa da punto a punto, ma piuttosto una lunghezza esatta lungo la forma del path, anche quando il path è liscio con un'alta precisione curvilinea.

Sintassi:

path_get_length(index);

ArgomentoTipo Descrizione
index The index of the path to measure.

Ritorna:

Esempio:

path_len = path_get_length(pth_AI);

Questo imposterà "path_len" alla lunghezza totale del path indicizzato in "pth_AI" in pixel.
