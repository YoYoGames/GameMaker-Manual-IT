Aggiungi percorso

Con questa funzione potete creare un path in GameMaker senza usare l'editor path. Questa funzione restituirà l'indice del path che dovrebbe essere memorizzato in una variabile e usato come riferimento per quel path da allora in poi. Notate che il path creato è vuoto, cioè non ha punti definiti, quindi dovrete usare le altre funzioni disponibili per aggiungere punti al path o usare le griglie MP per generare il path. Una volta che avete finito di usare il path, o volete crearne uno nuovo e memorizzare il suo indice nella stessa variabile dovete prima cancellare il vecchio path con path_delete per evitare perdite di memoria che possono causare il crash del gioco.

Sintassi:

path_add();

Ritorna:

Path Asset

Esempio:

global.newpath = path_add();

Questo creerà un nuovo path e assegnerà il suo indice a global.newpath.
