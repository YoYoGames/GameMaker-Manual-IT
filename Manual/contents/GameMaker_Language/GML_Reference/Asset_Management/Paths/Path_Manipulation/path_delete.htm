cancella percorso

Potete usare questo codice per rimuovere un path dalla memoria. Se questo path è stato creato dinamicamente usando path_add()la variabile che contiene l'indice path non sarà più valida per accedere a path perché non esiste più, e se path è stato creato usando l'editor di percorsi, quel path non è più accessibile in tutto il gioco perché lo state cancellando definitivamente.

Sintassi:

path_delete(index);

ArgomentoTipo Descrizione
index The index of the path to delete.

Ritorna:

Esempio:

var t_path = path_add();
if (mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1))
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

Il codice qui sopra creerà un path e memorizzerà il suo indice in una variabile locale. Questo path viene poi usato per memorizzare un mp_grid_path() generato path che, se riesce a trovare la sua strada verso l'obiettivo, viene poi assegnato al path indicizzato in "mypath". Infine il "t_path" viene cancellato.
