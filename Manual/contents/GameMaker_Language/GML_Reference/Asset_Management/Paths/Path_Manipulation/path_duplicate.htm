percorso_duplicato

Questa funzione prende un path e lo copia in un nuovo path. Il nuovo path viene creato nel processo, e il suo indice viene restituito per essere utilizzato in tutte le ulteriori chiamate per utilizzare questo nuovo path.

Sintassi:

path_duplicate(index);

ArgomentoTipo Descrizione
index The index of the existing path to duplicate.

Ritorna:

Path Asset

Esempio:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

Il codice qui sopra sceglie una delle quattro risorse path e la duplica, memorizzando l'indice del nuovo path nella variabile "mypath".
