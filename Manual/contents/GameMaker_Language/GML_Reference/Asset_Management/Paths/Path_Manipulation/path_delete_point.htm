percorso_delete_point

Con questa funzione potete rimuovere un punto dal path specificato. I punti di Path sono numerati da 0, quindi il primo punto è sempre lo 0° punto e l'ultimo punto di path sarà path_get_number() - 1. Se chiamate questa funzione su un path asset, allora la rimozione del punto sarà permanente e tutte le istanze assegnate a path in futuro mancheranno del punto che è stato rimosso. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_delete_point(ind, n);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to delete a point from.
n Real The defining point to delete.

Ritorna:

N/A

Esempio:

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

Il codice di cui sopra genera un loop che controlla i punti all'indietro dalla fine per una collisione con il dato object. Se viene trovata una collisione, un punto di path viene cancellato, finché non c'è nessuna collisione, o il path è lungo solo 2 punti.
