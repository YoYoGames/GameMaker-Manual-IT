punto di aggiunta del percorso

Con questa funzione si può aggiungere un punto al sito path specificato e impostare il suo fattore di velocità. Questo punto viene aggiunto alla fine di path, indipendentemente dalla posizione del punto, e il fattore è uguale alla percentuale di path_speed a cui l'istanza seguente va effettivamente quando raggiunge questo punto in path. Se si desidera posizionare un punto di path in qualche altra posizione che non sia la fine, si dovrebbe usare path_insert_point().

Se usato su un path dal Browser Asset, notate che la funzione cambierà l'asset attuale, e quindi influenzerà permanentemente come il path è usato da tutte le istanze nel gioco dal momento in cui la funzione è usata fino alla fine del gioco. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_add_point(index, x, y, speed);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to add the point to.
x Real The x coordinate of the new point.
y Real The y coordinate of the new point.
speed Real The speed factor of the point (default value is 100).

Ritorna:

N/A

Esempio:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Il codice di cui sopra creerà un path circolare intorno alla posizione x/y dell'istanza che esegue il codice.
