path_flip

Questa funzione prende tutti i punti di path e li capovolge lungo l'asse orizzontale. Questa funzione cambia l'attuale risorsa path , e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione è usata fino alla fine del gioco.Se questo non è ciò che si richiede, allora si dovrebbe usare una funzione come path_duplicate() per creare una copia del path prima, poi chiamare questa funzione sul asset duplicato (non dimenticare di chiamare path_delete() sul asset quando non è più necessario).

Sintassi:

path_flip(index);

ArgomentoTipo Descrizione
index The index of the path to flip.

Ritorna:

Esempio:

path_flip(mypath);

Questo capovolgerebbe il path indicizzato nella variabile "mypath" lungo l'asse orizzontale.

Indietro: Manipolazione del percorso
Il prossimo: path_mirror
