applicare il percorso

Con questo codice potete aggiungere un path su un altro, unendoli effettivamente insieme, e perché questo funzioni, entrambi i paths devono esistere in precedenza. In generale si desidera utilizzare questa funzione su paths creato utilizzando path_add(), poiché se lo si utilizza su un path asset, influenzerà permanentemente il path per le istanze nel gioco dal momento in cui la funzione viene utilizzata fino alla fine del gioco.

NOTA Questo non rimuoverà il path aggiunto dal gioco. È ancora lì, solo che ora non ha più punti e se non volete più usarlo dovete rimuoverlo con la funzione path_delete().

Sintassi:

path_append(index, path);

ArgomentoTipo Descrizione
index The new index of the path.
path The index of the path that will be appended to 'index'.

Ritorna:

N/A

Esempio:

path_append(mypath, mppath);
path_delete(mppath);

Il codice qui sopra aggiunge i dati di path indicizzati nella variabile "mppath" al path indicizzato nella variabile "mypath" e poi cancella "mppath" dalla memoria.
