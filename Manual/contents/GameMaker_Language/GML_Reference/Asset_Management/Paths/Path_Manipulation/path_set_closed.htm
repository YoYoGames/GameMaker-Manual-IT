percorso_set_chiuso

Questa funzione può essere usata per segnalare un dato path come aperto (falso) o chiuso (vero). Un path chiuso ha il suo punto iniziale collegato al suo punto finale, formando un loop, e un path aperto ha un inizio e una fine definitivi e non collegati. Questa funzione cambia l'attuale risorsa path , e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione è usata fino alla fine del gioco.Se questo non è ciò che si richiede, allora si dovrebbe usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul asset duplicato (non dimenticare di chiamare path_delete() sul asset quando non è più necessario).

Sintassi:

path_set_closed(index, closed);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to change.
closed Booleano Whether the path is closed (true) or not (false).

Ritorna:

N/A

Esempio:

path_set_closed(pth_Patrol, true);

Questo imposterà il dato path come un path chiuso.
