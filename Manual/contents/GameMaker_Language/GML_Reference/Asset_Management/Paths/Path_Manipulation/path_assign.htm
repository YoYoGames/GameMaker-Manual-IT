percorso_assegnato

Con questa funzione è possibile copiare i dati di path da un path a un altro. Il path in cui viene copiato sarà cancellato per primo (se ha punti path ) e sarà completamente sovrascritto dal path da cui viene copiato da. Nessuno dei due path viene cancellato nel processo e il risultato sono due paths, con due indici diversi, ma la stessa forma e gli stessi punti. In generale vorreste usare questa funzione su un path creato usando path_add(), poiché se la usate su un path asset, influenzerà permanentemente il path per tutte le istanze nel gioco dal momento in cui la funzione viene usata fino alla fine del gioco.

Sintassi:

path_assign(index, path);

ArgomentoTipo Descrizione
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

Ritorna:

Esempio:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

Il codice qui sopra creerà un nuovo, vuoto path indicizzato nella variabile "mypath" e poi copierà i dati di path da una delle tre risorse disponibili path.

Il prossimo: path_delete
