punto di cancellazione del percorso

Con questa funzione potete rimuovere tutte le definizioni dei punti da un path, creando effettivamente un path"vuoto". Questo non cancella il sito path, per quello si dovrebbe usare path_delete()Tuttavia va notato che questa funzione cambia l'attuale asset path , e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione viene usata fino alla fine del gioco. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_clear_points(index);

ArgomentoTipo Descrizione
index The index of the path to clear.

Ritorna:

Esempio:

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

Il codice qui sopra controlla se ci sono dei punti su path indicizzati nella variabile "mypath" e se ci sono, cancella il path.
