percorso_esiste

Questa funzione restituisce se esiste o meno un path con l'indice dato. Notate che se controllate l'esistenza di un path attraverso una variabile che non è ancora stata dichiarata, questo darà un errore.

Sintassi:

path_exists(index);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to check for.

Ritorna:

Boolean

Esempio:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Questo codice controlla se la variabile data memorizza un indice path e se lo fa, allora il path viene cancellato dalla memoria.

Indietro: Manipolazione del percorso
Il prossimo: path_add
