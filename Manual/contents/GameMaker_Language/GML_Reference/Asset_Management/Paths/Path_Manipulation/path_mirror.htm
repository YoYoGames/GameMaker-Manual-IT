percorso_specchio

Questa funzione prende tutti i punti di path e li specchia lungo l'asse verticale. Questa funzione cambia l'attuale asset path , e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione viene usata fino alla fine del gioco. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_mirror(index);

ArgomentoTipo Descrizione
index The index of the path to mirror.

Ritorna:

Esempio:

path_mirror(mypath);

Questo rispecchierebbe il path indicizzato nella variabile "mypath" lungo l'asse verticale.

Indietro: Manipolazione del percorso
Il prossimo: path_reverse
