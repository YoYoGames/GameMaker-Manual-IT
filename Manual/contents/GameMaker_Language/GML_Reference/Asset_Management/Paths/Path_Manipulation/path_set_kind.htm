percorso_set_kind

Questa funzione può essere usata per impostare il tipo di path che si desidera che la risorsa specificata sia. Questa può essere sia una linea retta path (impostata a 0) o una path smussata (impostata a 1), nel qual caso deve essere presa in considerazione anche la precisione di path (la precisione può essere impostata anche utilizzando path_set_precision()). Questa funzione cambia l'attuale risorsa path, e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul asset duplicato (non dimenticate di chiamare path_delete() sul asset quando non è più necessario).

Sintassi:

path_set_kind(index, val);

ArgomentoTipo Descrizione
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Ritorna:

Esempio:

path_set_kind(pth_Patrol, true);

Questo imposterà il dato path per essere un "liscio" path.

Indietro: Manipolazione del percorso
Il prossimo: path_set_precision
