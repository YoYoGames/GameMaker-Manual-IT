percorso_rescale

Questa funzione può essere usata per ridimensionare il dato path lungo entrambi (o entrambi) gli assi verticale e orizzontale, in pratica spostando ciascuno dei punti di path in una nuova posizione corrispondente a questa scala intorno al centro di path. Questa funzione cambia l'attuale risorsa path, e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione viene usata fino alla fine del gioco. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_rescale(index, xscale, yscale);

ArgomentoTipo Descrizione
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

Ritorna:

Esempio:

path_rescale(pth_AI, 3, 3);

Questo ridimensionerebbe il dato path a tre volte la sua dimensione originale.

Indietro: Manipolazione del percorso
Il prossimo: path_set_closed
