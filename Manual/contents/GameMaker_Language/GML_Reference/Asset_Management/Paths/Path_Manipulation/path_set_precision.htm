precizione del percorso

Con questa funzione potete cambiare la "morbidezza" di un path. Questo valore deve essere compreso tra 1 e 8, con un valore basso che crea bordi più dritti con curve più nette tra i punti, mentre un valore più alto arrotonderà i punti e renderà il path molto più "sinuoso". Si noti che questa funzione non avrà alcun effetto visibile se il sito path non è stato impostato come liscio nell'editor path o utilizzando la funzione path_set_kind(). Questa funzione cambia l'attuale asset path , e quindi influenzerà permanentemente il modo in cui path è usato da tutte le istanze del gioco dal momento in cui la funzione viene usata fino alla fine del gioco. Se questo non è ciò di cui avete bisogno, allora dovreste usare una funzione come path_duplicate() per creare prima una copia di path, poi chiamare questa funzione sul duplicato asset (non dimenticate di chiamare path_delete() su asset quando non è più necessario).

Sintassi:

path_set_precision(index, prec);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to change.
prec Real The precision of the path. Must be an integer between 1 and 8.

Ritorna:

N/A

Esempio:

path_set_precision(pth_Patrol, 2);

Questo imposterà la precisione del percorso dato a 2.

Indietro: Manipolazione del percorso
