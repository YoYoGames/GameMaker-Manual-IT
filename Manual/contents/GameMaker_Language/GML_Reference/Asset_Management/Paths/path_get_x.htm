percorso_get_x

Con questa funzione si può ottenere la coordinata x di una posizione su qualsiasi dato path. 0 è l'inizio del path, 1 è la fine del path, e qualsiasi cosa in mezzo equivale a quella distanza attraverso il path. Questo non deve essere un punto di definizione del path, può essere ovunque su di esso.

Sintassi:

path_get_x(ind, pos);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Ritorna:

Real

Esempio:

x = path_get_x(pth_WallWalk, 0.5);

Questo imposterà la x dell'istanza chiamante alla coordinata x del punto esattamente a metà del dato path.
