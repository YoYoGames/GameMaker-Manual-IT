percorso_get_name

Questa funzione restituisce il nome del path a cui si fa riferimento come string. Il nome è quello che è stato usato per definire il path nell'editor o (se il path è stato creato attraverso una funzione di codice) restituirà un string con il formato " _newpathXX" dove "XX" è il numero del path generato, partendo da 0 e incrementando di uno ogni volta che viene creato un nuovo path. Si noti che questo è solo un string e non può essere usato per fare riferimento direttamente a path - per questo è necessario l'ID dell'indice del percorso. Potete, comunque, usare questo string per ottenere l'ID dell'indice del percorso usando il string restituito insieme a con la funzione asset_get_index().

Sintassi:

path_get_name(index);

ArgomentoTipo Descrizione
index The index of the path to check.

Ritorna:

Esempio:

path_name = path_get_name(path_array[0]);

Questo imposterà "path_name" al nome del path indicizzato nell'array dato alla posizione 0.
