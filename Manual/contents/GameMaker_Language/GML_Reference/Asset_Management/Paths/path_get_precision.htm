percorso_get_precisione

Questa funzione restituisce con quale precisione il dato path è stato "smussato", e sarà un valore intero da 1 a 8. Sebbene sia possibile ottenere (e impostare) questo valore per una linea retta path non avrà alcuna influenza su come un'istanza usa il path poiché è rilevante solo quando il tipo path è impostato su "smooth".

Sintassi:

path_get_precision(index);

ArgomentoTipo Descrizione
index The index of the path to check.

Ritorna:

Esempio:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

Il codice di cui sopra controlla il path indicizzato in "pth_Patrol" per vedere che tipo è e se è liscio, controlla poi quanto è preciso lo smoothing. Se è inferiore a 8, viene impostato ad un valore di 8.
