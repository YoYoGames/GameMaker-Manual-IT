fine del percorso

La chiamata di questa funzione terminerà l'attuale path che l'istanza sta seguendo, come impostato quando la funzione path_start() è stata chiamata.

Sintassi:

path_end();

Ritorna:

Esempio:

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

Il codice di cui sopra terminerà l'attuale path se l'istanza rileva qualsiasi collisione con qualsiasi istanza dell'oggetto dato.
