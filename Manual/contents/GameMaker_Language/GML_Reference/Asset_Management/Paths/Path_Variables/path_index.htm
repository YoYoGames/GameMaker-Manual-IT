indice del percorso

Tutte le risorse in GameMaker hanno un numero identificativo unico, e la path_index è una variabile di sola lettura che contiene il valore dell'indice per un dato path asset che è stato assegnato a un'istanza utilizzando la funzione path_start(). Se l'istanza non ha path assegnato, la variabile sarà impostata a -1.

Sintassi:

path_index;

Ritorna:

Path Asset

Esempio:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

Il codice di cui sopra controlla per vedere se un path è stato assegnato all'istanza, e se non lo è avvia un nuovo path (assegnandolo al path_index allo stesso tempo).
