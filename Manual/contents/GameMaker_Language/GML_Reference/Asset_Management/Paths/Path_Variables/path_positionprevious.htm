percorso_posizioneprecedente

Questa variabile può essere usata per ottenere o impostare la posizione di un'istanza lungo il suo attuale path nel passo precedente, ed è un valore normalizzato tra 0 e 1 cioè: 0 è la posizione iniziale del path e 1 sarebbe la posizione finale. È simile agli attributi xprevious e yprevious per come funziona, solo che è specifica per paths. Può essere utile per cose come fermare temporaneamente un path follower se c'è qualcosa in mezzo (vedi il codice di esempio qui sotto).

Sintassi:

path_positionprevious;

Ritorna:

Esempio:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(_x, _y, 16, obj_player, false, true))
{
    path_position = path_positionprevious;
}

Il codice precedente controlla un'area davanti all'istanza per una collisione con object "obj_Player" e se ce n'è una, riporta l'istanza alla posizione precedente che occupava sul corrente path, come assegnato dalla funzione path_start().
