azione_percorso

Questa variabile può essere usata per ottenere o cambiare la reazione di un'istanza quando raggiunge la fine della corrente path. Normalmente la impostereste quando avviate il sito path usando path_start() ma potreste voler cambiare questo comportamento a seconda di un qualsiasi numero di eventi nel vostro gioco. I valori disponibili sono espressi utilizzando le seguenti costanti:

Costante di azione di fine percorso
Costante Descrizione
path_action_stop Fermare il percorso
path_action_restart Continuare dalla posizione di partenza, saltando alla partenza se il path non è chiuso
path_action_continue Far ripartire path dalla posizione attuale
path_action_reverse Invertire la velocità del path (far funzionare il path al contrario)

Sintassi:

path_endaction;

Ritorna:

Path End Action Constant

Esempio:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

Il codice di cui sopra controllerà l'azione finale di path e se è impostata su stop, allora l'azione finale sarà cambiata in reverse.
