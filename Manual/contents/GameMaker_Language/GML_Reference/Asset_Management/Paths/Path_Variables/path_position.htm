posizione_percorso

Questa funzione può essere usata per ottenere o impostare la posizione di un'istanza lungo un path. Il valore è normalizzato da 0 - 1, quindi se lo impostate, per esempio, a 0,5, l'istanza sarà spostata esattamente a metà del path.

Sintassi:

path_position;

Ritorna:

Esempio:

path_position = random(1);

Il codice precedente imposta l'istanza in una posizione casuale in qualsiasi punto del corrente path.
