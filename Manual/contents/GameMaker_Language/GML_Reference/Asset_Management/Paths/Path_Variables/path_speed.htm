percorso_velocità

Potete usare questa funzione per ottenere o impostare la velocità di un path dopo che è stato avviato usando la funzione path_start(). Potete usare valori negativi per indicare che l'istanza dovrebbe seguire il path al contrario.

Sintassi:

path_speed;

Ritorna:

Esempio:

path_speed = -1 + random(2);

Il codice qui sopra imposterà l'istanza a viaggiare su path a una velocità casuale tra -1 e 1.
