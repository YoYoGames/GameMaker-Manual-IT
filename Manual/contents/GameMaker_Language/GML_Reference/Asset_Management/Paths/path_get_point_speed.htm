percorso_get_point_speed

Con questa funzione si può ottenere la velocità del punto (come definito nell'Editor di tracciato o quando si aggiunge dinamicamente un punto path usando path_add_point()) espressa in percentuale. Così, se avete un punto path impostato a 50 nell'editor path, questa funzione restituirà 50 quando viene utilizzata.

Sintassi:

path_get_point_speed(index, n);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to check.
n Real The point number to check.

Ritorna:

Real

Esempio:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

Il codice qui sopra controlla il punto path "0" del path indicizzato da "pth_Patrol" per vedere se la velocità non è uguale a 100. Se non lo è, imposta la velocità di quel punto a 100.
