path_get_point_y

Questa funzione restituirà la posizione y (in coordinate room ) del punto che avete inserito per il path che indicizzate. Se il punto è al di fuori dell'intervallo di path (cioè: un path ha 8 punti e voi chiedete la posizione y del punto 10) allora verrà restituito un valore di 0.

Sintassi:

path_get_point_y(index, n);

Argomento Tipo Descrizione
index Percorso patrimoniale The index of the path to check.
n Real The point number to check.

Ritorna:

Real

Esempio:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

Il codice qui sopra controllerà per vedere se un'istanza è alla fine di un path. Se lo è, sceglierà un punto casuale su path e sposterà l'istanza in quel punto.
