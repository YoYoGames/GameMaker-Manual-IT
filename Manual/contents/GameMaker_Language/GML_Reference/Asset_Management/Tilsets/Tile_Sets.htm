Le seguenti funzioni sono disponibili quando si lavora con tile set assets, come creato nel Browser delle risorse. Queste funzioni servono solo per lavorare con asset stesso, e non per lavorare con le mappe tiles o tile quando sono presenti nel vostro gioco (per quello vedete le funzioni di livello appropriate nella sezione Stanze).

Le funzioni disponibili sono:

  • tileset_get_name
  • tileset_get_texture
  • tileset_get_uvs
