nome_target

Questa funzione restituisce il nome come stringa del sito tile set asset specificato. Questo nome è quello specificato per tile set nel Browser Asset della finestra principale di GameMaker. Si noti che questo è solo un string e non può essere usato per fare riferimento direttamente al tile set - per questo è necessario l'indice del set di piastrelle. Tuttavia, è possibile utilizzare questo string per ottenere l'indice del set di piastrelle utilizzando il string restituito insieme alla funzione asset_get_index().

Sintassi:

tileset_get_name(index);

Argomento Tipo Descrizione
index Set di piastrelle Asset The index of the tile set to get the name of.

Restituzioni

String

Esempio:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

Il codice precedente otterrà il nome dell'indice tile set per il livello dato, memorizzando il ritorno string nella variabile tileset_name.

Indietro: Set di piastrelle
Il prossimo: tileset_get_texture
