nome dello script_get

Questa funzione restituisce il nome come stringa del script specificato. Questo nome è quello che è stato specificato per il script nel Browser Asset della finestra principale di GameMaker. Per maggiori informazioni su scripts, vedere L'editor Script .

Sintassi:

script_get_name(scr);

Argomento Tipo Descrizione
scr Script Asset The index of the script that you want to get the name of.

Ritorna:

String

Esempio:

scr_name = script_get_name(Help_File);

Il codice dell'esempio precedente memorizzerà il nome dell'indice indicato script nella variabile "scr_name".
