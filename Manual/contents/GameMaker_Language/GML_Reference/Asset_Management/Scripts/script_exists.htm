esiste uno script

Questa funzione restituirà true o false a seconda che esista la funzione script o script con l'indice dato. Si noti che questo non è un string, ma piuttosto il nome asset che contiene l'indice unico per ogni script (come apparirebbe in IDE) o la funzione script nominata, come definita all'interno di script asset (si noti che questo non funziona per le variabili di metodo). Per maggiori informazioni su scripts, vedere L'editor Script .

Sintassi:

script_exists(scr);

Argomento Tipo Descrizione
scr Script Asset The script index that you want to check.

Ritorna:

Boolean

Esempio:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

L'esempio precedente aggiunge due funzioni script e un valore in un array, poi procede a ottenere un numero casuale e lo usa per scegliere una funzione script da eseguire, a meno che non venga scelto il -1, nel qual caso non succederà nulla.
