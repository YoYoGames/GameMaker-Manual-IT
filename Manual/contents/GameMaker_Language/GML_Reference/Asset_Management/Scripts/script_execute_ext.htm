script_execute_ext

Questa funzione funziona in modo simile alla funzione script_execute() solo che potete fornire un array che contiene gli argomenti richiesti per la funzione/script che viene chiamata. Potete anche fornire due argomenti opzionali alla funzione per specificare un offset nell'array da cui prendere gli argomenti, così come il numero di argomenti da utilizzare dall'array (questo deve essere un massimo di array_length - offset).

Sintassi:

script_execute_ext(scr, array_args, [offset], [num_args]);

Argomento Tipo Descrizione
scr Script Asset o Script Function The function/script that you want to call
array_args Array The array containing the arguments for the function/script
[offset] Real OPTIONAL L'offset nell'array degli argomenti
[num_args] Real OPTIONAL Il numero di argomenti da usare (dall'offset in poi)

Ritorna:

Variable (Will depend on the return value from the function/script being called)

Esempio:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

Il codice dell'esempio precedente userà script_execute_ext per chiamare la funzione data, fornendo un array di quattro argomenti, ma usandone solo uno a caso.

NOTA Questa funzione non può essere usata con le funzioni integrate.
