gravità

gravity è una delle variabili "incorporate" che tutte le istanze hanno e, quando impostata, applicherà una forza costante nel gravity_direction dell'istanza, influenzando sia l'istanza speed che direction. Si noti che la gravità è una forza cumulativa e accelererà il object se si sceglie di non limitare la velocità finale, e di solito si imposta questa variabile a piccoli valori decimali come 0,01. Se impostate gravity a 0, allora nessuna gravità sarà applicata all'istanza (questo è il valore di default).

Sintassi:

gravity;

Ritorna:

(single precision floating point value)

Esempio:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

Il codice di cui sopra applicherà la gravità solo se l'istanza non trova nessuna istanza di "obj_Ground" sotto di essa.
