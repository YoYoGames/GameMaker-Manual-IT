vspeed

vspeed è una delle proprietà "incorporate" che tutte le istanze hanno e definisce la velocità di movimento verticale (lungo l'asse y) dell'istanza in pixel per passo. Così, un vspeed di 3 significa 3 pixel di movimento verso il basso (+y) ogni passo, e un vspeed di -3 significherebbe 3 pixel di movimento verso l'alto (-y) ogni passo.

Notate che se impostate le variabili speed e/o direction allora il valore di vspeed sarà aggiornato automaticamente per riflettere questi cambiamenti, e allo stesso modo, cambiando il valore di vspeed influenzerà anche i valori di speed e direction. Notate anche che gravity, gravity_direction e friction possono tutti modificare il valore di questa variabile quando sono usati nei vostri giochi.

Sintassi:

vspeed;

Ritorna:

(single precision floating point value)

Esempio:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

Il codice di cui sopra cambierà la velocità verticale a seconda dei tasti premuti.
