yprecedente

Questa variabile incorporata restituisce la precedente posizione y per l'istanza. Questa variabile sarà impostata alla posizione x corrente appena prima dell'inizio dell'evento di inizio passo, ma può anche essere impostata attraverso il codice in qualsiasi momento, il che significa che potete darle il vostro valore personalizzato se fosse necessario.

Sintassi:

yprevious;

Ritorna:

(single precision floating point value)

Esempio:

if x != xprevious || y != yprevious
{
    moved = true;
}

Il codice qui sopra controlla se c'è qualche differenza tra i valori xprevious e yprevious e gli attuali valori x e y. Se c'è, imposta una variabile a true.
