attrito

Friction è una delle proprietà "incorporate" che le istanze possono avere e può essere usata per rallentare l'istanza nel tempo quando il speed è diverso da zero. Funziona semplicemente sottraendo una quantità dalla velocità ogni passo fino a quando object ha una velocità di 0, quindi se l'attrito è impostato, per esempio, a 0,1 e la velocità dell'istanza è 1 (1 pixel per passo), essa rallenterà e si fermerà dopo 10 passi. Notate anche che l'attrito è applicato a velocità positive e negative allo stesso modo, con il risultato netto che object ha sempre una velocità di 0 dopo un dato tempo.

Sintassi:

friction;

Ritorna:

(single precision floating point value)

Esempio:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

Il codice di cui sopra applicherà l'attrito solo se la velocità assoluta dell'istanza è superiore a 0.
