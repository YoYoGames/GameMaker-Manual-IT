visibile

Un'istanza può essere contrassegnata come visible o no impostando questa variabile a true (visibile) o false (invisibile). Questo funziona dicendo a GameMaker di saltare l'evento di disegno per questa istanza, quindi bisogna fare attenzione quando si usa questo, perché significa che nessun codice inserito nell'evento di disegno sarà eseguito mentre l'istanza non è visibile. Notate anche che se il file layer a cui l'istanza è assegnata è contrassegnata come invisibile, l'impostazione di questa variabile su true non avrà alcun effetto finché il livello stesso non sarà contrassegnato come visibile.

Sintassi:

visible;

Ritorna:

Esempio:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

Il codice di cui sopra controllerà il flag visibile dell'"altra" istanza in un evento di collisione e se è impostato su true sposterà l'istanza nella sua posizione precedente.
