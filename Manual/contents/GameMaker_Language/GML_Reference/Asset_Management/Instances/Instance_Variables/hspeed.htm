hspeed

hspeed è una delle proprietà "built-in" che tutte le istanze hanno e definisce la velocità di movimento orizzontale (lungo l'asse x) dell'istanza in pixel per passo. Così, un hspeed di 3 significa 3 pixel di movimento verso destra (+x) ogni passo, e un hspeed di -3 significherebbe 3 pixel di movimento a sinistra (-x) ogni passo.

Notate che se impostate le variabili speed e/o direction allora il valore di hspeed sarà aggiornato automaticamente per riflettere questi cambiamenti, e allo stesso modo, cambiando il valore di hspeed influenzerà anche i valori di speed e direction. Notate anche che gravity, gravity_direction e friction possono tutti modificare il valore di questa variabile quando sono usati nei vostri giochi.

Sintassi:

hspeed;

Ritorna:

(single precision floating point value)

Esempio:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

Il codice di cui sopra cambierà la velocità orizzontale a seconda dei tasti premuti.
