x

Il valore x di un'istanza è la posizione orizzontale nel corrente room, misurata in pixel. Questo valore può essere 0, positivo o meno, dove 0 è il lato sinistro di room e spostandosi a destra aumenta x, spostandosi a sinistra diminuisce x (un valore negativo valore di x significa che l'istanza è andata fuori dal lato sinistro di room). Potete anche dare a x un valore reale come 12.345, ma poiché non potete ottenere .345 di un pixel, l'istanza potrebbe non sembrare essersi mossa, anche se questi valori saranno mantenuti e usati internamente da GameMaker e sono perfettamente validi.

Impostando i valori x (e y) potete far saltare object intorno a room nella posizione che preferite, oppure potete aggiungere e sottrarre ad essi quantità minori per dare l'illusione del movimento senza usare effettivamente i valori integrati speed e direction funzioni. Questo dovrebbe essere preso in considerazione quando fate i vostri giochi, perché spesso un tipo di movimento si presta a un particolare stile di gioco più di un altro.

Sintassi:

x;

Ritorna:

(single precision floating point value)

Esempio:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

Il codice di cui sopra userà i tasti freccia per aggiungere o sottrarre alla posizione x e creare così l'illusione del movimento.
