id

Questa variabile di sola lettura contiene il numero identificativo unico dell'istanza. A ogni istanza creata, sia attraverso il codice sia aggiungendola a room nell'editor Room, viene assegnato un numero che viene usato internamente per identificare l'istanza e la variabile id è quella che si può usare per farvi riferimento. Il numero id viene anche restituito (e può essere memorizzato in una variabile) quando si crea un'istanza usando instance_create_layer() o instance_create_depth(), così come altre funzioni di istanza.

Questa variabile è ora un riferimento a un'istanza, invece di essere semplicemente un numero reale. Si può ottenere il numero ID dell'istanza chiamando real() o int64() su questa variabile.

NOTA Il valore di questa variabile non è lo stesso dell'identificatore che viene dato all'istanza nell'editor di room ed è anche diverso dall'array instance_id che contiene gli id di tutte le istanze attualmente attive.

Sintassi:

id;

Restituzione:

Instance ID

Esempio:

for (var i = 0; i < instance_count; i++;)
{
    if (instance_id[i] != id)
    {
        instance_id[i].scr += 5;
    }
}

Il codice precedente aggiunge 5 alla variabile "scr" per ogni istanza attiva in room, tranne quella che sta eseguendo il codice. Lo fa scorrendo TUTTE le istanze attive (usando l'array "instance_id" per restituire l'ID di ciascuna istanza attiva) e confrontandole con la variabile "id" incorporata, che è l'ID dell'istanza originale che esegue il codice.
