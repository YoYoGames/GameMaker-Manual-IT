xstart

Questa variabile memorizza la posizione x iniziale dell'istanza quando viene creata per la prima volta in room. Questa non è una variabile di sola lettura e può essere impostata così come letta.

Sintassi:

xstart;

Ritorna:

(single precision floating point value)

Esempio:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

Il codice di cui sopra controllerà per una collisione con un'istanza di "obj_spike" e se ce n'è una, deduce 100 dal punteggio e sposta l'istanza indietro alla sua posizione di partenza.

Indietro: Variabili di istanza
