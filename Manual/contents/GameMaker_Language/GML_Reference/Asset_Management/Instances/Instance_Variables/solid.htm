solido

Un'istanza può essere contrassegnata come solida attraverso le proprietà object nell'Object Editor, o cambiando il valore di questa variabile incorporata. Se solid è impostato su true allora, quando viene rilevata una collisione, l'istanza in collisione viene riportata automaticamente alla posizione in cui si trovava nel passo precedente alla collisione (e quindi qualsiasi codice o azione viene eseguito nell'evento di collisione). Se è impostato su false, tutto il posizionamento deve essere gestito attraverso l'evento di collisione.

Sintassi:

solid;

Ritorna:

Esempio:

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

Il codice di cui sopra controllerà la bandiera solida dell'"altra" istanza in una collisione e se è impostata su true sposterà quell'istanza nella sua posizione precedente.

Indietro: Variabili di istanza
Il prossimo: persistent
