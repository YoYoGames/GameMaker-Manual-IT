direzione_gravità

gravity_direction è una delle proprietà "incorporate" che tutte le istanze hanno e può essere usata per impostare la direzione del movimento quando l'istanza gravity è maggiore di 0. Si noti che le direzioni in GameMaker sono solitamente calcolate come 0° a destra, 90° in alto, 180° a sinistra e 270° in basso.

Sintassi:

gravity_direction;

Ritorna:

(single precision floating point value)

Esempio:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

Il codice di cui sopra cambierà la direzione_gravità se il giocatore object incontra un'istanza di "obj_switch".
