cambio d'istanza

Potete usare questa funzione per cambiare un'istanza di un object in un'altra istanza di un diverso object, e mentre lo fate decidere se eseguire gli eventi Destroy e Clean Up delle istanze iniziali e l'evento Create delle nuove istanze. In questo modo, potete avere (per esempio) una bomba che si trasforma in un'esplosione - nel qual caso l'argomento "perf" sarebbe probabilmente vero poiché vorreste che la bomba eseguisse i suoi eventi Destroy e Clean Up, e che l'esplosione eseguisse il suo evento Create - o potreste avere il vostro personaggio giocatore che si trasforma in un altro - nel qual caso l'argomento "perf" sarebbe probabilmente falso poiché non volete che le istanze eseguano i loro eventi Create e Destroy/Clean Up.

Vale la pena notare che cambiare l'istanza significa che non si dovrebbero eseguire ulteriori azioni con quell'istanza fino al passo successivo, in particolare cercando di accedere a variabili ecc... poiché ciò causerà un errore. Fondamentalmente, si cambia l'istanza ma non è effettivamente disponibile fino alla fine del passo corrente, quindi per accedere a qualsiasi variabile che contiene direttamente (per esempio, chiamando obj_Changed.x) non funzionerà.

ATTENZIONE! Quando si cambia un'istanza abilitata alla fisica, le proprietà fisiche non saranno trasferite alla nuova istanza in cui si cambia. Pertanto dovreste avere del codice in atto per "trasferire" lo stato fisico dell'istanza corrente alla nuova istanza, o aver definito le proprietà fisiche della nuova istanza nel suo evento Create, o nell'editor object. Per questo motivo si raccomanda di non usare questa funzione con istanze abilitate alla fisica, ma piuttosto di usare una combinazione di instance_destroy() e instance_create_layer().

Sintassi:

instance_change(obj, perf);

Argomento Tipo Descrizione
obj Oggetto Attività The new object the calling object will change into.
perf Booleano Whether to perform that new object's Create and Destroy events (true) or not (false).

Ritorna:

N/A

Esempio:

if keyboard_check(vk_enter)
{
    instance_change(obj_Player_Swimming, false);
    exit;
}

Il codice di cui sopra cambierà l'istanza chiamante in un'istanza di "obj_Player_Swimming" senza eseguire l'evento di distruzione dell'istanza originale né l'evento di creazione della nuova istanza quando l'utente preme il tasto "Enter", uscendo dal blocco di codice al termine.
