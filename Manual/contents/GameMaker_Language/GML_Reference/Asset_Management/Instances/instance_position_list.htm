elenco istanze_posizione

Con questa funzione potete controllare una posizione per una collisione con tutte le istanze di un altro object. Quando usate questa funzione state controllando un singolo punto in room per tutte le istanze di un object, e il controllo sarà fatto contro il bounding box delle istanze o contro la maschera delle istanze se quel object ha collisioni precise controllate.

Si fornisce anche una lista DS, quindi i id di tutte le istanze che collidono con l'istanza chiamante saranno aggiunte alla fine della lista data. Potete cancellare la lista prima di chiamare questa funzione in modo che contenga solo i risultati di questa chiamata di funzione. Avete anche la possibilità di ordinare la lista in base alla distanza dalla posizione controllata all'origine delle istanze trovate in collisione.

Si noti che la funzione accetta anche la parola chiave speciale allnel qual caso saranno elencate tutte le istanze trovate in collisione. La funzione restituisce il numero di istanze trovate, o 0 se non ne viene trovata nessuna.

Sintassi:

instance_position_list(x, y, obj, list, ordered);

Argomento Tipo Descrizione
x Real The x position to check for instances.
y Real The y position to check for instances.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instances of.
list ID elenco DS The DS list to use to store the IDs of colliding instances.
ordered Booleano Whether the list should be ordered by distance (true) o no (false).

Ritorna:

Real (The number of instances found to be in collision)

Esempio:

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Il codice di cui sopra controllerà per una collisione con tutte le istanze trovate nella posizione del mouse. Queste saranno aggiunte ad una lista DS, che viene poi percorsa in loop per distruggere ciascuna delle istanze in collisione.
