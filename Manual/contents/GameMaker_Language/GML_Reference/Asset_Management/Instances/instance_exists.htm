istanza_esiste

Questa funzione può essere usata in due modi, a seconda di ciò che si vuole controllare. Potete darle un object_index da controllare, nel qual caso questa funzione restituirà true se esistono istanze attive dell'indirizzo object specificato nel corrente room, o potete anche fornirle un id di istanza, nel qual caso questa funzione restituirà true se quell'istanza specifica esiste ed è attiva nel corrente room.

Si prega di notare che questa funzione non tiene conto delle istanze che sono state disattivate utilizzando le funzioni di disattivazione delle istanze.

Sintassi:

instance_exists(obj);

Argomento Tipo Descrizione
obj ID dell'istanza o dell'oggetto Asset The object or instance to check for the existence of.

Ritorna:

Boolean

Esempio:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

Il codice di cui sopra controlla se esiste qualche istanza di object "obj_Enemy" e se non esiste aggiunge alla variabile "score" e cambia room.

Indietro: Istanze
Il prossimo: instance_change
