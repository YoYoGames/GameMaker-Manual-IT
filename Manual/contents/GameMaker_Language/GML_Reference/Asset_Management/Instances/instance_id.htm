instance_id

Questo array di sola lettura contiene tutti i iddi ogni istanza attiva all'interno di room. Questo significa che se avete usato una qualsiasi delle funzioni Instance Deactivate quelle istanze che sono state disattivate non saranno incluse in questo array (se avete usato un valore da questo array in precedenza, ora restituirà la parola chiave noone).

Sintassi:

instance_id[num];

Ritorna:

Instance ID

Esempio:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

Il codice di cui sopra loop attraverserà tutte le istanze all'interno di room e aggiungerà 0,1 alla loro velocità.

Indietro: Istanze
Il prossimo: instance_count
