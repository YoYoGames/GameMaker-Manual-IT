allarme_set

Questa funzione può essere usata per impostare un allarme. Si fornisce il numero di allarme da 0 a 11, e poi il valore su cui impostare l'allarme. Il valore deve essere un valore intero, e potete impostarlo a -1 per fermare l'allarme (i valori non interi saranno arrotondati al numero intero più vicino). Questo è un metodo alternativo all'impostazione diretta dell'array di allarmi.

Sintassi:

alarm_set(index, value);

Argomento Tipo Descrizione
index Real The alarm index to set, from 0 to 11.
value Real The value (an integer) to set the alarm to.

Ritorna:

N/A

Esempio:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Il codice di cui sopra controlla tutti gli allarmi nell'istanza chiamante e se sono maggiori di 0 li imposta a -1, impedendo loro di continuare il conto alla rovescia.
