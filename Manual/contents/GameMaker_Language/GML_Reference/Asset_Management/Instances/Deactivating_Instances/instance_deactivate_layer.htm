istanza_disattiva_strato

Con questa funzione si possono disattivare tutte le istanze assegnate a uno specifico livello. È necessario fornire l'ID del livello, che può essere sia il nome del livello come scritto nell'editor di codice (come string) o il valore effettivo dell'ID del livello come restituito dalla funzione layer_create()e si noti che si possono disattivare solo i livelli di istanza con questa funzione. Si noti che se si è disattivato un livello che ha istanze di objects contrassegnate come Persistent, allora sarà necessario riattivare nuovamente il livello con la funzione instance_activate_layer() prima di cambiare room, altrimenti qualsiasi istanza persistente sul livello non sarà riportata e sarà scartata. Si noti inoltre che la disattivazione non è istantanea, e un'istanza che è stata disattivata in questo modo non sarà considerata inattiva fino alla fine dell'evento in cui la funzione è stata chiamata.

NOTA: se si disattiva un'istanza all'inizio di room (cioè: dal codice di creazione di room, o da un evento di creazione di un'istanza all'interno di room) tutte le istanze che sono posizionate all'interno di room dall'editor di room eseguiranno ancora il loro evento di creazione prima di essere disattivate.

ATTENZIONE: Disattivare le istanze che hanno la fisica abilitata NON impedirà ai loro apparecchi di interagire all'interno della simulazione fisica. Per questo dovreste impostare la loro variabile phy_active su true o false quando attivate/disattivate le istanze.

Sintassi:

instance_deactivate_layer(obj);

Argomento Tipo Descrizione
layer ID del livello o stringa The layer name string (or ID value) to be used

Ritorna:

N/A

Esempio:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Il codice precedente disattiva tutte le istanze assegnate al livello "Enemy_Layer" e poi attiva una regione all'interno di room.
