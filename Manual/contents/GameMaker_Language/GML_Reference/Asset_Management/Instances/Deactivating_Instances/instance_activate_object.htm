istanza_attiva_oggetto

Con questa funzione potete attivare una singola istanza o tutte le istanze di uno specifico object da tutte quelle che sono state disattivate in precedenza. Notate che se avete disattivato un'istanza o object che è stato contrassegnato come persistente, allora sarà necessario riattivarla di nuovo con questa funzione prima di cambiare room, altrimenti non verrà riportata e verrà invece scartata. Si noti inoltre che l'attivazione non è istantanea, e un'istanza che è stata attivata in questo non sarà considerata attiva fino alla fine dell'evento in cui la funzione è stata chiamata.

Sintassi:

instance_activate_object(obj);

ArgomentoTipo Descrizione
obj The object or instance to activate (the keyword all can also be used).

Ritorna:

Esempio:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

Il codice di cui sopra attiva tutte le istanze all'interno di room e poi disattiva quelle che sono al di fuori dei limiti della vista della telecamera corrente, tranne che per object "obj_Lights" che vengono riattivate di nuovo alla fine.

Indietro: Disattivare le istanze
