istanza_disattiva_oggetto

Con questa funzione potete disattivare una singola istanza o tutte le istanze di uno specifico object da tutte quelle che sono state attivate in precedenza. Notate che se avete disattivato un'istanza o object che è stato contrassegnato come persistente, allora sarà necessario riattivarlo di nuovo con la funzione instance_activate_object() prima di cambiare room, altrimenti non verrà riportata e verrà invece scartata. Si noti inoltre che la disattivazione non è istantanea, e un'istanza che è stata disattivata in questo modo non sarà considerata inattiva fino alla fine dell'evento in cui la funzione è stata chiamata.

NOTA: se si disattiva un'istanza all'inizio di room (cioè: dal codice di creazione di room, o da un evento di creazione di un'istanza all'interno di room) tutte le istanze che sono posizionate all'interno di room dall'editor di room eseguiranno ancora il loro evento di creazione prima di essere disattivate.

ATTENZIONE: Disattivare le istanze che hanno la fisica abilitata NON impedirà ai loro apparecchi di interagire all'interno della simulazione fisica. Per questo dovreste impostare la loro phy_active variabile su true o false mentre attivate/disattivate le istanze.

Sintassi:

instance_deactivate_object(obj);

ArgomentoTipo Descrizione
obj The object or instance to deactivate (the keyword all can also be used).

Ritorna:

Esempio:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Il codice qui sopra disattiva tutte le istanze di object "obj_Enemy" e poi attiva una regione all'interno di room.

Indietro: Disattivare le istanze
Il prossimo: instance_deactivate_region
