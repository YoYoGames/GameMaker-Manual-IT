luogo_istanza

Con questa funzione potete controllare una posizione per una collisione con un'altra istanza o con tutte le istanze di un object usando la maschera di collisione dell'istanza che esegue il codice per il controllo. Quando usate questa funzione state effettivamente chiedendo a GameMaker di spostare l'istanza nella nuova posizione, controllare la collisione, tornare indietro e dirvi se una collisione è stata trovata o meno.

Questo funzionerà per le collisioni precise, ma solo se sia l'istanza che il object per cui si sta controllando hanno maschere di collisione precise selezionate, altrimenti vengono applicate solo le collisioni del bounding box.

Questa funzione restituirà l'unica istanza id del object in collisione, ma se questo non è necessario è leggermente più veloce usare la funzione place_meeting(). Questa funzione accetta anche le parole chiave speciali all e other e restituirà la parola chiave noone se non si verifica alcuna collisione, o il valore ID dell'istanza unica trovata se si verifica una collisione.

Sintassi:

instance_place(x, y, obj);

Argomento Tipo Descrizione
x Real The x position to check for instances.
y Real The y position to check for instances.
obj ID dell'asset dell'oggetto o dell'istanza The object to check for instances of.

Ritorna:

Instance ID or noone

Esempio:

var _inst = instance_place(x, y, obj_Enemy);
if (_inst != noone)
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

Il codice di cui sopra controllerà per una collisione con istanze di "obj_Enemy" e se ce n'è una, ridurrà la variabile "hp" della quantità memorizzata nella variabile "dmg" dell'istanza in collisione e poi distruggerà l'istanza in collisione.
