trova_istanza

Tutte le istanze hanno un identificatore unico (id) che può essere usato per modificarle e manipolarle mentre un gioco è in esecuzione, ma non sempre si può sapere quale sia l'id di una specifica istanza e quindi questa funzione può essere d'aiuto in quanto può essere usata per iterare attraverso tutte le istanze per trovare ciò di cui si ha bisogno. Specificate il sito object di cui volete trovare l'istanza e un numero, e se c'è un'istanza in quella posizione nella lista delle istanze, allora la funzione restituisce l'id di quell'istanza, altrimenti restituisce la parola chiave speciale noone. Potete anche usare la parola chiave all per iterare attraverso tutte le istanze in un room, così come un genitore object per iterare attraverso tutte le istanze che fanno parte di quella gerarchia genitore/figlio, e potete anche specificare un'istanza (se avete il suo id) come controllo per vedere se effettivamente esiste nel corrente room. Si prega di notare che, poiché le istanze sono ordinate in modo arbitrario, non c'è un ordine specifico per come le istanze sono controllate da questa funzione, e qualsiasi istanza può essere in qualsiasi posizione.

Il valore massimo di "n" in questa funzione sarebbe

  • Per la parola chiave all: instance_count - 1
  • Per un indice object: instance_number(OBJ) - 1

Sintassi:

instance_find(obj, n);

Argomento Tipo Descrizione
obj Oggetto Attività The object to find the nth instance of
n Real The number of the instance to find.

Ritorna:

Instance ID or noone

Esempio:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

Il codice di cui sopra userà un for loop per iterare tutte le istanze di "obj_Enemy" e memorizzare il loro id nell'array "enemy[]".
