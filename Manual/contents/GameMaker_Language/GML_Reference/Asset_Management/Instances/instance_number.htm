numero di istanza

Con questa funzione si può scoprire quante istanze attive del object specificato esistono nel room. Quando si controlla usando questa funzione, se il object è un genitore, allora anche tutti i figli objects saranno inclusi nel valore di ritorno, e si noti anche che quelle istanze che sono state disattivate con le funzioni instance deactivate non saranno incluse in questo controllo.

Sintassi:

instance_number(obj);

ArgomentoTipo Descrizione
obj The object to total the number of instances of.

Ritorna:

Esempio:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

Il codice di cui sopra controllerà il numero di istanze che sono create dallo stesso object come l'istanza corrente e poi se ci sono meno di 50, ne creerà un'altra in una posizione casuale all'interno di room.

Indietro: Istanze
