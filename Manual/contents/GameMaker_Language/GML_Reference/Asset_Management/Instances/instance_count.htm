numero di istanze

Con questa variabile di sola lettura potete ottenere un conteggio di tutte le istanze attive che si trovano in room. Questo include l'istanza che esegue il codice, ma non include le istanze che sono state disattivate usando le funzioni di disattivazione delle istanze. Notate che questa variabile vi darà solo il numero di istanze all'inizio del passo, quindi qualsiasi modifica alle istanze in room fatta dopo che il passo è iniziato non sarà presa in considerazione.

Sintassi:

instance_count;

Ritorna:

Esempio:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

Il codice di cui sopra creerà istanze multiple di object "obj_Star" fino a quando il numero totale di istanze raggiunge 100.
