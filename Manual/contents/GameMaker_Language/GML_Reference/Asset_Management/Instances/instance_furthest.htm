istanza_più lontana

Questa funzione controllerà tutte le istanze del dato object per vedere quale è più lontano dal dato punto x/y. Tutti i controlli saranno dalla posizione x/y data all'origine (la posizione x/y) delle istanze del object specificato. Se non esistono istanze di object, la funzione restituirà la parola chiave noone, ma se ci sono istanze allora restituirà la parola chiave id dell'istanza trovata. Si prega di notare che se l'istanza che esegue il codice ha lo stesso indice object del object che viene controllato, allora sarà incluso nel controllo (questo include i controlli per i genitori objects se l'istanza chiamante è anche un figlio del genitore).

Sintassi:

instance_furthest(x, y, obj);

ArgomentoTipo Descrizione
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

Ritorna:

Instance ID or noone

Esempio:

var inst;
inst = instance_furthest(x, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}

Il codice di cui sopra troverà l'istanza più lontana che condivide lo stesso indice object dell'istanza che esegue il codice e memorizzerà l'id in una variabile. Questa variabile viene poi controllata per vedere se è uguale all'id dell'istanza chiamante e, se non lo è, viene tracciata una linea tra le due istanze.
