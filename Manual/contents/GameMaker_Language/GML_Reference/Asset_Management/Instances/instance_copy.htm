copia_istanza

Con questa funzione potete "clonare" un'istanza, poiché questo creerà una nuova versione dell'istanza che esegue il codice nella stessa posizione. L'argomento "perf" è usato per istruire questa nuova istanza ad eseguire o meno l'evento di creazione. Questa funzione restituisce il id della nuova istanza che può quindi essere memorizzata in una variabile o utilizzata per accedere a tale istanza.

NOTA: se si sceglie di non eseguire l'evento di creazione, si possono incontrare errori se l'istanza dipende da qualsiasi variabile inizializzata in questo evento.

Sintassi:

instance_copy(perf);

Argomento Tipo Descrizione
perf Booleano Whether to perform the new instance's Create event (true) or not (false).

Ritorna:

Instance ID

Esempio:

var inst = instance_number(object_index);
if inst < 10
{
    instance_copy(true);
}

Il codice precedente crea una variabile locale e la usa per memorizzare il numero di istanze di object che eseguono il codice in room. Se il numero è inferiore a 10, l'istanza fa una copia di se stessa.
