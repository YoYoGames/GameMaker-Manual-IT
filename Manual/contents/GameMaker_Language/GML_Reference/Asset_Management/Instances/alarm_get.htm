ottenere l'allarme

Questa funzione può essere utilizzata per ottenere il valore corrente dell'allarme dato. Si fornisce il numero dell'allarme da 0 a 11 e questa funzione restituisce il valore che l'allarme ha attualmente. Questo è un metodo alternativo per ottenere direttamente il valore dell'array di allarmi.

Sintassi:

alarm_get(index);

Argomento Tipo Descrizione
index Real The alarm index to get, from 0 to 11.

Ritorna:

Real (integer)

Esempio:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Il codice di cui sopra controlla tutti gli allarmi nell'istanza chiamante e se sono maggiori di 0 li imposta a -1, impedendo loro di continuare il conto alla rovescia.

Indietro: Istanze
Il prossimo: alarm_set
