istanza_id_get

Con questa funzione potete ottenere il valore ID unico di qualsiasi istanza dalla lista di istanze attualmente attiva. Voi date l'indice nella lista delle istanze da cui ottenere l'ID e la funzione restituirà il valore da memorizzare in una variabile.

Sintassi:

instance_id_get(index);

Argomento Tipo Descrizione
index Real The index within the instance list from 0 al numero di istanze - 1.

Ritorna:

Instance ID

Esempio:

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

Il codice di cui sopra loop attraverserà tutte le istanze all'interno di room e aggiungerà 0,1 alla loro velocità.
