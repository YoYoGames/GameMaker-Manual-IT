indice del canale dell'animazione

Questa funzione restituisce il valore dell'indice per ogni dato canale animation curve. Si fornisce l'ID di animation curve o struct, dove l'ID della curva è il nome del animation curve come è stato definito nel Browser Asset, o il puntatore struct come restituito dalla funzione animcurve_create(). Si fornisce poi il nome del canale, come string, e la funzione restituirà il valore dell'indice associato a quel canale. Notate che se la curva o il canale non esiste, otterrete un errore.

Sintassi:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argomento Tipo Descrizione
curve_struct_or_id Struttura della curva di animazione The ID or struct pointer of the animation curve to target
channel_name Stringa The channel name (a string).

Ritorna:

Real

Esempio:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Il codice qui sopra recupera il canale struct per il canale chiamato "x_pos" nella curva asset "ac_ButtonTween", poi loops attraverso i punti sulla curva del canale e aggiunge uno al loro valore.
