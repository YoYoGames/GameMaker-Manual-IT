animcurve_get_channel

Questa funzione restituisce la struct contenente i dati del canale per il canale specificato in un animation curve asset. Voi fornite l'ID animation curve o struct così come il nome o l'indice del canale, e la funzione restituisce un struct con il seguente formato:

Nome della variabile Tipo di dati Descrizione
name stringa Questo è il nome del canale.
type Tipo di interpolazione della curva di animazione Costante Questa sarà una delle costanti animcurvetype_linear per l'interpolazione lineare tra punti, animcurvetype_catmullrom per l'interpolazione "liscia" tra i punti usando l'interpolazione catmull-rom, o animcurvetype_bezier per l'interpolazione Bezier.
iterations intero Se il canale sta usando l'interpolazione catmull-rom ("smooth") questo tiene quanti punti sono stati generati per ogni segmento della curva (nota che questi punti extra sono interni alla funzione e usati solo per i calcoli di runtime ). Se il canale sta usando l'interpolazione lineare, questo valore esisterà ancora ma può essere ignorato poiché non ha alcuna influenza su come la curva viene interpolata.
points puntatore di matrice Questo è un array, dove ogni elemento dell'array è un punto struct.


L'ID animation curve è il nome del animation curve come è stato definito nel Browser Asset, o il puntatore struct come restituito dalla funzione animcurve_create(). Il nome del canale è un string che si riferisce al canale come è stato definito nell'Animation Curve asset, o potete fornire un valore di indice, che va da 0 a n, dove n è l'ultimo canale nella curva asset (es: se un animation curve ha 4 canali, saranno indicizzati da 0 a 3). Si noti che passare un valore di indice richiederà meno elaborazione che passare il nome di un canale. Se la funzione fallisce (cioè: non esiste un canale con il nome o l'indice dato) allora si otterrà un errore.

I punti su un singolo canale sono memorizzati come structs in un array, dove ogni elemento dell'array è un singolo punto struct. Il punto struct ha le seguenti variabili:

Nome della variabile Tipo di dati Descrizione
posx vero La posizione nel tempo (normalizzata da 0 a 1) del punto.
value vero Il valore del punto.

Sintassi:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Argomento Tipo Descrizione
curve_struct_or_id Struttura della curva di animazione The ID or struct pointer of the animation curve to target
channel_name_or_index Stringa o Reale The channel name (a string) or the channel index (an integer).

Ritorna:

Animation Curve Channel Struct

Esempio:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Il codice qui sopra recupera il canale struct per il canale 0 nella curva asset "ac_ButtonTween", poi loops attraverso i punti sulla curva del canale e aggiunge uno al loro valore.
