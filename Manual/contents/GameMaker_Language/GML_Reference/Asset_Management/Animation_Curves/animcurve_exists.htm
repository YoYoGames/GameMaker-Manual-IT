esiste una curva animata

Questa funzione restituirà true se il dato Animazione Curva Asset o Struttura della curva di animazione esiste ed è una Curva di animazione valida, o falso se non lo è.

Sintassi:

animcurve_exists(curve_struct_or_id);

Argomento Tipo Descrizione
curve_struct_or_id Struttura della curva di animazione o Curva di animazione Asset The Animation Curve that will be checked.

Ritorna:

Boolean

Esempio:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

Il codice di cui sopra controlla se la curva di animazione memorizzata nella variabile spring_curve personalizzata esiste. Se esiste, esegue un metodo personalizzato per utilizzare quella curva di animazione.

Indietro: Curve di animazione
Dopo: animcurve_channel_new
