creare una curva animata

Questa funzione creerà una struct animation curve vuota, pronta per essere popolata con i dati dei canali. La funzione restituirà un puntatore a struct e questo è poi usato per aggiungere canali e altri dati a animation curve. Potete anche fornire questo a funzioni come animcurve_get() per ottenere successivamente i dati dalla curva.

Quando si usa questa funzione, il struct creato non avrà dati al suo interno, e per usarlo si deve aggiungere almeno un canale e il canale deve avere punti aggiunti. Per aggiungere un canale, vedi la funzione animcurve_channel_new()e per aggiungere punti al canale, vedere la funzione animcurve_point_new(). Inoltre puoi dare un nome a animation curve struct impostando la variabile "name" (come mostrato nel codice di esempio, sotto). I canali aggiunti a un nuovo animation curve dovrebbero essere aggiunti come un array, dove ogni elemento dell'array fa riferimento a un canale struct, e i punti di un canale dovrebbero essere aggiunti come un array, dove ogni elemento dell'array fa riferimento a un punto struct.

Notate che animation curves creato in questo modo deve essere cancellato quando non è più necessario, perché altrimenti occuperà spazio in memoria, il che può portare a una perdita di memoria, rallentando e infine mandando in crash il gioco. È possibile rimuovere un animation curve quando non è più necessario utilizzando la funzione animcurve_destroy(). Non c'è bisogno di ripulire i dati dei canali o dei punti all'interno della curva, in quanto questo verrà fatto automaticamente dal garbage collector quando la curva stessa verrà distrutta.

Sintassi:

animcurve_create();

Ritorna:

Animation Curve Struct

Esempio:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Il codice precedente crea un nuovo animation curve struct e lo memorizza nella variabile "my_curve". Questo struct è poi popolato con un nome e un array di canali. L'array di canali contiene un singolo canale con tre punti.

Indietro: Curve di animazione
