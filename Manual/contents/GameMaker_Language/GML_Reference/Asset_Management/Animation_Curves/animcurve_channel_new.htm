nuovo canale animato

Questa funzione può essere usata per creare una nuova struct di canale animation curve. Un canale struct ha le seguenti variabili:

Struttura del canale della curva di animazione
Nome della variabile Tipo di dati Descrizione
nome stringa Questo è il nome del canale.
tipo Tipo di interpolazione della curva di animazione Costante Questa sarà una delle costanti date nella tabella sottostante.
iterazioni intero Se il canale sta usando l'interpolazione catmull-rom ("smooth") questo tiene quanti punti sono stati generati per ogni segmento della curva (nota che questi punti extra sono interni alla funzione e usati solo per i calcoli di runtime ). Se il canale sta usando l'interpolazione lineare, questo valore esisterà ancora ma può essere ignorato poiché non ha alcuna influenza su come la curva viene interpolata.
punti array Questo è un array, dove ogni elemento dell'array è un Struttura del punto della curva di animazione.

Tipo di interpolazione della curva di animazione Costante
Costante Descrizione
animcurvetype_linear Utilizzato per l'interpolazione lineare tra punti.
animcurvetype_catmullrom Utilizzato per l'interpolazione liscia tra i punti usando l'interpolazione catmull-rom.
animcurvetype_bezier Utilizzato per l'interpolazione di Bezier tra i punti.


Per default quando create un nuovo canale struct, la variabile "name" sarà un string vuoto, il "type" sarà animcurvetype_linear e le "iterazioni" saranno 16. Tutte queste variabili possono essere impostate ai valori che desideri (nota che il valore "iterazioni" non ha effetto sui tipi di curve lineari). Una volta creata, è necessario aggiungere punti alla curva, il che viene fatto aggiungendo diversi punti structs all'array"points". Questi punti structs possono essere creati utilizzando la funzione animcurve_point_new().

Sintassi:

animcurve_channel_new();

Ritorna:

Animation Curve Channel Struct

Esempio:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Il codice precedente crea un nuovo animation curve struct e lo memorizza nella variabile "my_curve". Questo struct è poi popolato con un nome e un array di canali. L'array di canali contiene un singolo canale con tre punti.
