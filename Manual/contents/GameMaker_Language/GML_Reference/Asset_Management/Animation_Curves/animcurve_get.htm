ottenere la curva dell'anima

Questa funzione restituirà una struttura contenente tutti i dati per il dato animation curve. Si fornisce l'ID animation curve asset (come definito nel Browser Asset ), e la funzione restituirà un struct con le seguenti variabili:

Struttura della curva di animazione
Nome della variabile Tipo di dati Descrizione
name Stringa Questo è il nome del animation curve.
channels Array di Struttura del canale della curva di animaziones Questo è un array, dove ogni elemento dell'array è un canale struct.


La variabile channels in struct è un array dove ogni elemento dell'array è un Struttura del canale della curva di animazione con dati relativi ad un canale all'interno della curva. Il canale struct è spiegato in questa pagina.

Come per i canali, i punti di un singolo canale sono memorizzati come structs in un array, dove ogni elemento dell'array è un singolo punto struct. Il punto struct è spiegato in questa pagina.

Si noti che se la funzione fallisce (ad esempio, la curva di animazione data asset non esiste), allora la funzione restituirà -1.

Sintassi:

animcurve_get(curve_id);

Argomento Tipo Descrizione
curve_id Animazione Curve Asset The asset browser ID (index) of the animation curve to get.

Ritorna:

Animation Curve Struct or -1

Esempio:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

Il codice qui sopra recupera il animation curve struct per la curva asset "ac_ButtonTween", poi controlla il tipo di curva per il primo canale, e se non è lineare lo imposta a lineare.
