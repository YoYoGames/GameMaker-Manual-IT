Curve di animazione

Le curve di animazione possono essere impostate nell'Asset Browser come assets usando l'Animation Curve Editor e possono poi essere usate nell'Editor delle Sequenze per controllare le tracce in un sequence. Tuttavia possono anche essere usate nel vostro codice per cambiare i valori nel tempo nei vostri giochi per qualsiasi cosa, dagli effetti di tweening sui pulsanti, al movimento, al cambio di colore e altro. Potete anche creare il vostro animation curves su runtime usando le funzioni qui elencate, rendendole uno strumento molto flessibile per i vostri giochi.

Le funzioni disponibili per le curve di animazione sono le seguenti:

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • esiste una curva animata
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
