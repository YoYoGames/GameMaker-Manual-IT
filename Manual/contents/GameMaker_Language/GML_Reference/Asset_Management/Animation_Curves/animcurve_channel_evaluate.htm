animcurve_canali_valutati

Questa funzione può essere utilizzata per ottenere il valore in un momento specifico da una struttura di canale. Si fornisce il puntatore struct del canale (come restituito dalla funzione animcurve_get_channel()o come restituito in animation curve struct dalla funzione animcurve_get()) e la posizione "x" (tempo) lungo il canale della curva da valutare. Questa posizione deve essere compresa tra 0 e 1 e la funzione restituirà il valore della curva in quella posizione, oppure restituirà 0 se il canale struct fornito non è valido.

Sintassi:

animcurve_channel_evaluate(channel_struct, posx);

Argomento Tipo Descrizione
channel_struct Strutture del canale della curva di animazione The struct pointer for the channel to evaluate.
posx Reale The position in time to check (from 0 to 1).

Restituzione:

Real

Esempio:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

Il codice precedente ottiene il canale struct per animation curve asset "ac_AlphaCurve". Utilizza quindi il valore di valutazione restituito per impostare l'alfa dell'immagine per l'istanza.

Indietro: Curve di animazione
Il prossimo: animcurve_create
