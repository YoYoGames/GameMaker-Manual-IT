ombreggiatore di livello

Con questa funzione si può assegnare una risorsa shader a qualsiasi dato livello e il livello sarà poi reso usando quel shader. Si fornisce il valore ID unico del livello o il nome del livello (come string - questo avrà un impatto sulle prestazioni), insieme all'ID del shader da usare. Il shader deve essere stato creato in precedenza nel Browser Asset e l'indice shader (il nome della risorsa shader ) viene poi passato a questa funzione. Se il livello assegnato ha istanze aggiunte ad esso, allora il shader sarà applicato a tutti gli eventi di disegno che l'istanza usa - per esempio se l'istanza ha un evento Draw GUI Begin, allora il shader sarà applicato automaticamente ad esso. Il shader influenzerà anche qualsiasi altro elemento grafico disegnato su quel livello, come le mappe sprite assets o tile.

Si noti che la funzione non deve essere chiamata in nessun evento di disegno o di passo, ma piuttosto deve essere chiamata solo all'inizio del room nel codice di creazione della stanza o nell'evento Create / Room Start Event di un'istanza.

Sintassi:

layer_shader(layer_id, shader)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
shader Shader Asset The shader index to assign to the layer

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

Il codice qui sopra assegnerà la risorsa shader "shd_Sepia" al dato livello per tutti i disegni.
