layer_get_shader

Questa funzione può essere usata per controllare se il dato livello ha un shader assegnato ad esso. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni), e la funzione restituirà o l'indice shader del shader assegnato, o -1 se nessun shader è assegnato.

Sintassi:

layer_get_shader(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)

Ritorna:

Shader Asset or -1 (if no shader is assigned)

Esempio:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

Il codice di cui sopra controllerà per vedere se il livello che l'istanza che esegue il codice ha un shader assegnato ad esso e se non ce l'ha viene assegnato.
