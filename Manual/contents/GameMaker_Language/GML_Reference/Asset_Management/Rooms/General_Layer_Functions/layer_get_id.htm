ottenere l'ID del livello

Questa funzione può essere usata per ottenere il valore ID unico di un dato livello. In IDE, tutti i livelli hanno un nome e un tipo, e per essere in grado di modificarli o cambiarli attraverso il codice è necessario dare il valore ID del livello. Questa funzione è usata per recuperare questo ID usando il nome (a string) del livello (come scritto in IDE). Se si crea un nuovo livello attraverso il codice usando la funzione layer_create() allora quella funzione restituirà invece il valore unico dell'ID (i livelli creati dinamicamente non ottengono nomi). Si noti che se si dà il nome di un livello che non esiste nel corrente room, allora si otterrà un errore e il progetto andrà in crash.

Sintassi:

layer_get_id(layer_name)

ArgomentoTipo Descrizione
layer_name The name of the layer (a string)

Ritorna:

Layer ID or -1 (if the layer specified doesn't exist)

Esempio:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Il codice qui sopra otterrà prima l'indice dell'istanza più vicina alla posizione x/y data e lo memorizzerà in una variabile locale. Poi ottiene l'ID univoco del livello di istanza per il livello chiamato "Instances Front", e sposta l'istanza trovata su quel livello.
