layer_y

Puoi usare questa funzione per impostare la posizione y del livello all'interno dello scope corrente room. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione sposterà il livello del dato numero di pixel lungo l'asse verticale di room.

Sintassi:

layer_y(layer_id, y)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to set the y position of
y Real The y position in the room to set the layer to

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Il codice di cui sopra controlla la posizione del livello dato e se non è impostato su (0, 0) allora è impostato su quella posizione.
