ha un'istanza

Questa funzione può essere usata per controllare se una data istanza è attualmente assegnata al dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e l'ID dell'istanza da controllare. Puoi anche dare un object_index (cioè: il nome del object nel Browser Asset ) e la funzione controllerà se ci sono istanze di quel object sul livello dato. La funzione restituirà true se l'istanza è sul livello e false se non lo è.

Sintassi:

layer_has_instance(layer_id, instance_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
instance_id ID istanza The unique instance ID or the object index of the instance to check for

Ritorna:

Boolean

Esempio:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Il codice di cui sopra controllerà per vedere se il livello dato contiene qualche istanza di object "obj_Bullet_Parent" e se non lo è distruggerà il livello.
