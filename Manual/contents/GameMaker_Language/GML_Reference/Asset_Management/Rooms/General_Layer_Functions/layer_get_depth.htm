layer_get_depth

Puoi usare questa funzione per ottenere il valore di profondità associato ad un dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituirà la profondità del livello come numero reale. Nota che la profondità è definita come più alta quanto più "lontano" dalla telecamera e più bassa quanto più "vicino" alla telecamera, quindi se hai tre livelli con profondità -100, 0, 100, i livelli saranno disegnati nell'ordine 100, 0, -100, così che il livello "superiore" (cioè il più vicino alla telecamera e quindi disegnato sopra tutto il resto) sarà il livello con profondità -100.

L'immagine seguente mostra uno schema di come funziona la profondità per i livelli:

Nota che se fornisci un valore di ID di livello non valido otterrai un errore.

Sintassi:

layer_get_depth(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to get the depth of (or the layer name as a string)

Ritorna:

Real

Esempio:

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

Il codice qui sopra controlla la profondità dell'ID di un livello memorizzato in una variabile globale e se non è -100 distrugge il livello e lo ricrea alla profondità di -100.
