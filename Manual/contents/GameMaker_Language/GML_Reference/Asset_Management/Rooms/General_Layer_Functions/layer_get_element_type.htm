tipo_di_elemento_strato

Potete usare questa funzione per ottenere il tipo di elemento per l'elemento dato. Fornite il valore unico dell'ID dell'elemento (per esempio, come restituito dalla funzione che ha creato l'elemento o dall'editor room ) e la funzione restituirà una delle seguenti costanti (o -1 se l'elemento non esiste o il valore ID è errato):

Elemento di strato Tipo di costante
Costante Descrizione
layerelementtype_background L'elemento è uno sfondo.
layerelementtype_instance L'elemento è un'istanza.
layerelementtype_sprite L'elemento è un sprite asset.
layerelementtype_tilemap L'elemento è una tilemap.
layerelementtype_particlesystem L'elemento è un sistema di particelle.
layerelementtype_tile L'elemento è uno sfondo legacy tile (questo è valido solo per progetti che sono stati importati da versioni precedenti di GameMaker).
layerelementtype_sequence L'elemento è un sequence asset.

Notate che questa funzione è più utile quando avete più tipi di elementi diversi assegnati a un livello, e che potete ottenere una lista di tutti gli elementi su un dato livello usando la funzione layer_get_all_elements().

Sintassi:

layer_get_element_type(element_id)

Argomento Tipo Descrizione
layer ID dell'elemento di strato The unique ID value of the element to get the type of

Ritorna:

Layer Element Type Constant or -1 (if element does not exist or is invalid)

Esempio:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i]);
    }
}

Il codice di cui sopra ottiene gli ID per tutti gli elementi dell'istanza assegnati al livello dell'istanza che esegue il codice. Il codice controlla poi per vedere se qualcuno degli elementi restituiti è sprite assets e se lo è, viene distrutto.
