Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e questa funzione restituirà l'indice della funzione script della funzione assegnata da eseguire all'inizio del rendering per quel livello, o restituirà -1 se nessuna funzione è assegnata. È possibile assegnare funzioni script a un livello con layer_script_begin() e layer_script_end().

Sintassi:

layer_get_script_begin(layer_id);

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)

Ritorna:

Script Function or -1 (if no function is assigned)

Esempio:

if layer_get_script_begin(layer) == -1
{
    layer_script_begin(layer, scr_SetShaderValues);
}

Il codice qui sopra controllerà per vedere se il livello che l'istanza che esegue il codice ha una funzione script assegnata ad esso e se non ce l'ha viene assegnata.
