velocità_strato

Potete usare questa funzione per impostare la velocità orizzontale (in pixel per fotogramma di gioco) del livello all'interno dello scope corrente room. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e il valore di velocità da impostare, dove un valore positivo è verso destra e un valore negativo verso sinistra.

Sintassi:

layer_hspeed(layer_id, hspd)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to set the horizontal speed of
hspd Real The horizontal speed (in pixels per game frame) to set

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Il codice di cui sopra controlla le velocità orizzontali e verticali del livello dato e se non sono entrambe impostate su 0, allora le imposta su 0.
