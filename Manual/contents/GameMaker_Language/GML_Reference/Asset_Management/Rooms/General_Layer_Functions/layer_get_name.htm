nome_di_strato

Puoi usare questa funzione per ottenere il nome del dato livello. Si fornisce il valore ID univoco del livello e se il livello è uno dei livelli nominati creati nell'editor room, allora la funzione restituirà un string con il nome del livello. Se il livello non è uno di quelli dell'editor room (cioè: è stato creato usando layer_create()) allora verrà restituito un string vuoto.

Sintassi:

layer_get_name(layer_id)

Argomento Tipo Descrizione
layer_id ID strato The unique ID value of the layer to get the name of

Ritorna:

String

Esempio:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

Il codice qui sopra ottiene gli ID per tutti i livelli in room e poi loops anche se controlla se qualcuno ha un nome. Se lo sono, il loro ID viene aggiunto ad una lista.
