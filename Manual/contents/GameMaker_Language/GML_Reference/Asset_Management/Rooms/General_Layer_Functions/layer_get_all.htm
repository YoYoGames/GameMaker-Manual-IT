ottenere tutti i livelli

Questa funzione restituirà un array popolato con i valori ID unici di ogni strato nel room.

Sintassi:

layer_get_all()

Ritorna:

Array (populated with Layer IDs)

Esempio:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

Il codice qui sopra recupera tutti i livelli in un room e aggiunge i loro valori ID ad un array. Questo array viene poi analizzato per distruggere o i livelli room.
