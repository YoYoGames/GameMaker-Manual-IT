ottieni tutti gli elementi

Puoi usare questa funzione per ottenere gli ID degli elementi del dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituirà un array di ID, dove ogni voce nell'array è un ID unico per un elemento su quel livello. Per esempio, se il livello è un Asset Layer, l'array sarà popolato con i valori ID per ogni sprite asset che è assegnato al livello. Si noti che usare il codice per lavorare con i livelli significa che si possono assegnare diversi tipi di elementi allo stesso livello - così si può avere sprite assets insieme alle istanze, per esempio - nel qual caso si può poi usare la funzione layer_get_element_type() per ottenere il tipo di elemento a cui l'ID si riferisce.

Sintassi:

layer_get_all_elements(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to get the elements from (or the layer name as a string)

Ritorna:

Array of Layer Element IDs

Esempio:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i])
    }
}

Il codice di cui sopra ottiene gli ID per tutti gli elementi dell'istanza assegnati al livello dell'istanza che esegue il codice. Il codice controlla poi per vedere se qualcuno degli elementi restituiti è sprite assets e se lo è, viene distrutto.
