layer_is_draw_depth_forced

Questa funzione può essere usata per controllare e vedere se la profondità Z forzata è abilitata per il rendering dei livelli nel progetto. Vedere layer_force_draw_depth() per maggiori informazioni.

Sintassi:

layer_is_draw_depth_forced()

Ritorna:

Esempio:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

Il codice qui sopra controlla se la profondità Z del livello è forzata o no e se non lo è, imposta la profondità Z a 0 e la abilita.
