esiste un livello

Questa funzione può essere usata per controllare se il dato livello esiste. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituirà un valore booleano di true se esiste o false se non esiste.

NOTA: questa funzione lavora nell'ambito del target corrente room - di default il room in cui la funzione è chiamata - che può essere impostato usando la funzione layer_set_target_room().

Sintassi:

layer_exists(layer_name)

Argomento Tipo Descrizione
layer_name Stringa o ID del livello The name of the layer (a string or ID value)

Ritorna:

Boolean

Esempio:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

Il codice qui sopra controllerà per vedere se il livello memorizzato nella variabile globale esiste effettivamente, e se non esiste allora viene creato.
