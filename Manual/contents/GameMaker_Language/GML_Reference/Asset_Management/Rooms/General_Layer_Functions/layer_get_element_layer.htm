layer_get_element_layer

Potete usare questa funzione per ottenere l'ID del livello su cui si trova l'elemento dato. Si fornisce il valore unico dell'ID dell'elemento (per esempio, come restituito dalla funzione che ha creato l'elemento o dall'editor di room ) e la funzione restituirà l'ID unico del livello su cui si trova l'elemento. Se l'ID dell'elemento fornito non è valido, allora la funzione restituirà -1.

Sintassi:

layer_get_element_layer(element_id)

Argomento Tipo Descrizione
element_id ID dell'elemento di strato The unique ID value of the element to get the layer of

Ritorna:

Layer ID or -1 (if the element is invalid)

Esempio:

element = layer_get_element_layer(asset_1);

Il codice precedente ottiene l'ID del livello per l'elemento con l'ID memorizzato nella variabile "asset_1" e lo memorizza in una variabile.
