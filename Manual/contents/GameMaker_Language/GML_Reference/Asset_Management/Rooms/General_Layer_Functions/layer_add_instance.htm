aggiungere un'istanza

Questa funzione può essere usata per spostare una data istanza dal livello su cui si trova attualmente ad un altro livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e l'ID dell'istanza da spostare tra i livelli. Per esempio, diciamo che il tuo giocatore è su un livello che è ad una profondità inferiore rispetto ad un altro livello e vuoi che appaia dietro i livelli tra i due. Puoi usare questa funzione per switch il giocatore dal livello di primo piano al livello di sfondo e farlo apparire dietro gli altri livelli disegnati.

NOTA Questa funzione non può essere usata per aggiungere una nuova istanza ad un livello. È necessario aver creato l'istanza in precedenza e aver memorizzato il suo ID in una variabile.

NOTA: Questa funzione è valida solo nell'ambito dell'attuale room e non può essere usata quando l'ambito è stato modificato usando la funzione layer_set_target_room().

Sintassi:

layer_add_instance(layer_id, instance_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the instance layer to target (or the layer name as a string)
instance_id ID istanza The unique instance ID value of the instance to move over to the target layer

Ritorna:

N/A

Esempio:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Il codice qui sopra otterrà prima l'indice dell'istanza più vicina alla posizione x/y data e lo memorizzerà in una variabile locale. Poi ottiene l'ID univoco del livello di istanza per il livello chiamato "Instances Front", e sposta l'istanza trovata su quel livello.
