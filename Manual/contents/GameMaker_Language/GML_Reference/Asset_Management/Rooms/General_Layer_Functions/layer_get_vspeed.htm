layer_get_vspeed

Potete usare questa funzione per recuperare la velocità verticale (in pixel per fotogramma di gioco) del livello all'interno dello scope corrente room. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituisce un numero reale per la velocità verticale, dove un valore positivo è giù e un valore negativo su. Il valore predefinito è 0 (a meno che non sia impostato nell'editor room ).

Sintassi:

layer_get_vspeed(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to get the vertical speed from

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Il codice di cui sopra controlla le velocità orizzontali e verticali del livello dato e se non sono entrambe impostate a 0 allora le imposta a 0.
