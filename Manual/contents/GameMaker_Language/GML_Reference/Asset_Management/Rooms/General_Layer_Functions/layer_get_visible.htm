ottieni_visibile

Con questa funzione puoi controllare se un livello è visibile o meno. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituirà true se è visibile, e false altrimenti.

Sintassi:

layer_get_visible(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Il codice qui sopra ottiene il valore dell'ID per il livello chiamato "Instances" nell'editor room, poi usa l'ID per controllare se il livello è visibile o no, alternando la visibilità del livello a seconda del valore restituito.
