stanza_reset_target

Questa funzione è usata per resettare il target del livello all'attuale room. Vedere la funzione layer_set_target_room() per ulteriori informazioni.

Sintassi:

layer_reset_target_room()

Ritorna:

Esempio:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

Il codice qui sopra imposta il target room al room "rm_Game" e poi ottiene l'ID di livello per il livello chiamato "SpriteAssets" in quel room. Questo ID di livello viene poi usato per aggiungere 50 sprite assets casuali al livello, prima che l'obiettivo del livello venga riportato al corrente room.
