imposta_livello_visibile

Con questa funzione puoi alternare la visibilità di un livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) così come il valore di attivazione per il livello, dove visible è true e invisible è false.

Quando un livello è invisibile, nulla di ciò che è assegnato al livello sarà disegnato, e se qualsiasi istanza è assegnata al livello allora non eseguirà nemmeno il suo evento di disegno (più o meno come se impostassi la variabile di istanza visible a false). Nota che qualsiasi istanza che è già contrassegnata come invisibile non sarà contrassegnata come visibile se il livello su cui si trova è impostato su visible, poiché queste sono due impostazioni indipendenti, sebbene il loro comportamento sia lo stesso.

Sintassi:

layer_set_visible(layer_id, visible)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
visible Booleano Whether the layer should be visible (true) o non (false)

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Il codice qui sopra ottiene il valore dell'ID per il livello chiamato "Instances" nell'editor room, poi usa l'ID per controllare se il livello è visibile o no, alternando la visibilità del livello a seconda del valore restituito.
