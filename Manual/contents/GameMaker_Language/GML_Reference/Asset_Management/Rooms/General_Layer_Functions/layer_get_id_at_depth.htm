layer_get_id_at_depth

Puoi usare questa funzione per ottenere gli ID di tutti i livelli assegnati a una specifica profondità. Tu dai la profondità da controllare e la funzione restituirà un array con 1 o più voci a seconda che ci siano o meno dei livelli alla profondità data. Se non ci sono livelli alla profondità data, allora l'array avrà una singola voce alla posizione [0] con un valore di -1, ma, se ci sono livelli alla profondità, allora verrà fatta una voce nell'array per ogni livello trovato - il valore della voce sarà il valore ID unico per un livello.

Sintassi:

layer_get_id_at_depth(depth)

Argomento Tipo Descrizione
depth Real The depth to check and retrieve the layer IDs from.

Ritorna:

Array

Esempio:

var a = layer_get_id_at_depth(0);
if (a[0] != -1)
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

Il codice qui sopra recupera i dati sui livelli con una profondità di 0. Viene fatto un controllo per vedere se esistono livelli a quella profondità, e se ci sono allora l'array restituito viene analizzato e ciascuno dei livelli trovati viene distrutto.
