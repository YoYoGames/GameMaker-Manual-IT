installazione_strato_get_instance

Questa funzione può essere usata per ottenere l'ID di istanza unico di un dato elemento di istanza. Voi date l'ID dell'elemento di istanza (vedete l'esempio di codice qui sotto per come ottenerlo), e la funzione restituirà un valore reale che rappresenta l'ID di istanza unico per l'elemento. Se l'elemento non è un'istanza, la funzione restituirà -1.

Sintassi:

layer_instance_get_instance(element_id)

Argomento Tipo Descrizione
element_id ID dell'elemento dell'istanza The unique ID value of the instance element to get the ID of

Ritorna:

Instance ID

Esempio:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(_elements[i]) == layerelementtype_instance)
     {
         var _element = _elements[i];
         var _inst = layer_instance_get_instance(_element);
         _inst.x = _inst.x + 10;
     }
}

Il codice qui sopra controllerà tutti gli elementi di istanza su un livello, poi otterrà il loro valore ID unico e lo userà per spostarli di 10px a destra.
