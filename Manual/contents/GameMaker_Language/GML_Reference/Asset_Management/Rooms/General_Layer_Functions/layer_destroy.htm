distruggi_strato

Questa funzione distruggerà il dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e questo lo rimuoverà dal corrente room. Se il livello è stato progettato nell'editor room, allora la prossima volta che si lascia room e poi si ritorna, il livello sarà ricreato di nuovo con i contenuti originali, tuttavia se room è persistente, il livello sarà rimosso a meno che room persistenza sia disattivata di nuovo. Quando si distrugge un livello in questo modo, anche tutti i suoi contenuti saranno rimossi, quindi qualsiasi ID di riferimento per sfondi o mappe tile, ecc... non sarà più valido e qualsiasi istanza assegnata al livello sarà distrutta (eseguendo il loro evento Destroy allo stesso tempo, se ne hanno uno, così come l'evento Clean Up).

Sintassi:

layer_destroy(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to destroy (or the layer name as a string)

Ritorna:

N/A

Esempio:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Il codice qui sopra controllerà se esiste qualche istanza di object "obj_Bullet_Parent", e se non esiste distruggerà il livello con l'ID memorizzato nella variabile globale.
