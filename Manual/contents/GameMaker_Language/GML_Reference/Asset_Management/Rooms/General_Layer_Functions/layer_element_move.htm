Spostamento di un elemento

Puoi usare questa funzione per spostare un elemento da un livello ad un altro. Si dà l'ID dell'elemento, come restituito dalla funzione usata per creare l'elemento o dall'editor room o dalla funzione layer_get_all_elements()e poi dai l'ID del livello (che ottieni quando crei il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni).

Sintassi:

layer_element_move(element_id, layer_id)

Argomento Tipo Descrizione
element_id ID dell'elemento di strato The unique ID value of the element to move
layer_id Stringa o ID del livello The unique ID value of the layer to move the element to (or the layer name as a string)

Ritorna:

N/A

Esempio:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

Il codice di cui sopra ottiene gli elementi sul livello a cui è assegnata l'istanza che esegue il codice, poi li controlla per vedere se sono sprite assets, e se lo sono allora vengono spostati sul livello con l'ID memorizzato nella variabile "asset_layer".
