stanza_get_target

Questa funzione restituisce l'attuale room che viene preso di mira dalle funzioni di livello. Vedere layer_set_target_room() per maggiori informazioni.

Sintassi:

layer_get_target_room()

Ritorna:

Room Asset

Esempio:

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

Il codice di cui sopra controlla il target corrente room e se non è il target corrente room allora il target di livello room viene resettato.

Indietro: Funzioni generali del livello
Il prossimo: layer_reset_target_room
