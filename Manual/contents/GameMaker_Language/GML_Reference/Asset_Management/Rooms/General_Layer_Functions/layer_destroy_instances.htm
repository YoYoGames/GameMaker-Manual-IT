distruggere le istanze

Questa funzione può essere usata per distruggere tutte le istanze assegnate al dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni), e poi tutte le istanze che sono sul livello saranno rimosse dal gioco, innescando i loro eventi Destroy e Clean Up.

Sintassi:

layer_destroy_instances(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer (or the layer name as a string)

Ritorna:

N/A

Esempio:

if global.game_over
{
    layer_destroy_instances(layer);
}

Il codice di cui sopra controllerà una variabile globale e se è true allora tutte le istanze che sono sul livello dell'istanza chiamante saranno distrutte (inclusa l'istanza chiamante).
