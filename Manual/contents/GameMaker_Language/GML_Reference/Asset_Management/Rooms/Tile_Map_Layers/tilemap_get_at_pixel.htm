tilemap_get_at_pixel

Usando questa funzione puoi recuperare i dati di tile da una posizione (all'interno di room) dell'elemento di mappa tile. Date l'ID dell'elemento di mappa tile (che ottenete quando create un elemento di mappa tile usando layer_tilemap_create() o quando usate la funzione layer_tilemap_get_id()) così come la posizione x e y nella stanza da cui ottenere i dati tile e la funzione restituirà il "blob" di dati tile. Questi dati sono essenzialmente una maschera di bit che contiene l'indice tile, i booleani flip/rotate/mirror e qualsiasi maschera dati che sia stata applicata (vedi tilemap_set_mask() per i dettagli), e il valore dei dati risultante può essere usato nelle funzioni tile_* per cambiare le proprietà di tiles. Se avete bisogno di ottenere i dati di tile da una cella specifica tile dovreste usare la funzione tilemap_get() invece.

IMPORTANTE! Se i tiles nella mappa tile sono rimasti invariati (cioè: non sono ruotati o capovolti ecc...), allora il valore di ritorno dei dati tile set sarà esattamente uguale all'indice del tile sul tile set. Quindi puoi creare "mappe di collisione" di tiles usando un tile all'indice 1 nel tile set - per esempio - poi usare questa funzione per controllare 1 o 0 (un tile vuoto) per calcolare le collisioni.

Sintassi:

tilemap_get_at_pixel(tilemap_element_id, x, y);

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to get the tile-data of
x Real The position along the x-axis to get the tile data from (in room pixels)
y Real The position along the y-axis to get the tile data from (in room pixels)

Ritorna:

Tile Data (-1 if there is an error)

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

Il codice qui sopra ottiene l'ID per la mappa tile "Clouds" sul livello "Tiles_Sky" e poi lo usa per ottenere i dati da tile nella posizione (64, 128). Questi dati di tile vengono poi capovolti prima di essere usati per impostare nuovamente tile sulla mappa tile.
