esiste una mappa a strati

Puoi usare questa funzione per controllare e vedere se un elemento della mappa tile esiste su un dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e l'ID dell'elemento mappa tile (che si ottiene quando si crea un elemento mappa tile usando layer_tilemap_create() o quando si usa la funzione layer_tilemap_get_id()) e la funzione restituirà o true se l'elemento esiste, o false se non esiste.

NOTA: questa funzione lavora nell'ambito del target corrente room - di default il room in cui la funzione è chiamata - che può essere impostato usando la funzione layer_set_target_room().

Sintassi:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to check

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("tilemap_trees");
if (layer_tilemap_exists(lay_id, global.Treestilemap))
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

Il codice qui sopra controlla il livello "tilemap_trees" per vedere se l'elemento di mappa dato tile esiste e se esiste, allora viene distrutto (ma non il livello).
