tilemap_clear

Usando questa funzione potete cancellare/impostare tutti i tiles su una data tile-map. Si dà l'ID dell'elemento della mappa tile (che si ottiene quando si crea un elemento della mappa tile usando layer_tilemap_create() o quando si usa la funzione layer_tilemap_get_id()), e poi fornisci i dati di tile con cui vuoi cancellare il livello. Un valore predefinito di 0 cancellerà tutti i tiles dal livello (essenzialmente rendendo tutti i tiles "vuoti"), mentre si possono usare le funzioni dedicate tile_* per creare i propri dati tile con cui cancellare la mappa tile.

Sintassi:

tilemap_clear(tilemap_element_id, tiledata)

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to change
tiledata Dati delle mattonelle (reali) The tile data to use to clear the layer

Ritorna:

N/A

Esempio:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

Il codice qui sopra ottiene il valore ID di una mappa tile creata nell'editor room, e poi la cancella usando "empty" tiles.
