tile_get_rotate

Questa funzione può essere usata per controllare se in un dato insieme di tile-data il sito tile è stato ruotato di 90 gradi o no. Si danno i tile-data, che possono essere recuperati usando la funzione tilemap_get()e la funzione restituirà true se tile è ruotato, o false in caso contrario.

Sintassi:

tile_get_rotate(tiledata)

Argomento Tipo Descrizione
tiledata Dati sulle piastrelle the tile-data to check

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Il codice qui sopra ottiene l'ID della mappa tile dal livello dato e poi ottiene la posizione della cella x e y per il tile sotto il mouse. Questa posizione viene poi usata per ottenere il tile-data che viene ruotato e poi usato per impostare nuovamente tile.
