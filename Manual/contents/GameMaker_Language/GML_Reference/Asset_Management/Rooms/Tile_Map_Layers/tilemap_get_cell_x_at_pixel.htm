tilemap_get_cell_x_at_pixel

Usando questa funzione potete recuperare la posizione dell'asse x di una singola cella della mappa tile dando la posizione relativa dell'asse x all'interno di room. Date l'ID dell'elemento della mappa tile (che ottenete quando create un elemento della mappa tile usando layer_tilemap_create() o quando si usa la funzione layer_tilemap_get_id()), così come la posizione x e y all'interno della stanza e la funzione restituirà la posizione x della cella all'interno della mappa tile per quel punto. Notate che se il valore è al di fuori dell'area della mappa tile, e nessuna cella è disponibile, restituirà -1.

Sintassi:

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

ArgomentoTipo Descrizione
tilemap_element_id The unique ID value of the tile map element to get the cell x position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Ritorna:

(x-axis cell position or -1 if there is an error)

Esempio:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Il codice qui sopra usa l'ID della mappa recuperato tile per ottenere la posizione della cella da un insieme di coordinate room e poi memorizzare i dati per ogni tile trovato lì in una variabile di istanza.
