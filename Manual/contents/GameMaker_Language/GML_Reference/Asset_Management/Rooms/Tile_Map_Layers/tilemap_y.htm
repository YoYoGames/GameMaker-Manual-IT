tilemap_y

Questa funzione controlla la posizione lungo l'asse y del room dell'elemento di mappa asset tile sul livello. Si dà l'ID dell'elemento di mappa tile (che si ottiene quando si crea un elemento di mappa tile usando layer_tilemap_create() o quando usi la funzione layer_tilemap_get_id()), e poi imposti il valore y da usare (basato sulle coordinate di room ).

Sintassi:

tilemap_y(tilemap_element_id, y);

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to change
y Real The y position for the tile map

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Il codice qui sopra ottiene il valore ID della mappa tile asset assegnata al livello "Asset_sky" e poi imposta la sua posizione y a un valore casuale tra 0 e l'altezza di room.
