Distruggi mappa di livello

Questa funzione distruggerà il dato elemento della mappa tile. Si fornisce l'ID della mappa tile (che si ottiene quando si crea la mappa tile usando layer_tilemap_create() o quando si usa l'ID del livello insieme a layer_get_tilemap_id()) e questo lo rimuoverà. Nota che questo non rimuove il livello, solo la mappa tile da esso, e se la mappa tile è una che è stata aggiunta nell'editor room, allora la prossima volta che lasci il room e poi ritorni, la mappa tile sarà ricreata di nuovo. Tuttavia, se room è persistente, la mappa tile sarà rimossa a meno che la persistenza di room sia nuovamente disattivata.

Sintassi:

layer_tilemap_destroy(tilemap_element_id)

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map to be destroyed

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

Il codice qui sopra controlla il livello "Tiles_trees" per vedere se l'elemento dato della mappa tile esiste e se esiste, allora viene distrutto (ma non il livello).
