mappa_strato_get_id

Questa funzione può essere usata per recuperare il valore ID unico dell'elemento della mappa tile su un livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e la funzione restituirà il valore ID associato all'elemento della mappa tile sul livello. Si noti che questa funzione è specificamente progettata per l'uso con le mappe tile che sono state aggiunte in IDE, poiché se si aggiunge una mappa tile ad un livello attraverso il codice utilizzando la funzione layer_tilemap_create(), allora restituirà l'ID unico per l'elemento di mappa tile aggiunto. Se l'ID della mappa dato non è corretto o la mappa non esiste, la funzione restituirà -1.

Sintassi:

layer_tilemap_get_id(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)

Ritorna:

Tile Map Element ID or -1

Esempio:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Il codice qui sopra otterrà l'ID del livello chiamato "tilemap_trees" e poi lo userà per ottenere l'ID dell'elemento della mappa tile su quel livello. Questo ID viene poi usato per rimuovere la mappa tile dal livello.

Indietro: Livelli di mappa a mattonelle
