tilemap_get_frame

Poiché tiles può essere animato, a volte può essere utile sapere quale fotogramma è attualmente disegnato e reagire di conseguenza, quindi con questa funzione potete recuperare l'indice del fotogramma corrente per una data mappa tile. Date l'ID dell'elemento della mappa tile (che ottenete quando create un elemento della mappa tile usando layer_tilemap_create() o quando usate la funzione layer_tilemap_get_id()), e la funzione restituirà l'indice del frame.

Sintassi:

tilemap_get_frame(tilemap_element_id)

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to get the frame index of

Ritorna:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Esempio:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4)
{
    global.activate = true;
}
else
{
    global.activate = false;
}

Il codice qui sopra controlla il fotogramma di animazione corrente per la data mappa tile sul dato livello, e imposta una variabile globale basata sul valore di ritorno.
