Livelli di mappa a mattonelle

L'editor di stanze di GameMaker permette di aggiungere tiles in qualsiasi dato room usando i livelli. Come implica il nome, un livello di mappa tile è semplicemente una collezione di tile set tiles che sono tutti considerati alla stessa profondità all'interno di room, e impilando i livelli è possibile far disegnare alcune cose sopra o sotto altre. Puoi anche controllare certi aspetti dei livelli dal codice, aggiungendo o rimuovendo cose, o cambiando certe proprietà del livello o di ciò che contiene in fase di esecuzione. Si noti che nell'editor di room si è limitati ad una mappa tile per livello (il simbolo Layer di piastrelle), ma quando si lavora con loro nel codice, si possono avere più mappe tile assegnate ad un singolo livello, con ogni mappa che ha il suo unico ID e proprietà, così:

  • Il set di piastrelle è la risorsa che si usa per creare i livelli tile e le mappe tile
  • Il livello delle piastrelle è quello che si crea nell'Editor Room per aggiungere tile sets a
  • La mappa di piastrelle è ciò che si chiama l'insieme di tiles che sono aggiunti a un livello, sia nell'editor room o attraverso il codice, come un singolo elemento su quel livello

Oltre alle funzioni elencate sopra per impostare le mappe di tile, avete anche un certo numero di funzioni che possono essere usate per manipolare i dati dei singoli tiles. Un "blob" di dati tile è composto dall'indice tile più alcuni bit extra per mostrare se il tile è stato ruotato, capovolto o specchiato. Avete anche la possibilità di aggiungere la vostra maschera di bit ai dati di tile, anche se questo è solo per utenti avanzati, poiché dovrete mascherare dei bit che sono riservati all'indice tile - essenzialmente ci sono 19 bit riservati ai vostri indici tile, ma se ne usate solo 8, allora vi restano undici bit che possono essere usati per creare una maschera personalizzata. Questa maschera può poi essere usata e controllata nel codice per creare collisioni tile personalizzate o altro. Tuttavia, normalmente non avreste bisogno di creare i vostri blob di dati tile, e usereste invece la funzione tilemap_get() per ottenere i dati di tile, poi li manipolereste usando le funzioni sottostanti, e poi li impostereste di nuovo usando tilemap_set().
