tile_get_empty

Questa funzione può essere usata per controllare se un dato insieme di tile-data è per un tile vuoto o no. Si danno i tile-data, che possono essere recuperati utilizzando la funzione tilemap_get()e la funzione restituirà true se il tile è vuoto, o false se c'è un indice tile.

Sintassi:

tile_get_empty(tiledata)

Argomento Tipo Descrizione
tiledata Dati sulle piastrelle the tile-data to check

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++;)
{
    for (var j = 0; j < tilemap_get_height(map_id); j++;)
    {
        var data = tilemap_get(map_id, i, j);
        if (!tile_get_empty(data))
        {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
        }
    }
}

Il codice qui sopra ottiene l'ID della mappa tile dal livello dato e poi procede a controllare ogni cella di tile sulla mappa per vedere se ha dati o no. Se ce l'ha, il tile è impostato come vuoto.
