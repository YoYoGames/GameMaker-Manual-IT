tile_set_rotate

Questa funzione può essere usata per impostare un dato set di tile-data per ruotare il tile di 90 gradi o meno. Voi date i tile-data, che possono essere recuperati usando la funzione tilemap_get()e poi impostate l'argomento rotate su true se volete che tile sia ruotato, o su false se volete che tile sia nel suo stato predefinito, non ruotato. La funzione restituirà un set di dati tile modificato che può essere applicato utilizzando la funzione tilemap_set() funzione.

Sintassi:

tile_set_rotate(tiledata, rotate)

Argomento Tipo Descrizione
tiledata Dati sulle piastrelle The tile-data to set
rotate Booleano Set to true per ruotare e false per lasciarlo così com'è

Ritorna:

Real

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Il codice qui sopra ottiene l'ID della mappa tile dal livello dato e poi ottiene la posizione della cella x e y per il tile sotto il mouse. Questa posizione viene poi usata per ottenere il tile-data che viene ruotato e poi usato per impostare nuovamente tile.
