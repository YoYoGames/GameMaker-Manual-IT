tilemap_x

Questa funzione controlla la posizione lungo l'asse x del room dell'elemento di mappa asset tile sul livello. Si dà l'ID dell'elemento di mappa tile (che si ottiene quando si crea un elemento di mappa tile usando layer_tilemap_create() o quando usi la funzione layer_tilemap_get_id()), e poi imposti il valore x da usare (basato sulle coordinate di room ).

Sintassi:

tilemap_x(tilemap_element_id, x);

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to change
x Real The x position for the tile map

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

Il codice qui sopra ottiene il valore ID della mappa tile asset assegnato al livello "Asset_sky" e poi imposta la sua posizione x a un valore casuale tra 0 e la larghezza del room.
