tilemap_get_global_mask

Questa funzione può essere usata per recuperare il valore della maschera di bit per tutte le mappe tile, restituendo il valore corrente della maschera o -1 se c'è un errore o 0 se non è specificata alcuna maschera. Per ulteriori informazioni sulle maschere di bit della mappa globale tile, vedere la funzione tilemap_set_global_mask().

Sintassi:

tilemap_get_global_mask();

Ritorna:

(0 for no mask, -1 for an error, mask value otherwise)

Esempio:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_global_mask(new_mask);
}

Il codice di cui sopra ottiene il valore globale della maschera associata a tutte le mappe tile. Se non è uguale al valore definito nella variabile "new_mask", allora viene impostato a quel valore.
