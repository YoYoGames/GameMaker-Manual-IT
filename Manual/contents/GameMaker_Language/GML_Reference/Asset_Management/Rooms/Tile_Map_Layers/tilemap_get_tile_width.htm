tilemap_get_tile_width

Usando questa funzione puoi recuperare la larghezza (in pixel) di ogni cella di tile dell'elemento mappa tile. Si dà l'ID dell'elemento mappa tile (che si ottiene quando si crea un elemento mappa tile usando layer_tilemap_create() o quando usate la funzione layer_tilemap_get_id()), e la funzione restituirà la larghezza della cella di tile set.

Sintassi:

tilemap_get_tile_width(tilemap_element_id);

ArgomentoTipo Descrizione
tilemap_element_id The unique ID value of the tile map element to get the tile cell width of

Ritorna:

Esempio:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Il codice qui sopra usa l'ID della mappa recuperata tile per ottenere la larghezza e l'altezza della cella tile e usarle per impostare due variabili globali.
