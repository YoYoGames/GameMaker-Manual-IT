tile_get_index

Questa funzione può essere usata per ottenere l'indice tile (la posizione di tile all'interno dell'immagine tile set ) da un insieme di tile-data. Si specificano i tile-data, che possono essere recuperati usando la funzione tilemap_get()e la funzione restituisce un valore intero per l'indice.

Quando si usa questa funzione, assicurarsi che il tile-data dato sia valido, altrimenti anche il valore dell'indice restituito non sarà valido, il che potrebbe causare un comportamento inaspettato.

Sintassi:

tile_get_index(tiledata)

Argomento Tipo Descrizione
tiledata Dati sulle piastrelle the tile-data to check

Ritorna:

Real

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Il codice di cui sopra ottiene l'ID della mappa tile dal dato livello e poi lo usa per ottenere i dati delle piastrelle per la cella sotto la posizione del mouse. Questi dati vengono poi utilizzati per impostare l'indice tile su un numero casuale.
