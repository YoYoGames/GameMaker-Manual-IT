tilemap_tileset

Con questa funzione è possibile modificare il tile set asset assegnato a un dato elemento di mappa tile su un livello. Si dà l'ID dell'elemento di mappa tile (che si ottiene quando si crea un elemento di mappa tile usando layer_tilemap_create() o quando si usa la funzione layer_tilemap_get_id()), quindi si fornisce un indice tile set e alla mappa tile verrà assegnato il nuovo sprite.

Sintassi:

tilemap_tileset(tilemap_element_id, tileset_index)

Argomento Tipo Descrizione
tilemap_element_id ID elemento mappa piastrelle The unique ID value of the tile map element to change
tileset_index Set di piastrelle Asset The new tile set index to use

Restituzione:

N/A

Esempio:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

Il codice precedente controlla l'attuale tile set assegnato alla mappa tile sul layer "Tiles_trees" e se non è "ts_Nighttime" allora quel tile set viene assegnato alla mappa tile.
