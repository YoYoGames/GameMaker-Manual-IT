creare una mappa a strati

Con questa funzione puoi assegnare una risorsa tile-set ad un livello da usare come mappa tile nel tuo progetto. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e poi una posizione iniziale (x, y) per aggiungere la mappa tile al room, la risorsa tile set da usare, e poi la larghezza e l'altezza della mappa tile in celle (cioè: una larghezza di 20 e un'altezza di 10 creerà una mappa tile con 200 celle che è larga 20 celle e alta 10 celle), con la dimensione delle celle stesse definita dal tile set scelto. Vale la pena notare che non è possibile posizionare tiles in posizioni negative all'interno della mappa tile, quindi tutti i tiles devono essere posizionati all'interno degli spazi delle celle da 0 a larghezza - 1, da 0 a altezza - 1.

Sintassi:

layer_tilemap_create(layer_id, x, y, tileset, width, height)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
x Real The x position of the tile map in the room
y Real The y position of the tile map in the room
tileset Tile Set Asset The Tile Set asset to be used
width Real The width tile map (in cells)
height Real The height of the tile map (in cells)

Ritorna:

Tile Map Element ID

Esempio:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Il codice qui sopra crea un nuovo livello e poi vi aggiunge un nuovo elemento di mappa tile, impostando la posizione della mappa tile a (0,0) così come il tile set da usare e la larghezza e l'altezza della mappa tile.

Indietro: Livelli di mappa a mattonelle
Il prossimo: layer_tilemap_destroy
