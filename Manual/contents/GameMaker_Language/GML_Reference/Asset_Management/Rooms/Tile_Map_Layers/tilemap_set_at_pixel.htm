tilemap_set_at_pixel

Questa funzione può essere usata per impostare una cella all'interno dell'elemento di mappa tile sul livello su un nuovo tile usando la posizione attuale del tile per cambiare all'interno del room. Si dà l'ID dell'elemento di mappa tile (che si ottiene quando si crea un elemento di mappa tile usando layer_tilemap_create() o quando si usa la funzione layer_tilemap_get_id()), e poi i dati di tile da impostare così come la posizione all'interno di room. Potete recuperare i dati di tile usando la funzione tilemap_get_at_pixel() e poi usare le funzioni tile_ per cambiarli prima di impostarli di nuovo usando questa funzione. La funzione restituirà true se tile è stato impostato con successo e false se c'è stato un problema e non è stato impostato.

Sintassi:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Argomento Tipo Descrizione
tilemap_element_id ID dell'elemento della mappa delle mattonelle The unique ID value of the tile map element to change
tiledata Dati delle mattonelle (reali) The tile set data to set
xcell Real The x position (in the room)
ycell Real The y position (in the room)

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

Il codice qui sopra ottiene l'ID per la mappa tile sul livello "Tiles_Sky" e poi lo usa per ottenere i dati da tile nella posizione (4, 4). Questi dati di tile sono poi capovolti prima di essere usati per impostare nuovamente tile sulla mappa tile.
