indice_di_sfondo_get_index

Questa funzione può essere usata per ottenere il valore corrente dell'indice dell'immagine dell'elemento di sfondo. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e la funzione restituirà un valore reale che rappresenta l'indice dell'immagine mostrata per sprite. La funzione restituirà -1 se l'elemento di sfondo non esiste o se l'elemento non ha un valido sprite assegnato ad esso.

Sintassi:

layer_background_get_index(background_element_id);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to get the information from

Ritorna:

Real (the current sprite image index or -1)

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_index(back_id) < 4)
{
    layer_background_index(back_id, 4);
}

Il codice qui sopra otterrà l'ID del livello chiamato "Background_sky" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per controllare se l'indice dell'immagine per l'elemento è inferiore a 4, e se è così viene impostato a 4.
