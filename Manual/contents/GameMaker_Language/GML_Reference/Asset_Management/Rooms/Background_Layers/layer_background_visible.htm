strato_sfondo_visibile

Usando questa funzione potete alterare la visibilità di un elemento di sfondo. Si fornisce l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi impostare l'argomento visible su true o false. Quando impostato su true l'elemento sarà visualizzato, e quando impostato su false, l'elemento non sarà visualizzato. Nota che questo dipende dalla visibilità del livello, e anche se l'elemento di sfondo è contrassegnato come visibile, non sarà disegnato se il livello su cui si trova è contrassegnato come non visibile.

Sintassi:

layer_background_visible(background_element_id, visible)

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
visible Booleano The visibility toggle, which can be true o false

Ritorna:

N/A

Esempio:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_visible(back))
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

Il codice di cui sopra controlla la pressione di un pulsante del mouse e, se ne viene rilevato uno, altera la visibilità dello sfondo dell'elemento assegnato al livello su cui si trova l'istanza che esegue il codice.
