distruggi_sfondo

Questa funzione distruggerà l'elemento di sfondo dato. Si fornisce l'ID dello sfondo (che si ottiene quando si crea lo sfondo usando layer_background_create() o quando usi l'ID del livello insieme a layer_get_background_id()) e questo lo rimuoverà. Nota che questo non rimuove il livello, solo lo sfondo da esso, e se lo sfondo è uno che è stato aggiunto nell'editor room, allora la prossima volta che lascerai room e tornerai, lo sfondo sarà ricreato di nuovo. Comunque se room è persistente, lo sfondo sarà rimosso a meno che room persistenza sia disattivata di nuovo.

Sintassi:

layer_background_destroy(background_element_id)

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to be destroyed

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if (layer_background_exists(lay_id, bck_id))
{
    layer_background_destroy(bck_id);
}

Il codice qui sopra controlla il livello "Background_trees" per vedere se l'elemento di sfondo dato esiste e se esiste, allora viene distrutto (ma non il livello).
