livello_sfondo_velocità

Questa funzione può essere usata per impostare la velocità di un sprite animato che è stato assegnato a un elemento di sfondo. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi si imposta la velocità di animazione. Se IDE ha impostato la velocità di animazione per il sprite usato per essere calcolato come Frames Per Game Frame, allora il valore di velocità dato è un moltiplicatore, con 1 che è il valore predefinito, e impostandolo a 0,5 si dimezza la velocità di animazione, mentre impostandolo a 2 la si raddoppia. Tuttavia, se IDE ha impostato la velocità dell'animazione per il sprite usato per essere calcolato come Frames Per Second, allora il valore di velocità dato è un valore letterale di frames per secondo, quindi impostandolo a 5, per esempio, imposterà l'animazione a loop 5 frames ogni secondo che il gioco gira. Se l'elemento di sfondo non ha sprite collegato o il sprite usato non ha sotto-immagini, questa funzione non avrà alcun effetto.

Sintassi:

layer_background_speed(background_element_id, speed);

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
speed Real The speed multiplier/value to use for the element

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

Il codice qui sopra otterrà l'ID del livello chiamato "Background_water" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per cambiare la velocità dell'immagine dell'elemento.
