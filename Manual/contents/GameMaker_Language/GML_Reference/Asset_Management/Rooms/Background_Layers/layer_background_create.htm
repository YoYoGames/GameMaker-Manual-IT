creazione di un livello di sfondo

Con questa funzione puoi assegnare una risorsa sprite ad un livello da usare come sfondo nel tuo progetto. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e un indice sprite (che sarebbe il nome del sprite come mostrato nel Browser Asset ), e sarà aggiunto al livello. La funzione restituisce il valore ID unico per l'elemento, che può poi essere usato in ulteriori funzioni di livello per gli sfondi.

Sintassi:

layer_background_create(layer_id, sprite)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
sprite Sprite Asset The sprite index to be used

Ritorna:

Background Element ID

Esempio:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Il codice qui sopra crea un nuovo livello e poi aggiunge un nuovo elemento di sfondo ad esso, impostando un sprite per essere l'immagine di sfondo utilizzata.
