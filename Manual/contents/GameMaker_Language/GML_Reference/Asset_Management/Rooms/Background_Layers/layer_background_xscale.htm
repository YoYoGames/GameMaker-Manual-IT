scala_di_sfondo_x

Questa funzione può essere usata per impostare la scala lungo l'asse x di un elemento di sfondo. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi imposti il valore di scala. Il valore di scala che dai è un moltiplicatore che verrà usato per cambiare il modo in cui l'elemento di sfondo viene visualizzato, dove un valore di 0,5 visualizzerebbe l'elemento a metà scala, e un valore di 2 lo visualizzerebbe a scala doppia. Nota che i valori negativi sono validi, e "capovolgeranno" l'elemento intorno alla posizione (0,0), quindi una scala x di -1 mostrerà l immagine invertita.

Sintassi:

layer_background_xscale(background_element_id, xscale);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

Ritorna:

Esempio:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

Il codice qui sopra otterrà l'ID del livello chiamato "Background_water" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per cambiare la scala x dell'elemento.

Indietro: Livelli di sfondo
