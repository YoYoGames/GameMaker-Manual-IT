strato_sfondo_esistente

Puoi usare questa funzione per controllare e vedere se un elemento di sfondo esiste su un dato livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create()) o il nome del livello (come string - questo avrà un impatto sulle prestazioni) e l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()) e la funzione restituirà o true se l'elemento esiste, o false se non esiste.

NOTA: questa funzione lavora nell'ambito del target corrente room - di default il room in cui la funzione è chiamata - che può essere impostato usando la funzione layer_set_target_room().

Sintassi:

layer_background_exists(layer_id, background_element_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target (or the layer name as a string)
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to check

Ritorna:

Boolean

Esempio:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

Il codice qui sopra controlla il livello "Background_trees" per vedere se l'elemento di sfondo dato esiste e se esiste, allora viene distrutto (ma non il livello).
