miscela_di_sfondo

Questa funzione controlla la miscelazione (o "tinta") dello sfondo sprite e il valore predefinito è -1 (che rappresenta la costante c_white, che può anche essere utilizzata). Qualsiasi altro valore (comprese le costanti di colore interne come c_red, o c_aqua) fonderà il colore specificato con l'originale sprite. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando si usa la funzione layer_background_get_id()), e poi imposti il colore di fusione da usare. Qui sotto potete vedere un esempio di un sprite che è stato miscelato con diversi colori:

Si prega di notare che si dovrebbe cercare di limitare il blending sulla piattaforma HTML5 (a meno che non si usi WebGL), poiché ogni mescolato sprite deve essere memorizzato nella cache separatamente e quindi avere molti mescolati sprites può influire negativamente sulle prestazioni (si può anche impostare la dimensione di cache usando la funzione sprite_set_cache_size()).

Sintassi:

layer_background_blend(background_element_id, blend);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

Ritorna:

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

Il codice qui sopra ottiene il valore ID dello sfondo assegnato al livello "Background_sky" e poi lo tinge di un colore.
