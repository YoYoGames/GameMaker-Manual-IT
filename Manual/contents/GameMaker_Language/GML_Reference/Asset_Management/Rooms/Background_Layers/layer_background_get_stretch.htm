layer_background_get_stretch

Questa funzione può essere usata per ottenere lo stato allungato dell'elemento di sfondo sprite. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando si usa la funzione layer_background_get_id()), e la funzione restituirà o true se l'elemento sprite è attualmente stirato per adattarsi al room, o false se non lo è.

Sintassi:

layer_background_get_stretch(background_element_id);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to get the information from

Ritorna:

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
    layer_background_stretch(back_id, false);
}

Il codice qui sopra otterrà l'ID del livello chiamato "Background_sky" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID è poi usato per controllare e vedere se l'elemento sprite sarà allungato per adattarsi a room o no e se è allungato, allora questa proprietà è impostata su false.
