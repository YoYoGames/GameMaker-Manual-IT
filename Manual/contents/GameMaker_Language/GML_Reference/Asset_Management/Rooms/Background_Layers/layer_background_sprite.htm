sfondo_strato_sprite

Con questa funzione potete impostare l'indice sprite dell'elemento di sfondo. Si fornisce l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi dai un indice sprite da usare. L'immagine dell'elemento di sfondo verrà sostituita con il nuovo sprite. Se date un valore di -1, all'elemento non verrà assegnato alcun sprite (ma esisterà ancora e potrà avere un sprite assegnato nuovamente in seguito).

Questa funzione ha anche l'alias layer_background_change().

Sintassi:

layer_background_sprite(background_element_id, sprite_index)

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
sprite_index Sprite Asset The sprite index of the sprite to use for the background element

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

Il codice qui sopra otterrà l'ID del livello chiamato "Background_sky" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per controllare il sprite assegnato all'elemento, impostandolo su sprite "spr_Clouds" se non lo è già.
