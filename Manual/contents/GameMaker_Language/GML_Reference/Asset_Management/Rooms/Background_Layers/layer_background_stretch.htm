sfondo_stretch

Usando questa funzione potete alternare un elemento di sfondo sprite per allungarlo per adattarlo a room o rimanere a 1:1 con la risoluzione. Si fornisce l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi si imposta l'argomento stretch su true o false. Quando è impostato su true l'elemento sprite sarà allungato per adattarsi a room (sia in scala superiore che inferiore a seconda delle dimensioni di sprite e room), e quando è impostato su false, l'elemento sprite sarà visualizzato alla sua risoluzione nativa.

Sintassi:

layer_background_stretch(background_element_id, stretch)

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
stretch Booleano The stretch toggle, which can be true o false

Ritorna:

N/A

Esempio:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

Il codice di cui sopra controlla la pressione di un pulsante del mouse e, se viene rilevato, altera l'allungamento dell'elemento di sfondo sprite assegnato al livello su cui si trova l'istanza che esegue il codice.
