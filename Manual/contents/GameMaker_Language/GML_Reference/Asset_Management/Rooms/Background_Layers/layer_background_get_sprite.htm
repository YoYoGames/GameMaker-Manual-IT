layer_background_get_sprite

Questa funzione può essere usata per ottenere il valore corrente dell'indice sprite dell'elemento di sfondo. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usate la funzione layer_background_get_id()), e la funzione restituirà un valore reale che rappresenta l'indice sprite mostrato. Se l'elemento non ha sprite assegnato, la funzione restituirà -1.

Sintassi:

layer_background_get_sprite(background_element_id);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to get the information from

Ritorna:

Sprite Asset or -1

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

Il codice qui sopra otterrà l'ID del livello chiamato "Background_sky" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per controllare il sprite assegnato all'elemento, impostandolo su sprite "spr_Clouds" se non lo è già.
