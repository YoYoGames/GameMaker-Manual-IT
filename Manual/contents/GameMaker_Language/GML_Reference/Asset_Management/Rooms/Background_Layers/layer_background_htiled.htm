strato_sfondo_htiled

Usando questa funzione puoi cambiare se l'elemento di sfondo dato su un livello deve essere piastrellato orizzontalmente o no. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi imposti il valore di tiling. Se impostato su true allora l'elemento sarà visualizzato affiancato orizzontalmente attraverso room, e quando impostato su false, l'elemento non sarà affiancato. Questa funzione è solo per progetti 2D, e non funzionerà correttamente quando si usa una proiezione con telecamera 3D.

Sintassi:

layer_background_htiled(background_element_id, tiled);

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
tiled Booleano The horizontal tiling toggle, which can be true o false

Ritorna:

N/A

Esempio:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

Il codice di cui sopra controllerà l'elemento di sfondo assegnato al livello su cui si trova l'istanza che esegue il codice e se non è piastrellato in entrambe le direzioni lo imposterà su tile in verticale e in orizzontale.
