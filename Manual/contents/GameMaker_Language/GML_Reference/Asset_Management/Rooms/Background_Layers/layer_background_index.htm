indice di sfondo del livello

Questa funzione può essere usata per impostare l'indice dell'immagine dello sfondo sprite che ha più sotto-immagini. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e poi imposti l'indice dell'immagine da utilizzare. Se imposti un valore al di fuori della gamma di sotto-immagini, allora l'indice dell'immagine sarà loop circa.

Sintassi:

layer_background_index(background_element_id, image_index);

Argomento Tipo Descrizione
background_element_id ID dell'elemento di sfondo The unique ID value of the background element to change
index Real The image index to use for the background

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Il codice qui sopra otterrà l'ID del livello chiamato "Background_trees" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per cambiare l'indice dell'immagine dell'elemento.
