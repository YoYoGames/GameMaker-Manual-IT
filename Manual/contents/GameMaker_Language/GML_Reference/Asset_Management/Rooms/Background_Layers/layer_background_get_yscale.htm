layer_background_get_yscale

Questa funzione può essere usata per ottenere il valore corrente del moltiplicatore di scala dell'elemento di sfondo. Si dà l'ID dell'elemento di sfondo (che si ottiene quando si crea un elemento di sfondo usando layer_background_create() o quando usi la funzione layer_background_get_id()), e la funzione restituirà un valore reale che rappresenta il moltiplicatore di scala utilizzato per disegnare il sprite. Il valore predefinito è 1.

Sintassi:

layer_background_get_yscale(background_element_id);

ArgomentoTipo Descrizione
background_element_id The unique ID value of the background element to get the information from

Ritorna:

Esempio:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_yscale(back_id) != -1
{
    layer_background_yscale(back_id, -1);
}

Il codice qui sopra otterrà l'ID del livello chiamato "Background_sky" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per controllare la scala y per l'elemento e se non è -1, viene impostato a -1.
