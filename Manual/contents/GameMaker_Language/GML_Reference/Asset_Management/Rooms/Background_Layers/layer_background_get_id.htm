layer_background_get_id

Questa funzione può essere usata per recuperare il valore ID unico dell'elemento di sfondo su un livello. Si fornisce l'ID del livello (che si ottiene quando si usa il nome del livello insieme a layer_get_id()) e la funzione restituirà il valore ID associato all'elemento di sfondo sul livello. Nota che questa funzione è specificamente progettata per l'uso con gli sfondi che sono stati aggiunti in IDE, poiché se aggiungi uno sfondo ad un livello attraverso il codice usando la funzione layer_background_create()allora restituirà l'ID unico per l'elemento di sfondo aggiunto.

Sintassi:

layer_background_get_id(layer_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target

Ritorna:

Background Element ID

Esempio:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Il codice qui sopra otterrà l'ID del livello per il livello chiamato "Background_trees" e poi lo userà per ottenere l'ID dell'elemento di sfondo su quel livello. Questo ID viene poi usato per cambiare l'elemento sprite.

Indietro: Livelli di sfondo
