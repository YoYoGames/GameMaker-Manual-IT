stanza_persistente

Questa variabile può essere usata per ottenere e impostare il flag persistente per il corrente room. Se impostato su true il room è considerato persistente, nel qual caso ogni volta che si lascia il room e si ritorna di nuovo lo stato delle istanze all'interno di quel room sarà stato mantenuto. Tuttavia, se è contrassegnato come false, ogni volta che si ritorna a room verrà ripristinato il suo stato iniziale. Dovreste notare che un room persistente usa molta più memoria di un normale room e non è raccomandato averne troppi nel vostro gioco.

Sintassi:

room_persistent;

Ritorna:

Esempio:

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

Il codice qui sopra controlla la variabile "lives" e se è inferiore a 1, imposterà la persistenza room a false e poi cambierà rooms.
