fx_set_parameters

Questa funzione è usata per cambiare i parametri di un filtro/effetto. Si specifica un FX struct (come restituito da fx_create() o layer_get_fx()) e un struct contenente i suoi parametri (come restituito da fx_get_parameters()). Questo renderà le tue modifiche visibili nel livello a cui è assegnato l'FX struct.

Il tuo parametro struct non deve contenere tutti i parametri applicabili al filtro/effetto, e solo le variabili che sono presenti nel dato struct saranno aggiornate per il filtro/effetto.

Sintassi:

fx_set_parameters(filter_or_effect, parameter_struct);

ArgomentoTipo Descrizione
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

Ritorna:

Esempio:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Il codice qui sopra ottiene l'FX struct per un livello che si presume abbia il filtro "Colour Tint" applicato ad esso, e recupera il suo parametro struct chiamando fx_get_parameters(). Dopo di che crea un valore oscillante usando sin() e current_time, che è poi usato per i valori RGBA per l'effetto tinta in modo che continui ad animarsi. L'array RGBA viene assegnato alla variabile g_TintCol nel parametro struct, e il struct viene poi applicato nuovamente all'FX struct chiamando fx_set_parameters().
