layer_fx_is_enabled

Questa funzione restituisce se il filtro/effetto per un livello è abilitato.

Si specifica l'ID o il nome del livello che si vuole puntare, e la funzione restituisce un valore booleano che indica se l'FX del livello è abilitato(true) o disabilitato(false).

Sintassi:

layer_fx_is_enabled(layer_name_or_id);

Argomento Tipo Descrizione
layer_name_or_id Stringa o ID del livello The name or ID of the layer to check

Ritorna:

Boolean

Esempio:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

Il codice qui sopra "respawna" l'istanza, spostandola nella sua posizione originale e riempiendo i suoi HP. Poi controlla se il livello Desaturate FX è abilitato, e se lo è, lo disabilita.
