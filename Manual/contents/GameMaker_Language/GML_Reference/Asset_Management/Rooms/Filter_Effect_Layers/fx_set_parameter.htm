fx_set_parameter

Questa funzione è usata per cambiare il valore di un parametro nel dato FX struct. Si specifica un FX struct (come restituito da fx_create() o layer_get_fx()), un nome di parametro come string, e il suo nuovo valore. Questo renderà le tue modifiche visibili nel livello a cui l'FX struct è assegnato.

Siete in grado di passare più argomenti per il parametro nel caso in cui sia un vettore, tuttavia potete anche semplicemente specificare un array come unico argomento che conterrebbe i vostri valori multipli.

Trova i nomi dei parametri di un filtro/effetto in questa pagina: Tipi di FX e parametri.

Sintassi:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Argomento Tipo Descrizione
filter_or_effect Struttura FX The FX struct to modify
parameter_name Stringa The name of the parameter to change the value of (as a string)
val... Real o Array The new value(s) for that parameter as a real or array

Ritorna:

N/A

Esempio:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Il codice qui sopra crea un nuovo FX struct dall'effetto "_filter_tintfilter", che è il filtro "Colour Tint" trovato in Room Editor. Assegna un valore al suo parametro "g_TintCol" che è il colore della tinta, e poiché è un vec4 internamente, prende un array contenente quattro valori (corrispondenti ai suoi valori rosso, verde, blu e alfa). L'FX struct per la tinta viene poi applicato ad un livello FX.

Notate che siete anche in grado di passare valori al vettore come argomenti separati invece che come array:

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
