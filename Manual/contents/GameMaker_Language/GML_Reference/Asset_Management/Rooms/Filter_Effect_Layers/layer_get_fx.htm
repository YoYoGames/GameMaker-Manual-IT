layer_get_fx

Questa funzione è usata per recuperare l'FX struct per un livello. Si specifica l'ID o il nome del livello a cui si vuole mirare e la funzione restituirà un struct contenente informazioni sul suo effetto applicato. Questo struct sarà simile al struct che si ottiene da fx_create(), e le funzioni fx_get_parameter/s e fx_set_parameter/s possono essere utilizzate su di esso per leggere e modificare i suoi parametri.

Se il livello specificato non ha filtri/effetti applicati ad esso, la funzione restituisce -1.

Sintassi:

layer_get_fx(layer_name_or_id);

Argomento Tipo Descrizione
layer_name_or_id Stringa o ID del livello The name or ID of the layer to read

Ritorna:

FX Struct (or -1 if not found)

Esempio:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Il codice di cui sopra esegue un for ciclo attraverso tutti i livelli presenti in room, e controlla ogni livello per un FX struct. Se un livello ha un FX struct, controlla il nome di quel filtro/effetto chiamando fx_get_name() su di esso; se è uguale a "_filter_tintfilter" che significa che è un filtro "Colour Tint", cambia il suo colore di tinta in blu.
