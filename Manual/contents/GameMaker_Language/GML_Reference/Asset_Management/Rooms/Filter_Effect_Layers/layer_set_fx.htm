layer_set_fx

Questa funzione serve ad assegnare un FX struct a un Room livello. Si specifica l'ID o il nome del livello che si desidera modificare, quindi un FX struct (come restituito da fx_create() o layer_get_fx()) che verrà applicato al livello specificato, rendendo l'effetto visibile in room.

Se si specifica un livello che non è un livello filtro/effetto ma è di un altro tipo (come un livello istanza, Asset, Tile, ecc.), l'effetto verrà applicato al contenuto di quel livello e a tutti i livelli sottostanti.

Se al layer specificato è già assegnato un FX struct, questo verrà sostituito dal nuovo.

NOTA La chiamata di questa funzione al livello principale di un script non avrà alcun effetto, poiché scripts viene eseguito prima del caricamento del primo room. È possibile impostare prima una stanza di destinazione e poi chiamare questa funzione.

Sintassi:

layer_set_fx(layer_name_or_id, filter_or_effect);

Argomento Tipo Descrizione
layer_name_or_id Stringa o ID del livello The name or ID of the layer to modify
filter_or_effect Struttura FX The FX struct to apply to the layer

Restituzione:

N/A

Esempio:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Il codice qui sopra crea un nuovo FX struct dall'effetto "_filter_tintfilter", che è il filtro "Colour Tint" che si trova nell'editor Room. Assegna al suo parametro "g_TintCol" un valore che è il colore della tinta e, poiché internamente è un vec4, prende un array contenente quattro valori (corrispondenti ai valori di rosso, verde, blu e alfa). L'FX struct per la tinta viene quindi applicato a un livello FX.
