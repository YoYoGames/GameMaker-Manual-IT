fx_get_name

Questa funzione è usata per recuperare il nome di un filtro/effetto dal suo struct. Si specifica l'FX struct da leggere (come restituito da fx_create() o layer_get_fx()) e la funzione restituisce il suo nome come string. Questo nome può poi essere passato in fx_create() per creare un nuovo FX struct usando lo stesso tipo di filtro/effetto.

Sintassi:

fx_get_name(filter_or_effect);

ArgomentoTipo Descrizione
filter_or_effect The FX struct to get the name of

Ritorna:

Esempio:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Il codice di cui sopra esegue un for ciclo attraverso tutti i livelli presenti in room, e controlla ogni livello per un FX struct. Se un livello ha un FX struct, controlla il nome di quel filtro/effetto chiamando fx_get_name() su di esso; se è uguale a "_filter_tintfilter" che significa che è un filtro "Colour Tint", cambia il suo colore di tinta in blu.
