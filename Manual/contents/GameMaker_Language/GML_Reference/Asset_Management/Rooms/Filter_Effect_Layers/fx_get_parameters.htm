fx_get_parametri

Questa funzione viene utilizzata per recuperare una struttura contenente tutti i parametri per un determinato FX struct. Si specifica un FX struct (come restituito da fx_create() o layer_get_fx()) e questa funzione restituisce una struct contenente le variabili per tutti i parametri applicabili a quel filtro/effetto. I parametri che utilizzano vettori (cioè gruppi di valori) saranno presenti come array.

È possibile ottenere i nomi di tutti i parametri in un array separato chiamando fx_get_parameter_names(), oppure trovarli in questa pagina: Tipi e parametri FX.

La modifica dei valori in questo struct non applicherà le modifiche al filtro/effetto; sarà necessario richiamare fx_set_parameters() per applicare nuovamente il struct modificato al filtro/effetto affinché le modifiche abbiano effetto.

Sintassi:

fx_get_parameters(filter_or_effect);

Argomento Tipo Descrizione
filter_or_effect Struttura FX The FX struct for which the parameters will be returned

Restituzione:

Struct (or -1 on failure)

Esempio:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Il codice precedente ottiene l'FX struct per un livello a cui si presume sia stato applicato il filtro "Tinta colore" e recupera il suo parametro struct chiamando fx_get_parameters(). Poi crea un valore oscillante usando sin() e current_time, che viene usato per i valori RGBA dell'effetto tinta, in modo che continui ad animarsi. L'array RGBA viene assegnato alla variabile g_TintCol nel parametro struct, e il valore struct viene poi applicato nuovamente all'FX struct chiamando fx_set_parameters().
