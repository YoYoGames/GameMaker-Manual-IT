abilita_livello_fx

Questa funzione abilita/disabilita il filtro/effetto assegnato a un livello Room. Si specifica o l'ID o il nome del livello che si vuole modificare, e poi un valore booleano che dice se l'FX deve essere abilitato(true) o disabilitato(false).

Passando in false non rimuoverà l'effetto dal livello, ma semplicemente lo renderà invisibile. Usa layer_clear_fx() per rimuovere un FX da un livello.

Allo stesso modo, passare in true non farà nulla se non c'è un FX applicato al livello. Usa layer_set_fx() per applicare un FX a un livello.

Sintassi:

layer_enable_fx(layer_name_or_id, enable);

Argomento Tipo Descrizione
layer_name_or_id Stringa o ID del livello The name or ID of the layer to modify
enable Booleano Whether to enable or disable the FX

Ritorna:

N/A

Esempio:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

Il codice qui sopra abilita un livello Desaturate FX se il valore degli HP dell'istanza è inferiore o uguale a 3, altrimenti lo disabilita. Questo indica al giocatore che i suoi HP sono bassi, desaturando lo schermo.
