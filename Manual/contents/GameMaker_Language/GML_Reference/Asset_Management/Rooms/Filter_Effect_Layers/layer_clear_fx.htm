filtro_clear_fx

Questa funzione è usata per cancellare l'FX struct che è applicato al livello specificato. Si specifica l'ID o il nome del livello che si vuole modificare e questa funzione rimuoverà l'FX struct che è applicato a quel livello, disabilitando completamente i suoi effetti.

Se usi questa funzione per disabilitare temporaneamente un effetto e hai bisogno di ripristinarlo sul livello, dovresti prima salvare l'FX struct per il livello chiamando layer_get_fx() e poi chiamare questa funzione per cancellarlo sul livello. Puoi in seguito applicarlo di nuovo al livello chiamando layer_set_fx(), che riabiliterà l'effetto.

Sintassi:

layer_clear_fx(layer_name_or_id);

Argomento Tipo Descrizione
layer_name_or_id Stringa o ID del livello The name or ID of the layer to clear the FX for

Ritorna:

N/A

Esempio:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

Il codice qui sopra controlla se il tasto "E" è premuto, e poi controlla se il livello "EffectLayer" ha un filtro/effetto applicato ad esso; in tal caso memorizza il suo FX struct in una variabile temporanea e cancella il livello da qualsiasi filtro/effetto. Quando il tasto "F" viene premuto, controlla se la variabile temporanea contiene qualcosa di diverso da -1, e in tal caso applica nuovamente quell'FX al livello e resetta la variabile temporanea a -1. Tutto questo essenzialmente ti permette di attivare e disattivare un effetto ogni volta che ne hai bisogno senza perdere i suoi parametri.
