imposta la porta d'immagine della stanza

Con questa funzione potete impostare le proprietà della view port per qualsiasi room nel vostro gioco eccetto quello corrente. Si fornisce l'indice room in cui impostare la porta di visualizzazione, l'indice della porta di visualizzazione (da 0 a 7) e poi se la porta di visualizzazione deve essere visibile all'inizio del room o no (impostare su true per rendere la porta visibile, e false altrimenti). Dopo di che si impostano le coordinate x e y (corrispondenti alla posizione in alto a sinistra della porta di visualizzazione) e poi la larghezza e l'altezza in pixel.

Sintassi:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Argomento Tipo Descrizione
rm Camera Asset The index of the room to set
vind Real The index of the view port to set
vis Booleano The visibility of the view port (true è visibile, false è invisibile)
xport Real The x position for the view port in the room
yport Real The y position of the view port in the room
wport Real The width (in pixels) of the view port
hport Real The height (in pixels) of the view port

Ritorna:

N/A

Esempio:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Questo imposterà le proprietà della porta di vista [0] in room indicizzate nella variabile "global.myroom".
