larghezza_camera

Questa variabile contiene la larghezza dell'attuale room in pixel. Puoi cambiare questa variabile per cambiare la larghezza di room in qualsiasi momento.

Sintassi:

room_width;

Ritorna:

Esempio:

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

Il codice di cui sopra controlla se il bounding box dell'istanza corrente sprite è maggiore della larghezza di room, e se lo è sposta l'istanza verso l'alto.
