nome_della_stanza

Questa funzione può essere usata per restituire il nome del room specificato come string. Si prega di notare che questo è solo un string e non può essere usato per fare riferimento direttamente al room - per quello avreste bisogno dell'indice della stanza. Potete, tuttavia, usare questo string per ottenere l'indice della stanza usando il string restituito insieme alla funzione asset_get_index().

Sintassi:

room_get_name(index);

ArgomentoTipo Descrizione
index The index of the room to check the name of.

Ritorna:

Esempio:

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

Il codice di cui sopra otterrà il nome dell'attuale room e lo disegnerà sullo schermo.
