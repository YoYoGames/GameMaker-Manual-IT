ottenere la porta di vista della stanza

Con questa funzione è possibile recuperare i dettagli di una porta di vista in un room diverso da quello corrente. Si fornisce l'ID room e l'indice della porta di vista da recuperare (da 0 a 7) e la funzione restituirà un array di 5 indici, dove:

  • [0] = visibile (booleano: true / false)
  • [1] = posizione x (reale)
  • [2] = posizione y (reale)
  • [3] = larghezza (reale)
  • [4] = altezza (reale)

Sintassi:

room_get_viewport(rm, vind);

Argomento Tipo Descrizione
rm Camera Asset The index of the room to get viewport data from
vind Real The index of the view port to get

Ritorna:

Array (5 elements: visible, x, y, width, height)

Esempio:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

Il codice qui sopra recupera i dati della porta di visualizzazione per il dato room poi controlla se la porta è contrassegnata come visibile. Se non lo è, i dati della porta della vista vengono impostati per renderla visibile.
