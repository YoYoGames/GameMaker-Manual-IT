sequenza_strato_x

Con questa funzione potete impostare la posizione lungo l'asse X (orizzontale) di room per l'elemento dato sequence. Si fornisce l'ID dell'elemento sequence come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello insieme alla posizione X da impostare e il sequence sarà spostato nella nuova posizione.

Sintassi:

layer_sequence_x(sequence_element_id, pos_x)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target
pos_x The X position to move the sequence element to

Ritorna:

Esempio:

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

Il codice qui sopra controlla se l'elemento sequence a cui si fa riferimento nella variabile "my_seq" esiste, e se esiste imposta la posizione x/y a quella dell'istanza che esegue il codice.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_y
