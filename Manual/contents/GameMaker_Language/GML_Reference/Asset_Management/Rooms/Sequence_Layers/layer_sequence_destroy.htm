distruggi_sequenza

Con questa funzione potete distruggere (rimuovere da room) un elemento sequence. Si fornisce l'ID dell'elemento sequence come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello e l'elemento sequence sarà distrutto.

Sintassi:

layer_sequence_destroy(sequence_element_id)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target

Ritorna:

Esempio:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

Il codice di cui sopra ottiene gli ID per tutti gli elementi assegnati al livello dell'istanza che esegue il codice. Il codice controlla poi per vedere se qualcuno degli elementi restituiti è sequence assets e se lo è, viene distrutto.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_x
