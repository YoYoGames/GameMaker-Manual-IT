sequenze_get_sequenza

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e restituirà la struttura dell'oggetto sequence. Questa funzione bypassa la necessità di ottenere prima l'istanza sequence struct e permette di accedere direttamente ai dati di sequence. Potete trovare maggiori informazioni sul formato di sequence object struct in questa pagina.

Sintassi:

layer_sequence_get_sequence(sequence_element_id)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target

Ritorna:

Sequence Object Struct

Esempio:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Il codice precedente crea un nuovo elemento sequence e poi recupera i suoi dati sequence struct . Questo viene poi usato per cambiare la velocità di riproduzione e il tipo di sequence.
