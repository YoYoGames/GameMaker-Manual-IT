sequenza_livello_è_finito

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e controllerà se l'elemento sequence è finito di suonare o no, restituendo true se lo è, e false se non lo è. Si noti che questo è applicabile solo quando sequence non è impostato su loop o ping-pong nella modalità di riproduzione.

Sintassi:

layer_sequence_is_finished(sequence_element_id)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target

Ritorna:

Esempio:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

Il codice qui sopra controlla se l'ID dell'elemento sequence memorizzato nella variabile "title_sequence" è finito, e se lo è lo fa ripartire e imposta un allarme.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_exists
