pausa_di_sequenza

Con questa funzione potete mettere in pausa la riproduzione del dato sequence. Si fornisce l'ID dell'elemento sequence come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello e la funzione metterà in pausa il sequence fino a quando non inizierete di nuovo la riproduzione usando la funzione layer_sequence_play().

Sintassi:

layer_sequence_pause(sequence_element_id)

Argomento Tipo Descrizione
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target

Ritorna:

N/A

Esempio:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

Il codice qui sopra crea un nuovo sequence e memorizza il suo ID in una variabile locale. Poi controlla se il gioco è in pausa, e se lo è mette in pausa la riproduzione di sequence.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_play
