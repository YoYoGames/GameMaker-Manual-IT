sequenza_strato_get_y

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e restituirà la posizione Y corrente nel gioco room per l'elemento sequence.

Sintassi:

layer_sequence_get_y(sequence_element_id)

Argomento Tipo Descrizione
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target

Ritorna:

Real

Esempio:

if layer_sequence_get_y(title_sequence) != room_height / 2
{
    layer_sequence_y(title_sequence, room_height / 2);
}

Il codice qui sopra controlla la posizione Y dell'ID dell'elemento sequence memorizzato nella variabile "title_sequence", e se non è impostato a metà dell'altezza di room, viene impostato a questo valore.
