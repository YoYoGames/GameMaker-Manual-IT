angolo di acquisizione di una sequenza di livelli

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e restituirà l'angolo corrente dell'elemento sequence nel gioco room. Notate che gli angoli sono restituiti in gradi, e 0º è a destra, 90º è in alto, 180º è a sinistra e 270º è in basso.

Sintassi:

layer_sequence_get_angle(sequence_element_id)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target

Ritorna:

Esempio:

var _ang = layer_sequence_get_angle(title_sequence)
if (_ang > 0)
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

Il codice qui sopra recupera l'angolo corrente dell'elemento sequence con l'ID memorizzato nella variabile "title_sequence", e se non è 0, allora viene sottratto 1 all'angolo corrente.
