la sequenza è in pausa

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e controllerà se sequence è attualmente in pausa o no, restituendo true se è in pausa, e false se non lo è.

Sintassi:

layer_sequence_is_paused(sequence_element_id)

ArgomentoTipo Descrizione
sequence_element_id The unique ID value of the sequence element to target

Ritorna:

Esempio:

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

Il codice qui sopra controlla se l'ID dell'elemento sequence memorizzato nella variabile "title_sequence" è in pausa, e se lo è ne inizia la riproduzione.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_is_finished
