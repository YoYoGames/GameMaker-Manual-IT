sequenza_strato_get_instanza

Con questa funzione si fornisce l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello - e restituirà la struttura di istanza di sequence. Puoi trovare maggiori informazioni sul formato della struct di istanza di sequence in questa pagina.

Sintassi:

layer_sequence_get_instance(sequence_element_id)

Argomento Tipo Descrizione
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target

Ritorna:

Sequence Instance Struct

Esempio:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

Il codice qui sopra crea un nuovo elemento sequence e poi recupera il suo ID di istanza sequence. Questo viene poi usato per cambiare la proprietà di scala della velocità di riproduzione dell'istanza sequence.
