angolo_di_sequenza

Con questa funzione puoi impostare l'angolo di rotazione per l'elemento dato sequence. Si fornisce l'ID dell'elemento sequence come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello insieme al nuovo angolo da impostare e sequence sarà ruotato intorno alla sua origine nella nuova posizione. Gli angoli in GameMaker sono calcolati con 0º a destra, e vanno in senso antiorario - quindi 90º è in alto, 180º è a sinistra e 270º è in basso - e l'angolo predefinito per un sequence sarebbe 0º.

Notate che non dovreste cambiare le variabili image_xscale \ image_yscale \ image_angle \ x \ y per qualsiasi istanza in un sequence che usa questa funzione, poiché saranno sovrascritte non appena il sequence si aggiorna dopo aver chiamato la funzione.

Sintassi:

layer_sequence_angle(sequence_element_id, angle)

Argomento Tipo Descrizione
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target
angle Real The new angle to rotate the sequence element to

Ritorna:

N/A

Esempio:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

Il codice qui sopra controlla il valore tenuto nella variabile current_angle, e se è inferiore a 90 allora lo aggiunge e poi usa il valore per impostare l'angolo dell'elemento sequence a cui si fa riferimento nella variabile my_seq.
