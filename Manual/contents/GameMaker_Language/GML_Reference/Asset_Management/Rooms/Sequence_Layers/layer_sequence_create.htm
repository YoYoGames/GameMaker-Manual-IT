creare una sequenza di livelli

Con questa funzione puoi creare un'istanza di un sequence asset sul dato livello. Si fornisce l'ID del livello che può essere un string del nome del livello - come definito nell'editor di room - o l'ID unico del livello - come restituito dalla funzione layer_get_id(), così come la posizione X e Y in room per creare il sequence, e infine l'ID del sequence da creare. L'ID di sequence è la costante del nome che avete definito nel Browser di Asset per l'elemento sequence. La funzione restituirà l'ID univoco dell'elemento sequence, che può poi essere usato in tutte le ulteriori funzioni di livello per sequences, o può essere usato per recuperare l'istanza sequence struct usando la funzione layer_sequence_get_instance().

Vale la pena notare che se il sequence contiene qualsiasi traccia object, le loro istanze saranno create non appena l'elemento sequence stesso viene creato, indipendentemente da dove sono posizionate le loro chiavi asset sul Dope Sheet. Il controllore Sequence semplicemente alterna la visibilità dell'istanza per nasconderla e mostrarla a seconda della posizione e della durata della chiave della traccia asset e non la crea e distrugge ripetutamente. Di conseguenza, le istanze eseguiranno i loro eventi Create quando l'elemento sequence viene creato e non quando iniziano le loro chiavi di risorsa.

Sintassi:

layer_sequence_create(layer_id, x, y, sequence_id)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the instance layer to target
x Real The x position in the room to create the sequence at
y Real The y position in the room to create the sequence at
sequence_id Sequenza Attività The sequence asset to use, as defined in the Asset Browser

Ritorna:

Sequence Element ID

Esempio:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Il codice qui sopra crea un nuovo sequence sul livello "Background" poi lo mette in pausa.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_destroy
