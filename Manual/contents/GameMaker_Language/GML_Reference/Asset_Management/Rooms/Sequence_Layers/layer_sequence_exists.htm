esiste una sequenza di livelli

Con questa funzione puoi controllare se un elemento sequence esiste sul dato livello. Si fornisce l'ID del livello che può essere un string del nome del livello - come definito nell'editor di room - o l'ID unico del livello - come restituito dalla funzione layer_get_id(), così come l'ID dell'elemento sequence - come restituito da layer_sequence_create() o da una delle funzioni degli elementi del livello - e restituirà true se l'elemento dato esiste o false altrimenti.

Sintassi:

layer_sequence_exists(layer_id, sequence_element_id)

Argomento Tipo Descrizione
layer_ID Stringa o ID del livello The unique ID or name of the layer to check
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target

Ritorna:

Boolean

Esempio:

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

Il codice qui sopra controlla se l'elemento dato sequence esiste, e se non esiste allora crea un nuovo sequence sul livello dato e lo mette in pausa.

Indietro: Strati di sequenza
Il prossimo: layer_sequence_create
