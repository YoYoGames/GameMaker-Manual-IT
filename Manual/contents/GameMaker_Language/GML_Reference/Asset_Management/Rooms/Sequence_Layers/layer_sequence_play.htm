gioco_sequenza_strato

Con questa funzione potete avviare la riproduzione del dato sequence. Si fornisce l'ID dell'elemento sequence come restituito da layer_sequence_create() o da una delle funzioni degli elementi di livello e la funzione riprodurrà il sequence, che potrete poi mettere in pausa se necessario usando la funzione layer_sequence_pause().

IMPORTANTE! Se il tuo sequence ha delle istanze in esso, queste istanze non dovrebbero cambiare le loro variabili image_xscale / image_yscale / image_angle / x / y perché saranno sovrascritte quando il sequence aggiorna ogni passo dopo l'inizio della riproduzione.

Sintassi:

layer_sequence_play(sequence_element_id)

Argomento Tipo Descrizione
sequence_element_id ID dell'elemento di sequenza The unique ID value of the sequence element to target

Ritorna:

N/A

Esempio:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if (layer_get_element_type(a[i]) == layerelementtype_sequence)
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

Il codice qui sopra controlla se il gioco è stato messo in pausa o meno quando viene premuto un tasto. Se il gioco è in pausa, allora loops attraverso tutti gli elementi sequence sul livello corrente (il livello dell'istanza chiamante) e mette in pausa la loro riproduzione, e se il gioco non è in pausa, allora il loop inizierà di nuovo la riproduzione.
