camera_goto

Questa funzione vi permette di andare a qualsiasi room nel vostro progetto di gioco, sia esso creato con il codice o nel Browser Asset. Si fornisce l'indice room (memorizzato nella variabile del nome room, o come variabile restituita dalla funzione room_add()). Notate che il room non cambierà fino alla fine dell'evento in cui la funzione è stata chiamata, quindi qualsiasi codice dopo che questa è stata chiamata verrà comunque eseguito se nello stesso evento. Questa funzione attiverà anche l'evento Room End.

NOTA Room Gli ID non sono basati sul loro ordine nel Browser Asset o nel Manager Room, e quindi si dovrebbe evitare di fornire direttamente un valore numerico. Invece, usate la costante di stanza per la asset a cui volete fare riferimento (che è semplicemente il suo nome) o recuperatela attraverso una funzione.

Sintassi:

room_goto(index);

ArgomentoTipo Descrizione
index The index of the room to go to.

Ritorna:

Esempio:

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

Il codice qui sopra controllerà una variabile globale e cambierà room in base al valore che contiene.
