livello_sprite_x

Questa funzione controlla la posizione lungo l'asse x del room dell'elemento asset sprite sul livello. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usi la funzione layer_sprite_get_id()), e poi imposti il valore x da usare (basato sulle coordinate di room ).

Sintassi:

layer_sprite_x(sprite_element_id, x);

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite element to change
x The x position for the sprite

Ritorna:

Esempio:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

Il codice qui sopra ottiene il valore ID del sprite asset "Clouds" assegnato al livello "Asset_sky" e poi imposta la sua posizione x a un valore casuale tra 0 e la larghezza del room.
