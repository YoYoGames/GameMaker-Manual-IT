Creazione di un livello di sprite

Con questa funzione puoi assegnare una risorsa sprite ad un livello da usare nel tuo progetto. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create() o quando usi il nome del livello insieme a layer_get_id()), una posizione all'interno di room, e un indice sprite (che sarebbe il nome del sprite come mostrato nel Browser Asset ), e sarà aggiunto al livello. La funzione restituisce il valore ID unico per l'elemento, che può quindi essere usato in ulteriori funzioni di livello per sprites.

Sintassi:

layer_sprite_create(layer_id, x, y, sprite)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target
x Real The x position to use
y Real The y position to use
sprite Sprite Asset The sprite index to be used

Ritorna:

Sprite Element ID

Esempio:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

Il codice qui sopra crea un nuovo livello e poi vi aggiunge 10 nuovi elementi sprite, memorizzando l'ID di ogni elemento in un array.
