layer_sprite_get_id

Questa funzione può essere usata per recuperare il valore ID unico di un elemento sprite su un livello. Si fornisce l'ID del livello (che si ottiene quando si crea il livello usando layer_create() o quando si usa il nome del livello insieme a layer_get_id()) e il nome dell'elemento sprite come definito nell'editor Room. La funzione restituirà il valore ID associato a quell'elemento sprite sul livello.

Si noti che questa funzione è specificamente progettata per l'uso con sprites che sono stati aggiunti nel IDE su un livello asset, e se si è aggiunto un sprite a un livello attraverso il codice utilizzando la funzione layer_sprite_create()allora l'ID restituito da quella funzione dovrebbe essere usato per tutti i riferimenti futuri (poiché quell'elemento sprite non avrà un nome da passare in questa funzione).

Se il livello specificato non esiste, o l'elemento dato sprite non può essere trovato, la funzione restituisce -1.

Sintassi:

layer_sprite_get_id(layer_id, sprite_element_name)

Argomento Tipo Descrizione
layer_id Stringa o ID del livello The unique ID value of the layer to target
sprite_element_name Stringa The unique name of the sprite element on the layer as defined in the Room Editor

Ritorna:

Sprite Element ID

Esempio:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

Il codice qui sopra otterrà l'ID del livello chiamato "Assets_trees" e poi lo userà per recuperare l'ID dell'elemento sprite "graphic_254367CB" su quel livello. L'ID dell'elemento sprite recuperato viene poi usato per cambiare l'elemento sprite.
