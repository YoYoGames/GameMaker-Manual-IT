distruggi_strato

Questa funzione distruggerà l'elemento dato sprite. Si fornisce l'ID di sprite (che si ottiene quando si crea sprite usando layer_sprite_create() o quando si usa l'ID del livello insieme a layer_get_sprite_id()) e questo lo rimuoverà. Nota che questo non rimuove il livello, solo il sprite da esso, e se il sprite è uno che è stato aggiunto nell'editor room, allora la prossima volta che lasci il room e poi ritorni, il sprite sarà ricreato di nuovo. Tuttavia se room è persistente, sprite sarà rimosso a meno che la persistenza di room sia nuovamente disattivata.

Sintassi:

layer_sprite_destroy(sprite_element_id)

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite to be destroyed

Ritorna:

Esempio:

var lay_id = layer_get_id("Asset_Trees");
var spr_id = layer_sprite_get_id(lay_id, "graphic_254367CB");

if (layer_sprite_exists(lay_id, spr_id))
{
    layer_sprite_destroy(lay_id);
}

Il codice qui sopra controlla il livello "Asset_Trees" per vedere se l'elemento dato sprite esiste e se esiste, allora viene distrutto (ma non il livello).
