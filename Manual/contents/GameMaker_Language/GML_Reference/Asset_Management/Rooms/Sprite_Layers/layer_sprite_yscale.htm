scala dei livelli

Usando questa funzione puoi cambiare se l'elemento dato sprite su un livello deve essere scalato lungo l'asse y o no. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usi la funzione layer_sprite_get_id()), e poi imposti il valore di scala. Una scala di 1 indica nessuna scala (1:1), valori più piccoli ridurranno la scala (0.5, per esempio, dimezzerà l'altezza dell'elemento sprite usato), valori più grandi la scaleranno, e valori negativi rispecchieranno l'elemento sprite e lo scaleranno a meno che il valore usato sia esattamente -1 (nel qual caso l'elemento sprite usato è semplicemente rispecchiato dall'alto in basso sulla sua posizione (0, 0) senza alcuna scala).

Sintassi:

layer_sprite_yscale(sprite_element_id, yscale);

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

Ritorna:

Esempio:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Il codice di cui sopra controllerà l'elemento sprite assegnato al livello su cui si trova l'istanza che esegue il codice e se è scalato in entrambe le direzioni, imposterà sia la scala dell'asse x che quella dell'asse y a 1.
