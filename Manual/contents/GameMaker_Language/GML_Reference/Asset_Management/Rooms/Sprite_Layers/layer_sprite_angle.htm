angolo di sprite

Usando questa funzione puoi cambiare l'angolo per il dato elemento sprite su un livello. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usate la funzione layer_sprite_get_id()), e poi imposti il valore dell'angolo, da 0 a 359 in senso antiorario, dove 0 è a destra, 90 è in alto, 180 è a sinistra e 270 è in basso. Se impostate un valore maggiore di 360, questo sarà ripetuto per portarlo all'interno della gamma 0 - 359.

Sintassi:

layer_sprite_angle(sprite_element_id, angle);

Argomento Tipo Descrizione
sprite_element_id ID dell'elemento sprite The unique ID value of the sprite element to change
angle Real The angle of the sprite (default is 0)

Ritorna:

N/A

Esempio:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if (layer_sprite_get_angle(asset_sprite) != 0)
{
    layer_sprite_angle(asset_sprite, 0);
}

Il codice di cui sopra controllerà l'elemento sprite assegnato al livello su cui si trova l'istanza che esegue il codice e se il suo angolo non è 0 lo imposterà a 0.
