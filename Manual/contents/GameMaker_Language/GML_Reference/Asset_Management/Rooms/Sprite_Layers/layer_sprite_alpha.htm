livello_sprite_alpha

Questa funzione controlla l'alfa (trasparenza) del sprite sul livello asset. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usate la funzione layer_sprite_get_id()), e poi imposti il valore alfa da usare. Alpha può essere compreso tra 0 (completamente trasparente) e 1 (completamente opaco) con il valore alpha di default per l'elemento sprite che è 1. Nota che se il livello a cui è stato assegnato l'elemento sprite non è visibile - o l'elemento stesso è stato reso invisibile - non vedrete alcuna differenza con questa funzione finché il livello o l'elemento non saranno resi nuovamente visibili.

Sintassi:

layer_sprite_alpha(sprite_element_id, alpha);

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

Ritorna:

Esempio:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

Il codice qui sopra ottiene il valore ID del sprite asset chiamato "Clouds" assegnato al livello "Asset_sky" e poi imposta la sua alfa ad un valore casuale tra 0 e 1.
