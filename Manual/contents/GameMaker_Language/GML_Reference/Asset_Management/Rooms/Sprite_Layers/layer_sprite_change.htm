cambio_sprite_strato

Usando questa funzione puoi cambiare la risorsa sprite assegnata ad un dato elemento sprite su un livello. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usate la funzione layer_sprite_get_id()), e poi fornisci un elemento di indice sprite asset sarà cambiato. Notate che se sprite ha delle sotto-immagini, allora anch'esso si animerà (questo può essere controllato usando i tasti layer_sprite_index() e layer_sprite_speed() ). Notate che potete assegnare un valore di -1 come nuovo indice sprite e nessun sprite sarà mostrato, anche se l'elemento esisterà ancora e potrà essere cambiato di nuovo in seguito.

Sintassi:

layer_sprite_change(sprite_element_id, sprite_index)

Argomento Tipo Descrizione
sprite_element_id ID dell'elemento sprite The unique ID value of the sprite element to change
sprite_index Sprite Asset The new sprite index to use

Ritorna:

N/A

Esempio:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if (layer_sprite_get_sprite(s) != spr_Nighttime)
{
    layer_sprite_change(s, spr_nighttime);
}

Il codice di cui sopra controlla l'indice sprite dell'elemento con l'ID memorizzato nella variabile globale global.Asset_sprite e se non è "spr_Nighttime" allora quel sprite viene assegnato all'elemento.

Indietro: Livelli di sprite
Il prossimo: layer_sprite_index
