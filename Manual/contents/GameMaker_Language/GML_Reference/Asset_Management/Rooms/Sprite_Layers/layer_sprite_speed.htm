velocità dello sprite

Questa funzione può essere usata per impostare la velocità di un'immagine animata che è stata assegnata a un elemento sprite. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usate la funzione layer_sprite_get_id()), e poi impostate la velocità di animazione. Se il IDE ha impostato la velocità di animazione per il sprite usato per essere calcolato come Frames Per Game Frame, allora il valore di velocità dato è un moltiplicatore, con 1 che è il valore predefinito, e impostandolo a 0,5 si dimezza la velocità di animazione, mentre impostandolo a 1,5 la si raddoppia. Tuttavia, se il IDE ha impostato la velocità di animazione per il sprite usato per essere calcolato come Frames Per Second, allora il valore di velocità dato è un valore letterale di frames per secondo, quindi impostandolo a 5, per esempio, imposterà l'animazione a loop 5 frames ogni secondo che il gioco gira. Se il sprite usato non ha sotto-immagini, questa funzione non avrà alcun effetto.

Sintassi:

layer_sprite_speed(sprite_element_id, speed);

Argomento Tipo Descrizione
sprite_element_id Sprite Asset The unique ID value of the sprite element to change
speed Real The speed multiplier/value to use for the element

Ritorna:

N/A

Esempio:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Il codice qui sopra otterrà l'ID del livello chiamato "Assets_water" e poi lo userà per ottenere l'ID del dato elemento sprite su quel livello. Questo ID viene poi usato per cambiare la velocità dell'immagine dell'elemento.
