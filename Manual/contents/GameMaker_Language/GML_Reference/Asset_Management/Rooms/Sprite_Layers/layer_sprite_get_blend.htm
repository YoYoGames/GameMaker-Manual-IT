mescola_sprite_get_blend

Questa funzione può essere usata per ottenere il colore di fusione dell'elemento sprite. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando usate la funzione layer_sprite_get_id()), e la funzione restituirà un valore reale che rappresenta il colore assegnato.

Sintassi:

layer_sprite_get_blend(sprite_element_id);

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite element to get the information from

Ritorna:

Colour Constant

Esempio:

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

Il codice qui sopra otterrà l'ID del livello chiamato "sprite_sky" e poi lo userà per ottenere l'ID dell'elemento sprite su quel livello. Questo ID viene poi usato per controllare il colore della miscela dell'elemento e se è equivalente alla costante c_white, allora la miscela dell'elemento viene impostata su un colore casuale.
