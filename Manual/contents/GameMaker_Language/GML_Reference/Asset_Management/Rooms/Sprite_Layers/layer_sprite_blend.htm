miscela_sprite

Questa funzione controlla la miscelazione (o "tinta") di sprite sprite e il valore predefinito è -1 (che rappresenta la costante c_white, che può anche essere usata). Qualsiasi altro valore (comprese le costanti di colore interne come c_red, o c_aqua) fonderà il colore specificato con l'originale sprite. Si dà l'ID dell'elemento sprite (che si ottiene quando si crea un elemento sprite usando layer_sprite_create() o quando si usa la funzione layer_sprite_get_id()), e poi impostate il colore di miscelazione da usare. Qui sotto potete vedere un esempio di un sprite che è stato miscelato con diversi colori:

Si prega di notare che si dovrebbe cercare di limitare il blending sulla piattaforma HTML5 (a meno che non si usi WebGL), poiché ogni sprite mescolato deve essere memorizzato nella cache separatamente e quindi avere molti sprites mescolati può influire negativamente sulle prestazioni (si può anche impostare la dimensione di cache usando la funzione sprite_set_cache_size()).

Sintassi:

layer_sprite_blend(sprite_element_id, blend);

ArgomentoTipo Descrizione
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

Ritorna:

Esempio:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

Il codice qui sopra ottiene il valore ID del sprite chiamato "Clouds" assegnato al livello "Asset_sky" e poi lo tinge di un colore.
