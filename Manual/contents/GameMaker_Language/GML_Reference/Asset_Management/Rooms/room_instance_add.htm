aggiungi istanza stanza

Con questa funzione potete aggiungere un'istanza in qualsiasi room diverso da quello corrente e in qualsiasi posizione all'interno di quel room. La funzione restituisce l'univoco id dell'istanza che può poi essere usata in ulteriori chiamate di funzione per impostare proprietà ecc... di quell'istanza, ma solo una volta che il gioco è entrato nella stanza specificata. Se volete creare un'istanza nel corrente room dovreste usare la funzione instance_create_layer(). Si noti che chiamando questa funzione su un room asset creato nel Browser Asset si aggiungerà permanentemente l'istanza alla stanza, e anche chiamando game_restart() non si riporterà il room al suo stato originale (solo terminando il gioco e riaprendolo si ripartirà con il room nel suo stato originale).

Sintassi:

room_instance_add(index, x, y, obj);

ArgomentoTipo Descrizione
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

Ritorna:

Instance ID

Esempio:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

Il codice qui sopra aggiungerà un nuovo room al gioco e poi copierà il contenuto del room indicizzato come "rm_Base" in esso. Quindi aggiungerà un'istanza di object "obj_player" alla posizione 100,100 di questo nuovo room.

Indietro: Camere
Il prossimo: room_instance_clear
