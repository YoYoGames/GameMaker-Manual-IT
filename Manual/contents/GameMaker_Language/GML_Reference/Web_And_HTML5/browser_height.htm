altezza del browser

Questa variabile contiene l'altezza (in pixel) del browser in cui il gioco viene eseguito. Se nessun browser è presente, allora viene restituita la dimensione della finestra. L'immagine seguente illustra come questa variabile si riferisce al resto delle funzioni di posizionamento disponibili attraverso le funzioni Window.

Sintassi:

browser_height

Ritorna:

Esempio:

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

Il codice qui sopra controlla la larghezza e l'altezza del browser per vedere se una delle due è più grande della larghezza e dell'altezza della finestra corrente. Se lo sono, calcola una nuova posizione per la finestra di gioco all'interno del browser.
