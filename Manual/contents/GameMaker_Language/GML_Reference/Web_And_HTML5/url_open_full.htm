url_open_full

Questo aprirà l'URL specificato nel browser del dispositivo di destinazione scelto o, se si utilizza il modulo HTML5, nel browser attualmente aperto. Il parametro "target" che si specifica è lo stesso del valore "name" standard di JavaScript quando si usa il metodo open() (si tenga presente che tutti gli elementi tranne "_self" possono causare il blocco del browser o chiedere all'utente se desidera consentirlo) e il parametro "options" è lo stesso del parametro "specs" di JavaScript per controllare le proprietà che la nuova finestra/tab deve visualizzare (non tutti i browser possono supportare tutte le caratteristiche).

Gli obiettivi validi sono:

Obiettivo Descrizione
_blank Apre il documento collegato in una nuova finestra o scheda (non funziona se i pop-up sono bloccati dall'utente, nel qual caso si possono usare le funzioni clickable_* al posto delle funzioni).
sé stesso Apre il documento collegato nella stessa cornice in cui è stato cliccato (impostazione predefinita).
genitore Apre il documento collegato nel riquadro principale.
In alto Apre il documento collegato nell'intero corpo della finestra.

Le opzioni valide sono:

Parametri Descrizione
'height=[px]' L'altezza della finestra, con un valore minimo pari a 100.
'width=[px]' Larghezza della finestra, il cui valore minimo è 100.
sinistra=[px]' La posizione sinistra della finestra.
top=[px]' La posizione superiore della finestra (solo IE).
location=[boolean] Se visualizzare o meno il campo dell'indirizzo (l'impostazione predefinita è 1).
'menubar=[boolean]' Se visualizzare o meno la barra dei menu (l'impostazione predefinita è 1).
'ridimensionabile=[booleano]' Se la finestra è ridimensionabile o meno (l'impostazione predefinita è 1).
scrollbars=[boolean]' Se visualizzare o meno le barre di scorrimento (l'impostazione predefinita è 1).
status=[boolean] Se aggiungere o meno una barra di stato (l'impostazione predefinita è 1).
Barra del titolo=[booleano]' Se visualizzare o meno la barra del titolo. Questa opzione viene ignorata a meno che l'applicazione chiamante non sia un'applicazione HTML o una finestra di dialogo affidabile (l'impostazione predefinita è 1);
barra degli strumenti=[booleano]' Se visualizzare o meno la barra degli strumenti del browser (l'impostazione predefinita è sì).

Sintassi:

url_open_full(url, target, options);

Argomento Tipo Descrizione
url Stringa The URL (website address) to link to.
target Stringa This is the target area to open the URL in (see description).
options Stringa Standard browser options (see description).

Restituzione:

N/A

Esempio:

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

In questo modo si aprirebbe la homepage di GameMaker in una nuova finestra che non può essere ridimensionata, con un'altezza di 200 pixel e senza barre di scorrimento.
