aggiungi cliccabile

Con questa funzione è possibile creare un'icona personalizzata, cliccabile, che agisce come un "pulsante" al di fuori della tela di gioco, ma all'interno della finestra del browser stesso. La funzione restituirà un valore ID che deve essere memorizzato in una variabile per essere utilizzato in tutte le ulteriori funzioni del pulsante che fanno riferimento al nuovo cliccabile. Questa funzione è molto utile per evitare che il browser crei una finestra pop-up quando si clicca sul pulsante, in quanto è tutto fatto attraverso GameMaker stesso.

Questa funzione funziona esattamente come la funzione clickable_add() dove si deve impostare la posizione relativa al (0,0) della finestra del browser, dare la voce della pagina texture per il sprite che si desidera che l'icona del pulsante abbia (vedi: sprite_get_tpe()), specificare o un URL da aprire o un script (dal Browser Asset di GameMaker) da aprire o eseguire quindi impostare il target e i parametri da utilizzare. Tuttavia questa funzione ha anche due opzioni aggiuntive che permettono di impostare l'alfa dell'icona da 0 (completamente trasparente) a 1 (completamente opaca) e la scala dell'icona, che è un valore moltiplicatore applicato al sprite usato. Così un valore di 1 è la scala predefinita, mentre (per esempio) 0,5 sarebbe metà scala e 2 sarebbe il doppio.

Vale anche la pena notare che se l'argomento URL che passate è il nome di una funzione di script (cioè: non inizia con "http://") E la script è chiamata gmcallback_* allora potete ottenere un feed diretto in un evento html5. I nomi delle funzioni script di GameMaker non sono offuscati se iniziano con gmcallback_.

Sintassi:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argomento Tipo Descrizione
x Real The x position within the window.
y Real The y position within the window.
tpe Inserimento pagina texture The texture page entry for the sprite to be used.
url Stringa The URL (website address) to link to, or the script to be run.
target Stringa This is the target area to open the URL in (see description).
params Stringa The various parameters for the icon (see description).
alpha Real The image alpha of the icon (default 1).
scale Real The scale of the icon (default 1).

Ritorna:

Clickable ID

Esempio:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

Il codice qui sopra crea un'icona DOM cliccabile nella posizione (32, 32) della pagina su cui sta girando il game canvas. L'icona usa il sprite referenziato dalla pagina texture come "spr_MS_Home" e quando viene cliccata l'icona aprirà una nuova finestra per l'URL specificato e con le proprietà definite per quella finestra. il pulsante sarà anche scalato al doppio della dimensione originale sprite e a metà di un'immagine alfa di 1.
