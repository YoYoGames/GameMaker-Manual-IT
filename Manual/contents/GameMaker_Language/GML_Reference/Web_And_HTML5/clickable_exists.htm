cliccabile_esiste

Questa funzione restituisce se un'icona DOM cliccabile è stata creata con l'indice specificato esiste o no. Si prega di notare che il valore utilizzato per il controllo deve essere stato inizializzato in precedenza, altrimenti si otterrà un errore che causerà la chiusura di GameMaker.

Sintassi:

clickable_exists(index);

Argomento Tipo Descrizione
index ID cliccabile The index of the clickable to check.

Ritorna:

Boolean

Esempio:

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

Il codice qui sopra controlla la variabile "home_but" per vedere se esiste già, e se non esiste crea un'icona DOM cliccabile nella posizione (32, 32) della pagina su cui sta girando il game canvas. L'icona usa il sprite referenziato dalla pagina texture come "spr_MS_Home" e quando viene cliccata l'icona aprirà una nuova finestra per l'URL specificato e con le proprietà definite per quella finestra.

Indietro: Web e HTML5
