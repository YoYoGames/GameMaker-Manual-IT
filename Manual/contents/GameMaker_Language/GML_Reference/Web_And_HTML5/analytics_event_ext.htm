analytics_event_ext

Questa funzione invierà il testo specificato al fornitore di analisi che avete impostato attraverso le Opzioni di gioco HTML5. Questa funzione può essere usata per creare un'analisi personalizzata per tracciare qualcosa al di fuori dell'ambito del provider utilizzato, e accetterà anche coppie parametro/valore personalizzate, dove il parametro è un string e il valore un numero reale. Per Google Analytics, è possibile aggiungere solo una coppia extra, mentre Flurry accetta fino a 7.

Sintassi:

analytics_event_ext(string, string_param, value);

ArgomentoTipo Descrizione
string A string to send to the provider.
string_param[0 -9] The parameter to send (a string).
value[0 - 9] The value of the parameter (a real number).

Ritorna:

Esempio:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

Il codice di cui sopra otterrà le informazioni di riproduzione da un file ini e l'ora corrente e poi invierà questi dettagli al fornitore di analisi.
