cliccabile_cambiamento_ext

Con questa funzione potete cambiare il sprite e la posizione di un'icona cliccabile creata precedentemente con clickable_add(). Tenete presente che la posizione è basata sulla posizione della finestra, non del canvas, (0,0) e che il sprite deve essere referenziato direttamente dalla pagina texture (vedi: sprite_get_tpe()). Questa funzione permette anche di cambiare la scala del sprite usato (come moltiplicatore in modo che 1 sia il valore predefinito, 0,5 sarebbe la metà e 2 sarebbe il doppio) e il valore alfa da 0 (completamente trasparente) a 1 (completamente opaco) per l'icona finale sullo schermo.

Sintassi:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Argomento Tipo Descrizione
index ID cliccabile The index of the clickable icon to change.
tpe Inserimento pagina texture The texture page entry for the sprite to be used
x Real The new x position within the window.
y Real The new y position within the window.
scale Real The scale of the icon (default 1).
alpha Real The image alpha of the icon (default 1).

Ritorna:

N/A

Esempio:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Il codice di cui sopra cambierà l'indice dell'immagine e la posizione dell'icona cliccabile indicizzata nella variabile "global.Help" a seconda dell'attuale room, cambiando anche la scala e l'alfa dell'icona.
