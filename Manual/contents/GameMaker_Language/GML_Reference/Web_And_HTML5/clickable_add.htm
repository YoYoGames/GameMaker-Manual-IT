aggiungi cliccabile

Con questa funzione è possibile creare un'icona personalizzata, cliccabile, che agisce come un "pulsante" al di fuori della tela di gioco, ma all'interno della finestra del browser stesso. La funzione restituirà un valore ID che deve essere memorizzato in una variabile per essere utilizzato in tutte le ulteriori funzioni del pulsante che fanno riferimento al nuovo cliccabile. Questa funzione è molto utile per evitare che il browser crei una finestra pop-up quando si clicca sul pulsante, in quanto è tutto fatto attraverso GameMaker stesso.

Devi prima impostare la posizione, che è relativa al (0,0) della finestra del browser (angolo in alto a sinistra) e poi dare la voce della pagina texture per il sprite che desideri che l'icona del pulsante abbia (vedi: sprite_get_tpe()). Potete quindi specificare un URL da aprire o un script (dal Browser Asset di GameMaker) da aprire o eseguire. Il parametro "target" che specificate è lo stesso del valore standard di JavaScript "name" quando usate il metodo open() (siate consapevoli che tutto tranne '_self' può risultare nel blocco del browser, o chiedere all'utente se desiderano permetterlo) e l'argomento "params" è lo stesso del parametro JavaScript "specs" per controllare quali proprietà la nuova finestra/tab dovrebbe mostrare (non tutti i browser possono supportare tutte le caratteristiche). Se state usando l'icona per eseguire un script interno, questi ultimi due argomenti possono essere lasciati vuoti strings "".

Gli obiettivi validi sono:

Obiettivo Descrizione
_blank Apre il documento collegato in una nuova finestra o scheda.
se stesso Apre il documento collegato nella stessa cornice in cui è stato cliccato (questo è il default).
_parent Apre il documento collegato nella cornice madre.
{\an8}Si tratta di un'esperienza di vita, ma non di un'esperienza di vita. Apre il documento collegato nel corpo intero della finestra.

I parametri validi sono:

Parametri Descrizione
'altezza=[px]' L'altezza della finestra, con il valore minimo di 100.
'width=[px]' La larghezza della finestra, con il valore minimo di 100.
'left=[px]' La posizione sinistra della finestra.
'top=[px]' La posizione superiore della finestra (solo IE).
'posizione=[booleano]' Se visualizzare o meno il campo dell'indirizzo (il valore predefinito è 1).
'menubar=[booleano]' Se visualizzare o meno la barra dei menu (il valore predefinito è 1).
'resizable=[booleano]' Se la finestra è ridimensionabile o meno (il valore predefinito è 1).
'scrollbars=[booleano]' Se visualizzare o meno le barre di scorrimento (il valore predefinito è 1).
'status=[booleano]' Se aggiungere o meno una barra di stato (il valore predefinito è 1).
'titlebar=[booleano]' Se visualizzare o meno la barra del titolo. Questo viene ignorato a meno che l'applicazione chiamante sia un'applicazione HTML o una finestra di dialogo affidabile (il valore predefinito è 1);
'toolbar=[booleano]' Se visualizzare o meno la barra degli strumenti del browser (l'impostazione predefinita è yes).


Vale anche la pena notare che se l'argomento URL che passate è il nome di una funzione di script (cioè: non inizia con "http://") E la script è chiamata gmcallback_* allora potete ottenere un feed diretto in un evento html5. I nomi delle funzioni script di GameMaker non sono offuscati se iniziano con gmcallback_.

Sintassi:

clickable_add(x, y, tpe, url, target, params);

Argomento Tipo Descrizione
x Real The x position within the window.
y Real The y position within the window.
tpe Inserimento pagina texture The texture page entry for the sprite to be used.
url Stringa The URL (website address) to link to, or the script to be run.
target Stringa This is the target area to open the URL in (see description).
params Stringa The various parameters for the icon (see description).

Ritorna:

Clickable ID

Esempio:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Il codice qui sopra crea un'icona DOM cliccabile nella posizione (32, 32) della pagina su cui sta girando il game canvas. L'icona usa il sprite referenziato dalla pagina texture come "spr_MS_Home" e quando viene cliccata l'icona aprirà una nuova finestra per l'URL specificato e con le proprietà definite per quella finestra.
