cancellare con un clic

Questa funzione deve essere usata per rimuovere un'icona cliccabile creata precedentemente con clickable_add() dalla finestra di gioco.

Sintassi:

clickable_delete(index);

Argomento Tipo Descrizione
index ID cliccabile Index of the clickable icon to remove.

Ritorna:

N/A

Esempio:

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

Il codice qui sopra rimuove l'icona cliccabile indicizzata nella variabile globale "Help_Icon" dalla finestra di gioco, se esiste.
