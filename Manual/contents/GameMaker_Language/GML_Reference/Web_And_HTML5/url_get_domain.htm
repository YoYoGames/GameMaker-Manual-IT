url_get_domain

Puoi usarlo per ottenere il dominio dove il tuo gioco HTML5 viene giocato.

Sintassi:

url_get_domain();

Ritorna:

Esempio:

dom_name=url_get_domain();

Questo memorizzerà il dominio del vostro gioco nella variabile "dom_name".
