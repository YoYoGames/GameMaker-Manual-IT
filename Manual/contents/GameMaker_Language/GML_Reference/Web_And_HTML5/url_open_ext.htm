url_open_ext

Questo aprirà l'URL specificato nel browser del dispositivo di destinazione scelto o, se si utilizza il modulo HTML5, nel browser attualmente aperto. Il parametro "target" che si specifica è lo stesso del valore "name" standard di JavaScript quando si usa il metodo open() e si deve essere consapevoli che tutti i parametri tranne "_self" possono provocare il blocco del browser o chiedere all'utente se desidera consentirlo.

Gli obiettivi validi sono:

Obiettivo Descrizione
_blank Apre il documento collegato in una nuova finestra o scheda (non funziona se i pop-up sono bloccati dall'utente, nel qual caso si possono usare le funzioni clickable_* al posto delle funzioni).
sé stesso Apre il documento collegato nella stessa cornice in cui è stato cliccato (impostazione predefinita).
genitore Apre il documento collegato nel riquadro principale.
In alto Apre il documento collegato nell'intero corpo della finestra.

Sintassi:

url_open_ext(url, target);

Argomento Tipo Descrizione
url Stringa The URL (website address) to link to.
target Stringa This is the target area to open the URL in.

Restituzione:

N/A

Esempio:

url_open_ext("http://gamemaker.io", "_blank");

In questo modo si aprirà la homepage di GameMaker in una nuova finestra.
