Stile cliccabile

Questa funzione permette di impostare le proprietà di stile CSS per il dato pulsante tramite le coppie chiave/valore nel DS Map. È necessario aver creato in precedenza sia l'elemento pulsante (usando clickable_add()) e il DS Map e fornire gli indici memorizzati per ciascuno come argomenti.

Sintassi:

clickable_set_style(index, map)

Argomento Tipo Descrizione
index ID cliccabile The index of the clickable icon to style.
map ID della mappa DS The index of the DS Map da cui impostare lo stile.

Ritorna:

N/A

Esempio:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Il codice qui sopra cambierà lo stile del pulsante cliccabile con l'indice memorizzato nella variabile "button".
