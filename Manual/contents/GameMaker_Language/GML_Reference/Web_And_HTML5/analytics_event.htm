analitica_evento

Questa funzione invierà il testo specificato al provider di analisi che avete impostato attraverso le Opzioni di gioco HTML5. Questa funzione può essere utilizzata per creare un'analitica personalizzata per tracciare qualcosa al di fuori dell'ambito del provider utilizzato.

Sintassi:

analytics_event(string);

ArgomentoTipo Descrizione
string A string to send to the provider

Ritorna:

Esempio:

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

Il codice di cui sopra controllerà una variabile per vedere se è true e se lo è allora un evento analitico speciale sarà attivato con il string specificato.
