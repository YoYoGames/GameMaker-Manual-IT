abilitato a webgl

Questa variabile di sola lettura restituirà se WebGL è abilitato (true) o no (false) per il vostro gioco. Funzionerà solo per quei giochi che girano attraverso un browser (cioè: HTML5), e per tutte le altre piattaforme restituirà true.

Sintassi:

webgl_enabled

Ritorna:

Esempio:

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

Il codice qui sopra controlla il flag WebGL e poi imposta la variabile globale "quality" di conseguenza.
