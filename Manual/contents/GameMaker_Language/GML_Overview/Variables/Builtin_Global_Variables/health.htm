salute

Questa variabile è globale ed è usata per tenere un valore numerico che è solitamente usato per la salute del giocatore. Questa variabile è progettata solo per supportare progetti legacy di versioni precedenti di GameMaker e dovrebbe non essere usato in nuovi progetti perché potrebbe essere deprecata in futuro.

Sintassi:

health;

Ritorna:

Real (single precision floating point value)

Esempio

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

Il codice precedente controlla la variabile health e se è minore o uguale a 0, viene impostata una variabile globale e l'istanza viene distrutta.
