async_load

Questa variabile è globale e viene utilizzata per contenere una mappa DS quando viene utilizzata negli eventi asincroni, e -1 in tutte le altre occasioni. Il contenuto effettivo della mappa DS dipenderà dal tipo di callback dell'evento asincrono, così come dalla funzione che è stata usata per innescare l'evento, quindi fate riferimento alle singole pagine per quegli eventi per i dettagli completi di tutti i possibili contenuti della mappa DS.

Sintassi:

async_load;

Ritorna:

DS Map ID

Esempio esteso:

sprite =  sprite_add("site.com/path/image.png", 16, true, true, 0, 0);

Il codice di cui sopra verrebbe chiamato in un evento per caricare un sprite da un URL esterno. Questo innescherebbe poi l'evento asincrono Image Loaded, dove si analizzerebbe la mappa async_load:

if ds_map_find_value(async_load, "id") == sprite
{
    if (ds_map_find_value(async_load, "status") >= 0)
    {
        sprite_index = sprite;
    }
}

Il codice di cui sopra controllerà prima l'ID della mappa async_load, poi controllerà lo stato della callback. Se il valore è maggiore o uguale a 0 (segnalando il successo) il risultato della callback sarà usato per impostare l'indice sprite dell'istanza sull'immagine appena caricata.
