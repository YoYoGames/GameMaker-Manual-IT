vite

Questa variabile è di portata globale e viene utilizzata per contenere un valore numerico che di solito viene utilizzato per le vite dei giocatori. Questa variabile è progettata solo per supportare progetti legacy di versioni precedenti di GameMaker e dovrebbe non essere usato in nuovi progetti perché potrebbe essere deprecata in futuro.

Sintassi:

lives;

Ritorna:

Real (single precision floating point value)

Esempio

if (lives < 5)
{
    lives += 1;
}

Il codice di cui sopra controlla la variabile lives e se è inferiore a 5, viene aggiunto 1.
