argomento

Questa variabile contiene un array che viene utilizzato insieme alla variabile di sola lettura argument_count nelle funzioni o nei metodi degli script. Ogni posizione dell'array contiene un valore di input per la funzione ed è specificamente per l'uso con funzioni ad argomento variabile .

Si noti che ci sono anche una serie di variabili indipendenti di ambito globale che possono anche essere utilizzate nelle funzioni definite dall'utente per fare riferimento ai diversi argomenti di input: argument0, argument1, argument2, ecc... fino a argument15.

NOTA: questo non è più strettamente richiesto ed è fornito più che altro per il supporto legacy. Tutte le funzioni definite dall'utente accetteranno argomenti variabili, e qualsiasi argomento passato in una funzione che non fa parte degli input nominati sarà inizializzato a undefined per default, che può essere controllato usando la funzione is_undefined().

Sintassi:

argument[n]
argument1
argument2
...
argument15

Ritorna:

Variable (can be of any data type supplied to the function)

Esempio:

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

Il codice qui sopra usa la variabile argument_count insieme all'array argument per inizializzare le variabili, ed è un esempio di come permettere a una funzione definita dall'utente di accettare un numero variabile di argomenti.
