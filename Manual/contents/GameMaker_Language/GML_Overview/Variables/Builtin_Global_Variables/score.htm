punteggio

Questa variabile è globale ed è usata per tenere un valore numerico che è solitamente usato per il punteggio del giocatore. Questa variabile è progettata solo per supportare progetti legacy di versioni precedenti di GameMaker e dovrebbe non essere usato in nuovi progetti perché potrebbe essere deprecata in futuro.

Sintassi:

score;

Ritorna:

Real (single precision floating point value)

Esempio

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

Il codice di cui sopra controlla l'indice object dell'istanza che esegue il codice usando un switche poi aggiunge diverse quantità alla variabile score a seconda di quale object sia.
