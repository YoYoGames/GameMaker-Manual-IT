break

La dichiarazione break è usata per terminare prematuramente un for, repeat, while o do / until loop di qualche tipo, o per dire a un'istruzione switch di terminare in quel punto, o per terminare prematuramente una with chiamata. Consultate le singole pagine per queste diverse funzioni per avere una spiegazione più approfondita di come può essere usata in ogni circostanza. Notate che se break è usato al di fuori di uno di questi contesti darà un errore.

Qui sotto potete vedere alcuni esempi di base di come break può essere usato, e la sua sintassi è semplice:

break;

break in un ciclo for:

for (var i = 0; i < 10; i += 1)
{
    if (array[i] == 234)
    {
        break;
    }
}

num = i;

break in un ciclo repeat:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

break in un ciclo while:

var i = 0;
while (!place_free(x, y))
{
    x = random(room_width);
    y = random(room_height);
    if (i > 50)
    {
        break;
    }
    else
    {
        i+=1;
    }
}

break in un ciclo do / until:

var _id = noone;
do
{
    _id = list[| 0];
    if instance_exists(_id)
    {
        break;
    }
    ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

break quando si usa with:

var count = 0;
with (obj_Enemy)
{
    count++;
    if (count > 10)
    {
        break;
    }
    hp = 100;
}

break in un switch:

switch (keyboard_key)
{
    case vk_left:
    case ord("A"):
        x -= 4;
    break;

    case vk_right:
    case ord("D"):
        x += 4;
    break;

    case vk_up:
    case ord("W"):
        y -= 4;
    break;

    case vk_down:
    case ord("S"):
        y += 4;
    break;
}
