L'istruzione throw runtime vi permette di generare le vostre eccezioni di runtime, terminando il gioco e mostrando un messaggio di errore, usando la seguente sintassi:

throw (<expression>);

L'espressione usata può essere un valore o un string o qualsiasi altro data type, e questo genererà poi un errore di eccezione che viene - per default - mostrato sullo schermo e alla chiusura del messaggio di errore il gioco finirà. Per esempio chiamando questo:

throw ("Hello World!");

causerà il seguente errore di eccezione non gestita:

Potete, tuttavia, prendere il controllo di questo messaggio di errore e usare il vostro codice di gestione chiamando la funzione exception_unhandled_handler(). Questa funzione di runtime vi permette di fornire un codice personalizzato method da utilizzare che sarà chiamato ogni volta che si verifica un'eccezione non gestita nel vostro gioco.
