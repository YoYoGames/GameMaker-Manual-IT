nuovo

L'operatore new può essere usato per creare un struct con una funzione che viene chiamata con gli argomenti dati. La funzione può quindi popolare struct con le variabili degli argomenti - come l'evento create di un'istanza imposterà le variabili iniziali dell'istanza - e quindi l'operatore new restituirà struct. Prima di usare questo operatore, è importante notare che la funzione data deve essere contrassegnata come funzione constructor , altrimenti l'operatore new non creerà struct (l'esempio di codice qui sotto lo mostra e per ulteriori informazioni, si veda la pagina sulle strutture).

NOTA È possibile controllare un sito struct per scoprire quale funzione è stata usata per crearlo, usando la funzione runtime. instanceof().

Questo operatore ha la seguente sintassi:

<variable> = new <function> (<argument>, <argument>, ...);

Quando si utilizza l'operatore new insieme a una funzione, viene restituito un riferimento a struct che viene memorizzato nella variabile data, consentendo di accedere a struct e a qualsiasi variabile creata al suo interno dalla funzione.

L'esempio seguente mostra una funzione definita per essere utilizzata come costruttore:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Questa funzione può essere usata insieme all'operatore new per creare un struct e popolarlo con le variabili impostate sui valori degli argomenti usati nella funzione, come in questo caso:

mystruct = new init_struct(10, 100, "Hello World");

È importante notare che la chiamata a new su qualsiasi funzione che non sia stata segnalata come costruttore causerà un'eccezione di runtime.
