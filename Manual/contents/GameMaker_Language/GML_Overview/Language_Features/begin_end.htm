Il GameMaker Language permette di usare le parole chiave begin e end invece delle più usuali parentesi graffe {} quando si creano blocchi di codice. L'esempio di codice qui sotto mostra come funziona:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

Si noti che l'uso di queste parole chiave non è tipico ed è fornito come parte del linguaggio più per il supporto legacy che per qualsiasi altra cosa, e in qualsiasi momento nel futuro potrebbero essere deprecate.
