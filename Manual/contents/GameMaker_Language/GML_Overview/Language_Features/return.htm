ritorna

La dichiarazione return ha la seguente sintassi:

return (<expression>)

Si usa solo l'istruzione return nelle funzioni e nei metodi di script, ed è usata per restituire un valore dalla funzione da usare in ulteriore codice o chiamate di funzioni. Va notato che l'esecuzione della funzione termina all'istruzione return , il che significa che qualsiasi codice che viene dopo che return è stato chiamato non sarà eseguito. Ecco un breve esempio script chiamato sqr_calc che calcola il quadrato di qualsiasi valore gli venga passato, e include la cattura degli errori nel caso in cui il parametro che gli viene passato non sia un numero reale:

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

Per chiamare una funzione dall'interno di un pezzo di codice, basta usarla nello stesso modo in cui si chiamano le funzioni runtime - cioè, scrivere il nome della funzione con i valori dei parametri tra parentesi. Quindi, la funzione di cui sopra verrebbe chiamata in questo modo:

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

Qui la variabile val sarà o 0 (perché la variabile amount non è un numero reale) o il totale per value * value.
