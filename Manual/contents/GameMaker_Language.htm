Linguaggio GameMaker

Potete usare GameMaker Language per programmare i vostri giochi in due modi diversi: posizionando blocchi per "scrivere" codice visivamente, o scrivendo codice manualmente.

La prima opzione, GML Visual, è più adatta ai principianti e agli hobbisti che vogliono iniziare a fare i loro giochi velocemente.

La seconda opzione, GML Code, vi permette di sfruttare tutta la potenza di GameMaker Language scrivendo il vostro codice manualmente.

Mentre GML Visual è più facile per iniziare, GML Code è ancora un linguaggio di programmazione facile da usare con potenti caratteristiche(funzioni, metodi, strutture e costruttori, ecc.).

Creazione di script visivi/codici

Vi viene chiesto di scegliere tra GML Code e GML Visual per creare un nuovo evento Object, un nuovo Script asset, o qualsiasi altra cosa che si traduce nella creazione di uno script:

Oltre alla scelta tra GML Code e GML Visual, questa finestra contiene due caselle di controllo:

  • Non chiedere di nuovo per questo progetto: Quando questo è abilitato, imposta l'opzione di gioco "Default scripting language" alla tua nuova selezione, così non ti viene chiesto di scegliere di nuovo per il progetto corrente.
  • Non chiederlo mai più: Quando questo è abilitato, imposta la preferenza "Linguaggio di scripting predefinito" alla tua nuova selezione, così non ti viene mai chiesto di scegliere di nuovo, per nessun progetto.
