Il menu File

All'indirizzo file menu è possibile scegliere di modificare, salvare o importare i progetti e impostare le preferenze di GameMaker IDE. Le opzioni disponibili sono:

  • Nuovo progetto - Serve per creare un nuovo progetto da zero in GameMaker. Si aprirà la finestra di dialogo "Salva" e si chiederà di scegliere una posizione in cui salvare il nuovo progetto. Dopo aver dato un nome al progetto e una posizione in cui salvarlo, si accede a una nuova area di lavoro vuota per iniziare a lavorarci. I file di progetto vengono salvati con il file *.yyp extension .
  • Apri progetto - Apre un file explorer per selezionare un file di progetto *.yyz precedentemente creato per il caricamento. Se il progetto è stato creato con una versione inferiore alla 2.3.0, al momento dell'importazione verrà visualizzato il seguente messaggio di avviso: Se si sceglie di procedere, verrà aperto un file explorer e sarà possibile assegnare al progetto un nome e una posizione di salvataggio aggiornati, in modo che il progetto originale non venga sovrascritto. Una volta convertito nel nuovo formato 2.3.0+, il progetto si aprirà in IDE e verrà visualizzato un messaggio che indica che la conversione è riuscita. È necessario controllare il rapporto di compatibilità generato dal processo di conversione e verificare se c'è qualcosa da correggere prima di provare a eseguire o compilare il progetto. Se si verificano errori ripetuti durante il tentativo di aprire un progetto realizzato con una versione precedente alla 2.3.0, è necessario contattare YoYo Games (utilizzando l'opzione "Segnala un bug" del menu Aiuto), fornendo il file originale *.yyp che si sta cercando di convertire.
  • Importa progetto - Qui è possibile importare progetti che sono stati precedentemente esportati come archivio *.yyz (vedere "Esporta progetto", sotto), oppure - se si dispone di un progetto realizzato con il precedente GameMaker: Studio 1.4 - è possibile selezionare questa opzione per importare il file *.gmx o *.gmz di quella linea di prodotti. Come per l'apertura di un progetto, se il file *.yyz è stato creato con una versione di GameMaker precedente all'aggiornamento 2.3.0, dovrà essere convertito poiché il formato del file di progetto è cambiato con l'aggiornamento.

NOTA: Per importare un progetto precedente di GameMaker in GameMaker è necessario che sia stato creato con la versione 1.4, e anche in questo caso potrebbe non funzionare direttamente all'importazione. Molte cose sono diverse tra GameMaker e GameMaker: Studio 1.4, per cui i progetti più vecchi potrebbero richiedere alcune modifiche per funzionare. Consultate l'articolo Porting A GM:S 1.4 Game To GameMaker sulla Knowledge Base di GameMaker e la sezione del manuale dedicata alle funzioni obsolete e agli script di compatibilità.

IMPORTANTE! L'importazione di un progetto *.yyz che contiene risorse font può richiedere la "rigenerazione" di fonts rimuovendole/aggiungendole, oppure modificando font nell'editor font e poi reimpostandole. Questo perché le informazioni esistenti di font vengono mantenute da *.yyz quando viene esportato e, al momento dell'importazione successiva o su un'altra macchina, il progetto potrebbe non avere già installato il fonts richiesto (o potrebbe avere un font diverso con lo stesso nome). Questa operazione deve essere eseguita solo se si riscontrano problemi con il rendering di font dopo l'importazione.

  • Progetti recenti - Questa opzione consente di visualizzare i progetti utilizzati di recente. Selezionandola, si riapre la Pagina iniziale (se è stata chiusa) e si può selezionare un progetto usato di recente dall'elenco che vi si trova.
  • Salva progetto - Salva il gioco nella stessa posizione definita al momento della creazione del progetto (o da dove è stato aperto).
  • Salva progetto con nome - Come sopra, solo che si specifica la posizione e il nuovo nome del progetto.
  • Esporta progetto - Consente di esportare il progetto come file o come modello:
    • YYZ - Esporta il progetto come un singolo file *.yyz ed è molto utile per inviare copie a persone o per salvare backup.
    • Come modello - Consente di esportare il progetto come modello, da utilizzare in seguito come base per altri progetti di The Start Page. Selezionando questa opzione, verrà visualizzato il seguente messaggio:
      Per prima cosa si deve fare clic su "Modifica informazioni sul modello" per assicurarsi che le informazioni sul modello siano corrette, quindi selezionare "Esporta" quando si apre questa richiesta una seconda volta per salvare il progetto come modello. Il modello verrà quindi visualizzato nel menu Modello della schermata iniziale.
  • Preferenze - Selezionando questa opzione si accede alla finestra delle preferenze, dove è possibile impostare l'aspetto di GameMaker.
  • New IDE - Quando si seleziona questa opzione, viene aperta una nuova istanza di IDE da utilizzare (si aprirà nella pagina iniziale).
  • Il mio account - Questa opzione apre il browser Web predefinito e porta alla pagina del proprio account YoYo.
  • Logout - Selezionando questa opzione si esce da GameMaker. Sarà necessario effettuare nuovamente il login con la stessa licenza (o con un'altra) per poter utilizzare il prodotto in seguito.

IMPORTANTE! Se si utilizza una licenza gratuita per GameMaker e poi si acquista una delle altre licenze, potrebbe essere necessario uscire dal sito IDE, riavviare GameMaker e riaccedervi per assicurarsi che la nuova licenza venga applicata correttamente (anche se ciò dovrebbe avvenire automaticamente).

  • Esci - Esce da GameMaker sul desktop.
