Il menu dei layout

Questo menu è relativo al salvataggio e al caricamento di diversi layout IDE. Si può impostare il IDE come si vuole, con alcuni elementi (come il Browser Asset, o l'Output Windows) disposti diversamente da quelli di default. Questo vi permette di impostare GameMaker per adattarlo al vostro flusso di lavoro e con le opzioni di questo menu potete salvare i layout del progetto corrente come un file *.xml che può essere caricato in qualsiasi progetto per impostare gli stessi layout. Le opzioni in questo menu sono le seguenti:

  • Carica Layout - Questo aprirà un esploratore di file dove è possibile selezionare un file *.xml di layout precedentemente salvato per il caricamento.
  • Save Layout - Questo aprirà un esploratore di file dove è possibile nominare e salvare il layout corrente IDE come un file *.xml.
  • Reset Layout - Questo ripristina il layout corrente di IDE alle impostazioni predefinite.
