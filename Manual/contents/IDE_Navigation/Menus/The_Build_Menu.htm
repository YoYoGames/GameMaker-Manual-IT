Il menu Build

Il menu build è dove puoi scegliere di costruire il tuo progetto sia per i test, debugging o come eseguibile finale. Le opzioni disponibili sono:

  • Esegui - Esegui il tuo progetto attuale sulla piattaforma di destinazione scelta per il test. Per le diverse opzioni di compilazione disponibili durante la compilazione, vedi la sezione Compilazione. Nota che questo comando è lo stesso che premere il pulsante Run nel sito IDE.
  • Debug - Esegui il tuo progetto corrente per il test con debugger in modo da poter vedere informazioni dettagliate sulle prestazioni e fare il debug di qualsiasi errore o problema che potresti avere. Per maggiori informazioni su debugging, vedi la sezione Debug. Nota che questo comando è lo stesso che premere il pulsante Debug in IDE.
  • Pulisci - GameMaker mantiene una Cache di file per aiutare a velocizzare il tempo di compilazione durante i test. Tuttavia a volte un cache "stantio" può portare a strani e inaspettati errori nel vostro gioco, nel qual caso la prima cosa da fare è pulire il cache da qui e ri-testare. È anche una buona idea pulire cache prima di creare un eseguibile finale per la piattaforma di destinazione. Notate che questo comando è lo stesso che premere il pulsante Clean Cache nel IDE.
  • Create Executable - Questo compilerà il progetto corrente e creerà un pacchetto eseguibile (o pacchetti) per la piattaforma di destinazione. Prima aprirà il file explorer in modo che tu possa selezionare dove vuoi salvare il progetto e poi compilare il gioco. Vedi la sezione sulla compilazione per maggiori informazioni sull'impostazione degli obiettivi di compilazione. Nota che questo comando è la stessa cosa che premere il pulsante Create Executable (Crea eseguibile ) in IDE.
