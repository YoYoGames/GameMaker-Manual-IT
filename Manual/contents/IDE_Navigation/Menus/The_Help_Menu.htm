Il menu di aiuto

Questo menu è relativo alla richiesta di aiuto per l'utilizzo di GameMaker. Contiene le seguenti voci di menu:

  • Apri manuale - Apre il manuale di aiuto. Si noti che anche facendo clic sul pulsante della guida nella barra dei menu si apre il documento, così come premendo .
  • Knowledge Base - Questa opzione vi porterà alla Knowledge Base di GameMaker, dove potrete trovare articoli sulla configurazione della piattaforma, sui problemi noti, sull'uso di GameMaker e su molti altri argomenti.
  • Segnala un bug - Se si riscontrano bug o problemi durante l'utilizzo di GameMaker, è possibile segnalarli qui.
  • Open Log In Explorer - Apre la directory del sistema in cui si trova il file ui.log, solitamente necessario per inviare segnalazioni di bug relativi all'IDE.
  • GameMaker.io - Apre il sito web di GameMaker.
  • GameMaker Community - Questo aprirà i forum della GameMaker Community nel browser. Si noti che si tratta di un forum gestito dalla comunità , quindi se avete una segnalazione di bug o avete bisogno di assistenza tecnica dovete prima rivolgervi al sito di supporto ufficiale (vedi sopra).
  • Note di rilascio - Questa opzione apre le note di rilascio di GameMaker nel browser. Nel caso in cui sia necessario eseguire il rollback di un aggiornamento, è possibile ottenere i programmi di installazione della versione precedente facendo clic sul collegamento al numero di versione in questa pagina.
  • Note di rilascio attuali di Runtime - Poiché il Runtime può essere aggiornato indipendentemente da IDE, ha il proprio set di note di rilascio del Runtime. Selezionando questa opzione si apriranno nel browser, ma si noti che l'ultimo runtime potrebbe non essere quello in uso. Per sapere su quale runtime ci si trova o per informazioni su come aggiungere, rimuovere o attivare runtimes, consultare la sezione Preferenze del Runtime.
  • SDK necessari - A causa della natura multipiattaforma di GameMaker, sono necessari alcuni SDK di terze parti. Facendo clic su questo link si accede a un articolo di supporto che illustra in dettaglio i requisiti necessari per ogni piattaforma.
  • Apri progetto in Explorer - Apre la posizione di salvataggio del progetto attualmente caricato nel file explorer del sistema operativo.
  • Aggiorna caratteri di sistema - Questa operazione aggiorna icaratteri di sistema fonts che GameMaker è in grado di vedere, ed è utile per rilevare nuovi fonts installati mentre IDE è aperto.
  • Licenze - Apre una finestra contenente le licenze di tutte le librerie di terze parti utilizzate in GameMaker.
  • Informazioni su - Apre la finestra "About GameMaker", dove è possibile ottenere informazioni sulla versione del prodotto (essenziale per segnalare eventuali bug o problemi) e vedere i crediti.
