Menu

La parte superiore di GameMaker IDE è occupata dalla barra dei menu. Qui potete trovare una serie di menu a discesa che possono essere utilizzati per navigare IDE , eseguire operazioni o aprire finestre aggiuntive con strumenti che vi aiuteranno a creare i vostri giochi. Notate che qui elenchiamo i menu di default che sono disponibili, ma alcuni degli editor di asset aggiungeranno ulteriori menu a seconda del contesto. Questi menu aggiuntivi sono spiegati nelle pagine relative all'editor che li ha creati.

I menu predefiniti disponibili sono:

  • Il menu File
  • Il menu Modifica
  • Il menu Build
  • Il menu Windows
  • Il menu Strumenti
  • Il menu Marketplace
  • Il menu dei layout
  • Il menu Aiuto
