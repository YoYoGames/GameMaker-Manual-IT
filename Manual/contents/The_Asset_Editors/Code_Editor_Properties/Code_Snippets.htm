Frammenti di codice

Uno strumento molto utile che avete a disposizione quando modificate il vostro codice è l'uso dei Code Snippets. Quando si usa l'editor script, si può premere F4 per aprire il pop-up di code snippet, che permette di selezionare uno dei metodi di codice comunemente usati da una lista:

Da questa finestra puoi selezionare con il mouse lo snippet da usare, o puoi premere il tasto di scelta rapida associato elencato sulla destra. Questo aggiungerà lo snippet al tuo codice per la modifica.

Puoi anche definire i tuoi frammenti di codice se vuoi. Prima di fare questo, è necessario creare un file chiamato "snippets.txt" in una delle seguenti directory:

  • Finestre: %ProgramData%\GameMaker\User\
  • macOS: /Users/Shared/GameMaker/User/

Questa cartella non viene modificata durante qualsiasi aggiornamento di GameMaker, quindi il tuo file modificato rimarrà intatto (ma qualsiasi modifica al file snippets di base nella directory di installazione verrà ripristinata).

Una volta che il file è stato creato nella directory User, è possibile modificarlo con qualsiasi editor di testo seguendo queste regole:

  • Ogni snippet deve essere su una linea separata (non ci dovrebbero essere righe vuote)
  • Ogni snippet deve iniziare con il tasto di scelta rapida da utilizzare seguito da "-" e poi il nome dello snippet (che è quello che viene mostrato nel menu) seguito da due punti ":", ad esempio:

I - Creare un'istanza:

Dopo i due punti aggiungete il frammento di codice

I - Instance Create:instance_create_layer(x, y, |layer|, object);

Il codice che si aggiunge deve anche seguire un formato specifico dove:

  • Tutto il codice è su una sola linea
  • Si usa "$" per un newline
    • NOTA Quando si creano più newlines, ci deve essere uno spazio tra ogni segno "$". Per esempio, si dovrebbe scrivere $ $ $ per aggiungere tre newline.
  • Si mette il testo che deve essere evidenziato tra i simboli "|". L'editor di codice selezionerà automaticamente questo testo per la modifica quando lo snippet viene aggiunto.

Vedi gli esempi che sono già nel file base snippets per vedere come è impostato seguendo le regole di cui sopra. Potete trovare il file base nella directory di installazione di GameMaker:

  • Windows: GameMaker\TextEditor\snippets.txt
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
