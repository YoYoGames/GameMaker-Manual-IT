Notifiche push

L'evento di notifica push viene attivato dal callback di notifiche push sul sistema operativo del dispositivo, sia da una fonte locale che da una fonte remota. Genera una mappa DS che è esclusiva di questo evento ed è memorizzata nella variabile speciale async_load.

Questo evento è utilizzato esclusivamente da extensions (come Firebase Cloud Messaging extension), pertanto si consiglia di consultare la documentazione dell'estensione per le istruzioni sull'uso di questo evento.

Indietro: Eventi asincroni
Avanti: Salva / Carica
