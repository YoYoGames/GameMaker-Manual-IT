Variabili oggetto

Quando cliccate sul pulsante Variables si aprirà la finestra Object Variables , che potete usare per generare qualsiasi numero di variabili prima dell 'evento Create dell'oggetto:

In questa finestra potete cliccare sul pulsante Add per aggiungere una nuova variabile alla lista. Potete quindi dare un nome alla variabile così come impostare il suo tipo e regolare le sue proprietà:

Il nome della variabile deve iniziare con una lettera e può contenere solo lettere, numeri e il simbolo di sottolineatura '_' con una lunghezza massima di 64 simboli. Quindi, le variabili valide sono cose come fish, foo_bar, num1, e le variabili non valide sarebbero 6fish, foo bar, o *num. Una volta dato il nome della variabile è necessario impostare il suo tipo, che può essere uno qualsiasi dei seguenti:Vale la pena notare che questa caratteristica è particolarmente utile quando si lavora con le istanze genitore/figlio - poiché si può avere un genitore object con un set di variabili definite e poi semplicemente modificarle nel figlio object (vedere la sezione Variabili oggetto e oggetti figlio più avanti per maggiori dettagli) - e per quando si aggiungono istanze nell'Editor delle stanze (vedere la sezione sui livelli > Livello istanza), poiché si possono anche modificare queste variabili per istanze individuali che sono state posizionate in room.

I diversi tipi di variabili che potete creare sono elencati qui sotto:

  • Reale: Un numero reale è qualsiasi numero che non è un intero e può essere positivo o negativo. Quindi, 124.5, 45639.566546456, 0.9, -45.5 ecc... sono tutti esempi di numeri reali. Tutti i numeri reali sono memorizzati come valori in virgola mobile a 64bit (doppi). I dati reali (e interi) hanno un'opzione per impostare un intervallo di valori a cui si può accedere cliccando il pulsante Options :Quando si seleziona questa opzione si può inserire un valore iniziale e uno finale e invece di avere un valore fisso mostrato per la variabile nella finestra Object Variables, avrete un cursore che è vincolato a questi valori:
  • Numero intero: Un intero è un numero intero e può essere positivo o negativo, per esempio 30004, 19, 0, -300. Notate che potete impostare un intervallo di valori per l'intero in uscita, come spiegato sopra per i numeri reali.
  • Stringa: Un string è qualsiasi cosa che è stata messa tra virgolette doppie, per esempio "fish", "Hello World", o "12345".
  • Booleano: Un booleano è un valore che può essere vero o falso. Nella finestra Variabili dell'editor Object, è semplicemente mostrato come una casella che si seleziona per vero e si deseleziona per falso.
  • Espressione: Un'espressione è una frase matematica che può contenere numeri ordinari, variabili, strings, o funzioni GML così come uno o più operatori. Per esempio sqrt(85 * 6) + 5.5 è un'espressione.
  • Risorsa: Una risorsa è semplicemente una qualsiasi delle risorse che avete definito nel Browser di Asset. Quando si seleziona questo tipo di risorsa, è possibile fare clic sul pulsante Open Asset Explorer per aprire il Asset Explorer e selezionare la risorsa richiesta.
  • Lista: Selezionare un input di tipo lista significa che è possibile creare una selezione di valori (questi possono essere strings, reali, espressioni ecc...) di cui si può poi scegliere uno o più per la variabile da restituire. Per definire gli elementi che vanno nella lista dovete prima aggiungerli nelle Opzioni (cliccate sul pulsante ):

  • Colore: Il tipo di colore serve per definire un valore di colore da memorizzare nella variabile. Puoi inserire un numero reale (da 0 a 16777216, questo avrà un valore alfa di 255), o un valore esadecimale (nel formato di $RRGGBBAA), o puoi potete fare doppio clic su sul campione di colore per aprire il Colour Picker e definire il colore lì.

Variabili oggetto e oggetti figli

Una caratteristica importante delle variabili di Object è che sono ereditate da qualsiasi figlio objects che avete nel Browser di Asset (vedere la sezione sugli oggetti padre per maggiori informazioni), il che significa che potete può scegliere di sovrascrivere o cambiare una o tutte queste variabili, se lo desidera.

Quando si crea un figlio object di un'altra risorsa object che ha variabili definite per essa, queste appariranno nella finestra Object Variables come questa:

Potete vedere nell'immagine superiore che il genitore object ha quattro variabili Object e nell'immagine inferiore queste sono mostrate, solo "in grigio" e hanno l'icona Inherited From Parent per indicare che sono state ereditate. Queste variabili ereditate possono essere modificate se si fa clic sul pulsante Override Variable , in modo da poter regolare il cursore dell'intervallo o i valori, o selezionare elementi diversi dalle liste, ecc... Notate che quando modificate una variabile padre potete solo cambiare il valore definito ma non il nome, né potete cambiare le opzioni della variabile. Notate anche che potete aggiungere nuove variabili Object anche ai figli objects (e nell'immagine di esempio qui sopra puoi vedere che "Character Type" è una nuova variabile Object solo per il figlio object).

Se avete modificato una variabile ereditata Object, allora potete cliccare sul pulsante Delete per rimuovere la modifica, ma non la variabile. La variabile sarà sarà ancora ereditata dal genitore, ma userà di nuovo il valore predefinito del genitore. Se avete bisogno di rimuovere completamente la variabile, allora dovete farlo dal genitore object.
