L'evento è progettato per catturare e trasmettere i dati che sono stati assegnati a particolari eventi dell'animazione sprite come definiti nell'editor utilizzato per realizzarli.

Quando l'evento di animazione viene attivato, viene creata una mappa DS speciale, chiamata mappa event_data (più o meno come la mappa async_load per un evento async). Questa mappa conterrà tutte le seguenti coppie chiave/valore, che possono essere analizzate per ottenere informazioni sull'evento:

  • "name": il nome dell'evento (come definito nel programma di animazione)
  • "track": l'indice della traccia su cui viene riprodotta l'animazione (l'impostazione predefinita è 0)
  • "integer": un valore intero associato all'evento (come definito nel programma di animazione, l'impostazione predefinita è 0)
  • "float": un valore float associato all'evento (come definito nel programma di animazione, l'impostazione predefinita è 0,0)
  • "string": un valore string associato all'evento (come definito nel programma di animazione, il valore predefinito è un string vuoto "")
