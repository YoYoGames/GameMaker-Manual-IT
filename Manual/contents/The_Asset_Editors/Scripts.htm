L'editore Script

Questa sezione si occupa dell'editor di script (chiamato anche editor di testo), in cui si inserisce tutto il codice necessario per l'esecuzione del gioco. Questo editor viene utilizzato per creare script assets, per codificare gli eventi degli oggetti, per aggiungere il codice di creazione delle stanze e per molte altre cose.

Questa sezione illustra semplicemente il funzionamento dell'editor script e non come creare script assets né come usare GML Code o GML Visual per creare scripts. Per questo si rimanda alle sezioni seguenti:

  • Codice GML Scripts
  • Script visivi GML

Quando si crea un sito script asset, può essere richiesto di scegliere tra GML Visual e GML Code. Per ulteriori informazioni, consultare GameMaker Language.

L'editor script è il luogo in cui si scrive il codice GameMaker Language, che è il linguaggio di programmazione integrato di GameMaker (per maggiori dettagli, vedere la sezione Linguaggio di GameMaker ). Quando si acquisisce familiarità con GameMaker e si desidera utilizzarlo al meglio, è consigliabile iniziare a imparare a usare questo linguaggio, che amplia notevolmente le possibilità di creazione di giochi.

Un script asset può essere rinominato facendo clic con il tasto destro del mouse su nel Browser delle risorse e selezionando Rinomina, ma si noti che il nome script deve essere conforme alle regole di scripting per assets, quindi può essere solo alfanumerico, deve iniziare con una lettera e l'unico simbolo che può contenere è il simbolo "_" sotto la barra. L'editor script in altri luoghi non può essere rinominato e avrà un nome specifico per ciò che si sta modificando, come ad esempio un evento object.

Editor del codice principale

L'editor di codice principale è il luogo in cui si scriverà tutto il codice per creare il sito script asset o per riempire un evento object. L'edizione vera e propria del codice è trattata nella sezione Modifica, più avanti in questa pagina.
